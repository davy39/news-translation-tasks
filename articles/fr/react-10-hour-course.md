React est une bibliothèque JavaScript open-source populaire pour construire des interfaces utilisateur.

React a une API légère, un écosystème robuste et en évolution, et une grande communauté.

Nous avons publié un cours complet de 10 heures sur la chaîne YouTube freeCodeCamp.org qui enseigne tout ce que vous devez savoir sur React.

Ce cours a été créé par John Smilga de Coding Addict. John est un enseignant expérimenté qui a créé plus de 500 tutoriels de codage.

Dans ce cours, vous apprendrez :

* Les fondamentaux de React
* Les hooks React
* L'API de contexte
* Le routeur React
* Les hooks personnalisés
* Et plus encore

À la fin de cette vidéo, vous aurez une bonne compréhension de l'utilisation de React pour votre prochain projet.
