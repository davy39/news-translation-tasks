Par Vishnu Chilamakuru

Dans cet article, je vais partager quelques-unes des extensions que j'utilise pour rendre le développement avec Ruby on Rails plus facile et plus amusant.

## Pourquoi utiliser ces outils ?

Les outils de développement jouent un rôle significatif dans la vie d'un développeur. Si vous êtes un développeur junior et que vous commencez tout juste à travailler sur des projets, alors connaître les outils appropriés est une nécessité. 

Ces outils peuvent vous faire gagner beaucoup de temps et ils vous permettent de coder plus efficacement et ainsi augmenter votre productivité.

Si vous êtes un développeur Ruby on Rails à la recherche d'outils de développement gratuits, je vous recommande Visual Studio. Il dispose d'une tonne d'extensions, comme celles mentionnées ci-dessous, et elles m'ont aidé à augmenter considérablement ma productivité.

*Note : Toutes les extensions Visual Studio sont disponibles sur le* [_Visual Studio Marketplace_](https://marketplace.visualstudio.com/) *gratuitement.*

Alors, plongeons-nous dans le sujet.

## [Ruby](https://marketplace.visualstudio.com/items?itemName=rebornix.Ruby)

Avec ~1,3 million de téléchargements, il s'agit de l'une des extensions les plus populaires pour Ruby. Elle fournit un support amélioré pour le langage Ruby et le débogage. 

Avec un support de débogage amélioré, les développeurs peuvent définir des points d'arrêt et inspecter les variables locales et globales en mode débogage. Cela aide à déboguer facilement et rapidement tout problème. 

Cette extension prend également en charge la mise en forme du code via **_rubocop_**, ce qui est très utile lorsque vous travaillez avec une équipe de développeurs pour maintenir un format de code cohérent.

L'extension Ruby offre les fonctionnalités suivantes :

> - Détection automatique de l'environnement Ruby avec support pour rvm, rbenv, chruby et asdf  
>   
> - Support de linting via RuboCop, Standard et Reek  
>   
> - Support de mise en forme via RuboCop, Standard, Rufo et RubyFMT  
>   
> - Support de base pour IntelliSense  
>   
> - Support de débogage Ruby  
>   
> Source : [Ruby](https://marketplace.visualstudio.com/items?itemName=rebornix.Ruby)

## [Rails](https://marketplace.visualstudio.com/items?itemName=bung87.rails)

Il s'agit d'une autre extension populaire pour Rails qui fournit un support Rails amélioré. 

La templating ERB HTML est largement utilisée dans l'écosystème Rails en tant que vues pour rendre les pages HTML des sites web. L'extension Rails prend en charge la syntaxe `.erb` et fournit également une auto-complétion pour les balises HTML populaires comme stylesheet, meta tags, asset tags, et ainsi de suite. 

Cette extension aide également à basculer facilement entre les vues Rails (`*.erb` files). Elle vous aide également à voir la documentation en ligne de toute méthode ou commande facilement côte à côte.

Voici quelques-unes des fonctionnalités prises en charge par cette extension :

> - Fragments de code Ruby on Rails « Asset Helpers » et « Tag Helpers ».  
>   
> - Mise en évidence de la syntaxe .erb.  
>   
> - Navigation entre les fichiers liés via une commande.  
>   
> - Aller à la Définition.  
>   
> - Suggestion de chemin de vue, suggestion de méthode statique du Modèle et suggestion de champ du Modèle.  
>   
> - Ouvrir la documentation en ligne sur le côté via une commande.  
>   
> Source : [Rails](https://marketplace.visualstudio.com/items?itemName=bung87.rails)

## [Ruby Solargraph](https://marketplace.visualstudio.com/items?itemName=castwide.solargraph)

Ruby Solargraph est l'une des extensions les plus utiles de cette liste, et fournit IntelliSense, la complétion de code et la documentation en ligne pour Ruby. 

La documentation en ligne vous aide à visualiser toutes les méthodes autorisées de la classe/objet, et vous aide également à comprendre facilement la définition de chaque méthode et ses arguments.

Il s'agit de l'une des extensions que j'ai personnellement utilisées à de nombreuses reprises pour me référer à la documentation d'une méthode Ruby, aux arguments d'une méthode, et ainsi de suite.

## [Vscode Endwise](https://github.com/kaiwood/vscode-endwise)

Il s'agit de mon extension préférée qui peut vous faire gagner beaucoup de temps et vous éviter des maux de tête. Cette extension ajoute automatiquement `end` à tous vos blocs de code Ruby.

![Image](https://www.freecodecamp.org/news/content/images/2020/08/endwise.gif)
_Image de [VSCode Endwise](https://github.com/kaiwood/vscode-endwise)_

## [Rails Db Schema](https://marketplace.visualstudio.com/items?itemName=aki77.rails-db-schema)

Cette extension vous aide à définir un schéma de base de données et active également l'auto-complétion pour les schémas de base de données Rails. 

Lors de la définition de schémas ou de la création de tables pour toute entité, cette extension active et complète automatiquement la syntaxe pour tous vos DDL (langage de définition de base de données) comme `create_table`, `create_index`, `delete_table`, `update_table`, et ainsi de suite. 

![Image](https://www.freecodecamp.org/news/content/images/2020/08/definition.gif)
_Image de [VSCode DB Schema](https://marketplace.visualstudio.com/items?itemName=aki77.rails-db-schema)_

Elle aide à l'auto-complétion de tous les attributs de toute entité de base de données. Par exemple, si `User` a les attributs `email`, `name` et `date_of_birth`, cette extension détectera automatiquement la définition de l'entité et complétera automatiquement ses attributs lorsque vous tapez `User`. 

![Image](https://www.freecodecamp.org/news/content/images/2020/08/autocompletion.gif)

## Pourquoi utiliser Visual Studio ?

Il existe de nombreuses autres options d'IDE pour les développeurs Ruby on Rails comme RubyMine (la version entreprise), Sublime, Vim, et ainsi de suite. 

Mais mon préféré personnel est Visual Studio car il dispose d'un support extensif pour de multiples langages comme Golang, PHP, Node.js, et plus encore. C'est donc l'IDE par défaut, surtout pour les développeurs polyglottes.

Même si Visual Studio manque de quelques fonctionnalités par rapport à RubyMine comme l'obtention de support pour les dernières mises à jour de version de Rails, il couvre la majorité des fonctionnalités requises pour le développement via des extensions communautaires.
