Avec tous les outils et programmes disponibles, il peut être difficile de déterminer la meilleure façon de configurer votre environnement de développement sur votre ordinateur.

Dans cette présentation, le développeur web professionnel Daniel Deutsch partage comment il configure son environnement. Il présente les programmes qu'il utilise pour son travail et comment il les configure, y compris les avantages de sa configuration. Enfin, il montre une démonstration de son flux de travail.

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=K8tsFn9HbAQ) (19 minutes de visionnage).
