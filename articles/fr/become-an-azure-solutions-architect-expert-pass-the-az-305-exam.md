Êtes-vous prêt à élever votre expertise dans la conception de solutions cloud et hybrides sur Azure ?

L'examen de certification Azure Solutions Architect Expert évalue votre capacité à accomplir les tâches techniques suivantes : concevoir des solutions d'identité, de gouvernance et de surveillance ; concevoir des solutions de stockage de données ; concevoir des solutions de continuité d'activité ; et concevoir des solutions d'infrastructure.

Nous venons de publier un cours de 10 heures sur la chaîne YouTube freeCodeCamp.org qui vous préparera à passer cet examen de certification. Ce cours est votre clé pour maîtriser les essentiels de Microsoft Azure.

Andrew Brown d'ExamPro a créé ce cours. Il a créé de nombreux cours populaires de préparation aux examens qui ont aidé des milliers de personnes à obtenir une certification professionnelle.

Ce cours n'est pas simplement une série de conférences ; c'est un voyage à travers le vaste paysage de l'architecture Azure. Le cours couvre tous les sujets que vous devez apprendre pour passer l'examen AZ-305. Voici les sujets abordés dans le cours :

**☁️ Introduction :** Commencez votre voyage avec une analyse détaillée du guide de l'examen, offrant des informations sur ce à quoi vous attendre et comment naviguer efficacement dans le cours. Explorez la fonctionnalité Sandbox, un outil essentiel pour l'apprentissage pratique.

**☁️ Solutions d'identité :** Plongez dans le monde de Microsoft Entra ID, comprenez son récent changement de nom, ses applications et comparez AD vs. AAD. Cette section couvre en profondeur les locataires, les utilisateurs, les groupes et les aspects des droits d'accès, ainsi que l'authentification unique et l'accès conditionnel.

**☁️ Contrôle d'accès basé sur les rôles :** Découvrez les différents rôles Azure et leurs fonctionnalités. Apprenez-en davantage sur les contrôles d'accès, les administrateurs classiques, le RBAC Azure et les nuances des stratégies Azure vs. RBAC des rôles Azure.

**☁️ Stratégies Azure :** Une section dédiée pour comprendre la portée des stratégies Azure et l'anatomie des fichiers de définition des stratégies Azure, ainsi que la configuration de ces stratégies pour une gouvernance cloud optimale.

**☁️ Azure Resource Manager & Modèles ARM :** Ces sections sont essentielles pour comprendre la gestion et le déploiement des ressources dans Azure. Apprenez-en davantage sur les groupes de gestion Azure, les abonnements, les balises de ressources et plus encore. Les modèles ARM vous guideront à travers la gestion des ressources, des paramètres et des fonctions dans Azure.

**☁️ Solutions de surveillance :** Acquérez une expertise dans Azure Monitor, ses magasins de données, ses journaux et ses analyses. Ce module comprend des leçons détaillées sur Azure Sentinel et Application Insights.

**☁️ Bonnes pratiques de gestion de l'identité :** Une partie cruciale du cours axée sur les principes tels que le moindre privilège et la gestion des identités privilégiées, assurant une base solide en matière de sécurité des identités.

**☁️ Azure Key Vault & Intégration avec les solutions sur site :** Comprenez les mécanismes et la tarification d'Azure Key Vault et apprenez à intégrer les solutions Azure avec les infrastructures sur site.

**☁️ Couverture complète des services Azure :** Le cours s'étend à des explorations détaillées de la base de données Azure SQL, Azure Synapse Analytics, Azure Data Factory, Azure Databricks, Azure Data Lake, Cosmos DB et Azure Cache pour Redis. Chacune de ces sections est conçue pour vous fournir une compréhension approfondie des services de données d'Azure.

**☁️ Modules spécialisés Azure :** Apprenez-en davantage sur les services spécialisés d'Azure tels que les machines virtuelles Azure, Azure App Service, Azure Container Instances, Azure Functions et Azure Front Door, ainsi que sur Azure CDN et Azure Service Bus. Ces modules offrent une plongée en profondeur dans des services Azure spécifiques, améliorant vos compétences architecturales.

**☁️ Solutions de migration et de connectivité :** Équipez-vous de connaissances sur le cadre d'adoption du cloud Azure, Azure Migrate et les solutions de connectivité telles que les VNets et les sous-réseaux.

**☁️ Sécurité réseau et équilibrage de charge :** Comprenez comment optimiser la sécurité du réseau grâce au pare-feu Azure et aux groupes de sécurité réseau, et maîtrisez les solutions d'équilibrage de charge et de routage.

Que vous cherchiez à passer la certification AZ 305 ou à améliorer votre expertise Azure, ce cours est votre guide complet pour réussir.
