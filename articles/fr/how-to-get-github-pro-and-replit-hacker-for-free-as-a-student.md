Si vous êtes au lycée ou à l'université et que vous êtes intéressé par la programmation, vous devriez envisager de vous inscrire à GitHub Education. 

Cela vous permet d'utiliser GitHub Pro, ainsi que de nombreux autres avantages. 

Dans ce guide rapide, je vais vous montrer les étapes pour vous inscrire à GitHub Education et je vais discuter de ses avantages. Ensuite, je vous montrerai comment vous pouvez obtenir un accès gratuit à Replit Hacker pendant un an. 

## Comment obtenir un compte GitHub Pro gratuit en tant qu'étudiant

Si vous n'avez pas déjà de compte [GitHub](https://github.com/signup), vous devrez en créer un. Connectez-vous avec votre adresse e-mail personnelle (afin de pouvoir utiliser votre compte même après que votre adresse e-mail scolaire soit désactivée lorsque vous obtiendrez votre diplôme).

Ensuite, allez dans les paramètres et cliquez sur Emails. C'est ici que vous ajouterez votre e-mail scolaire à votre compte GitHub.

![image1](https://user-images.githubusercontent.com/97064249/215526808-7e2aeb8a-c32c-4f33-9ef7-ad0f6b88d287.png)
_Ajoutez votre e-mail scolaire dans votre compte GitHub_

Ajoutez votre adresse e-mail scolaire et vérifiez-la. Une fois vérifiée, allez sur [https://education.github.com/](https://education.github.com/). Cela vous permettra d'être éligible pour GitHub Pro. Connectez-vous avec votre compte GitHub.

Ensuite, allez dans Benefits. Faites défiler vers le bas et cliquez sur **Get student benefits** (assurez-vous simplement d'être [éligible](https://education.github.com/discount_requests/pack_application#student-info) d'abord).

![image2](https://user-images.githubusercontent.com/97064249/215526941-b1a1e2fc-bcd9-4edd-bf8e-61a4e1cd82ac.png)
_Avantages pour les étudiants_

Sélectionnez **Student** comme votre statut académique. Faites défiler vers le bas et utilisez votre adresse e-mail scolaire. Après cela, vous devez sélectionner le nom de votre école. En utilisant quelques phrases, expliquez pourquoi vous utilisez GitHub.

Une fois que vous avez terminé ces étapes, appuyez sur continuer.

Vous devrez vérifier que vous êtes un étudiant dans votre école. Pour ce faire, vous pouvez télécharger une capture d'écran d'un e-mail de votre conseil scolaire ou d'un autre document scolaire officiel. La vérification prendra quelques jours, vous devrez donc attendre jusqu'à ce moment-là.

Une fois cela terminé, vous devriez voir votre tableau de bord ressembler à ceci :

![image3](https://user-images.githubusercontent.com/97064249/215528452-d7f4de46-77ac-4df2-8352-f06329fdeb5f.png)

Vous aurez maintenant accès à de nombreux avantages pour les étudiants, y compris un crédit de 200 $ pour Digital Ocean, un domaine gratuit de Namecheap et un abonnement gratuit à l'IDE JetBrains, que vous pouvez renouveler chaque année. 

Ensuite dans ce tutoriel, vous apprendrez comment obtenir le plan Replit Hacker gratuitement (ce qui n'est pas mentionné comme une offre d'avantages pour les étudiants).

## Comment obtenir un plan Replit Hacker gratuit

Une fois que vous avez GitHub Pro, vous aurez également accès à de nombreux autres avantages. L'un des avantages les plus notables est un plan Replit Hacker gratuit pendant un an.

Si vous ne savez pas ce qu'est Replit, c'est un IDE basé sur le web qui permet aux utilisateurs de créer des projets incroyables (appelés Repls) qui sont stockés dans le cloud. 

J'ai trouvé Replit incroyablement utile pour coder en Rust, car il rend l'installation de packages et la configuration de l'environnement de développement vraiment simples et faciles. 

Vous pouvez également importer des dépôts GitHub directement dans Replit, afin de pouvoir travailler sur votre code depuis n'importe quel appareil, tant que vous avez une connexion Internet.

Replit est gratuit pour s'inscrire, mais avoir le **plan Hacker** vous donne certains avantages qui sont extrêmement utiles pour le développement. 

Un avantage notable est d'avoir des Repls privés illimités. D'autres avantages sont d'avoir 10 fois plus de stockage de compte et le double de stockage de Repl. Ce plan commence à 7,00 $ par mois, mais heureusement pour les étudiants, il existe un moyen d'obtenir le plan gratuitement ([Source](https://replit.com/site/github-students)).

Si vous n'avez pas de compte Replit, créez-en un avec votre adresse e-mail personnelle (afin de pouvoir utiliser votre compte même après que votre adresse e-mail scolaire soit désactivée lorsque vous obtiendrez votre diplôme).

Ensuite, allez dans Account dans le menu d'accueil.

![image4](https://user-images.githubusercontent.com/97064249/215528245-87cf3cae-db1f-4382-b75a-599c269cc95e.png)

Faites défiler vers le bas jusqu'à Connected services et connectez votre compte GitHub.

![image5](https://user-images.githubusercontent.com/97064249/215528283-8895bb3e-fb70-4d7c-87b3-3d3852e8c6c7.png)

Vous devriez maintenant avoir GitHub Pro (jusqu'à l'obtention de votre diplôme) et Replit Hacker (pendant un an).

Profitez-en !
