Beaucoup de cours de machine learning s'appuient sur des frameworks qui abstraient le fonctionnement interne de ce qui se passe. Si vous voulez devenir compétent, il est utile de comprendre comment les choses fonctionnent sous le capot.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser le machine learning et les réseaux de neurones sans aucun framework.

Le Dr. Radu Mariescu-Istodor a créé ce cours. Il est titulaire d'un doctorat en informatique et crée des tutoriels logiciels engageants et créatifs.

Dans un monde où le machine learning devient de plus en plus prévalent, comprendre les concepts et algorithmes sous-jacents n'a jamais été aussi crucial. Le cours ne repose pas sur des bibliothèques préexistantes. Au lieu de cela, il vous guide à travers la construction d'un système de machine learning à partir de zéro.

Le cours est centré autour d'un projet captivant : créer une application web qui apprend à reconnaître des dessins. Il s'agit de la Phase 2 du cours, où l'accent est mis sur l'amélioration de la précision des méthodes développées dans la Phase 1 (mais vous pouvez toujours suivre même si vous n'avez pas regardé la Phase 1).

À travers une série de sections complètes, vous explorerez des fonctionnalités avancées, des méthodes de classification, des techniques de nettoyage de données et une large gamme de concepts fondamentaux essentiels au machine learning.

Le cours est structuré pour construire progressivement votre compréhension à partir des bases. Voici les sections thématiques de ce cours :

* Introduction
* Révision du Code de la Phase 1
* Nettoyage des Données
* Matrice de Confusion
* Marqueur de Distance Euclidienne
* Mesure de l'Élongation
* Mesure de la Rondeur
* Vecteur vs Raster (Pixels)
* Réseaux de Neurones
* Optimisation des Réseaux de Neurones
* Réseaux de Neurones Profonds

En révélant le fonctionnement interne des systèmes de machine learning, vous développerez non seulement une compréhension profonde du sujet, mais vous affinerez également vos compétences en développement logiciel.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/3wwiOSxDAmg) (4 heures de visionnage).
