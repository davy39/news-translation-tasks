Par Gitter

Go, également connu sous le nom de [Golang](https://golang.org/), est un langage de programmation open-source développé chez Google en 2007 par Robert Griesemer, Rob Pike et Ken Thompson.

À l'origine destiné à être un descendant de C++ pour le 21e siècle, Golang se caractérise par une compilation rapide, une syntaxe plus simple et une gestion efficace de la mémoire. C'est un langage propre, rapide et typé, particulièrement adapté au développement web.

Voici une liste de communautés liées à Go que nous hébergeons sur [Gitter](http://gitter.im). N'hésitez pas à rejoindre la conversation.

* [**docker/docker**](https://gitter.im/docker/docker?utm_source=blog&utm_medium=content&utm_campaign=go) — Docker est un projet open source pour empaqueter, transporter et exécuter n'importe quelle application en tant que conteneur léger. Les conteneurs Docker sont à la fois _agnostiques matériels_ et _agnostiques plateformes_. Docker a commencé comme une implémentation open-source du moteur de déploiement qui alimentait [dotCloud](http://web.archive.org/web/20130530031104/https://www.dotcloud.com/), une plateforme populaire en tant que service. Il bénéficie directement de l'expérience accumulée sur plusieurs années d'exploitation à grande échelle et de support de centaines de milliers d'applications et de bases de données.
* [**drone/drone**](https://gitter.im/drone/drone?utm_source=blog&utm_medium=content&utm_campaign=go) — Drone est une plateforme de livraison continue basée sur Docker, écrite en Go. Chaque build est exécuté à l'intérieur d'un conteneur Docker éphémère, donnant aux développeurs un contrôle complet sur leur environnement de build avec une isolation garantie.
* [**minio/minio**](https://gitter.im/minio/minio) — [Minio](https://minio.io) est un serveur de stockage d'objets compatible avec Amazon S3 et sous licence Apache 2.0.
* [**gogits/gogs**](https://gitter.im/gogits/gogs?utm_source=blog&utm_medium=content&utm_campaign=go) — Le but de ce projet est de rendre la mise en place d'un service Git auto-hébergé la plus facile, rapide et sans douleur possible.
* [**google/gxui**](https://gitter.im/google/gxui?utm_source=blog&utm_medium=content&utm_campaign=go) **—** Une bibliothèque expérimentale d'interface utilisateur multiplateforme en Go.
* [**spf13/hugo**](https://gitter.im/spf13/hugo?utm_source=blog&utm_medium=content&utm_campaign=go) — Un générateur de site statique rapide et flexible construit avec amour en GoLang.
* [**Go-lang-idea-plugin**](https://gitter.im/go-lang-plugin-org/go-lang-idea-plugin?utm_source=blog&utm_medium=content&utm_campaign=go) — IDE pour le langage Google Go construit en utilisant la plateforme IntelliJ [https://plugins.jetbrains.com/plugin/5047](https://plugins.jetbrains.com/plugin/5047).
* [**gin-gonic/gin**](https://gitter.im/gin-gonic/gin?utm_source=blog&utm_medium=content&utm_campaign=go) — Gin est un framework web HTTP écrit en Go (Golang). Il propose une API similaire à Martini avec des performances bien meilleures — jusqu'à 40 fois plus rapide.
* [**labstack/echo**](https://gitter.im/labstack/echo?utm_source=blog&utm_medium=content&utm_campaign=go) — Framework serveur HTTP rapide et sans fioritures pour Go (Golang). Jusqu'à 10 fois plus rapide que les autres. [https://echo.labstack.com](https://echo.labstack.com/)
* [**influxdata/influxdb**](https://gitter.im/influxdata/influxdb?utm_source=blog&utm_medium=content&utm_campaign=go) — Base de données open source de séries temporelles sans dépendances externes. Elle est utile pour enregistrer des métriques, des événements et effectuer des analyses.
* [**aws/aws-sdk-go**](https://gitter.im/aws/aws-sdk-go?utm_source=blog&utm_medium=content&utm_campaign=go) **—** aws-sdk-go est le SDK AWS officiel pour le langage de programmation Go.
* [**FreeCodeCamp/go**](https://gitter.im/FreeCodeCamp/go?utm_source=blog&utm_medium=content&utm_campaign=go) — Si vous êtes débutant, c'est le meilleur endroit pour discuter de Go et obtenir de l'aide.
* [**derekparker/delve**](https://gitter.im/derekparker/delve) **—** Delve est un débogueur pour le langage de programmation Go.
* [**CodisLabs/codis**](https://gitter.im/CodisLabs/codis) **—** Codis est une solution de cluster Redis haute performance basée sur un proxy, écrite en Go. Elle est prête pour la production et largement utilisée dans de nombreuses entreprises.
* [**blevesearch/bleve**](https://gitter.im/blevesearch/bleve) **—** Indexation de texte moderne en Go.
* [**go-mangos/mangos**](https://gitter.im/go-mangos/mangos) **—** Le package mangos est une implémentation en Go pur des protocoles SP ("Scalable Protocols").
* [**linkeddata/gold**](https://gitter.im/linkeddata/gold) **—** gold est un serveur de référence pour la plateforme Linked Data Platform pour la [**plateforme Solid**](https://github.com/solid/solid-spec). Écrit en Go, basé sur [le travail initial réalisé par William Waites](https://bitbucket.org/ww/gold).
