Embarquez pour un voyage passionnant afin de maîtriser le développement frontend avec ce cours créé par Jess, l'experte derrière la chaîne YouTube acclamée Coder Coder.

Ce cours complet sur la chaîne YouTube de freeCodeCamp.org est conçu pour vous donner les connaissances et les compétences nécessaires pour exceller en JavaScript, HTML et CSS, tout en construisant un tableau de bord de médias sociaux attrayant équipé d'un basculeur de thème sombre/clair.

Conçu autour du défi Frontend Mentor orienté débutants, ce cours vous guide à travers le processus de développement d'une page de destination qui reflète fidèlement le design fourni. N'hésitez pas à utiliser les outils avec lesquels vous êtes à l'aise ou que vous souhaitez pratiquer, car l'objectif principal est de mettre en œuvre un basculeur de thème de couleur fluide via JavaScript.

En suivant ce cours, les utilisateurs seront capables de :

* Utiliser HTML, CSS et JavaScript pour construire un site simple
* Optimiser les mises en page du site pour différentes tailles d'écran
* Incorporer des états de survol pour tous les éléments interactifs
* Basculer les thèmes de couleur en fonction de leurs préférences

Le cours est divisé en cinq parties :

**Partie 1 : Poser les bases**

* Introduction
* Comprendre les exigences de conception fonctionnelle
* Créer des contrôles de formulaire accessibles
* Mettre à jour les propriétés personnalisées CSS avec JS
* Implémenter le texte réservé aux lecteurs d'écran

**Partie 2 : Préparer l'environnement de développement**

* Créer un dépôt GitHub
* Configurer les fichiers SCSS et JS
* Établir un flux de travail Gulp

**Partie 3 : Plonger dans la conception**

* Analyser la conception
* Construire la barre supérieure
* Utiliser un balisage accessible
* Styliser la barre supérieure
* Styliser le basculeur de thème et améliorer l'accessibilité

**Partie 4 : Construire le tableau de bord des médias sociaux**

* Introduction à BEM et planification des noms de classe
* Ajouter le balisage et les sélecteurs SCSS pour les cartes
* Appliquer des styles aux cartes
* Créer des mises en page de grille de cartes avec flexbox et CSS grid
* Styliser les barres supérieures des cartes
* Implémenter le balisage et les styles de la carte inférieure

**Partie 5 : Améliorer le basculeur de thème**

* Modifier le basculeur en fonction des commentaires des utilisateurs
* Charger des thèmes clairs ou sombres en fonction des préférences système
* Développer la logique de basculement pour passer manuellement entre les thèmes clair et sombre
* Enregistrer les paramètres de basculement dans le stockage local

Rejoignez Jess de Coder Coder dans ce voyage passionnant et améliorez vos compétences en développement frontend. Que vous soyez novice ou que vous ayez déjà une certaine expérience, ce cours est l'occasion parfaite d'apprendre et de grandir en tant que développeur web.
