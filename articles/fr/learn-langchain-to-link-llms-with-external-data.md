LangChain est un framework axé sur l'IA conçu pour permettre aux développeurs de créer des applications de raisonnement contextuel en liant des modèles de langage puissants avec des sources de données externes.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra tout sur LangChain. Ce cours vous fournira les compétences de pointe nécessaires pour construire un chatbot hautement informé en utilisant le LangChain Expression Language.

Tom Chant est un instructeur populaire chez Scrimba. Dans ce cours, Tom vous emmènera dans un voyage des bases de LangChain.js aux concepts avancés. Vous explorerez une variété de sujets, y compris les embeddings, les diagrammes de flux d'application, le stockage vectoriel Supabase, la division de texte, et bien plus encore. Le cours est structuré pour rendre l'apprentissage de LangChain.js accessible et agréable, avec un accent sur les applications pratiques.

Le cours inclut même une introduction à LangChain par Jacob Lee, le mainteneur principal de LangChain.js.

Dans ce cours, vous apprendrez :

* La division avec un outil LangChain textSplitter
* La vectorisation des morceaux de texte
* L'utilisation des modèles d'embeddings
* Le stockage vectoriel Supabase
* Les templates avec input_variables
* Les prompts à partir de templates
* Le LangChain Expression Language
* Les chaînes de base avec la méthode .Pipe()
* La récupération à partir d'un stockage vectoriel
* Les chaînes complexes avec RunnableSequence()
* La classe StringOutputParser()
* Le dépannage des problèmes de performance

Dans ce cours, vous apprendrez à utiliser LangChain.js pour construire un chatbot capable de répondre à des questions sur un texte spécifique que vous lui fournissez.

Dans la première partie du projet, vous apprendrez à utiliser LangChain pour diviser le texte en morceaux, convertir les morceaux en vecteurs en utilisant un modèle d'embeddings OpenAI, et les stocker ensemble dans un stockage vectoriel Supabase.

Ensuite, vous apprendrez les chaînes, qui sont les éléments de base de LangChain. Et nous faisons cela en utilisant le LangChain Expression Language. Cela rend le processus de codage dans LangChain beaucoup plus fluide et plus facile à comprendre.

Enfin, vous apprendrez la récupération : utiliser la correspondance vectorielle pour sélectionner les morceaux de texte de notre stockage vectoriel qui sont les plus susceptibles de contenir la réponse à une requête utilisateur. Cela permet au chatbot de répondre à des questions spécifiques à vos données - une compétence critique lors de la travail avec l'IA et l'un des cas d'utilisation les plus courants pour l'IA dans le développement web.
