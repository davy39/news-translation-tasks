Vous vous demandez quelles sont les étapes pour mettre votre site web en ligne ? Peut-être que vous n'avez même pas encore de site web, mais vous avez une idée de site web ?

Nous avons publié un cours accéléré sur la chaîne YouTube freeCodeCamp.org qui vous mènera d'une idée de site web à la mise en ligne du site avec une URL personnalisée. Si vous vous êtes déjà demandé comment un domaine comme `www.nom-de-votre-site.tech` pointe vers votre site personnel, ce cours est fait pour vous.

Tout d'abord, vous apprendrez à créer un site web simple en utilisant HTML, CSS et JavaScript, en commençant par un modèle de site web. (Si vous avez déjà un site web que vous souhaitez mettre en ligne—super ! Passez simplement cette section et allez à la suivante.)

Ensuite, vous apprendrez à obtenir une URL personnalisée. C'est le nom que les gens tapent dans leur navigateur web pour accéder à votre site.

Enfin, vous apprendrez à héberger vos fichiers de site web et à faire pointer votre URL personnalisée vers vos fichiers de site web. Vous apprendrez deux méthodes différentes pour héberger votre site web, y compris une méthode pour héberger votre site web gratuitement.

En cours de route, vous apprendrez également à configurer vos paramètres DNS, à créer une adresse email personnalisée et à installer WordPress.

Ce cours a été développé par moi, Beau Carnes. J'ai créé de nombreux cours de logiciels au fil des ans sur YouTube et d'autres plateformes, et j'ai conçu des sections entières du [programme principal de freeCodeCamp](https://www.freecodecamp.org/learn).

Voici une liste des compétences que je couvre dans ce cours accéléré :

* Comment créer un site web statique simple
* Comment obtenir une URL personnalisée
* Comment obtenir un hébergement web
* Ou comment héberger votre site web gratuitement avec GitHub Pages si vous voulez économiser de l'argent
* Comment créer une adresse email personnalisée à votre nouveau domaine
* Comment mettre à jour vos Nameservers
* Et comment mettre à jour vos paramètres DNS

Et oui, je couvre tout cela en une heure. Attachez votre ceinture et préparez-vous pour un sérieux apprentissage.

Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/NQP89ish9t8) (1 heure de visionnage).
