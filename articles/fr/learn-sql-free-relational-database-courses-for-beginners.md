SQL (Structured Query Language) est un langage de programmation populaire utilisé pour les bases de données relationnelles. Et il existe de nombreuses bonnes ressources gratuites que vous pouvez utiliser pour commencer à l'apprendre.

Dans cet article, je vais lister 8 ressources en ligne gratuites et adaptées aux débutants pour apprendre SQL, PostgreSQL et MySQL.

## Programme de Bases de Données Relationnelles freeCodeCamp

En septembre 2021, freeCodeCamp a publié la [version bêta](https://www.freecodecamp.org/news/how-to-run-freecodecamps-relational-databases-curriculum-using-docker-vscode-and-coderoad/) de leur programme de bases de données relationnelles. Cette version bêta s'exécute entièrement dans Visual Studio Code en utilisant Docker et une extension CodeRoad.

Dans ce cours, vous apprendrez Git, GitHub, Bash, Nano, SQL, PostgreSQL et plus encore. Il s'agit d'un programme d'apprentissage entièrement basé sur des projets.

Parmi les projets que vous construirez dans le cours, on trouve une base de données Mario, une base de données Monde, une base de données Corps Célestes et un magasin de location de vélos.

Pour en savoir plus, vous pouvez lire [cet article](https://www.freecodecamp.org/news/how-to-run-freecodecamps-relational-databases-curriculum-using-docker-vscode-and-coderoad/) de Tom Mondloch qui a créé le cours.

## Tutoriel SQL - Cours Complet de Bases de Données pour Débutants

Dans [ce cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=HXV3zeQKqGY), Mike Dane vous enseignera les bases de SQL. Vous apprendrez à créer des tables, insérer des données, utiliser des caractères génériques, des unions, des jointures et plus encore.

Dans ce cours, vous apprendrez à installer MySQL sur votre ordinateur Mac ou Windows. Vous pratiquerez également vos compétences SQL en construisant une base de données d'entreprise et un diagramme ER.

Si vous aimez ce cours de Mike Dane, alors n'hésitez pas à consulter leur [chaîne YouTube](https://www.youtube.com/c/GiraffeAcademy/videos) pour plus de cours gratuits.

## Apprendre SQL en 60 Minutes

Dans [ce cours YouTube de Web Dev Simplified](https://www.youtube.com/watch?v=p3qvj9hO_Bo), Kyle vous enseignera les bases pour commencer avec SQL. Vous apprendrez à créer des tables, insérer des données dans des tables, les clés primaires, les clés étrangères, les fonctions d'agrégation et plus encore.

Il s'agit d'un excellent cours à suivre pour une introduction rapide aux bases. À partir de là, vous pouvez passer à la construction de vos premiers projets utilisant SQL.

## Tutoriel SQL pour Débutants

Dans [ce cours YouTube d'Amigoscode](https://www.youtube.com/watch?v=5hzZtqCNQKk), Nelson vous enseignera comment vous installer en utilisant PostgreSQL sur Mac et Windows. Vous apprendrez ensuite les bases de SQL, y compris la création de tables, les clés primaires, les clés étrangères, les fonctions d'agrégation et plus encore.

Si vous appréciez ce cours YouTube gratuit de Nelson, vous pouvez consulter ses autres cours sur sa [chaîne](https://www.youtube.com/c/amigoscode/videos) ou jeter un œil à ses cours payants sur son [site web](https://amigoscode.com/).

## Base de Données MySQL - Cours Complet

Dans ce [cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=ER8oKX5myE0), Bharath vous enseignera comment installer MySQL sur votre machine locale. Vous apprendrez ensuite à créer des tables, insérer des données dans des tables, faire de la modélisation de données, utiliser des verrous, utiliser SQL Explain et plus encore.

Bharath fournit également des ressources utiles pour accompagner le cours, y compris un [guide de démarrage](https://docs.google.com/document/d/1tDPgLoE1SakXJlwBF283nG12iUaBVy-OngmFaC4KHbI/edit) et un [dépôt GitHub](https://github.com/jmbharathram/executeoncommand/tree/master/mysql) des scripts MySQL.

Si vous avez aimé ce cours, vous pouvez visiter d'autres cours sur leur [chaîne YouTube](https://www.youtube.com/channel/UCwfA8RML42R-a65NRA4MQ2A).

## Tutoriel MySQL pour Débutants - Cours Complet

Dans ce [cours YouTube de Programming with Mosh](https://www.youtube.com/watch?v=7S_tz1z_5bA), Mosh vous enseignera comment installer MySQL sur Mac ou Windows. Vous apprendrez ensuite à créer des tables, insérer des données dans des bases de données, faire des jointures, utiliser des opérateurs, des clauses SQL, des opérateurs SQL et plus encore.

Si vous appréciez ce cours de Mosh, vous pouvez visiter leur [chaîne YouTube](https://www.youtube.com/channel/UCWv7vMbMWH4-V0ZXdmDpPBA) ou son [site web](https://codewithmosh.com/) pour des cours payants.

## Cours Accéléré MySQL | Apprendre SQL

Dans ce [cours YouTube de Traversy Media](https://www.youtube.com/watch?v=9ylj9NR0Lcg), Brad vous enseignera les bases pour commencer avec MySQL. Vous apprendrez à créer des tables, insérer des données dans des tables, modifier des tables, les clés primaires, les clés étrangères, les fonctions d'agrégation et plus encore.

Il s'agit d'un excellent cours à suivre pour une introduction rapide aux bases. À partir de là, vous pouvez passer à la construction de vos premiers projets utilisant MySQL.

Brad inclut également une [feuille de triche MySQL](https://gist.github.com/bradtraversy/c831baaad44343cc945e76c2e30927b3) avec ce cours.

## SQL pour la Science des Données

Dans ce [cours de l'UC Davis](https://www.coursera.org/learn/sql-for-data-science?specialization=learn-sql-basics-data-science), vous apprendrez les bases de l'utilisation de SQL dans le contexte de la Science des Données. Vous apprendrez à créer des tables, insérer des données dans des tables, les clés primaires, les clés étrangères, les fonctions d'agrégation, les jointures, les sous-requêtes et plus encore.

Ce cours est gratuit à auditer sur Coursera. Il fait également partie de la [Spécialisation Apprendre les Bases de SQL pour la Science des Données](https://www.coursera.org/specializations/learn-sql-basics-data-science).

Si vous êtes intéressé à continuer avec la spécialisation, je vous suggère de regarder les trois autres cours proposés après ce cours d'introduction.
