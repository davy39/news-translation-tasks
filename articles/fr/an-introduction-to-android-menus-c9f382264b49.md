#### Tout ce que vous voulez savoir sur les menus Android

Il existe trois types de menus dans Android : Popup, Contextuel et Options. Chacun a un cas d'utilisation spécifique et du code qui l'accompagne. Pour apprendre à les utiliser, continuez votre lecture.
