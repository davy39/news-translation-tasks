J'entends certains de mes collègues parler d'atteindre l'inbox zéro. J'ai donc commencé à réfléchir à un moyen de vider mes plus de 4000 messages non lus. Et après des jours de recherche, j'ai trouvé une solution.

J'ai même supprimé 20 000 messages de la corbeille – et j'ai économisé un peu plus d'1 gigaoctet d'espace disque dans le processus.
![inbox0](https://www.freecodecamp.org/news/content/images/2022/07/inbox0.png)

Aujourd'hui, je veux vous montrer comment supprimer des emails en masse dans l'application Gmail – peu importe les milliers de messages que vous avez.

## Comment supprimer tous les emails non lus dans Gmail

**Étape 1** : Connectez-vous à votre compte Gmail 

**Étape 2** : Dans la barre de recherche, tapez `in:unread` et appuyez sur `ENTRÉE`. Cela vous montrera 50 messages non lus.
![ss4-2](https://www.freecodecamp.org/news/content/images/2022/07/ss4-2.png)

**Étape 3** : Sélectionnez la case à cocher en haut à droite pour sélectionner 50 des emails non lus.
![ss5-2](https://www.freecodecamp.org/news/content/images/2022/07/ss5-2.png)

**Étape 4** : Cliquez sur le message qui dit « sélectionner toutes les conversations qui correspondent à cette recherche ». Cela sélectionnera tous vos messages non lus.
![ss6-2](https://www.freecodecamp.org/news/content/images/2022/07/ss6-2.png)

**Étape 5** : Cliquez sur l'icône de suppression en haut.
![ss7-3](https://www.freecodecamp.org/news/content/images/2022/07/ss7-3.png)

**Étape 6** : Une fenêtre contextuelle apparaîtra vous demandant si vous souhaitez supprimer les messages en masse. Cliquez sur « OK ».
![ss8-2](https://www.freecodecamp.org/news/content/images/2022/07/ss8-2.png) 

C'est ainsi que vous pouvez supprimer des messages en masse dans Gmail.
![ss9-2](https://www.freecodecamp.org/news/content/images/2022/07/ss9-2.png) 


## Comment vider la corbeille Gmail

**Étape 1** : Pour vider les messages de votre corbeille, cliquez sur « Plus » à gauche et sélectionnez Corbeille.
![ss10-2](https://www.freecodecamp.org/news/content/images/2022/07/ss10-2.png)

**Étape 2** : Cliquez sur la case à cocher en haut à droite pour sélectionner les messages dans la corbeille.
![ss11-2](https://www.freecodecamp.org/news/content/images/2022/07/ss11-2.png)

**Étape 3** : Sélectionnez tous les messages dans la corbeille et cliquez sur « Supprimer définitivement ».
![ss12-2](https://www.freecodecamp.org/news/content/images/2022/07/ss12-2.png)

**Étape 4** : Cliquez sur OK pour confirmer que vous souhaitez supprimer tous les messages. 
![ss13-1](https://www.freecodecamp.org/news/content/images/2022/07/ss13-1.png)

Vous devriez recevoir un message indiquant que x nombre d'emails ont été supprimés définitivement. Si vous ne recevez pas le message même si tout semble clair, actualisez la page.
![ss14-1](https://www.freecodecamp.org/news/content/images/2022/07/ss14-1.png)

Vous pouvez également supprimer des messages dans l'onglet promotion ou social.


## Comment supprimer les emails de la liste des promotions ?

Pour supprimer des messages dans l'onglet promotion, suivez les étapes ci-dessous.

**Étape 1** : Cliquez sur Plus à droite et sélectionnez Catégories.
![ss15](https://www.freecodecamp.org/news/content/images/2022/07/ss15.png)

**Étape 2** : Cliquez sur Promotions.
![ss16](https://www.freecodecamp.org/news/content/images/2022/07/ss16.png)
 
**Étape 3** : Cliquez sur la case à cocher en haut à droite pour sélectionner les 50 messages dans l'onglet promotion.
![ss17](https://www.freecodecamp.org/news/content/images/2022/07/ss17.png) 

**Étape 4** : Cliquez sur sélectionner toutes les conversations dans l'onglet promotion.
![ss18](https://www.freecodecamp.org/news/content/images/2022/07/ss18.png) 

**Étape 5** : Cliquez sur l'icône de suppression en haut.
![ss19](https://www.freecodecamp.org/news/content/images/2022/07/ss19.png) 

**Étape 6** : Confirmez que vous souhaitez supprimer tous les messages dans l'onglet promotion.	
![ss20](https://www.freecodecamp.org/news/content/images/2022/07/ss20.png) 

Vous devriez recevoir un message indiquant que les conversations ont été déplacées vers la corbeille.
![ss21](https://www.freecodecamp.org/news/content/images/2022/07/ss21.png) 

Si vous souhaitez supprimer des messages dans d'autres onglets tels que social ou forum, répétez le processus que vous avez suivi pour supprimer tous les messages dans l'onglet promotion.
