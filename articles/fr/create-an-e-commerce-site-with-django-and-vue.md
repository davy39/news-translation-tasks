On estime qu'il existe 24 millions de sites eCommerce sur Internet. Voulez-vous apprendre à en créer un de plus ?

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un site e-commerce en utilisant Django et Vue.

Ce cours a été développé par CodeWithStein. Il a créé de nombreux tutoriels sur sa chaîne YouTube et nous voulions le faire connaître à un public plus large.

Pour suivre ce cours, il serait utile d'avoir au moins quelques connaissances en Python et en JavaScript, car Django utilise Python et Vue utilise JavaScript.

Dans ce cours, vous utiliserez le Django Rest Framework pour construire l'API et le backend, et Vue pour créer un frontend séparé. Vous implémenterez les paiements avec Stripe.

Voici les sections abordées dans ce cours :

* Installation et configuration (Django)
* Installation et configuration (Vue)
* Inclure Font Awesome
* Comment configurer le modèle de base
* Comment créer l'application Django et les modèles pour les produits
* Comment créer le sérialiseur et les vues pour les produits
* Comment créer une page d'accueil simple (derniers produits)
* Comment afficher un produit
* Comment configurer Vuex / État
* Comment rendre possible l'ajout au panier
* Comment implémenter une barre de chargement
* Comment définir le titre du document (Toutes les pages)
* Comment rendre possible l'affichage d'une catégorie
* Comment ajouter une fonctionnalité de recherche
* Comment afficher le panier
* Comment modifier le contenu du panier (Incrémenter/Décrémenter le contenu, Retirer du panier)
* Comment activer l'inscription
* Comment activer la connexion
* Comment créer une page simple "Mon compte"
* Comment procéder au paiement (Authentification)
* Comment créer une page de succès simple
* Comment afficher mes commandes sur la page "Mon compte"
* Comment déployer Django
* Comment déployer Vue (Générer des fichiers localement, Envoyer des fichiers sur le serveur, Configurer l'hôte virtuel NGINX)
