Django REST Framework est un outil puissant et flexible pour construire des APIs Web.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Python et le Django REST Framework (DRF) pour construire une API web.

Bobby Stearman a développé ce cours. Bobby a créé de nombreux cours populaires sur le développement logiciel, à la fois sur sa propre chaîne et sur la chaîne freeCodeCamp.org.

Le Django REST Framework offre une suite complète de fonctionnalités pour développer des APIs RESTful, incluant la sérialisation, l'authentification, l'autorisation et la limitation de débit. Il facilite également la gestion des données stockées dans la base de données, permettant aux développeurs d'apporter rapidement des modifications au modèle de données et de maintenir une base de code propre et cohérente.

Voici ce que vous ferez dans ce cours :

* Cloner le code et configurer notre système
* Ajouter des paramètres et des configurations d'URL
* Créer le premier point de terminaison de l'API
* Écrire des tests unitaires
* Créer des points de terminaison avec authentification
* Créer des modèles et des sérialiseurs pour les articles et les commandes
* Coder des ensembles de vues et des routeurs
* Écrire les tests unitaires finaux

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/tujhGdn1EMI) (2 heures de visionnage).
