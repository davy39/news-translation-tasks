Par Nick Karnik

Voici les extensions GitHub que j'utilise. Elles vous permettront d'améliorer votre productivité sur GitHub. Merci de partager vos extensions préférées dans les commentaires.

[**Octotree**](https://github.com/buunguyen/octotree) **permet une exploration facile du code source directement dans le navigateur grâce à un panneau sur la gauche :**

![Image](https://cdn-media-1.freecodecamp.org/images/K91lj5pBvV8HsN0TX37LruGJCDJG1blhqOht)

[**github-dashboard**](https://github.com/muan/github-dashboard) **permet de filtrer les événements sur le tableau d'activité de Github.com**

![Image](https://cdn-media-1.freecodecamp.org/images/iN1AAIkMUQTO2tm3ks1g8lYaaAa9M4fXpcuX)

[**Sourcegraph**](https://about.sourcegraph.com/) **permet d'accéder à la définition et de trouver les références dans le code**

![Image](https://cdn-media-1.freecodecamp.org/images/34Q7gJSeHq3DelG3YEvimvn7d5ipOdqBVJjj)

[**Octolinker**](https://github.com/OctoLinker/OctoLinker) **permet de naviguer plus efficacement dans le code en transformant les imports de modules en liens cliquables pour voir facilement leur source**

![Image](https://cdn-media-1.freecodecamp.org/images/bod3D8IorChh9HIRZwTdyjELHSXqARbIEkUm)

[**github-repo-size**](https://github.com/harshjv/github-repo-size) **affiche la taille du dépôt et des fichiers individuels sur GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/Qsrd1b3RRBr9mSvSJiITQSrSfJuflTS0yKwX)

[**refined-github**](https://github.com/sindresorhus/refined-github) **simplifie l'interface de GitHub et ajoute plusieurs fonctionnalités utiles**

**_Plus de sauts causés par les branches récemment poussées (elles sont déplacées sur le côté)_**

![Image](https://cdn-media-1.freecodecamp.org/images/rEZM8tEWaLOAcumeAyigcRIRROL2lktOa6wn)

**_Avatars de réactions montrant qui a réagi à un commentaire_**

![Image](https://cdn-media-1.freecodecamp.org/images/ofubT38pFl9a1t1qYwaO-e3sSSVgiZk4A4RX)

**_L'option d'attendre les vérifications lors de la fusion d'une PR_**

![Image](https://cdn-media-1.freecodecamp.org/images/-JrEtND7QVMoeFZhN0bkl2CfDqjN9hwP48Pc)

**_Références cliquables vers les issues et les pull requests_**

![Image](https://cdn-media-1.freecodecamp.org/images/Nfyrn7vHoEy0JD83y-77SEUUtd4dvp3izwaD)

**_Liens vers le commit ou la pull request de clôture d'une issue_**

![Image](https://cdn-media-1.freecodecamp.org/images/BnipoAfM0PazABnMq3vvP1rJJplOyLi5Y23Q)

[**ZenHub**](https://www.zenhub.com/) **est une meilleure façon de gérer vos issues GitHub. Il supporte les tableaux multi-dépôts, les épopées et les rapports — le tout directement dans GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/vC0BibG0DNmiPTvvg1jRYtyTNmslgpLNhiJg)

[**github-npm-stats**](https://github.com/katranci/github-npm-stats) **affiche les statistiques de téléchargement npm sur un dépôt GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/XA3PgTHXVBEgwgGypM2LiKWIqZytzToWFATh)

[**npmhub**](https://github.com/npmhub/npmhub) **affiche les dépendances npm sur les dépôts GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/gxzbn81yIRTpC2YYC280lb50K6vqnwiJqZud)

[**isometric-contributions**](https://github.com/jasonlong/isometric-contributions) **affiche une version en pixel art isométrique des graphiques de contributions GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/kY4gzq0z4qeboA9gRBy2NfA6dzsPL-VPGZSu)
