MongoDB est la base de données NoSQL / documentaire la plus populaire. Dans ce cours de démarrage rapide, vous serez opérationnel avec MongoDB et Python.

Votre enseignant est Michael Kennedy. Il est l'animateur du podcast Talk Python to Me et il travaille directement avec l'équipe MongoDB pour aider les gens à utiliser MongoDB plus facilement.

![Image](https://cdn-media-1.freecodecamp.org/ghost/2019/02/download.png)

Voici quelques-uns des sujets abordés dans le cours :

* Pourquoi NoSQL et MongoDB
* Modélisation des données dans les bases de données documentaires
* Introduction à MongoEngine
* Construction de la couche de données avec MongoEngine

À la fin du cours, vous serez prêt à commencer à utiliser MongoDB dans vos propres projets Python.

Vous pouvez regarder la vidéo complète sur la chaîne [freeCodeCamp.org YouTube](https://www.youtube.com/watch?v=E-1xI85Zog8) (2 heures de visionnage).
