Les expressions régulières, ou simplement regex, sont utilisées dans presque tous les langages de programmation pour définir un motif de recherche qui peut être utilisé pour rechercher des éléments dans une chaîne de caractères.

Vous pouvez apprendre les bases des expressions régulières gratuitement dans ce cours intensif complet. Il se concentre sur l'utilisation des regex en JavaScript, mais les principes s'appliquent à de nombreux autres langages de programmation que vous pourriez choisir d'utiliser.

Ce cours suit [le programme gratuit sur les expressions régulières de freeCodeCamp.org](https://learn.freecodecamp.org/javascript-algorithms-and-data-structures/regular-expressions).

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=ZfQFUJhPqMM) (45 minutes de visionnage).
