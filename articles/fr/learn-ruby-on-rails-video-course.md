Ruby on Rails est un framework de développement web qui simplifie et accélère la création d'applications web full stack par rapport à de nombreux autres frameworks. Ruby on Rails fournit la structure de base nécessaire pour créer des pages web, des services web et des bases de données.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous aidera à apprendre Ruby on Rails en construisant une application de liste d'amis.

Le cours a été créé par John Elder de Codemy.com. John a écrit deux livres best-sellers sur Ruby on Rails et a créé de nombreuses formations vidéo.

Dans ce cours, vous allez construire une application qui garde une trace des informations de contact de vos amis. Vous apprendrez à implémenter la fonctionnalité CRUD. CRUD signifie Create, Read, Update, et Destroy et est utilisé dans presque toutes les applications web.

Voici les sujets abordés dans ce cours :

* Introduction et Installation
* Première Page Web et Vue d'ensemble MVC
* Liens Partiels de l'Application et Nouvelles Pages
* Échafaudage CRUD
* Styliser l'Application avec Bootstrap
* Styliser les Vues Devise
* Associations
* Modifications de Style
* S'amuser avec le Contrôleur
* Git, GitHub, et Heroku

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=fmyvWz5TUWg) (4 heures de visionnage).
