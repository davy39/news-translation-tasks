_Note : nous avons initialement publié cela sur notre blog maintenant disparu en janvier 2015._

Nous sommes ravis d'annoncer que tout le monde chez Free Code Camp a maintenant un portfolio public gratuit !  
   
Vous avez travaillé dur pour compléter nos défis de codage et construire des applications JavaScript. Il est maintenant temps de montrer au monde vos progrès !  
   
Votre nouveau Portfolio Free Code Camp inclut :

* Votre nom et où vous êtes basé
* Votre bio de 140 caractères
* Liens vers votre flux Twitter, profil LinkedIn, projets CodePen, badges CoderByte et compte Github
* Une liste des défis Free Code Camp que vous avez complétés, et quand
* Un portfolio de jusqu'à 3 sites web que vous avez construits, avec un lien et une capture d'écran pour chacun

Votre Portfolio Free Code Camp est un lien personnalisé gratuit qui peut servir de vitrine complète de vos progrès en tant que codeur.

Jen a été l'une des premières Code Campers à personnaliser complètement sa page de portfolio. Félicitations, Jen ! Votre portfolio sera également indexé par les moteurs de recherche, vous offrant une exposition supplémentaire et boostant votre marque personnelle en tant que codeur. Vous pouvez faire apparaître votre portfolio plus haut dans les résultats de recherche en y faisant des liens avec vos autres profils de médias sociaux.  
   
Vous possédez maintenant une partie de Free Code Camp. Votre profil sera situé à freecodecamp.com/[votre nom d'utilisateur]. Vous pouvez le personnaliser à [www.freecodecamp.com/account](http://www.freecodecamp.com/account).  
   
Vous remarquerez également que nous avons ajouté Angular.js à Free Code Camp et "angularisé" tous nos formulaires web. Cela devrait faciliter la personnalisation de votre portfolio public.  
   
Nous sommes enfin une application MEAN Stack complète (MongoDB, Express.js, Angular.js, Node.js) ! Un grand merci au Conseiller de Camp [Terakilobyte](https://github.com/terakilobyte) pour avoir dirigé l'effort d'implémentation d'Angular.js sur Free Code Camp !  
   
Si vous voyez des endroits où Angular.js pourrait être utilisé dans notre application, [forkez-nous sur GitHub](https://github.com/FreeCodeCamp/freecodecamp) et soumettez une pull request. Nous aimerions Angulariser toute l'application !
