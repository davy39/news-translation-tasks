Un nom d'utilisateur est un nom qui identifie un utilisateur dans un système informatique ou un site web.

Les noms d'utilisateur sont souvent uniques, ce qui signifie que deux utilisateurs ne peuvent pas partager le même nom d'utilisateur. De plus, les noms d'utilisateur sont généralement associés à un mot de passe, et les deux peuvent être requis pour se connecter à un ordinateur ou à un site web.

Les noms d'utilisateur prennent de nombreuses formes. Par exemple, sur Twitter, les noms d'utilisateur doivent être uniques et commencent par un symbole @. Ainsi, @freeCodeCamp signifie que le nom d'utilisateur est freeCodeCamp.

Mais pour une adresse e-mail, le nom d'utilisateur de votre compte est tout ce qui précède le symbole @. Ainsi, pour l'adresse e-mail ghost@freecodecamp.org, le nom d'utilisateur est ghost.
