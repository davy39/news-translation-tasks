Scikit-learn est une bibliothèque d'apprentissage automatique open-source pour Python, connue pour sa simplicité, sa polyvalence et son accessibilité. La bibliothèque est bien documentée et soutenue par une grande communauté, ce qui en fait un choix populaire pour les débutants et les praticiens expérimentés dans le domaine de l'apprentissage automatique.

Nous venons de publier un cours de 18 heures sur la chaîne YouTube freeCodeCamp.org qui est une introduction pratique et concrète à l'apprentissage automatique avec Python et Scikit-Learn. Il est destiné aux débutants ayant des connaissances de base en Python et en statistiques.

Le cours est conçu et enseigné par Aakash N S, PDG et cofondateur de Jovian. Aakash a créé de nombreux cours populaires sur l'apprentissage automatique.

Le cours commence par les bases de l'apprentissage automatique en explorant des modèles comme la régression linéaire et logistique, puis passe à des modèles basés sur des arbres comme les arbres de décision, les forêts aléatoires et les machines à gradient boosting.

Le cours aborde également les meilleures pratiques pour aborder et gérer des projets d'apprentissage automatique et démontre comment construire un modèle d'apprentissage automatique de pointe pour un ensemble de données réel à partir de zéro. Ensuite, le cours examine brièvement l'apprentissage non supervisé et les recommandations, et guide à travers le processus de déploiement d'un modèle d'apprentissage automatique dans le cloud en utilisant le framework web Flask.

Vous apprendrez tout ce que vous devez savoir pour commencer à utiliser Scikit-learn pour l'apprentissage automatique. Scikit-learn offre une large gamme d'outils pour diverses tâches d'apprentissage automatique, y compris la classification, la régression, le clustering, la réduction de dimensionnalité, la sélection de modèles et le prétraitement. Scikit-learn est construit sur NumPy, SciPy et Matplotlib, et son interface conviviale permet une intégration facile dans les applications Python.

À la fin de ce cours, vous serez en mesure de construire, d'entraîner et de déployer des modèles d'apprentissage automatique dans le monde réel avec confiance. Pour tirer le meilleur parti de ce cours, suivez-le et tapez tout le code vous-même, et appliquez les techniques couvertes ici à d'autres ensembles de données réels et compétitions que vous pouvez trouver sur des plateformes comme Kaggle.

Voici les leçons de ce cours :

* Leçon 1 - Régression linéaire et descente de gradient
* Leçon 2 - Régression logistique pour la classification
* Leçon 3 - Arbres de décision et forêts aléatoires
* Leçon 4 - Comment aborder les projets d'apprentissage automatique
* Leçon 5 - Machines à gradient boosting avec XGBoost
* Leçon 6 - Apprentissage non supervisé avec Scikit-Learn
* Leçon 7 - Projet d'apprentissage automatique à partir de zéro
* Leçon 8 - Déploiement d'un projet d'apprentissage automatique avec Flask

Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=hDKCxebp88A) (18 heures de visionnage).
