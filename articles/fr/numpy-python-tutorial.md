Apprenez les bases de la bibliothèque NumPy pour Python dans ce tutoriel de Keith Galli. Le tutoriel explique comment NumPy fonctionne et comment écrire du code avec NumPy. Vous apprendrez à créer des tableaux, à les indexer, à faire des mathématiques, des statistiques, à les remodeler, et plus encore.

Voici les sujets abordés :

* Qu'est-ce que NumPy
* NumPy vs Listes (vitesse, fonctionnalité)
* Applications de NumPy
* Les bases (création de tableaux, forme, taille, type de données)
* Accéder/Changer des éléments spécifiques, des lignes, des colonnes, etc. (découpage)
* Initialisation de différents tableaux (1s, 0s, full, random, etc...)
* Copier des variables
* Mathématiques de base (arithmétique, trigonométrie, etc.)
* Algèbre linéaire
* Statistiques
* Réorganiser les tableaux (reshape, vstack, hstack)
* Charger des données à partir d'un fichier
* Indexation avancée et masquage booléen

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=QUT1VHiLmmI) (1 heure de visionnage).
