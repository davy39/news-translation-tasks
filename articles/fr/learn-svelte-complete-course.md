Svelte est un framework JavaScript front-end pour créer des pages web interactives. Svelte a gagné en popularité ces dernières années.

Nous venons de publier un cours gratuit de 23 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Svelte.

Li Hau Tan a développé ce cours. Li est un mainteneur principal de Svelte, ce qui en fait la personne idéale pour enseigner ce cours.

Lee commence par les bases absolues avec la création de votre premier composant Svelte et continue pour enseigner chaque concept que vous devez connaître pour développer des applications avec Svelte.
