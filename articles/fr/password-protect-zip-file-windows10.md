Les fichiers Zip contiennent souvent plusieurs fichiers volumineux, vous pourriez donc vouloir les chiffrer ou les protéger avec un mot de passe. Ainsi, seules certaines personnes y auront accès.

Dans ce guide, je vais vous montrer une excellente façon de protéger un fichier Zip par mot de passe, afin que vous puissiez être assuré que seules les personnes qui doivent le voir y auront accès.

## Qu'est-ce qu'un fichier Zip ?

Un fichier Zip, également appelé dossier Zip, vous aide à compresser plusieurs fichiers en un seul fichier géant. Cela vous permet de les sauvegarder facilement dans un seul fichier, ou de les envoyer à qui vous voulez en une seule fois. Les fichiers Zip ont l'extension .zip.

En plus de pouvoir compresser plusieurs fichiers en un seul, vous avez également l'avantage de réduire la taille des fichiers et de pouvoir protéger le fichier Zip avec un mot de passe.

## Comment protéger un fichier Zip par mot de passe sur Windows 10

Windows 10 ne propose qu'une façon de chiffrer un fichier Zip, mais pas de le protéger par mot de passe. Ainsi, pour protéger un fichier Zip par mot de passe sur Windows 10, vous avez besoin d'une application tierce qui fonctionne sur Windows. Un exemple d'une telle application tierce est WinRAR.

### Comment protéger un fichier Zip par mot de passe avec WinRAR

**Étape 1** : La première chose à faire est de télécharger WinRAR depuis son site officiel. 
![dowwnloadd-winRAR](https://www.freecodecamp.org/news/content/images/2021/11/dowwnloadd-winRAR.png)

Lorsque vous téléchargez l'installateur (généralement avec une extension .exe), ouvrez-le et suivez l'assistant d'installation pour installer WinRAR.

**Étape 2** : Ouvrez le fichier Zip avec WinRAR
![ss-1-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-1-2.jpg)

**Étape 3** : Dans les éléments du menu, sélectionnez "Outils" et choisissez "Convertir les archives". Vous pouvez également appuyer sur `Alt` + `Q` sur votre clavier pour faire cela rapidement.
![ss-2-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-2-2.jpg)

**Étape 4** : Cliquez sur le bouton "Compression..." dans la fenêtre contextuelle qui apparaît.
![ss-3-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-3-2.jpg)

**Étape 5** : Cliquez sur "Définir le mot de passe..." dans la fenêtre contextuelle suivante qui apparaît.
![ss-4-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-4-2.jpg)

**Étape 6** : Saisissez le mot de passe de votre choix dans le champ "Entrez le mot de passe", et confirmez-le dans le champ "Réentrez le mot de passe pour vérification" et cliquez sur "Ok".
![ss-5-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-5-2.jpg)

**Étape 7** : Cliquez à nouveau sur "Ok". Une fenêtre contextuelle apparaîtra vous demandant si vous souhaitez chiffrer les archives converties. Cliquez sur Oui.
![ss-6-1](https://www.freecodecamp.org/news/content/images/2021/11/ss-6-1.jpg)

**Étape 8** : Cliquez à nouveau sur "Ok". WinRAR va maintenant procéder à la protection de votre fichier Zip avec le mot de passe défini. Plus le fichier est volumineux, plus cela prend du temps. 

Une fois terminé, cliquez sur le bouton Fermer et c'est tout.
![ss-7](https://www.freecodecamp.org/news/content/images/2021/11/ss-7.png)

Vous aurez maintenant un fichier Zip séparé avec l'extension `.rar`. C'est celui qui sera protégé par un mot de passe.
![ss-8](https://www.freecodecamp.org/news/content/images/2021/11/ss-8.jpg)

### Comment protéger un fichier Zip par mot de passe avec 7-Zip

Une autre application tierce pour protéger vos fichiers Zip avec des mots de passe est 7-Zip.

Suivez les étapes ci-dessous pour protéger vos fichiers Zip avec 7-Zip.

**Étape 1** : Téléchargez l'application 7-Zip depuis leur site web et installez-la.
![downlooad-7-zip](https://www.freecodecamp.org/news/content/images/2021/11/downlooad-7-zip.png)

**Étape 2** : Faites un clic droit sur le dossier que vous souhaitez compresser et survolez l'option 7-Zip. Dans le menu qui apparaît pendant que vous survolez, sélectionnez "Ajouter à l'archive".
![ss-9](https://www.freecodecamp.org/news/content/images/2021/11/ss-9.png)

**Étape 4** : Dans la section "Chiffrement", entrez votre mot de passe souhaité dans le champ "Entrez le mot de passe" et confirmez-le dans le champ "Réentrez le mot de passe".
![ss-10](https://www.freecodecamp.org/news/content/images/2021/11/ss-10.jpg)

**Étape 5** : Cliquez sur "Ok" pour enfin créer le fichier Zip et le protéger avec un mot de passe.

## Conclusion

Puisque vous pouvez compresser plusieurs fichiers en un seul dossier Zip géant, vous pourriez avoir besoin de le protéger avec un mot de passe. Cela garantit que seules les personnes qui ont accès peuvent ouvrir le fichier Zip.
