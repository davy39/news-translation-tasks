Dans ce cours de visualisation de données pour débutants de Curran Kelleher, vous apprendrez à transformer des données en formes graphiques significatives. Vous apprendrez à créer des visualisations en utilisant D3.js et les technologies web. D3 est une bibliothèque JavaScript pour visualiser des données avec HTML, SVG et CSS.

En plus d'enseigner tout sur D3, le cours couvre également les bases de JavaScript, HTML, CSS et SVG afin que vous ayez toutes les connaissances préalables pour créer des visualisations de données époustouflantes.

Ce cours complet couvre les sujets suivants :

* Pourquoi visualiser des données ?
* Entrées pour la visualisation de données : Données et tâches
* Introduction à JavaScript
* Introduction à HTML, CSS et SVG
* Introduction à D3.js - Créons un visage !
* Créer un graphique à barres avec D3.js et SVG
* Personnalisation des axes d'un graphique à barres avec D3.js
* Créer un nuage de points avec D3.js
* Créer des graphiques en ligne et en aire avec D3.js
* Le modèle général de mise à jour de D3.js
* Marques et canaux en visualisation de données
* Interaction avec un flux de données unidirectionnel en utilisant D3.js
* Créer une carte du monde avec D3
* Astuces économiques pour l'interaction sur une carte du monde D3.js
* Toile vierge
* Construction d'une visualisation arborescente des pays du monde avec D3.js
* Légendes de couleur et de taille avec D3.js
* Carte choroplèthe avec D3.js
* Filtrage interactif sur une carte choroplèthe
* Utilisation de la couleur dans la visualisation
* Nuage de points avec menus
* Cercles sur une carte
* Graphique en ligne avec plusieurs lignes
* Fusion et transformation de données avec JavaScript
* Sélection d'une année sur un graphique en ligne

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=_8V5o2UHG0E) (13 heures de visionnage).
