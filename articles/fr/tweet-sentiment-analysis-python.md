Apprenez à analyser des tweets dans ce tutoriel Python. Vincent Russo montre comment utiliser le module Tweepy pour diffuser des tweets en direct directement depuis Twitter en temps réel. Les tweets sont visualisés, puis le module TextBlob est utilisé pour effectuer une analyse de sentiments sur les tweets.

Voici les sections de la vidéo :

* Diffusion de tweets en direct
* Curseur et pagination
* Analyse des données de tweets
* Visualisation des données de tweets
* Analyse de sentiments

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=1gQ6uG5Ujiw) (90 minutes de visionnage).
