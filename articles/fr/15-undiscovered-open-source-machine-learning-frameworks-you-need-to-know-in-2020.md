Par Davis David

Le Machine Learning (ML) est l'une des technologies émergentes les plus rapides aujourd'hui. Et l'application du machine learning à différents domaines de l'informatique gagne rapidement en popularité.

Ce n'est pas seulement grâce à l'existence de matériel bon marché et puissant. C'est aussi grâce à la disponibilité croissante de [frameworks de **Machine Learning**](https://machinelearningmastery.com/types-of-learning-in-machine-learning/) **libres et open-source**, qui permettent aux développeurs de mettre en œuvre facilement le machine learning.

Cette large gamme de frameworks open-source de machine learning permet aux data scientists et aux ingénieurs en machine learning de construire, implémenter et maintenir des systèmes de machine learning, générer de nouveaux projets et créer de nouveaux systèmes de machine learning impactants.

Choisir un Framework ou une bibliothèque de Machine Learning pour résoudre votre cas d'utilisation implique de faire une évaluation pour décider ce qui est adapté à votre cas d'utilisation. Plusieurs facteurs sont importants pour cette évaluation, tels que :

* Facilité d'utilisation.
* Support sur le marché (Communauté).
* Vitesse d'exécution.
* Ouverture.

### **À qui s'adresse cet article ?**

Cet article s'adresse à ceux qui veulent utiliser les connaissances en pratique après avoir appris la théorie.

Il s'adresse également à ceux qui veulent explorer d'autres frameworks open-source potentiels de machine learning pour leur futur projet ML.

Voici maintenant la liste des frameworks ou bibliothèques open-source et méconnus que les entreprises et les particuliers peuvent utiliser pour construire des systèmes de machine learning.

## 1. Blocks

> Blocks est un framework qui vous aide à construire des modèles de réseaux de neurones sur Theano. Actuellement, il supporte et fournit la construction d'opérations Theano paramétrées, appelées « bricks », la correspondance de motifs pour sélectionner des variables et des bricks dans de grands modèles, des algorithmes pour optimiser votre modèle et la sauvegarde et la reprise de l'entraînement. [Dépôt de Blocks](https://github.com/mila-iqia/blocks)

Vous pouvez également en apprendre davantage sur [Fuel](https://github.com/mila-udem/fuel), le moteur de traitement de données développé principalement pour Blocks.

Langage de programmation : Python  
Lien GitHub : [https://github.com/mila-iqia/blocks](https://github.com/mila-iqia/blocks)

## 2. Analytics Zoo

![Image](https://www.freecodecamp.org/news/content/images/2020/05/1_U0_D44bljlEYgkwynkf9VQ.jpeg)

> Analytics Zoo fournit une plateforme unifiée d'analyse de données et d'IA qui unit de manière transparente les programmes TensorFlow, Keras, PyTorch, Spark, Flink et Ray en un pipeline intégré, qui peut être mis à l'échelle de manière transparente d'un ordinateur portable à de grands clusters pour traiter des big data de production. [Dépôt d'Analytics Zoo](https://github.com/intel-analytics/analytics-zoo)

Quand utiliser Analytics Zoo pour développer votre solution IA :

* Vous voulez prototyper facilement des modèles IA.
* Quand la mise à l'échelle est importante pour vous.
* Quand vous voulez ajouter des processus d'automatisation dans votre pipeline de machine learning, tels que l'ingénierie des caractéristiques et la sélection de modèles.

Ce projet est maintenu par [Intel-analytics](https://github.com/intel-analytics).

Langage de programmation : Python  
Lien GitHub : [https://github.com/intel-analytics/analytics-zoo](https://github.com/intel-analytics/analytics-zoo)

## 3. ML5.js

![Image](https://www.freecodecamp.org/news/content/images/2020/05/5-1.png)

> Ml5.js vise à rendre le machine learning accessible à un large public d'artistes, de codeurs créatifs et d'étudiants. La bibliothèque fournit un accès aux algorithmes et modèles de machine learning dans le navigateur, en s'appuyant sur [TensorFlow.js](https://js.tensorflow.org/). ["Dépôt Ml5.js"](https://github.com/ml5js/ml5-library)

ml5.js est inspiré par [Processing](https://processing.org/) et [p5.js](https://p5js.org/).

Ce projet open source est développé et maintenu par le programme Interactive Telecommunications/Interactive Media Arts de NYU et par des artistes, designers, étudiants, technologues et développeurs du monde entier.

**NOTE :** Ce projet est actuellement en développement.

Langage de programmation : Javascript  
Lien GitHub : [https://github.com/ml5js/ml5-library](https://github.com/ml5js/ml5-library)

## 4. AdaNet

![Image](https://www.freecodecamp.org/news/content/images/2020/05/adanet.png)

> **AdaNet** est un framework léger basé sur TensorFlow pour l'apprentissage automatique de modèles de haute qualité avec une intervention minimale d'experts. AdaNet s'appuie sur les efforts récents d'AutoML pour être rapide et flexible tout en fournissant des garanties d'apprentissage. Importamment, AdaNet fournit un cadre général non seulement pour l'apprentissage d'une architecture de réseau de neurones, mais aussi pour l'apprentissage d'un ensemble afin d'obtenir des modèles encore meilleurs. ["Dépôt AdaNet"](https://github.com/tensorflow/adanet)

AdaNet fournit une API familière comme Keras pour l'entraînement, l'évaluation et le service de vos modèles en production.

Langage de programmation : Python  
Lien GitHub : [https://github.com/tensorflow/adanet](https://github.com/tensorflow/adanet)

## 5. Mljar

Si vous cherchez une plateforme pour créer des modèles prototypes et un service de déploiement, Mljar est le bon choix pour vous. Mljar tend à rechercher différents algorithmes et à effectuer l'ajustement des hyper-paramètres pour trouver le meilleur modèle.

Il fournit également des résultats rapides en exécutant tous les calculs dans le cloud et en créant finalement des modèles d'ensemble. Ensuite, il crée des rapports markdown à partir de l'entraînement AutoML.

![Image](https://www.freecodecamp.org/news/content/images/2020/06/automl_summary.gif)
_rapport markdown_

Mljar peut entraîner des modèles ML pour :

* Classification binaire,
* Classification multi-classes,
* Régression.

Mljar propose deux types d'interfaces :

* Un wrapper Python sur l'API Mljar.
* Exécution de modèles de Machine Learning dans votre navigateur web.

Langage de programmation : Python  
Lien GitHub : [https://github.com/mljar/mljar-supervised](https://github.com/mljar/mljar-supervised).

## 6. ConvNetJS

![Image](https://www.freecodecamp.org/news/content/images/2020/05/js.png)

> Deep Learning en Javascript. Entraînez des réseaux de neurones convolutionnels (ou ordinaires) dans votre navigateur. ["Dépôt convnetjs"](https://github.com/karpathy/convnetjs)

Comme Tensorflow.js, ConvNetJS est une bibliothèque JavaScript qui supporte l'entraînement de différents modèles de Deep Learning dans votre navigateur web. Vous n'avez pas besoin de GPUs et d'autres logiciels lourds.

ConvNetJS supporte :

* Modules de réseaux de neurones.
* Entraînement de réseaux convolutionnels pour les images.
* Fonctions de coût de régression et de classification.
* Module d'apprentissage par renforcement, basé sur le Deep Q Learning.

**Note :** Non activement maintenu.

Langage de programmation : Javascript  
Lien GitHub : [https://github.com/karpathy/convnetjs](https://github.com/karpathy/convnetjs)

## 7. NNI (Neural Network Intelligence)

![Image](https://www.freecodecamp.org/news/content/images/2020/05/nni_logo.png)
_Logo NNI_

> **NNI (Neural Network Intelligence)** est un outil léger mais puissant pour aider les utilisateurs à **automatiser** l'ingénierie des caractéristiques, la recherche d'architecture neuronale, l'ajustement des hyper-paramètres et la compression de modèles. L'outil gère les expériences d'apprentissage automatique automatisé (AutoML), **dispatche et exécute** les travaux d'essai des expériences générés par les algorithmes d'ajustement pour rechercher la meilleure architecture neuronale et/ou les meilleurs hyper-paramètres dans **différents environnements d'entraînement** comme la machine locale, les serveurs distants, OpenPAI, Kubeflow et d'autres options cloud. [Dépôt NNI](https://github.com/Microsoft/nni)

**Quand utiliser NNI**

* Si vous voulez essayer différents algorithmes AutoML.
* Si vous voulez exécuter des travaux d'essai AutoML dans différents environnements.
* Si vous voulez supporter AutoML sur votre plateforme.

NOTE : Projet open source de Microsoft.

Langage de programmation : Python  
Lien GitHub : [https://github.com/Microsoft/nni](https://github.com/Microsoft/nni)

## 8. Datumbox

![Image](https://www.freecodecamp.org/news/content/images/2020/05/687474703a2f2f7777772e646174756d626f782e636f6d2f696d672f6c6f676f2e706e67-1.png)
_Logo Datumbox_

> Le Framework de Machine Learning Datumbox est un framework open-source écrit en Java qui permet le développement rapide d'applications de Machine Learning et de statistiques. L'objectif principal du framework est d'inclure un grand nombre d'algorithmes de machine learning et de méthodes statistiques et de pouvoir gérer des ensembles de données de grande taille. ["Dépôt DatumBox"](https://github.com/datumbox/datumbox-framework)

Datumbox fournit un certain nombre de modèles pré-entraînés pour différentes tâches telles que la détection de spam, l'analyse de sentiment, la détection de langue, la classification de sujets, etc.

Langage de programmation : Java  
Lien GitHub : [https://github.com/datumbox/datumbox-framework](https://github.com/datumbox/datumbox-framework)

## 9. XAI (Une boîte à outils d'explicabilité pour le ML)

> XAI est une bibliothèque de Machine Learning conçue avec l'explicabilité de l'IA au cœur. XAI contient divers outils qui permettent l'analyse et l'évaluation des données et des modèles. La bibliothèque XAI est maintenue par [The Institute for Ethical AI & ML](http://ethical.institute/), et elle a été développée sur la base des [8 principes pour un Machine Learning Responsable](http://ethical.institute/principles.html). ["Dépôt XAI"](https://github.com/EthicalML/xai)

Les 8 principes pour un Machine Learning Responsable incluent :

* Augmentation humaine
* Évaluation des biais
* Explicabilité par justification
* Opérations reproductibles
* Stratégie de déplacement
* Précision pratique
* Confiance par la confidentialité
* Sensibilisation aux risques des données

Pour en savoir plus sur XAI, vous pouvez consulter cette [conférence à Tensorflow London](https://www.youtube.com/watch?v=GZpfBhQJ0H4). Elle contient des informations sur les définitions et les principes de cette bibliothèque.

XAI est actuellement en développement précoce, la version actuelle est 0.05 (Alpha).

Langage de programmation : Python  
Lien GitHub : [https://github.com/EthicalML/xai](https://github.com/EthicalML/xai)

## 10. Plato

![Image](https://www.freecodecamp.org/news/content/images/2020/05/plat.png)
_Logo Plato_

Plato est un framework flexible pour le développement d'agents d'IA conversationnels dans différents environnements. Plato a été conçu à la fois pour les utilisateurs ayant une expérience limitée en IA conversationnelle et pour les chercheurs expérimentés dans ce domaine. Il fournit un design clair et compréhensible, s'intègre avec les frameworks existants de deep learning et d'optimisation bayésienne, et réduit le besoin d'écrire du code.

Il supporte les interactions via le texte, la parole et les actes de dialogue. Pour apprendre comment fonctionne le système de dialogue de recherche Plato, lisez l'article [ici](https://eng.uber.com/plato-research-dialogue-system/).

**NOTE :** Plato est un projet open source d'Uber.

Langage de programmation : Python  
Lien GitHub : [https://github.com/uber-research/plato-research-dialogue-system](https://github.com/uber-research/plato-research-dialogue-system)

## 11. DeepDetect

![Image](https://www.freecodecamp.org/news/content/images/2020/05/deepdetect.png)

> DeepDetect est une API et un serveur de machine learning écrits en C++. Il rend le machine learning de pointe facile à utiliser et à intégrer dans les applications existantes.
> DeepDetect implémente le support pour l'apprentissage supervisé et non supervisé de deep learning d'images, de texte, de séries temporelles et d'autres données, avec un accent sur la simplicité et la facilité d'utilisation, de test et de connexion dans les applications existantes. Il supporte la classification, la détection d'objets, la segmentation, la régression et les autoencodeurs. [Dépôt DeepDetect](https://github.com/jolibrain/deepdetect)

DeepDetect s'appuie sur des bibliothèques externes de machine learning telles que :

* Bibliothèque de boosting de gradient [XGBoost](https://github.com/dmlc/xgboost).
* Bibliothèques de deep learning ([Caffe](https://github.com/BVLC/caffe), [Tensorflow](https://tensorflow.org/), [Caffe2](https://caffe2.ai/), [Torch](https://pytorch.org/), [NCNN](https://github.com/Tencent/ncnn), et [Dlib](http://dlib.net/ml.html)).
* Clustering avec [T-SNE](https://github.com/DmitryUlyanov/Multicore-TSNE).
* Recherche de similarité avec [Annoy](https://github.com/spotify/annoy/) et [FAISS](https://github.com/facebookresearch/faiss).

![Image](https://www.freecodecamp.org/news/content/images/2020/05/faces_emo.jpg)
_Détection d'émotions faciales_

DeepDetect est conçu, implémenté et supporté par [Jolibrain](http://jolibrain.com/) avec l'aide d'autres contributeurs.

Langage de programmation : C++  
Lien GitHub : [https://github.com/jolibrain/deepdetect](https://github.com/jolibrain/deepdetect)

## 13. Dopamine

![Image](https://www.freecodecamp.org/news/content/images/2020/05/68747470733a2f2f676f6f676c652e6769746875622e696f2f646f70616d696e652f696d616765732f646f70616d696e655f6c6f676f2e706e67.png)

> Dopamine est un framework de recherche pour le prototypage rapide d'algorithmes d'apprentissage par renforcement. Il vise à combler le besoin d'une base de code petite et facilement compréhensible dans laquelle les utilisateurs peuvent librement expérimenter avec des idées sauvages (recherche spéculative). [Dépôt Dopamine](https://github.com/google/dopamine)

Les principes de conception de Dopamine incluent :

* Expérimentation facile.
* Développement flexible.
* Compact et fiable.
* Reproductible.

L'année dernière (2019), Dopamine a basculé ses définitions de réseau pour utiliser **tf.keras.Model**. Les réseaux précédents basés sur **tf.contrib.slim** ont été supprimés.

Pour apprendre à utiliser Dopamine, consultez les [cahiers Colaboratory](https://github.com/google/dopamine/tree/master/dopamine/colab).

## 14. TuriCreate

![Image](https://www.freecodecamp.org/news/content/images/2020/05/1_kUkBXPEDhxB6UaWNKPgqTQ.jpeg)
_TuriCreate_

> TuriCreate est un ensemble d'outils open-source pour créer des modèles Core ML personnalisés.

Avec TuriCreate, vous pouvez accomplir différentes tâches ML telles que la classification d'images, la classification de sons, la détection d'objets, le transfert de style, la classification d'activités, le système de recommandation de similarité d'images, la classification de texte et le clustering.

Le framework est simple à utiliser, flexible et visuel. Il fonctionne sur de grands ensembles de données et est prêt à être déployé. Les modèles entraînés peuvent être utilisés immédiatement dans les applications iOS, macOS, tvOS et watchOS sans aucune conversion supplémentaire.

Consultez les conférences TuriCreate à [WWDC 2019](https://developer.apple.com/videos/play/wwdc2019/420/) et [WWDC 2018](https://developer.apple.com/videos/play/wwdc2018/712/) pour en savoir plus sur TuriCreate.

**NOTE :** TuriCreate est un projet open source d'Apple.

Langage de programmation : Python  
Lien GitHub : [https://github.com/apple/turicreate](https://github.com/apple/turicreate)

## 15. Flair

![Image](https://www.freecodecamp.org/news/content/images/2020/05/flair_logo_2020.png)
_Logo Flair_

Flair est un framework simple de traitement du langage naturel (NLP), développé et open-sourcé par l'[Université Humboldt de Berlin](https://www.informatik.hu-berlin.de/en/forschung-en/gebiete/ml-en/). Flair fait officiellement partie de l'**écosystème PyTorch** et est utilisé dans des centaines de projets industriels et académiques.

> Flair vous permet d'appliquer nos modèles de traitement du langage naturel (NLP) de pointe à votre texte, tels que la reconnaissance d'entités nommées (NER), l'étiquetage morpho-syntaxique (PoS), la désambiguïsation de sens et la classification. [Dépôt Flair](https://github.com/flairNLP/flair)

Flair surpasse les meilleures méthodes précédentes sur une gamme de tâches NLP : Reconnaissance d'entités nommées, étiquetage morpho-syntaxique et segmentation. Consultez ce tableau :

**Note** : Le score F1 est une métrique d'évaluation principalement utilisée pour les tâches de classification. Le score F1 prend en considération la distribution des classes présentes.

## Conclusion

Avant de commencer à construire une application de machine learning, vous devez sélectionner un framework ML parmi les nombreuses options disponibles. Cela peut être une tâche difficile.

Par conséquent, il est important d'évaluer _plusieurs options_ avant de prendre une décision finale. Les frameworks open-source de machine learning mentionnés ci-dessus peuvent aider quiconque à construire des modèles de machine learning efficacement et facilement.
