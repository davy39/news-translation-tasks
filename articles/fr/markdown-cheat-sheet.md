Markdown a gagné en popularité car il est facile à utiliser et largement accepté sur différentes plateformes. 

Vous pouvez utiliser Markdown pour écrire du contenu qui peut être transmis en texte brut. Un bon exemple serait un article de blog.

Dans cet article, vous apprendrez ce qu'est Markdown et comment l'utiliser.

## Qu'est-ce que Markdown ?

Markdown est un langage de balisage tout comme HTML. Nous l'utilisons pour analyser du texte et le convertir dans un format spécifique. Vous pouvez également le considérer comme un outil de conversion de texte en HTML. 

De nombreux développeurs aiment écrire en Markdown car cela leur donne un contrôle précis sur leur texte et leur code. Nous verrons comment et pourquoi dans les paragraphes suivants.

## Dans ce guide, nous aborderons les sujets suivants.

* Comment créer votre premier fichier Markdown.
* Créer une feuille de triche pour Markdown
* Discuter de la façon dont Markdown peut être rendu dans VS Code

## Outils qui supportent Markdown

Markdown fonctionne dans n'importe quel navigateur, même si vous utilisez un simple bloc-notes. Mais il existe certains outils qui peuvent aider à améliorer votre productivité en fournissant une vue en temps réel (de Markdown et de texte riche) côte à côte.

Voici quelques-uns des outils qui supportent le travail avec Markdown :

* VSCode (Nous allons couvrir cela dans cet article)
* Atom
* Haroopad
* Sublime Text
* MarkPad

### Échapper des caractères en Markdown

Si vous souhaitez que le navigateur ignore la syntaxe et conserve les caractères, les caractères peuvent être échappés en utilisant l'antislash `\`.

Par exemple, \* ne parsera pas ses caractères suivants comme de l'italique.

Un excellent exemple pour Markdown est le fichier README.md de GitHub. Là, les blocs de code sont facilement combinés avec du texte bien formaté.
