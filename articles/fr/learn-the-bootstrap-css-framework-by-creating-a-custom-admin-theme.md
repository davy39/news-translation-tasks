Bootstrap est un framework CSS populaire qui aide les développeurs web à créer des sites web responsives, mobile-first. Apprenez à utiliser Bootstrap dans ce cours vidéo complet d'Andrew Wilson.

Dans ce cours intermédiaire, vous apprendrez Bootstrap en créant un thème d'administration personnalisé à partir de zéro en utilisant webpack.

Tout d'abord, Andrew démontre comment configurer npm et webpack afin de pouvoir configurer les fichiers sources de Bootstrap qui sont écrits en SASS. Ensuite, il enseigne les bases de Bootstrap et crée quelques templates de base qui sont courants dans les formulaires d'administration. Enfin, il personnalise le thème en écrivant des styles personnalisés et en écrasant certains des JavaScript de Bootstrap.

Regardez la vidéo complète ci-dessous ou sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=RyTRgQ7k6QE) (3 heures de visionnage).
