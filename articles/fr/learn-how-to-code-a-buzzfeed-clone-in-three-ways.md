Vous cherchez à améliorer vos compétences en développement web ? Vous voulez savoir comment construire un clone de BuzzFeed en utilisant trois technologies différentes ? Ne cherchez plus !

Nous venons de publier un cours vidéo de 4 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à coder un clone de BuzzFeed en utilisant trois méthodes différentes.

Ania Kubów a développé ce cours. Elle est une créatrice de cours de logiciels très populaire et une contributrice fréquente de la chaîne YouTube freeCodeCamp.org.

Dans ce cours complet, vous apprendrez à construire un clone de BuzzFeed en JavaScript, suivi d'une version en React utilisant JSON server pour simuler une API, et enfin à construire un clone BuzzFeed en TypeScript avec un mini backend utilisant Node.js pour communiquer avec une base de données. À la fin du cours, vous aurez une solide compréhension de la construction d'un clone de BuzzFeed en trois technologies différentes, allant du niveau débutant à intermédiaire.

Dans le cadre du projet, vous construirez un quiz pour les utilisateurs qui pose la question "Quel fromage êtes-vous ?". En utilisant une logique de défilement que vous développerez dans le cours, le quiz passera automatiquement à la question suivante après qu'un utilisateur ait répondu à une question. Vous apprendrez à utiliser JavaScript pour créer un quiz interactif qui maintiendra les utilisateurs engagés et divertis.

Le cours est conçu pour les débutants qui ont une compréhension de base de HTML, CSS et JavaScript, mais toute personne intéressée par le développement web peut en bénéficier. L'instructrice est expérimentée et compétente, et le cours est conçu pour être facile à suivre, afin que vous puissiez commencer à construire votre clone de BuzzFeed immédiatement.

Ce cours est parfait pour ceux qui cherchent à améliorer leurs compétences en développement web ou pour ceux qui veulent simplement apprendre à construire un quiz interactif.

Regardez le cours complet [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=qkQr42DmaEE) (4 heures de visionnage).
