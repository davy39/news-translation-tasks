Apprenez à créer une application iOS complète de A à Z en utilisant SwiftUI et Xcode.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer une application de navigation de films et de séries télévisées riche en fonctionnalités, avec un écran d'accueil dynamique, une recherche puissante et des écrans de détails permettant de lire des bandes-annonces YouTube.

Vous mettrez également en œuvre un gestionnaire de téléchargement pour enregistrer des titres pour un visionnage hors ligne à l'aide de SwiftData. En intégrant les API de The Movie Database et de YouTube, ce projet offre une excellente pratique concrète de SwiftUI, de l'intégration d'API et du stockage de données moderne. Carlos Valentin a créé ce cours.

Voici les sections de ce cours :

* Essentiels
    
* Réseautage (Networking)
    
* Requêtes réseau API
    
* Navigation
    
* Recherche
    
* SwiftData
