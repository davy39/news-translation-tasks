Kivy est un framework Python gratuit et open source pour le développement d'applications mobiles et de jeux avec une interface utilisateur naturelle.

Nous venons de publier un cours Kivy de 5 heures sur la chaîne YouTube de freeCodeCamp.org. Dans ce cours Kivy, vous apprendrez à créer des jeux et des applications avec Python pour toutes les plateformes (Windows, Mac, iOS, Android).

Jonathan Roux a développé ce cours. Jonathan a créé de nombreux cours sur diverses plateformes.

Tout d'abord, vous apprendrez à utiliser la bibliothèque Kivy. Vous créerez des interfaces utilisateur graphiques avec des boutons, des étiquettes (labels) et des images. Vous apprendrez à implémenter de nombreux types de layouts pour créer des interfaces capables de s'adapter à toutes les tailles d'écran, de l'ordinateur au mobile.

Ensuite, vous créerez un jeu complet avec Kivy et Python. Le projet GALAXY est un jeu en perspective où vous devez maintenir votre vaisseau spatial sur la piste.
