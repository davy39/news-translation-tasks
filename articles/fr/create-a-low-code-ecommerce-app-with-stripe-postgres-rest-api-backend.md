La création d'applications peut être beaucoup plus rapide avec des outils low-code. Une plateforme low-code peut vous fournir des composants d'interface utilisateur pré-construits dotés de fonctionnalités telles que des tableaux, des boutons et des entrées de texte. Cela vous permet de vous passer des modèles frontend et de construire les vôtres rapidement.

Nous venons de publier un tutoriel sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer une application e-commerce B2B avec Stripe, Postgres et un backend API REST. Et vous ferez tout cela avec peu ou pas de code.

Ania Kubów a développé ce cours. Ania est une développeuse de logiciels et une formatrice populaire. 

Ania vous montrera comment créer un tableau de bord de gestion des commandes, un tableau de bord pour les employés et un portail pour les développeurs. Et vous apprendrez comment implémenter l'intégration d'e-mails dans les applications.

Toutes les applications que vous créerez seront liées à une seule base de données. Ces applications sont similaires à ce que vous verriez dans une petite ou moyenne entreprise.

Ania vous guidera étape par étape. Elle commencera par une explication, puis enseignera chaque application dans trois tutoriels distincts. À la fin du tutoriel, vous saurez comment construire vos propres outils internes.

![Image](https://www.freecodecamp.org/news/content/images/2022/09/image-468.png)

Vous utiliserez la plateforme Retool pour construire les outils métier. Retool est un éditeur no-code en glisser-déposer avec de nombreux composants pré-construits pour créer des applications CRUD (créer, lire, mettre à jour, supprimer) internes aussi rapidement que possible.
