Avez-vous un entretien sur React qui approche ? Ou souhaitez-vous simplement tester vos compétences en React ?

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous aidera à vous préparer pour un entretien sur React en vous enseignant 30 questions et concepts courants d'entretien sur React.

Nishant Singh a développé ce cours. Nishant est ingénieur frontend et créateur de tutoriels.

Voici les sujets que vous allez apprendre dans ce cours :

* Map et Filter
* États dans React
* Props dans React
* Expressions conditionnelles en ligne
* Gestion des événements dans React
* Clés dans React
* Formulaires dans React
* Entrées dynamiques dans React
* Stylisation CSS dans React
* Composants non contrôlés vs contrôlés
* Virtual DOM
* InnerHTML dans React
* Fragments React
* Composants sans état vs avec état
* Requêtes API REST
* Debouncing dans React
* Context API dans React
* Class vs ClassNames dans React
* Composants d'ordre supérieur
* Chargement paresseux dans React
* Fonctions d'assistance dans React
* Implémentation de la récursivité dans React
* Exécution de tableaux de fonctions
* Hooks personnalisés dans React
* Promesses et Async/Await
* Code Splitting dans React (Imports dynamiques)
* Création d'un filtre de recherche dans React
* Ajout de TypeScript à React
* React Testing Library
* Mise en cache d'une réponse API

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/XBTJDpT2XaI) (5 heures de visionnage).
