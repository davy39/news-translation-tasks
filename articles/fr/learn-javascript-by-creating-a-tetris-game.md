Construire des projets concrets est l'une des meilleures façons d'apprendre à coder. Et si vous allez construire un projet, autant en faire l'un des jeux les plus populaires et les plus amusants de tous les temps : Tetris.

<iframe src="//commons.wikimedia.org/wiki/File:Tetris_theme.ogg?embedplayer=yes" width="300" height="20" frameborder="0" webkitAllowFullScreen mozallowfullscreen allowFullScreen></iframe><em>Un peu de musique de lecture. ?</em><br>

Ce cours vidéo d'Ania Kubow vous aidera à apprendre les bases de JavaScript en codant un jeu Tetris à partir de zéro. Ce cours est idéal pour les personnes qui n'ont jamais touché de code auparavant ou pour celles qui souhaitent réviser les fondamentaux.

Le cours ne couvre pas tous les petits détails de JavaScript – mais c'est le but. Il vous aide à vous lancer directement dans la création d'un jeu réel que vous pourrez jouer.

Vous apprendrez d'abord comment configurer un projet à partir de zéro en utilisant l'éditeur de code Atom et un outil d'invite de commande. Ensuite, vous acquerrez une compréhension fondamentale de JavaScript en construisant votre propre version de Tetris. En cours de route, vous obtiendrez également un aperçu de l'utilisation de HTML et CSS.

Tout au long du cours, il y a des sections explicatives qui enseignent les concepts de base de JavaScript tels que les variables, les fonctions, les tableaux, et plus encore. Ces sections explicatives font une pause dans le projet Tetris pour approfondir un concept spécifique. Elles incluent des images et des diagrammes pour vous aider à mieux comprendre les concepts.

De plus, à la fin de certaines sections, il y a des questions à choix multiples pour vous aider à tester vos connaissances.

![Image](https://www.freecodecamp.org/news/content/images/2020/05/tetris.gif)

Voici quelques-unes des méthodes JavaScript que vous apprendrez dans ce cours :

* addEventListener()
* querySelector()
* querySelectorAll()
* keyCode()
* Math.floor()
* Math.random()
* length
* forEach()
* splice()
* clearInterval()
* setInterval()
* some()
* innerHTML()
* every()
* add()
* remove()
* contains()

Une fois que vous aurez terminé le cours, vous aurez créé votre propre jeu Tetris que vous pourrez jouer. Faites simplement attention à l'endroit où vous le jouez, car le record du monde Guinness pour "[La plus longue peine de prison pour avoir joué à un jeu vidéo](https://www.guinnessworldrecords.com/world-records/longest-prison-sentence-for-playing-a-video-game)" impliquait une partie de Tetris.

Vous pouvez regarder le cours ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=rAUn1Lom6dw) (1,5 heure de visionnage).
