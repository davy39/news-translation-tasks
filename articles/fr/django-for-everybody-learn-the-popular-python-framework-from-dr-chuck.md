Django est l'une des façons les plus populaires de créer des sites web full-stack avec Python.

Nous venons de publier un cours Django de 18 heures sur la chaîne YouTube freeCodeCamp.org.

Charles Severance (alias Dr. Chuck) a créé ce cours, ainsi que le populaire cours Python pour Tous. Il est professeur clinicien à l'École d'Information de l'Université du Michigan, où il enseigne divers cours orientés technologie, y compris la programmation, la conception de bases de données et le développement Web.

Ce cours initie les programmeurs Python à la création de sites web en utilisant la bibliothèque Django. Dans le cadre du cours, vous apprendrez HTML, CSS, SQL, Django, JavaScript, jQuery et les services Web JSON.
