Ce cours complet de 134 parties sur JavaScript pour débutants vous enseignera tout ce que vous devez savoir pour commencer avec le langage de programmation JavaScript. La taille de la police dans ce cours est grande, ce qui le rend parfait pour une visualisation sur les petits écrans.

J'ai créé ce cours pour accompagner le programme JavaScript de freeCodeCamp.org. Cependant, le cours fonctionne de manière autonome, donc suivre le programme gratuit sur [freeCodeCamp.org](https://freecodecamp.org) n'est pas obligatoire.

À la fin de ce cours, vous serez prêt à commencer à utiliser JavaScript pour créer vos propres projets.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=PkZNo7MFNFg) (3,5 heures de visionnage).
