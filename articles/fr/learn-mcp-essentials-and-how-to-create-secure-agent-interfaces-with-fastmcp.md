Le MCP est l'ensemble de règles standard qui permet aux agents d'IA, comme GitHub Copilot et Gemini, d'interagir de manière sécurisée et intelligente avec vos bases de données, fonctions et applications.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer des serveurs Model-Context Protocol (ou MCP) en utilisant la bibliothèque Python open source FastMCP. Carlos Leon a créé ce cours.  
  
Vous apprendrez ce qu'est le MCP et pourquoi il est important. Ensuite, vous plongerez dans trois projets de codage pratiques.  
  
Vous construirez des applications de calculatrice en utilisant le Standard I/O (STIO) et le HTTP Stream, intégrerez des API pour du contenu dynamique, et appliquerez des tests avec GitHub Copilot.  
  
Vous terminerez par le déploiement de votre MCP sur FastMCP Cloud pour un flux de travail complet de bout en bout.

Voici les sections du cours :

* À propos du MCP
    
* Scénario de codage 1
    
* Scénario de codage 2
    
* Scénario de codage 3
    
* Tester les MCP avec GitHub Copilot
    
* Bonus : Déployer un MCP sur FastMCP Cloud
