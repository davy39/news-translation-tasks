Si vous voulez vraiment comprendre le développement de jeux, envisagez de coder votre propre moteur de jeu.

Nous venons de publier un cours de 25 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à coder votre propre moteur de jeu 2D en utilisant Java.

Vous utiliserez le moteur de jeu pour créer un clone de Super Mario, mais le moteur peut également être utilisé pour créer d'autres jeux.

![Image](https://www.freecodecamp.org/news/content/images/2022/01/super-mario-video-game.gif)
