Créer un clone de Google Docs vous aidera à apprendre de nombreux nouveaux concepts.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Flutter pour créer un clone de Google Docs.

Rivaan Ranawat a créé ce cours. Rivaan a créé de nombreux cours techniques populaires sur une variété de sujets.

Ce tutoriel est conçu pour les débutants en Node.js et aucune connaissance préalable en Javascript n'est requise. Vous utiliserez Flutter, Node, Express, Socket, MongoDB et Riverpod.

Le clone que vous construisez possède les fonctionnalités suivantes :

* Authentification Google (sans Firebase)
* Persistance de l'état
* Création de nouveaux documents
* Affichage de la liste des documents
* Mise à jour du titre du document
* Partage de lien
* Sauvegarde automatique
* Édition collaborative dans l'éditeur de texte riche
* Déconnexion
