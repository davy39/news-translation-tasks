Avez-vous déjà voulu créer un bot Twitter ? Dans ce tutoriel, John G. Fisher montre comment créer un bot avec Python qui tweete des images ou des mises à jour de statut à intervalles réguliers.

Le script scrape le web pour des images et les publie ensuite sur Twitter. Il peut également être utilisé pour tweeter des phrases et du texte.

Vous pouvez regarder la vidéo sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=8u-zJVVVhT4) (17 minutes de visionnage).
