React est l'un des frameworks front-end les plus populaires pour JavaScript.

Nous venons de publier un autre cours sur React sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser React en construisant une application e-Commerce avec des composants de classe.

Le cours couvre des sujets tels que les bases des composants, le rendu de divers éléments dans les composants, la communication entre composants parent-enfant, les méthodes de cycle de vie, les formulaires, les appels REST-API avec JSON server, et le routage de base.

Voici les sujets abordés dans ce cours :

* Qu'est-ce que React
* Comprendre les composants
* Créer une nouvelle application React
* Créer une application React à partir de zéro
* Charger Bootstrap
* Créer des composants React
* Ajouter des styles CSS
* Ajouter une barre de navigation Bootstrap
* Corriger l'erreur - propriété DOM class
* Composants imbriqués
* Rendre des expressions
* Gérer les événements
* Mettre à jour l'état du composant
* Rendre une liste
* Rendre conditionnellement
* Méthodes de rendu
* Rendre des images
* Rendre des styles CSS
* Rendre des classes CSS
* Passer des arguments d'événement
* Rendre des composants enfants
* Props
* Props vs State
* Props.Children
* Gérer les événements enfants
* Supprimer des composants enfants
* Phases du cycle de vie des composants
* Phase de montage
* Phase de mise à jour
* Phase de démontage
* Cycle de vie des composants enfants
* Phase de gestion des erreurs
* Formulaire de base
* Liaison bidirectionnelle
* Bouton de soumission de formulaire
* Requêtes HTTP
* Async et Await
* Formulaire de connexion avec requête HTTP
* Routage de base
* Page "404"
* Switch
* Link
