Vous cherchez à améliorer vos compétences en développement web frontend avec HTML, CSS, JavaScript, TypeScript et React ? Si c'est le cas, nous avons le cours vidéo parfait pour vous !

Nous venons de publier un cours de développement web frontend basé sur un projet de 10 heures sur la chaîne YouTube freeCodeCamp.org.

Zach Gollwitzer a publié ce cours. Zach est un développeur full stack et son cours Frontend Developer Bootcamp, que nous avons publié il y a quelques mois, a dépassé les 1 000 000 de vues. Ce nouveau cours est une suite à ce cours précédent. Cependant, vous pouvez passer directement à ce cours si vous avez déjà une compréhension de base de HTML, CSS et JavaScript.

Ce cours vidéo est divisé en quatre sections, chaque section s'appuyant sur la précédente. Dans la Section 1, vous apprendrez à créer un jeu de Tic Tac Toe avec du HTML/CSS/JS Vanilla. Cela servira de base pour le reste du cours.

Dans cette première section, vous apprendrez à configurer votre projet et les extensions VSCode nécessaires. Ensuite, vous construirez l'interface utilisateur avec HTML et CSS et ajouterez de l'interactivité JavaScript au projet.

Dans la Section 2, vous refactoriserez le jeu Tic Tac Toe pour utiliser le modèle Model-View-Controller (MVC). Vous explorerez pourquoi ce modèle est utile et comment il peut améliorer la structure de votre code.

La Section 3 vous présentera TypeScript et vous apprendra à le configurer à partir de zéro. Vous découvrirez les avantages de l'utilisation de TypeScript et comment il peut améliorer la fiabilité et la maintenabilité de votre code.

Enfin, dans la Section 4, vous refactoriserez le jeu Tic Tac Toe pour utiliser React et TypeScript. Vous apprendrez à configurer une application React à partir de zéro et explorerez pourquoi React est utile et quels problèmes il résout par rapport à une approche Vanilla. Vous apprendrez également à initialiser TypeScript dans une application React à partir de zéro et à refactoriser votre application Vanilla en une application React.

Ce cours vidéo est parfait pour toute personne cherchant à améliorer ses compétences en développement web frontend, des débutants aux développeurs intermédiaires.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=MsnQ5uepIaE) (10 heures de visionnage).
