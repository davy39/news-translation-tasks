Découvrez l'environnement d'exécution JavaScript Node.js dans ce cours complet de Samer Buna. Node.js permet l'exécution de code JavaScript en dehors d'un navigateur.

À la fin de ce cours, vous comprendrez les fondamentaux de Node.js et devriez être à l'aise pour écrire du code pour Node.js.

Voici les sujets abordés dans ce cours :

* Obtenir Node
* Qu'est-ce que Node ? Qu'est-ce que V8 ? Qu'est-ce qu'un REPL ?
* L'interface en ligne de commande (CLI) de Node et l'environnement de processus
* Exécuter des fichiers avec Node
* Encapsulation des modules et exports/require
* Mise en cache des modules et APIs
* Modules ECMAScript dans Node
* Utilisation des packages Node/npm
* Publication de packages
* Nettoyage et modules globaux
* Scripts package.json, npx et dépendances prod/dev
* Création d'un serveur web avec http intégré
* Création d'un serveur web avec express
* Require vs import vs import()
* Surveillance des fichiers pour les changements en développement
* Callbacks et promesses
* Événements et flux
* Clusters et PM2
* Autres bibliothèques principales dans Node
* Débogage des applications Node
* Révision du JavaScript moderne

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=gG3pytAY2MY) (3 heures de visionnage).
