Recréer une application populaire est une manière amusante d'apprendre de nombreuses compétences complexes en programmation.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire un clone d'Instagram en utilisant React Native, Firebase Firestore, Firebase Storage, Redux et Expo. L'application inclut un système d'authentification, une base de données, un stockage et bien plus encore.

SimCoder a créé ce cours. Il a enseigné des dizaines de cours de logiciels excellents sur sa chaîne YouTube populaire.

Créer l'application avec React Native vous permet d'utiliser la même base de code pour les appareils Android et iOS.

À la fin de ce cours, vous saurez comment créer une application de A à Z et vous pourrez utiliser ces compétences dans vos propres projets.

Voici les sections de ce cours :

* Introduction
* Installation de React Native Expo
* Système d'authentification avec Firebase et React Native
* Sauvegarder des données avec Firebase et Redux
* Gestion de la navigation sur React Native
* Utilisation de la caméra et de la galerie d'images avec React Native
* Sauvegarder une publication d'image avec Firebase Firestore et Firebase Storage
* Affichage du profil utilisateur avec Firebase Firestore
* Recherche d'utilisateurs par nom similaire avec Firebase Firestore
* Système de suivi avec Firestore et React Native
* Système de flux de publications avec Firebase et React Native
* Déconnexion de Firebase Auth et Redux React Native
* Ajout de commentaires utilisateurs aux publications avec Firebase Firestore
* Système de likes avec Firestore

**Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/1hPgQWbWmEk) (6 heures de visionnage).**
