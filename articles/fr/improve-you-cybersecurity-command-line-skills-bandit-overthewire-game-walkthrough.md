Les CTFs et les Wargames peuvent aider les gens à améliorer leurs compétences en ligne de commande et en hacking. 

Le wargame Bandit d'OverTheWire est destiné aux débutants absolus. Il s'agit d'un jeu auquel vous vous connectez via SSH et qui vous aidera à améliorer vos compétences en ligne de commande, vos compétences Linux et vos compétences en hacking.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous fournira un guide complet des 34 niveaux faisant partie du wargame Bandit.

Sabyasachi Paul a créé ce tutoriel. Il possède une grande expérience dans le domaine de la cybersécurité. Sabyasachi vous guidera à travers une série de défis de sécurité impliquant le shell Linux, les connexions à distance et SSH.

Avant de regarder la solution de chaque défi, essayez d'abord de résoudre chaque défi par vous-même.

Dans ce jeu en ligne de commande, vous commencez au Niveau 0 et essayez de le "battre" ou de le "terminer". Terminer un niveau donne des informations sur la façon de commencer le niveau suivant. 

Si vous suivez le jeu par vous-même, vous pourriez rencontrer de nombreuses situations où vous n'avez aucune idée de ce que vous êtes censé faire. Il y a plusieurs choses que vous pouvez essayer lorsque vous ne savez pas comment continuer dans le jeu (avant de regarder la solution).

Premièrement, si vous connaissez une commande mais ne savez pas comment l'utiliser, essayez le manuel (page man) en entrant `man <command>`. Par exemple, `man ls` pour en savoir plus sur la commande "ls". La commande "man" a également un manuel, essayez-la ! Lorsque vous utilisez man, appuyez sur q pour quitter (vous pouvez également utiliser / et n et N pour rechercher).

Deuxièmement, s'il n'y a pas de page man, la commande pourrait être une commande intégrée au shell. Dans ce cas, utilisez la commande `help <X>`. Par exemple, `help cd`.

Vous pouvez trouver les instructions pour le jeu ici : [https://overthewire.org/wargames/bandit/](https://overthewire.org/wargames/bandit/)
