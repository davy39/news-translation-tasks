Vous avez un fichier Photoshop (.psd) que vous devez ouvrir mais vous n'avez pas Photoshop ? Dans ce guide, vous apprendrez plusieurs façons d'ouvrir un fichier Photoshop gratuitement sans payer pour Photoshop.

### Prévisualisation sur Google Drive

Vous pouvez utiliser cette méthode peu importe le type de système d'exploitation que vous avez. Il suffit de télécharger votre fichier PSD sur Google Drive. Vous pourrez prévisualiser le fichier.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/image-179.png)

![Image](https://www.freecodecamp.org/news/content/images/2019/09/image-180.png)

Une fois que vous prévisualisez le fichier, vous pouvez prendre une capture d'écran et ouvrir le fichier de capture d'écran dans un autre programme.

Voici quelques raccourcis pour prendre des captures d'écran sur divers systèmes d'exploitation :

* Windows : Win+Shift+S
* Mac : Command+Shift+4
* ChromeOS : Ctrl + Shift + []]] (touche de changement de fenêtre)

### IrfanView (Windows uniquement)

InfranView est un programme gratuit que vous pouvez télécharger sur Windows. Il vous permettra d'afficher un fichier PSD Photoshop, ainsi que de l'enregistrer dans plusieurs autres formats populaires. Cependant, il ne supporte pas les calques dans les fichiers Photoshop, donc tout sera fusionné en une seule couche.

[Téléchargez IrfanView ici.](https://www.irfanview.com/main_download_engl.htm)

### Aperçu (Mac uniquement)

Le programme intégré Aperçu sur MacOS peut ouvrir les fichiers Photoshop. Il suffit de faire un clic droit sur le fichier, de choisir "Ouvrir avec", puis "Aperçu". Une fois le fichier ouvert dans Aperçu, vous pouvez l'exporter dans d'autres formats en choisissant "Fichier", puis "Exporter...". Le menu Exporter dispose d'un menu déroulant où vous pouvez sélectionner le format. Tout comme IrfanView, les informations sur les calques ne sont pas supportées.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/image-182.png)

### GIMP

GIMP est une alternative gratuite et open source à Photoshop. Il peut ouvrir des fichiers PSD et même préserver les informations sur les calques. GIMP est l'option la plus puissante de cette liste et vous permettra d'apporter d'autres modifications au fichier.

Pour ouvrir un fichier Photoshop dans GIMP, commencez par [télécharger et installer GIMP](https://www.gimp.org/downloads/).

Ensuite, ouvrez le fichier Photoshop dans GIMP. Utilisez simplement les paramètres par défaut pour convertir le fichier.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/image-190.png)

Vous avez maintenant accès à l'image dans GIMP, y compris toutes les couches ! Vous pouvez modifier l'image ou l'exporter dans n'importe quel format. GIMP peut ne pas préserver toutes les informations de texte ou les styles de calque.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/image-186.png)

Maintenant, vous savez que vous n'avez pas besoin de Photoshop pour ouvrir des fichiers PSD !
