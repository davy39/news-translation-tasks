C# est un langage de programmation polyvalent et orienté objet. Ce cours complet de Mike Dane vous donnera une introduction complète à tous les concepts de base de C#.

Les sujets suivants sont abordés :

* [Installation et configuration](#heading-installation)
* Dessiner une forme
* Variables
* Types de données
* Travailler avec des chaînes de caractères
* Travailler avec des nombres
* Obtenir l'entrée de l'utilisateur
* Construire une calculatrice
* Construire un Mad Lib
* Tableaux
* Méthodes
* Instruction Return
* Instructions If
* Instructions If (suite)
* Construire une meilleure calculatrice
* Instructions Switch
* Boucles While
* Construire un jeu de devinettes
* Boucles For
* Construire une méthode d'exposant
* Tableaux 2D
* Commentaires
* Gestion des exceptions
* Classes et objets
* Constructeurs
* Méthodes d'objet
* Getters et Setters
* Attributs de classe statique
* Méthodes et classes statiques
* Héritage

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=GhQdlIFylQ8) (4,5 heures de visionnage).
