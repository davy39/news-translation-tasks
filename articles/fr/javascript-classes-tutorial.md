En JavaScript, vous pouvez utiliser la fonctionnalité OOP (Programmation Orientée Objet) des "classes" pour construire des objets qui sont ensuite des ressources utiles pour votre script/application.

Apprenez tout sur les classes en JavaScript dans cette vidéo. Vous apprendrez :

* Les bases des classes
* Les getters et setters
* Les méthodes statiques
* L'héritage et l'extension
* Le polymorphisme
* Les classes en pratique

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=2ZphE5HcQPQ) (1 heure de visionnage).
