Un nom de domaine est l'adresse d'un site web. C'est le nom que vous tapez dans un navigateur web pour accéder à ce site spécifique. Par exemple, le nom de domaine de freeCodeCamp est freecodecamp.org.

Les noms de domaine sont importants car ils permettent aux gens de mémoriser facilement l'adresse de votre site web. 

Imaginez ce qui se passerait si vous deviez mémoriser l'adresse IP de chaque site web que vous souhaitez visiter. Ce serait un vrai casse-tête pour naviguer sur le web.

## Quels caractères peuvent figurer dans un nom de domaine ?

Un nom de domaine peut être une combinaison de lettres de l'alphabet anglais (A-Z) et de chiffres (0-9). 

Vous pouvez également utiliser le caractère tiret. Par exemple, le domaine de Class Central était à l'origine class-central.com, mais ils ont pu acheter classcentral.com sans le tiret, ce qui semble plus officiel et est plus facile à taper.

Vous pouvez utiliser ces noms en combinaison avec diverses extensions de noms de domaine, telles que .com, .net et .org. Ces extensions sont également appelées domaines de premier niveau ou TLD (Top Level Domains).

## Qu'est-ce qu'un TLD ? Signification et exemples de domaine de premier niveau

TLD signifie Top Level Domain. Lorsque le système de noms de domaine (DNS) est apparu pour la première fois dans les années 1980, il n'existait que quelques TLD : **.com**, **.org** et **.net**.

Aux États-Unis, il existait également trois TLD très restreints et qui le sont toujours :

* Les domaines .edu sont réservés aux universités accréditées
* .gov est réservé aux organisations gouvernementales américaines
* .mil est réservé à l'armée américaine

De nombreux pays ont leurs propres TLD. Par exemple, le Royaume-Uni a .uk et .co.uk. Vous avez peut-être déjà vu le TLD .io – il est réservé au territoire britannique de l'océan Indien, mais est populaire auprès des entreprises technologiques car io est similaire à I/O, ou Input Output.

D'autres TLD populaires sont .co (qui est similaire à .com, mais pour les startups).

Notez que .org est l'un des TLD originaux et est conçu pour les organisations à but non lucratif. Mais en pratique, n'importe qui peut enregistrer un domaine .org. Par exemple, Craigslist.org est une entreprise à but lucratif.

Soyez donc prudent : le fait qu'une entreprise utilise un domaine .org ne signifie pas qu'elle est une organisation à but non lucratif exonérée d'impôts. (Au fait, si vous vous posez la question, freeCodeCamp est une organisation à but non lucratif exonérée d'impôts.)

## Comment enregistrer un nom de domaine ?

Lorsque vous enregistrez un nom de domaine, vous réservez essentiellement ce nom pour votre propre usage. Personne d'autre ne peut utiliser ce nom de domaine tant que vous continuez à le renouveler. 

Et vous devrez le renouveler chaque année auprès d'un service d'enregistrement de domaines. Les frais annuels pour cela peuvent varier de 12 $ pour un domaine .com à plusieurs centaines de dollars pour certains noms de domaine spécialisés.

Les noms de domaine sont généralement enregistrés auprès de registrars de noms de domaine, tels que Google Domains, Name.com ou Namecheap. Le processus est généralement assez simple et direct.

Une fois que vous avez enregistré un nom de domaine, vous devrez le pointer vers votre fournisseur d'hébergement web ou vos propres serveurs. Cela indiquera au nom de domaine où trouver votre site web.
