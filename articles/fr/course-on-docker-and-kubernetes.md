Docker et Kubernetes simplifient l'exécution de code et de logiciels dans différents environnements. 

Les conteneurs sont un moyen d'emballer des logiciels. Exécuter un conteneur revient à exécuter une machine virtuelle. Vous pouvez utiliser Docker pour construire et déployer des conteneurs et Kubernetes pour les gérer.

Nous avons publié un cours pratique de 4 heures sur la chaîne YouTube freeCodeCamp.org qui vous enseignera les bases de Docker et Kubernetes.

Ce cours a été développé par Nelson Jamal d'Amigoscode et Nana Janashia de Techworld with Nana. Ces deux instructeurs ont une expérience considérable dans l'enseignement sur YouTube et d'autres plateformes.

Dans ce cours, vous apprendrez d'abord tout sur Docker, y compris les images, les conteneurs, les volumes, le registre de conteneurs et l'architecture Docker. Ensuite, vous apprendrez Kubernetes, y compris l'architecture, les pods, les services, les config maps et les déploiements.

Vous apprendrez les différences entre Docker et Kubernetes, ce qui vous permettra de commencer à utiliser ces deux technologies en toute confiance.
