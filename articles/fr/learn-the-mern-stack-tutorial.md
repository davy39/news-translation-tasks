La pile MERN est une pile de technologies populaire pour construire une application moderne à page unique. Dans ce cours vidéo que j'ai développé, vous apprendrez la pile MERN en construisant une application de suivi d'exercices full stack.

Vous apprendrez les bases de toutes les technologies de la pile MERN, y compris :

* MongoDB : Une base de données open source basée sur des documents.
* Express : Un framework d'application web pour Node.js.
* React : Une bibliothèque JavaScript front-end pour construire des interfaces utilisateur.
* Node.js : Un environnement d'exécution JavaScript qui exécute du code JavaScript en dehors d'un navigateur (comme un serveur).

Vous apprendrez également comment héberger la base de données dans le cloud en utilisant MongoDB Atlas.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=7CqJlxBYj-M) (2 heures de visionnage).
