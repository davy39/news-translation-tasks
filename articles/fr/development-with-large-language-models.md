Les grands modèles de langage (LLMs) ouvrent un large éventail de capacités pour les développeurs. Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous équipera des compétences nécessaires pour utiliser efficacement les LLMs pour vos projets.

Développé par Akshath de Luup.ai, ce cours vous aidera à libérer le potentiel des LLMs. Vous créerez des projets qui utilisent les LLMs pour :

* Développer des interfaces dynamiques capables d'interagir avec les utilisateurs en temps réel.
* Naviguer facilement à travers de grandes quantités de données textuelles.
* Permettre aux LLMs de rechercher sur Internet des articles de recherche et des informations pertinentes.

Avec un accent sur l'application pratique, ce cours vous guidera à travers une série de projets concrets qui montrent le potentiel incroyable des LLMs. Voici les sujets abordés :

* Brève introduction aux LLMs
* Playground ChatGPT
* Bases de l'API OpenAI
* Brève introduction à chainlit
* Clonage de l'interface utilisateur de ChatGPT
* Limites de notre interface
* Ajout de streaming, vue backend, bouton de séquence d'arrêt
* Introduction aux bases de données vectorielles
* Bases de données vectorielles pratiques
* QnA avec des documents - .txt et .pdf
* Test de notre système de Q&A
* Introduction à la navigation web et aux agents
* Chercheur IA
* L'humain comme outil
* Mini plugin d'interpréteur de code (outil Replit)
* Recherche sur YouTube à l'aide d'agents
* Guide pour explorer davantage
* Outil Shell
* Créez vos propres outils
* Notes de fin

Regardez le cours complet sur la chaîne YouTube freeCodeCamp.org (2 heures de visionnage).
