Les composants React ont plusieurs méthodes de cycle de vie que vous pouvez remplacer pour exécuter votre code à un moment particulier du processus.

Dans cette vidéo, Nick Karnik démontre toutes les méthodes du cycle de vie des composants React. Vous apprendrez quand utiliser ces méthodes dans une application React.

Vous pouvez regarder la vidéo sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=m_mtV4YaI8c) (26 minutes de visionnage).
