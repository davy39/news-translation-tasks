La certification GitHub Foundations a été lancée il y a quelques semaines seulement. Et vous pouvez maintenant regarder un cours de préparation à l'examen de 9 heures sur la chaîne YouTube de freeCodeCamp.org !

Andrew Brown d'ExamPro a créé ce cours. Il a créé une série de cours de préparation aux examens populaires pour la chaîne YouTube de freeCodeCamp.org.

Cette certification est une excellente opportunité pour les professionnels de montrer leur maîtrise des concepts fondamentaux de GitHub, des techniques de collaboration et de l'éventail des fonctionnalités offertes par cette plateforme.

Le cours est structuré pour offrir une compréhension approfondie de chaque domaine couvert par la certification. Voici un aperçu de ce que vous trouverez dans ce cours complet :

#### Introduction

La partie introductive de notre cours offre une base solide sur l'écosystème de GitHub, préparant le terrain pour une exploration plus approfondie de ses fonctionnalités et de ses utilisations. Elle commence par familiariser les apprenants avec les aspects fondamentaux de GitHub, offrant des informations sur la structure et les sujets essentiels de la certification GitHub Foundations. De plus, elle met en lumière l'initiative unique de GitHub pour reconnaître et honorer les contributeurs remarquables au sein de sa communauté.

#### Bases de GitHub

Couvrant tout, des **systèmes de contrôle de version** aux différences entre **Git et GitHub**, cette section est une plongée en profondeur dans les essentiels. Apprenez à configurer l'authentification multifacteur, à gérer plusieurs comptes et à comprendre les intricacies des dépôts.

#### Développement GitHub

Maîtrisez le **GitHub CLI** et les API, explorez les SDK et sécurisez vos connexions avec des clés SSH. Ce module est pratique avec des sessions de suivi pour un apprentissage pratique.

#### GitHub Desktop et Mobile

Comprenez les nuances de **GitHub Desktop** et **Mobile**, y compris la gestion des notifications et les comparaisons de fonctionnalités.

#### Entités GitHub

Découvrez les différents types de comptes, comparez les versions gratuites et pro, et apprenez-en davantage sur les plans organisationnels et les options de déploiement en entreprise.

#### Markdown

De la syntaxe de base à la syntaxe étendue, cette section vous aide à maîtriser le **Markdown**, une compétence cruciale pour une documentation efficace sur GitHub.

#### Profil utilisateur GitHub

Un guide détaillé pour configurer et optimiser votre profil utilisateur GitHub.

#### Comprendre les dépôts GitHub

Un module complet sur la gestion des dépôts, couvrant tout, des fichiers README aux insights, y compris plusieurs sessions de suivi.

#### Problèmes et demandes de tirage GitHub

Plongez dans le monde des problèmes et des demandes de tirage, en comprenant leur création, leur gestion et les meilleures pratiques.

#### Discussions, notifications, gists, wiki, pages GitHub

Ces sections offrent des connaissances complètes sur les fonctionnalités de collaboration, de communication et de partage de contenu de GitHub.

#### Actions, Copilot, Codespaces, GitHub.dev

Axées sur les outils avancés de GitHub comme Actions, Copilot et Codespaces, ces modules sont conçus pour ceux qui cherchent à exploiter tout le potentiel de GitHub.

#### Open Source

Apprenez-en davantage sur les contributions open source, les sponsors GitHub et la distinction entre InnerSource et OpenSource.

#### Étiquettes, jalons, projets

Maîtrisez l'art d'organiser votre travail sur GitHub avec des étiquettes, des jalons et des outils de gestion de projet.

#### Authentification et sécurité

Un module critique couvrant la sécurité des comptes, les permissions d'accès et les rôles organisationnels.

#### Administrateur et entreprise GitHub

Conçu pour les administrateurs, cette section couvre les permissions de dépôt, la protection des branches et les fonctionnalités de sécurité avancées de GitHub.

#### Réservez votre examen

Dernières étapes pour réserver votre examen de certification, en vous assurant d'être pleinement préparé.

Ce cours ne vous préparera pas seulement à la certification, mais améliorera également votre compréhension globale et vos compétences dans l'utilisation efficace de GitHub. Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=Jdc0i7RcBv8) (9 heures de visionnage).
