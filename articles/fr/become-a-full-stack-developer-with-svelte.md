Le développement web full stack inclut le backend qui s'exécute sur un serveur et le frontend qui s'exécute sur un navigateur web. De nouveaux outils rendent plus facile que jamais la création d'une application web full stack.

Nous venons de publier un cours de développement web full stack sur la chaîne YouTube freeCodeCamp.org. Et le meilleur : tout peut être fait dans un navigateur en utilisant Gitpod.

Mike Nikles a développé ce cours. Mike est un développeur senior chez Gitpod et un excellent enseignant.

Ce cours couvre tout, de la création d'un nouveau dépôt à la mise en production, et vous apprendrez la raison de chaque ligne de code. Le framework JavaScript Svelte est utilisé pour le frontend avec Postgres comme base de données. L'application est déployée en utilisant Vercel et Railway.

Voici toutes les sections couvertes dans ce cours :

* L'environnement de développement Gitpod
* Aperçu du projet
* Architecture du projet
* Initialiser l'application
* Introduction à SvelteKit
* Configurer Gitpod
* Développer la structure HTML
* Appliquer les styles CSS
* Développer les endpoints CRUD : GET et POST
* Développer les endpoints CRUD : DELETE
* Développer les endpoints CRUD : PATCH - Éditer le texte
* Développer les endpoints CRUD : PATCH - Marquer comme terminé
* Améliorer les formulaires HTML
* Installer et démarrer la base de données de développement
* Utiliser Prisma ORM
* Mettre à jour l'API pour utiliser Prisma
* Déployer la base de données sur Railway
* Déployer l'application web et l'API sur Vercel
* Revue du projet
