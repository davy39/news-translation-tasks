Vous cherchez un autre projet d'application web à construire ? Êtes-vous intéressé par l'incorporation de l'accessibilité dans vos sites web ? Voulez-vous en savoir plus sur la création d'applications mobile-first ?

Si vous avez répondu "oui" à l'une de ces questions, vous allez aimer ce nouveau tutoriel que nous venons de publier sur la chaîne YouTube freeCodeCamp.org.

Dans ce tutoriel, vous allez construire une application de liste de tâches avec HTML, Sass et JavaScript vanilla. Votre application sera responsive, avec une approche mobile-first.

L'application sera également construite en gardant à l'esprit l'accessibilité (A11y) pour la navigation au clavier et les lecteurs d'écran. Vous utiliserez également l'API Web Storage pour créer un stockage de données persistant afin que les éléments de la liste réapparaissent à chaque visite de l'application web.

Ce tutoriel a été créé par Dave Gray. Dave est un instructeur universitaire qui enseigne le développement web. Il possède également une chaîne YouTube avec de nombreux tutoriels excellents.

Voici les sujets que Dave aborde dans ce tutoriel :

* Fondamentaux HTML
* Styles SASS / CSS
* Classes JavaScript
* Fonctions JavaScript
* Test des fonctionnalités d'accessibilité
* Nettoyage et améliorations du code pour l'A11y
* Test des corrections et améliorations pour l'application
* Amélioration de l'application

Regardez le tutoriel complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=y51Cv4wnsPw) (2 heures de visionnage).
