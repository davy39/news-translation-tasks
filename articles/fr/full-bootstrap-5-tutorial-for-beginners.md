Bootstrap est le framework CSS le plus populaire. Il permet aux développeurs web de concevoir et de personnaliser rapidement des sites mobiles-first et responsives.

Nous venons de publier un cours de 3 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Bootstrap 5.

Dawid Adach a développé ce cours. Dawid travaille avec Bootstrap depuis des années et a aidé à créer l'une des bibliothèques Bootstrap les plus populaires appelée Material Design for Bootstrap.

Voici toutes les sections couvertes dans ce cours :

* Qu'est-ce que Bootstrap ?
* Pourquoi utiliser Bootstrap ?
* Télécharger le CSS et le JS compilés
* Installation via CDN
* Installation NPM
* Système de grille
* Composants
* Boutons
* Cartes
* Typographie
* Images responsives
* Utilitaires
* Tableaux
* Alertes
* Navs et Navbars
* Icônes
* Formulaires
* Autres composants
* Créer un projet en utilisant Bootstrap 5

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=-qfEOE4vtxE) (3 heures de visionnage).
