En matière de CSS, il existe de nombreuses propriétés qui contrôlent l'espacement des éléments sur une page. Deux des propriétés les plus courantes sont margin et padding.

Margin et padding contrôlent tous deux la quantité d'espace autour d'un élément, mais ils le font de différentes manières.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Artboard-1.jpeg)
_Cette image provient de l'excellent tutoriel gratuit de Kevin Powell sur [les marges CSS](https://www.freecodecamp.org/news/css-margins/)._

Margin est l'espace à l'extérieur d'un élément. C'est l'espace entre un élément et les éléments qui l'entourent.

Si vous souhaitez qu'un élément soit plus éloigné des autres éléments, augmentez la propriété margin.

Padding est l'espace à l'intérieur d'un élément. C'est l'espace entre un élément et son contenu.

### Exemple de Margin en CSS

Voici comment donner une marge à un élément en CSS :

```css
.element {

  margin: 10px;

}

```

### Exemple de Margin en HTML

Et voici comment styliser l'élément pour qu'il ait une marge en utilisant le style en ligne en HTML. (Gardez à l'esprit que cela est considéré comme une mauvaise pratique dans la plupart des cas.)

```html
<div style="margin-top:100px;">Ceci est un texte.</div>
```

### Exemple de Padding en CSS

Voici comment donner un padding à un élément en CSS :

```css
.element {

  padding: 10px;

}

```

Et bien sûr, vous pouvez également le faire avec un style en ligne en HTML :

```html
<div style="padding-top:20px;">Ceci est un texte.</div>
```
