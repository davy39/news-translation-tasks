Git est essentiel pour tout développeur.

Nous venons de publier un cours qui vous aidera à apprendre Git et GitHub de zéro avec des exemples clairs, des workflows réels, le branching, le merging, le stashing, le rebase, les pull requests, et plus encore. Ce cours est idéal pour les débutants qui souhaitent des bases solides. Sumit Saha a développé ce cours.

Voici ce que vous apprendrez dans cette vidéo :

* La différence entre Git et GitHub de la manière la plus simple
    
* Comment Git suit votre travail en utilisant le working directory, la staging area et le repository
    
* Commandes Git fondamentales : add, commit, status, log, reset, restore, rm
    
* Comment travailler avec les branches, fusionner les mises à jour et résoudre les conflits de merge
    
* Comment push, pull et synchroniser votre code avec GitHub
    
* Utiliser le stash, le revert et le rebase pour gérer des workflows réels
    
* Comment fonctionnent les pull requests et pourquoi les équipes les utilisent
    

Regardez le cours complet sur la chaîne YouTube de freeCodeCamp.org (durée : 1 heure).
