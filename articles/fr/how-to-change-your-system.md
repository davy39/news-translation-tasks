MacOS permet de changer facilement le nom de votre Mac dans AirDrop.

Suivez simplement les 4 étapes suivantes et tout ira bien.

## Étape 1 : Ouvrir les Préférences Système

La manière la plus rapide d'ouvrir vos paramètres est d'appuyer sur ⌘ + espace, puis de taper "pref" et de choisir "Préférences Système" dans le menu déroulant.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/Spotlight_and_How_to_Change_Your_Airdrop_Name_-_Rename_your_Mac_in_30_seconds_-_freeCodeCamp_org.png)

## Étape 2 : Cliquer sur l'icône Partage

C'est le dossier avec le losange jaune. Je l'ai indiqué ci-dessous pour votre commodité.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/System_Preferences_and_How_to_change_your_Mac_computer_s_name.png)

## Étape 3 : Modifier le nom de l'ordinateur

Vous verrez le nom de votre ordinateur en haut, et vous pourrez cliquer dans le champ de texte pour le mettre à jour.

Une fois que vous l'aurez changé pour le nom souhaité, il vous suffira de cliquer sur Modifier.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/Sharing-2.png)

## Étape 4 : Confirmer le nouveau nom de l'ordinateur

Maintenant, vous pouvez simplement cliquer sur le bouton "OK". Vos paramètres seront enregistrés.

![Image](https://www.freecodecamp.org/news/content/images/2019/09/Sharing-1.png)

## Le nom de votre AirDrop est maintenant mis à jour.

Vous pouvez fermer les Préférences Système.

Maintenant, lorsque les gens vous verront sur AirDrop, ils verront votre nouveau nom AirDrop.
