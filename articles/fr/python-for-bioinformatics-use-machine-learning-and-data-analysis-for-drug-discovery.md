Vous cherchez un moyen d'appliquer Python et le machine learning à une application concrète ? Vous devriez envisager la Bioinformatique.

La bioinformatique est un domaine interdisciplinaire qui développe des méthodes et des outils logiciels pour comprendre les données biologiques, en particulier lorsque les ensembles de données sont volumineux et complexes.

Nous venons de publier un cours qui vous apprendra à utiliser Python et le machine learning pour construire un projet de bioinformatique dédié à la découverte de médicaments.

Chanin Nantasenamat (alias Data Professor) a développé ce cours. Il est professeur associé en bioinformatique et sait comment simplifier les concepts pour les débutants.

Vous n'avez pas besoin de connaître quoi que ce soit en bioinformatique pour suivre ce cours.

Le cours couvre la collecte de jeux de données, le prétraitement des données et la réalisation d'analyses exploratoires. Vous apprendrez à construire des modèles de machine learning afin de faire des prédictions et d'obtenir des insights basés sur les données, utiles pour la découverte de médicaments. Ensuite, vous apprendrez à comparer les modèles et à sélectionner celui qui convient le mieux au cas d'usage.

Voici les sections abordées dans ce cours :

* Collecte de données
* Analyse exploratoire des données
* Calcul de descripteurs
* Construction de modèles
* Comparaison de modèles
* Déploiement de modèles
