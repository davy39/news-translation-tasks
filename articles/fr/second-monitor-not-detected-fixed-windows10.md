Parfois, votre PC ne détecte pas votre deuxième écran. Dans ce tutoriel, vous apprendrez quelques étapes pour résoudre ce problème.

## Vérifier les connexions incorrectes

Avant de faire quoi que ce soit de technique, assurez-vous de vérifier vos connexions. Assurez-vous que tous les câbles sont bien branchés et connectés dans le bon ordre.

## Mettre à jour l'adaptateur d'affichage

C'est probablement la meilleure solution pour faire reconnaître votre deuxième écran par votre PC. Voici les étapes à suivre :

### **Étape 1 - Ouvrir le Gestionnaire de périphériques**

* Cliquez sur la barre de recherche dans votre barre des tâches.
* Tapez "gestionnaire de périphériques".
* Cliquez sur Gestionnaire de périphériques dans les résultats de recherche pour lancer l'application.

### **Étape 2 - Mettre à jour l'adaptateur d'affichage**

* Développez l'option Adaptateurs d'affichage dans le Gestionnaire de périphériques.
* Faites un clic droit sur votre adaptateur d'affichage et cliquez sur "Mettre à jour le pilote"

![Image](https://www.freecodecamp.org/news/content/images/2022/01/update-display-adapter.png)

* On vous demandera : "Comment voulez-vous rechercher les pilotes ?" Choisissez l'option qui dit "Rechercher automatiquement les pilotes".

![Image](https://www.freecodecamp.org/news/content/images/2022/01/how-do-you-want-to-search-for-drivers.png)

Attendez que le pilote soit mis à jour, après quoi vous pouvez aller dans vos paramètres d'affichage et voir si le deuxième écran est détecté.

Si vous obtenez une réponse indiquant que votre pilote est déjà à jour, vous devrez alors faire la mise à jour manuellement. Vous pouvez le faire en suivant les étapes ci-dessous :

* Faites un clic droit sur l'adaptateur d'affichage comme nous l'avons fait auparavant – mais cette fois, vous devez cliquer sur **propriétés**.
* Cliquez sur l'onglet **Pilote**.
* Copiez les valeurs **Fournisseur du pilote** et **Version du pilote**.
* Recherchez les deux valeurs sur Google.

Après avoir fait cela, vous devriez trouver le site web de votre fournisseur de pilotes où vous pourrez installer la dernière version du pilote.

Une fois que vous avez installé le pilote, vérifiez si votre PC reconnaît votre deuxième écran.

## Conclusion

Dans cet article, nous avons vu quelques-unes des étapes nécessaires pour mettre à jour votre adaptateur d'affichage afin de rendre votre deuxième écran visible. J'espère que cela vous aide.
