Dans ce cours complet de John Smilga, vous apprendrez React en construisant un projet de site web pour une station balnéaire. Le projet utilise React Router pour le routage, l'API de contexte React pour la gestion d'état, le CMS sans tête Contentful pour la gestion des données, et Netlify pour héberger l'application.

Vous construirez un site web responsive qui affichera des chambres d'hôtel et permettra aux utilisateurs de filtrer les chambres. Les données proviennent directement du CMS sans tête Contentful. Ce cours complet de 6 heures couvrira tout ce que vous devez savoir en détail.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=LXJOvkVYQqA) (6 heures de visionnage).
