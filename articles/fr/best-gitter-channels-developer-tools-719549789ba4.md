Par Gitter

Les outils pour développeurs sont devenus essentiels à tout type de développement logiciel sérieux, y compris dans le cadre de l'open source. Ils peuvent atténuer les difficultés quotidiennes des développeurs — aider à la maintenabilité, rendre le travail plus rapide et plus amusant. Découvrez ces canaux Gitter dédiés aux outils pour développeurs et impliquez-vous !

* [**nodejs/node**](https://gitter.im/nodejs/node?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Node.js est un environnement d'exécution JavaScript basé sur le moteur V8 JavaScript de Chrome. Ceci est un chat général sur l'environnement d'exécution JavaScript Node.js
* [**cmderdev/cmder**](https://gitter.im/cmderdev/cmder?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Un bel émulateur de console pour Windows.
* [**git-for-windows/git**](https://gitter.im/git-for-windows/git?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Une version de Git contenant des correctifs spécifiques à Windows. [http://git-scm.com/](http://git-scm.com/)
* [**jsfiddle/jsfiddle**](https://gitter.im/jsfiddle/jsfiddle?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Éditeur de code en ligne et espace de jeu. Hub front-end.
* [**sindresorhus/ava**](https://gitter.im/sindresorhus/ava?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Exécuteur de tests futuriste.
* [**princejwesley/Mancy**](https://gitter.im/princejwesley/Mancy?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** REPL NodeJS basé sur Electron.
* [**bucaran/fly**](https://gitter.im/bucaran/fly?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** *Fly* est un système de build moderne pour [Node](https://nodejs.org/) basé sur les [_co_-routines](https://medium.com/@tjholowaychuk/callbacks-vs-coroutines-174f1fe66127), les [générateurs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/function*) et les [promesses](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise).
* [**webdriverio/webdriverio**](https://gitter.im/webdriverio/webdriverio?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Cette bibliothèque est un module [webdriver](https://w3c.github.io/webdriver/webdriver-spec.html) (automatisation de navigateur) pour Node.JS. Elle permet d'écrire des tests [Selenium](https://en.wikipedia.org/wiki/Selenium_(software)) très facilement dans votre framework de test BDD/TDD préféré, qui s'exécuteront localement ou dans le cloud en utilisant Sauce Labs, BrowserStack ou TestingBot.
* [**x64dbg/x64dbg**](https://gitter.im/x64dbg/x64dbg?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Un débogueur x64/x32 open-source pour Windows.
* [**neovim/neovim**](https://gitter.im/neovim/neovim?utm_source=blog&utm_medium=content&utm_campaign=devtools) **—** Un fork de Vim axé sur l'extensibilité et l'agilité.

Vous cherchez autre chose ? Consultez notre toute nouvelle section [Explore](https://gitter.im/explore/tags/javascript,php,ruby) ou [créez facilement votre propre canal ici](https://gitter.im/home#createroom).

Avons-nous oublié un canal que vous pensez devrait être mis en avant ? Faites-nous signe dans le [Gitter HQ](https://gitter.im/gitterHQ/gitter) et nous l'ajouterons à la liste.

Bon codage !
