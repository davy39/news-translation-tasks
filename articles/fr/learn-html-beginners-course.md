Le HTML est la colonne vertébrale de chaque site web sur Internet. Si vous souhaitez vous lancer dans le développement web, apprendre le HTML est l'endroit idéal pour commencer.

Nous venons de publier un cours vidéo HTML pour débutants sur la chaîne YouTube de freeCodeCamp.org. (Et nous proposons toujours notre [cours interactif gratuit sur le HTML](https://www.freecodecamp.org/learn/responsive-web-design/).)

Dave Gray a créé ce cours. Dave est instructeur universitaire, créateur de cours et excellent enseignant.

Ce cours complet sur le HTML5 est un excellent point de départ pour votre voyage vers l'apprentissage du développement web. Considérez ce cours comme un manuel vidéo avec 10 chapitres clairement définis.

![Image](https://www.freecodecamp.org/news/content/images/2022/03/html-intro.gif)
_Le HTML crée la structure d'un site web._

Voici les sections couvertes dans ce cours :

* Introduction
* Élément Head
* Bases du texte
* Types de listes
* Ajouter des liens
* Ajouter des images
* Balises sémantiques
* Créer des tableaux
* Formulaires et entrées
* Projet HTML

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/kUMe1FH4CHE) (4 heures de visionnage).
