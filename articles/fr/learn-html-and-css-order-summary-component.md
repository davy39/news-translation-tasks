Le HTML et le CSS sont les technologies les plus importantes lorsqu'il s'agit de créer des sites web. Et construire des projets est l'une des meilleures façons de les apprendre.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous aidera à apprendre le HTML et le CSS en construisant un composant de résumé de commande.

Madison Kanna a créé ce cours. Elle est une ingénieure logicielle expérimentée et sait comment décomposer les choses pour enseigner aux débutants.

Dans ce cours, Madison vous montrera comment résoudre un défi de Frontend Mentor. Le défi consiste à construire un composant de résumé de commande et à le faire ressembler le plus possible au design.

Voici les sections couvertes dans ce cours :

* Création du projet

* Hello world

* HTML, CSS et JS
