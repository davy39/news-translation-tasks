Une grande partie de votre expérience en tant que développeur dépendra du programme que vous avez choisi pour écrire votre code. Un bon environnement de développement intégré (IDE) ou un éditeur de code peut vraiment augmenter votre productivité.

Le problème avec les langages populaires comme Python est que chaque IDE ou éditeur de code semble offrir un bon support pour le langage. Bien que cela puisse être génial, choisir le meilleur devient un peu délicat.

Dans cet article, je vais vous présenter trois IDE et éditeurs de code qui peuvent rendre votre parcours Python plus fluide.

Mais avant de commencer, je tiens à clarifier le fait que cette liste n'est pas exhaustive. Comme je l'ai dit, Python est l'un des langages de programmation les plus populaires, donc il est supporté par un grand nombre d'éditeurs de code et d'IDE.

J'aurais pu en inclure autant que possible, mais j'ai plutôt choisi d'inclure ceux que j'ai utilisés à un moment donné dans ma vie et que je n'hésiterais pas à réutiliser. Parce que je pense que cela sera plus utile.

Sans plus tarder, plongeons-nous dans le sujet !

## IDE vs Éditeur de Code – Quelle est la Différence ?

Avant de commencer à lire sur les IDE et les éditeurs de code que j'ai en réserve pour vous, clarifions les définitions d'un IDE et d'un éditeur de code.

Comme vous le savez peut-être déjà, les fichiers de code source sont simplement des fichiers texte avec certaines extensions qui leur sont ajoutées. Tout éditeur de texte qui possède certaines fonctionnalités spéciales telles que la coloration syntaxique, l'indentation automatique du code, et ainsi de suite pour faciliter l'édition des fichiers de code est appelé un éditeur de code.

Les éditeurs de code populaires incluent Visual Studio Code, Sublime Text, Atom, Notepad++ et bien d'autres.

Un IDE ou Environnement de Développement Intégré, en revanche, est une suite de logiciels beaucoup plus complexe qui combine plusieurs outils tels qu'un éditeur de code, un navigateur de fichiers, un émulateur de terminal, un explorateur de base de données et bien plus encore dans un seul package.

Les IDE populaires incluent PyCharm, IntelliJ Idea, Microsoft Visual Studio, et bien d'autres.

Mais grâce aux éditeurs de code modernes et hautement extensibles comme Microsoft Visual Studio Code, la ligne entre un IDE et un éditeur de code a commencé à s'estomper.

Maintenant que vous avez une meilleure idée de ce qu'est un éditeur par rapport à un IDE complet, examinons certains des meilleurs pour le codage Python.

## PyCharm

Le premier de notre liste est un IDE de JetBrains. [PyCharm](https://www.jetbrains.com/pycharm/) est l'un des IDE Python les plus utilisés (sinon le plus utilisé).

![Image](https://www.freecodecamp.org/news/content/images/2022/04/complexLook@2x.jpg)
_https://www.jetbrains.com/pycharm/_

L'IDE existe en deux éditions. L'édition professionnelle vous coûtera 8,90 $ par mois et 89,00 $ par an si facturé annuellement. Il existe également l'édition communautaire qui est complètement gratuite et basée sur des logiciels open-source. Dans cet article, je vais discuter de l'édition communautaire.

Les deux éditions sont disponibles pour Windows, macOS et Linux. Vous pouvez télécharger l'essai de 30 jours de l'édition professionnelle ou l'édition communautaire depuis la page officielle de [téléchargement](https://www.jetbrains.com/pycharm/download/).

Le processus d'installation est assez simple, quelle que soit la plateforme que vous utilisez. Une fois que vous avez téléchargé et installé PyCharm sur votre ordinateur, vous devriez pouvoir démarrer l'IDE. Vous pouvez utiliser le raccourci du menu démarrer sur Windows, le répertoire Applications sur macOS ou votre lanceur d'applications sur Linux.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-17.png)

Vous pouvez créer un nouveau projet Python en cliquant sur le bouton _Nouveau Projet_.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-18.png)

À l'étape suivante, choisissez où vous souhaitez stocker votre projet. Vous pouvez soit créer un nouvel environnement virtuel, soit utiliser un interpréteur précédemment configuré. Dans ce cas, je crée un nouvel environnement.

Si vous cochez l'option _Créer un script main.py de bienvenue_, un nouveau fichier Python avec un peu de code standard sera créé dans votre projet. Une fois que vous êtes satisfait de tous les choix, cliquez sur le bouton _Créer_.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-19.png)

Voici à quoi ressemble l'éditeur de code une fois le projet créé. Sur le côté gauche, vous pouvez parcourir tous vos fichiers sources, et vous pouvez cliquer sur le bouton de lecture en haut à droite de la fenêtre pour exécuter les scripts sélectionnés dans la liste déroulante.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/run-code-pycharm.png)

Comme vous pouvez le voir, PyCharm dispose d'un terminal intégré en bas de la fenêtre et vous pouvez voir les sorties de votre programme sans jamais quitter PyCharm.

L'édition communautaire est assez complète et vous pouvez faire plus ou moins tout ce que vous pouvez faire sur l'édition professionnelle. L'édition professionnelle offre un meilleur support pour les frameworks web comme Django et quelques autres fonctionnalités supplémentaires.

Si vous êtes étudiant, vous pouvez obtenir PyCharm Professional Edition et tous les autres produits JetBrains gratuitement [en faisant une demande sur leur site web](https://www.jetbrains.com/community/education/#students). Vous pouvez également obtenir une licence gratuite pour tous les produits JetBrains si vous êtes [un mainteneur open-source](https://www.jetbrains.com/community/opensource/).

## Microsoft Visual Studio Code

Le suivant dans la liste de mes favoris est Microsoft Visual Studio Code ou VSCode pour faire court. C'est un éditeur de code open-source, alimenté par Electron, multiplateforme de Microsoft avec une tonne d'options de personnalisation et d'extensions.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-21.png)

En installant le bon ensemble d'extensions, vous pouvez transformer Visual Studio Code en un IDE Python presque complet. Vous pouvez télécharger VSCode gratuitement depuis le [site officiel de téléchargement](https://code.visualstudio.com/#alt-downloads) pour Windows, macOS et Linux.

Une fois que vous avez installé VSCode sur votre système, ouvrez le logiciel et allez dans l'onglet des extensions en appuyant sur la combinaison de touches _Ctrl + Shift + X_.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-22.png)

Utilisez la barre de recherche pour rechercher et installer les extensions suivantes :

* [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python) – fournit des fonctionnalités telles que IntelliSense (Pylance), linting, débogage, navigation de code, formatage de code, refactoring, explorateur de variables, explorateur de tests, et plus encore !
* [PyLance](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance) – fonctionne aux côtés de Python dans Visual Studio Code pour fournir un support de langage performant.
* [Visual Studio IntelliCode](https://marketplace.visualstudio.com/items?itemName=VisualStudioExptTeam.vscodeintellicode) – fournit des fonctionnalités de développement assisté par IA pour les développeurs Python, TypeScript/JavaScript et Java dans Visual Studio Code.

Une fois que vous avez installé ces trois extensions, vous êtes prêt à partir. Créez un répertoire n'importe où sur votre machine et ouvrez ce dossier en utilisant VSCode. Vous pouvez utiliser le terminal intégré pour exécuter votre code ou exécuter n'importe quelle commande en général.

Vous pouvez définir des points d'arrêt en cliquant sur le côté gauche de n'importe quel numéro de ligne. Ensuite, vous pouvez appuyer sur _F5_ pour commencer le débogage ou _Ctrl + F5_ pour exécuter le programme sans débogage. VSCode a beaucoup d'autres astuces que vous découvrirez au fur et à mesure que vous continuerez à l'utiliser.

## Sublime Text

Sublime Text est l'un des éditeurs de code OG qui est utilisé par les développeurs depuis des années. C'est un éditeur de code très performant et puissant avec un riche support pour les packages.

Vous pouvez télécharger Sublime Text depuis leur [page officielle de téléchargement](https://www.sublimetext.com/download) pour Windows, macOS et Linux. Une fois que vous l'avez installé, démarrez Sublime Text comme n'importe quel autre logiciel.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-23.png)

Maintenant, cliquez sur _Tools > Install Package Control..._

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-24.png)

Cela installera le gestionnaire de packages Sublime. Attendez qu'un message de succès apparaisse.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-25.png)

Maintenant, allez dans la Palette de Commandes en utilisant la combinaison de touches _Ctrl + Shift + P_ et tapez _Install Package_ :

![Image](https://www.freecodecamp.org/news/content/images/2022/04/image-26.png)

Sélectionnez la première option et recherchez le package [Anaconda](https://packagecontrol.io/packages/Anaconda). C'est le package Python ultime qui transforme Sublime Text en un IDE Python avec des fonctionnalités comme l'autocomplétion, le linting de code, les fonctionnalités d'IDE, le formatage autopep8, le vérificateur de complexité McCabe, le support Vagrant et Docker, et bien plus encore.

Il existe également des packages plus spécifiques tels que [Djaneiro](https://packagecontrol.io/packages/Djaneiro) pour le support Django et [requirementstxt](https://packagecontrol.io/packages/requirementstxt) pour le support des fichiers requirements.txt sur Sublime Text. Il suffit de regarder autour du site [Package Control](https://packagecontrol.io/browse) et vous pourriez trouver quelques packages très utiles.
