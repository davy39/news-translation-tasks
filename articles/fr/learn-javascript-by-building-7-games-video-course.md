JavaScript est l'un des langages de programmation les plus populaires. Nous avons publié un cours vidéo complet où vous pouvez apprendre JavaScript en construisant 7 jeux rétro basés sur une grille. 

Lorsque vous apprenez en construisant des projets, vous pouvez immédiatement voir comment le code fonctionne dans une situation réelle. Et à la fin de ce cours d'Ania Kubow, vous aurez 7 excellents projets pour votre portfolio de carrière. Chaque jeu enseigné dans ce cours est créé avec un style minimal pour que vous puissiez les prendre et les personnaliser.

Vous apprendrez les bases de JavaScript en créant les sept jeux suivants :

* Jeu de mémoire
* Whack-a-mole
* Puissance 4
* Snake
* Space Invaders
* Frogger
* Tetris

Voici quelques-unes des méthodes et fonctions JavaScript que vous apprendrez dans ce cours :

* querySelector()
* addEventListener()
* Array.from()
* getElementsByClassName()
* Math.floor()
* Math.random()
* forEach()
* classList.contains()
* classList.add()
* classList.remove()
* setInterval()
* clearInterval()
* some()
* style.backgroundImage
* splice()
* concat()
* appendChild()
* Fonctions fléchées

Regardez le cours ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=lhNdUVh3qCc) (2 heures de visionnage).
