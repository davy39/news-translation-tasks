Lorsque vous travaillez sur votre ordinateur, vous voulez faire les choses aussi rapidement que possible. Et vous ne voulez probablement pas répéter la saisie des mêmes choses encore et encore. 

Savoir comment copier et coller du texte et d'autres données vous aidera à travailler efficacement sans vous répéter. 

Dans ce guide, je vais vous montrer les différentes façons de copier et coller du texte, des fichiers et des dossiers sur votre PC Windows, y compris depuis la ligne de commande. 

Ensuite, je vous montrerai également comment accéder au presse-papiers de votre ordinateur afin que vous puissiez voir l'historique de vos éléments copiés.

## Comment copier et coller du texte sur un PC Windows

Vous pouvez copier du texte en le surlignant dans n'importe quel éditeur, en cliquant avec le bouton droit sur le texte surligné, puis en sélectionnant « Copier ». 
![ss-1-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-1-2.jpg)

Pour surligner plusieurs mots, maintenez le bouton gauche de votre souris enfoncé, puis faites défiler vers le bas jusqu'au point où vous souhaitez arrêter de copier le texte. Cliquez avec le bouton droit et sélectionnez « Copier » pour copier le texte que vous avez surligné.

Pour surligner tout le texte dans un document, appuyez sur `CTRL` + `A`.

Si vous travaillez avec beaucoup de texte dans MS Word, par exemple, vous pouvez double-cliquer sur un mot pour le sélectionner, triple-cliquer sur n'importe quel mot dans une ligne pour sélectionner tout le paragraphe, cliquer avec le bouton droit, puis sélectionner « Copier ».

Pour coller le texte copié, cliquez à nouveau avec le bouton droit sur n'importe quel éditeur et sélectionnez « Coller ». Assurez-vous de cliquer avec le bouton droit exactement là où vous voulez que le texte collé soit.
![ss-2-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-2-2.jpg)
![ss-3-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-3-1.png)

Le raccourci que vous pouvez utiliser pour copier du texte dans n'importe quel éditeur est `CTRL` + `C`. Pour coller le texte, appuyez et maintenez `CTRL` puis `V`.

## Comment copier et coller des fichiers et des dossiers

Pour copier un fichier sur votre PC Windows, cliquez avec le bouton droit sur le fichier et sélectionnez « Copier » dans la liste des éléments qui apparaît :
![ss-4-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-4-1.jpg)

Pour coller le fichier ou le dossier copié dans un autre dossier, cliquez avec le bouton droit sur un espace vide dans le dossier et sélectionnez « Coller » :
![ss-5](https://www.freecodecamp.org/news/content/images/2021/10/ss-5.jpg)

Vous pouvez également copier des fichiers et des dossiers en masse. Pour ce faire, maintenez la touche `SHIFT` enfoncée et utilisez les touches fléchées haut et bas pour vous déplacer vers le haut et vers le bas dans vos fichiers. Ensuite, cliquez avec le bouton droit et sélectionnez « Copier ».

Si vous ne voulez pas que certains fichiers ou dossiers soient sélectionnés, maintenez la touche `CTRL` au lieu de `SHIFT` et utilisez votre souris pour sélectionner chaque fichier et dossier que vous voulez. Ensuite, cliquez simplement avec le bouton droit et sélectionnez « Copier ».

Pour sélectionner tous les fichiers ou dossiers dans un dossier, ou sélectionner tous les fichiers et dossiers dans un dossier ensemble, appuyez sur `CTRL` + `A`.

## Comment copier et coller du texte dans la ligne de commande

Dans la ligne de commande, `CTRL` + `C` et `CTRL` + `V` signifient différentes choses, mais vous pouvez toujours copier et coller directement là.

Ouvrez la ligne de commande en tapant « cmd » dans la barre de recherche Windows. Pour copier un texte, surlignez le texte avec votre souris ou votre pavé tactile, puis cliquez avec le bouton droit sur le texte surligné. 

Pour coller le texte dans la ligne de commande, cliquez à nouveau avec le bouton droit.

Dans la ligne de commande, vous pouvez toujours utiliser `CTRL` + `C` et `CTRL` + `V` pour copier et coller. Pour ce faire, pointez votre curseur en haut de la fenêtre et cliquez avec le bouton droit pour sélectionner les propriétés :
![ss-6](https://www.freecodecamp.org/news/content/images/2021/10/ss-6.jpg)

Cochez « Utiliser Ctrl+Maj+C/V comme Copier/Coller » :
![ss-7](https://www.freecodecamp.org/news/content/images/2021/10/ss-7.jpg)

Maintenant, vous pouvez utiliser `CTRL` + `C` et `CTRL` + `V` pour copier et coller dans la ligne de commande.

## Comment accéder à votre presse-papiers Windows

Si vous faites beaucoup de copier-coller, vous ne pouvez coller que le dernier texte que vous copiez. 

Mais que faire si vous voulez coller l'avant-dernier (deuxième depuis le dernier) ou l'antépénultième (troisième depuis le dernier) élément que vous avez copié ? Si vous appuyez sur `CTRL V` ou si vous cliquez avec le bouton droit et sélectionnez « Coller », le dernier élément que vous avez copié est collé.

Pour accéder aux éléments précédents que vous avez copiés sur un ordinateur Windows, appuyez et maintenez simplement la touche `WIN` (Windows) enfoncée, puis `V`. Là, vous verrez une liste des éléments que vous avez copiés depuis que vous avez allumé votre PC.
![ss-8](https://www.freecodecamp.org/news/content/images/2021/10/ss-8.png)

Si vous n'avez jamais accédé au presse-papiers auparavant, vous serez invité à activer l'historique du presse-papiers pour votre machine Windows.

Pour coller un élément depuis le presse-papiers, cliquez dessus avec le bouton gauche.

## Conclusion

Dans ce guide, vous avez appris les différentes façons de copier et coller des textes et des fichiers, y compris comment le faire dans la ligne de commande. 

Vous avez également appris à voir l'historique de votre presse-papiers, afin de pouvoir coller l'un des éléments précédents que vous avez copiés.
