Par Evaristo Caraballo

La communauté freeCodeCamp génère des gigaoctets de données chaque semaine. L'une des parties les plus actives de la communauté est le système de salons de discussion. Des milliers de personnes s'y retrouvent, discutent de technologie et s'entraident pour améliorer leurs compétences en codage.

Je pose souvent des questions sur les données. Cette semaine, je me demandais quels dépôts GitHub les gens trouvaient les plus pertinents pour leurs études. J'ai donc analysé l'historique de discussion de plusieurs gigaoctets du [principal salon de discussion de freeCodeCamp](https://gitter.im/freecodecamp/freecodecamp).

Parmi les milliers de dépôts GitHub mentionnés, voici les 10 les plus fréquemment cités :

#### [freeCodeCamp/freeCodeCamp](https://github.com/freeCodeCamp/freeCodeCamp)

Vous avez probablement vu celui-ci venir. Mais ce n'est pas seulement à cause de l'origine des données. La plateforme d'apprentissage de la communauté freeCodeCamp elle-même est assez populaire, avec près de 250k étoiles, plus de 10k forks, et des centaines de nouvelles issues et pull requests chaque semaine.

Ainsi, les gens en parlent beaucoup dans le salon de discussion.

#### [getify/You-Dont-Know-JS](https://github.com/getify/You-Dont-Know-JS)

You Don't Know JavaScript de Kyle Simpson est sans aucun doute le livre de référence non officiel pour la communauté freeCodeCamp. [Kyle](https://www.freecodecamp.org/news/the-10-github-repos-people-mention-the-most-in-freecodecamps-main-chat-room-189750600fa4/undefined) travaille également actuellement sur un autre projet similaire à YDKJS, **"Functional Light JS"**, qui est de plus en plus mentionné.

#### [vhf/free-programming-books](https://github.com/vhf/free-programming-books)

Une liste fréquemment mise à jour de ressources gratuites. Ce dépôt est une compilation bien organisée de matériel collecté dans de nombreuses parties du monde. Il comprend des livres, des podcasts, des sites web, des outils pour développeurs... vous l'appelez. Définitivement un "must" pour les personnes qui apprennent à coder.

#### [twbs/bootstrap](https://github.com/twbs/bootstrap)

Le compte GitHub de Bootstrap, le célèbre framework de conception web responsive. Le dépôt fournit principalement des informations techniques, mais dispose également de liens supplémentaires vers d'autres sites utiles. Si vous cherchez une documentation plus détaillée sur le framework, vous devriez également visiter [getbootstrap.com](https://getbootstrap.com/).

#### [jwasham/coding-interview-university](https://github.com/jwasham/coding-interview-university)

[Googley as Heck](https://www.freecodecamp.org/news/the-10-github-repos-people-mention-the-most-in-freecodecamps-main-chat-room-189750600fa4/undefined) a créé ce dépôt pour documenter son [processus de 8 mois de préparation pour l'entretien chez Google](https://medium.freecodecamp.com/why-i-studied-full-time-for-8-months-for-a-google-interview-cc662ce9bb13). Il comprend une liste détaillée des choses que vous devez savoir pour passer les redoutables tests de tableau blanc de Google, ainsi que des liens vers des références qui expliquent clairement divers concepts d'informatique. (Au fait, après toute cette préparation, il [a pris un emploi chez Amazon à la place](https://medium.freecodecamp.com/ive-been-hired-by-amazon-8b21f7c27de5#.6e1kc7fes).)

#### [ericelliott/essential-javascript-links](https://github.com/ericelliott/essential-javascript-links)

La liste de liens JavaScript d'[Eric Elliott](https://www.freecodecamp.org/news/the-10-github-repos-people-mention-the-most-in-freecodecamps-main-chat-room-189750600fa4/undefined) a été pendant longtemps la liste de ressources la plus populaire au sein de la communauté freeCodeCamp. (Elle a récemment été détrônée par [vhf/free-programming-books](https://github.com/vhf/free-programming-books).)

#### [d3/d3](https://github.com/d3/d3/wiki)

Si vous voulez apprendre d3.js, ce dépôt GitHub officiel est l'un des meilleurs endroits pour commencer. Les deux parties de ce dépôt que la communauté freeCodeCamp mentionne le plus se trouvent en fait dans son wiki :

* [la galerie](https://github.com/d3/d3/wiki/Gallery)
* [les tutoriels](https://github.com/d3/d3/wiki/Tutorials)

Ces sections contiennent d'excellentes informations, ainsi qu'une liste de références que vous pouvez utiliser pour apprendre d3.js par vous-même.

#### [vinta/awesome-python](https://github.com/vinta/awesome-python)

Présenté comme une "liste organisée de frameworks, bibliothèques, logiciels et ressources Python géniaux", c'est un excellent point de départ si vous voulez vous améliorer en Python.

#### [oneuijs/You-Dont-Need-jQuery](https://github.com/oneuijs/You-Dont-Need-jQuery)

Il s'agit d'un dépôt bien connu qui se concentre sur la résolution de problèmes de programmation typiques en utilisant du JavaScript vanilla. L'essor de ce dépôt a été étroitement lié à celui de React.

#### [toddmotto/public-apis](https://github.com/toddmotto/public-apis)

Une belle collection d'API publiques, fréquemment mise à jour. Les API sont même classées par catégories, ce qui facilite grandement leur exploration.

Au cas où vous vous poseriez la question, toutes les données ont été obtenues à partir du [principal salon de discussion Gitter de freeCodeCamp](https://gitter.im/FreeCodeCamp/FreeCodeCamp) entre juin 2016 et mars 2017. J'ai exécuté un script Python pour évaluer les données, puis j'ai classé les liens GitHub observés en fonction du nombre de mentions ainsi que de la récente des mentions. La sélection finale a été basée sur la pertinence.

Encore une fois, cette liste ne fait qu'effleurer la surface. La communauté de freeCodeCamp référence des centaines de dépôts chaque jour.

Merci d'avoir lu, et bon codage !
