Pour améliorer vos compétences en machine learning et en intelligence artificielle, il est important de résoudre des problèmes réels. Quel meilleur problème à résoudre que celui d'aider à sauver des vies ?

Le machine learning est de plus en plus utilisé dans le domaine de la santé. PyTorch et Monai peuvent être utilisés pour détecter des tumeurs dans le foie.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser PyTorch, Monai et Python pour la segmentation 3D du foie. Vous utiliserez le machine learning et la vision par ordinateur pour trouver des tumeurs dans le foie.

Mohammed El Amine MOKHTARI a développé ce cours. Il est doctorant en vision par ordinateur et créateur de contenu en ligne.

Voici les sections de ce cours :

* Qu'est-ce que U-Net
* Installation du logiciel
* Trouver les jeux de données
* Préparation des données
* Installation des packages
* Prétraitement
* Erreurs que vous pourriez rencontrer
* Dice Loss
* Entropie croisée pondérée
* La partie entraînement
* La partie test
* Utilisation du dépôt GitHub

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/M3ZWfamWrBM) (5 heures de visionnage).
