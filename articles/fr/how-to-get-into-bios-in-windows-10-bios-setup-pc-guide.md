BIOS signifie Basic Input/Output System. Il est intégré aux ordinateurs sous forme de puce sur la carte mère et est principalement responsable du démarrage du système informatique.

En plus de démarrer votre ordinateur, le BIOS est responsable de nombreuses autres fonctionnalités en arrière-plan – comme la gestion du flux de données entre un système d'exploitation et des périphériques matériels connectés tels que les imprimantes, la souris, le clavier, le disque dur et les scanners.

Une fois que tout a été certifié OK lors du démarrage, votre ordinateur démarrera avec succès. Sinon, un message d'erreur tel que le BSOD (écran bleu de la mort) ou plusieurs autres sera affiché.

Si vous pouvez accéder au BIOS sur votre PC, vous pouvez faire diverses choses telles que changer les paramètres du disque dur, créer un mot de passe pour le BIOS afin de rendre votre ordinateur plus sécurisé, changer l'ordre de démarrage, et plus encore.

Donc, dans cet article, je vais vous montrer deux façons d'accéder au BIOS dans Windows 10.

## Comment accéder au BIOS dans Windows 10 via les paramètres

**Étape 1** : Cliquez sur Démarrer et sélectionnez Paramètres pour ouvrir l'application Paramètres. Ou appuyez simplement sur `WIN` (touche Windows) + `I`.
![ss-1-3](https://www.freecodecamp.org/news/content/images/2021/10/ss-1-3.jpg)

**Étape 2** : Cliquez sur « Mise à jour et sécurité ».
![ss-2-4](https://www.freecodecamp.org/news/content/images/2021/10/ss-2-4.jpg)

**Étape 3** : Sélectionnez « Récupération ».
![ss-3-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-3-2.jpg)

**Étape 4** : Sous « Démarrage avancé », cliquez sur « Redémarrer maintenant ». Cela vous permettra de redémarrer votre ordinateur pour la restauration et certaines configurations.
![ss-4-3](https://www.freecodecamp.org/news/content/images/2021/10/ss-4-3.jpg)

**Étape 5** : Après le redémarrage de votre PC, vous verrez un menu avec un fond bleu contenant des options comme « Continuer », « Dépannage » et « Éteindre votre PC ». Cliquez sur Dépannage.

**Étape 6** : Cliquez sur « Options avancées »

**Étape 7** : Sélectionnez « Paramètres du micrologiciel UEFI » pour enfin entrer dans le BIOS de votre ordinateur.

Et voilà - vous devriez être dedans.

Si vous ne trouvez pas les paramètres du micrologiciel UEFI, l'utilisation de touches de raccourci pour entrer dans le BIOS viendra à la rescousse. Voyons comment les utiliser maintenant.

## Comment accéder au BIOS dans Windows 10 via les touches de raccourci

Certains fabricants d'ordinateurs donnent aux utilisateurs plusieurs touches ou combinaisons de touches à appuyer pendant le démarrage pour qu'ils puissent entrer dans le BIOS. Il s'agit généralement de la touche F2 sur de nombreux appareils, mais il existe quelques exceptions.

Le tableau ci-dessous montre les touches de raccourci pour entrer dans le BIOS sur différentes marques d'ordinateurs.

HP : ESC (Échap) pour entrer dans le menu de démarrage, puis F10 ou F12.

| Marque | Touche(s) |
| ----------- | ----------- |
| ACER      | `F2` ou `DEL` (Supprimer) |
| Toshiba | `F2` |
| Lenovo Thinkpad | `ENTRÉE` + `F1` |
| Lenovo Desktop | `F1` |
| Dell | `F2` |
| MSI | `DEL` |
| Samsung | `F2` |
| Sony | `F2` |
| Origin PC | `F2` |

## Conclusion

Dans ce guide, vous avez appris comment accéder au BIOS dans Windows 10, afin d'avoir plus de contrôle sur votre appareil. Cela vous permet également de plonger dans plus de paramètres que vous ne trouverez peut-être pas dans l'application Paramètres.

Les paramètres de Windows 10 offrent une meilleure façon d'entrer dans le BIOS sur les ordinateurs modernes, mais si vous utilisez des appareils plus anciens, l'utilisation de touches de raccourci pourrait être votre meilleure option pour accéder au BIOS.

Si vous vous demandez ce qu'est l'UEFI, cela signifie Unified Extensible Firmware Interface. Il fonctionne comme le BIOS mais offre un temps de démarrage plus rapide et peut supporter des tailles de disque jusqu'à 9 zettaoctets, tandis que le BIOS ne supporte que jusqu'à 2,2 téraoctets.

Faites attention à ce que vous faites dans le BIOS car tout ce que vous modifiez pourrait avoir un effet durable sur votre ordinateur.
