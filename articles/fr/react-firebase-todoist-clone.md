Dans ce cours intermédiaire sur React de Karl Hadwen, vous apprendrez à créer l'application populaire Todoist à partir de zéro en utilisant React, des Hooks personnalisés, Firebase et la React Testing Library. Vous apprendrez à utiliser SCSS pour styliser l'application et à appliquer la méthodologie de nommage BEM.

Cette application est entièrement responsive. Vous verrez comment utiliser Lighthouse (extension Chrome) pour vous assurer que votre site web est entièrement accessible en intégrant toutes les fonctionnalités d'accessibilité dans notre application.

Tout d'abord, vous verrez comment initialiser l'application en utilisant create-react-app comme base. Ensuite, vous apprendrez à développer l'application en utilisant React (Hooks personnalisés, Context), Firebase et la React Testing Library (tests unitaires et d'intégration). Vous apprendrez même à ajouter un mode sombre !

Vous pouvez regarder le [cours vidéo complet](https://www.youtube.com/watch?v=hT3j87FMR6M) sur la chaîne YouTube de freeCodeCamp.org (8 heures de visionnage).
