Par Ayo Isaiah

Choisir le bon ordinateur portable pour la programmation peut être un processus difficile.

Il est facile de se sentir confus en recherchant les différentes options. Il existe de nombreux modèles d'ordinateurs portables, chacun avec un ensemble différent de compromis.

Vous pouvez écrire du code sur la plupart des ordinateurs portables. Pourtant, votre productivité s'améliorera si vous utilisez une machine adaptée au type de tâches que vous effectuez.

Il existe différents types de développement, et divers outils sont requis pour chaque spécialisation. Il n'y a donc pas d'approche universelle pour acheter une machine de développement.

J'ai fait les hypothèses suivantes dans cet article :

* Vous êtes un développeur web
* Votre ordinateur portable est votre machine de développement principale

Voici quelques considérations avant d'acheter votre prochain ordinateur portable.

### Mobilité

Les ordinateurs portables existent en toutes formes et tailles. Vous devez déterminer à quel point vous voulez que votre ordinateur portable soit portable.

Si vous n'avez pas besoin de transporter votre ordinateur portable souvent, vous pourriez envisager un ordinateur portable de 15 pouces. Ceux-ci auront généralement de meilleures spécifications et plus d'espace à l'écran pour le multitâche.

Si vous travaillez dans différents endroits ou voyagez beaucoup, un ordinateur portable de 13 ou 14 pouces peut être le meilleur pour vous. Ils sont plus légers et offrent une autonomie de batterie plus longue.

Sauf si vous achetez un ordinateur portable 2-en-1, un écran tactile ne fournit pas assez d'avantages pour justifier le coût supplémentaire. Je vous suggère d'éviter l'écran tactile.

![Image](https://cdn-media-1.freecodecamp.org/images/8HXfTyW1Mmvq9vxU3rlJp2XCVlbP1Kc7Wu1F)

### Affichage

L'écran d'un ordinateur portable est l'une de ses caractéristiques les plus importantes, surtout pour les programmeurs. Le développement d'applications implique de fixer l'écran pendant de longues périodes. Vous devez prêter une attention particulière aux détails.

La plupart des ordinateurs portables économiques sont livrés avec un écran de 1366 x 768, que je considère comme médiocre au mieux. L'écran n'a pas assez d'espace pour le multitâche. De plus, le texte n'est pas assez net pour que vous ayez une expérience de lecture confortable.

Un écran 4k est excessif pour un ordinateur portable, surtout lorsque vous considérez les coûts supplémentaires et la consommation de batterie que vous rencontrerez.

Quoi que vous fassiez, n'achetez pas un ordinateur portable avec moins d'un écran Full HD 1920 x 1080 (1080p). Si vous devez payer un peu plus pour obtenir du 1080p, faites-le.

Assurez-vous également que l'écran a de bons angles de vision ; l'écran de votre ordinateur portable ne devrait pas servir de miroir !

### Puissance de traitement (CPU)

Le CPU de votre ordinateur portable a une énorme influence sur les performances, vous ne pouvez donc pas vous permettre de lésiner sur celui-ci. Il existe de nombreux types de processeurs avec différentes spécifications. Assurez-vous de considérer ces spécifications. Certaines des plus importantes sont la taille du cache, le nombre de cœurs, la fréquence et la puissance thermique.

En général, un bon processeur Intel Core i5 ou i7 avec une fréquence de 3 GHz ou plus devrait suffire pour la plupart des gens.

### Mémoire (RAM)

Je ne pense pas qu'un travail de programmation sérieux puisse être fait sur un ordinateur portable avec moins de 4 Go de RAM. Ma recommandation minimale de RAM est de 8 Go. Même cela devient à peine suffisant avec l'avènement des applications Electron, qui adorent consommer de grandes quantités de RAM. Si vous avez de l'argent supplémentaire, investissez dans 16 Go de RAM.

### Type et capacité de stockage

L'obtention d'un SSD (Solid State Drive) devrait être en haut de vos priorités. Cela vous donnera des améliorations significatives de performance par rapport à un disque dur standard. Chaque opération sera beaucoup plus rapide avec un SSD : y compris le démarrage du système d'exploitation, la compilation de code, le lancement d'applications et le chargement de projets.

Un SSD de 256 Go devrait être la base. Si vous avez plus d'argent, un SSD de 512 Go ou 1 To est mieux. Si le coût est un facteur, optez pour un SSD plus petit, où votre système d'exploitation vivra aux côtés de vos applications et documents fréquemment consultés (comme les fichiers de projet). Le reste de vos affaires, comme la musique ou les vidéos, peut reposer sur un disque dur externe plus grand.

### Clavier

![Image](https://cdn-media-1.freecodecamp.org/images/Css3cm3OqXWd5Ty2SEQ2YYPLrbjQjOivRN8i)

Vous ne pouvez pas vous permettre de compromettre la qualité du clavier de votre ordinateur portable, car c'est ce que vous utiliserez pour taper du code toute la journée. J'ai tendance à opter pour des ordinateurs portables avec une disposition de clavier plus compacte.

La chose la plus importante est d'essayer soigneusement le clavier d'un ordinateur portable avant de l'acheter. Assurez-vous que les touches sont confortables et faciles à atteindre avec un bon déplacement. Un clavier rétroéclairé est utile si vous prévoyez de travailler souvent dans des conditions de faible luminosité.

### Autonomie

Une bonne autonomie de batterie peut ne pas être si importante pour vous si vous passez la plupart de votre temps près d'une prise de courant. Néanmoins, visez au moins 6 heures d'autonomie.

Ne vous fiez pas à l'autonomie de la batterie telle qu'indiquée par le fabricant. Lisez les évaluations tierces de sites fiables et voyez ce que les utilisateurs disent du produit dans les forums et les avis.

### Système d'exploitation

Votre choix de système d'exploitation déterminera en grande partie quel ordinateur portable acheter. Les utilisateurs de Windows ont beaucoup d'options, mais si vous préférez macOS, vous êtes limité à l'une des offres MacBook.

![Image](https://cdn-media-1.freecodecamp.org/images/Hwds0pSgIAumS3UybTfys4fbldWXwyKR9Dgo)

Linux fonctionnera sur la plupart des matériels, mais il est préférable d'acheter des ordinateurs portables qui ont un support Linux officiel. Certains vendeurs, comme Dell et [System 76](https://system76.com/), fournissent des machines de haute qualité avec Linux préinstallé. Vous pourriez vouloir regarder ceux-ci en premier.

Sinon, faites vos recherches pour vous assurer que l'ordinateur portable que vous prévoyez d'acheter fonctionne bien avec vos distributions Linux préférées.

### Graphiques dédiés ou intégrés ?

Une carte graphique dédiée (également connue sous le nom de discrète) n'est pas très importante pour les besoins de codage. Économisez de l'argent en optant pour une carte graphique intégrée. Investissez l'argent que vous économisez dans un SSD ou un meilleur processeur, ce qui vous offrira plus de valeur pour votre argent.

J'aimerais savoir quels facteurs vous considérez comme les plus importants pour une machine de développement et comment cela affecte votre travail au quotidien.
