La meilleure façon d'apprendre un nouveau langage de programmation est de construire des projets avec celui-ci.

J'ai créé une liste de 25 tutoriels de projets adaptés aux débutants en Python.

Mon conseil pour les tutoriels serait de regarder la vidéo, de construire le projet, de le décomposer et de le reconstruire à votre manière. Expérimentez en ajoutant de nouvelles fonctionnalités ou en utilisant différentes méthodes.

Cela testera si vous avez vraiment appris les concepts ou non.

Vous pouvez cliquer sur l'un des projets listés ci-dessous pour accéder directement à cette section de l'article.

Si vous n'êtes pas familier avec les bases de Python, je vous suggère de regarder [ce tutoriel Python pour débutants de freeCodeCamp](https://www.youtube.com/watch?v=rfscVS0vtbw).

## Projet Python Cadavres Exquis

[Dans ce tutoriel de Kylie Ying,](https://www.youtube.com/watch?v=8ext9G7xspg&t=100s) vous apprendrez comment obtenir des entrées de l'utilisateur, travailler avec des f-strings, et voir vos résultats imprimés dans la console.

C'est un excellent projet de départ pour se familiariser avec la concaténation de chaînes en Python.

## Projet Python Jeu Devine le Nombre (ordinateur)

[Dans ce tutoriel de Kylie Ying,](https://www.youtube.com/watch?v=8ext9G7xspg&t=414s) vous apprendrez comment travailler avec le [module random de Python](https://docs.python.org/3/library/random.html), construire des fonctions, travailler avec des boucles while et des conditionnelles, et obtenir des entrées de l'utilisateur.

## Projet Python Jeu Devine le Nombre (utilisateur)

[Dans ce tutoriel de Kylie Ying,](https://www.youtube.com/watch?v=8ext9G7xspg&t=797s) vous construirez un jeu de devinettes où l'ordinateur doit deviner le bon nombre. Vous travaillerez avec le [module random de Python](https://docs.python.org/3/library/random.html), construirez des fonctions, travaillerez avec des boucles while et des conditionnelles, et obtiendrez des entrées de l'utilisateur.

## Projet Python Pierre, papier, ciseaux

[Dans ce tutoriel de Kylie Ying](https://www.youtube.com/watch?v=8ext9G7xspg&t=1274s), vous travaillerez avec `random.choice()`, des instructions if, et l'obtention d'entrées de l'utilisateur. C'est un excellent projet pour vous aider à construire les bases comme les conditionnelles et les fonctions.

## Projet Python Pendu

[Dans ce tutoriel de Kylie Ying,](https://www.youtube.com/watch?v=8ext9G7xspg&t=1465s) vous apprendrez comment travailler avec des dictionnaires, des listes, et des instructions if imbriquées. Vous apprendrez également comment travailler avec les modules string et random de Python.

## Projet Python Compteur à rebours

[Dans ce tutoriel de Code With Tomi](https://www.youtube.com/watch?v=SqvVm3QiQVk&t=1992s), vous apprendrez comment construire un compteur à rebours en utilisant le module `time` de Python. C'est un excellent projet pour débutants pour vous habituer à travailler avec des boucles while en Python.

## Projet Python Générateur de mots de passe

[Dans ce tutoriel de Code With Tomi](https://www.youtube.com/watch?v=SqvVm3QiQVk&t=2531s), vous apprendrez comment construire un générateur de mots de passe aléatoires. Vous collecterez des données de l'utilisateur sur le nombre de mots de passe et leurs longueurs et générerez une collection de mots de passe avec des caractères aléatoires.

Ce projet vous donnera plus de pratique avec les `for loops` et le module `random` de Python.

## Projet Python Encodeur / décodeur de codes QR

[Dans ce tutoriel de Code With Tomi](https://www.youtube.com/watch?v=SqvVm3QiQVk&t=3192s), vous apprendrez comment créer vos propres codes QR et encoder/décoder des informations à partir de ceux-ci. Ce projet utilise la bibliothèque `qrcode`.

C'est un excellent projet pour les débutants pour se familiariser avec l'installation et l'utilisation de différents modules Python.

## Projet Python Morpion

## Projet Python Recherche binaire

## Coder un bot Discord avec Python - Hébergement gratuit dans le cloud

Après la sortie de cette vidéo, Replit a changé la manière dont vous pouvez stocker vos variables d'environnement dans votre programme.
