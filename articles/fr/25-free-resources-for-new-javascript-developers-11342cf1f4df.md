Par freeCodeCamp

Nous avons demandé à nos campeurs de partager leurs ressources gratuites préférées pour les nouveaux développeurs JavaScript sur [Camper News](http://freecodecamp.com/news). La liste comprend quelques livres éprouvés ainsi que des podcasts et des vidéos que vous ne connaissez peut-être pas encore.

#### Livres

![Image](https://cdn-media-1.freecodecamp.org/images/0*rfscP_J7mO4Asdfh.png)

[Eloquent Javascript](http://eloquentjavascript.net//) est une introduction moderne à la programmation et à JavaScript par [Marijn Haverbeke](https://twitter.com/intent/user?screen_name=marijnjh). Il existe également une [version annotée](https://watchandcode.com/courses/eloquent-javascript-the-annotated-version) d'Eloquent JavaScript par [Gordon Zhu](https://twitter.com/intent/user?screen_name=gordon_zhu).

![Image](https://cdn-media-1.freecodecamp.org/images/0*2g9wybpLWgnx02Y0.png)

[JavaScript pour les Chats](http://jsforcats.com/) par [Max Ogden](https://twitter.com/intent/user?screen_name=maxogden) est une introduction pour les nouveaux programmeurs si facile que votre compagnon humain pourrait le faire aussi !

![Image](https://cdn-media-1.freecodecamp.org/images/1*NPMHrp7TulegtvIu6_pVGw.png)

[The JavaScript Way](https://twitter.com/bpesquet) est écrit par le contributeur de freeCodeCamp [Baptiste Pesquet](https://twitter.com/bpesquet). C'est une introduction gratuite et open source à JavaScript.

![Image](https://cdn-media-1.freecodecamp.org/images/1*sNjdsG4jbcDXbd4Gj5DVAg.png)

[You Don't Know JS](https://github.com/getify/You-Dont-Know-JS) est une série de livres de [Kyle Simpson](https://twitter.com/intent/user?screen_name=getify) qui plonge en profondeur dans les mécanismes centraux du langage JavaScript.

![Image](https://cdn-media-1.freecodecamp.org/images/0*8fGZ0evPj1UNMFod.png)

[Human JavaScript](http://read.humanjavascript.com/ch01-introduction.html) par [Henrik Joreteg](https://twitter.com/intent/user?screen_name=HenrikJoreteg) est un livre sur un ensemble spécifique d'outils, de modèles et d'approches optimisés pour les personnes.

![Image](https://cdn-media-1.freecodecamp.org/images/0*zUGh-oio1IWvA7aL.jpg)

[Speaking JavaScript](http://speakingjs.com/es5/) par [Axel Rauschmayer](https://twitter.com/intent/user?screen_name=rauschma) a été écrit pour aider les programmeurs à apprendre JavaScript rapidement et correctement, ainsi que pour approfondir vos compétences existantes et rechercher des sujets spécifiques.

![Image](https://cdn-media-1.freecodecamp.org/images/0*lBfQTN0_XNyCafrB.jpeg)

[ExploringJS](http://exploringjs.com/es6/) couvre ECMAScript 6 en détail, mais est structuré de manière à ce que vous puissiez également obtenir rapidement un aperçu si vous le souhaitez.

![Image](https://cdn-media-1.freecodecamp.org/images/0*Rwv5gbZsV-J22p1l.png)

[JavaScript Allonge](https://leanpub.com/javascriptallongesix/read) par [Reginald Braithwaite](https://twitter.com/intent/user?screen_name=raganwald) couvre la programmation fonctionnelle en JavaScript.

![Image](https://cdn-media-1.freecodecamp.org/images/0*XwRkBgkHhJ940Vyp.png)

[JavaScript Spessore](https://leanpub.com/javascript-spessore/read) par [Reginald Braithwaite](https://twitter.com/intent/user?screen_name=raganwald) est écrit pour le lecteur qui a lu [JavaScript Allongé](https://leanpub.com/javascriptallongesix/read). Il couvre les fonctions, les fermetures et les prototypes.

![Image](https://cdn-media-1.freecodecamp.org/images/1*mxwmvIn18mLoXxPH9DBxJA.png)

[What is Code?](http://www.bloomberg.com/graphics/2015-paul-ford-what-is-code/) est un essai interactif de Paul Ford sur ce qu'est le code et pourquoi c'est important.

![Image](https://cdn-media-1.freecodecamp.org/images/0*D9H2nJhHKgyi_yl-.png)

[Le Guide Principalement Adéquat du Professeur Frisby à la Programmation Fonctionnelle](https://github.com/DrBoolean/mostly-adequate-guide) couvre la programmation fonctionnelle en JavaScript.

![Image](https://cdn-media-1.freecodecamp.org/images/0*r7qP2VdbOk0472qa.jpg)

[Programming JavaScript Applications](http://chimera.labs.oreilly.com/books/1234000000262/index.html) par [Eric Elliot](http://%28@_ericelliott%29/) se concentre sur le codage JavaScript intermédiaire.

#### Podcasts

![Image](https://cdn-media-1.freecodecamp.org/images/1*tAe2lujHUc4I5o58lhuCrw.png)

[NodeUp](http://nodeup.com/) est un podcast Node.js réalisé par [@ffloat](https://twitter.com/intent/user?screen_name=ffloat) et [@dshaw](https://twitter.com/intent/user?screen_name=dshaw).

![Image](https://cdn-media-1.freecodecamp.org/images/0*PDdOHaKxHHFg2MH3.png)

[JavaScript Jabber](http://devchat.tv/js-jabber/) est un podcast hebdomadaire sur JavaScript, incluant Node.js, les technologies Front-End, les carrières, les équipes et plus encore.

![Image](https://cdn-media-1.freecodecamp.org/images/0*XggCSS2-KauJqiiE.png)

[Five JS Podcast](https://fivejs.codeschool.com/) est un podcast hebdomadaire de cinq minutes publié chaque jeudi. Vous pouvez [les suivre sur Twitter](https://twitter.com/intent/user?screen_name=FiveJSPodcast) ou vous abonner avec [iTunes](https://itunes.apple.com/us/podcast/5-minutes-of-javascript/id775261328?mt=2) ou [RSS](https://fivejs.codeschool.com/feed.rss).

![Image](https://cdn-media-1.freecodecamp.org/images/0*_oE2N5Vr-3h_NDzm.jpg)

[This Developer's Life](http://thisdeveloperslife.com/) explore différents aspects de la vie des développeurs. Modélisé d'après This American Life de NPR, il présente des interviews et une musique éclectique.

![Image](https://cdn-media-1.freecodecamp.org/images/1*FRCe94UQvCqt3IoFfSktVg.png)

Le [CodeNewbie Podcast](http://www.codenewbie.org/podcast) est un podcast avec des histoires de personnes sur leur parcours de codage. De nouveaux épisodes sont publiés chaque lundi. Vous pouvez les suivre sur Twitter [@CodeNewbies](https://twitter.com/intent/user?screen_name=CodeNewbies)

#### Vidéos

![Image](https://cdn-media-1.freecodecamp.org/images/1*_btjhMh4TKccihFtG1MmFw.png)

[JavaScript: The Good Parts](https://www.youtube.com/watch?v=hQVTIJBZook) est couvert dans cette vidéo YouTube par son créateur, [Douglas Crockford](http://javascript.crockford.com/).

![Image](https://cdn-media-1.freecodecamp.org/images/1*KI9UUD0IATBE9q9bTI2CQQ.png)

[Level Up Tuts](http://leveluptuts.com/) est né du besoin d'une meilleure documentation instructionnelle.

![Image](https://cdn-media-1.freecodecamp.org/images/1*w3LYSYUtGO2tjiqcBSalyg.png)

[DevTips](https://www.youtube.com/user/DevTipsForDesigners) propose des vidéos hebdomadaires sur le sujet de la conception et du développement web.

#### Autres Ressources

![Image](https://cdn-media-1.freecodecamp.org/images/0*p7SvyBW6-3ZAxnwj.png)

Le projet [repl.it](http://repl.it/) est un environnement en ligne pour explorer de manière interactive les langages de programmation.

![Image](https://cdn-media-1.freecodecamp.org/images/0*k-U6tFX1V8N1qx8n.png)

Le [Guide MDN JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) est une excellente référence et fournit un aperçu du langage.

![Image](https://cdn-media-1.freecodecamp.org/images/0*b5Ftp5V8--i5FzDC.png)

[Codewars](http://www.codewars.com/) propose des défis de codage interactifs soumis par les utilisateurs.

![Image](https://cdn-media-1.freecodecamp.org/images/1*H6-v_wPFYz9S_9VcV5HkSQ.png)

[CodinGame](https://www.codingame.com/) vous permet de pratiquer la syntaxe JavaScript à travers des jeux de codage interactifs amusants.

![Image](https://cdn-media-1.freecodecamp.org/images/1*0LVbWebePrQOf-Pry_RbBg.png)

[jQuery Fundamentals](http://jqfundamentals.com/) vous guide à travers les problèmes courants que vous pourrez résoudre en utilisant jQuery.
