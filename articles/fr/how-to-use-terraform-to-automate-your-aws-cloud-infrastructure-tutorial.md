Terraform est un outil logiciel open-source d'infrastructure en tant que code. Il est utilisé pour construire, modifier et versionner l'infrastructure de manière sûre et efficace. Nous avons publié un [cours complet sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=SLB_c_ayRMo) pour vous aider à apprendre à utiliser Terraform.

Dans ce cours de Sanjeev Thiyagarajan, vous apprendrez comment Terraform peut être utilisé pour gérer et automatiser votre infrastructure cloud AWS. Le cours commence par les bases absolues : la création d'un compte AWS gratuit et l'installation de Terraform sur votre machine.

Ensuite, vous apprendrez à déployer une ressource via Terraform et à maîtriser les concepts de base de son utilisation. Vous aurez même l'occasion de configurer un simple serveur web dans AWS.

Voici les sections de ce cours :

* Configuration AWS
* Configuration Windows
* Configuration Mac
* Installation Linux
* VSCode
* Vue d'ensemble de Terraform
* Modification des ressources
* Suppression des ressources
* Référencement des ressources
* Fichiers Terraform
* Projet pratique
* Commandes d'état Terraform
* Sortie Terraform
* Ressources cibles
* Variables Terraform

Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=SLB_c_ayRMo) (2 heures de visionnage).
