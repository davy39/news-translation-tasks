L'une des meilleures choses à propos de VS Code est la riche bibliothèque d'extensions que vous pouvez utiliser pour accélérer votre codage et faciliter votre vie. 

Certaines de ces extensions sont payantes, mais il existe de nombreuses options de qualité disponibles gratuitement dans la Place de marché VS Code. 

Dans cet article, je vais partager 10 extensions gratuites pour votre plaisir de lecture. Plongeons-nous dedans.

## 1. VS Code Auto Rename Tag

Cette extension vous permet de renommer à la fois les balises d'ouverture et de fermeture en une seule fois. Cela signifie que, où que vous choisissiez de renommer - soit la balise d'ouverture ou la balise de fermeture - vous n'avez pas à revenir à l'autre balise pour faire la même chose. Elle est simplement renommée automatiquement pour vous.

![autorenametags](https://www.freecodecamp.org/news/content/images/2021/08/autorenametags.gif)

## 2. Prettier dans VS Code

Le nom « Prettier » est assez explicite - cette extension rend votre code plus joli en le formatant de manière très lisible. 

Vous pouvez choisir de formater votre code à l'enregistrement ou au collage.

![prettier](https://www.freecodecamp.org/news/content/images/2021/08/prettier.gif)

## 3. Convertir les pixels en Rem dans VS Code

Parfois, vous pourriez vouloir que vos unités soient en rem (root em) au lieu de pixels (px) et vice versa. 

Bonne nouvelle - vous n'avez pas besoin de faire les conversions mathématiques vous-même car c'est ce que cette extension fait pour vous. 

Pour convertir de rem en px et vice versa, appuyez et maintenez les touches alt + z.

![pxtoem](https://www.freecodecamp.org/news/content/images/2021/08/pxtoem.gif)

## 4. Cheatsheet Flexbox pour VS Code
L'extension Cheatsheet Flexbox apporte une cheatsheet pratique de Flexbox directement dans VS Code. 

Vous pouvez l'utiliser de deux manières : en appuyant sur Ctrl + Shift + P sous Windows et Cmd + Shift + P sur Mac pour la faire apparaître, ou vous pouvez simplement survoler n'importe où où l'affichage est défini sur flex.

![flex](https://www.freecodecamp.org/news/content/images/2021/08/flex.gif)

## 5. Extension Grid Snippets pour VS Code

L'extension Grid Snippets vous donne la possibilité de coder en CSS Grid avec des raccourcis simples qui apportent les snippets pour vous. 

Certains de ces raccourcis incluent :

- dg – display grid
- dig – display inline-grid
- gg – grid gap
- gtc – grid-template-columns
- gta – grid-template-areas

Il existe de nombreux autres raccourcis disponibles dans la documentation.

![grid](https://www.freecodecamp.org/news/content/images/2021/08/grid.gif)

## 6. Bracket Pair Colorizer

Il est facile de se perdre dans les accolades de notre code. Ce coloriseur de paires d'accolades atténue cette confusion en attribuant différentes couleurs aux accolades (parenthèses, accolades, crochets).

Vous pouvez voir dans la capture d'écran ci-dessous que toutes les accolades ont des couleurs différentes afin que vous puissiez garder une trace de celles qui sont lesquelles.

![bracket](https://www.freecodecamp.org/news/content/images/2021/08/bracket.png)

## 7. Extension Live Server dans VS Code

Les jours où nous devions tous recharger le navigateur pour voir les modifications apportées à notre code sont révolus. 

De nos jours, il vous suffit d'installer l'extension Live Server depuis la Place de marché VS Code, et hop ! Chaque modification que vous apportez est immédiatement mise à jour dans le navigateur. 

Le fichier que vous ouvrez avec Live Server doit être le fichier HTML. De plus, le fichier HTML doit avoir des balises head et body d'ouverture et de fermeture.

![liveserver](https://www.freecodecamp.org/news/content/images/2021/08/liveserver.gif)

## 8. Icônes VS Code

VS Code Icons est l'une de mes extensions préférées. Avec elle, vous pouvez voir l'icône de votre fichier représentée dans le langage ou le framework/bibliothèque dans lequel vous codez.

![vscideicons](https://www.freecodecamp.org/news/content/images/2021/08/vscideicons.gif)

## 9. Polacode

Cette extension est très intéressante à utiliser. Elle vous permet de prendre de belles captures d'écran de votre code. 

Pour prendre une capture d'écran, vous appuyez simplement sur Ctrl + Shift + P sous Windows et Cmd + Shift + P sur Mac pour faire apparaître la palette de commandes. Ensuite, vous pouvez rechercher « Polacode » et le sélectionner. Il vous suffit de surligner la partie que vous souhaitez capturer et d'appuyer sur l'icône en forme d'appareil photo pour prendre la capture d'écran et l'enregistrer sur votre appareil.

![polacode](https://www.freecodecamp.org/news/content/images/2021/08/polacode.gif)

## 10. Code Time

Code Time est une extension qui vous aide à suivre votre progression et vos métriques directement dans VS Code. 

Mais ce n'est que la partie émergée de l'iceberg. Avec cette extension super cool, vous pouvez désactiver les notifications, connecter un espace de travail Slack et définir votre présence comme absente, connaître votre temps de codage, votre progression, et bien plus encore. 

De plus, vous pouvez également inviter vos membres d'équipe, ce qui vous aidera à découvrir l'impact des réunions ou de toute autre activité. 

Vous obtenez une petite icône en forme de pied lorsqu'elle est installée. À partir de là, vous pouvez explorer toutes les fonctionnalités. Étonnamment, elle n'est pas payante - elle est gratuite ! 

![codetime](https://www.freecodecamp.org/news/content/images/2021/08/codetime.png)
