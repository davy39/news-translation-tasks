Par Karlijn Willems

Vous avez peut-être entendu parler de l'apprentissage automatique grâce à des applications intéressantes comme le filtrage de spam, la reconnaissance optique de caractères et la vision par ordinateur.

Se lancer dans l'apprentissage automatique est un long processus qui implique de parcourir plusieurs ressources. Il existe des livres pour débutants, des articles académiques, des exercices guidés et des projets autonomes. Il est facile de perdre de vue ce que vous devez apprendre parmi toutes ces options.

Alors dans l'article d'aujourd'hui, je vais lister sept étapes (et 50+ ressources) qui peuvent vous aider à vous lancer dans ce domaine passionnant de l'informatique, et à progresser vers devenir un héros de l'apprentissage automatique.

Notez que cette liste de ressources n'est pas exhaustive et est destinée à vous aider à commencer. Il existe de nombreuses autres ressources.

### 1. Acquérez les connaissances de base nécessaires

Vous vous souvenez peut-être de l'infographie [Learn Data Science](https://www.datacamp.com/community/tutorials/learn-data-science-infographic) de DataCamp selon laquelle les mathématiques et les statistiques sont essentielles pour commencer l'apprentissage automatique (ML). Les fondements peuvent sembler assez simples car il ne s'agit que de trois sujets. Mais n'oubliez pas que ce sont en fait trois sujets très vastes.

Il y a deux choses très importantes à garder à l'esprit ici :

* Premièrement, vous aurez définitivement besoin de quelques conseils supplémentaires sur ce que vous devez exactement couvrir pour commencer.
* Deuxièmement, ce sont les fondements de votre apprentissage futur. Ne craignez pas de prendre votre temps. Acquérez les connaissances sur lesquelles vous allez tout construire.

Le premier point est simple : il est bon de couvrir l'algèbre linéaire et les statistiques. Ces deux sujets sont le minimum que l'on devrait comprendre. Mais pendant que vous y êtes, vous devriez également essayer de couvrir des sujets tels que l'optimisation et le calcul avancé. Ils seront utiles lorsque vous approfondirez le ML.

Voici quelques conseils pour savoir où commencer si vous partez de zéro :

* [Khan Academy](http://www.khanacademy.org/) est une bonne ressource pour les débutants. Pensez à suivre les cours d'Algèbre Linéaire et de Calcul.
* Allez sur [MIT OpenCourseWare](https://ocw.mit.edu/index.htm) et suivez le cours d'[Algèbre Linéaire](https://ocw.mit.edu/courses/mathematics/18-06-linear-algebra-spring-2010/).
* Suivez [ce cours Coursera](https://www.coursera.org/learn/basic-statistics) pour une introduction aux statistiques descriptives, à la théorie des probabilités et aux statistiques inférentielles.

![Image](https://cdn-media-1.freecodecamp.org/images/4LbhpCaTCjArtcb6Q48GoKqaOePSKpPZan4O)
_Les statistiques sont l'une des clés pour apprendre le ML_

Si vous préférez les livres, envisagez les suivants :

* [_Linear Algebra and Its Applications_](https://www.amazon.com/Linear-Algebra-Its-Applications-4th/dp/0030105676),
* [_Applied Linear Algebra_](https://www.amazon.com/Applied-Linear-Algebra-3rd-Noble/dp/0130412600),
* [_3,000 Solved Problems in Linear Algebra_](https://www.amazon.de/Solved-Problems-Linear-Algebra-Schaums/dp/0070380236),
* [MIT Online Texbooks](https://ocw.mit.edu/courses/online-textbooks/)

Cependant, dans la plupart des cas, vous commencerez déjà en connaissant certaines choses sur les statistiques et les mathématiques. Ou peut-être avez-vous déjà parcouru toutes les ressources théoriques listées ci-dessus.

Dans ces cas, il est bon de récapituler et d'évaluer honnêtement vos connaissances. Y a-t-il des domaines que vous devez réviser ou êtes-vous prêt pour l'instant ?

Si vous êtes prêt, il est temps de passer à l'étape suivante et d'appliquer toutes ces connaissances avec R ou Python. En règle générale, il est bon d'en choisir un et de commencer avec ce langage. Plus tard, vous pourrez toujours ajouter l'autre langage de programmation à votre ensemble de compétences.

Pourquoi toutes ces connaissances en programmation sont-elles nécessaires ?

Eh bien, vous verrez que les cours listés ci-dessus (ou ceux que vous avez suivis à l'école ou à l'université) vous fourniront une introduction plus théorique (et non appliquée) aux sujets de mathématiques et de statistiques. Cependant, le ML est très appliqué et vous devrez être capable d'appliquer tous les sujets que vous avez appris. Il est donc bon de parcourir à nouveau les matériaux, mais cette fois de manière appliquée.

Si vous voulez maîtriser les bases de R et Python, envisagez les cours suivants :

* Les cours introductifs Python ou R de DataCamp : [Intro to Python for Data Science](https://www.datacamp.com/courses/intro-to-python-for-data-science) ou [Introduction to R Programming](https://www.datacamp.com/courses/free-introduction-to-r).
* Les cours introductifs Python et R d'Edx : [Introduction to Python for Data Science](https://www.edx.org/course/introduction-python-data-science-microsoft-dat208x-5) et [Introduction to R for Data Science](https://www.edx.org/course/introduction-r-data-science-microsoft-dat204x-4).
* Il existe de nombreux autres cours gratuits. Consultez [Coursera](http://www.coursera.org/) ou [Codeacademy](https://www.codecademy.com/) pour plus d'options.

Lorsque vous aurez maîtrisé les bases, consultez le blog de DataCamp sur les [40+ Python Statistics For Data Science Resources](https://www.datacamp.com/community/tutorials/python-statistics-data-science). Cet article propose 40+ ressources sur les sujets de statistiques que vous devez connaître pour commencer avec la science des données (et par extension aussi le ML).

Assurez-vous également de consulter [ce tutoriel SciPy](https://www.datacamp.com/community/tutorials/python-scipy-tutorial) sur les vecteurs et les tableaux et [cet atelier](http://www.math.pitt.edu/~siam/workshops/python10/python.pdf) sur le calcul scientifique avec Python.

Pour vous exercer avec Python et le calcul, vous pouvez consulter le [package SymPy](http://docs.sympy.org/latest/tutorial/calculus.html).

### 2. N'ayez pas peur d'investir dans la "théorie" du ML

Beaucoup de gens ne font pas l'effort de parcourir des matériaux plus théoriques parce que c'est "sec" ou "ennuyeux". Mais parcourir la théorie et vraiment investir votre temps dedans est essentiel et inestimable à long terme. Vous comprendrez mieux les nouvelles avancées en apprentissage automatique, et vous pourrez vous référer à vos connaissances de base. Cela vous aidera à rester motivé.

De plus, la théorie n'a pas besoin d'être ennuyeuse. Comme vous l'avez lu dans l'introduction, il existe de nombreuses ressources qui vous faciliteront la tâche.

Les **livres** sont l'un des meilleurs moyens d'absorber les connaissances théoriques. Ils vous obligent à vous arrêter et à réfléchir de temps en temps. Bien sûr, lire des livres est une activité très statique et cela peut ne pas convenir à votre style d'apprentissage. Néanmoins, essayez les livres suivants et voyez si cela pourrait vous convenir :

* [_Machine Learning textbook_](http://www.cs.cmu.edu/~tom/mlbook.html), de Tom Mitchell, peut être ancien mais c'est de l'or. Ce livre passe en revue les sujets les plus importants de l'apprentissage automatique de manière bien expliquée et étape par étape.
* _Machine Learning: The Art and Science of Algorithms that Make Sense of Data_ (vous pouvez voir les diapositives du livre [ici](http://www.cs.bris.ac.uk/~flach/mlbook/materials/mlbook-beamer.pdf)) : ce livre est idéal pour les débutants. De nombreuses applications réelles sont discutées, ce qui peut manquer dans le livre de Tom Mitchell.
* [_Machine Learning Yearning_](http://www.mlyearning.org/) : ce livre d'Andrew Ng n'est pas encore complet, mais il est destiné à être une excellente référence pour ceux qui apprennent le ML.
* [_Algorithms and Data Structures_](https://www.amazon.com/Algorithms-Data-Structures-Applications-Practitioner/dp/0134894286) par Jurg Nievergelt et Klaus Hinrichs
* Consultez également [_Data Mining for the Masses_](https://www.amazon.com/Data-Mining-Masses-Matthew-North/dp/0615684378) par Matthew North. Vous trouverez que ce livre vous guide à travers certains des sujets les plus difficiles.
* [_Introduction to Machine Learning_](http://alex.smola.org/drafts/thebook.pdf) par Alex Smola et S.V.N. Vishwanathan.

![Image](https://cdn-media-1.freecodecamp.org/images/QZcUedk1XakAKTLRS90ptnGRLAoAwv-eBYPO)
_Prenez votre temps pour lire des livres et étudier le matériel qu'ils couvrent_

Les **vidéos / MOOCs** sont excellents pour ceux qui apprennent en regardant et en écoutant. Il existe de nombreux MOOCs et vidéos, mais il peut aussi être difficile de trouver votre chemin à travers toutes ces ressources. Voici une liste des plus notables :

* [Ce MOOC bien connu sur l'apprentissage automatique](https://www.coursera.org/learn/machine-learning), enseigné par Andrew Ng, vous introduit à l'apprentissage automatique et à la théorie. Ne vous inquiétez pas — c'est bien expliqué et prend les choses étape par étape, donc c'est excellent pour les débutants.
* La [playlist du cours MIT Open Courseware 6034](https://youtu.be/TjZBTDzGeGg?list=PLnvKubj2-I2LhIibS8TOGC42xsD3-liux) : déjà un peu plus avancé. Vous aurez définitivement besoin de quelques travaux précédents sur la théorie du ML avant de commencer cette série, mais vous ne le regretterez pas.

À ce stade, il est important pour vous de passer en revue les techniques séparées et de saisir l'ensemble du tableau. Cela commence par la compréhension des concepts clés : la distinction entre l'apprentissage supervisé et non supervisé, la classification et la régression, et ainsi de suite. Les exercices manuels (écrits) peuvent être utiles. Ils peuvent vous aider à comprendre comment les algorithmes fonctionnent et comment vous devriez les aborder. Vous trouverez le plus souvent ces exercices écrits dans les cours des universités. Consultez [ce cours de ML](http://web.cecs.pdx.edu/~mm/MachineLearningSpring2017/) de l'Université d'État de Portland.

### 3. Passez à la pratique

Connaître la théorie et comprendre les algorithmes en lisant et en regardant, c'est bien. Mais vous devez aussi dépasser cette étape et commencer quelques exercices. Vous apprendrez à implémenter ces algorithmes et à appliquer la théorie que vous avez apprise.

Tout d'abord, vous avez des tutoriels qui vous introduisent aux bases de l'apprentissage automatique en Python et R. La meilleure façon est, bien sûr, d'opter pour des tutoriels interactifs :

* Dans [Python Machine Learning: Scikit-Learn Tutorial](https://www.datacamp.com/community/tutorials/machine-learning-python), vous en apprendrez davantage sur les algorithmes bien connus KMeans et Support Vector Machines (SVM) pour construire des modèles avec Scikit-Learn.
* [Machine Learning in R for beginners](https://www.datacamp.com/community/tutorials/machine-learning-in-r) vous introduit au ML en R avec les packages class et caret.
* [Keras Tutorial: Deep Learning in Python](https://www.datacamp.com/community/tutorials/deep-learning-python) couvre comment construire des Perceptrons Multi-Couches (MLPs) pour des tâches de classification et de régression, étape par étape.

Consultez également les tutoriels suivants, qui sont statiques et nécessiteront de travailler dans un IDE :

* [Machine Learning in Python, Step By Step](http://machinelearningmastery.com/machine-learning-in-python-step-by-step/) : tutoriel étape par étape avec Scikit-Learn.
* [Develop Your First Neural Network in Python With Keras Step-By-Step](http://machinelearningmastery.com/tutorial-first-neural-network-python-keras/) : apprenez à développer votre premier réseau de neurones avec Keras grâce à ce tutoriel.
* Il y en a beaucoup d'autres que vous pouvez considérer, mais les tutoriels de [Machine Learning Mastery](http://www.machinelearningmastery.com/) sont très bons.

Outre les tutoriels, il y a aussi des cours. Suivre des cours vous aidera à appliquer les concepts que vous avez appris de manière ciblée. Des instructeurs expérimentés vous aideront. Voici quelques cours interactifs pour Python et ML :

* [Supervised Learning with scikit-learn](https://www.datacamp.com/courses/supervised-learning-with-scikit-learn) : vous apprendrez à construire des modèles prédictifs, à ajuster leurs paramètres et à prédire leur performance sur des données invisibles. Tout cela en utilisant des jeux de données réels. Vous le ferez avec Scikit-Learn.
* [Unsupervised Learning in Python](https://www.datacamp.com/courses/unsupervised-learning-in-python) : montre comment clusteriser, transformer, visualiser et extraire des insights à partir de jeux de données non étiquetés. À la fin du cours, vous construirez un système de recommandation.
* [Deep Learning in Python](https://www.datacamp.com/courses/deep-learning-in-python) : vous acquerrez des connaissances pratiques sur l'utilisation de l'apprentissage profond avec Keras 2.0, la dernière version d'une bibliothèque de pointe pour l'apprentissage profond en Python.
* [Applied Machine Learning in Python](https://www.coursera.org/learn/python-machine-learning) : introduit l'apprenant à l'apprentissage automatique appliqué et se concentre davantage sur les techniques et méthodes que sur les statistiques derrière ces méthodes.

![Image](https://cdn-media-1.freecodecamp.org/images/3V20GPHNc3Ubmi3HdE1rWaKy9w7nX8sJVMkQ)
_Après la théorie, prenez votre temps pour appliquer les connaissances que vous avez acquises._

Pour ceux qui apprennent le ML avec R, il y a aussi ces cours interactifs :

* [Introduction to Machine Learning](https://www.datacamp.com/courses/introduction-to-machine-learning-with-r) vous donne un aperçu général des techniques et applications les plus courantes de la discipline. Vous obtiendrez plus d'informations sur l'évaluation et l'entraînement de différents modèles ML. Le reste du cours se concentre sur une introduction à trois des tâches ML les plus basiques : la classification, la régression et le clustering.
* [R: Unsupervised Learning](https://www.datacamp.com/courses/unsupervised-learning-in-r) fournit une introduction de base au clustering et à la réduction de dimensionnalité en R d'un point de vue ML. Cela vous permet de passer des données aux insights aussi rapidement que possible.
* [Practical Machine Learning](https://www.coursera.org/learn/practical-machine-learning) couvre les composants de base de la construction et de l'application de fonctions de prédiction avec un accent sur les applications pratiques.

Enfin, il y a aussi des livres qui abordent les sujets de ML de manière très appliquée. Si vous cherchez à apprendre avec l'aide de texte et d'un IDE, consultez ces livres :

* Le [_Python Machine Learning Book_](https://github.com/rasbt/python-machine-learning-book) de Sebastian Raschka
* L'[Introduction to Artificial Neural Networks and Deep Learning: A Practical Guide with Applications in Python](https://github.com/rasbt/deep-learning-book) de Sebastian Raschka
* [_Machine Learning with R_](https://books.google.be/books/about/Machine_Learning_with_R.html?id=ZQu8AQAAQBAJ&source=kp_cover&redir_esc=y) de Brett Lantz

### 4. Pratiquez

La pratique est encore plus importante que de se lancer et de réviser le matériel avec Python. Cette étape a probablement été la plus difficile pour moi. Consultez comment d'autres personnes ont implémenté des algorithmes ML lorsque vous avez fait quelques exercices. Ensuite, lancez-vous dans vos propres projets qui illustrent votre compréhension des algorithmes et théories ML.

L'une des manières les plus directes est de voir les exercices un peu plus grands. Vous voulez faire un exercice plus grand qui vous oblige à faire plus de nettoyage de données et d'ingénierie de caractéristiques.

* Commencez avec [Kaggle](http://www.kaggle.com/). Si vous avez besoin d'aide supplémentaire pour vaincre la soi-disant "peur des données", consultez le [Kaggle Python Tutorial on Machine Learning](https://www.datacamp.com/community/open-courses/kaggle-python-tutorial-on-machine-learning) et le [Kaggle R Tutorial on Machine Learning](https://www.datacamp.com/community/open-courses/kaggle-tutorial-on-machine-learing-the-sinking-of-the-titanic). Ceux-ci vous mettront à niveau en un rien de temps.
* Ensuite, vous pouvez également commencer à faire des défis par vous-même. Consultez ces sites, où vous pouvez trouver de nombreux jeux de données ML : [UCI Machine Learning Repository](http://archive.ics.uci.edu/ml/), [Public datasets for machine learning](http://homepages.inf.ed.ac.uk/rbf/IAPR/researchers/MLPAGES/mldat.htm), et [data.world](https://data.world/).

**Astuce** : n'oubliez pas qu'il existe des ressources pratiques pour vous aider lorsque vous pratiquez — Consultez ces [aides-mémoire de science des données](https://www.datacamp.com/community/data-science-cheatsheets).

### 6. Ne vous arrêtez pas

Apprendre le ML est quelque chose qui ne devrait jamais s'arrêter. Comme beaucoup le confirmeront, il y a toujours de nouvelles choses à apprendre — même lorsque vous travaillez dans ce domaine depuis une décennie.

Il y a, par exemple, des **tendances ML** comme l'apprentissage profond qui sont très populaires en ce moment. Vous pourriez également vous concentrer sur d'autres sujets qui ne sont pas centraux à ce stade mais qui pourraient l'être à l'avenir. Consultez cette [question intéressante et les réponses](https://www.quora.com/Should-I-quit-machine-learning) si vous voulez en savoir plus.

Les **articles** ne sont peut-être pas la première chose qui vient à l'esprit lorsque vous êtes préoccupé par la maîtrise des bases. Mais ils sont votre moyen de vous tenir au courant des dernières recherches. Les articles ne sont pas destinés à ceux qui commencent. Ils conviennent définitivement à ceux qui sont plus avancés.

* [Top 20 des articles de recherche récents sur l'apprentissage automatique et l'apprentissage profond](http://www.kdnuggets.com/2017/04/top-20-papers-machine-learning.html)
* [Journal of Machine Learning Research](http://www.jmlr.org/)
* [Awesome Deep Learning Papers](https://github.com/terryum/awesome-deep-learning-papers)
* [Quels sont certains des meilleurs articles de recherche/livres pour l'apprentissage automatique ?](https://www.quora.com/What-are-some-of-the-best-research-papers-books-for-Machine-learning)

Les **autres technologies** sont également quelque chose à considérer. Mais ne vous inquiétez pas d'elles lorsque vous commencez. Vous pouvez, par exemple, vous concentrer sur l'ajout de Python ou R (selon celui que vous connaissez déjà) à votre ensemble de compétences. Vous pouvez parcourir cet article pour trouver des ressources intéressantes.

Si vous souhaitez également vous orienter vers le big data, vous pourriez envisager de vous pencher sur Spark. Voici quelques ressources intéressantes :

### 7. Utilisez tout le matériel disponible

L'apprentissage automatique est un sujet difficile qui peut vous faire perdre votre motivation à un moment donné. Ou peut-être avez-vous besoin d'un changement. Dans de tels cas, rappelez-vous qu'il existe de nombreuses ressources sur lesquelles vous pouvez vous appuyer. Consultez les ressources suivantes :

**Podcasts**. Excellente ressource pour continuer votre voyage dans le ML et rester à jour avec les derniers développements dans le domaine :

* [DataFramed](https://www.datacamp.com/community/podcast)
* [Talking Machines](http://www.thetalkingmachines.com/)
* [Data Skeptic](https://dataskeptic.com/)
* [Linear Digressions](http://lineardigressions.com/)
* [This Week in Machine Learning & AI](https://twimlai.com/)
* [Learning Machines 101](http://www.learningmachines101.com/)

Il existe, bien sûr, de nombreux autres podcasts, mais cette liste est juste pour vous aider à commencer !

La **documentation** et le **code source des packages** sont deux moyens de vous plonger plus profondément dans l'implémentation des algorithmes ML. Consultez certains de ces dépôts :

* [Scikit-Learn](https://github.com/scikit-learn/scikit-learn) : Package ML Python bien connu
* [Keras](http://www.github.com/fchollet/keras) : Package d'apprentissage profond pour Python
* [caret](http://topepo/caret) : package R très populaire pour la Classification et l'Entraînement de Régression

Les **visualisations** sont l'une des manières les plus récentes et les plus tendance de se plonger dans la théorie du ML. Elles sont fantastiques pour les débutants, mais aussi très intéressantes pour les apprenants plus avancés. Les visualisations suivantes vous intrigueront et vous aideront à mieux comprendre le fonctionnement du ML :

* [Une introduction visuelle à l'apprentissage automatique](http://www.r2d3.us/visual-intro-to-machine-learning-part-1/)
