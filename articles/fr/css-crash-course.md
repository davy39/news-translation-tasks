Il est important pour tous les développeurs web de connaître quelque chose sur CSS. Ce cours accéléré de Web Dev Simplified couvre tout ce que vous devez savoir pour commencer avec CSS en juste un peu plus de 20 minutes.

Il couvre la syntaxe CSS, comment ajouter CSS à votre HTML, les couleurs CSS, les unités CSS, le modèle de boîte, et les meilleures pratiques pour CSS. Vous verrez un exemple complet de CSS utilisé pour styliser une page HTML et à la fin de cette vidéo, vous en saurez assez sur CSS pour styliser n'importe quelle page web basique dans vos propres projets !

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=r1xBCi5SOjw) (24 minutes de visionnage).
