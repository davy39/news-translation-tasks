Apprenez les bases de HTML et CSS dans ce tutoriel complet de Web Dev Simplified. Vous allez créer un projet de site web pour un groupe de musique en utilisant HTML et apprendre à appliquer un design moderne afin de styliser le site avec CSS.

En plus de couvrir les éléments de base et avancés de HTML, ce cours aborde également les meilleures pratiques pour utiliser CSS afin de styliser HTML.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=kMT54MPz9oE) (2 heures de visionnage).
