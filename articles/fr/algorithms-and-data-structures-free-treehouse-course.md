Les algorithmes et les structures de données sont deux des sujets fondamentaux en informatique. Tous les programmeurs les rencontreront, et ils sont souvent abordés lors des entretiens d'embauche.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous offrira une excellente introduction aux algorithmes et aux structures de données.

Ce cours a été développé par les enseignants Pasan Premaratne et Jay McGavren. Pour la première fois, ce cours est désormais disponible gratuitement.

Le cours est en fait une combinaison de ces trois cours plus courts :

* Introduction aux algorithmes
* Introduction aux structures de données
* Algorithmes : Tri et recherche

Dans la première section, vous apprendrez ce que sont les algorithmes, comment ils fonctionnent et où ils peuvent être trouvés.

Dans la deuxième section, vous en apprendrez davantage sur deux structures de données introductives – les tableaux et les listes chaînées. Vous examinerez les opérations courantes et comment les temps d'exécution de ces opérations affectent notre code quotidien.

Pour le reste du cours, vous combinerez vos connaissances des algorithmes et des structures de données pour résoudre le problème du tri des données en utilisant l'algorithme de tri par fusion.

La dernière section examine les algorithmes dans deux catégories : le tri et la recherche. Vous implémenterez des algorithmes de tri bien connus comme le tri par sélection, le tri rapide et le tri par fusion. Vous apprendrez également les algorithmes de recherche de base comme la recherche séquentielle et la recherche binaire.

À la fin de ce cours de 6 heures, vous comprendrez ce que sont les algorithmes et les structures de données, comment ils sont mesurés et évalués, et comment les développeurs les utilisent pour résoudre des problèmes.

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=8hly31xKli0). Assurez-vous de vérifier la description pour les extraits de code inclus.
