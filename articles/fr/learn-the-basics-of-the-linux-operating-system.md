Linux est un système d'exploitation open source et gratuit utilisé par des ordinateurs du monde entier. Il est particulièrement populaire pour une utilisation sur les serveurs.

Nous venons de publier un cours accéléré sur Linux sur la chaîne YouTube freeCodeCamp.org qui vous enseignera les bases de ce système d'exploitation open source.

Josh de KeepItTechie a développé ce cours. Josh s'engage à faire connaître les différents systèmes d'exploitation Linux et à aider les gens à apprendre à utiliser Linux pour un usage domestique et professionnel de base.

Linux est un clone du système d'exploitation UNIX, donc comprendre l'un revient à comprendre l'autre. Dans ce cours, vous obtiendrez une compréhension complète du fonctionnement du système d'exploitation Linux.

Vous pouvez suivre ce cours sur un ordinateur Windows ou Mac. Vous apprendrez à installer facilement Linux en utilisant une machine virtuelle.

Voici les sections couvertes dans ce cours :

* Installer Linux (#heading-installation)
* Environnement de bureau
* Terminal
* Travailler avec les répertoires
* Travailler avec les fichiers
* Travailler avec le contenu des fichiers
* Structure des fichiers Linux
* Informations système
* Réseautage
* Gestionnaire de paquets Linux
* Éditeur de texte
* Conclusion
