Apprenez à réussir la certification HashiCorp Terraform Associate (003) avec ce cours gratuit de 7 heures.

## **Qu'est-ce que la certification HashiCorp Terraform Associate ?**

HashiCorp est une entreprise spécialisée dans les outils open-source pour les charges de travail multi-cloud. L'outil le plus populaire de HashiCorp est Terraform, qui permet aux ingénieurs DevOps d'écrire du code pour provisionner des infrastructures pour plusieurs fournisseurs de services cloud (CSP), par exemple AWS, Azure et GCP.

La certification HashiCorp Terraform Associate est une certification qui prouve qu'un ingénieur a une compréhension pratique de l'outil Terraform ainsi que de l'offre SaaS connue sous le nom de Terraform Cloud.

Pour ceux qui cherchent à faire carrière en tant qu'ingénieur DevOps, la certification Terraform Associate est essentielle puisque Terraform est devenu la norme industrielle pour l'Infrastructure as Code (IaC) et apparaît fréquemment comme une compétence attendue dans les offres d'emploi DevOps.

La certification Terraform Associate n'est pas un examen difficile, mais elle repose fortement sur des connaissances pratiques de Terraform. C'est pourquoi ce cours de préparation dure 7 heures – nous avons ajouté de nombreux exercices pratiques et cas particuliers courants que vous ne rencontrerez qu'en pratiquant.

## **Aperçu de la certification Terraform Associate**

La certification Terraform Associate est composée des domaines suivants :

1. Comprendre les concepts de l'Infrastructure as Code (IaC)
2. Comprendre le but de Terraform (par rapport à d'autres IaC)
3. Comprendre les bases de Terraform
4. Utiliser la CLI de Terraform (en dehors du flux de travail principal)
5. Interagir avec les modules Terraform
6. Naviguer dans le flux de travail Terraform
7. Implémenter et maintenir l'état
8. Lire, générer et modifier la configuration
9. Comprendre les capacités de Terraform Cloud et Enterprise

Bien que la certification Terraform comporte beaucoup plus de domaines que d'autres certifications cloud, les exigences attendues pour chaque domaine sont finies, et presque tout ce que vous voyez dans le guide de l'examen apparaît à l'examen.

## **Puis-je simplement regarder les vidéos et réussir l'examen ?**

Oui, la certification HashiCorp Terraform Associate est suffisamment facile pour que vous puissiez simplement regarder les vidéos et réussir l'examen.

Cependant, comme l'examen est toujours basé sur les trois dernières versions mineures de Terraform, si vous ne faites pas les exercices pratiques sur votre machine, vous pourriez manquer des cas particuliers émergents de Terraform.
