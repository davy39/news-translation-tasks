Apprenez à utiliser GraphQL avec Laravel et Vue.js dans ce cours d'Andre Madarang. Le cours utilise le package Lighthouse sur le backend avec Laravel et Vue Apollo sur le frontend. Vous apprendrez à construire une application de recommandation de livres à partir de zéro comme projet. 

Voici les sujets abordés dans le cours :

* Package Lighthouse Backend
* Requêtes Vue Apollo
* Mutations Vue Apollo
* Stylisation Tailwind CSS
* Déploiement
* Client GraphQL dans Laravel
* Serveur GraphQL Laravel avec graphql-laravel
* Authentification GraphQL dans Laravel avec Lighthouse & Passport
* Authentification GraphQL dans Vue Apollo

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=4z3EMCc4bP4) (5 heures de visionnage).
