Il existe une tonne de cours de programmation sur la façon de coder des jeux et des applications. Cependant, beaucoup d'entre eux ne montrent pas le processus complet de codage d'un programme jusqu'à sa mise en ligne sur le magasin d'applications pour que les gens puissent le télécharger.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer et publier une application Android à partir de zéro.

Rahul Pandey enseigne ce cours. Rahul est ingénieur chez Facebook. Il donne également des cours à l'Université Stanford et a publié de nombreuses applications.

Dans ce cours pour débutants, vous apprendrez à utiliser Kotlin pour créer et publier un jeu de mémoire personnalisable qui peut sauvegarder du contenu dans le stockage cloud Firebase.

L'application que vous allez créer vous permet de jouer à l'un des jeux prédéfinis, ou de jouer à un jeu personnalisé créé par vous ou un ami ! Il s'agit d'un jeu de mémoire classique qui vous permet d'utiliser des photos de votre téléphone.

Rahul a déjà publié l'application de ce cours sur le Play Store. Vous pouvez [la consulter ici](https://play.google.com/store/apps/details?id=com.rkpandey.mymemory) puis regarder le cours pour apprendre à la créer vous-même.

Dans ce cours, vous apprendrez :

* Les mises en page dynamiques
* La navigation multi-écrans
* Le chargement d'images
* La persistance avec Firebase

Vous apprendrez également des conseils et astuces pratiques sur le développement d'applications et la checklist de Rahul qu'il utilise avant de publier des applications sur le Google Play Store.

Il s'agit d'un cours pour débutants, mais vous devriez avoir une certaine expérience en programmation. Il serait également utile de connaître un peu Kotlin, mais il est possible de suivre même si vous êtes complètement nouveau dans ce langage.

Voici toutes les sections du cours :

* Introduction + vue d'ensemble de l'application
* Créer la mise en page principale
* Configurer le RecyclerView pour afficher les cartes mémoire
* Ajouter l'énumération BoardSize + créer des icônes
* Créer les classes de cartes mémoire et de jeu
* Implémenter la logique du jeu
* Afficher les informations du jeu
* Ajouter des options de menu pour le contrôle du jeu
* Utiliser une intention pour naviguer vers CreateActivity
* Construire la grille d'images dans le flux de création
* Choisir l'intention de l'image
* Préparation du téléchargement de l'image
* Télécharger vers Firebase Storage
* Sauvegarder le jeu de mémoire dans Firebase
* Jouer à la mémoire avec des images personnalisées
* Ajouter une option de menu pour télécharger un jeu
* Mises à jour de style
* Créer une version de release
* Publier votre application sur Google Play
* Révision des concepts

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=C2DBDZKkLss) (4 heures de visionnage).
