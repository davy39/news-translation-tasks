Il est simple et facile de créer un site web en utilisant WordPress. 

WordPress est un système de gestion de contenu qui vous aide à créer un site web même si vous ne connaissez rien à la programmation. Vous n'avez pas besoin de connaître HTML, CSS ou JavaScript. Vous pouvez tout faire graphiquement avec WordPress.

WordPress est de loin le moyen le plus populaire de créer un site web. Il est utilisé par 40 % de TOUS les sites web. Et surtout, WordPress est gratuit à utiliser.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un site web en utilisant WordPress.

Ce cours est parfait pour les débutants. À la fin du cours, vous saurez comment créer de nombreux types de sites web différents et comment les personnaliser.

Tout d'abord, vous apprendrez à faire la configuration initiale. Cela inclut l'obtention d'un nom de domaine, l'hébergement et l'installation de WordPress.

[Hostinger](https://www.hostinger.com/) a fourni une subvention pour rendre ce cours possible. Vous apprendrez à configurer l'hébergement du site web en utilisant Hostinger. Et ils ont même fourni à nos téléspectateurs un code de réduction que vous pouvez trouver dans la description de la vidéo.

Après avoir installé WordPress, vous apprendrez à configurer l'installation et à la personnaliser pour vos besoins. Vous pourriez vouloir transformer le site web en un blog, un site de commerce électronique ou un site web pour une entreprise locale. Vous apprendrez comment vous pouvez facilement le personnaliser pour votre cas d'utilisation. Vous apprendrez à mettre à jour le design, à ajouter des images, à utiliser des couleurs personnalisées, et plus encore.
