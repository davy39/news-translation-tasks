Par Adam Naor

Un fichier ISO est une copie exacte d'un disque optique entier tel qu'un CD, un DVD ou un Blu-ray archivé dans un seul fichier. Ce fichier, qui est également parfois appelé une image ISO, est un duplicata de petite taille de grands ensembles de données. 

Pour remplir sa fonction, le fichier ISO doit être ouvert et assemblé afin que les données puissent être consultées.

De nombreuses personnes utilisent les fichiers ISO pour sauvegarder leurs disques optiques ou stocker leurs données de manière plus fonctionnelle. 

Sa fonction réelle est de répliquer un disque optique original et de le stocker jusqu'à ce qu'il soit nécessaire de graver un nouveau disque contenant les mêmes données. Vous pouvez l'utiliser pour transférer un jeu, par exemple, d'un ancien disque vers votre ordinateur portable lorsqu'il n'y a pas de disque physique à proximité. 

Cependant, les fichiers ISO sont utilisés pour des fins bien plus grandes lorsqu'il s'agit de systèmes d'exploitation. Ils peuvent être d'excellents conteneurs pour de grands programmes ou des systèmes d'exploitation eux-mêmes, puisqu'ils sont un seul fichier contenant d'énormes quantités de données. 

En les téléchargeant dans un fichier ISO, ces systèmes d'exploitation peuvent être téléchargés, montés ou gravés sur un disque optique facilement.

Les systèmes d'exploitation comme Windows 8, 8.1 et 10 sont disponibles en téléchargement sous forme d'image ISO. Ils vous permettront même de monter une image ISO en tant que disque virtuel afin d'assurer le bon fonctionnement du matériel ou du système d'exploitation de l'ordinateur. 

Dans cette situation, il est traité par tous les programmes comme s'il s'agissait d'un disque optique réel placé à l'intérieur du lecteur de disque optique de l'ordinateur. 

## Comment créer un fichier ISO

Si vous cherchez à faire une sauvegarde numérique de n'importe quel disque physique, créer un fichier ISO est une excellente façon de préserver ces données et d'y revenir à tout moment souhaité. 

Cette copie stockée dans un fichier ISO peut être facilement accessible pour installer des logiciels, des jeux et plus encore sur d'autres ordinateurs. Vous pouvez même créer plus de répliques de ce même fichier ISO sans avoir besoin du disque original dont vous l'avez copié. Cela signifie que le partager avec d'autres est un jeu d'enfant !

Vous pouvez facilement créer des images ISO d'un disque physique en utilisant les outils préinstallés qui accompagnent les systèmes d'exploitation comme MacOS et Linux. 

Cependant, ce n'est pas le cas avec le système d'exploitation Windows, vous devrez donc utiliser des applications tierces. Ces applications peuvent être trouvées partout sur Internet, vous pouvez donc choisir celle qui fonctionne le mieux pour vous.

Néanmoins, le système d'exploitation Windows vous donne la possibilité de monter, graver ou copier un fichier ISO sur votre ordinateur complètement gratuitement, donc il a ses avantages. Vous pouvez l'utiliser pour sauvegarder des logiciels, des jeux, des films, et ainsi de suite.

## Comment graver des fichiers ISO

### Windows 8/8.1/10

1. Téléchargez l'image ISO du CD dans un dossier sur votre ordinateur. Je préfère la placer sur mon bureau pour un accès facile et rapide.
2. Ouvrez le dossier.
3. Cliquez avec le bouton droit de la souris --> fichier .iso.
4. Dans le Menu --> Graver l'image disque.

![Image](https://www.freecodecamp.org/news/content/images/2021/02/Screen-Shot-2021-02-18-at-3.40.51-PM.png)

5. L'utilitaire de gravure d'image disque Windows s'ouvrira.  
  
6. Sélectionnez le graveur de disque.  
  
7. Cliquez sur "Graver".

![Image](https://www.freecodecamp.org/news/content/images/2021/02/Screen-Shot-2021-02-18-at-3.39.33-PM.png)

Vous pouvez monter des images ISO dans Mac OS X en utilisant l'Utilitaire de disque, situé dans le répertoire /Applications/Utilities/. 

1. Après avoir lancé l'Utilitaire de disque, naviguez depuis le menu Utilitaire de disque jusqu'à « Ouvrir l'image disque » 
2. Sélectionnez votre fichier ISO. 
3. L'ISO devrait maintenant apparaître monté sur le bureau de Mac OS. 
4. Une fenêtre Sélectionner l'image à graver apparaîtra. 
5. Sélectionnez le fichier .iso que vous souhaitez graver sur un CD/DVD. 
6. Assurez-vous d'avoir un disque inséré dans votre lecteur, puis cliquez sur Graver. 
7. Une fenêtre Utilitaire de disque apparaîtra montrant la progression de l'enregistrement. 
8. Une fois le processus d'enregistrement terminé, l'Utilitaire de disque vérifiera que l'image a été gravée correctement. 
9. Cliquez sur OK pour éjecter le disque.

Graver une image ISO peut ne pas être aussi simple que graver des documents ou de la musique sur un disque, car leur nature est si différente. Graver un fichier ISO sur un CD, un DVD ou un disque Blu-ray signifie assembler le contenu du fichier ISO sur le disque.

Une fois de plus, vous pouvez utiliser l'assistant du système d'exploitation Windows pour graver une image ISO sur un disque physique ou un périphérique USB en double-cliquant sur le fichier ISO. Vous n'aurez pas besoin de tiers pour cela, il suffit de suivre les instructions de base de l'assistant. 

Par exemple, Microsoft Office est disponible sous forme de fichier ISO qui ne peut être gravé ou monté, vous devrez donc absolument le graver directement sur votre ordinateur. En revanche, il existe certains programmes qui nécessitent de graver leur fichier ISO en dehors du système d'exploitation pour être utilisés. 

Ces programmes peuvent être des outils de diagnostic de disque dur à partir desquels votre ordinateur peut démarrer depuis un disque ou une clé USB.

## Comment extraire des fichiers ISO

Extraire le contenu d'un fichier ISO dans un dossier peut être extrêmement utile si vous souhaitez savoir exactement ce qu'il contient ou si vous souhaitez accéder à certains fichiers au sein de l'image ISO. 

C'est exact, graver un fichier ISO sur un disque physique ou un périphérique USB n'est pas votre seule option pour le déverrouiller.

## Tout rassembler : les fichiers ISO dans votre vie quotidienne

Les programmes de compression et de décompression peuvent extraire le contenu d'un fichier ISO dans un dossier afin que vous puissiez parcourir son contenu comme n'importe quel autre dossier sur votre ordinateur. 

Dans le cas de Microsoft Office, par exemple, vous pouvez installer le programme en extrayant uniquement les fichiers d'installation.

ISO n'est pas seulement pour Windows, et il est utile de le mentionner pour aider ceux qui travaillent et construisent sur MacOS.
