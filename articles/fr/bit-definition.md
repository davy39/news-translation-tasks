Un bit, qui est l'abréviation de "binary digit" (chiffre binaire), est la plus petite forme de donnée sur un ordinateur.

Les bits ne peuvent être que 1 ou 0. Si un bit est 1, alors il est "allumé" ou vrai. Et si un bit est éteint, alors il est "éteint", ou faux. Le concept de bits est basé sur [l'Algèbre Booléenne](https://www.freecodecamp.org/news/boolean-definition/).

Les termes bit et byte sont souvent confondus. Un bit est une unité unique, et un byte est un groupe de 8 bits. L'abréviation pour les bits est "b", et pour les bytes c'est "B".

De nos jours, nous mesurons les unités plus grandes comme la taille de stockage en bytes (Mo pour mégaoctets ou Go pour gigaoctets). Mais pour des choses comme la vitesse internet, nous utilisons les bits (100 Mbit/s ou mégabits par seconde).
