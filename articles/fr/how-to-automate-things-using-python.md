Si vous utilisez souvent un ordinateur, vous avez probablement dû effectuer des tâches répétitives. Python peut être utilisé pour automatiser de nombreuses tâches ennuyeuses afin de vous libérer du temps pour d'autres choses.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à automatiser des choses avec Python.

Vous acquerrez des compétences en construisant les projets d'automatisation suivants :

* Envoi d'emails avec les titres de Hacker News
* Téléchargeur de conférences TED
* Extracteur de tableaux depuis des PDF
* Analyseur automatique de CV en masse
* Convertisseur de types d'images
* Construction d'un résumeur automatique de nouvelles

Ce cours a été créé par 1littlecoder. Il a réalisé de nombreux excellents cours par le passé et est un enseignant exceptionnel.
