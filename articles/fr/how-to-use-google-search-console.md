Par Adam Naor

Si vous construisez des produits, vous voulez savoir comment les gens utilisent les produits, comment ils entendent parler de ce que vous construisez et, finalement, comment ils vous trouvent. La Search Console joue un rôle important en vous aidant à suivre et à mesurer ces métriques opérationnelles.

La Console est gratuite. Vous devez simplement prouver à Google que vous possédez un site web. Ensuite, Google vous donnera accès à cette mine d'or de données de recherche.

Je vais vous guider à travers les fonctionnalités principales de la Google Search Console et les décrire en anglais simple. J'espère que cela vous aidera à l'utiliser pour mesurer les points de données qui comptent le plus pour votre produit.

## Google Search Console : Fonctionnalités et fonctionnalités principales

La Google Search Console est un service web de Google qui permet aux webmasters de vérifier l'état d'indexation et d'optimiser la visibilité de leurs sites web. Ci-dessous, une capture d'écran de la partie principale de la Search Console afin que vous puissiez voir une disposition visuelle.

![Image](https://lh4.googleusercontent.com/VmfGjjHEwTjmyBhv3VVoGSN7GfZaZN9_ZNyF4VvTo0JHZMLzEBBncCStxjjzd0z-CqOquS82rgXDsLj1hVsGqiWKQjNvnviU0Lepm3UQoc-0OTEYSPcC8Xl_DtEHB4eL6xQCBVQD)
_Les principaux onglets de l'interface Google Search Console_

Il existe cinq domaines différents de performance de recherche que la Console vous permet de voir rapidement et d'approfondir. Ces 5 domaines sont :

1. Requêtes
2. Pages
3. Pays
4. Apparence de recherche
5. Dates

Je souhaite fournir un aperçu de chacun. Je vais ensuite expliquer comment la performance et le rendu de votre site web dans chaque champ vous raconteront une histoire sur le trafic de votre site web et l'expérience utilisateur.

### Comment utiliser les requêtes dans Google Search Console

Les constructeurs opèrent à tous les niveaux, restent connectés aux détails et audent les données fréquemment. Avant de plonger plus profondément dans la Console, un constructeur voudra savoir ce que les gens tapent dans Google.com pour atteindre la page de destination ou le site web.

En d'autres termes, quelles sont les phrases, mots ou requêtes spécifiques que les gens tapent lorsqu'ils recherchent du contenu concernant votre page ? Quelle est l'intention de recherche ?

La Console permet aux constructeurs de voir les requêtes exactes que les utilisateurs saisissent pour atteindre votre site web. Les informations concernant les requêtes sont extrêmement importantes car elles vous permettent de voir ce que les gens tapent réellement dans leur moteur de recherche avant d'atteindre votre page.

Savoir ce que les gens recherchent est un excellent guide pour allouer des ressources ou optimiser vos pages pour ce que les gens veulent apprendre.

### Comment utiliser les pages dans Google Search Console

Pages est la section de la Console qui vous permet de voir quelles sont vos pages qui sont classées, le nombre de clics que chaque page reçoit et le nombre total d'impressions que la page a obtenues. Par défaut, Google liste vos pages les plus fréquentées en premier.

Si vous ne savez pas quelles pages reçoivent le plus de trafic, vous ne saurez pas comment les gens découvrent votre site web. Les pages peuvent être révélatrices, car vous pourriez penser que votre site web parle du sujet X, mais la plupart des gens vous découvrent en atterrissant sur la page Y.

Lorsque j'ai construit pour la première fois un site web pour travailler depuis chez soi, je voulais créer un site polyvalent pour aider les gens à travailler depuis chez eux.

À ma grande surprise, les pages qui étaient les mieux classées, y compris les supports pour quatre moniteurs et les chaises de bureau rouges, m'ont semblé quelque peu ésotériques.

Néanmoins, c'est ce que les gens voulaient lire et où ils passaient leur temps. Sans la section des pages de la Console, je n'aurais pas pu obtenir cette information.

Vous pouvez également utiliser la Console pour comparer les performances de votre site sur différents appareils et systèmes d'exploitation. Si vous souhaitez savoir si des pages spécifiques sont plus pertinentes pour les utilisateurs sur leur mobile ou sur leur ordinateur, il y a des données pour cela.

Il serait intéressant de découvrir si vous avez beaucoup d'utilisateurs venant de Gmail via leur application de bureau, par exemple, et ensuite de voir s'ils passent plus de temps sur la page que les utilisateurs de Gmail qui sont sur leur téléphone.

### Comment utiliser les pays dans Google Search Console

Cette section sera la plus importante pour les personnes qui s'attendent à et veulent générer du trafic depuis différentes géographies. Si vous êtes un site web basé sur l'éducation, par exemple, vous pourriez vouloir des utilisateurs du monde entier qui peuvent accéder à votre contenu.

D'où viennent les gens peut influencer la langue des pages que vous affichez ou vous aider à mieux interagir avec vos utilisateurs. Si vous êtes un site web qui vend un produit physique, vous pourriez valoriser le trafic en provenance de lieux où vous pouvez réellement expédier.

L'onglet pays est utile pour confirmer votre hypothèse que votre trafic provient de certaines localités.

### Comment utiliser l'apparence de recherche sur Google Search Console

Une partie de votre trafic proviendra de requêtes standard, d'autres sources de trafic proviendront de vidéos ou des "Accelerated Mobile Pages" de Google, plus communément connues sous le nom d'AMP. L'onglet apparence capture comment Google classe l'apparence de votre contenu.

Cet onglet est le moins important à mon avis, car vous ne changerez probablement pas votre comportement en fonction de ce que vous voyez ici.

Cependant, si vous pratiquez le développement de produits agile et souhaitez itérer et voir comment chaque changement impacte les résultats de recherche, cet onglet sera un guide utile.

### Comment utiliser les dates dans Google Search Console

Le dernier onglet est le plus explicite et concerne le temps.

Vous pouvez suivre vos résultats de recherche organique par jour ou voir les tendances en zoomant sur les performances mensuelles, trimestrielles ou annuelles. En général, il est utile de prendre une perspective à long terme sur la performance de recherche, car tout ce que vous faites pour améliorer votre site web prendra probablement de nombreux jours ou semaines pour que Google l'indexe.

Google recherche des sites web performants, utiles aux utilisateurs, rapides, réactifs et bien structurés. Si votre site web offre ces avantages aux utilisateurs – et génère l'engagement des utilisateurs au fil du temps – votre trafic augmentera probablement et l'onglet dates vous aidera à le visualiser.

## Conclusion : Utilisez la Search Console pour construire de meilleurs produits

La Google Search Console est un outil puissant et nécessaire pour examiner, utiliser et explorer les métriques opérationnelles principales de votre site web. La Console vous aidera à comprendre comment les utilisateurs découvrent votre site dans la recherche organique et quelles pages ils visitent sur votre domaine.

En mesurant ces points de données, vous pouvez améliorer votre site web en améliorant les pages individuelles, en améliorant la qualité de votre apparence de recherche et en voyant comment vous vous améliorez au fil du temps.

Google fournit l'accès à la console sans frais supplémentaires afin que vous puissiez construire et développer sans payer par requête.

La console est très puissante et c'est un outil utile à utiliser lors de la construction et du déploiement de tout site sur l'internet public.
