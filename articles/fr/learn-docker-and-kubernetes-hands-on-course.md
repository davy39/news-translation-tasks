Docker et Kubernetes sont des technologies importantes qui peuvent vous aider à mettre à l'échelle vos applications.

Docker est une plateforme pour empaqueter, déployer et exécuter des applications dans des conteneurs. Kubernetes est un système pour gérer des applications conteneurisées sur un cluster de nœuds. Il fournit des mécanismes de base pour le déploiement, la maintenance et la mise à l'échelle des applications.

Nous venons de publier un cours de 6 heures sur la chaîne YouTube freeCodeCamp.org qui vous fournira une introduction pratique à Docker et Kubernetes.

Guy Barrette enseigne ce cours sur les fondamentaux des conteneurs Docker et Kubernetes pour débutants. Guy est un développeur et formateur avec plus de 25 ans d'expérience. Il est un Microsoft MVP, un conférencier fréquent et a été le leader du groupe d'utilisateurs .NET de Montréal pendant plus de 23 ans. Tout cela pour dire qu'il est la personne idéale pour vous enseigner Docker et Kubernetes.

Apprenez à conteneuriser des applications avec la CLI Docker et Visual Studio Code. Apprenez à utiliser les commandes push et pull pour les images de conteneurs sur Docker Hub, un registre de conteneurs, et créez des applications multi-conteneurs en utilisant Docker Compose. Découvrez l'architecture de Kubernetes et son fonctionnement. Exécutez-le localement et utilisez la CLI Kubernetes et Visual Studio Code pour exécuter, déboguer et résoudre les problèmes des applications et services. Apprenez-en davantage sur les pods et les différentes charges de travail disponibles.
