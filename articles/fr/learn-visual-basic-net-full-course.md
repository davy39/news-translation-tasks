Saviez-vous que Visual Basic est actuellement un langage de programmation plus populaire que JavaScript ? C'est selon [l'index TIOBE](https://www.tiobe.com/tiobe-index/), qui est l'un des indicateurs les plus respectés de la popularité des langages de programmation.

Visual Basic est un langage de programmation orienté objet développé par Microsoft. Il permet de créer rapidement et facilement des applications .NET sûres en termes de typage. Certaines utilisations courantes de Visual Basic incluent la création d'applications basées sur Windows, d'utilitaires pour effectuer des tâches spécifiques et l'ajout de fonctionnalités à des applications existantes.

Nous venons de publier un cours complet de Visual Basic pour débutants sur la chaîne YouTube freeCodeCamp.org.

Kevin Drumm a créé ce cours. Kevin est le responsable de l'informatique dans une école au Royaume-Uni. Il a également créé des centaines de tutoriels de programmation.

Dans ce cours, vous apprendrez les concepts de base des langages de programmation de haut niveau, y compris la séquence, la sélection et l'itération. Vous apprendrez à créer une interface utilisateur basée sur des événements et des formulaires pour capturer les entrées, et vous apprendrez à écrire du code pour valider et traiter les données collectées.

Voici les sections de ce cours :

* Bonjour Visual Studio
* Personnaliser l'IDE Visual Studio
* Sortie et Variables
* Types de Données des Variables
* Saisie avec Windows Forms
* Débogage de Code
* Opérateurs Arithmétiques
* Expressions Arithmétiques Complexes
* Sélection avec les Instructions If
* Opérateurs Logiques et Relationnels 1
* Opérateurs Logiques et Relationnels 2
* Select Case
* For Next
* Pratique des Boucles For Next et des Blocs If
* Do While
* Boucles Contrôlées par Condition
* Variables de Tableau

* Recherche Linéaire
* Tableaux à Deux Dimensions
* Tableaux 2D et Boucles Imbriquées
