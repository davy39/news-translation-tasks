Dans le monde d'aujourd'hui, nous dépendons tous fortement de divers produits logiciels qui ont été soigneusement programmés par des développeurs. Mais qu'est-ce que la programmation informatique ?

Dans cet article, je vais fournir une définition de la programmation, ainsi que des exemples de différents types de programmeurs. Je vais également suggérer une liste de cours que vous pouvez suivre pour commencer à apprendre à coder. 

## Définition de la programmation

Un programme informatique se compose de code qui est exécuté sur un ordinateur pour effectuer des tâches particulières. Ce code est écrit par des programmeurs. 

La programmation est le processus qui consiste à donner aux machines un ensemble d'instructions décrivant comment un programme doit être exécuté. Les programmeurs passeront toute leur carrière à apprendre une variété de langages de programmation et d'outils afin de pouvoir construire efficacement des programmes informatiques. 

Les programmeurs commenceront par utiliser un éditeur de code ou un IDE pour écrire ce que l'on appelle le code source. Il s'agit d'une collection de code écrit dans un langage de programmation que d'autres programmeurs peuvent lire.

Le code source doit être converti en langage machine afin que les machines puissent comprendre les instructions et exécuter le programme. Ce processus de conversion du code source en langage machine est appelé compilation. 

Des exemples de langages de programmation compilés seraient C et C++. 

Il existe d'autres langages qui n'utilisent pas de compilateurs. Au lieu de cela, ces langages utiliseront un interpréteur qui lira et exécutera le code.

Des exemples de langages de programmation interprétés seraient JavaScript et PHP. 

Une fois le code exécuté, le programme informatique peut alors s'exécuter. Les différents types de programmes informatiques incluent les traitements de texte, les systèmes de bases de données, les jeux vidéo et les sites web.

Ces programmes informatiques nous permettent d'interagir avec différents appareils et services logiciels comme les téléphones, les sites web et les ordinateurs eux-mêmes. 

## Histoire de la programmation

De nombreux historiens pensent qu'Ada Lovelace a été la première à créer un programme informatique publié au milieu du 19ème siècle. Elle était mathématicienne et a traduit les écrits de Luigi Federico Menabrea sur la machine analytique.

Dans cette traduction, Ada avait ajouté ses propres notes approfondies et créé une méthode pour calculer les nombres de Bernoulli pour la machine analytique. 

Les travaux du mathématicien Alonzo Church ainsi que la machine de Turing ont été les premières introductions des codes informatiques. 

Dans les années 1920, les ordinateurs ont été créés, suivis par les premiers langages de programmation. Au cours du siècle suivant, à mesure que les machines et la technologie continuaient d'évoluer, le nombre de langages de programmation a également augmenté. 

## Types de langages de programmation

Il existe des centaines de langages de programmation. Les développeurs considéreront d'abord les besoins de l'application afin de décider quels langages seraient appropriés à utiliser. 

Voici quelques langages de programmation populaires.  

* Python
* JavaScript
* C/C++
* Java
* C#
* Ruby
* PHP

Certains de ces langages sont principalement utilisés dans un domaine de développement, tandis que d'autres sont des langages de programmation plus polyvalents. 

JavaScript est principalement utilisé dans le développement web et est généralement le premier langage de programmation appris par les développeurs web débutants. JavaScript peut également être utilisé pour le développement mobile et de jeux. 

Python peut être utilisé dans divers domaines comme l'analyse de données, l'apprentissage automatique et le développement web.

Les langages de programmation sont regroupés dans diverses catégories. Voici une liste de certaines de ces catégories. 

* Langage machine – un langage de bas niveau qui se compose de 0 et de 1 (binaire). Les langages de haut niveau sont compilés en code machine afin que le code puisse être exécuté par l'ordinateur. 
* Langage d'assemblage – un langage de bas niveau qui est compilé par un assembleur. Les assembleurs traduisent le code humain en code machine. 
* Langages procéduraux – cette approche passe par une série de procédures avant qu'un programme ne soit exécuté sur l'ordinateur. (Par exemple, Go et Julia)
* Langages de script – ces langages n'ont souvent pas besoin d'être compilés mais plutôt interprétés. Interprété signifie qu'un interpréteur lira et exécutera le code au lieu d'être compilé en code machine. (Par exemple, JavaScript et PHP)
* Langages fonctionnels – cela fonctionne avec l'idée de construire des programmes complexes à travers une collection de fonctions plus petites. (Par exemple, Haskell et Scala)
* Langages orientés objet – cela fonctionne avec l'idée de construire des programmes autour de collections d'objets. (Par exemple, Java et Python)

Il existe de nombreux autres paradigmes logiciels et langages à explorer que nous n'avons pas couverts dans cet article. Mais cela devrait vous donner un bon point de départ dans les divers types de langages de programmation existants.

## Compétences que les programmeurs devraient avoir

Un programmeur professionnel aura une variété de responsabilités lorsqu'il s'agit de construire des programmes informatiques.

Voici quelques-unes des compétences importantes nécessaires en programmation. 

* Écrire un code propre et efficace
* Résolution de problèmes 
* Test des applications
* Correction des erreurs ("bugs") dans le code
* Bonnes compétences en communication et en écoute
* Capacité à bien travailler au sein d'une équipe
* Patience et persévérance 

C'est le travail d'un programmeur de prendre les besoins du client et de créer une application solide et fonctionnelle. Les développeurs passeront beaucoup de temps à rechercher, construire, corriger et tester leur code. 

Après la sortie de l'application, les développeurs passeront du temps à maintenir le produit et à travailler pour l'améliorer. 

## Conclusion

La programmation informatique peut être une carrière très gratifiante, tant sur le plan intellectuel que financier. Les programmeurs doivent être prêts à apprendre de nouvelles choses et à résoudre des problèmes complexes. 

Il existe de nombreux domaines différents de la programmation à explorer, comme le web, le mobile, le développement de jeux ou l'ingénierie des systèmes. Certains domaines nécessiteront une formation formelle, tandis que d'autres sont ouverts aux programmeurs autodidactes.
