React Native vous permet d'utiliser JavaScript pour écrire des applications qui s'exécutent nativement sur Android, iOS et Blackberry. Ce cours d'introduction complet de _codedamn_ vous enseignera les bases de React Native et vous aidera à créer votre première application React Native.

Les sujets abordés dans ce cours sont :

* [Installation de React Native](#heading-installation)
* Structure des fichiers
* Hello World
* Explication de Hello World
* Props
* State
* Feuilles de style et dispositions Flexbox
* Obtenir le texte de l'utilisateur
* Composants non contrôlés vs contrôlés
* Création d'une calculatrice

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=frvXANSaSec) (2,3 heures de visionnage).
