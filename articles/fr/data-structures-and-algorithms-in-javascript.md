Apprenez les structures de données et les algorithmes courants dans ce cours de Beau Carnes. En plus d'apprendre la théorie derrière les structures de données et les algorithmes, vous apprendrez à les implémenter en JavaScript.

Voici les sujets abordés :

* Piles
* Ensembles
* Files d'attente et files d'attente prioritaires
* Arbre binaire de recherche
* Arbre binaire de recherche : parcours et hauteur
* Tables de hachage
* Liste chaînée
* Trie
* Tas
* Graphes : liste d'adjacence, matrice d'adjacence, matrice d'incidence
* Graphes : recherche en largeur d'abord

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=t2CEgPsws3U) (2 heures de visionnage).
