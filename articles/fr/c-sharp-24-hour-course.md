Dans ce cours complet de 24 heures proposé par Tim Corey, vous apprendrez à utiliser C# pour construire une application complète de A à Z.  

Le cours vous guidera à travers tout le processus de conception et de construction d'une application réelle. En utilisant .net et Visual Studio, vous apprendrez à construire une application de suivi de tournoi prête à être lancée. 

Voici quelques-uns des sujets abordés dans ce cours :

* Interfaces
* Email
* SQL
* Fichiers texte
* Stockage et récupération des données App.config
* Programmation Orientée Objet (OOP)

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=wfWxdh-_k_4) (24 heures de visionnage).
