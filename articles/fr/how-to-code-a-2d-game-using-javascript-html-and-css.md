JavaScript peut être un langage de programmation étonnamment efficace pour créer des jeux 2D. Surtout pour les jeux que vous voulez rendre jouables sur presque n'importe quel appareil. 

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à coder un jeu 2D en utilisant JavaScript, HTML, CSS et HTML Canvas.

Frank de Frank's Laboratory a développé ce cours. Frank est un instructeur populaire qui propose toujours des tutoriels super créatifs.

Vous allez construire un jeu entièrement à partir de zéro, incluant tout, de l'animation de sprites aux arrière-plans parallaxes. Le jeu utilise uniquement HTML, CSS et du JavaScript pur, sans frameworks ni bibliothèques. 

![Image](https://www.freecodecamp.org/news/content/images/2022/09/game.gif)
_Vous allez créer ce jeu._

* Configuration JavaScript de base
* Programmation Orientée Objet avec JavaScript
* Création des objets Player et Game
* Boucle d'animation
* Entrées clavier
* Création de projectiles
* Événements périodiques
* Dessiner l'interface utilisateur du jeu
* Classe d'ennemi de base
* Détection de collision entre rectangles
* Affichage du score du jeu
* Conditions de victoire et de défaite
* Compter le temps de jeu
* Arrière-plans parallaxes animés
* Animation de sprites avec JavaScript
* Création d'un mode débogage
* Animation des feuilles de sprites (sprite sheets) d'ennemis
* Classe d'ennemi Night Angler
* Classe d'ennemi Lucky Fish
* Collecter des bonus (power ups)
* Dessiner des projectiles sous forme d'images
* Polices personnalisées et texte du jeu
* Nettoyage du code
* Effets de particules et physique
* Rotation des particules
* Ajustements et corrections
* Classe d'ennemi Hive Whale
* Classe d'ennemi Drone
* Animation d'effet de poussière
* Animation d'effet de feu
* Ajuster la difficulté du jeu
* Que faire ensuite ?
