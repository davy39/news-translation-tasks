Il est probable que vous soyez tombé sur un ou deux fichiers avec l'extension `.rtf`. Si vous vous êtes déjà demandé quel type de fichier c'était et comment l'ouvrir, vous n'êtes pas seul.

Dans cet article, nous allons expliquer ce qu'est un fichier RTF et comment l'ouvrir, l'éditer et le convertir dans différents formats qui vous sont peut-être plus familiers.

## Qu'est-ce que le format RTF ?

RTF signifie Rich Text Format et a été développé pour la première fois par Microsoft en 1987.

Le format RTF a été initialement développé pour faciliter le travail avec des documents texte, quel que soit le logiciel de traitement de texte ou le système d'exploitation que vous utilisez.

Bien que Microsoft ait cessé de maintenir le format en 2008, il a été largement adopté. De nos jours, la plupart des systèmes d'exploitation modernes disposent d'un moyen d'ouvrir les fichiers RTF directement.

## Comment ouvrir et éditer un fichier RTF

Pour ouvrir un fichier au format Rich Text, commencez par double-cliquer dessus si vous êtes sur un ordinateur de bureau ou en tapant dessus si vous êtes sur un appareil mobile.

### Logiciels de traitement de texte sur bureau et en ligne

La plupart des systèmes d'exploitation de bureau sont livrés avec un logiciel capable d'ouvrir des fichiers RTF, comme WordPad sur Windows ou TextEdit sur macOS.

De nombreuses distributions Linux incluent une suite bureautique comme LibreOffice qui peut ouvrir des fichiers RTF. Si ce n'est pas le cas, vous pouvez l'installer via la ligne de commande ou vous rendre sur [https://www.libreoffice.org/](https://www.libreoffice.org/) pour télécharger le package directement.

De plus, la plupart des logiciels de traitement de texte en ligne peuvent ouvrir des fichiers RTF. Google Docs, Dropbox Paper et Office Online sont parmi les logiciels de traitement de texte en ligne les plus populaires, et tous sont capables d'ouvrir des fichiers avec l'extension `.rtf`.

Une fois que vous avez ouvert un fichier RTF en utilisant l'une des méthodes ci-dessus, vous pouvez éditer le fichier comme vous le feriez pour tout autre fichier texte :

### Comment ouvrir un fichier RTF avec un autre programme

Si vous souhaitez ouvrir un autre programme, faites simplement un clic droit sur le fichier et survolez "Ouvrir avec". Ensuite, sélectionnez simplement le programme avec lequel vous souhaitez ouvrir le fichier :

![Image](https://www.freecodecamp.org/news/content/images/2020/10/Peek-2020-10-08-16-03.gif)
_Ouverture d'un fichier RTF avec un autre programme dans Linux Mint_

Remarque : Ce processus peut varier légèrement en fonction de votre système d'exploitation.

### Logiciels de traitement de texte sur mobile

Les appareils mobiles peuvent avoir un peu plus de difficulté à ouvrir les fichiers RTF directement. Néanmoins, il existe de nombreuses options comme Google Docs ou WPS Office pour Android et iOS.

Une fois que vous avez trouvé un logiciel de traitement de texte mobile avec lequel vous êtes à l'aise, il y a de bonnes chances qu'il puisse ouvrir et éditer des fichiers RTF.

## Comment convertir un fichier RTF dans un autre format

La méthode exacte de conversion d'un fichier RTF varie en fonction du logiciel que vous utilisez pour l'ouvrir. Néanmoins, le processus est généralement similaire, quel que soit le logiciel de traitement de texte que vous utilisez.

Voici comment convertir un fichier RTF dans un autre format dans LibreOffice Writer.

### Étape 1 : Sélectionnez "Enregistrer sous"

Avec le fichier ouvert dans LibreOffice Writer comme décrit ci-dessus, cliquez sur "Fichier", puis sélectionnez "Enregistrer sous..." dans le menu déroulant :

![Image](https://www.freecodecamp.org/news/content/images/2020/10/image-11.png)
_"Enregistrer sous..." dans LibreOffice Writer_

### Étape 2 : Sélectionnez un emplacement

Dans le menu contextuel "Enregistrer", sélectionnez un emplacement pour enregistrer votre nouveau fichier.

Vous pouvez également en profiter pour renommer le fichier si vous le souhaitez :

![Image](https://www.freecodecamp.org/news/content/images/2020/10/image-12.png)
_Le menu contextuel "Enregistrer" dans LibreOffice Writer_

J'ai choisi d'enregistrer le fichier sur mon bureau, mais vous pouvez l'enregistrer où vous le souhaitez.

Remarque : Ce menu peut avoir un aspect différent selon votre système d'exploitation.

### Étape 3 : Sélectionnez un nouveau format de fichier

Maintenant, la partie amusante – choisir le format différent dans lequel vous allez enregistrer le fichier.

Ouvrez le menu déroulant dans le coin inférieur droit et sélectionnez un autre format :

![Image](https://www.freecodecamp.org/news/content/images/2020/10/image-13.png)
_Le menu déroulant des formats de fichier dans LibreOffice Writer_

Après cela, cliquez simplement sur "Enregistrer". Votre nouveau fichier dans un autre format sera à l'emplacement que vous avez spécifié. Mieux encore, votre fichier RTF original restera intact là où vous l'avez ouvert.

Maintenant, tout ce que vous avez à faire est d'ouvrir votre nouveau fichier et de continuer à l'éditer.

Mais gardez à l'esprit que tous les formats de fichier ne sont pas aussi universels que le RTF. Si vous partagez votre nouveau fichier avec quelqu'un et qu'il a du mal à l'ouvrir, vous devrez peut-être le convertir dans un autre format.

Néanmoins, maintenant que vous connaissez le processus, il devrait être facile de convertir vos fichiers dans un format avec lequel vous et les autres pouvez travailler.
