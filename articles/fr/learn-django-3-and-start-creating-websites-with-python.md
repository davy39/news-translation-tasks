Django est le Framework web full-stack le plus populaire pour Python. Il aide à créer des applications web rapidement.

Nous venons de publier un cours de 7 heures qui vous apprendra à utiliser la version la plus récente de Django pour créer des applications web. Le cours couvre également pandas, matplotlib, JavaScript, ajax, xhtml2pdf, dropzone.js, et plus encore.

Luke de pyplane a développé ce cours. C'est un développeur logiciel expérimenté qui a créé de nombreux cours populaires.

Dans ce cours, vous apprendrez :

* Les concepts Django (models, views, templates, signals, et plus encore)
* Les dataframes pandas
* L'intégration de matplotlib et seaborn
* L'intégration de pdf
* L'intégration de JavaScript ajax
* dropzone.js pour les fichiers csv
* Travailler avec base64
* Et plus encore !

Ce cours est destiné aux débutants. Le seul prérequis est d'avoir une certaine expérience avec Python.
