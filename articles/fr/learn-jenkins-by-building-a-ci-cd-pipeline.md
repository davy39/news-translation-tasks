Jenkins est un serveur d'automatisation open source qui facilite la construction, les tests et le déploiement de logiciels.

Nous venons de publier un cours vidéo sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Jenkins en vous montrant comment construire un pipeline CI/CD pour une application web.

Gwendolyn Faraday a développé ce cours. Gwen est une développeuse logicielle expérimentée et elle a créé plusieurs cours populaires sur sa propre chaîne ainsi que sur la chaîne freeCodeCamp.

Outre Jenkins, le projet de ce cours utilise également les technologies suivantes :

* Serveurs Debian exécutés sur Linode
* Docker & Dockerhub
* Github
* Quelques commandes en ligne pour configurer les choses

Jenkins peut aider les développeurs à automatiser leur processus de développement logiciel et à améliorer leur productivité. Il peut également aider les utilisateurs à obtenir une nouvelle version de leur projet logiciel plus facilement. Jenkins est un outil important pour créer un pipeline DevOps.

Un pipeline DevOps est un ensemble de processus et d'outils qui permettent la livraison continue des applications logicielles. Le terme "DevOps" est une combinaison des mots "développement" et "opérations". Les pipelines DevOps sont utilisés pour automatiser les phases de construction, de test et de déploiement du cycle de vie du développement logiciel.
