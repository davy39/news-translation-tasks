Pendant des années, les gens m'ont demandé quand nous allions ajouter The Odin Project à freeCodeCamp, comme nous l'avons fait avec d'autres ressources de programmation open source telles que Project Euler et Rosetta Code.

Eh bien, aujourd'hui, je suis ravi d'annoncer que freeCodeCamp intègre le légendaire programme open source The Odin Project dans le navigateur avec notre nouvelle version interactive.

%[https://www.youtube.com/watch?v=MkjsaMk3nKE]

The Odin Project a été créé en 2013 par mon ami et soutien de freeCodeCamp, Erik Trautman. Au fil des années, une communauté open source s'est développée pour maintenir et étendre le projet d'Erik.

freeCodeCamp a enrichi le programme open source pour le rendre interactif dans le navigateur, avec des tests pour évaluer votre code et vous assurer que vous avez compris les concepts clés.

![Image](https://www.freecodecamp.org/news/content/images/2023/12/Cursor_and_The_Odin_Project__Beta____freeCodeCamp_org_--.png)
_Une capture d'écran de la version freeCodeCamp de The Odin Project_

Vous pouvez essayer la version freeCodeCamp de [The Odin Project](https://www.freecodecamp.org/learn/the-odin-project/) ici.

### Voici la liste complète des sections actuellement disponibles

1. Apprendre les bases du HTML
2. Travailler avec du texte
3. Liens et images
4. Projet : Construire une page de recette
5. Apprendre les bases du CSS
6. Exercices sur les bases du CSS
7. Apprendre la spécificité CSS
8. Apprendre le modèle de boîte
9. Apprendre la différence entre Block et Inline
10. Introduction à Flexbox

### À venir en 2024 : encore plus de contenu de The Odin Project

Et nous ne faisons que commencer. Nous travaillons à porter toutes les leçons HTML, CSS et JavaScript de The Odin Project sur freeCodeCamp et à les rendre interactives dans le navigateur.

Comme tout ce que fait freeCodeCamp, tout cela est open source, et nous espérons que le code que nous écrivons pourra être intégré dans le site officiel de The Odin Project.

Si vous souhaitez vivre l'expérience originale de configuration de The Odin Project pour qu'il fonctionne sur votre ordinateur local, vous pouvez consulter le projet original sur [le site de The Odin Project](https://www.theodinproject.com/).

Un grand merci à la communauté de The Odin Project pour continuer à maintenir cette ressource d'apprentissage précieuse pour les développeurs du monde entier.
