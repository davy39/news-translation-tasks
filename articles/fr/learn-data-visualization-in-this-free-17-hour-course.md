Les entreprises deviennent de plus en plus axées sur les données et ces données doivent être visualisées pour être plus faciles à comprendre. La visualisation de données est un domaine en croissance avec de nombreuses opportunités.

Nous avons publié un cours complet de 17 heures sur la visualisation de données, parfait pour toute personne nouvelle dans le domaine. Le cours se concentre sur la visualisation de données en utilisant D3.js, JavaScript et React.

Curran Kelleher a développé ce cours. Curran a précédemment créé l'un des cours de visualisation de données les plus populaires sur Internet et il a maintenant complètement refait et amélioré le cours. Ce cours est le fruit de son expérience approfondie dans le développement de visualisations de données et l'enseignement aux nouveaux développeurs.

Vous apprendrez à conceptualiser, concevoir et construire des visualisations de données interactives avec les technologies Web.
