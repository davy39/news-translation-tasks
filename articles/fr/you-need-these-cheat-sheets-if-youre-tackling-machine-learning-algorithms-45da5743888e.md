Par Shubhi Asthana

Lorsque j'ai commencé à apprendre le Machine Learning (ML) il y a deux ans, j'avais de nombreuses questions sur les algorithmes à utiliser, comment les corrélier avec les ensembles de données, etc. La réponse dépendait de nombreux facteurs comme la taille des données, la sortie attendue et les ressources computationnelles disponibles. J'ai ensuite été introduite aux fiches mémo de ML qui m'ont familiarisée avec les algorithmes, packages et fonctions fréquemment utilisés.

Cet article contient les trois meilleures fiches mémo que je recommanderais à un débutant intéressé par l'identification et l'application des algorithmes de ML à différents problèmes. Étant donné la rapidité avec laquelle ce domaine évolue, les algorithmes tendance progressent également. Il est donc important de comprendre les algorithmes qui aident à couvrir les domaines de l'apprentissage supervisé et non supervisé, de la classification et de la régression, et ainsi de suite.

### **Organigramme des algorithmes SAS**

![Image](https://cdn-media-1.freecodecamp.org/images/hXQAYC2tMyc28bTxzz6P-qpz8q7R8M4rAjBN)

Source : [https://blogs.sas.com/content/subconsciousmusings/2017/04/12/machine-learning-algorithm-use/](https://blogs.sas.com/content/subconsciousmusings/2017/04/12/machine-learning-algorithm-use/)

Le blog SAS lui-même est une excellente lecture. Le lien montre comment utiliser la fiche mémo ainsi que les considérations lors du choix d'un algorithme. La fiche mémo montre un organigramme facile à utiliser corrélant les données aux algorithmes.

### **Fiches mémo Python et Scikit**

La plupart des développeurs travaillent en Python ou en R pour implémenter les algorithmes de ML. Je travaille en Python, et donc les deux fiches mémo suivantes m'ont été très utiles.

![Image](https://cdn-media-1.freecodecamp.org/images/m2pwU-zdlB8OazkzAftqZoFCSq8z3DZlhfjC)

Source : [https://s3.amazonaws.com/assets.datacamp.com/blog_assets/PythonForDataScience.pdf](https://s3.amazonaws.com/assets.datacamp.com/blog_assets/PythonForDataScience.pdf)

La fiche mémo Python a été préparée par DataCamp et peut être utilisée comme référence rapide pour guider à travers les packages Python de ML et les structures de données également.

![Image](https://cdn-media-1.freecodecamp.org/images/4yS6uVoqDm8Vt6u9T-fpMr9PUEgwHWFVOvnO)

Source : [https://s3.amazonaws.com/assets.datacamp.com/blog_assets/Scikit_Learn_Cheat_Sheet_Python.pdf](https://s3.amazonaws.com/assets.datacamp.com/blog_assets/Scikit_Learn_Cheat_Sheet_Python.pdf)

Scikit-learn est une bibliothèque Python open-source qui implémente une large variété de ML, de prétraitement des données, de validation croisée ainsi que de visualisation des algorithmes. Cette bibliothèque fait partie des incontournables pour tout aspirant data scientist, donc je recommande vivement cette fiche mémo.
