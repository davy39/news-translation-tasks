Webpack est un bundler de modules JavaScript et est utile dans de nombreux workflows de développement. Apprenez Webpack avec Colt Steele dans ce cours complet.

Le cours commence par une question simple : "Qu'est-ce que webpack ?" Ensuite, vous apprendrez à installer Webpack, à le configurer et à ajouter des loaders pour CSS, SASS, HTML et les fichiers.

Le cours couvre le cache busting, la configuration d'un serveur de développement et la séparation de vos fichiers de configuration pour le développement et la production. Vous apprendrez à utiliser plusieurs plugins Webpack, notamment clean-webpack-plugin, mini-css-extract-plugin, optimize-css-assets-webpack-plugin, terser-webpack-plugin et html-webpack-plugin.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=MpGLUVbqoYQ) (2 heures de visionnage).
