Apprenez tout ce que vous devez savoir sur Foundation par ZERB dans ce cours accéléré de Shadow Teck. Foundation est un framework CSS front-end réactif qui facilite la conception de sites Web réactifs magnifiques. Ce tutoriel pour débutants couvre tous les composants de Foundation.

Tout d'abord, vous apprendrez à utiliser les modèles HTML5 de Foundation. Ensuite, vous verrez comment construire un site Web de portfolio complet à partir de zéro en utilisant Foundation. Lorsque vous aurez terminé ce cours, vous devriez avoir une compréhension intermédiaire du fonctionnement de Foundation.

Vous pouvez regarder la vidéo complète sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=_drzb7j9Bg4) (2 heures de visionnage).
