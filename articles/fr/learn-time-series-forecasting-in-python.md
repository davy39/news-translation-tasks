Apprendre la prévision des séries temporelles en Python est important car cela peut aider à prédire les tendances futures.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui est une introduction à la prévision des séries temporelles avec Python. Vous apprendrez ce qu'est une série temporelle et comment la décomposer en ses principaux composants tels que la tendance, la saisonnalité et les résidus.

Vous commencerez par construire des modèles de base simples avant d'apprendre des techniques de prévision puissantes telles que ARIMA et ARIMA saisonnier. Vous découvrirez comment prédire les valeurs futures, évaluer vos modèles à l'aide de la validation croisée et incorporer des caractéristiques exogènes pour améliorer les prédictions.

Le cours couvre également comment générer des intervalles de prédiction et sélectionner les métriques d'évaluation les plus appropriées pour vos projets. Vous acquerrez une compréhension claire des principales techniques de prévision et comment les appliquer en pratique.

Marco Peixeiro enseigne ce cours. Il est l'auteur du livre *Time Series Forecasting in Python* publié par Manning Publications.

Cette vidéo est le point de départ idéal pour les débutants souhaitant prédire des données de séries temporelles. Le cours utilise du code Python à 100 % pour couvrir les concepts fondamentaux de la prévision des séries temporelles, y compris :

* définir les données de séries temporelles

* décomposition des séries temporelles

* prévision avec ARIMA

* validation croisée dans les séries temporelles

* utilisation de caractéristiques exogènes

* génération d'intervalles de prédiction

* métriques d'évaluation pour les modèles de prévision

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/fxx_E0ojKrc) (1,5 heure de visionnage).
