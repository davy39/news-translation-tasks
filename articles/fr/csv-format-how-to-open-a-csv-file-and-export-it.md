Vous avez peut-être entendu parler d'un fichier CSV, mais que signifie-t-il ?

Dans cet article, nous allons apprendre ce qu'est un fichier CSV, et comment les ouvrir et les exporter en utilisant Microsoft Excel et Google Sheets.

## Qu'est-ce qu'un fichier CSV ?

CSV signifie Comma Separated Values (Valeurs séparées par des virgules). Il s'agit d'un fichier qui contient des données séparées par des virgules et qui est généralement utilisé dans les tableurs et les bases de données.

Voici un exemple de fichier CSV. Nous avons deux catégories : Nom complet et Spécialité, et les données sont séparées par des virgules.

```txt
Full Name,Major
Dave Wilson,Computer Science Major
Kelly Clarks,Music Major
James Anderson,Business Major
```

Parfois, la séparation des données n'a pas besoin d'être une virgule. Cela pourrait être un espace ou un point-virgule.

Ce séparateur est appelé un délimiteur, mais le plus couramment utilisé est la virgule.

Les fichiers CSV sont très utiles car vous pouvez partager des informations entre différents programmes tant qu'ils peuvent ouvrir le fichier.

Par exemple, je peux créer un fichier CSV dans Microsoft Excel et importer ces données dans mes contacts.

## Comment créer et exporter un fichier CSV dans Microsoft Excel

Microsoft Excel est un outil puissant qui vous permet d'organiser vos données sous forme de tableurs. Les cas d'utilisation les plus courants pour Excel seraient de créer des budgets, trier et stocker des données, et créer des graphiques ou des tableaux.

Voici les étapes pour créer et exporter un fichier CSV dans Excel.

1. Cliquez pour ouvrir Microsoft Excel et allez dans Fichier > Nouveau

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.49.16-AM.png)

2. Ajoutez des données à l'intérieur du tableur

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.52.46-AM.png)

3. Allez dans Fichier > Enregistrer

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.53.58-AM.png)

4. Pour le format de fichier, choisissez `.csv` et cliquez sur Enregistrer

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.55.00-AM.png)

## Comment ouvrir un fichier CSV dans Microsoft Excel

1. Ouvrez Excel et allez dans Fichier > Ouvrir...

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.30.33-AM.png)

2. Trouvez le fichier CSV et cliquez sur Ouvrir

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.32.30-AM.png)

Vous devriez alors voir vos données ouvertes dans un tableur :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.33.30-AM.png)

## Comment créer et exporter un fichier CSV dans Google Sheets

Si vous avez un compte Google, vous pouvez créer et exporter des fichiers CSV en utilisant Google Sheets.

1. Connectez-vous à votre compte Google

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-7.57.54-AM.png)

2. Dans les applications Google, faites défiler vers le bas et cliquez sur Sheets

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.00.22-AM.png)

3. Pour créer une nouvelle feuille Google Sheets, cliquez sur Vide

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.01.16-AM.png)

4. Ajoutez des données et nommez votre feuille de calcul

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.04.23-AM.png)

5. Cliquez sur Fichier > Télécharger et choisissez l'option Valeurs séparées par des virgules

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.08.31-AM.png)

Le fichier devrait alors être téléchargé sur votre machine où vous pouvez l'ouvrir dans n'importe quel programme qui ouvre les fichiers CSV.

## Comment ouvrir un fichier CSV dans Google Sheets

1. Ouvrez une feuille de calcul Google vide

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.12.16-AM.png)

2. Cliquez sur Fichier > Importer

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.12.53-AM.png)

Vous avez quatre options parmi lesquelles choisir.

* Importer un fichier depuis Google Drive
* Importer des fichiers partagés avec vous
* Importer des fichiers récents
* Importer des fichiers depuis votre ordinateur

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.14.51-AM.png)

3. Pour importer un fichier depuis votre ordinateur, cliquez sur Télécharger

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.15.38-AM.png)

4. Sélectionnez un fichier depuis votre ordinateur. Vous devriez alors voir une fenêtre contextuelle comme celle-ci :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.17.20-AM.png)

Vous pouvez choisir de laisser l'option par défaut ou de changer votre type de séparateur.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.17.58-AM.png)

Vous pouvez également choisir de laisser l'option par défaut ou de changer l'emplacement d'importation.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.19.19-AM.png)

Une fois que vous avez terminé, cliquez sur le bouton Importer les données.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.20.00-AM.png)

Vous devriez maintenant voir les informations importées apparaître dans votre feuille de calcul Google.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-17-at-8.20.46-AM.png)

C'est ainsi que vous pouvez créer, ouvrir et exporter des fichiers CSV dans Microsoft Excel et Google Sheets.
