Coder des projets est une étape importante dans l'apprentissage de la programmation. Lorsque vous construisez un projet complet de A à Z, même s'il s'agit d'un projet simple, cela vous aide à comprendre comment tout fonctionne ensemble.

Nous avons publié un cours de 8 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à coder 15 projets simples différents en JavaScript. Le cours a été créé par John Smilga de la chaîne YouTube Coding Addict.

Un code de départ est fourni pour chaque projet afin que vous puissiez vous concentrer uniquement sur le JavaScript et ne pas avoir à vous soucier du HTML ou du CSS.

Voici les projets que vous réaliserez dans le cours :

1. Color Flipper
2. Compteur
3. Avis
4. Barre de navigation
5. Barre latérale
6. Modale
7. Questions
8. Menu
9. Vidéo
10. Défilement
11. Onglets
12. Compte à rebours
13. Lorem Ipsum
14. Épicerie
15. Curseur

Ces projets vous donneront de l'expérience avec toutes les fonctionnalités les plus courantes de JavaScript. À la fin du cours, vous serez prêt à commencer à construire vos propres projets.

Il sera utile d'avoir au moins une expérience de base avec JavaScript avant de suivre ce cours.

Vous pouvez regarder le cours complet de 8 heures sans publicité ci-dessous ou sur la [chaîne YouTube freeCodeCamp.org](https://youtu.be/3PHXvlpOkf4).
