JavaScript peut être utilisé pour développer une variété de types de jeux différents. Et les jeux créés avec JavaScript peuvent fonctionner sur presque n'importe quel appareil disposant d'un navigateur web.

Nous venons de publier un cours de développement de jeux JavaScript de 10 heures sur la chaîne YouTube freeCodeCamp.org. Vous apprendrez à créer des jeux 2D avec HTML, CSS et du JavaScript vanilla pur, sans utiliser de frameworks ni de bibliothèques !

Frank Dvorak a développé ce cours. Il a créé de nombreux cours populaires sur JavaScript sur diverses plateformes. Frank se spécialise dans le codage créatif avec JavaScript vanilla et HTML canvas.

Frank vous enseignera étape par étape en expliquant chaque technique sur une petite base de code autonome, puis il vous montrera comment utiliser tout ce que vous avez appris pour créer un jeu final unique.

![Image](https://www.freecodecamp.org/news/content/images/2022/07/jsgames.gif)
_Vous créerez ce jeu dans le cours._

Ce cours est composé de 9 projets distincts. Dans cette série de projets, vous apprendrez tout ce dont vous avez besoin pour créer vos propres jeux animés 2D, de l'animation de sprites à la gestion d'état.

Voici les projets et sections présentés dans ce cours :

* Projet 1 : Techniques d'animation de sprites en JavaScript vanilla
* Projet 2 : Arrière-plans parallaxes
* Projet 3 : Modèles de mouvement des ennemis
* Détection de collision entre rectangles
* Détection de collision entre cercles
* Projet 4 : Animations de collision à partir d'une feuille de sprites
* Projet 5 : Jeu de tir à la cible
* Projet 6 : Variété d'ennemis
* Projet 7 : Jeu à défilement latéral avec support mobile
* Projet 8 : Gestion d'état
* Projet 9 : Jeu final de course sans fin avec toutes les fonctionnalités

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/GFO_txvwK_c) (10 heures de visionnage).
