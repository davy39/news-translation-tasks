Un développeur web basé à Berlin — qui code en JavaScript pour gagner sa vie — a décidé de passer une journée entière sans JavaScript.

Admettons-le — dans un monde fou où la page web moyenne fait 2,4 mégaoctets — [la même taille que le jeu Doom de 1993](https://fcc.im/2rClKqV) — désactiver complètement JavaScript peut sembler une chose sensée à faire.

Voici ses principales observations après avoir désactivé JavaScript dans le navigateur pendant une journée :

* Le web est vraiment rapide sans JavaScript
* Il n'y a aucune publicité. Pas de vidéos en lecture automatique ni de pop-ups "donnez-nous votre adresse e-mail".
* Mais… beaucoup de sites web ne fonctionnent pas du tout. Les images ne se chargent pas, et les formulaires ne peuvent pas être soumis.

Voici à quoi ressemblaient plusieurs sites web populaires qu'elle a visités avec JavaScript désactivé :
