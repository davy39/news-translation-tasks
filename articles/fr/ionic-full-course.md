Apprenez à utiliser Ionic 4 dans ce cours complet pour débutants d'Awais Mirza. Ionic Framework est l'outil gratuit et open source pour développer des applications mobiles de haute qualité pour iOS, Android et le web, le tout à partir d'une seule base de code JavaScript.

Dans ce tutoriel, vous apprendrez à créer une application mobile avec Ionic et Angular à partir de zéro. Vous apprendrez les concepts de base d'Ionic et de l'Ionic CLI.

Voici quelques-uns des sujets abordés :

* [Installation d'Ionic](#heading-installation)
* Structure du projet
* Émulation iOS et Android
* Routage et chargement paresseux
* Feuille d'action
* Composants
* Alerte ION et documentation
* Ionic Native
* Débogage
* Plugins HTTP natifs
* Plugins SMS natifs
* Stockage natif

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=AvbuIRg8_Jg) (3 heures de visionnage).
