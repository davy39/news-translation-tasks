Le sens du mot "variable" dépend du domaine dans lequel il est utilisé. 

En programmation, une variable est une donnée particulière qui contient une valeur. Selon la configuration, cette valeur peut changer ou rester fixe. 

Par exemple, en JavaScript, vous pouvez implémenter une variable pour qu'elle change au fil du temps avec les mots-clés "var" et "let". Mais si vous le souhaitez, vous pouvez implémenter une variable pour qu'elle ne change pas avec le mot-clé "const".

Dans cet article, le type de variable que nous allons examiner n'est pas celui de la programmation, mais celui que vous rencontrerez dans la recherche. Plus précisément, nous allons examiner les différences entre les deux principaux types de variables en recherche : les variables dépendantes et indépendantes.

Mais avant d'examiner les différences entre les variables dépendantes et indépendantes, nous devons d'abord comprendre ce qu'est une variable.

## Qu'est-ce qu'une variable en recherche ?
Si vous mènez une recherche, vous allez mesurer beaucoup de valeurs. Donc, en recherche, une variable est tout ce que vous essayez de mesurer. Cela pourrait être l'âge, la température, la longueur, la hauteur, la masse, le poids, ou toute autre chose qui peut avoir une valeur.

De plus, vous allez mesurer ces variables dans différentes unités - centimètres (cm), mètres (m), grammes (g), kilogrammes (kg), et bien d'autres. 

Ces unités ne peuvent pas être négligées, mais en ce qui concerne les variables, qu'elles soient dépendantes ou indépendantes, ce sont les valeurs qui comptent vraiment.


## Quelles sont les variables dépendantes et indépendantes ?
Les variables dépendantes et indépendantes dépendent du fait qu'une variable détermine le résultat de l'autre ou non.

Une variable dépendante est une variable dont les changements et son résultat dépendent d'une autre variable. Dans presque tous les cas, la variable dont dépend la variable dépendante est une variable indépendante.

Les variables dépendantes sont également appelées variables de réponse ou de résultat car elles représentent le résultat des valeurs que vous mesurez. C'est-à-dire ce que vous enregistrez après avoir manipulé les variables indépendantes.

Une variable indépendante est une variable dont le résultat ou les changements ne dépendent pas d'une autre variable. C'est l'exact opposé de la variable dépendante, du moins selon ce que le nom implique.

Les variables indépendantes sont également appelées variables prédictives car vous pouvez les utiliser pour prédire le résultat d'une variable dépendante. C'est-à-dire que lorsque vous manipulez des variables indépendantes, elles peuvent vous donner le résultat d'une variable dépendante.


## Quelles sont les différences entre les variables dépendantes et indépendantes ?
| **Base** | **Variable dépendante** | **Variable indépendante** |
|-----------| ----------- | ----------- |
|**Type**| C'est la variable "réponse" | C'est la variable "effet" |
|**Résultat**| Le résultat dépend d'une autre variable (généralement la variable indépendante) | Le résultat ne dépend pas d'une autre variable |
|**Changements** | Cette variable change au fil du temps. Considérez la variable dépendante comme une variable que vous déclarez avec le mot-clé "let" ou "var" en JavaScript. Vous pouvez la changer plus tard. | Cette variable ne change jamais. Considérez la variable indépendante comme la variable que vous déclarez avec le mot-clé "const" en JavaScript. Elle est fixe sauf si vous changez explicitement la valeur. |
|**Manipulation**| Les variables dépendantes ne peuvent pas être manipulées car leur valeur dépend de la variable indépendante. | Les variables indépendantes peuvent être manipulées pour déterminer le résultat d'une variable dépendante. |
|**Position sur un graphique**| Les variables dépendantes sont placées sur l'axe des y (axe vertical) sur un graphique | Les variables indépendantes sont placées sur l'axe des x (axe horizontal) sur un graphique |


## Comment identifier une variable dépendante d'une variable indépendante
Nous avons examiné ce que sont les variables, ce que sont les variables dépendantes et indépendantes, et les différences exactes entre les variables dépendantes et indépendantes. 

Mais comment différencier exactement une variable dépendante d'une variable indépendante ? Nous allons examiner deux expériences ou exemples :

- comment la vitamine A aide les mères à produire du lait

- le niveau de lumière auquel les insectes nocturnes (insectes actifs la nuit) sont attirés

Pour la première expérience, les sources de vitamine A que la mère absorbe à partir d'aliments comme l'huile de poisson et les légumes verts sont la variable indépendante. C'est parce que le chercheur peut changer [diminuer ou augmenter] la quantité donnée aux mères. La façon dont le corps de la mère réagit en produisant plus de lait est la variable dépendante.

Pour la deuxième expérience, le niveau de lumière est la variable indépendante parce que le chercheur peut le changer. La façon dont les insectes nocturnes réagissent à ce niveau de lumière est la variable dépendante.
