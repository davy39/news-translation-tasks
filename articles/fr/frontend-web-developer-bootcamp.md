Il y a beaucoup d'emplois disponibles pour les développeurs web frontend. Et vous pouvez apprendre toutes les compétences nécessaires sans payer quoi que ce soit.

Vous connaissez probablement déjà le [curriculum interactif gratuit](https://freecodecamp.org) de freeCodeCamp qui vous enseignera les compétences nécessaires pour devenir un développeur web fullstack. Mais nous avons également de nombreuses vidéos qui vous enseigneront les compétences en développement web sur la chaîne YouTube freeCodeCamp.org.

Et nous venons de publier un cours vidéo bootcamp de 21 heures qui vous enseignera les compétences de base nécessaires pour devenir un développeur web frontend. Le cours vous enseignera JavaScript, HTML et CSS.

Zach Gollwitzer enseigne ce cours. Zack est un ingénieur logiciel expérimenté et un développeur fullstack. Il est également un excellent instructeur.

Dans ce cours massif, vous passerez de aucune expérience en codage à avoir les compétences essentielles d'un développeur web frontend. Vous apprendrez diverses technologies de développement web et créerez quelques projets en cours de route.

Voici les sections couvertes dans ce cours massif :

* Votre premier programme JavaScript
* Qu'est-ce que JavaScript ?
* Variables JavaScript
* Défis JavaScript de base
* Opérateurs JavaScript
* Plus de défis JavaScript
* Fonctions, conditionnelles, boucles JavaScript
* Problèmes de pratique JavaScript pour débutants
* Méthodes intégrées JavaScript
* Fonctions de rappel
* Primitives JavaScript
* Dates JavaScript
* Expressions régulières
* Méthodes de chaîne JavaScript
* Méthodes de tableau JavaScript
* Utilitaires mathématiques avec JavaScript
* Types d'erreurs et autres concepts JavaScript
* 10 problèmes de pratique JavaScript pour débutants/intermédiaires
* Cours accéléré HTML
* Défis HTML "Page à propos de moi"
* Modèle d'objet de document (DOM)
* Défis HTML + JavaScript "Générateur de mèmes"
* Cours accéléré CSS
* Le modèle de boîte CSS
* Propriétés CSS les plus courantes
* Défis CSS Frontend Mentor "Carte de tarification"
* Introduction au design web réactif
* Cours accéléré CSS Flexbox
* Défis CSS Flexbox Frontend Mentor "Témoignages"
* Réflexions finales et prochaines étapes

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/zJSY8tbf_ys) (21 heures de visionnage).
