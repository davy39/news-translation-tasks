Apprenez les tests d'intrusion réseau dans ce cours vidéo complet de The Cyber Mentor. Ce cours enseigne tout ce que vous devez savoir pour commencer avec le hacking éthique et les tests d'intrusion. Vous apprendrez les compétences pratiques nécessaires pour travailler dans ce domaine. 

Tout au long de ce cours gratuit et complet pour débutants, vous développerez un laboratoire Active Directory sous Windows, le rendrez vulnérable, le piraterez et le corrigerez. Nous aborderons tout, des côtés rouge/bleu à la rédaction de rapports de tests d'intrusion.

Voici quelques-uns des sujets abordés :

* Linux
* Python
* OSINT Passif
* Outils et Tactiques de Scanning
* Énumération
* Exploitation
* Shells
* Bourrage d'identifiants
* Construction d'un Lab AD
* Empoisonnement LLMNR
* Craquage NTLMv2 avec Hashcat
* Relais NTLM
* Usurpation de Token
* Pass the Hash
* PsExec
* MS17-010
* GPP/cPasswords
* Kerberoasting
* Transfers de Fichiers
* Pivoting
* Rédaction de Rapports
* Conseils de Carrière

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=3Kq1MIfTWCE) (15 heures de visionnage).
