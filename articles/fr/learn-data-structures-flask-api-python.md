Les gens apprennent souvent les structures de données hors contexte. Il peut être utile de voir comment elles sont utilisées dans un projet réel.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous aidera à apprendre les concepts fondamentaux des structures de données en construisant une API réelle avec Python et Flask.

Ce cours a été développé par Georgio Tunson. Il a un talent pour aider les gens à comprendre les structures de données de manière pratique.

Le seul prérequis pour ce cours est une compréhension de base de Python. Vous apprendrez à installer Python et Flask, à configurer une base de données et à implémenter des routes API en utilisant de nombreuses structures de données populaires.

Voici les sujets abordés dans ce cours :

* Installer Python et Flask
* Configuration de la base de données et squelette de l'API
* Installer le navigateur de base de données
* Liste chaînée
* Installer Postman
* Table de hachage
* Arbre binaire de recherche
* Piles et files d'attente

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=74NW-84BqbA) (3 heures de visionnage).
