Terraform est un outil logiciel open source d'infrastructure en tant que code qui facilite la configuration programmatique de l'infrastructure sur une variété de fournisseurs de services cloud.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Terraform et Azure pour configurer un environnement de développement.

Nous avons précédemment publié un cours similaire avec AWS au lieu d'Azure.

Derek Morgan a créé ce cours. Il a créé de nombreux cours pour préparer les étudiants à une variété de certifications techniques.

Ce cours est conçu pour vous aider à apprendre les bases de Terraform rapidement avec un projet pratique qui peut servir de cadre pour vos propres projets !

Ce projet vous guidera à travers les bases de Terraform alors que vous utiliserez Visual Studio Code (sur Windows, Mac ou Linux !) pour déployer des ressources Azure et une machine virtuelle Azure à laquelle vous pourrez vous connecter via SSH pour avoir votre propre environnement redéployable qui sera parfait pour vos futurs projets !

Voici les sections couvertes dans ce cours :

* Bienvenue et Installation
* Initialisation du Fournisseur Terraform
* Un Groupe de Ressources
* Un Réseau Virtuel et Référencement d'autres Ressources
* État de Terraform
* Terraform Destroy
* Un Sous-réseau
* Un Groupe de Sécurité
* Associations de Groupes de Sécurité
* Une IP Publique
* Une Interface Réseau
* Une Paire de Clés
* Données Personnalisées
* Scripts de Configuration SSH
* Le Provisionneur
* Sources de Données
* Sorties
* Variables
* Précédence des Variables
* Conditionnelles
