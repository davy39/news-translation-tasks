Avez-vous déjà entendu parler de "Netflix and Chill" ? Essayez plutôt "Netflix and Code". Codez votre propre clone de Netflix !

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un clone de Netflix en utilisant Django et Tailwind CSS.

Django est un framework web gratuit et open-source basé sur Python qui permet aux développeurs de créer des back-ends de sites web en suivant le modèle architectural modèle–template–vue. Tailwind CSS est un framework CSS populaire qui simplifie le stylisme des sites web.

Oliver Otchere a créé ce cours. Oliver est un développeur logiciel expérimenté qui a créé de nombreux cours techniques.

Créer un clone de Netflix est une manière amusante d'améliorer vos compétences en programmation. L'application que vous créerez dispose d'un système d'authentification complet. Elle permet aux utilisateurs de créer des profils et de sélectionner des vidéos à regarder.

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=gbyYXgiSgdM) (2 heures de visionnage).
