Avez-vous déjà essayé de consulter votre historique de navigation ? Peut-être souhaitez-vous vérifier ce restaurant que vous avez recherché hier soir (mais dont vous avez oublié le nom), ou vous voulez revisiter un site de voyage que vous avez consulté il y a quelques jours. Cela peut être super utile.

Mais, d'un autre côté, que faire si vous ne voulez pas laisser une trace des sites que vous avez visités sur votre ordinateur ? Peut-être que votre conjoint ou partenaire découvrira que vous avez recherché leur cadeau d'anniversaire (à la dernière minute, en plus !), ou vous avez visité un site que vous préféreriez garder privé et personnel.

Vous pourriez également utiliser un ordinateur public et ne pas vouloir que des inconnus voient votre historique. C'est bien compréhensible.

Quelle que soit la raison, il est bon de savoir comment consulter - et effacer, ou supprimer - votre historique de navigation. Et c'est assez facile à faire dans différents navigateurs. Alors, examinons cela.

## Comment effacer votre historique dans Chrome

Chrome stocke les sites que vous avez visités au cours des 90 derniers jours dans votre historique de navigation. Si vous utilisez le mode incognito (que vous pouvez lire ici), il ne stockera pas ces sites. Mais tout ce que vous avez consulté dans un navigateur Chrome régulier apparaîtra là.

De plus, **quelque chose à noter** : si vous avez synchronisé tous vos appareils (ordinateur portable, iPhone, tablette, etc.), cet historique sera visible sur tous les appareils. De même, lorsque vous l'effacez, il sera effacé de tous les appareils.

### Effacer tout votre historique de navigation

Sur votre ordinateur, ouvrez le navigateur Chrome. Dans le coin supérieur droit, vous verrez trois points verticaux. Cliquez dessus pour ouvrir ce menu, et sélectionnez l'onglet "Historique" parmi les options.

![Image](https://www.freecodecamp.org/news/content/images/2019/10/Chrome-browsing-history.png)
_Comment accéder à votre historique de navigation dans Chrome_

Une fois que vous avez cliqué sur cette option, vous serez redirigé vers un résumé de votre historique de navigation récent. Si vous voulez simplement le consulter, c'est parfait.

![Image](https://www.freecodecamp.org/news/content/images/2019/10/chrome-history-list.png)
_Votre historique de navigation prêt pour votre consultation (ou suppression) plaisir._

Si vous voulez l'effacer, cliquez simplement sur l'onglet "Effacer les données de navigation" à gauche. Vous serez redirigé vers cet écran (ci-dessous), où vous avez la possibilité d'effacer les données des dernières heures, 24 heures, 7 jours, 4 semaines ou tout le temps. Choisissez votre période et les catégories que vous souhaitez effacer :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/chrome-browsing-history-clear-data.png)
_Cliquez sur le menu déroulant ("Période") pour choisir votre période. Cochez et décochez les cases en fonction de vos besoins._

Comme vous pouvez le voir sur la capture d'écran ci-dessus, Chrome stocke non seulement votre historique de navigation, mais aussi les cookies des sites que vous visitez et les fichiers mis en cache.

Une fois que vous avez effacé ces données, elles n'apparaîtront plus dans votre historique. Chrome n'autocomplétera pas non plus ces sites pour vous si vous commencez à les taper dans la barre d'adresse.

### Effacer une partie de votre historique

Que faire si vous voulez simplement effacer quelques sites, mais garder le reste ? C'est possible aussi.

Ouvrez Chrome, cliquez sur ces trois points verticaux, choisissez "Historique". Même processus jusqu'à présent. Maintenant, sur cette première page d'historique, au lieu de cliquer sur "Effacer les données de navigation" à gauche, sélectionnez/cochez simplement les cases pour ce que vous voulez supprimer :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/chrome-clear-specific-history.png)
_Il suffit de sélectionner les sites que vous souhaitez supprimer._

Une fois que vous avez sélectionné tous les sites que vous souhaitez effacer, cliquez simplement sur "Supprimer" dans le coin supérieur droit. Seuls ces sites seront supprimés de votre historique.

## Comment effacer votre historique dans Firefox

Pour effacer votre historique dans Firefox, le processus est assez similaire à Chrome - les boutons ont juste un aspect un peu différent.

### Effacer tout l'historique de navigation

Ouvrez votre navigateur Firefox, et cliquez sur l'onglet bibliothèque en haut à droite (il ressemble à quelques livres penchés ensemble) :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/firefox-history-2.png)
_Une fois que vous avez sélectionné "Historique", une nouvelle fenêtre s'ouvrira au même endroit._

Sélectionnez l'onglet "Historique", puis l'option "Effacer l'historique récent" dans le menu qui remplacera le précédent.

Vous verrez cette boîte apparaître :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/firefox-clear-history-tab.png)
_Prêt à tout supprimer !_

Et vous remarquerez qu'il y a un menu déroulant "Période à effacer". Vous pouvez effacer votre historique des dernières heures, deux heures, quatre heures, aujourd'hui, ou tout le temps.

Vous pouvez également sélectionner quelles informations vous souhaitez effacer, similaire à Chrome.

Maintenant, soyez prêt : si vous sélectionnez simplement "Effacer maintenant" en bas à droite, il effacera tout votre historique, sans poser de questions.

### Effacer une partie de votre historique

Si vous souhaitez sélectionner les éléments à effacer, le processus est un peu différent. Cliquez sur cet onglet de bibliothèque, sélectionnez "Historique" puis "Afficher tout l'historique" en bas.

Cette boîte apparaîtra :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/firefox-show-all-history.png)
_Vous pouvez sélectionner l'historique que vous souhaitez voir (aujourd'hui, hier, etc.) via la barre latérale de gauche._

Ensuite, vous verrez votre historique. Si vous n'avez que quelques sites, vous pouvez simplement cliquer en maintenant la touche ctrl (ou clic droit) sur le site que vous souhaitez supprimer.

Lorsque vous maintenez la touche ctrl enfoncée et cliquez sur le site, un menu apparaîtra. Il suffit de sélectionner "oublier ce site" dans ce menu et Firefox le supprimera de votre historique.

![Image](https://www.freecodecamp.org/news/content/images/2019/10/firefox-forget-site.png)
_Ctrl+clic (ou clic droit) pour sélectionner un site. Un par un._

Tout l'historique associé à ce site (cookies, cache, navigation et connexions, etc.) sera supprimé. Hourra !

## Comment effacer votre historique dans Safari

Enfin, voyons comment effacer votre historique dans Safari.

### Effacer tout votre historique de navigation

Une fois de plus, le processus est assez similaire à Chrome et Firefox. Mais dans Safari, vous cliquez simplement sur l'onglet "Historique" en haut de votre menu de navigateur. Dans le menu déroulant, sélectionnez "Effacer l'historique" :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/Safari-clear-history.png)
_Facile comme bonjour - il suffit de cliquer sur l'onglet "Historique" puis sur "Effacer l'historique"._

Une boîte apparaîtra qui vous demandera si vous souhaitez effacer tout votre historique (ce qui inclut les caches, etc.) et pour quelle période (dernière heure, aujourd'hui, aujourd'hui et hier, tout l'historique).

![Image](https://www.freecodecamp.org/news/content/images/2019/10/Safari-clear-all.png)
_Options similaires._

Sélectionnez l'option appropriée, et cliquez sur "Effacer l'historique". Boom, l'historique est effacé pour cette période (ou tout le temps).

### Effacer une partie de votre historique

Encore une fois, vous ne voulez peut-être pas tout effacer. Juste un site. Tout à fait faisable.

Cliquez sur l'onglet "Historique" et sélectionnez "Afficher tout l'historique" en haut du menu déroulant. Tous les sites que vous avez consultés seront listés par jour.

Si vous voulez choisir un site spécifique à supprimer, il suffit de cliquer en maintenant la touche ctrl (ou clic droit) sur ce site, et de sélectionner "Supprimer" dans le menu déroulant :

![Image](https://www.freecodecamp.org/news/content/images/2019/10/safari-select-history.png)
_Similaire à Firefox - ctrl+clic ou clic droit pour sélectionner les sites à supprimer._

Seul ce site sera supprimé de votre historique de navigation.

Maintenant, vous savez comment supprimer votre historique de navigation - tout ou partie - de certains des principaux navigateurs utilisés aujourd'hui.
