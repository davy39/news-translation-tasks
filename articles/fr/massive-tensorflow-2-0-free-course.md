TensorFlow est l'une des plateformes de machine learning les plus populaires—et elle est complètement open source. Avec TensorFlow 2.0, il n'a jamais été aussi facile de construire et de déployer des modèles de machine learning.

Nous avons publié un cours de 7 heures sur TensorFlow 2.0 sur la chaîne YouTube de freeCodeCamp.org. Le cours est conçu pour les programmeurs Python qui cherchent à améliorer leurs connaissances et compétences en machine learning et en intelligence artificielle.  

Non seulement ce cours vous apprendra à utiliser TensorFlow, mais il vous donnera également un excellent aperçu du machine learning et de l'intelligence artificielle.

Le créateur de ce cours est Tim Ruscica, connu pour sa populaire chaîne YouTube "Tech With Tim". Tout au long de huit modules, Tim couvre les concepts et méthodes fondamentaux en machine learning et en intelligence artificielle comme :

* les algorithmes d'apprentissage de base, 
* le deep learning avec les réseaux de neurones, 
* la vision par ordinateur avec les réseaux de neurones convolutifs, 
* le traitement du langage naturel avec les réseaux de neurones récurrents, 
* et l'apprentissage par renforcement. 

Pour accompagner la partie vidéo de ce cours, il y a six fichiers Jupyter Notebook riches en informations. Ces fichiers contiennent des notes détaillées, des instructions et des diagrammes. Ils incluent également tout le code utilisé dans le cours afin que vous puissiez facilement essayer le code vous-même. Et vous pouvez accéder aux fichiers sur Google Colaboratory, ce qui vous permet d'exécuter tout le code dans votre navigateur.

Après avoir terminé ce cours, vous aurez une connaissance approfondie des techniques de base en machine learning et en IA, et vous aurez les compétences nécessaires pour appliquer ces techniques à vos propres ensembles de données.	

Voici une répartition de chaque module.

### Module 1 : Fondamentaux du Machine Learning

Le premier module couvre la différence entre l'intelligence artificielle, les réseaux de neurones et le machine learning. Les fondamentaux du machine learning présentés dans ce module fourniront les bases pour le reste du cours.

### Module 2 : Introduction à TensorFlow

Ce module fournit une introduction générale à TensorFlow. Vous apprendrez ce qu'est un Tensor et découvrirez les formes et la représentation des données. Vous apprendrez également comment TensorFlow fonctionne à un niveau plus bas. 

Bien que vous puissiez créer des modèles de machine learning sans savoir comment tout fonctionne, une compréhension plus approfondie facilite l'ajustement des modèles et l'obtention des meilleurs résultats.

### Module 3 : Algorithmes d'Apprentissage de Base

Vous apprendrez quatre des algorithmes fondamentaux de machine learning. Chacun des algorithmes sera appliqué à des problèmes et ensembles de données uniques.

Les algorithmes couverts sont :

* Régression linéaire
* Classification
* Clustering
* Modèles de Markov cachés

### Module 4 : Réseaux de Neurones avec TensorFlow

Dans ce module, vous apprendrez comment fonctionnent les réseaux de neurones et les mathématiques qui les sous-tendent. Vous apprendrez la descente de gradient, la rétropropagation et comment l'information circule à travers un réseau de neurones.

Dans la deuxième partie du module, vous verrez comment créer un réseau de neurones avec Keras pour classer des articles de vêtements.

### Module 5 : Vision par Ordinateur Approfondie - Réseaux de Neurones Convolutifs

Ce module vous apprendra à utiliser un réseau de neurones convolutif pour effectuer la classification d'images et la détection/reconnaissance d'objets.

Vous apprendrez les concepts suivants :

* Données d'image
* Couches convolutives
* Couches de pooling
* Architectures de CNN

### Module 6 : Traitement du Langage Naturel avec les RNN

Le traitement du langage naturel (NLP pour Natural Language Processing) est une discipline informatique qui traite de la communication entre les langues naturelles (humaines) et les langues informatiques. Un exemple courant de NLP est quelque chose comme la vérification orthographique ou l'autocomplétion. 

Ce module introduit un nouveau type de réseau de neurones appelé réseau de neurones récurrent (RNN pour Recurrent Neural Network). Ces réseaux sont souvent utilisés pour le NLP.

Vous apprendrez à utiliser un RNN pour l'analyse de sentiment et la génération de caractères.

### Module 7 : Apprentissage par Renforcement avec Q-Learning

Dans ce module, vous apprendrez l'apprentissage par renforcement.

Cette technique est différente de nombreuses autres techniques de machine learning couvertes précédemment dans le cours. Plutôt que de nourrir notre modèle de machine learning avec des millions d'exemples, nous laissons notre modèle générer ses propres exemples en explorant un environnement.

Vous apprendrez à créer un modèle de machine learning en utilisant l'apprentissage par renforcement.

### Module 8 : Conclusion et Prochaines Étapes

Dans le module final, vous apprendrez les prochaines étapes pour en savoir plus sur TensorFlow et le machine learning.
