Java est le langage de programmation le plus populaire selon l'index TIOBE. Apprenez à programmer en Java dans ce cours complet de Tim Ruscica.

Ce cours complet sur Java est destiné aux débutants absolus. Aucune expérience préalable en programmation n'est requise.

Voici les sujets abordés dans le cours.

* Variables et types de données
* Opérateurs de base
* Entrées et Scanners
* Conditions et booléens
* If/Else/Else If
* Instructions imbriquées
* Tableaux
* Boucles For
* Boucles For (Suite)
* Boucles While
* Ensembles et listes
* Maps et HashMaps
* Exemple de Maps
* Introduction aux objets
* Création de classes
* Héritage
* Que signifie STATIC ?
* Surcharge de méthodes et comparaisons d'objets
* Classes internes
* Interfaces
* Enums

Vous pouvez regarder le cours vidéo complet sur la chaîne [freeCodeCamp.org YouTube](https://www.youtube.com/watch?v=GoXwIVyNvX0) (4 heures de visionnage).
