## Comment installer les bibliothèques nécessaires pour Matplotlib

### Option 1 : Comment installer Matplotlib directement avec pip install

1. Ouvrez une fenêtre de terminal et tapez
2. pip install matplotlib
3. pip install numpy
4. pip install pandas

### Option 2 : Comment installer Anaconda

Téléchargez Anaconda, qui contiendra tous les packages dont nous avons besoin. Voici [un tutoriel vidéo qui vous guide à travers le processus](https://youtu.be/YJC6ldI3hWk).
