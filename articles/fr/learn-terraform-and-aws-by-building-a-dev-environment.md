Terraform est un outil logiciel open source d'infrastructure en tant que code qui facilite la configuration programmatique de l'infrastructure sur divers fournisseurs de services cloud.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Terraform et AWS pour configurer un environnement de développement.

Derek Morgan a créé ce cours. Il a créé de nombreux cours pour préparer les étudiants à diverses certifications techniques.

Ce cours est conçu pour vous aider à apprendre rapidement les bases de Terraform avec un projet pratique qui peut servir de cadre pour vos propres projets !

Ce projet vous guidera à travers les bases de Terraform alors que vous utiliserez Visual Studio Code (sur Windows, Mac ou Linux) pour déployer des ressources AWS et une instance EC2 à laquelle vous pourrez vous connecter via SSH pour avoir votre propre environnement redéployable que vous pourrez utiliser pour des projets futurs.

* Terraform Destroy
* Un sous-réseau et les références
* Un IGW et Terraform fmt
* Une table de routage
