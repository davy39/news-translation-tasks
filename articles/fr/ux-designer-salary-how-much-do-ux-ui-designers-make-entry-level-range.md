Dans cet article, je vais fournir des définitions pour le design UI/UX ainsi que des informations sur les salaires de départ à travers le monde. 

## Qu'est-ce que le design UX ?

Le terme Expérience Utilisateur (UX) fait référence à la manière dont les utilisateurs interagissent avec un produit ou un service. Chaque fois que vous vous connectez à un site web ou jouez à un jeu sur votre téléphone, votre interaction avec cette application est considérée comme une expérience utilisateur.

Si vous trouvez un produit facile à utiliser et à comprendre, alors cela serait considéré comme une expérience utilisateur positive. Mais si le produit est confus et frustrant, cela serait considéré comme une expérience utilisateur négative.

Si vous souhaitez en savoir plus sur le design UX et trouver des ressources pour commencer à apprendre, je vous suggère de lire mon article sur [Qu'est-ce que le design UX ?](https://www.freecodecamp.org/news/user-experience-design-what-does-a-ux-designer-do/)

## Qu'est-ce que le design UI ?

Le terme Interface Utilisateur (UI) fait référence à l'apparence et à la sensation d'un produit. Pour les produits numériques, le design UI se concentre sur tous les aspects visuels, y compris les polices, les couleurs, l'espacement, le design réactif et plus encore. 

Un bon design UI est considéré comme simple, facile à comprendre et cohérent. Vous ne voulez pas que les utilisateurs aient à lutter pour naviguer dans le produit parce qu'il n'est pas intuitif. 

Un mauvais design UI est considéré comme incohérent et difficile à comprendre. Si les utilisateurs sont confrontés à des images de mauvaise qualité, des polices incohérentes ou un mauvais contraste dans le contenu visuel, alors l'utilisateur voudra quitter le produit dès que possible.

Si vous êtes intéressé à apprendre les bases du design UI, je vous suggère de lire [cet article de Per Harald Borgen](https://www.freecodecamp.org/news/learn-ui-design-fundamentals-with-this-free-one-hour-course/).
