## Algorithmes et Structures de Données en JavaScript

Dans [ce cours freeCodeCamp](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/), vous apprendrez les bases de JavaScript, ES6, les expressions régulières, les structures de données de base, la programmation orientée objet et la programmation fonctionnelle. 

Il s'agit d'un environnement d'apprentissage interactif où vous apprendrez JavaScript en relevant des centaines de défis et cinq défis d'algorithmes de certification. 

À la fin, vous pouvez recevoir une certification gratuite à partager sur Twitter ou à ajouter à votre profil LinkedIn. 

## Apprendre JavaScript – Cours Complet pour Débutants

Dans [ce cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=PkZNo7MFNFg), vous apprendrez les conditions, les boucles, les fonctions, les objets, les tableaux et ES6.

Ce cours populaire compte plus de 7 millions de vues sur YouTube. Beau Carnes vous guidera à travers tous les [défis JavaScript de base de freeCodeCamp](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/) et fournira des explications détaillées tout au long du parcours. 

## Full Stack JavaScript

The Odin Project propose un cours de JavaScript pour débutants dans la [section Fondamentaux](https://www.theodinproject.com/paths/foundations/courses/foundations). Vous y apprendrez les bases, y compris les conditions, les boucles, les fonctions, les objets, les tableaux et ES6.

Ensuite, vous pourrez passer à [ce cours de The Odin Project](https://www.theodinproject.com/paths/full-stack-javascript?) qui vous enseignera des concepts plus avancés et vous fera construire des projets plus complexes. 

Voici une liste de certains des projets que vous réaliserez tout au long du cours :

* Un jeu Pierre, Papier, Ciseaux
* Tic Tac Toe
* Liste de tâches
* Page de restaurant
* et plus...

## Le Tutoriel Moderne de JavaScript

Dans [ce tutoriel](https://javascript.info/), vous apprendrez les bases de JavaScript, la gestion des erreurs, les promesses, Async/await, le DOM et plus encore.

Il s'agit d'un cours très détaillé rempli de centaines de leçons et de défis à compléter. Ils disposent également d'une communauté en ligne très active où vous pouvez poser des questions dans la section des commentaires et recevoir de l'aide. 

## LearnJS

Dans [ce tutoriel](https://www.learn-js.org/), vous apprendrez les bases de JavaScript, la programmation orientée objet, l'héritage et les structures de données.

Il s'agit d'une excellente introduction au langage, remplie de dizaines de défis tout au long du parcours. 

## Cours de JavaScript

Dans [ce cours SoloLearn](https://www.sololearn.com/learning/1024), vous apprendrez les conditions, les boucles, les fonctions, les objets, les tableaux, le DOM et les événements.

Il y a des dizaines de défis, de quiz et de projets à réaliser tout au long du cours.

Voici une liste de certains des projets que vous réaliserez en cours de route.

* Planificateur de voyage
* L'escargot dans le puits
* Convertisseur de devises
* Gestionnaire de contacts
* et plus...

## Programmation JavaScript

Dans [ce cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=jS4aFq5-91M), vous apprendrez les bases de JavaScript, y compris les conditions, les boucles, les fonctions, les objets, les tableaux et ES6. Vous construirez également un jeu de blackjack et une extension Google Chrome.

Il s'agit d'un cours très complet de plus de 7 heures qui vous donnera une bonne base de départ en JavaScript. 

## Tutoriel JavaScript pour Débutants : Apprendre JavaScript en 1 Heure

Dans [ce cours de Programming with Mosh](https://www.youtube.com/watch?v=W6NZfCO5SIk), vous apprendrez les bases de JavaScript, y compris les conditions, les boucles, les fonctions, les objets, les tableaux et les fonctionnalités ES6.

Il s'agit d'un excellent court cours d'introduction et Mosh est un développeur expérimenté qui vous guidera à travers tous ces concepts de manière facile à comprendre.
