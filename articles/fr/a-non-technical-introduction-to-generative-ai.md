Si vous souhaitez apprendre les concepts autour de l'IA générative sans vous perdre dans les détails techniques, nous avons un excellent cours pour vous. Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui est une introduction non technique à l'IA générative. Ce cours offre une exploration approfondie de l'IA générative, la rendant accessible aux programmeurs et aux débutants. Abdul de 1littlecoder a développé ce cours.

## Aperçu du cours

Le cours commence par une brève introduction à l'IA générative, suivie d'une comparaison entre le passé et le présent de l'IA. Vous découvrirez pourquoi l'IA générative est possible aujourd'hui et explorerez certains aspects moins connus de cette technologie.

Abdul aborde le concept de l'IA décentralisée et examine les API LLM (Large Language Model). Vous acquerrez une compréhension systémique des différents niveaux d'applications LLM, y compris les systèmes de questions-réponses, les chatbots, les solutions RAG (Retrieval-Augmented Generation), et comment les grands modèles de langage peuvent être utilisés pour des tâches de traitement du langage naturel en aval et le développement d'agents IA intelligents.

Enfin, le cours explore le potentiel des systèmes d'exploitation de grands modèles de langage, offrant un aperçu complet de l'évolution de l'IA générative, de son état actuel et de ses implications futures.

## Sections du cours

Voici les sections abordées dans ce cours.

* Introduction rapide à l'IA générative

* L'IA : avant vs maintenant

* Pourquoi l'IA générative est possible aujourd'hui

* Les aspects moins connus de l'IA générative

* Qu'est-ce que l'IA décentralisée ?

* API LLM

* Cadre d'application LLM

* Complétion de texte

* Chatbots

* RAG : LLM avec connaissances

* LLM pour les tâches de TAL en aval

* Agents IA basés sur les LLM

* Systèmes d'exploitation de grands modèles de langage

Consultez le cours complet sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=y4SLh1cpap4) et commencez votre voyage dans le domaine fascinant de l'IA générative (90 minutes de visionnage).
