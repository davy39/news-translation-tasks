MongoDB (le "M" dans les stacks MEAN et MERN) est l'une des bases de données les plus dominantes utilisées aujourd'hui.

Avec MySQL et PostgreSQL, MongoDB est une base de données standard dans l'industrie, considérée aussi bien par les startups que par les grandes entreprises. Mais contrairement aux deux autres, MongoDB est le standard de facto pour les bases de données basées sur des documents et a connu une explosion de popularité depuis sa sortie en 2009.

Dans ce tutoriel, Dan Kwon montre comment implémenter un projet MongoDB de base avec Node.js et Express. Il explique comment déployer complètement le projet.

Vous pouvez regarder la vidéo sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=4yqu8YF29cU) (75 minutes de visionnage).
