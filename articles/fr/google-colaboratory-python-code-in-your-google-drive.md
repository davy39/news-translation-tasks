Google Colaboratory (« Colab ») est un notebook (comme un Jupyter Notebook) où vous pouvez exécuter du code Python dans votre Google Drive.

Vous pouvez écrire du texte, écrire du code, exécuter ce code et voir le résultat – tout en ligne dans le même notebook.

## Avantages de Google Colab

Le partage de notebooks est aussi simple que le partage de n'importe quel document Google. Vous pouvez également obtenir l'application et exécuter le code depuis votre téléphone.

Vous pouvez utiliser le langage Python puissant et populaire dans votre Google Drive, et la configuration prendra moins de cinq minutes.

Parce que Python s'exécute sur un serveur (et non dans votre navigateur local ou sur votre ordinateur local), vous pouvez facilement l'utiliser pour interagir avec une base de données en ligne et analyser des données dans des situations où vous devez garder le code privé.

## Comment utiliser Google Colab

Maintenant, lorsque vous cliquez sur le bouton « + nouveau » et cliquez sur « plus » (en bas de cette première liste), vous verrez « Google Colaboratory » dans la liste suivante.

Cliquez dessus pour ouvrir un nouveau notebook Colab. Donnez un nom à votre notebook, comme vous le feriez pour n'importe quel document ou feuille de calcul Google. Ce notebook est en mode sombre :

![Image](https://lh3.googleusercontent.com/ttxQIV3tGb3kaiMlu-ix-J0939nbXX7Xx_Ke5UMcBnT_mRNVcNfJevAbMWm8nhIYbCM6zNjSMY_d3CwqIi-euqnf8HSAZlLG5oZST84kDnw9JpxKnLNkj-SioTtL_xhYHfiSgS1c)

Tapez une commande simple comme test. Pour exécuter le code dans n'importe quelle cellule, vous pouvez cliquer sur le bouton d'exécution sur le côté gauche de la cellule de code (ressemble à un bouton « lecture » avec un triangle dans un cercle) ou vous pouvez appuyer sur [shift] + [enter]. Le résultat apparaîtra juste en dessous de la cellule de code.

![Image](https://lh4.googleusercontent.com/DCX2F45Bewre17_E27tFm0liy5l155iNB7vt4ohbFhCS7QUpwc47JHJ0ipkgJU6AcfKDcmLY8u2q8N-JHdBl1BwkTeM5-BQ250YbH-UwEKiLC8D6gjuo96vGcwwSFPJi0fxqbSkS)

Vous pouvez importer de nombreuses bibliothèques populaires sans avoir à les installer au préalable.

![Image](https://lh6.googleusercontent.com/Ya_VZMKr0gAEs-UJq7BZa-gnjQ3_AQSAq1YK-eCNQBMgibEeFuAl9BwYZvSrhOyC518v5bjJD9gIGs7WQoI87S3cy_cbJdzIScUvYP8pxpTHEhbbRwSLZwX5qojvn7MQPEfOu0F6)

Si vous importez une bibliothèque ou définissez une fonction dans une cellule, celle-ci restera disponible dans d'autres cellules pendant au moins quelques minutes. Le runtime se déconnectera si vous ne lancez aucune cellule pendant 30 minutes ou si vous avez le notebook ouvert pendant 12 heures.

## Explorer Colab

Si vous avez suivi les étapes et que vous avez maintenant installé Google Colaboratory, vous êtes prêt à créer vos propres projets.

Si vous cliquez sur le symbole « **< >** » dans le coin inférieur gauche du notebook, vous trouverez des extraits de code que vous pouvez utiliser. Google propose également de nombreuses ressources pour vous, dont certaines se trouvent sur [https://colab.research.google.com/](https://colab.research.google.com/) (il suffit de fermer la fenêtre contextuelle qui apparaît).

Chez freeCodeCamp, nous construisons un programme pour vous montrer comment utiliser Python pour résoudre des problèmes mathématiques. Vous avez maintenant les outils à portée de main. Bon codage !
