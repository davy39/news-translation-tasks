Vous voulez utiliser vos compétences en Java pour construire un projet concret ? Que diriez-vous d'un jeu Sudoku ?

Nous venons de publier un cours complet de Ryan Kay sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire une application de bureau de jeu Sudoku en utilisant Java.

Ryan utilise une application Sudoku comme outil pédagogique afin d'expliquer comment concevoir, architecturer et construire des applications de bureau Java. Le cours est très pratique et l'idée est que vous suiviez les instructions dans votre propre IDE.

Voici quelques-uns des sujets abordés dans ce cours :

* Classes de domaine de problème
* Constantes avec Enums et variables static final
* Conteneur d'application/Point d'entrée
* Interface utilisateur avec les interfaces Java
* Logique de contrôle
* Logique de calcul
* Logique de construction
* Persistance/Stockage de données
* Remerciements et ressources supplémentaires

Regardez le cours sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=qH9mWpYMtYU) (2 heures de visionnage).
