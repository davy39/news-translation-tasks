Les API Rest sont utilisées partout. Si vous apprenez le framework Spring Boot basé sur Java, vous devrez savoir comment en créer une. Nous avons publié un cours vidéo complet qui vous apprendra à créer une API Rest en utilisant Spring Boot.

Ce cours de Pair Learning démontrera également comment utiliser PostgreSQL comme base de données relationnelle et Spring JdbcTemplate pour interagir avec celle-ci. Vous apprendrez à ajouter une authentification en utilisant JWT (JSON Web Tokens).

Voici les sections de ce cours :

* Installation du Projet et Création des Objets de la Base de Données
* Persistance des Informations Utilisateur lors de l'Enregistrement
* Connexion et Hachage du Mot de Passe
* Authentification JWT
* Ajout de Nouvelles Catégories
* Catégorie - Fonctionnalités de Recherche et de Mise à Jour
* Ajout de Transactions de Catégorie
* Transaction - Recherche et Mise à Jour
* Suppression - Catégorie et Transactions
* CORS et Test depuis un Client Web
* Résumé et Code

Vous pouvez regarder le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/5VUjP1wMqoE) (2 heures de visionnage).
