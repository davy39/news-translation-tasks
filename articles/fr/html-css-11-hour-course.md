Le HTML et le CSS sont des compétences essentielles pour une carrière dans le développement web. Ce cours de onze heures de John Smilga de Coding Addict vous enseignera le HTML et le CSS à partir de zéro. À la fin de ce cours, vous serez en mesure de créer vos propres projets.

Le HTML est un langage de balisage utilisé pour développer des pages web. Le CSS est un langage responsable de la mise en page et du style des pages web. Ils sont utilisés sur presque tous les sites web. Ce cours vidéo est le moyen idéal pour compléter le programme gratuit de HTML/CSS sur [freecodecamp.org](https://freecodecamp.org).

Vous pouvez regarder le cours vidéo complet sur la chaîne YouTube de [freeCodeCamp.org](https://www.youtube.com/watch?v=mU6anWqZJcc) (11,5 heures de visionnage).
