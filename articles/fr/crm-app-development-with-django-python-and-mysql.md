Souhaitez-vous apprendre à créer une application de gestion de la relation client (CRM) avec Django, Python et MySQL ?

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous guidera étape par étape pour construire une application CRM entièrement fonctionnelle en utilisant Django, Python et MySQL.

John Elder a développé ce cours. C'est un développeur expérimenté et un créateur de cours populaire. Il a travaillé avec une variété de langages de programmation, y compris Python, et il est passionné par l'enseignement du code. Grâce à ses conseils, vous serez en mesure de suivre ce cours en toute confiance et de créer votre propre application CRM.

Tout au long de ce cours, vous apprendrez à créer une application CRM fonctionnelle qui utilise MySQL pour la base de données. Vous apprendrez comment s'enregistrer, se connecter, se déconnecter, ajouter des enregistrements, consulter des enregistrements, mettre à jour des enregistrements et supprimer des enregistrements. À la fin du cours, vous aurez une solide compréhension de la création d'une application de base avec ces technologies.

Le cours commence par une introduction aux technologies et logiciels nécessaires pour le projet. Vous apprendrez comment installer et configurer l'application et MySQL. De plus, le cours couvre le contrôle de version avec Git et Github.com.

Ensuite, le cours plonge dans la construction de l'application de base. Vous apprendrez à vous connecter, vous déconnecter et enregistrer des utilisateurs, ainsi qu'à configurer le modèle de base de données. Le cours explique ensuite comment afficher les enregistrements sur le site Web à l'aide d'un tableau Bootstrap et comment afficher des enregistrements individuels à l'aide d'une carte Bootstrap.

Le cours explique également comment ajouter de nouveaux enregistrements, mettre à jour des enregistrements et supprimer des enregistrements. De plus, vous apprendrez à pousser (push) le code vers GitHub pour le contrôle de version.

À la fin de ce cours, vous aurez une solide compréhension de la création d'une application CRM de base avec Django, Python et MySQL. Vous pourrez utiliser vos nouvelles compétences pour créer des applications similaires pour vous-même ou pour vos clients.

En conclusion, si vous cherchez à élargir vos compétences en codage et à apprendre comment construire une application CRM avec Django, Python et MySQL, ce cours vidéo est parfait pour vous. Avec des instructions étape par étape et les conseils d'un développeur chevronné, vous serez sur la bonne voie pour créer votre propre application fonctionnelle.

Regardez le cours complet [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/t10QcFx7d5k) (durée de 2 heures).
