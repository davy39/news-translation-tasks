Apprenez à créer un site e-commerce avec Django et Python dans ce cours complet de JustDjango

Vous apprendrez toutes les étapes pour développer un site e-commerce complet, y compris :

* Installation et configuration du projet
* Ajout d'articles à un panier
* Amélioration de l'interface utilisateur
* Création d'un récapitulatif de commande
* Le processus de paiement
* Gestion des paiements avec Stripe
* Codes de réduction
* Gestion des remboursements
* Adresses par défaut
* Sauvegarde des informations de carte de crédit
