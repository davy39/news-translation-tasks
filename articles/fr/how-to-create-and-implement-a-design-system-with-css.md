Si vous allez construire un site web multi-pages, il est bon de créer un système de design pour aider à tout lier ensemble.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra comment créer et implémenter un système de design avec CSS.

Kevin Powell a créé ce cours. Kevin est un guru CSS qui a créé de nombreux cours populaires de développement web sur diverses plateformes.

Ce cours vous apprendra comment donner vie à des designs Figma de haute qualité dans le navigateur. Vous apprendrez à créer un site web de voyage spatial à partir de zéro.

Dans ce cours, vous testerez vos compétences en HTML, CSS et JavaScript en construisant un site multi-pages et responsive mettant en vedette une page d'accueil, des boutons, une barre de navigation, des onglets, une navigation au clavier et des indicateurs de soulignement, de points et de numéros.

Vous apprendrez également à créer un système de design efficace pour vous assurer que votre site est net et professionnel, et que votre codage est aussi efficace que possible.

Après avoir terminé ce cours, vous aurez une compréhension approfondie de CSS et du design responsive, ainsi que les connaissances pour construire un site interactif que les utilisateurs aimeront.

Voici tous les sujets abordés dans ce cours :

* Création de systèmes de design
* Réinitialisations CSS
* Classes utilitaires
* Utilisation de la couleur
* Typographie responsive
* Ajout d'espacement
* Boutons efficaces
* Indicateurs de soulignement
* Indicateurs de points
* Indicateurs de numéros
* Conteneurs de grille
* Responsivité
* Ajout d'images de fond
* Navigation
* Boutons d'ouverture/fermeture
* Ajustements d'accessibilité
* Aller au contenu
* Alignement
* L'élément picture
* Navigation au clavier
* Onglets
* Refactoring

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=lRaL-8qZ0mM) (7 heures de visionnage).
