Microsoft Excel affiche les données dans un *format tabulaire*. Cela signifie que les informations sont organisées dans un tableau composé de lignes et de colonnes.

Les lignes et les colonnes sont des propriétés différentes qui ensemble constituent un tableau.

Ce sont les deux fonctionnalités les plus importantes d'Excel qui permettent aux utilisateurs de stocker et de manipuler leurs données.

Ci-dessous, nous discuterons des définitions d'une ligne et d'une colonne, ainsi que des différences entre ces deux fonctionnalités.


## Qu'est-ce qu'une ligne dans Excel ?

Chaque ligne est désignée et identifiée par une valeur numérique unique que vous verrez sur le côté gauche.

![Screenshot-2021-08-05-at-10.53.07-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-05-at-10.53.07-PM.jpeg)

Les numéros de ligne sont disposés verticalement sur la feuille de calcul, allant de 1 à 1 048 576 (vous pouvez avoir un total de 1 048 576 lignes dans Excel).

Les lignes elles-mêmes s'étendent horizontalement sur une feuille de calcul.

Les données sont placées horizontalement dans le tableau et vont de gauche à droite.

![Screenshot-2021-08-06-at-5.28.11-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-06-at-5.28.11-PM.jpeg)

La ligne 1 est la première ligne dans Excel.

Comme vous pouvez le voir dans l'exemple ci-dessous, vous pouvez sélectionner toute la ligne avec le numéro 3 en cliquant sur le numéro lui-même.

![Screenshot-2021-08-05-at-10.38.03-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-05-at-10.38.03-PM.png)

Pour naviguer à travers les numéros et atteindre la dernière ligne, vous pouvez utiliser :

- Pour les utilisateurs Windows : `Control flèche de navigation vers le bas`. Vous appuyez d'abord sur la touche Control puis, tout en la maintenant enfoncée, vous appuyez sur la flèche de navigation vers le bas.
- Pour les utilisateurs Mac : `Command flèche de navigation vers le bas`. Vous appuyez d'abord sur la touche Command puis, tout en la maintenant enfoncée, vous appuyez sur la flèche de navigation vers le bas.

Pour revenir à la première ligne (le haut) à nouveau, appuyez sur `Control flèche de navigation vers le haut` pour Windows et `Command flèche de navigation vers le haut` pour Mac.


## Qu'est-ce qu'une colonne dans Excel ?

Les colonnes sont désignées et identifiées par une lettre d'en-tête alphabétique unique, qui se trouve en haut de la feuille de calcul.

![Screenshot-2021-08-06-at-12.39.38-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-06-at-12.39.38-PM.jpeg)

Les en-têtes de colonne vont de A à XFD, car les feuilles de calcul Excel peuvent avoir 16 384 colonnes au total.

Les colonnes s'étendent verticalement dans la feuille de calcul, et les données vont de haut en bas.

![Screenshot-2021-08-06-at-12.39.38-PM-2](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-06-at-12.39.38-PM-2.jpeg)

La colonne A est la première colonne dans Excel.

Dans l'exemple ci-dessous, vous pouvez voir que toute la colonne avec l'en-tête B est sélectionnée en appuyant/cliquant sur la lettre en haut.

![Screenshot-2021-08-06-at-12.42.39-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-06-at-12.42.39-PM.png)

Pour vous déplacer vers la dernière colonne :

- Pour les utilisateurs Windows : `Control flèche de navigation vers la droite`. Appuyez d'abord sur la touche Control puis, tout en la maintenant enfoncée, appuyez sur la flèche de navigation vers la droite.
- Pour les utilisateurs Mac : `Command flèche de navigation vers la droite`. Appuyez d'abord sur la touche Command puis, tout en la maintenant enfoncée, appuyez sur la flèche de navigation vers la droite.

Pour revenir à la première colonne à nouveau, appuyez sur `Control flèche de navigation vers la gauche` pour Windows et `Command flèche de navigation vers la gauche` pour Mac.


## Qu'est-ce qu'une cellule dans Excel ?

Une cellule est l'intersection d'une ligne et d'une colonne. Une ligne et une colonne adjacentes constituent une cellule.

Vous pouvez définir une cellule par la combinaison d'un numéro de ligne et d'un en-tête de colonne.

Par exemple, ci-dessous, la cellule sélectionnée est C3. Elle a un en-tête de colonne C et un numéro de ligne 3.

![Screenshot-2021-08-06-at-2.19.09-PM](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot-2021-08-06-at-2.19.09-PM.png)

Nous pouvons également sélectionner une ligne ou une colonne entière à partir d'une cellule.

Pour sélectionner toute la ligne lorsque vous êtes dans une cellule, appuyez sur `Shift Espace`.

Pour sélectionner toute la colonne lorsque vous êtes dans une cellule, appuyez sur `Ctrl Espace`.


## Conclusion

Maintenant, vous connaissez les définitions des lignes et des colonnes dans Excel. Vous avez appris leurs principales différences et leur fonctionnement.

En résumé, les informations dans une ligne sont présentées horizontalement, tandis que dans une colonne, les informations sont verticales.
