L'IA d'entreprise nécessite des compétences et des technologies différentes de celles des applications basiques basées sur OpenAI que l'on voit partout.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui enseigne tout sur les Embeddings, RAG, les modèles multimodaux et les agents avec Amazon Nova. Ce cours couvre l'ingénierie de l'IA. Vous apprendrez de nombreuses technologies, des Amazon Tian Text Embeddings à l'intégration de LangChain avec Amazon Bedrock. Vous construirez une application complète en utilisant les agents et les bases de connaissances d'Amazon Bedrock.

Ce cours démontre des applications pratiques de ces technologies, telles que l'automatisation et l'optimisation du processus de réclamation d'assurance avec l'IA. Vous apprendrez comment utiliser des agents alimentés par Bedrock pour aider dans des tâches comme la création de réclamations, la gestion de documents et la récupération de données, améliorant significativement l'efficacité et les capacités de prise de décision dans les opérations de service client. Suman Debnath d'Amazon a développé ce cours.

Voici les sections couvertes dans ce cours :

* Embeddings en NLP et LLMs

* Encodage par paires d'octets (BPE)

* Amazon Tian Text Embeddings

* LLMs multimodaux

* Prétraitement contrastif langage-image (CLIP)

* Bootstrapping Language-Image Pre-training with Frozen Image Encoders and Large Language Models (BLIP-2)

* Modèle multimodal Amazon Nova

* RAG multimodal

* Agents avec bases de connaissances

* Ressources
