Par Andrew Brown

Le stockage cloud vous permet de stocker vos données sur le disque dur de quelqu'un d'autre, dans des centres de données à travers le monde. Vous n'avez pas à vous soucier de perdre vos données et vous pouvez y accéder depuis n'importe où.

Et croyez-le ou non, beaucoup de ces services offrent des niveaux gratuits assez généreux.

Dans cet article, je vais décomposer les principaux fournisseurs de stockage cloud afin que vous puissiez décider lesquels vous souhaitez utiliser.

# Pourquoi utiliser le stockage cloud ?

Il y a de nombreux avantages et peu d'inconvénients à adopter le stockage cloud. Voici les cas d'utilisation les plus courants :

## Étendre votre stockage local

Vous pouvez avoir un ordinateur portable ou un téléphone avec un stockage limité. Il peut être possible d'installer un disque dur SSD supplémentaire ou une carte SD, mais cela peut être coûteux. De plus, certains appareils ne disposent pas d'un emplacement pour un disque dur supplémentaire.

### Sauvegarder vos données

La plupart des fournisseurs de stockage cloud sont résistants à la perte de données. Lorsque vous téléchargez des données dans le cloud, le fournisseur stocke des copies de vos données jusqu'à six fois sur six disques durs différents dans des centres de données isolés distincts.

De votre point de vue, il semblera qu'il n'y a qu'un seul fichier, mais ces copies existent. Cette stratégie cloud de copies redondantes est appelée haute disponibilité et garantit des données récupérables en cas de catastrophes virtuelles ou physiques. Donc, si vous avez des photos précieuses que vous ne voulez pas perdre, le cloud est le meilleur endroit pour elles.

### Partage et collaboration

Le stockage cloud vous permet de partager des fichiers de votre choix avec vos amis et collègues. Lorsque vous partagez des fichiers par e-mail, vous êtes limité à environ 20 Mo par e-mail. Avec le stockage cloud, vous pourriez partager des téraoctets de données.

### Sécuriser vos données

Lorsque vous téléchargez vos données vers le stockage cloud, ils chiffrent vos données au repos. Certains fournisseurs vous permettent de protéger par mot de passe, d'exiger une clé de chiffrement ou d'appliquer une authentification multifactorielle nécessitant une étape de vérification supplémentaire pour accéder.

Il existe une variété de fournisseurs de stockage cloud, et la majorité d'entre eux offrent un niveau gratuit allant jusqu'à plusieurs gigaoctets. En utilisant divers fournisseurs de cloud, vous pourriez avoir des téraoctets de stockage gratuit.

J'ai catégorisé le stockage cloud gratuit en deux catégories : une pour le consommateur général, et des options supplémentaires pour les développeurs.

# Stockage cloud gratuit convivial pour les utilisateurs

### Dropbox - 2 Go

L'un des plus anciens et des plus populaires fournisseurs de stockage cloud pour le consommateur général. Dropbox a une interface simple, et la synchronisation des fichiers avec votre ordinateur local est une expérience transparente.

Dropbox a l'offre gratuite la plus petite de cette liste. Il peut vous intéresser de savoir que Dropbox utilise AWS S3 en arrière-plan.

[https://www.dropbox.com/basic](https://www.dropbox.com/basic)

### Amazon Drive - 5 Go avec abonnement Prime

Amazon est mieux connu pour son stockage convivial pour les développeurs AWS S3 que pour Amazon Drive. Si vous voulez obtenir les 5 Go gratuits, vous devez acheter un abonnement Amazon Prime.

Cela fait toujours partie de notre liste puisque les abonnements Prime deviennent très courants, et il vaut la peine de souligner le stockage gratuit.

[https://www.amazon.com/b?ie=UTF8&node=15547130011](https://www.amazon.com/b?ie=UTF8&node=15547130011)

### OneDrive - 5 Go

Si vous êtes sur Windows 10, vous avez déjà OneDrive préinstallé, ce qui est logique puisque OneDrive est le service de stockage cloud de Microsoft.

[https://onedrive.live.com/about/en-us/plans/](https://onedrive.live.com/about/en-us/plans/)

### Google Drive - 15 Go

Si vous avez Gmail ou un téléphone ou une tablette Android, alors vous êtes à un clic d'accéder à Google Drive.

Google Drive a la meilleure offre de notre liste, et son interface web est très facile à utiliser.

[https://www.google.com/drive/](https://www.google.com/drive/)

### iCloud - 5 Go

Si vous êtes sur un Mac ou utilisez un iPhone ou un iPad, alors on vous a demandé lors de la configuration de votre compte Apple si vous souhaitiez utiliser iCloud.

Vous n'avez pas besoin de posséder des produits Apple pour profiter du stockage d'iCloud.

[https://support.apple.com/en-ca/HT201318?](https://support.apple.com/en-ca/HT201318?)

### Box - 10 Go

Box était autrefois aussi populaire que Dropbox avec la même offre. Mais à un moment donné, Box a changé son orientation en ajoutant plusieurs fonctionnalités adaptées aux industries professionnelles.

Ils ont toujours un compte individuel qui vient avec la deuxième meilleure offre de notre liste.

[https://www.box.com/pricing/individual](https://www.box.com/pricing/individual)

# Stockage cloud gratuit **convivial pour les développeurs**

Ces solutions de stockage cloud sont destinées aux développeurs qui construisent des applications web et mobiles. Pour accéder à ces fournisseurs de stockage cloud, vous devez fournir une carte de crédit qui peut appliquer un frais de 1 $ pour vérification.

Ces services n'offrent pas de limites strictes, ce qui signifie que si vous stockez au-delà du niveau gratuit, vous serez facturé. Il est donc nécessaire de surveiller attentivement pour éviter des factures élevées.

De plus, les prix sont un peu plus compliqués où vous devez considérer les coûts de transfert de données sortantes, de transfert de données entrantes et de stockage. Ces fournisseurs de stockage cloud sont beaucoup plus économiques que les fournisseurs de stockage conviviaux pour les utilisateurs, car payer pour des gigaoctets supplémentaires peut coûter quelques centimes, mais encore une fois, il n'y a pas de garde-fous.

### Amazon S3 - 5 Go

Amazon a créé une société dérivée appelée AWS (Amazon Web Services), et ils offrent plus de 175+ services, S3 étant l'un d'eux. AWS est actuellement le fournisseur le plus populaire parmi les startups technologiques. S3, si vous vous demandiez, signifie Simple Storage Service.

[https://aws.amazon.com/s3/pricing/?nc=sn&loc=4](https://aws.amazon.com/s3/pricing/?nc=sn&loc=4)

### Google Cloud Storage - 5 Go

Google a Google Drive pour le consommateur régulier et Google Cloud Storage pour les développeurs.

Contrairement à Google Drive, où vous obtenez 15 Go gratuitement avec Google Cloud Storage, vous n'obtenez que 5 Go.

Google Drive utilise probablement Google Cloud Storage en arrière-plan, ce qui vous fait vous demander pourquoi ils ne laissent pas les développeurs avoir 15 Go gratuits.

[https://cloud.google.com/storage/pricing](https://cloud.google.com/storage/pricing)

### Azure Blob Storage - 5 Go

Azure est la société de services cloud de Microsoft. Vous vous demandez peut-être pourquoi elle est nommée Blob Storage. Blob signifie Binary Large Object et est une collection de données binaires stockées comme un seul objet dans une base de données.

En fait, tous nos fournisseurs de stockage conviviaux pour les développeurs ici sont connus sous le nom de "stockage d'objets".

[https://azure.microsoft.com/en-ca/free/](https://azure.microsoft.com/en-ca/free/)

### Object Storage Service (Alibaba Cloud) - 5 Go

Alibaba est l'Amazon de l'Asie et a sa propre société de services cloud appelée Alibaba Cloud. Leur offre est à la hauteur de tous les autres fournisseurs de stockage cloud conviviaux pour les développeurs. Mais peut-être dans les années à venir, ils seront mieux connus alors qu'ils continuent à ajouter plus de centres de données à travers le monde.

[https://www.alibabacloud.com/product/oss/pricing?spm=a3c0i.7950270.1167928.1.3310ab91MGJk9V](https://www.alibabacloud.com/product/oss/pricing?spm=a3c0i.7950270.1167928.1.3310ab91MGJk9V)

# Le coût caché du transfert de données sortantes

La plupart des fournisseurs de cloud mettent en avant la quantité de données que vous pouvez stocker par mois en taille de gigaoctets. Ce qu'ils tendent souvent à cacher, ce sont les limites sur le transfert de données sortantes.

Le transfert de données sortantes est l'acte de télécharger, et cela peut représenter un coût élevé pour les fournisseurs de stockage cloud. Les fichiers vidéo, par exemple, peuvent aller de centaines de mégaoctets à plusieurs gigaoctets.

Si vous prévoyez de partager des fichiers vidéo avec plusieurs amis, vous pourriez constater que vous serez limité, ou vous pourriez payer une facture élevée pour un dépassement.

# Gestionnaire de stockage multi-cloud

Il peut être difficile de gérer plusieurs fournisseurs de stockage cloud, car ils ont tous des interfaces différentes. Vous pourriez utiliser un service tel que Odrive, qui unifie tout votre stockage cloud dans une interface agnostique.

Odrive a un niveau gratuit. Mais si vous avez besoin de synchroniser vos fichiers entre plusieurs appareils, vous devrez passer à une version payante.

[https://www.odrive.com/](https://www.odrive.com/)

## Il existe une multitude d'options de stockage cloud.

Je vous encourage à utiliser autant de ces options que vous en avez besoin. En faisant cela, vous pourriez vous en sortir sans payer quoi que ce soit.

Cela dit, si vous trouvez une option de stockage cloud que vous aimez vraiment, vous pouvez toujours payer un peu d'argent chaque année pour obtenir plus d'espace.
