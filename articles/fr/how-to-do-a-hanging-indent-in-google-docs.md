Une indentation suspendue est un moyen simple de rendre vos documents plus professionnels. Google Docs dispose d'une fonctionnalité intégrée pour créer cela.

### Qu'est-ce qu'une indentation suspendue ?

Une indentation suspendue est un type d'indentation où la première ligne d'un paragraphe n'est pas indentée, mais les lignes suivantes du même paragraphe le sont.

Voici un exemple de ce à quoi ressemble une indentation suspendue :

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Untitled_document_-_Google_Docs_---2.png)

Dans Google Docs, vous pouvez créer une indentation suspendue en suivant ces étapes simples :

**Étape 1** : Sélectionnez le texte que vous souhaitez formater. Vous pouvez le faire en triple-cliquant sur le paragraphe.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Untitled_document_-_Google_Docs_--.png)

**Étape 2** : Cliquez sur le menu "Format". Ensuite, choisissez "Aligner et indenté". Puis choisissez "Options d'indentation".

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Untitled_document_-_Google_Docs_---1.png)

**Étape 3** : Dans ce menu, vous pouvez choisir "Suspendue". Si vous le souhaitez, vous pouvez spécifier de combien vous voulez indenter. Pour ce tutoriel, j'ai laissé cette valeur à la valeur par défaut de 0,5.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Untitled_document_-_Google_Docs_---3.png)

Et voilà ! Vous avez un paragraphe élégant qui utilise l'indentation suspendue.
