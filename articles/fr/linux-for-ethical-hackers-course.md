Apprenez les bases de Kali Linux dans ce cours de The Cyber Mentor.

Le cours couvre l'installation, l'utilisation du terminal / ligne de commande, la script bash, les outils et fonctionnalités pour le hacking éthique, et plus encore. Le cours se concentre sur Kali Linux, mais une grande partie du contenu s'applique également à d'autres versions de Linux.

Ce cours couvre :

* Installation de VMWare / Kali Linux
* Aperçu de Kali Linux
* Navigation dans le système de fichiers
* Utilisateurs et privilèges
* Commandes réseau courantes
* Visualisation, création et édition de fichiers
* Démarrage et arrêt des services
* Installation et mise à jour des outils
* Scripting avec Bash

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=lZAoFs75_cs) (2 heures de visionnage).
