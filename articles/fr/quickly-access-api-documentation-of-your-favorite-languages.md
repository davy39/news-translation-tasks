Il n'est pas surprenant que vous deviez travailler avec différents langages de programmation, ou au moins utiliser de nombreuses méthodes. Vous ne savez peut-être pas comment appeler une méthode, combien de paramètres elle possède, ou quel type d'objet est utilisé pour chaque paramètre. Dans ces cas, avoir un moyen rapide de rechercher dans la documentation API aide énormément.

Il existe de nombreuses façons de consulter la documentation API officielle pour différents langages. Cependant, j'ai trouvé deux applications qui sont beaucoup plus faciles que tout ce que j'ai essayé. Ces outils peuvent vous aider à accéder à la documentation avec très peu de touches.

### Installation des outils

[Dash](http://t.dripemail2.com/c/eyJhY2NvdW50X2lkIjoiMzUyMjAyOCIsImRlbGl2ZXJ5X2lkIjoiODc5MDA5NTcwIiwidXJsIjoiaHR0cDovL2thcGVsaS5jb20vZGFzaD9fX3M9aDdhcHkyM3Y5ZWV3YThxbjF1aHUifQ) est adapté pour Mac et [Zeal](http://t.dripemail2.com/c/eyJhY2NvdW50X2lkIjoiMzUyMjAyOCIsImRlbGl2ZXJ5X2lkIjoiODc5MDA5NTcwIiwidXJsIjoiaHR0cDovL3plYWxkb2NzLm9yZz9fX3M9aDdhcHkyM3Y5ZWV3YThxbjF1aHUifQ) peut être utilisé pour tout autre système d'exploitation de votre choix.

Une fois que vous les avez téléchargés, la première chose à faire est de télécharger la documentation pour le langage que vous allez utiliser. Par exemple, vous pouvez télécharger Java, Ruby, JavaScript, ou même des frameworks comme jQuery, Ruby on Rails, etc.

Ensuite, vous pouvez configurer des raccourcis clavier qui vous aideront à rechercher rapidement n'importe quoi avec seulement quelques touches.

### Conclusion

Maintenant, lorsque vous avez des doutes sur une méthode ou une fonctionnalité particulière que vous êtes sur le point d'utiliser, vous pouvez rapidement vous référer à la documentation que vous avez téléchargée sur votre ordinateur.

Il peut y avoir des cas où quelque chose manque dans ces ensembles de documentation, mais dans la plupart des cas, ils sont très utiles.
