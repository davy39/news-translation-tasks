Apprenez à créer une application full stack avec Next.js, TypeScript et AWS Amplify.

Nous venons de publier un nouveau cours sur les applications full stack sur la chaîne YouTube freeCodeCamp.org. Brian H. Hough, un ingénieur logiciel renommé et créateur de cours, est votre guide. Il vous aidera à construire une application génératrice de citations inspirantes.

Vous travaillerez avec Next.js, un framework React populaire pour construire des applications web scalables et efficaces. TypeScript, un sur-ensemble fortement typé de JavaScript qui offre une programmation plus robuste, sera votre langage de programmation. Pour assurer un déploiement et un hébergement sans faille, vous exploiterez la puissance d'AWS Amplify, un ensemble d'outils et de services d'Amazon Web Services qui permet aux développeurs de construire des applications serverless full-stack.

Votre parcours commence par la configuration d'un dépôt GitHub. Ensuite, vous créez un projet en utilisant Next.js et Styled Components. Vous donnerez à votre application une touche unique. Vous ajouterez des polices Google et un fond dynamique. De plus, vous concevrez un pied de page et une fenêtre contextuelle Material-UI.

Ce cours est pratique. Vous écrirez un script Node.js pour générer des images dans CodeSandbox. Ensuite, vous utiliserez Node.js pour appeler l'API de ZenQuotes.

Le cours approfondit le développement backend. Vous explorerez l'écosystème AWS. Vous utiliserez l'interface de ligne de commande AWS Amplify, créerez une API GraphQL avec AWS AppSync et ajouterez une authentification. Cela se fait avec Amazon Cognito et IAM. De plus, vous écrirez et interrogerez des données NoSQL avec Amazon DynamoDB et AWS AppSync.

Le déploiement est une autre partie cruciale de ce cours. Vous déploierez votre pile AWS CloudFormation dans le cloud. Ensuite, vous déploierez votre script Node.js vers AWS Lambda et vous vous assurerez qu'il a accès à Amazon DynamoDB.

Le débogage est une compétence critique. Vous apprendrez à déboguer en utilisant les journaux Amazon CloudWatch. Vous dépannerez également votre pipeline CI/CD avec les paramètres de construction AWS Amplify.

À la fin, vous récupérerez des citations avec AWS Lambda, AWS AppSync et IAM. Vous ajouterez l'hébergement de l'application avec un pipeline CI/CD. Vous utiliserez AWS Amplify et GitHub. Ensuite, vous célébrerez votre build final.

Que vous soyez débutant ou développeur expérimenté, ce cours est fait pour vous. C'est une excellente opportunité d'améliorer vos compétences. Regardez le [cours complet](https://youtu.be/FRmCxj9K7II) sur la chaîne YouTube freeCodeCamp.org (6 heures de visionnage).
