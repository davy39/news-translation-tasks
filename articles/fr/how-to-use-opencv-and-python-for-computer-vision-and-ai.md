OpenCV est une bibliothèque Python populaire pour la vision par ordinateur en temps réel.

Nous venons de publier un nouveau cours OpenCV sur la chaîne YouTube freeCodeCamp.org. Ce cours provient directement des créateurs d'OpenCV et est le cours parfait pour les débutants.

Vous apprendrez à utiliser OpenCV pour la vision par ordinateur et l'IA. Vous apprendrez et serez exposé à une large gamme de sujets passionnants comme la manipulation d'images et de vidéos, l'amélioration d'images, le filtrage, la détection de contours, la détection et le suivi d'objets, la détection de visages et le module d'apprentissage profond d'OpenCV.

À la fin de ce cours, il y a une interview avec le Dr Satya Mallick, le PDG d'OpenCV.org. Le Dr Mallick partage ses opinions sur les nombreuses opportunités dans le marché de l'emploi en vision par ordinateur et en IA. Il donne des conseils sur la façon de se préparer et de se faire embaucher pour un emploi dans l'IA.

Voici les sections de ce cours :

* Module 1 : Prise en main des images
* Module 2 : Manipulation basique d'images
* Module 3 : Annotation d'images
* Module 4 : Amélioration d'images
* Module 5 : Accès à la caméra
* Module 6 : Lire et écrire des vidéos
* Module 7 : Filtrage d'images et détection de contours
* Module 8 : Caractéristiques d'images et alignement d'images
* Module 9 : Assemblage d'images et création de panoramas
* Module 10 : Imagerie à haute dynamique (HDR)
* Module 11 : Suivi d'objets
* Module 12 : Détection de visages
* Module 13 : Détection d'objets
* Module 14 : Estimation de pose avec OpenPose
* Interview avec le PDG d'OpenCV, le Dr Satya Mallick

Regardez le cours ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=P4Z8_qe2Cu0) (3 heures de visionnage).
