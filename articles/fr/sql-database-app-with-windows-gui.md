Êtes-vous intéressé par l'apprentissage de la conception et de l'utilisation de bases de données pour les applications Windows ?

Nous venons de publier un cours complet sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer une application de base de données SQL avec une interface graphique Windows.

Animé par Shad Sluiter, professeur d'informatique et de développement logiciel à l'Université Grand Canyon, ce cours vous guidera à travers le processus de création d'une application Windows graphique avec C# capable d'afficher et de modifier des données provenant d'un serveur de base de données MySQL.

Tout au long du cours, vous acquerrez une expérience pratique sur une gamme de sujets, notamment la création d'une nouvelle base de données dans MySQL, l'écriture de requêtes SQL, la création d'une application front-end avec interface graphique (GUI) et la connexion à une base de données avec un Data Access Object. Vous apprendrez également comment interroger une base de données, insérer de nouveaux enregistrements et effectuer des actions de sélection composées.

Mais ce n'est pas tout – ce cours couvre également des sujets plus avancés tels que la création d'une deuxième table avec une clé étrangère, la réalisation de jointures de tables, la conception avec des diagrammes UML et ER, et même l'ajout d'un lecteur vidéo à l'application.

Pour suivre ce cours, vous aurez besoin de Microsoft Windows. Avec des tutoriels vidéo faciles à suivre, ce cours est parfait pour tous ceux qui cherchent à acquérir des compétences pratiques en conception de bases de données et en développement d'applications.

Donc, si vous êtes prêt à faire passer vos compétences en base de données au niveau supérieur, suivez ce cours et commencez à créer votre propre application Windows en toute simplicité.

Vous pouvez regarder le cours complet [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/VX4wl7qIcbA) (3 heures de visionnage).
