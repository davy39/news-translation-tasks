Apprenez à créer des Dapp Ethereum dans ce cours vidéo complet pour débutants de EatTheBlocks. Ce cours enseigne comment construire des applications décentralisées sur la blockchain Ethereum. 

Vous apprendrez à construire 5 Dapp Ethereum. Pour chacune d'elles, ce cours démontre comment :

* développer des contrats intelligents,
* tester des contrats intelligents,
* et construire une interface web pour interagir avec le contrat intelligent.

Le cours couvre :

* L'architecture des Dapps
* Solidity, le langage de programmation pour les contrats intelligents Ethereum
* Remix, l'IDE en ligne pour Solidity
* Truffle, le framework pour les contrats intelligents Solidity
* Ganache, la blockchain de développement local
* Web3, la bibliothèque JS pour intégrer une interface de Dapp
* Metamask, le portefeuille Ethereum utilisé par la plupart des Dapps (extension de navigateur)
* Le déploiement sur le testnet public (Ropsten) et le mainnet, en utilisant l'API Infura

Regardez le cours complet ci-dessous ou sur la [chaîne YouTube freeCodeCamp](https://www.youtube.com/watch?v=8wMKq7HvbKw) (5 heures de visionnage).
