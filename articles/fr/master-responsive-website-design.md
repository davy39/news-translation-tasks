Ce tutoriel vidéo de Kevin Powell vous apprend à créer des sites web responsives à partir de zéro. 

Le cours commence par expliquer les concepts de base nécessaires pour commencer à penser de manière responsive, puis il vous guide à travers chaque étape de la création d'un site web responsive de niveau professionnel.

Vous apprendrez ces éléments essentiels pour créer un site web responsive, tout en travaillant sur un projet concret :

1. Comment aborder une mise en page
2. Les unités CSS
3. Les bases de Flexbox
4. Les bases des media queries

Vous pouvez regarder [la vidéo complète sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=srvUrASNj0s) (4 heures de visionnage).
