Avez-vous déjà essayé d'écrire du code ou de naviguer sur internet, pour découvrir que votre clavier ne fonctionne pas ?

Lorsque votre clavier ne fonctionne pas, cela peut être dû à : 
- Des dommages physiques
- De la saleté
- Des touches de filtre
- Un pilote obsolète ou corrompu
- Une batterie faible (si vous utilisez un clavier externe).

Vous pouvez vérifier le port USB dans lequel votre clavier est inséré et vous assurer qu'il est correctement connecté pour le faire fonctionner à nouveau. Vous pouvez également nettoyer votre clavier. Mais ces solutions rapides ne résoudront peut-être pas le problème.

Si vous continuez à appuyer sur les touches de votre clavier et que rien ne s'affiche à l'écran, vous êtes au bon endroit. Car dans ce guide, je vais vous montrer 3 façons de le réparer.

Notez que si vous avez des problèmes avec votre clavier externe ou intégré, vous devrez activer le clavier à l'écran pour le réparer avec les solutions fournies dans ce guide.

### Comment activer le clavier à l'écran
Allez dans les paramètres et sélectionnez Facilité d'accès.
![ss-1](https://www.freecodecamp.org/news/content/images/2022/01/ss-1.jpg)

Sous interaction, sélectionnez Claviers.
![ss-2](https://www.freecodecamp.org/news/content/images/2022/01/ss-2.jpg)

Activez "Utiliser le clavier à l'écran".
![ss-3](https://www.freecodecamp.org/news/content/images/2022/01/ss-3.jpg)

## Comment réparer votre clavier qui ne fonctionne pas en désactivant les touches de filtre

Les touches de filtre aident les personnes qui ne peuvent pas appuyer sur 2 boutons en même temps à exécuter des fonctionnalités qui ne peuvent être accessibles que de cette manière. Mais parfois, cela peut faire en sorte que votre clavier cesse de fonctionner.

**Pour désactiver les touches de filtre, suivez les étapes ci-dessous :**

**Étape 1** : Cliquez sur Démarrer et recherchez "panneau de configuration", puis appuyez sur ENTRÉE pour ouvrir le premier résultat de recherche - qui est toujours le Panneau de configuration.
![cpanel](https://www.freecodecamp.org/news/content/images/2022/01/cpanel.jpg)

**Étape 2** : Assurez-vous que vous affichez par Catégorie et non par petites icônes ou grandes icônes, puis choisissez "Facilité d'accès".
![ss-4](https://www.freecodecamp.org/news/content/images/2022/01/ss-4.jpg)

**Étape 3** : Cliquez sur le lien "Modifier le fonctionnement de votre clavier".
![ss-5](https://www.freecodecamp.org/news/content/images/2022/01/ss-5.jpg)

**Étape 4** : Décochez la case "Activer les touches de filtre".

**Étape 5** : Cliquez sur "Appliquer", puis sur "Ok".
![ss-6](https://www.freecodecamp.org/news/content/images/2022/01/ss-6.jpg)

## Comment réparer votre clavier qui ne tape pas en désactivant le démarrage rapide

Le démarrage rapide est une option qui permet aux ordinateurs Windows 10 de démarrer rapidement.

Si vous utilisez un clavier externe, la fonction de démarrage rapide pourrait être la raison pour laquelle il ne fonctionne pas.

**Suivez les recommandations ci-dessous pour désactiver la fonction de démarrage rapide.**

**Étape 1** : Cliquez sur Démarrer et recherchez "panneau de configuration". Cliquez sur le résultat de recherche du Panneau de configuration ou appuyez sur `ENTRÉE` sur votre clavier.
![cpanel](https://www.freecodecamp.org/news/content/images/2022/01/cpanel.jpg)

**Étape 2** : Assurez-vous que vous affichez par Catégorie, puis sélectionnez "Matériel et audio".
![ss-7_LI](https://www.freecodecamp.org/news/content/images/2022/01/ss-7_LI.jpg)

**Étape 3** : Sélectionnez Options d'alimentation.
![ss-8](https://www.freecodecamp.org/news/content/images/2022/01/ss-8.jpg)

**Étape 4** : Cliquez sur le lien qui dit "Choisir le comportement des boutons d'alimentation".
![ss-9](https://www.freecodecamp.org/news/content/images/2022/01/ss-9.jpg)

**Étape 5** : Cliquez sur le lien "Modifier les paramètres actuellement non disponibles".
![ss-10](https://www.freecodecamp.org/news/content/images/2022/01/ss-10.jpg)

**Étape 6** : Décochez "Activer le démarrage rapide" et cliquez sur le bouton "Enregistrer les modifications".
![ss-11](https://www.freecodecamp.org/news/content/images/2022/01/ss-11.jpg)

## Comment réparer votre clavier qui ne fonctionne pas en mettant à jour le pilote du clavier

Un pilote obsolète ou corrompu pourrait être la raison pour laquelle votre clavier ne fonctionne pas.

**Vous pouvez mettre à jour le pilote de votre clavier en suivant les étapes expliquées ci-dessous :**

**Étape 1** : Faites un clic droit sur Démarrer et sélectionnez Gestionnaire de périphériques.
![devicemanager](https://www.freecodecamp.org/news/content/images/2022/01/devicemanager.jpg)

**Étape 2** : Développez Claviers.
![ss-12](https://www.freecodecamp.org/news/content/images/2022/01/ss-12.jpg)

**Étape 3** : Faites un clic droit sur le clavier concerné et sélectionnez Mettre à jour le pilote.
![ss-13](https://www.freecodecamp.org/news/content/images/2022/01/ss-13.jpg)

**Étape 4** : Sélectionnez rechercher automatiquement un logiciel de pilote mis à jour.
![ss-14](https://www.freecodecamp.org/news/content/images/2022/01/ss-14.jpg)

Votre ordinateur va maintenant rechercher sur internet les pilotes récents et les installer.
![ss-15](https://www.freecodecamp.org/news/content/images/2022/01/ss-15.png)
