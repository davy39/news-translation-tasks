Dans ce cours vidéo, vous apprendrez React en créant une application de chat.

À la fin de la vidéo, vous aurez une solide compréhension de React.js et aurez votre propre application de chat que vous pourrez utiliser pour discuter avec vos amis.

Ce cours utilise l'API Chatkit, vous n'aurez donc pas à vous soucier du codage back-end. Et à tout moment pendant la vidéo, vous pouvez obtenir une version fonctionnelle du code et expérimenter avec sur Scrimba.

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=jFNHerJqvFw) (90 minutes de visionnage).
