Les APIs, ou Interfaces de Programmation d'Applications, sont un composant critique du développement web et de la programmation. Elles fournissent un moyen pour différentes applications et systèmes de communiquer entre eux, échangeant des données et permettant des fonctionnalités.

Nous venons de publier un cours sur les bases des APIs sur la chaîne YouTube de freeCodeCamp.org. Il s'agit d'une version mise à jour de ce qui est déjà l'un des cours sur les APIs les plus populaires sur Internet.

Craig Dennis enseigne ce cours. Il est un excellent éducateur en développement et il travaille pour Twilio.

![Image](https://www.freecodecamp.org/news/content/images/2023/02/api.gif)

Dans ce cours pour débutants, vous apprendrez ce que sont les APIs, pourquoi elles existent, et les nombreux avantages qu'elles offrent. Vous aurez une expérience pratique en travaillant avec quelques APIs web populaires et vous acquerrez une compréhension approfondie de leur fonctionnement.

Le cours est divisé en trois unités, chacune s'appuyant sur la précédente :

* Qu'est-ce qu'une API ?
* Exploration des APIs
* Tout ensemble maintenant

Dans la première unité, vous apprendrez les bases des APIs, y compris ce qu'elles sont et comment elles sont utilisées. Vous apprendrez également les APIs RESTful et le fonctionnement du web.

Dans la deuxième unité, vous explorerez les APIs et acquerrez une expérience pratique en travaillant avec elles. Vous apprendrez à utiliser les APIs depuis la ligne de commande, à utiliser des outils pour explorer les APIs, et plus encore.

Enfin, dans la troisième unité, vous mettrez tout ensemble en travaillant sur un projet qui implique l'écriture d'une API côté serveur, la récupération des résultats sur le client depuis votre serveur, et la conclusion du cours.

Ce cours est adapté à ceux qui ont une compréhension de base de la programmation et du web. Donc, si vous cherchez à en apprendre davantage sur les APIs et comment travailler avec elles, ce cours vidéo est un excellent point de départ.

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=WXsD0ZgxjRw) (3 heures de visionnage).
