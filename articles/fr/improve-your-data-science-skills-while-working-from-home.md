## Prérequis

Pour commencer ce voyage, vous devrez couvrir les prérequis. Peu importe le domaine spécifique dans lequel vous vous trouvez, vous devrez apprendre les prérequis suivants pour la science des données.

### Logique/Algorithmes :

Il est important de savoir pourquoi nous avons besoin d'un prérequis particulier avant de l'apprendre. Les algorithmes sont essentiellement un ensemble d'instructions données à un ordinateur pour lui faire accomplir une tâche spécifique.

L'apprentissage automatique est construit à partir de divers algorithmes complexes. Vous devez donc comprendre comment les algorithmes et la logique fonctionnent à un niveau de base avant de vous lancer dans des algorithmes complexes nécessaires pour l'apprentissage automatique. 

Si vous êtes capable d'écrire la logique pour n'importe quel puzzle donné avec les étapes appropriées, il vous sera facile de comprendre comment ces algorithmes fonctionnent et vous pourrez en écrire un pour vous-même.

Ressources : [Quelques ressources gratuites pour apprendre les structures de données et les algorithmes en profondeur](https://www.freecodecamp.org/news/these-are-the-best-free-courses-to-learn-data-structures-and-algorithms-in-depth-4d52f0d6b35a/).

### Statistiques :

Les statistiques sont une collection d'outils que vous pouvez utiliser pour obtenir des réponses à des questions importantes sur les données. 

L'apprentissage automatique et les statistiques sont deux domaines d'étude étroitement liés. Tellement liés que les statisticiens désignent l'apprentissage automatique comme des "statistiques appliquées" ou de l'"apprentissage statistique".

![Image](https://www.freecodecamp.org/news/content/images/2020/03/image-111.png)
_Source de l'image : http://me.me/_

Les sujets suivants doivent être couverts par les aspirants scientifiques des données avant de commencer l'apprentissage automatique.

* Mesures de la tendance centrale — moyenne, médiane, mode, etc
* Mesures de variabilité — variance, écart-type, score z, etc
* Probabilité — fonction de densité de probabilité, probabilité conditionnelle, etc
* Précision — vrai positif, faux positif, sensibilité, etc
* Test d'hypothèses et signification statistique — valeur p, hypothèse nulle, etc

Ressources : [Apprendre les statistiques de niveau universitaire dans ce cours gratuit de 8 heures](https://www.freecodecamp.org/news/free-statistics-course/).

### Business :

Cela dépend du domaine sur lequel vous souhaitez vous concentrer. Cela implique essentiellement de comprendre le domaine particulier et d'acquérir une expertise dans ce domaine avant de vous lancer dans un projet de science des données. Cela est important car cela aide à définir notre problème avec précision.

Ressources : [Science des données pour les entreprises](https://www.oreilly.com/library/view/data-science-for/9781449374273/)

## Collecte de données

Il est maintenant temps pour nous d'explorer toutes les façons dont vous pouvez collecter vos données. Vous ne savez jamais où vos données peuvent se cacher. Voici quelques façons de collecter vos données.

### Web scraping 

Le web scraping vous aide à collecter des données structurées sur le web, à sélectionner certaines de ces données et à conserver ce que vous avez sélectionné pour l'utilisation dont vous avez besoin.

Vous pouvez [commencer à apprendre BeautifulSoup4](https://www.freecodecamp.org/news/better-web-scraping-in-python-with-selenium-beautiful-soup-and-pandas-d6390592e251/) qui vous aide à scraper des sites web et à créer vos propres ensembles de données.

Conseil avancé : Vous pouvez automatiser les navigateurs et obtenir des données à partir de pages web interactives telles que Firebase en utilisant [Selenium](https://www.selenium.dev/). Il est utile pour automatiser les applications web et automatiser l'administration basée sur le web.

Ressources : [Web Scraping 101 en Python](https://www.freecodecamp.org/news/p/51e02007-8108-4fb9-bbbb-f3b99e81d492/)

### Serveurs cloud

Si vos données sont stockées sur des serveurs cloud tels que S3, vous devrez peut-être vous familiariser avec la façon d'obtenir des données à partir de là. Le lien suivant vous aidera à comprendre comment les implémenter en utilisant Amazon S3.

Ressources : [Commencer avec Amazon S3](https://aws.amazon.com/s3/getting-started/), [Comment déployer votre site ou application sur AWS S3 avec CloudFront](https://www.freecodecamp.org/news/how-to-host-and-deploy-a-static-website-or-jamstack-app-to-s3-and-cloudfront/)

### APIs

Il existe des millions de sites web qui fournissent des données via des APIs tels que Facebook, Twitter, etc. Il est donc important d'apprendre comment elles sont utilisées et d'avoir une bonne idée de leur implémentation.

Ressources : [Qu'est-ce qu'une API ? En anglais, s'il vous plaît](https://www.freecodecamp.org/news/what-is-an-api-in-english-please-b880a3214a82/), [Comment construire une API JSON avec Python](https://www.freecodecamp.org/news/build-a-simple-json-api-in-python/), et [Commencer avec l'API Python](https://www.dataquest.io/blog/python-api-tutorial/).

## Prétraitement des données

Ce sujet inclut tout, de la nettoyage des données à l'ingénierie des caractéristiques. Cela prend beaucoup de temps et d'efforts. Nous devons donc consacrer beaucoup de temps à l'apprendre.

![Image](https://www.freecodecamp.org/news/content/images/2020/03/image-113.png)
_Source de l'image : [https://www.pinterest.com/pin/293648838181843463/](https://www.pinterest.com/pin/293648838181843463/" rel="noopener)_

**Le nettoyage des données** implique différentes techniques en fonction du problème et du type de données. Les données doivent être nettoyées des données irrelevantes, des erreurs de syntaxe, des incohérences de données et des données manquantes. Le guide suivant vous aidera à commencer avec le nettoyage des données.

Ressources : [Guide ultime du nettoyage des données](https://towardsdatascience.com/the-ultimate-guide-to-data-cleaning-3969843991d4)

**Le prétraitement des données** est une étape importante dans laquelle les données sont transformées ou encodées, afin que la machine puisse les analyser facilement. Cela nécessite du temps ainsi que des efforts pour prétraiter différents types de données, y compris les données numériques, textuelles et image.

Ressources : [Concepts de prétraitement des données](https://towardsdatascience.com/data-preprocessing-concepts-fa946d11c825), [Tout ce que vous devez savoir sur le prétraitement de texte pour le NLP et l'apprentissage automatique](https://www.freecodecamp.org/news/all-you-need-to-know-about-text-preprocessing-for-nlp-and-machine-learning-bc1c5765ff67/), [Prétraitement pour l'apprentissage profond](https://www.freecodecamp.org/news/https-medium-com-hadrienj-preprocessing-for-deep-learning-9e2b9c75165c/).

## Intelligence d'affaires

Après avoir obtenu les résultats de votre projet, il est maintenant temps de prendre des décisions commerciales à partir de ces résultats. L'intelligence d'affaires est une suite de logiciels et de services qui aide à transformer les données en intelligence et en connaissances exploitables.

Cela peut être fait en créant un tableau de bord à partir de la sortie de notre modèle. **Tableau** est un outil puissant et le plus rapide de visualisation de données utilisé dans l'industrie de l'intelligence d'affaires. Il aide à simplifier les données brutes dans un format très facilement compréhensible. L'analyse des données est très rapide avec Tableau et les visualisations créées sont sous forme de tableaux de bord et de feuilles de calcul.

Ressources : [Commencer avec Tableau](https://www.tableau.com/learn/tutorials/on-demand/getting-started), [Cours Tableau pour la science des données](https://www.freecodecamp.org/news/tableau-for-data-science-and-data-visualization-crash-course/)
