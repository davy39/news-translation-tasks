Apprenez à écrire du code plus professionnel avec Dylan Israel. Dylan enseigne des outils et des principes pour améliorer votre code qu'il a trouvés utiles dans son travail en tant que développeur.

Ce tutoriel couvre :

* Linting
* Variables
* Fonctions
* Principes SOLID
* Le Principe de Responsabilité Unique
* Le Principe Ouvert/Fermé
* Le Principe de Substitution de Liskov
* Le Principe de Ségrégation des Interfaces
* Le Principe d'Inversion des Dépendances
* et plus encore

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=ZihKWQXRBmE) (2 heures de visionnage).
