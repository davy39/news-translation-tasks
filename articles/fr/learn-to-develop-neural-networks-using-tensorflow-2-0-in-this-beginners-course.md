Apprenez à utiliser TensorFlow 2.0 dans ce cours vidéo complet de Tech with Tim. Ce cours vous montrera comment créer des réseaux de neurones avec Python et TensorFlow 2.0.

Cette version de TensorFlow rend les modèles plus faciles à comprendre, à maintenir et à développer. Que vous soyez nouveau dans TensorFlow ou que vous souhaitiez simplement en savoir plus sur les mises à jour de la version 2.0, ce cours est fait pour vous.

Voici quelques-uns des sujets que vous apprendrez dans le cours :

* Qu'est-ce qu'un réseau de neurones ?
* Comment charger et examiner les données
* Comment créer un modèle
* Comment utiliser le modèle pour faire des prédictions
* Comment classifier du texte
* Comment installer TensorFlow GPU sur Linux
