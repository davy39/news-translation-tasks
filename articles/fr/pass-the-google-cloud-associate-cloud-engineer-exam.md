Préparez-vous à l'examen Google Cloud Associate Cloud Engineer (ACE) et réussissez-le !

Nous venons de publier un cours mis à jour sur Google Cloud Associate Cloud Engineer sur la chaîne YouTube freeCodeCamp.org. Andrew Brown d'ExamPro a développé ce cours.

Les Associate Cloud Engineers déploient des applications, surveillent les opérations et gèrent des solutions d'entreprise. Ils utilisent Google Cloud Console et l'interface de ligne de commande pour effectuer des tâches courantes basées sur la plateforme afin de maintenir une ou plusieurs solutions déployées qui exploitent des services gérés par Google ou auto-gérés sur Google Cloud.

Voici les sections de ce cours :

* Introduction

* Accès à l'identité et configuration des ressources

* Configuration de la facturation

* Déploiement serverless et d'applications

* Options de stockage de données

* Déploiement de Compute Engine

* Déploiement de Google Kubernetes Engine (GKE)

* Ressources de mise en réseau

* Infrastructure as Code (IaC)

* Opérations de Compute Engine

* Solutions de stockage et de base de données

* Surveillance et journalisation

* Services d'identité et d'annuaire cloud

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/OlAmyf8_4O4) (11 heures de visionnage).
