GraphQL a transformé la manière dont les développeurs interagissent avec les API et les applications basées sur des données. Et il y a de bonnes chances que vous deviez le connaître en tant que développeur.

Nous venons de publier un cours complet sur GraphQL sur la chaîne YouTube de freeCodeCamp.org. Dans ce cours complet, conçu spécialement pour ceux qui sont nouveaux dans GraphQL, vous apprendrez les connaissances et les compétences nécessaires pour créer des applications robustes et efficaces basées sur des données.

The Net Ninja a créé ce cours. Il est un créateur prolifique de cours techniques et est également l'un des instructeurs GraphQL les plus vus sur Internet. Il est donc la personne idéale pour enseigner ce nouveau cours sur GraphQL.

GraphQL est un langage de requête pour votre API, et un runtime pour exécuter ces requêtes en utilisant un système de types que vous définissez pour vos données. Développé par Facebook en 2012 et publié en tant que projet open-source en 2015, GraphQL offre une alternative plus efficace, puissante et flexible à l'API REST traditionnelle. Au lieu d'avoir plusieurs endpoints pour différents besoins de données, avec GraphQL, vous pouvez demander exactement ce dont vous avez besoin, le tout en une seule requête. Cela peut entraîner une récupération de données plus rapide et plus précise.

Ce cours vous fournira une base solide en GraphQL, vous permettant de comprendre pleinement ses principes fondamentaux et de voir comment il diffère des API REST traditionnelles.

Voici les sections de ce cours :

* Qu'est-ce que GraphQL ?
* Bases des requêtes
* Création d'un serveur GraphQL (avec Apollo)
* Schéma et types
* Fonctions de résolution
* Variables de requête
* Données associées
* Mutations (ajout et suppression de données)
* Mutation de mise à jour

À la fin de ce cours, vous serez équipé des compétences pratiques nécessaires pour concevoir, implémenter et optimiser des applications basées sur des données en utilisant GraphQL. Le style d'enseignement captivant de The Net Ninja et ses tutoriels étape par étape garantissent que vous pourrez suivre et appliquer immédiatement ce que vous avez appris.
