Python est considéré comme l'un des langages de programmation les plus populaires au monde et sa popularité ne cesse de croître. C'est un excellent premier langage à apprendre.

Nous venons de publier un tout nouveau cours de programmation Python sur la chaîne YouTube freeCodeCamp.org.

Je m'appelle Beau Carnes et j'ai créé ce cours. J'ai précédemment créé l'un des cours de JavaScript les plus populaires sur YouTube ainsi que de nombreux tutoriels Python. C'est mon meilleur cours à ce jour.

Ce cours complet sur Python est destiné aux débutants. Vous n'avez besoin d'aucune expérience préalable en programmation pour suivre et tout ce dont vous avez besoin pour coder en Python est un navigateur web.

Dans ce cours, je vous enseignerai tous les aspects fondamentaux du langage de programmation Python et je simplifierai les sujets plus complexes.

Python excelle dans une grande variété de scénarios tels que les scripts Shell, l'automatisation des tâches et le développement web. C'est le langage de choix pour l'analyse de données et l'apprentissage automatique, mais il peut également s'adapter pour créer des jeux et travailler avec des dispositifs embarqués.

Nous allons plonger directement dans le sujet afin que vous puissiez commencer à coder votre premier programme Python dès que possible. Dans la première section, vous serez guidé ligne par ligne sur la façon de créer un jeu de pierre-papier-ciseaux.

Après le premier projet, j'entrerai dans plus de détails sur chacune des principales fonctionnalités de Python. Cette section est complète et détaillée.

Dans la section finale, vous utiliserez ce que vous avez appris pour coder un jeu de blackjack avec moi qui vous guiderai à chaque étape. J'utiliserai ce projet comme une opportunité pour vous enseigner davantage sur la programmation orientée objet en Python.

Tout au long du cours, il y aura une petite répétition de certains concepts clés de la programmation Python pour vous assurer une compréhension approfondie du langage.
