Que vous soyez un développeur expérimenté ou que vous débutiez dans le monde du développement web, il est essentiel de se tenir au courant des derniers outils et frameworks.

Et vous pouvez en apprendre davantage sur deux outils populaires de développement web dans le nouveau cours que nous venons de publier sur la chaîne YouTube freeCodeCamp.org. Dans ce cours, vous apprendrez à créer un portfolio web en utilisant SvelteKit et TailwindCSS.

SvelteKit est bien plus qu'un simple effet de mode ; c'est un changement de jeu dans la création d'applications web. Il offre un rendu côté serveur intégré, un routage et de nombreuses autres fonctionnalités qui simplifient le travail du développeur.

Mais qu'est-ce que le développement web sans une touche d'esthétique ? Voici Tailwind CSS. Un framework CSS basé sur les utilitaires, Tailwind permet aux développeurs de créer rapidement des interfaces utilisateur personnalisées sans se perdre dans du CSS personnalisé.

La combinaison de SvelteKit et Tailwind CSS promet une expérience de développement comme aucune autre, permettant la création d'applications web interactives, réactives et belles.

James McArthur a créé ce cours. Il est un développeur full-stack avec un talent pour créer des tutoriels engageants. James plonge en profondeur dans le processus de création d'un portfolio web à partir de zéro, ce qui en fait une vidéo parfaite pour ceux qui souhaitent présenter leurs compétences et projets à des employeurs potentiels ou simplement marquer leur présence sur Internet.

![Image](https://www.freecodecamp.org/news/content/images/2023/10/image-3.png)
_Ceci est la page d'accueil de la page que vous allez créer._

Le tutoriel adopte une approche pratique, guidant les spectateurs étape par étape. À la fin, non seulement vous aurez un portfolio web entièrement fonctionnel, mais vous l'aurez également en ligne sur Internet. Comme le dit amusamment James, "vous pouvez l'envoyer à votre mère".

La démonstration du portfolio présente une page d'accueil captivante pour un personnage fictif, Samuel Oak (ou comme James l'appelle, "le petit faux James"). Il possède des éléments esthétiquement plaisants comme des effets d'animation de particules qui ajoutent de la profondeur et du dynamisme à la page. La navigation est fluide, menant à des sections qui affichent des projets, un segment "à propos de moi" et un tableau unique qui vous présente de manière ludique mais efficace comme le meilleur candidat pour les demandes d'emploi.

Pour toute personne désireuse d'établir une présence en ligne frappante, ce cours est un excellent point de départ. Non seulement il enseigne les aspects techniques du développement web, mais il souligne également l'importance de l'esthétique et de l'expérience utilisateur.

Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/-2UjwQzxvBQ) (1 heure de visionnage).
