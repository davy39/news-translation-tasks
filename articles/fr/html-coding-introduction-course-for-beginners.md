Êtes-vous intéressé par la programmation mais ne l'avez-vous jamais essayée auparavant ? Êtes-vous curieux d'apprendre à créer votre propre site web ou application ? Si c'est le cas, alors ce cours vidéo est fait pour vous !

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous enseignera les bases du HTML et comment commencer à coder.

Présenté par Ania Kubów, une codeuse expérimentée et enseignante chevronnée, ce cours vous introduira aux bases du HTML. En seulement deux heures, vous apprendrez les fondamentaux de ce langage de programmation essentiel, qui fournit la structure de tous les sites web et applications.

Même si vous n'avez aucune expérience préalable en programmation, Ania vous guidera à travers chaque étape du processus, en commençant par les outils dont vous avez besoin pour démarrer. À la fin du cours, vous aurez les compétences et les connaissances nécessaires pour commencer à construire vos propres projets web et poursuivre votre parcours en programmation.
