Gatsby est un framework pour créer des sites web et des applications web ultra-rapides. Alimenté par React et GraphQL, Gatsby vous offre tout ce dont vous avez besoin pour construire et lancer votre prochain projet.

Dans ce cours d'Andrew Mead, vous apprendrez tout ce dont vous avez besoin pour construire et lancer votre premier site avec Gatsby. Vous construirez un site web à partir de zéro et apprendrez à le déployer en production avec des outils comme GraphQL. Voici quelques-unes des choses que vous apprendrez :

1. Comment créer un site Gatsby
2. Travailler avec les pages Gatsby
3. Lier les pages entre elles avec Gatsby
4. Créer des composants de page partagés
5. Créer des mises en page de pages Gatsby
6. Styliser les projets Gatsby
7. Styliser Gatsby avec les modules CSS
8. Les données Gatsby avec GraphQL
9. GraphQL Playground
10. Source de contenu à partir du système de fichiers
11. Travailler avec les publications Markdown
12. Générer des slugs pour les publications
13. Générer des pages dynamiquement
14. Rendre les données des publications dans un modèle de blog
15. Ajouter des images aux publications
16. Commencer avec Contentful
17. Rendre les publications Contentful
18. Pages dynamiques à partir de Contentful
19. Pages 404 et React Helmet
20. Déployer votre site Gatsby

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=kzWIUX3CpuI) (5 heures de visionnage).
