Syncfusion fournit une suite de composants d'interface utilisateur pour créer des applications web, de bureau et mobiles puissantes.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à exploiter la puissance des composants d'interface utilisateur Syncfusion au sein d'une application serveur Blazor. Vous apprendrez également à intégrer la technologie Microsoft Identity dans l'application Blazor pour utiliser les fonctionnalités de connexion, d'enregistrement, d'autorisation et d'authentification.

Gavin Lon a développé ce cours. Gavin est un développeur et enseignant expérimenté. Il a créé de nombreux cours populaires, à la fois sur sa propre chaîne et sur la chaîne YouTube freeCodeCamp.org.

L'utilisation des composants Syncfusion permet à un développeur d'inclure des fonctionnalités d'interface utilisateur puissantes dans une application grâce à la mise en œuvre de code déclaratif simple.

Vous apprendrez à intégrer les composants Syncfusion pertinents dans une application de gestion des ventes basée sur le web, entièrement fonctionnelle, qui pourrait être très utile à un service commercial et qui est également agréable à utiliser. Et vous le ferez avec des quantités minimales de code déclaratif.

L'application de gestion des ventes est multiplateforme et entièrement réactive lorsqu'elle est consultée sur plusieurs tailles d'écran.

Voici quelques-unes des choses que vous apprendrez à faire dans ce cours :

* Créer le projet Blazor via Visual Studio 2022
* Comprendre le composant DataGrid de Syncfusion
* Créer la base de données en utilisant les migrations Ef Core Code First
* Intégrer le composant DataGrid de Syncfusion dans l'application
* Intégrer le composant ListView de Syncfusion dans l'application de gestion des ventes
* Intégrer les graphiques Syncfusion dans l'application de gestion des ventes pour afficher les données analytiques des commandes de vente
* Créer des tableaux de bord pour les employés
* Intégrer le diagramme Syncfusion dans l'application de gestion des ventes
* Intégrer le planificateur Syncfusion dans l'application de gestion des ventes
* Intégrer Microsoft Identity dans l'application de gestion des ventes
