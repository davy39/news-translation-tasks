Apprenez Svelte dans ce tutoriel vidéo complet de [Classsed](https://www.youtube.com/channel/UC2-slOJImuSc20Drbf88qvg). Vous allez construire votre première application Svelte qui utilise plusieurs routes, gère des formulaires et consomme une API CRUD complète.

Svelte est une nouvelle approche pour construire des interfaces utilisateur. C'est un framework qui n'en est vraiment pas un. Alors que les frameworks traditionnels comme React et Vue font la majorité de leur travail dans le navigateur, Svelte déplace ce travail dans une étape de compilation qui se produit lorsque vous construisez votre application.

Après avoir regardé ce tutoriel vidéo, vous aurez une bonne idée de comment Svelte fonctionne. Et vous pourriez même décider que vous l'aimez plus que React ou Vue.

Vous pouvez regarder le cours vidéo complet sur la chaîne YouTube [freeCodeCamp.org](https://www.youtube.com/watch?v=vhGiGqZ78Rs) (1 heure de visionnage).
