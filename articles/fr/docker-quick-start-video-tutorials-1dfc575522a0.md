Par Mark Hopson

#### Nouveau sur Docker ? Alors cette série vidéo vous aidera à comprendre les bases et à commencer rapidement.

![Image](https://cdn-media-1.freecodecamp.org/images/1*easlVE_DOqRDUDkVINRI9g.png)

Les vidéos suivantes sont destinées à aider toute personne nouvelle sur Docker à se familiariser avec les bases. Les vidéos sont courtes car j'essaie de couvrir uniquement l'essentiel pour que vous puissiez coder rapidement.

**Quelques informations de contexte.** Ces vidéos ont été initialement réalisées pour mon employeur ([Flipp Corp.](https://corp.flipp.com/)) afin d'évangéliser Docker, mais j'ai pensé qu'elles pourraient être utiles ici aussi.

Je suis relativement nouveau sur Docker (et dans la création de screencasts), et beaucoup de ce que j'ai mis dans ces vidéos étaient des choses que j'ai trouvées manquantes dans d'autres. N'hésitez pas à laisser des commentaires ou des suggestions dans les commentaires ci-dessous. Merci !

### Partie 1 : Qu'est-ce que Docker et les concepts clés (13 min)

Cette première vidéo explique pourquoi Docker existe et le problème qu'il est conçu pour résoudre. Nous passerons en revue le fonctionnement de Docker à un niveau élevé, ainsi que quelques concepts clés (Images Docker, Conteneurs et le Dockerfile).

### Partie 2 : Commencer avec Docker sur un Mac (21 min)

Dans ce screencast, nous installerons Docker pour OSX et construirons notre première image Docker en utilisant une simple application NodeJS.

### Partie 3 : Utilisation de CMD dans Dockerfile (3 min)

Dans ce court screencast, nous passerons en revue la commande CMD et comment l'utiliser.

### Partie 4 : Débogage des builds Docker cassés (4 min)

Dans ce screencast, nous passerons en revue une tactique utile pour déboguer les builds Docker cassés. Essentiellement, nous exécutons le programme "Shell" à l'intérieur d'une image Docker partiellement construite, pour inspecter ce qui échoue et pour déterminer comment le réparer.

### Partie 5 (À venir) : Docker Compose

Il n'y a pas encore de vidéo ici, mais s'il y a de l'intérêt, je créerai une autre vidéo qui couvrira ce qu'est Docker Compose et pourquoi c'est génial. En attendant, voici un screencast de [Full Stack Talks](https://www.youtube.com/channel/UCpq2_V-9AYYUHSsjoJZPUTg) que j'ai trouvé très utile.
