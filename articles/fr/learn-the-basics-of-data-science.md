Apprenez les composants de base de la Data Science dans ce cours pour débutants de Coding With Max. Ce cours couvre :

1.  Statistiques : apprenez les types de données que vous rencontrerez, les types de moyennes, la variance, l'écart-type, la corrélation, et plus encore. 
2.  Visualisation de données : apprenez pourquoi nous devons visualiser nos données, et les différentes façons de le faire (graphiques à 1 variable, graphiques à 2 variables et graphiques à 3 variables.) 
3.  Programmation : apprenez pourquoi la programmation nous aide avec la data science, y compris la facilité d'automatisation et les bibliothèques Python recommandées pour commencer avec la data science. 

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=N6BghzuFLIg) (1,5 heure de visionnage).
