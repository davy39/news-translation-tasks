Apprendre Laravel en créant un clone d'Instagram dans ce cours complet pour débutants développé par Coder's Tape. Laravel est un framework web PHP gratuit et open-source utilisé pour créer des applications web. Ce cours vous préparera à utiliser Laravel pour vos propres projets web.

Ce cours couvre des sujets tels que :

* Qu'est-ce que Laravel ?
* Installation de Laravel
* Configuration des vues et des contrôleurs
* Authentification avec Laravel
* Front End avec Node et NPM
* Utilisation de SQLite avec Laravel
* Contrôleurs de ressources RESTful
* Passage de données aux vues
* Liaison de modèle de route
* Laravel Telescope
* Envoi d'e-mails aux utilisateurs
* et plus...

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=ImtZ5yENzgE) (4,5 heures de visionnage).
