Apprenez à coder le jeu classique Tetris en utilisant Python et Pygame. Ce tutoriel complet est destiné aux programmeurs intermédiaires ayant une certaine expérience, mais tout le monde est le bienvenu pour suivre.

À la fin de ce tutoriel de Tech With Tim, vous aurez créé un jeu Tetris entièrement fonctionnel.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=zfvxp7PgQ6c) (2 heures de visionnage).
