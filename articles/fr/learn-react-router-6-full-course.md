React Router est une bibliothèque puissante qui vous aide à construire des applications monopages avec facilité. Elle vous permet de créer plusieurs routes, des routes imbriquées et de gérer les transitions entre elles, tout en maintenant une base de code propre et organisée.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous enseignera tout sur React Router 6.

Dans ce cours, vous construirez une application web de style Airbnb appelée "VanLife", dédiée à la location de vans de voyage pour votre prochain grand road trip. En cours de route, vous apprendrez toutes les parties importantes de React Router 6, telles que les routes de mise en page et d'index, les routes imbriquées, le filtrage des résultats avec des paramètres de recherche, la protection des routes pour les utilisateurs authentifiés, et plus encore.

Bob Ziroll, responsable de l'éducation chez Scrimba, a créé ce cours. Il est conçu pour avoir une forte participation de votre part, en tant qu'étudiant.

Avant de suivre ce cours, vous devriez déjà bien maîtriser HTML, CSS, JavaScript et React. Mais si vous êtes prêt à faire passer vos compétences en React Router au niveau supérieur, ce cours est fait pour vous.

Le cours est divisé en plusieurs sections, incluant une introduction à React Router 6, les applications multipages vs. monopages, la configuration de React Router avec BrowserRouter, la création de routes et de chemins, et la construction de l'application VanLife. Vous apprendrez également les nouvelles API de routage de données inspirées de Remix, incluant les Loaders et les Actions.

Le cours est structuré de manière à ce que vous puissiez suivre chaque leçon et construire l'application VanLife en temps réel. Vous aurez également l'opportunité de participer à des défis qui testent votre compréhension du matériel.

À la fin de ce cours, vous aurez construit une application réelle en utilisant React Router 6, et vous serez prêt à relever des projets encore plus complexes à l'avenir.

Alors, qu'attendez-vous ? Regardez le [cours complet sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/nDGA3km5He4) (10 heures de visionnage).
