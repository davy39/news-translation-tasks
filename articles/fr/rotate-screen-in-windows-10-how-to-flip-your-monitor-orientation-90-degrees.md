Windows 10 dispose d'une fonction intégrée qui vous permet de faire pivoter ou d'inverser l'écran de votre moniteur, tout comme vous pouvez le faire sur votre smartphone.

Dans cet article, je vais vous montrer comment faire pivoter ou inverser le moniteur de votre machine Windows 10 dans plusieurs orientations telles que le paysage inversé, le portrait et le portrait inversé.

## Pourquoi voudriez-vous faire pivoter l'écran de votre moniteur ?

Eh bien, vous avez peut-être appuyé par erreur sur une combinaison de touches de votre clavier et l'écran de votre ordinateur apparaît maintenant inversé.

Ou, si vous travaillez avec beaucoup d'infographies, ou si vous êtes designer, il peut y avoir des moments où vous souhaitez voir un design ou un autre projet en mode portrait et paysage inversé. Et pour cela, vous devrez faire pivoter l'écran de votre ordinateur.

## Comment faire pivoter l'orientation de votre moniteur dans Windows 10

**Étape 1** : Faites un clic droit dans un espace vide sur le bureau Windows et sélectionnez Paramètres d'affichage :
![ss-1-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-1-1.jpg)

**Étape 2** : Cliquez sur le menu déroulant Orientation de l'affichage :
![ss-2-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-2-1.jpg)

**Étape 3** : Sélectionnez l'orientation souhaitée. Les options disponibles sont paysage (par défaut), paysage inversé, portrait et portrait inversé.
![ss-3-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-3-1.jpg)

**Étape 4** : Pour conserver les modifications apportées, cliquez sur le bouton « Conserver les modifications ». Pour revenir à l'orientation précédente, cliquez sur « Annuler ».

Si vous ne cliquez sur aucune des deux options, l'orientation sera rétablie à la précédente en 15 secondes.
![ss-6-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-6-1.png)

Le mode portrait ressemble à ceci :

### Raccourcis clavier pour faire pivoter votre écran dans Windows

Vous pouvez également utiliser des raccourcis clavier pour faire pivoter l'écran de votre moniteur :

- `CTRL` + `ALT` + `Flèche Haut` pour le mode paysage
- `CTRL` + `ALT` + `Flèche Droite` pour le mode portrait
- `CTRL` + `ALT` + `Flèche Bas` pour le mode paysage inversé
- `CTRL` + `ALT` + `Flèche Gauche` pour le mode portrait inversé

Gardez à l'esprit que ces raccourcis clavier ont été désactivés dans la dernière version de Windows 10. Mais ils devraient toujours fonctionner sur les versions précédentes de Windows.
