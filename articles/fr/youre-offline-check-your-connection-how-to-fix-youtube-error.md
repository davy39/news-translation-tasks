Avez-vous déjà essayé de vous connecter à YouTube pour regarder des vidéos, seulement pour voir l'erreur "Vous êtes hors ligne. Vérifiez votre connexion" ?

![ss1-2](https://www.freecodecamp.org/news/content/images/2023/05/ss1-2.png)

Cette erreur peut se produire à la fois sur le site web de YouTube et sur les applications mobiles pour Android et iPhone. Dans le cas des applications mobiles, vous pourriez obtenir l'erreur "Vous êtes hors ligne. Regardez les téléchargements sans connexion."

![Screenshot_20230504-105247-1](https://www.freecodecamp.org/news/content/images/2023/05/Screenshot_20230504-105247-1.png)

C'est parce que vous avez des vidéos téléchargées sur cette application mobile. Si vous n'avez pas de vidéos téléchargées, le message d'erreur est le même que celui que vous verriez sur la version web de YouTube.

Dans cet article, je vais vous montrer comment corriger cette erreur sur les ordinateurs et les téléphones mobiles afin que vous puissiez recommencer à regarder vos vidéos préférées sur YouTube. Mais avant cela, examinons ce qui cause cette erreur.

## Qu'est-ce qui cause l'erreur YouTube "Vous êtes hors ligne. Vérifiez votre connexion." ?
La plupart des erreurs que vous rencontrez en ligne contiennent des messages d'erreur qui vous indiquent ce qui n'a pas fonctionné et comment les corriger.

Dans le cas de cette erreur, elle est liée à votre connexion Internet. C'est pourquoi une partie du message d'erreur dit "Vous êtes hors ligne". Par conséquent, les causes incluent :

* Un réseau pauvre ou instable.
* Une connexion d'adaptateur Wi-Fi défectueuse.
* Une application ou un navigateur YouTube obsolète.
* Des données de cache corrompues.

L'autre partie du message d'erreur dit "Vérifiez votre connexion". Assurez-vous donc que votre connexion Internet est stable.

Mais votre connexion Internet peut ne pas être la seule raison de cette erreur. Continuez à lire pour voir d'autres solutions que vous pouvez utiliser pour corriger cette erreur.

## Comment corriger l'erreur YouTube "Vous êtes hors ligne. Vérifiez votre connexion."
Puisque l'erreur est liée au réseau, voici quelques **solutions rapides** que vous pouvez essayer :

* Assurez-vous que votre connexion Internet est bonne.
* Actualisez le site web de YouTube ou appuyez sur réessayer dans l'application mobile YouTube.
* Actualisez l'application mobile YouTube.
* Redémarrez l'application mobile YouTube.
* Effacez le cache de votre navigateur.
* Assurez-vous que votre ordinateur ou votre téléphone mobile n'est pas en mode avion.

Si toutes les solutions rapides ci-dessus ne fonctionnent pas, continuez à lire pour voir d'autres solutions que vous pouvez utiliser pour corriger l'erreur sur les ordinateurs et les mobiles.

## Comment corriger l'erreur YouTube "Vous êtes hors ligne. Vérifiez votre connexion." sur un ordinateur

### Désactiver votre bloqueur de publicités
Les bloqueur de publicités sont des programmes logiciels qui empêchent les publicités de s'afficher sur les sites web. Ils fonctionnent en bloquant le code qui affiche les publicités, telles que les images ou les vidéos, avant qu'elles ne puissent être chargées dans le navigateur.

Bien que les bloqueur de publicités puissent améliorer l'expérience utilisateur en supprimant les publicités ennuyeuses et intrusives, ils peuvent également interférer avec certaines fonctionnalités des sites web, y compris YouTube.

Ainsi, désactiver le(s) bloqueur(s) de publicités sur votre ordinateur pourrait suffire à faire fonctionner YouTube à nouveau.

### Désactiver votre réseau privé virtuel (VPN)
Un réseau privé virtuel (VPN) peut interférer avec des sites web comme YouTube et les empêcher de fonctionner correctement en raison de restrictions liées à l'emplacement du serveur virtuel, de problèmes de configuration, de congestion du réseau, etc.

Ainsi, tout comme la désactivation des bloqueur de publicités, la désactivation de votre VPN peut être une solution à l'erreur.

### Vider le DNS de votre ordinateur dans l'invite de commandes
Parfois, vider le cache de votre navigateur peut ne pas suffire à éliminer les données de cache corrompues, car le cache peut se trouver sur votre ordinateur lui-même.

Dans ce cas, vous devez également vider le cache de votre ordinateur. Et vider votre DNS s'en charge pour vous.

Pour vider le DNS de votre PC, ouvrez l'invite de commandes et exécutez les commandes suivantes une par une :

* `ipconfig /flushdns`
* `ipconfig /release`
* `ipconfig /renew`

## Comment corriger l'erreur YouTube "Vous êtes hors ligne. Vérifiez votre connexion." sur un téléphone mobile

### Mettre à jour l'application mobile YouTube
Si vous utilisez un appareil Android, rendez-vous sur le Play Store et mettez à jour l'application YouTube si des mises à jour sont disponibles. Et si vous utilisez un iPhone, vous pouvez télécharger la dernière version de l'application YouTube sur l'App Store iOS.

Mais parfois, l'erreur se produit même si votre application mobile est à jour. Dans ce cas, la solution suivante pourrait la corriger pour vous.

### Forcer l'arrêt et effacer les données du cache de votre application mobile
Sur un téléphone Android, appuyez longuement sur l'application YouTube et sélectionnez "Infos sur l'application" :
![Screenshot_20230504-115912-1](https://www.freecodecamp.org/news/content/images/2023/05/Screenshot_20230504-115912-1.png)

Appuyez sur "Forcer l'arrêt" :
![Screenshot_20230504-1159361](https://www.freecodecamp.org/news/content/images/2023/05/Screenshot_20230504-1159361.png)

Après cela, sélectionnez "Stockage et cache" :
![Screenshot_20230504-115936-1](https://www.freecodecamp.org/news/content/images/2023/05/Screenshot_20230504-115936-1.png)

Appuyez sur "Effacer le cache" :
![Screenshot_20230504-1159361-1](https://www.freecodecamp.org/news/content/images/2023/05/Screenshot_20230504-1159361-1.png)

Si cela ne fonctionne pas, sélectionnez "Effacer le stockage" ou réinstallez l'application YouTube.

Sur un iPhone, vous pouvez supprimer et réinstaller l'application YouTube et les données du cache seront effacées.
