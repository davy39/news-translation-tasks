Redux Toolkit est un ensemble d'outils opinionnés pour le développement efficace de Redux créé par l'équipe Redux. Il s'agit de la méthode standard et recommandée pour écrire la logique Redux et gérer l'état dans vos applications JavaScript.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Redux Toolkit.

John Smilga a publié ce cours. Il a créé de nombreux cours très populaires sur la chaîne YouTube freeCodeCamp.org.

Redux est une bibliothèque JavaScript pour gérer l'état des applications. Elle est le plus souvent utilisée avec React, mais peut également être utilisée avec d'autres frameworks JavaScript.

Redux Toolkit est un ensemble d'outils qui aide à simplifier le développement Redux. Il inclut des utilitaires pour créer et gérer les stores Redux, ainsi que pour écrire les actions et les réducteurs Redux.

L'équipe Redux recommande d'utiliser Redux Toolkit chaque fois que vous devez utiliser Redux. Dans ce cours, vous apprendrez à utiliser Redux Toolkit dans vos propres applications.
