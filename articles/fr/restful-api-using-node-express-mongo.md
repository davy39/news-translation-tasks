Une API REST est un composant intégral d'une application web. Vous n'avez pas besoin d'être un développeur backend expert pour créer la vôtre.

Dans ce tutoriel, Nick Karnik démontre les éléments de base de la création d'une API REST en utilisant Node.js, Express et MongoDB.

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=o3ka5fYysBM) (1,5 heure de visionnage).
