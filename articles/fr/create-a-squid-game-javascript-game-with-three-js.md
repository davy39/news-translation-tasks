Survivriez-vous à l'émission de télévision Squid Game ? Survivriez-vous à coder l'émission de télévision Squid Game ?

Il est temps de le découvrir.

Nous venons de publier un cours JavaScript sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer le jeu Red Light / Green Light de Squid Game.

Vous utiliserez Three.js et vous apprendrez à utiliser un modèle 3D dans un jeu JavaScript.

![Image](https://www.freecodecamp.org/news/content/images/2021/11/squid-game-red-light-green-light.gif)
_Ne vous faites pas prendre._

Shuvo de la chaîne YouTube Angle Brace a développé ce cours. Il a créé de nombreux tutoriels JavaScript amusants sur sa chaîne.

Voici les sections de ce cours :

* Installation
* Bases de Three.js
* Chargement du modèle 3D
* Classe Doll
* Animation GSAP
* Création de la piste
* Classe Player
* Gestion des touches
* Fonctionnement de la poupée
* Logique du jeu

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/4HSxX-EKJjw) (1 heure de visionnage).
