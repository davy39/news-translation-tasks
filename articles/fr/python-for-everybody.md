Ce cours vise à enseigner à tous les bases de la programmation informatique en utilisant Python 3. Le cours n'a pas de prérequis et toute personne ayant une expérience modérée en informatique devrait être capable de maîtriser les matériaux de ce cours.

Le cours a été créé par le Dr. Charles Severance (alias Dr. Chuck). Il est professeur clinique à l'École d'Information de l'Université du Michigan, où il enseigne divers cours orientés technologie, y compris la programmation, la conception de bases de données et le développement Web.

Voici les sujets abordés dans ce cours :

* [Installation de Python](#heading-installation)
* Pourquoi programmer ?
* Variables, expressions et instructions
* Exécution conditionnelle
* Fonctions
* Boucles et itérations
* Chaînes de caractères
* Fichiers
* Listes
* Dictionnaires
* Tuples
* Expressions régulières
* Programmation réseau
* Utilisation des services Web
* Programmation orientée objet
* Bases de données
* Visualisation de données

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=8DvywoWv6fI) (14 heures de visionnage).
