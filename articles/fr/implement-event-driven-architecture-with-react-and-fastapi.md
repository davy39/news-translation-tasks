Un système piloté par événements est une méthode courante pour configurer des logiciels. Dans une architecture pilotée par événements, des gestionnaires d'événements sont enregistrés pour des événements spécifiques. Lorsqu'un événement se produit, les gestionnaires sont invoqués.

Une excellente façon de comprendre ce concept est de construire un projet. Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à coder un système piloté par événements avec React, FastAPI et Redis.

Antonio de Scalable Scripts a développé ce cours. Antonio est un développeur expérimenté. Nous avons publié plusieurs cours de lui et ils sont toujours excellents.

Vous apprendrez à utiliser le modèle de Machine à États Finis. Le programme changera d'état à chaque nouvel événement.

Voici les sujets abordés dans ce cours :

* Base de données Redis
* Installation de FastAPI
* Créer un Point de Livraison
* Construction de l'État
* Point de Terminaison de l'État
* Dispatching Multiple Events
* Construction de l'État sans Redis
* Construction du Frontend
