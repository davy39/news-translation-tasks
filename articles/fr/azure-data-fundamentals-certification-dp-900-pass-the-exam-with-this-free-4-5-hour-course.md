# Qu'est-ce que la certification Azure Data Fundamentals ?

La certification Azure Data Fundamentals est destinée à ceux qui recherchent un rôle lié aux données tel que **Analyste de données, Ingénieur de données ou Data Scientist**.

![Image](https://www.freecodecamp.org/news/content/images/2021/06/dp-900roadmap.jpg)

La majorité des nouvelles entreprises technologiques sont axées sur les données, ce qui signifie que la valeur de leur entreprise n'est pas directement responsable du service qu'elles fournissent, mais des données qu'elles collectent, organisent et redistribuent éventuellement.

Connaître les fondamentaux des architectures axées sur les données dès le premier jour aidera les ingénieurs cloud à comprendre à long terme le coût et l'effort nécessaires pour stocker, transformer, analyser et sécuriser les données.

# Aperçu des Azure Data Fundamentals

Les Azure Data Fundamentals couvrent les sujets suivants :

* Concepts fondamentaux des données
* Data Lakes et analyse des données
* Entrepôts de données
* Transformations ELT
* Charges de travail Hadoop et Apache Spark
* Power BI
* Bases de données relationnelles et Azure SQL
* Bases de données non relationnelles et CosmosDB

# **Comment obtenir la certification Azure Data Fundamentals ?**

Vous pouvez obtenir la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft Azure.

Microsoft Azure est partenaire avec Pearson Vue et PSI Online, qui disposent d'un réseau de centres de test dans le monde entier. Ils proposent des examens en personne et en ligne. Si vous en avez la possibilité, je vous recommande de passer l'examen en personne.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous pouvez vous inscrire et réserver votre examen.

Les frais d'examen sont de **99 USD**.

# Puis-je simplement regarder les vidéos et réussir l'examen ?

Pour une certification fondamentale comme la DP-900, vous pouvez réussir en regardant simplement le contenu vidéo sans explorer les services Azure de manière pratique par vous-même.

Azure a une fréquence de mises à jour beaucoup plus élevée que les autres fournisseurs de services cloud. Parfois, il y a de nouvelles mises à jour chaque mois pour une certification. Cependant, la DP-900 n'est pas axée sur la pratique, donc les cours d'étude sont moins susceptibles de devenir obsolètes.

* L'examen comporte **40 à 60** questions avec un temps imparti de **60 minutes**.
* L'examen contient de nombreux types de questions différents.
* Une note de passage est **d'environ 70 %.**
