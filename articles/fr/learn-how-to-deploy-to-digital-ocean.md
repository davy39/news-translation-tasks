DigitalOcean est un fournisseur d'infrastructure cloud qui offre aux développeurs des services cloud les aidant à déployer et à mettre à l'échelle des applications qui s'exécutent simultanément sur plusieurs ordinateurs.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à déployer des applications Web sur Digital Ocean. Vous apprendrez à déployer des projets Django, mais les étapes sont similaires pour déployer des projets construits avec d'autres frameworks. Stein de Code with Stein a créé ce cours. Il a créé de nombreux cours populaires.

Ce cours vidéo fournit un guide complet sur le déploiement d'applications sur DigitalOcean. En commençant par les bases, il couvre la création et la gestion de Droplets, la configuration des fonctionnalités réseau et la sécurisation de vos instances. Le cours inclut des démonstrations étape par étape sur la façon de déployer divers types d'applications. En cours de route, vous apprendrez à utiliser des technologies comme Nginx, Supervisor, Git, Gunicorn, Postresql et bien d'autres.

Voici une liste des sections couvertes dans ce cours :

* Créer un compte
  
* Créer un serveur
  
* Installer des logiciels
  
* Configurer Postgresql
  
* Utilisateurs et environnement
  
* Requirements.txt
  
* Fichier de paramètres séparé
  
* Introduction à Git
  
* Variables d'environnement
  
* Initialiser la base de données
  
* Configurer Gunicorn
  
* Configurer Supervisor
  
* Configurer Nginx
  
* SSL / Let's Encrypt
  
* Répéter la procédure

À la fin de ce cours, vous aurez une solide compréhension de l'utilisation de DigitalOcean pour déployer non seulement des applications Django, mais aussi d'autres types de projets Web utilisant une variété de frameworks et de technologies. Stein s'assure que les concepts sont clairs et fournit des conseils pratiques et des astuces de dépannage tout au long du cours. Cela rend le cours adapté aux débutants qui commencent tout juste avec les déploiements cloud, ainsi qu'aux développeurs plus expérimentés cherchant à perfectionner leurs compétences avec les services de DigitalOcean.

Regardez le cours complet sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=Td3lirXIeRI) (1 heure de visionnage).
