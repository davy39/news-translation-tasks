React est l'un des frameworks frontend JavaScript les plus populaires. 

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire vingt-cinq projets différents en utilisant React. Chaque projet est conçu pour défier et améliorer votre compréhension des capacités de React. Sangam Mukherjee a développé ce cours.

#### Pourquoi React ?

React se distingue dans le monde du développement web pour son efficacité, sa flexibilité et le vaste écosystème qu'il supporte. Il permet aux développeurs de construire de grandes applications web qui peuvent changer les données sans recharger la page, assurant une expérience utilisateur fluide. L'architecture basée sur les composants de React le rend scalable et maintenable, ce qui en fait un choix de premier ordre pour les startups et les entreprises établies.

#### À propos du cours

Que vous soyez débutant avec React ou un développeur intermédiaire cherchant à approfondir vos connaissances de React, ce cours offre quelque chose de précieux pour tout le monde. Voici les projets que vous allez construire :

1. **Accordion** : Apprenez à créer des panneaux de contenu interactifs et pliables.
2. **Générateur de couleurs aléatoires** : Plongez dans la génération de couleurs dynamiques et apprenez la gestion d'état.
3. **Notation par étoiles** : Implémentez un système de notation convivial.
4. **Curseur d'images** : Créez un moyen visuellement attrayant de parcourir des images.
5. **Bouton Charger plus** : Améliorez l'expérience utilisateur en chargeant progressivement le contenu.
6. **Vue arborescente / Menu UI / Menu de navigation récursif** : Maîtrisez les composants récursifs pour les menus imbriqués.
7. **Générateur de QR Code** : Plongez dans la génération de QR codes pour un partage rapide d'informations.
8. **Mode clair et sombre / Changement de thème** : Implémentez la personnalisation de thème selon les préférences de l'utilisateur.
9. **Indicateur de défilement** : Ajoutez des indices visuels pour le progrès de défilement sur les pages web.
10. **Onglets** : Apprenez à créer des interfaces à onglets pour une meilleure organisation du contenu.
11. **Fenêtre modale Popup** : Concevez des dialogues pop-up engageants.
12. **Trouveur de profil GitHub** : Intégrez l'API de GitHub pour rechercher et afficher des profils utilisateurs.
13. **Recherche avec autocomplétion et implémentation d'API** : Améliorez la fonctionnalité de recherche avec des suggestions d'autocomplétion.
14. **Tic Tac Toe** : Construisez le jeu classique avec une touche React.
15. **Implémentation de Feature Flag** : Apprenez à utiliser les feature flags pour contrôler le déploiement des fonctionnalités.
16. **Hook personnalisé useFetch** : Simplifiez la récupération de données avec un hook React personnalisé.
17. **Hook personnalisé useOnclickOutside** : Détectez et répondez aux clics en dehors d'un élément spécifié.
18. **Hook personnalisé useWindowResize / useResponsive** : Créez des designs responsives avec des hooks personnalisés.
19. **Défilement vers le haut et le bas** : Améliorez la navigation avec des boutons de défilement.
20. **Défilement vers une section particulière** : Améliorez l'UX en défilant vers des sections de contenu spécifiques.
21. **Application météo** : Construisez une application qui récupère et affiche les données météo.
22. **Application de recettes de cuisine** : Créez une application pour parcourir et afficher des recettes.
23. **Application de panier d'achat** : Implémentez un panier d'achat basique pour des expériences de e-commerce.
24. **Application de suivi des dépenses** : Développez une application pour suivre et gérer les dépenses.
25. **Application de blog MERN Stack** : Combinez React avec MongoDB, Express.js et Node.js pour construire une application de blog full-stack.

À la fin de ce parcours, vous aurez un portfolio rempli de projets qui démontrent vos compétences et une compréhension approfondie des principes et des meilleures pratiques de React. Regardez le cours complet [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/5ZdHfJVAY-s) (10 heures de visionnage).
