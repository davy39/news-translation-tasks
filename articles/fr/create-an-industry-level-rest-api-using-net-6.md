Il existe de nombreux tutoriels sur les API REST, mais beaucoup d'entre eux ne vous apprennent pas à créer du code prêt pour la production en utilisant les meilleures pratiques de l'industrie.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer une API REST en utilisant du code .NET de niveau industriel. Vous apprendrez à écrire du code similaire à celui utilisé dans les meilleures entreprises technologiques.

Amichai Mantinband a créé ce cours. Amichai est ingénieur logiciel chez Microsoft et avant cela, il travaillait comme instructeur en informatique à l'université.

Ce cours est une vidéo concise, du début à la fin, que les développeurs peuvent utiliser comme modèle pour intégrer/structurer des applications CRUD de niveau industriel de petite à moyenne taille.

Voici les sections couvertes dans ce cours :

* Architecture du serveur backend
* Implémentation de la logique du modèle d'API
* Refactorisation des routes
* Création de modèle pour les données de requête
* Création de l'interface de service
* Implémentation de méthodes supplémentaires
* Gestion des erreurs
* Refactorisation du contrôleur et des services
* Refactorisation de la gestion des erreurs
* Test des requêtes API

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/PmDJIooZjBE) (1 heure de visionnage).
