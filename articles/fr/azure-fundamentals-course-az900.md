# Qu'est-ce que Microsoft Azure ?

Microsoft Azure est un fournisseur de services cloud. Un fournisseur de services cloud (CSP) est une entreprise technologique qui offre plusieurs services cloud. Azure propose plus de 600 services cloud, allant du calcul au stockage, en passant par les bases de données et l'apprentissage automatique.

Azure est la deuxième plateforme cloud la plus populaire derrière Amazon Web Services (AWS). Néanmoins, de nombreuses entreprises technologiques bien établies ou de grande taille adoptent les deux plateformes cloud. Cela peut être dû au fait qu'Azure est en mesure de rivaliser en termes de coût, d'offres cloud de niche, d'intégration facile et de la familiarité des développeurs avec les logiciels Microsoft.

Azure présente trois avantages concurrentiels par rapport à d'autres plateformes cloud comme AWS et Google Cloud.

## **Avantage n°1 : Azure et Microsoft ont fortement investi dans les outils de développement**

Microsoft a développé l'éditeur de code le plus populaire à ce jour, Visual Studio Code.

Ils ont également acquis GitHub et ont intégré ses fonctionnalités directement dans Azure.

Microsoft possède également Minecraft et l'a utilisé comme outil pour initier les jeunes développeurs à la programmation et à l'informatique en cloud.

## **Avantage n°2 : Microsoft Active Directory**

De nombreuses entreprises ont investi dans Active Directory pour contrôler l'accès aux logiciels et aux postes de travail. Maintenant, elles peuvent facilement étendre cela au cloud Azure et n'ont pas à apprendre un nouvel outil pour gérer l'accès.

Lorsque vous souhaitez vendre des logiciels à de grandes entreprises, une exigence générale pour l'achat est de réaliser une authentification unique avec Microsoft Active Directory, et Azure a ici l'avantage local.

## **Avantage n°3 : Machine Learning et Intelligence Artificielle**

Je pense qu'AWS propose une meilleure offre pour le ML et l'IA, mais Azure a fait un meilleur travail pour rendre ses offres plus attrayantes pour les entreprises technologiques.

Azure divise ses services d'IA en de nombreux petits services isolés, et ils ont des certifications pour les rôles basés sur les données au niveau Associé. Cela rend ces rôles plus accessibles aux développeurs aspirants.

# **La certification Azure Foundations**

La certification Azure Foundations est la certification cloud de niveau débutant pour Microsoft Azure. Elle est communément désignée par son code de cours, l'AZ-900.

L'examen couvre les domaines suivants :

* Concepts du cloud
* Services principaux d'Azure
* Sécurité, confidentialité, conformité et confiance
* Tarification et support

Le temps d'étude moyen – même pour les personnes nouvelles dans le développement cloud – est d'une seule semaine d'étude.

Il y a également un certain transfert de connaissances si vous avez déjà obtenu la certification AWS Cloud Certified Practitioner (la certification de niveau débutant d'AWS). Mais ne vous attendez pas à réussir l'AZ-900 sans étudier.

Microsoft et Azure avaient auparavant des certifications spécifiques à des services, mais ils sont passés à des certifications basées sur les rôles (certifications basées sur les titres de poste).

Voici toutes les certifications basées sur les rôles Azure disponibles :

* Azure Fundamental (AZ-900)
* Azure Administrator (AZ-103)
* Azure Developer (AZ-203)
* Azure AI Engineer (AI-100)
* Azure Data Scientist (DP-100)
* Azure Data Engineer (nécessite à la fois le DP-200 et le DP-201)
* Azure Security (AZ-500)
* Azure Solutions Architect Expert (nécessite à la fois le AZ-300 et le AZ-301)
* Azure DevOps Engineer Expert (AZ-400)
* Azure pour les charges de travail SAP (Az-120)
* Azure IoT Developer (Az-220)

## Comment obtenir la certification ?

Vous obtenez la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft Azure.

Microsoft Azure est partenaire de Pearson Vue, un réseau de centres de test dans le monde entier. Pearson Vue vous permet de passer l'examen en personne ou en ligne. Pour passer l'examen en ligne, vous devez vous assurer d'avoir une pièce calme, sans encombrement et une webcam fonctionnelle.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous vous inscrivez et réservez votre examen.

Les frais d'examen sont de **99 USD**.

## Puis-je simplement regarder les vidéos et réussir l'examen ?

Pour la certification de niveau débutant, oui, il est possible de réussir en regardant simplement les vidéos si vous avez déjà un bagage technique.

Pour les niveaux de certification plus difficiles, à partir du niveau Associé, il est nécessaire d'utiliser des examens pratiques payants proposés par des fournisseurs tiers.

L'examen comporte **40 à 60** questions avec un temps imparti d'**1 heure**.

Une note de passage est d'**environ 70%**.

Il est courant que la plupart des personnes nouvelles dans le développement cloud obtiennent 40 % à leur première tentative d'examen pratique, 65 % à leur deuxième tentative et un score de passage de 85 % à leur troisième tentative.

Je dirais qu'il n'est pas nécessaire d'obtenir un score supérieur à 90 %. Si vous êtes dans la fourchette de 75 à 85 %, vous êtes prêt à passer votre examen.

Si vous allez utiliser des examens pratiques payants, veuillez envisager d'utiliser les [examens pratiques d'Exam Pro](https://www.exampro.co/az-900) car les fonds iront directement à m'aider à produire plus de cours cloud gratuits.
