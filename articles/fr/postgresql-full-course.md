PostgreSQL est l'un des systèmes de gestion de bases de données les plus populaires. Vous pouvez l'apprendre gratuitement dans ce cours vidéo complet d'Amigoscode.

Dans ce cours, vous vous familiariserez avec les bases de données relationnelles et vous apprendrez les instructions SQL les plus couramment utilisées.

Le cours enseigne Postgres dès le début et les sujets incluent :

* Comment créer des tables
* Comment créer des bases de données
* Opérations CRUD
* Filtrage des données
* Jointures
* Travailler avec les dates
* Fonctions d'agrégation
* Extensions Postgres
* Séquences
* Contraintes de base de données

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=qw--VYLpxG4) (4 heures de visionnage).
