Vim est un éditeur de texte hautement configurable que vous pouvez utiliser pour le développement logiciel ou tout type d'édition de texte. 

Nous venons de publier un cours complet sur Vim sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Vim.

Florian Dedov de NeuralNine a développé ce cours.

Lorsque quelqu'un maîtrise Vim, il peut souvent être plus efficace avec celui-ci qu'avec d'autres éditeurs de texte. Vim vous oblige à utiliser uniquement le clavier et pas la souris. Cela limite le temps que vous perdez à déplacer votre main entre les appareils. 

Une autre grande chose à propos de Vim est qu'il peut être installé partout et est préinstallé sur de nombreuses machines. Si vous devez vous connecter en SSH à une machine Linux, Vim est souvent l'un des seuls éditeurs de texte disponibles.

Dans ce cours, vous apprendrez à utiliser toutes les commandes Vim les plus courantes et quelques extensions Vim courantes. Vous apprendrez même comment quitter Vim.
