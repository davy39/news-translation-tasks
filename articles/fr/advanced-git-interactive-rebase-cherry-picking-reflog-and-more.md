Si vous êtes un développeur logiciel, vous utilisez probablement Git tout le temps. Mais de nombreux développeurs n'utilisent pas les commandes git avancées qui peuvent accélérer leur flux de travail.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser les commandes git avancées.

Tobias Günther a créé ce cours. Il a de nombreuses années d'expérience avec git et a même cofondé Tower, une entreprise qui crée un client GUI pour git.

Dans ce cours, vous apprendrez à utiliser des outils Git tels que :

* Rebase interactif
* Cherry-Picking
* Reflog
* Sous-modules
* Recherche et recherche

![Image](https://www.freecodecamp.org/news/content/images/2021/11/image-50.png)
_Il n'existe pas d'outils git avancés pour vous aider à écrire de meilleurs messages de commit. (Source : [xkcd](https://xkcd.com/1296/))_

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/qsTthZi23VE) (35 minutes de visionnage).
