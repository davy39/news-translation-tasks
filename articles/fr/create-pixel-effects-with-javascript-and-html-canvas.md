Vous pouvez utiliser JavaScript pour créer des effets de pixels incroyables.

Nous venons de publier un cours qui vous apprendra à utiliser JavaScript pour créer des effets vraiment cool avec des images et du texte. Vous apprendrez plusieurs effets de particules différents avec JavaScript vanilla et HTML canvas.

Ce cours a été développé par Frank Dvorak. Frank aime démontrer ses expériences de codage créatif sur sa propre chaîne YouTube, et maintenant il partage certaines de ses expériences les plus cool avec la communauté freeCodeCamp.

Dans ce cours, vous apprendrez à donner vie à votre site web avec des logos ou des en-têtes animés interactifs. Vous aurez également l'occasion de pratiquer des techniques fondamentales de codage JavaScript en utilisant des exemples amusants.

Le tutoriel commence par un projet simple et adapté aux débutants, où vous apprendrez à analyser une image pour obtenir des données de pixels et à les utiliser pour décaler les couleurs de l'image et la transformer en niveaux de gris.

Ensuite, vous utiliserez ces connaissances de base de manière puissante pour créer de nombreuses animations avancées différentes avec JavaScript vanilla et HTML canvas, entièrement à partir de zéro.

Le cours comprend quatre projets différents. Voici les sections du cours :

### Projet 1 : Introduction à la manipulation de pixels pour débutants

* Comment importer une image dans un projet HTML canvas
* Comment analyser une image pour obtenir des données de pixels
* Comment transformer une image en niveaux de gris

### Projet 2 : Pluie de pixels noir et blanc utilisant les données d'image

* Système de particules utilisant les classes JavaScript
* Relier le mouvement des particules aux données de pixels

### Projet 3 : Effets de flux de pixels

* Dégradés, filtres et autres expériences avec HTML canvas

### Projet 4 : Texte interactif avec particules

* Création de particules
* Physique des particules et interactions avec la souris
* Comment façonner les particules en lettres
* Effet de constellations inspiré de particles.js avec JavaScript vanilla
