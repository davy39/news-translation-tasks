Scikit-learn est l'une des bibliothèques de machine learning les plus populaires pour Python. Elle fournit de nombreux algorithmes d'apprentissage non supervisé et supervisé qui simplifient le machine learning.

Nous venons de publier un cours sur scikit-learn sur la chaîne YouTube freeCodeCamp.org. Ce cours vous enseignera les bases de scikit-learn afin que vous puissiez commencer à l'utiliser dans vos propres projets de machine learning.

Vincent D. Warmerdam a créé ce cours. Vincent a enseigné de nombreux concepts de machine learning sur son [site web](https://calmcode.io/) et dans son travail en tant qu'avocat de la recherche. Il a également créé des bibliothèques open source utiles qui fonctionnent avec scikit-learn.

Vincent a un talent pour décomposer des sujets complexes de manière calme et simple.

Tout d'abord, vous aurez un aperçu de scikit-learn et apprendrez quelques concepts de haut niveau.

Ensuite, vous apprendrez les outils de prétraitement. Le prétraitement a un grand impact sur les performances d'un modèle.

Dans la troisième section, vous apprendrez les métriques et comment créer des métriques personnalisées pour évaluer vos modèles de machine learning.

Puis, vous apprendrez les méta-estimateurs. Ceux-ci concernent le post-traitement de vos données.

Enfin, vous apprendrez une bibliothèque de machine learning qui s'intègre avec scikit-learn et tente de rendre le machine learning plus humain.

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/0B5eIE_1vpU) (2 heures de visionnage).
