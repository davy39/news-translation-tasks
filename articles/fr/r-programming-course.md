Apprenez le langage de programmation R dans ce cours de Barton Poulson de [datalab.cc](https://datalab.cc/). Il s'agit d'un aperçu pratique du langage de programmation statistique R, l'un des outils les plus importants en science des données.

Le cours couvre :

* Installation de R
* RStudio
* Packages
* plot()
* Graphiques à barres
* Histogrammes
* Nuages de points
* Superposition de graphiques
* summary()
* describe()
* Sélection de cas
* Formats de données
* Facteurs
* Saisie de données
* Importation de données
* Clustering hiérarchique
* Composantes principales
* Régression
* Prochaines étapes

Vous pouvez regarder le cours vidéo complet sur la chaîne YouTube [freeCodeCamp.org](https://www.youtube.com/watch?v=_V8eKsto3Ug) (2 heures de visionnage).
