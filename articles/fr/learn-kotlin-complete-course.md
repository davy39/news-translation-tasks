Kotlin est le langage de programmation le plus populaire pour développer des applications Android et il peut également être utilisé pour tout type de développement.

Nous venons de publier un cours complet sur Kotlin pour débutants sur la chaîne YouTube freeCodeCamp.org.

Alex Cristian a développé ce cours. Alex est un développeur expérimenté qui a créé plusieurs applications Android.

Ce cours vous enseignera Kotlin à partir des bases absolues jusqu'aux concepts avancés. Le cours utilise des exemples intuitifs et pratiques, et à la fin de chaque section, un défi est proposé pour tester votre compréhension.
