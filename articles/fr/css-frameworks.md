# **Semantic UI**

Semantic UI est l'un des frameworks CSS les plus populaires pour développer des projets responsives, mobile first pour le web.

### Installation

Voici un simple modèle HTML qui inclut la dernière version compilée et minifiée du CSS et du JavaScript pour la bibliothèque Semantic UI. Nous avons utilisé un CDN dans cet exemple, mais vous pouvez consulter d'autres méthodes d'installation de Semantic UI [ici](https://semantic-ui.com/introduction/getting-started.html).

```html
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Modèle Semantic UI</title>
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/semantic-ui/2.2.13/semantic.min.css">
      </head>
      <body>
        <h1>Bonjour le monde</h1>
        <!-- Ajoutez tout le code HTML ici -->
        <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
        <script src="https://cdn.jsdelivr.net/semantic-ui/2.2.13/semantic.min.js"></script>
      </body>
    </html>
```

### Ressources d'apprentissage

* La documentation officielle pour Semantic UI est disponible [ici](https://semantic-ui.com/introduction/getting-started.html).
* Consultez le dépôt GitHub open source de Semantic UI [ici](https://github.com/Semantic-Org/Semantic-UI).

# **Skeleton**

Skeleton est un modèle de base très simple pour démarrer tout projet responsive.

### Installation

Voici un simple modèle HTML qui inclut la dernière version compilée et minifiée du CSS pour la bibliothèque Skeleton.

```html
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">

        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/normalize/4.1.1/normalize.min.css">
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/skeleton/2.0.4/skeleton.min.css">
      </head>
      <body>
        <h1>Bonjour le monde</h1>
        <!-- Ajoutez tout le code HTML ici -->

      </body>
    </html>
```

# **Materialize**

Materialize est un framework CSS Responsive basé sur le langage [Material Design](https://design.google.com/spec/) de Google.

### Installation

Voici un simple modèle HTML qui inclut la dernière version compilée et minifiée du CSS et du JavaScript pour la bibliothèque Materialize.

```html
    <!DOCTYPE html>
  <html>
    <head>
      <!--Import Google Icon Font-->
      <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
      <!--Import materialize.css-->
      <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.2/css/materialize.min.css">

      <!--Let browser know website is optimized for mobile-->
      <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    </head>

    <body>
      <h1>Bonjour le monde!</h1>
      <!--Import jQuery before materialize.js-->
      <script type="text/javascript" src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
      <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/0.100.2/js/materialize.min.js"></script>
    </body>
  </html>
```

Nous avons utilisé un CDN dans cet exemple, mais vous pouvez consulter d'autres méthodes d'installation de Materialize [ici](http://materializecss.com/getting-started).

### Ressources d'apprentissage

* La documentation officielle pour Materialize est disponible [ici](https://guide.freecodecamp.org/css/css-frameworks/css-framework-materialize/www.materializecss.com/).
* Consultez le dépôt GitHub open source de Materialize [ici](https://github.com/Dogfalo/materialize).
* [Voici](https://scotch.io/tutorials/make-material-design-websites-with-the-materialize-css-framework) un tutoriel pratique sur l'utilisation de Materialize.

## **UIKit**

UIKit est un framework front-end léger et modulaire pour développer des interfaces web rapides et puissantes.

### **Installation**

Voici un simple modèle HTML qui inclut la dernière version compilée et minifiée du CSS pour la bibliothèque UIKit.

```html
<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <link rel="stylesheet" href="css/uikit.min.css" />
        <script src="js/jquery.js"></script>
        <script src="js/uikit.min.js"></script>
        <script src="js/uikit-icons.min.js"></script>
    </head>
    <body>
    </body>
</html>
```

Nous avons utilisé un CDN dans cet exemple, mais vous pouvez consulter d'autres méthodes d'installation de UIKit [ici](https://getuikit.com/docs/introduction).

### **Ressources d'apprentissage**

* La documentation officielle pour UIKit est disponible [ici](https://getuikit.com/docs/introduction).
* Consultez le dépôt GitHub open source de UIKit [ici](https://github.com/uikit/uikit).

```html
    <!DOCTYPE html>

      <head>
```
