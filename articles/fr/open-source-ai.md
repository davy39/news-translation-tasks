## D'abord, qu'est-ce que l'Open Source ?

L'open source dans le développement logiciel fait référence à la pratique de rendre le code source d'un projet disponible au public pour être utilisé, modifié et redistribué.

Avec sa nature décentralisée, il encourage la collaboration ouverte et les boucles de rétroaction qui peuvent être significatives pour aider un projet à réussir.

Certains des principes clés du développement de logiciels open source incluent :

1. **Redistribution gratuite** : Les logiciels open source ne doivent pas restreindre quiconque de vendre ou de donner le logiciel dans le cadre d'une distribution logicielle. Si c'est le cas, aucune redevance ou frais ne doit être requis pour une telle distribution.

2. **Disponibilité du code source** : Les logiciels open source doivent inclure le code source, qui doit être facilement accessible et distribuable. Si un produit logiciel n'est pas distribué avec le code source, il doit y avoir un moyen bien publicisé de l'obtenir.

3. **Pas de discrimination** : Les licences open source ne doivent pas discriminer contre un individu ou un groupe de personnes, ce qui signifie qu'elles doivent être véritablement accessibles par tous.

![Image](https://www.freecodecamp.org/news/content/images/2023/07/image-6.png align="left")

*Source : Open Source Initiative (OSI)*

Ce modèle gagne en popularité, surtout récemment avec des plateformes de partage telles que Twitter et Product Hunt. En fait, 80 % des répondants à [une enquête](https://blog.opensource.org/the-2023-state-of-open-source-report-confirms-security-as-top-issue/) ont indiqué avoir augmenté leur utilisation de l'open source au cours de l'année passée, dont 41 % ont affirmé qu'il s'agissait d'une "augmentation significative".

## Pourquoi les projets d'IA open source sont-ils importants ?

Il y a de nombreux avantages aux projets open source. Pour la communauté, l'open source ouvre la voie à une plus grande transparence, à l'innovation et au développement de logiciels piloté par la communauté.

Alors que pour votre parcours personnel de développement logiciel, il y a une variété de raisons de contribuer et d'utiliser des projets open source, telles que :

* Participer à des communautés de développement logiciel qui ont les mêmes domaines de concentration que vous

* Cela vous permet de créer des modèles d'IA beaucoup plus finement réglés

* C'est souvent accrocheur dans votre portfolio ou CV

* Vous avez l'opportunité d'apprendre d'expérimentés développeurs de logiciels (et peut-être trouver un mentor)

* C'est une avenue de choix pour le réseautage professionnel

* C'est un excellent moyen de développer votre réputation et votre présence en ligne

* Cela vous aide à améliorer votre croissance personnelle et vos capacités de résolution de problèmes

Sans les projets open source, nous perdrions une grande avenue de croissance pour la communauté de développement logiciel. Et nous devrions nous fier uniquement au travail dans une entreprise technologique afin d'acquérir de l'expérience dans le travail sur de grandes bases de code et projets.

Avec les techniques et outils d'IA croissant à un rythme si rapide, avoir des projets d'IA open source nous permet de créer et d'entraîner nos propres modèles ou outils d'IA, sans avoir à dépendre de modèles corporatifs coûteux.

L'IA étant encore une technologie en pleine émergence, cela signifie des temps excitants pour les développeurs de logiciels qui pourraient être les premiers à créer leurs propres applications ou outils avant les grands acteurs.

# Outils de Deep Learning Open Source

## TensorFlow

![Image](https://www.freecodecamp.org/news/content/images/2023/07/tensorflow.png align="left")

*Crédits : Wikipedia*

[TensorFlow](https://github.com/tensorflow/tensorflow) est un framework open-source pour le machine learning. En utilisant le concept de graphe de calcul, TensorFlow représente les opérations et le flux de données sous forme de nœuds et d'arêtes.

Les tenseurs contiennent des données, et les modèles sont construits en connectant des opérations. Les algorithmes d'optimisation entraînent les modèles en minimisant la fonction de perte. Il permet aux développeurs de créer facilement des applications alimentées par le Machine Learning avec des résultats précis.

TensorFlow fournit également des API de haut niveau comme Keras, qui simplifie le processus de construction et d'entraînement des modèles. Cela en fait un outil incroyable pour créer des applications puissantes avec le Machine Learning.

**Principalement utilisé pour :** Deep Learning

**Étoiles GitHub :** 176 000

## PyTorch + Keras

![Image](https://www.freecodecamp.org/news/content/images/2023/07/Keras-or-PyTorch-1140x337.png align="left")

[PyTorch](https://github.com/pytorch/pytorch) et [Keras](https://github.com/keras-team/keras) sont deux frameworks populaires pour travailler avec le Deep Learning. Tous deux sont des frameworks populaires utilisés pour construire et entraîner des réseaux de neurones.

Keras soutient les développeurs en fournissant une API de haut niveau qui facilite la gestion des tâches de deep learning. PyTorch utilise un modèle de calcul de graphe dynamique où le graphe est construit au fur et à mesure que les opérations sont exécutées.

Les deux frameworks offrent une création, un entraînement et un déploiement faciles des modèles avec le support des backends TensorFlow et Theano.

**Principalement utilisé pour :** Deep Learning

**Étoiles GitHub :** 68 700 (Pytorch) et 58 800 (Keras)

## Apache MXNet

![Image](https://www.freecodecamp.org/news/content/images/2023/07/mxnet_logo_2.png align="left")

*Crédits : Apache MXNet Github*

[Apache MXNet](https://github.com/apache/mxnet) est un framework open-source pour le deep learning. Il accepte presque tous les langages de programmation populaires comme Scala, Python, R, et plus.

MXNet supporte à la fois la recherche et le déploiement de modèles de machine learning de niveau production. Il utilise un modèle de programmation symbolique et impérative, offrant les avantages des graphes de calcul dynamiques et statiques. Cela permet une parallélisation efficace et un calcul distribué sur plusieurs appareils ou machines.

**Principalement utilisé pour :** Deep Learning

**Étoiles GitHub :** 20 500

## tflearn

![Image](https://www.freecodecamp.org/news/content/images/2023/07/16848261.png align="left")

*Crédits : Tflearn Github*

[tflearn](https://github.com/tflearn/tflearn) est une bibliothèque avancée de deep learning. Elle simplifie le processus de construction et d'entraînement des réseaux de neurones. Elle permet aux utilisateurs de définir et d'entraîner des réseaux de neurones en utilisant une syntaxe concise et intuitive.

tflearn fournit une large gamme de couches, optimiseurs, fonctions d'activation et métriques d'évaluation intégrés.

Elle contient également des utilitaires pour la visualisation de modèles, le traitement des données et les points de contrôle. Le code écrit dans tflearn peut facilement être converti en code TensorFlow à tout moment. C'est la fonctionnalité la plus pratique de tflearn.

**Principalement utilisé pour :** Deep Learning

**Étoiles GitHub :** 9 600

## Theano

![Image](https://www.freecodecamp.org/news/content/images/2023/07/1200px-Theano_logo.svg.png align="left")

*Crédits : Wikidata*

[Theano](https://github.com/Theano/Theano) est une autre bibliothèque Python principalement conçue pour les calculs numériques et les tâches de deep learning. Cette bibliothèque permet aux utilisateurs de définir, optimiser et évaluer des expressions mathématiques efficacement, en tirant parti de la puissance de l'accélération GPU.

Theano fonctionne en construisant un graphe de calcul où les opérations mathématiques sont représentées sous forme de nœuds et leurs dépendances sous forme d'arêtes. Ce graphe est ensuite compilé en code C hautement optimisé, permettant son exécution sur diverses architectures matérielles.

Theano est construit sur NumPy et offre une intégration étroite avec celui-ci, une utilisation transparente du GPU, une optimisation de la vitesse et de la stabilité, et une intégration dynamique du code C. Ces fonctionnalités font de Theano un outil puissant pour les chercheurs et les praticiens dans le domaine du deep learning.

**Principalement utilisé pour :** Deep Learning

**Étoiles GitHub :** 9 800

# Outils de NLP et de Modèles de Langage Open Source

## Hugging Face Transformers

![Image](https://www.freecodecamp.org/news/content/images/2023/07/transformers-1.png align="left")

*Crédits : HuggingFace*

[HuggingFace Transformers](https://github.com/huggingface/transformers) est une bibliothèque open-source populaire pour les tâches de Traitement du Langage Naturel (NLP). Elle fournit un moyen simple et efficace d'utiliser des modèles de transformers.

HuggingFace Transformers tire parti des architectures de transformers, telles que GPT, BERT et RoBERTa, qui ont connu un succès remarquable dans le NLP.

Ces modèles ont été pré-entraînés sur d'énormes quantités de données, leur permettant de capturer des représentations contextuelles profondes du langage. Avec le support de modèles pré-entraînés, des outils pour le fine-tuning et un hub de modèles collaboratif, HuggingFace Transformers permet aux développeurs et aux chercheurs de tirer parti de la puissance des architectures de transformers pour une large gamme d'applications de traitement du langage naturel.

**Principalement utilisé pour :** Traitement du Langage Naturel (NLP) et Modèles de Langage

**Étoiles GitHub :** 107 000

## Fast.ai

![Image](https://www.freecodecamp.org/news/content/images/2023/07/ae21fe11-2ab9-4556-bd6a-0566fc4474a2.png align="left")

*Crédits : Fastai Github*

[Fast.ai](https://github.com/fastai/fastai) est une bibliothèque pour travailler avec des tâches de deep learning. Elle contient des modèles pré-entraînés qui soutiennent les développeurs pour gérer des tâches avec seulement quelques lignes de code.

Fast.ai fournit une gamme de fonctionnalités, y compris des architectures de modèles, des techniques d'optimisation, de prétraitement de données et des outils de visualisation.

En offrant une bibliothèque de haut niveau avec des fonctionnalités puissantes et des fonctionnalités pré-construites, Fast.ai [rend plus facile](https://www.freecodecamp.org/news/deep-learning-with-fastai/) l'apprentissage du deep learning.

**Principalement utilisé pour :** Traitement du Langage Naturel (NLP) et Modèles de Langage

**Étoiles GitHub :** 24 200

# Outils de Vision par Ordinateur et de Traitement d'Images Open Source

## OpenCV

![Image](https://www.freecodecamp.org/news/content/images/2023/07/opencv-ar21.png align="left")

*Crédits : Wikipedia*

[OpenCV](https://github.com/opencv/opencv) est une bibliothèque populaire de Vision par Ordinateur et de Traitement d'Images développée par Intel. Les langages de programmation populaires comme Python, Java et C++ sont acceptés par cette bibliothèque.

OpenCV possède des fonctions et des algorithmes pour manipuler, analyser et comprendre des images et des vidéos. OpenCV contient toutes les fonctions nécessaires pour toute tâche de traitement d'images comme la détection de caractéristiques, la reconnaissance d'objets, le filtrage d'images et l'étalonnage de caméra.

Ces fonctions peuvent travailler avec différentes images et vidéos efficacement, les rendant polyvalentes pour diverses applications.

**Principalement utilisé pour :** Vision par Ordinateur et Traitement d'Images

**Étoiles GitHub :** 69 900

## Detectron2

![Image](https://www.freecodecamp.org/news/content/images/2023/07/0_VbMjGBHMC6GnDKUp.png align="left")

*Crédits : Kaggle*

[Detectron2](https://github.com/facebookresearch/detectron2) est une bibliothèque de nouvelle génération qui fournit des algorithmes avancés de détection et de segmentation. La segmentation panoptique, Densepose, Cascade R-CNN, les boîtes englobantes tournées, PointRend, DeepLab, ViTDet et MViTv2 sont quelques-unes des nouvelles capacités incluses dans la version mise à jour de cette bibliothèque.

Detectron2 devient une plateforme flexible et efficace pour les développeurs afin de créer des modèles de vision par ordinateur. Il fonctionne en tirant parti d'une conception modulaire et d'une architecture de réseau dorsal puissante.

Il est depuis devenu un choix populaire pour construire et déployer des modèles de détection d'objets et de segmentation d'instances en raison de sa flexibilité, de son architecture puissante et de ses fonctionnalités étendues.

**Principalement utilisé pour :** Vision par Ordinateur et Traitement d'Images

**Étoiles GitHub :** 25 500

# Bibliothèques et Outils de Machine Learning Open Source

## Stable Diffusion

![Image](https://www.freecodecamp.org/news/content/images/2023/07/189541954-46afd772-d0c8-4005-874c-e2eca40c02f2.jpeg align="left")

*Crédits : Stable Diffusion Github*

[Stable Diffusion](https://github.com/AUTOMATIC1111/stable-diffusion-webui) est un modèle de diffusion latente qui est un type de réseau neuronal artificiel génératif profond. Il permet aux utilisateurs de générer des images basées sur des détails écrits au format texte.

Vous pouvez également effectuer des tâches comme la sortie, l'inpainting et la génération de traductions image-à-image en utilisant ce modèle.

Le code et les poids du modèle de Stable Diffusion ont été publiés publiquement, et il peut fonctionner sur la plupart des matériels grand public équipés d'un GPU modeste avec au moins 8 Go de VRAM.

**Principalement utilisé pour :** Bibliothèques et Outils de Machine Learning

**Étoiles GitHub :** 89 900 (WebUI) et 57 500 (Modèle)

## MindsDB

![Image](https://www.freecodecamp.org/news/content/images/2023/07/download.png align="left")

*Crédits : MindsDB*

[MindsDB](https://github.com/mindsdb/mindsdb) est un framework AutoML open-source. Créer un modèle prédictif est beaucoup plus facile en utilisant ce framework.

MindsDB permet aux développeurs de former, tester et déployer des modèles prédictifs en utilisant moins de code. Il peut analyser et comprendre automatiquement les données, sélectionner les algorithmes appropriés et former des modèles basés sur l'entrée de l'utilisateur.

Il accepte une large gamme de formats de données et s'intègre avec des bases de données populaires comme les bases de données SQL. MindsDB accepte un large ensemble de tâches de machine learning, y compris le traitement du langage naturel, la régression, la classification, et ainsi de suite.

**Principalement utilisé pour :** Bibliothèques et Outils de Machine Learning

**Étoiles GitHub :** 17 200

## Ivy

![Image](https://www.freecodecamp.org/news/content/images/2023/07/download--1-.png align="left")

*Crédits : Wikipedia*

[Ivy](https://github.com/unifyai/ivy) est une bibliothèque de deep learning open-source en Python axée sur la recherche et le développement. Elle contient une API avancée qui simplifie le processus de construction et d'entraînement des réseaux de neurones.

En utilisant des Graphes de Calcul Dynamiques avec Différentiation Automatique, Ivy offre aux développeurs un chemin flexible et intuitif pour définir ou effectuer des calculs et des architectures de réseau. Elle permet également aux développeurs de modifier la structure du réseau pendant l'exécution.

Ivy accepte les calculs CPU et GPU et s'intègre de manière transparente avec des frameworks de deep learning populaires comme TensorFlow et PyTorch.

**Principalement utilisé pour :** Bibliothèques et Outils de Machine Learning

**Étoiles GitHub :** 11 900

# Outils d'Assistants IA et de Chatbots Open Source

## GPT Engineer

![Image](https://www.freecodecamp.org/news/content/images/2023/07/243695075-6e362e45-4a94-4b0d-973d-393a31d92d9b--1-.gif align="left")

*Crédits : Anton Osika Github*

Celui-ci est un favori personnel que j'utilise pour aider à lancer des projets.

[GPT Engineer](https://github.com/AntonOsika/gpt-engineer) est conçu pour être facile à adapter et à étendre, et il permet à votre agent d'apprendre la structure de code souhaitée. Il génère une base de code entière basée sur une invite, offrant flexibilité et facilité d'utilisation. Il offre également des invites de haut niveau et peut fournir des commentaires à l'IA qu'elle retiendra au fil du temps.

GPT Engineer facilite les transferts rapides entre les interactions IA et humaines. Tous les calculs de GPT Engineer sont reprenables et persistés dans le système de fichiers.

**Principalement utilisé pour :** Votre Assistant IA et Chatbot

**Étoiles GitHub :** 37 300

## Open Assistant

![Image](https://www.freecodecamp.org/news/content/images/2023/07/logo.svg align="left")

*Crédits : Open Assistant*

[Open Assistant](https://github.com/LAION-AI/Open-Assistant) est un projet visant à donner à tous accès à un grand modèle de langage basé sur le chat. L'objectif du projet ne se limite pas à répliquer ChatGPT mais à construire l'assistant du futur.

Les contributeurs tentent de créer un assistant capable d'écrire des emails, des lettres de motivation, d'effectuer des tâches significatives, d'utiliser des API, de rechercher dynamiquement des informations, et plus encore.

De plus, l'assistant devrait avoir la capacité d'être personnalisé et étendu par n'importe qui. Que vous soyez débutant ou expert, vous pouvez explorer ce projet utile.

**Principalement utilisé pour :** Vos Assistants IA et Chatbots

**Étoiles GitHub :** 34 300

## Fauxpilot

![Image](https://www.freecodecamp.org/news/content/images/2023/07/120729571.png align="left")

*Crédits : Fauxpilot Github*

[FauxPilot](https://github.com/fauxpilot/fauxpilot) est une alternative open-source au serveur GitHub Copilot. Ce projet vise à développer une alternative hébergée localement à GitHub Copilot. Il utilise les modèles SalesForce CodeGen dans le serveur d'inférence Triton de NVIDIA, avec le backend FasterTransformer.

L'exécution de ce projet peut nécessiter la configuration de prérequis tels que Docker compose et un GPU NVIDIA avec une capacité de calcul compatible.

**Principalement utilisé pour :** Vos Assistants IA et Chatbots

**Étoiles GitHub :** 12 700
