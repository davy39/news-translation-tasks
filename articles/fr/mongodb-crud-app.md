Dans ce tutoriel MongoDB de NoobCoder, vous apprendrez à utiliser MongoDB pour créer une application Todo CRUD complète. Ce projet utilise MongoDB, Node.js, Express.js, jQuery, Bootstrap et l'API Fetch.

Vous apprendrez à :

1. connecter l'application à un serveur MongoDB via les pilotes MongoDB
2. développer des endpoints Rest
3. créer le fichier html
4. faire des requêtes au backend en utilisant l'API Fetch
5. valider les entrées utilisateur côté serveur avec JOI.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=CyTWPr_WwdI) (75 minutes de visionnage).
