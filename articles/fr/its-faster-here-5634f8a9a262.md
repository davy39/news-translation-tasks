1. Hyperdrive
2. Warp 9

4. le web mobile fonctionnant sur un ordinateur de bureau

### Astuce de vie : Utilisez les versions mobiles des sites web

Attendez. Nous sommes en 2017. Nous avons obtenu des sites web responsives dès 2011. Existe-t-il encore des sites web qui ont des versions mobiles dédiées ?

Étonnamment, oui. Facebook, Twitter et LinkedIn en sont trois exemples. Et tous sont bien meilleurs que leurs équivalents desktop et applications mobiles.

Voici pourquoi :

* Ils sont plus rapides
* Ils ont beaucoup moins de publicités
* Ils ont des interfaces plus propres et moins distractives

Alors commençons à utiliser le web en mode zen.

### Tiny Twitter

![Image](https://cdn-media-1.freecodecamp.org/images/m6VQxSFwt8UQ371hl8iFX8CacRfpJnIBEVmO)

Vous obtenez :

* votre fil d'actualité
* vos notifications
* vos messages

Vous n'obtenez pas :

* les tendances
* les vidéos qui jouent en marge
* un bouton "moments" sur lequel cliquer accidentellement

### Little LinkedIn

![Image](https://cdn-media-1.freecodecamp.org/images/uFwWWIU1hAJK48guCCYWxZRIjPuOM0YX-QN-)

Vous obtenez :

* vos notifications
* vos messages
* vos invitations à vous connecter

Vous n'obtenez pas :

* un fil d'actualité de personnes publiant des choses sans rapport avec le travail ou les carrières
* LinkedIn vous harcelant pour "compléter votre profil"
* LinkedIn vous harcelant pour [spammer tous vos contacts email](http://fortune.com/2015/10/05/linkedin-class-action/) pour la énième fois

### Fast Facebook

![Image](https://cdn-media-1.freecodecamp.org/images/PiVIStvCRob9Ij8H4AjJnTBMlzUkIswquZBf)

Vous obtenez :

* votre fil d'actualité
* vos demandes d'ami
* vos messages
* vos notifications

Vous n'obtenez pas :

* les tendances (qui sont essentiellement des publicités sur Facebook)
* les vidéos en lecture automatique
* des boîtes de chat qui apparaissent partout
* des histoires de fake news dans la barre latérale

### Commencez à utiliser le web mobile sur votre ordinateur de bureau.

Ajoutez ces liens à vos favoris :

* Notifications Twitter version mobile : [https://mobile.twitter.com/notifications](https://mobile.twitter.com/notifications)
* Messages LinkedIn version mobile : [https://touch.www.linkedin.com/#boite-de-reception](https://touch.www.linkedin.com/#inbox)
* Messages Facebook version mobile : [https://mobile.facebook.com/messages](https://mobile.facebook.com/messages/?folder=unread)

Et il peut y avoir des versions mobiles pour d'autres sites que vous utilisez souvent. Par exemple, voici l'expérience de shopping [mobile web zen-like d'Amazon](https://www.amazon.com/gp/aw) avec sa fonctionnalité de recherche principale.

Alors rejoignez-moi là où le web est plus rapide.
