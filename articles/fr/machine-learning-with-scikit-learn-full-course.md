Scikit-learn est une bibliothèque gratuite de machine learning pour le langage de programmation Python. Nous avons publié un cours complet sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/pqNCD_5r0IU) qui vous apprendra le machine learning en utilisant scikit-learn (également connu sous le nom de sklearn).

Tout d'abord, vous apprendrez les bases du machine learning et de scikit-learn. Ensuite, vous découvrirez quelques algorithmes courants de machine learning et comment les implémenter avec scikit-learn. Enfin, vous en apprendrez davantage sur l'intelligence artificielle et la science qui la sous-tend.

Ce cours a été créé par DLAcademy. Tout au long du cours, les concepts de machine learning seront enseignés à travers des exemples pratiques.

Voici les sujets abordés :

* Installation de scikit-learn
* Tracer un graphique
* Identifier les caractéristiques et les étiquettes
* Sauvegarder et ouvrir un modèle
* Classification
* Division entraînement/test
* Qu'est-ce que le KNN ?
* Qu'est-ce que le SVM ?
* Régression linéaire
* Régression logistique vs régression linéaire
* KMeans
* Réseaux de neurones
* Surapprentissage et sous-apprentissage
* Rétropropagation
* Fonction de coût et descente de gradient
* CNNs
* Implémentation d'un reconnaisseur de chiffres manuscrits

Regardez le cours sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/pqNCD_5r0IU) (3 heures de visionnage).
