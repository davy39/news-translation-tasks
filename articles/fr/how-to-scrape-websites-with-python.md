Parfois, vous pourriez vouloir extraire des données d'un site web dans un format différent. Mais que faire si le site web n'offre aucun moyen facile d'exporter ces données ? C'est là que le web scraping entre en jeu.

Nous avons publié un cours accéléré sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à effectuer du web scraping en utilisant la bibliothèque Python Beautiful Soup.

Ce cours a été développé par Jim Ergin de JimShapedCoding. Jim utilise et enseigne Python depuis de nombreuses années.

Beautiful Soup vous permettra de collecter toutes les informations que vous souhaitez à partir de n'importe quel site web. Il pourrait s'agir d'un site bancaire, de réseaux sociaux, de Wikipedia ou de tout autre site.

Beautiful Soup fournit des méthodes pour naviguer, rechercher et modifier un arbre d'analyse. Il facilite la dissection d'un document HTML et l'extraction des données dont vous avez besoin. Et cela ne nécessite pas beaucoup de code pour écrire une application.

Dans ce cours, vous apprendrez d'abord à scraper une page HTML basique pour comprendre les concepts. Ensuite, vous passerez au scraping d'un site web réel. À la fin, vous apprendrez à stocker les informations que vous scrapez à partir du site web.

Voici les sujets abordés dans ce cours :

* Structure HTML de base, Explication des balises HTML
* Installation des packages
* Scraping de fichiers locaux
* Méthodes find et find_all() de Beautiful Soup
* Outil d'inspection du navigateur web
* Récupération des prix dans un projet de web scraping basique
* Utilisation de la bibliothèque requests pour voir le HTML d'un site web
* Scraping d'un site web en production
* Boucle à travers des objets soup.find_all() similaires
* Filtrage des emplois scrapés
* Configuration d'un projet pour scraper toutes les 10 minutes
* Stockage du paragraphe des emplois dans des fichiers texte
