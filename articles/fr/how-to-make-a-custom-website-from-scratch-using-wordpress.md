Dans un nouveau cours de 4,5 heures de GTCoding, vous apprendrez à créer un site web entièrement réactif à partir de zéro en utilisant WordPress, HTML, CSS, JavaScript et PHP. Le cours vous guide à travers la création d'un thème WordPress personnalisé.

En plus de vous donner une solide compréhension du développement de thèmes WordPress, le cours couvre également la planification, la conception d'interface utilisateur et les bases de PHP.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=KibbYf9avko) (4,5 heures de visionnage).
