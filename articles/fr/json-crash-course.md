JSON (JavaScript Object Notation) est le format de représentation de données le plus populaire. Il permet de créer des API, des fichiers de configuration et des données structurées.

Dans cette vidéo de cours accéléré sur JSON, vous apprendrez :

* ce qu'est JSON,
* pourquoi JSON est important,
* à quoi sert JSON,
* la syntaxe de JSON,
* et vous verrez plusieurs exemples de JSON.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=GpOO5iKzOmY) (11 minutes de visionnage).
