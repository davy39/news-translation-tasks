Bien sûr, vous pouvez apprendre les systèmes de bases de données à partir d'une vidéo YouTube aléatoire. Ou vous pouvez les apprendre d'un professeur d'une des universités les mieux classées au monde.

Nous avons publié un cours de 25 heures sur les systèmes de bases de données enseigné par le professeur Immanuel Trummer, PhD. Il est professeur assistant en informatique à l'Université Cornell.

Il s'agit d'une introduction aux systèmes de gestion de bases de données relationnelles et non relationnelles. Vous apprendrez à interroger les systèmes de bases de données via des langages tels que SQL (le langage de requête structuré). Ensuite, vous verrez comment les systèmes de bases de données fonctionnent en interne, comment ils stockent et indexent les données, comment ils traitent et optimisent les requêtes, et comment ils traitent les transactions tout en fournissant des garanties telles que l'isolation, l'atomicité et la durabilité (garanties ACID).

Le cours aborde les questions de conception de bases de données et discute des approches novatrices de gestion des données via les systèmes NoSQL ou NewSQL. Enfin, le cours enseignera les données de graphes, de flux et spatiales, ainsi que les systèmes spécialisés pour ces types de données.

Les conférences sont issues du cours en ligne de l'automne 2020 à Cornell par le Prof. Trummer.

Voici les sections de ce cours complet :

* Le langage de requête structuré (SQL)
* Stockage et indexation des données
* Traitement des données relationnelles
* Traitement des transactions
* Conception de bases de données
* Traitement distribué des données
* Au-delà des données relationnelles

Vous pouvez regarder le cours gratuitement sur la chaîne YouTube de freeCodeCamp.org. Le cours est si volumineux qu'il a dû être divisé en deux vidéos. [Cliquez ici pour regarder la première partie](https://youtu.be/4cWkVbC2bNE) ou commencez à regarder ci-dessous.
