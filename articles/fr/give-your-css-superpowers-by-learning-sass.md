Apprenez les bases de Sass dans ce cours vidéo gratuit de 2 heures de codeSTACKr.

Le cours commence par un bref aperçu de ce qu'est Sass ainsi que des prérequis pour incorporer Sass dans votre environnement. 

Ensuite, il couvre comment configurer un compilateur Sass dans Visual Studio Code.

Vous apprendrez tout sur : 

* Les variables Sass
* les maps
* l'imbrication (nesting)
* les fonctions
* les mixins
* l'extension (extending)
* et les opérations.

Après avoir appris les bases, vous verrez comment construire un projet réel en utilisant Sass. 

Le cours passe en revue chaque partie étape par étape, y compris le déploiement du site.

Vous pouvez regarder [la vidéo complète sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=_a5j7KoflTs) (2 heures de visionnage).
