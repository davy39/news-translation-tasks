Il existe certains caractères qui sont des symboles auxquels vous n'avez pas accès sur les claviers. En HTML, ces caractères sont appelés entités et sont des caractères réservés.

Par exemple, il arrive que vous souhaitiez taper le symbole supérieur à (`>`) ou inférieur à (`<`) tout en travaillant sur le HTML d'un site web.

Si vous tapez supérieur à comme >, il sera traité comme une balise de fermeture, et si vous tapez inférieur à comme <, il sera traité comme une balise d'ouverture.

Pour taper ces caractères, vous devez utiliser des entités HTML. C'est pourquoi j'ai compilé pour vous ici une liste complète d'entités HTML approuvées par le W3C.

## Comment utiliser les entités HTML

Pour utiliser les entités fournies dans cet article, copiez le nom de l'entité ou le code HTML des entités et collez-les directement dans votre fichier HTML.

Si vous utilisez les noms d'entités, assurez-vous qu'ils sont précédés d'un esperluette (`&`) et suivis d'un point-virgule (`;`). Par exemple, `&gt;` pour le symbole supérieur à afin d'obtenir &gt; pour créer ce symbole.

Si vous utilisez le code HTML pour taper les entités, assurez-vous qu'ils sont précédés d'un esperluette (`&`) et d'un dièse (`#`), et suivis d'un point-virgule (`;`). Par exemple, `&#62;` pour le symbole supérieur à afin d'obtenir &#62; pour créer ce symbole.

À certaines occasions, le navigateur peut ne pas reconnaître les noms d'entités, il est donc préférable d'utiliser le code HTML pour cette entité.
