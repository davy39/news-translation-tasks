Par javinpaul

Une certification Java est très respectée dans l'industrie informatique et offre à un développeur Java une reconnaissance mondiale. Elle vous aide à trouver un meilleur emploi, à obtenir un meilleur salaire, et même une meilleure augmentation ou bonus dans votre emploi actuel.

C'est pourquoi des milliers de développeurs Java tentent de se certifier chaque année, mais le succès n'est pas facile à cet examen.

La certification Java d'Oracle a un taux de réussite difficile, **proche de 65 %** pour à la fois [OCAJP](https://javarevisited.blogspot.com/2017/08/java-se-8-certification-books-ocajp8-ocpjp8.html) (Oracle Certified Associate Java Programmer) et [OCPJP](https://javarevisited.blogspot.com/2018/07/do-you-need-to-pass-ocajp-before-taking-OCPJP-Java-certification.html#axzz5KQoN3YqB) (Oracle Certified Professional Java Programmer).

Vous avez besoin d'une connaissance approfondie du langage Java et de son API pour réussir l'examen.

À noter, la préparation pour l'OCPJP n'est pas très différente de celle pour l'OCAJP, bien que vous deviez utiliser des ressources axées uniquement sur l'OCPJP. Vous devez également couvrir plus de sujets et de manière plus approfondie, car [OCPJP 8](http://www.java67.com/2017/05/top-5-books-for-ocajp8-and-ocpjp8-Java-8-certification-exam.html) est certainement un défi difficile à relever.

Pour bien vous préparer et obtenir un bon score, voici quelques conseils que je vais partager. Ces conseils m'ont aidé par le passé à obtenir plus de 90 % aux certifications.

Alors, sans plus attendre, voici mes cinq conseils pour bien vous préparer aux examens [OCAJP](http://javarevisited.blogspot.com/2013/07/2-books-to-prepare-oracle-java-7-certification-OCA-OCP-OCPJP.html#axzz4gdx9UfxA) et [OCPJP](http://javarevisited.blogspot.com/2017/01/difference-between-ocpjp-8-upgrade-exams-1Z0-813-1Z0-810.html).

### 1. Codez chaque jour

Rien ne remplace la pratique du code. Vous devez coder quotidiennement pour développer un sens du code, ce qui vous aidera à lire le code donné dans l'examen réel.

Dans le cas de [OCPJP](https://javarevisited.blogspot.com/2017/01/difference-between-ocpjp-8-upgrade-exams-1Z0-813-1Z0-810.html#axzz5TRkZ9aeh), cela devient encore plus important car une attention aux détails est nécessaire pour comprendre chaque question de code avant de sélectionner les options.

### 3. Lisez un bon livre d'étude

Ils couvrent tous les sujets de l'examen, donnent des questions pratiques à la fin de chaque chapitre et partagent de nombreux conseils spécifiques à l'examen que vous ne trouverez dans aucun livre Java ordinaire.

Les simulateurs d'examen vous aideront à développer la vitesse et la précision nécessaires pour répondre à toutes les questions en 2,5 heures.

Je vous suggère de continuer à faire des tests blancs jusqu'à obtenir 80 % de manière constante avant de passer l'examen réel.

### 5. Participez régulièrement aux forums

Vous devez rester motivé et à jour sur l'examen et autres détails. Dans cette optique, rien de mieux que de participer à des forums comme [CodeRanch](http://coderanch.com/) ou le forum officiel de certification Java d'Oracle [Oracle](https://community.oracle.com/community/technology_network_community/certification/).

Vous pouvez poser des questions, clarifier vos doutes et vous familiariser avec d'autres candidats qui préparent les examens.

Vous pourrez peut-être trouver des partenaires d'étude qui peuvent se préparer avec vous pour les examens.
