ASP.NET Core est un framework de développement web open source pour créer des applications web sur la plateforme .NET. Bien qu'initialement conçu uniquement pour Windows, il est désormais disponible sur macOS et Linux également.

Nous avons publié un cours de 3 heures sur la chaîne YouTube freeCodeCamp.org afin que vous puissiez apprendre ASP.NET Core 3.1. Le cours est destiné aux débutants, mais il est attendu que vous ayez quelques connaissances en C#. (Si vous souhaitez apprendre le C#, consultez [ce excellent cours sur C#](https://www.youtube.com/watch?v=GhQdlIFylQ8).)

Dans ce cours, vous apprendrez d'abord l'histoire et les bases d'ASP.NET Core. Ensuite, vous apprendrez à créer une petite application de liste de livres avec des opérations CRUD. Vous apprendrez à utiliser Entity Framework pour l'intégration avec une base de données avec ASP.NET Core Razor Pages.

Vous apprendrez également à créer une application de liste de livres en utilisant ASP.NET MVC et verrez comment utiliser DataTables avec des appels API dans un projet Razor.

Ce cours a été développé par Bhrugen Patel. Regardez-le ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=C5cnZ-gZy2I).
