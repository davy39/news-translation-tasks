Êtes-vous un débutant complet intéressé par l'apprentissage du développement web ? Ne cherchez pas plus loin !

Nous venons de publier un cours complet sur le développement web avec HTML et CSS sur la chaîne YouTube de freeCodeCamp.org. Ce cours est conçu avec soin et enseigné par Aakash N S, le PDG et cofondateur de Jovian, une plateforme pour la science des données et l'apprentissage automatique.

Le cours commence par les bases de HTML et CSS, en approfondissant les concepts avancés au fur et à mesure. Vous apprendrez à utiliser Git, GitHub et le déploiement cloud avec Vercel. Nous aborderons également la conception responsive mobile-first, le framework CSS Bootstrap et le framework d'application web Express.

Les devoirs de codage pratique sont une partie intégrante du cours. Vous serez chargé de construire une page web à partir d'une maquettes, de créer un site web responsive mobile-first et de concevoir une calculatrice scientifique. Pour le grand final, vous construirez votre site web de portfolio personnel et le déploierez dans le cloud !

Voici un aperçu du contenu du cours :

### Leçon 1 : Les bases de HTML et CSS

* Balises HTML, head, title, body & div
* Texte, liens, listes, titres & images
* Sélecteurs CSS, propriétés & valeurs

### Leçon 2 : HTML et CSS avancés

* Formulaires, inputs, boutons & listes déroulantes
* Tableaux, CSS flexbox & grilles de mise en page
* Iframes, vidéo, audio & canvas

### Devoir 1 : De la maquettes à la page web

* Étudier et inspecter une maquettes de design
* Ajouter du contenu HTML et le styliser avec CSS
* Déployer la page web finale dans le cloud

### Leçon 3 : Contrôle de version et déploiement cloud

* Introduction à Git & GitHub
* Branches, commits, push & pull
* Déploiement cloud sur Vercel

### Leçon 4 : Design responsive et CSS Flexbox

* Requêtes média & design adapté aux mobiles
* Création de mises en page fluides avec CSS flexbox
* Construction et test d'un site web responsive

### Devoir 2 : Design web responsive mobile-first

* Inspecter les maquettes spécifiques aux appareils
* Configurer les points d'arrêt et les styles de base
* Implémenter le design mobile-first

### Leçon 5 : Framework CSS Bootstrap

* Installation de Bootstrap et personnalisation des styles de base
* Création de mises en page responsives en utilisant le système de grille
* Construction d'une page web en utilisant les composants Bootstrap

### Devoir 3 : Construire une calculatrice scientifique

* Construire l'interface utilisateur avec HTML & CSS
* Ajouter des gestionnaires d'événements aux nœuds DOM
* Implémenter la logique avec des fonctions JavaScript

### Leçon 6 : Framework d'application web Express

* Configuration des routes et méthodes HTTP
* Rendu des templates & fichiers statiques
* Déploiement cloud sur Vercel

### Projet : Construire votre site web personnel

* Choisir un template ou créer des wireframes
* Créer un site web avec plusieurs pages
* Ajouter du contenu et déployer dans le cloud

Rendez-vous sur la chaîne YouTube de freeCodeCamp.org pour [regarder le cours complet](https://www.youtube.com/watch?v=dX8396ZmSPk) (20 heures de visionnage).
