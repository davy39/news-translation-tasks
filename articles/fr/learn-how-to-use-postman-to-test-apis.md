Toute personne qui crée des API devrait également les tester. L'une des méthodes les plus populaires pour tester les API est d'utiliser Postman. Postman compte plus de 10 millions d'utilisateurs dans le monde.

Nous venons de publier un cours accéléré sur Postman sur la chaîne YouTube freeCodeCamp.org qui vous apprendra tout ce que vous devez savoir pour commencer à tester facilement vos API.

Valentin Despa a créé ce cours. Valentin est un éducateur développeur qui enseigne les concepts logiciels depuis de nombreuses années.

Dans ce cours, vous apprendrez à construire des requêtes API avec Postman et à inspecter les réponses et créer des flux de travail. La deuxième partie du cours abordera la fonctionnalité la plus passionnante de Postman : l'écriture de tests API et l'automatisation de leur exécution.

Vous en apprendrez davantage sur le collection runner, les moniteurs Postman et Newman. Vous acquerrez une expérience pratique et aurez de nombreuses opportunités de vous exercer. Des devoirs vous aideront à maîtriser le contenu.

Voici les sujets abordés dans ce cours :

* Qu'est-ce que Postman
* Comment installer Postman
* Votre première requête avec Postman
* HTTP
* Collections et variables Postman
* Paramètres de requête
* Devoir
* Variables de chemin
* Requête POST / Authentification API
* Format JSON
* Devoir
* Données de test aléatoires
* Postman est-il le bon outil pour moi ?
* Visualisation des commandes existantes
* Devoir
* Requête PATCH
* Requête DELETE
