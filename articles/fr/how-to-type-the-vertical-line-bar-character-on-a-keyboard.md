Nous utilisons le caractère de barre verticale, |, dans divers scénarios et contextes. Vous l'utiliserez en mathématiques et en programmation, sous Linux, et dans les domaines de l'informatique pour représenter différentes choses. 

Les programmeurs appellent également ce caractère le symbole pipe.

## À quoi sert le caractère Pipe ?

En mathématiques, le double pipe (||) signifie qu'une ligne est parallèle à l'autre. Par exemple A || C.

Sous Linux, vous pouvez l'utiliser pour combiner des commandes que vous souhaitez exécuter ensemble. 

Dans de nombreux langages de programmation, nous utilisons le caractère pipe comme opérateur OU, et il signifie souvent qu'une des déclarations doit être vraie.

En JavaScript par exemple, nous utilisons le caractère pipe comme ceci :

```js
let score1 = 25
let score2 = 35

console.log(score1 === 25 || score2 === 26)
// true
```

Vous vous demandez peut-être comment taper la barre verticale (caractère pipe) sur votre clavier. C'est ce que je vais vous montrer dans cet article.

## Comment taper le caractère de barre verticale sur un clavier.

Le caractère pipe n'est pas caché sur un clavier. Il est situé juste au-dessus de la touche Entrée.
![pipe-location](https://www.freecodecamp.org/news/content/images/2022/06/pipe-location.png)

Mais si vous tapez cette touche, ce que vous obtenez est un antislash. Donc, pour le taper, vous devez maintenir `SHIFT`, puis appuyer sur la touche antislash :
![type-pipe-shift](https://www.freecodecamp.org/news/content/images/2022/06/type-pipe-shift.png)

Une autre façon de taper le caractère de barre verticale est d'activer le pavé numérique, de maintenir `ALT`, puis d'appuyer sur 1, 2 et 4. 
![type-pipe-alt](https://www.freecodecamp.org/news/content/images/2022/06/type-pipe-alt.png)

Maintenant, vous savez comment taper la barre verticale ou le caractère pipe sur votre clavier, et vous pouvez l'utiliser dans vos tâches de mathématiques ou de programmation.
