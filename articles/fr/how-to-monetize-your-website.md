La Web Monetization signifie que vous pouvez gagner de l'argent pour chaque personne qui visite votre site web. Ça a l'air bien, n'est-ce pas ? Mais comment cela fonctionne-t-il ? 

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra comment monétiser votre site web en utilisant des microtransactions et le Interledger Protocol. 

À la fin de ce cours intensif, vous saurez :

* ce qu'est la Web Monetization, 
* comment l'implémenter sur votre site web pour être payé pour votre travail acharné,
* et comment recevoir des paiements.  


Ania Kubow a développé ce cours. Elle est développeuse de logiciels et créatrice de cours sur YouTube, ainsi que votre guide dans le monde de la Web Monetization.

Bien que vous découvrirez de nombreuses stratégies de monétisation différentes, ce cours se concentre sur une en particulier. Vous apprendrez comment gagner de l'argent avec votre site web sans avoir à dépendre de publicités tierces ou à vendre les données de vos utilisateurs.

La méthode implique que les utilisateurs, qui apprécient le service que vous leur offrez sur votre site web, vous versent un très petit paiement pour avoir visité votre site.

Vous apprendrez comment développer un site web où les utilisateurs peuvent choisir soit une version financée par la publicité, soit une expérience premium sans publicité moyennant une petite somme.

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/XwzcrhhyDkc) (1 heure de visionnage).
