Apprenez à utiliser les APIs avec React dans ce tutoriel vidéo intermédiaire de Karl Hadwen. Vous apprendrez à créer une petite application en React qui appelle l'API Hacker News. L'application utilise React memo, Axios (pour faire des appels API), des composants stylisés et React Testing Library.

Vous apprendrez également à tester les problèmes d'accessibilité et de performance en utilisant Lighthouse.

Voici quelques-unes des choses que vous apprendrez dans ce cours :

* Comment utiliser create-react-app
* Comment ajouter Axios en utilisant Yarn
* Comment obtenir des données de l'API Hacker News
* Comment useEffect fonctionne
* Comment ajouter une validation lors de la récupération des histoires
* Comment utiliser les composants stylisés
* Comment fonctionne le profileur React
* Un rapide aperçu de ce que nous avons construit jusqu'à présent
* Comment utiliser InfiniteScroll
* Comment ajouter react-testing-library
* Comment écrire des tests et ajouter une couverture Jest
* Comment exécuter l'application à travers Lighthouse
* et plus encore

Vous pouvez regarder le cours ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=LN6Dol_fX0w) (2,5 heures de visionnage).
