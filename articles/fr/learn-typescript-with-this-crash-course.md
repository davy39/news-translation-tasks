TypeScript est un langage de programmation développé et maintenu par Microsoft. Il devient de plus en plus populaire.

Nous avons publié un cours accéléré sur la chaîne YouTube freeCodeCamp.org qui vous apprendra le langage de programmation TypeScript. À la fin du cours, vous comprendrez toutes les fonctionnalités modernes de TypeScript.

Ce cours a été développé par Alexander Kochergin. Alexander a beaucoup d'expérience dans le développement de cours pour sa chaîne YouTube.

TypeScript est similaire à JavaScript, mais ajoute des fonctionnalités supplémentaires. TypeScript est utilisé par Microsoft, Asana, Lyft, Slack et de nombreuses autres grandes entreprises technologiques. De plus, c'est le langage de programmation principal utilisé dans Angular 2+.

Voici les sections de ce cours :

* Dois-je apprendre TypeScript ?
* Comment installer les outils TypeScript ?
* Introduction aux définitions de types TypeScript
* Introduction aux interfaces dans TypeScript
* Union et alias de type dans TypeScript
* Any, never, void, unknown dans TypeScript
* Travailler avec le DOM en TypeScript
* Création de classes dans TypeScript
* Interfaces et fonctions génériques dans TypeScript
* Comment utiliser les énumérations dans TypeScript

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=gp5H0Vw39yw).
