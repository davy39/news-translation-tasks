Le langage de programmation Python est très performant lorsqu'il s'agit de la mise en réseau. Nous avons publié un [cours accéléré](https://www.youtube.com/watch?v=FGdiSJakIS4) sur la chaîne YouTube freeCodeCamp.org qui vous aidera à apprendre les bases de la mise en réseau en Python.

Ce cours a été développé par Neural Nine. Vous apprendrez les principes de la mise en réseau en Python en construisant les projets de mise en réseau suivants :

* Client de messagerie
* Script DDOS
* Scanner de ports
* Salon de discussion TCP

Ces projets vous donneront une bonne introduction à ce dont Python est capable en matière de mise en réseau. Beaucoup des bases seront enseignées à travers l'application Client de messagerie. L'application Salon de discussion vous aidera à apprendre comment interagir avec TCP. Le Script DDOS et le Scanner de ports vous enseigneront les bases des tests de pénétration.

Vous pouvez regarder le cours sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=FGdiSJakIS4) (1 heure de visionnage).
