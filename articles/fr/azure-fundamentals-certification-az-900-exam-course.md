La certification Microsoft Azure Fundamentals est destinée aux professionnels de la technologie qui souhaitent démontrer des connaissances fondamentales des concepts cloud en général et de Microsoft Azure en particulier. Cet examen est un point de départ courant dans un parcours vers une carrière dans Azure.

Nous venons de publier un cours complet de préparation à l'examen sur la chaîne YouTube freeCodeCamp.org pour l'examen AZ-900. Andrew Brown d'ExamPro a développé ce cours.

Microsoft Azure est un fournisseur de services cloud. Un fournisseur de services cloud (CSP) est une entreprise technologique qui propose plusieurs services cloud. Azure répertorie plus de 600 offres de services cloud, allant du calcul au stockage, en passant par les bases de données et l'apprentissage automatique.

Azure est la deuxième plateforme cloud la plus populaire derrière Amazon Web Services (AWS). Néanmoins, de nombreuses entreprises technologiques bien établies ou de grande taille adoptent les deux plateformes cloud. Cela peut être dû au fait qu'Azure est en mesure de rivaliser en termes de coût, d'offres cloud de niche, d'intégration facile et de la familiarité des développeurs avec les logiciels Microsoft.

### La certification Azure Foundations

La certification Azure Foundations est la certification cloud de niveau débutant pour Microsoft Azure. Elle est communément désignée par son code de cours, l'AZ-900.

L'examen couvre les domaines suivants :

* Concepts cloud
* Services principaux d'Azure
* Sécurité, confidentialité, conformité et confiance
* Tarification et support

Le temps d'étude moyen – même pour les personnes nouvelles dans le développement cloud – est d'une seule semaine d'étude.

Il y a également un certain transfert de connaissances si vous avez déjà obtenu la certification AWS Cloud Certified Practitioner (la certification de niveau débutant d'AWS). Mais ne vous attendez pas à réussir l'AZ-900 sans étudier.

Microsoft et Azure avaient auparavant des certifications spécifiques à certains services, mais ils sont passés à des certifications basées sur les rôles (certifications basées sur les titres de poste).

## Comment obtenir la certification ?

Vous obtenez la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft Azure.

Microsoft Azure est partenaire de Pearson Vue, un réseau de centres de test dans le monde entier. Pearson Vue vous permet de passer l'examen en personne ou en ligne. Pour passer l'examen en ligne, vous devez vous assurer d'avoir une pièce calme, sans encombrement, et une webcam fonctionnelle.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous pouvez vous inscrire et réserver votre examen.

## Puis-je simplement regarder le cours et réussir l'examen ?

Pour la certification de niveau débutant, oui, il est possible de réussir en regardant simplement des vidéos si vous avez déjà un bagage technique.

Pour des niveaux de certification plus difficiles, à partir du niveau Associate, il est nécessaire d'utiliser des examens pratiques payants proposés par des tiers.

L'examen comporte **40 à 60** questions avec un temps imparti de **1 heure**.

Une note de passage est d'environ **70 %**.
