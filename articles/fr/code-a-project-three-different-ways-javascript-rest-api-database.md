Lorsque vous codez un projet similaire en utilisant différentes méthodes, cela vous aide à mieux comprendre chaque méthode.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à coder un widget de vol en JavaScript en utilisant plusieurs méthodes.

Tout d'abord, vous apprendrez à créer le projet en utilisant une approche JavaScript pour débutants. Ensuite, vous apprendrez à mettre à jour le projet pour utiliser une API afin d'obtenir les données pour le widget. Enfin, vous créerez le projet en tant qu'application React avec un mini-backend utilisant Node.js pour communiquer avec une base de données hébergée dans le cloud.

Ania Kubow a développé ce cours. Ania est une créatrice populaire et développeuse de logiciels. Elle a créé de nombreux cours pour la chaîne freeCodeCamp ainsi que pour sa propre chaîne.

![Image](https://www.freecodecamp.org/news/content/images/2022/10/Oct-13-2022-10-04-33.gif)
_Vous allez construire ce widget de vol._

Les trois sections sont les suivantes :

* Widget de vol en JavaScript pur
* Widget de vol en JavaScript et Node.js utilisant une API REST
* Widget de vol en React et Node.js utilisant une base de données

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/_uSrE-gIdFc) (2 heures de visionnage).
