Par javinpaul

Les structures de données et les algorithmes sont parmi les sujets les plus essentiels pour les programmeurs, tant pour obtenir un emploi que pour bien performer dans un emploi. Une bonne connaissance des structures de données et des algorithmes est la base de l'écriture de bon code.

Si vous êtes familier avec les structures de données essentielles, par exemple [array](http://javarevisited.blogspot.sg/2015/06/top-20-array-interview-questions-and-answers.html), [string](http://javarevisited.blogspot.com/2015/01/top-20-string-coding-interview-question-programming-interview.html#axzz54hiGw48b), [linked list](http://javarevisited.blogspot.sg/2017/07/top-10-linked-list-coding-questions-and.html), tree, map, et les structures de données avancées comme les tries, et les arbres auto-équilibrés comme les arbres AVL, etc., vous saurez quand utiliser quelle structure de données et calculer le coût CPU et mémoire de votre code.

Même si vous n'avez pas besoin d'écrire votre propre array, linked list, ou hashtable, étant donné que chaque SDK de programmation majeur les fournit, par exemple JDK ou la bibliothèque C++ STL, vous devrez les comprendre afin de pouvoir les utiliser au bon endroit.

**Utiliser la bonne structure de données peut améliorer considérablement les performances d'un algorithme.**

Idéalement, nous devrions tous apprendre les [structures de données](https://hackernoon.com/10-data-structure-algorithms-and-programming-courses-to-crack-any-coding-interview-e1c50b30b927) et les [algorithmes](https://dzone.com/articles/10-coding-interview-preparation-courses-crack-a-pr) dans nos écoles et universités, mais c'est rarement couvert. La plupart des programmeurs, y compris moi-même, n'ont été introduits aux structures de données que dans nos cours d'informatique, mais nous n'avons pas vraiment appris leur importance dans le monde réel, et c'est pourquoi nous ne les avons pas mieux comprises.

Pour nous, ce ne sont que des algorithmes et des structures de données, par exemple des concepts, pas un outil que vous pouvez utiliser pour écrire de bons programmes. Nous ne savions pas que Facebook les utiliserait pour stocker nos détails ou que Google les utiliserait pour stocker des pages web et des liens vers des requêtes de recherche.

Quoi qu'il en soit, il n'est jamais trop tard.

Si vous pensez que vos connaissances en structures de données ne sont pas à la hauteur ou si vous souhaitez améliorer vos connaissances en structures de données, alors vous êtes au bon endroit.

Auparavant, j'ai partagé quelques [livres gratuits pour apprendre les structures de données et les algorithmes](http://javarevisited.blogspot.sg/2016/05/5-free-data-structure-and-algorithm-books-in-java.html#axzz4uXETWjmV), et dans cet article, vous apprendrez quelques-uns des meilleurs cours sur les structures de données et les algorithmes qui sont disponibles gratuitement en ligne.

### Cours gratuits pour apprendre les structures de données et les algorithmes

Voici ma liste de certains des meilleurs cours pour apprendre les structures de données et les algorithmes, qui sont également gratuits. De nombreux programmeurs pensent que les ressources gratuites ne sont pas bonnes, ce qui n'est pas vrai.

Même s'ils ne correspondent parfois pas à la qualité et à la couverture des ressources payantes, ils sont, en fait, les meilleures ressources pour commencer.

Vous pouvez utiliser ces cours pour vous familiariser avec les structures de données essentielles et apprendre quelques bases.

Certains d'entre eux sont particulièrement bons du point de vue d'un entretien, vous pouvez donc les utiliser parallèlement à tout livre que vous utilisez pour vous préparer à votre [entretien d'embauche en programmation](http://www.java67.com/2018/05/top-75-programming-interview-questions-answers.html).

## Cours #1 : Concepts de structures de données en C

Il s'agit d'un autre cours de formation en ligne gratuit sur les algorithmes et les structures de données, qui vise à enseigner les structures de données de base en programmation informatique.

Les structures de données enseignées dans le cours incluent [Stack](http://www.java67.com/2013/08/ata-structures-in-java-programming-array-linked-list-map-set-stack-queue.html), [Queue](http://javarevisited.blogspot.sg/2017/03/difference-between-stack-and-queue-data-structure-in-java.html#axzz4rZnFiEOx), et [Linked List](https://javarevisited.blogspot.com/2015/02/simple-junit-example-unit-tests-for-linked-list-java.html) en utilisant le langage de programmation C.

L'objectif principal de ce [cours](https://www.udemy.com/course/data-structures-in-c/) est de permettre aux étudiants et aux ingénieurs logiciels de visualiser le fonctionnement des différentes structures de données.

![Image](https://cdn-media-1.freecodecamp.org/images/9KAnoe0SwLrdwMWSxZvutFKzyxq959oVVyZ7)
_**[Concepts de structures de données en C](https://www.udemy.com/course/data-structures-in-c/)**_

Ce n'est pas un cours exhaustif, mais vous apprendrez à propos de [Stack](https://javarevisited.blogspot.com/2017/03/how-to-implement-stack-in-java-using-array-example.html), Queue, et [Linked List](https://javarevisited.blogspot.com/2017/03/how-to-reverse-linked-list-in-java-using-iteration-and-recursion.html#axzz5bAq1c3jf).

En bref, c'est un excellent cours pour les programmeurs nouveaux dans les structures de données et ceux qui viennent d'apprendre un langage de programmation et veulent apprendre les bases du fonctionnement des structures de données.

## Cours #2 : Algorithmes Partie 1 - Coursera

Il s'agit de la [première partie](https://www.coursera.org/learn/algorithms-part1) d'une série de cours en deux parties couvrant les algorithmes et les structures de données sur Coursera par Robert Sedgewick.

Dans cette partie, vous apprendrez les structures de données essentielles comme les [linked lists](http://javarevisited.blogspot.sg/2017/07/top-10-linked-list-coding-questions-and.html#axzz4xXS86IVo), [stacks](https://javarevisited.blogspot.com/2017/03/difference-between-stack-and-queue-data-structure-in-java.html), queues, [binary trees](https://javarevisited.blogspot.com/2015/10/how-to-implement-binary-search-tree-in-java-example.html), et [hash tables](http://javarevisited.blogspot.sg/2011/02/how-hashmap-works-in-java.html#axzz5B9NXhuQw), et les algorithmes de recherche et de tri comme [binary search](https://javarevisited.blogspot.com/2017/04/recursive-binary-search-algorithm-in-java-example.html), [quicksort](https://javarevisited.blogspot.com/2014/08/quicksort-sorting-algorithm-in-java-in-place-example.html), [mergesort](http://www.java67.com/2018/03/mergesort-in-java-algorithm-example-and.html), [insertion sort](https://javarevisited.blogspot.com/2014/12/insertion-sort-algorithm-in-java-to-array-example.html), etc.

![Image](https://cdn-media-1.freecodecamp.org/images/Uqip9-SbrXSJg53ERcVRpQIvCRwBl9MQsozM)
_[Algorithmes Partie 1](https://www.coursera.org/learn/algorithms-part1)_

Vous apprendrez également les structures de données et algorithmes de base utilisés dans les applications quotidiennes et comprendrez les compromis impliqués dans le choix de chaque structure de données ainsi que les algorithmes de [traversal](http://www.java67.com/2017/05/binary-tree-post-order-traversal-in-java-without-recursion.html), retrieval, et update.

Toutes les fonctionnalités de ce cours sont disponibles gratuitement, mais il ne propose pas de certificat à la fin.

### **Cours #3 : Algorithmes - Partie 2**

Il s'agit de la [deuxième partie](https://www.coursera.org/learn/algorithms-part2) d'une série de cours en ligne gratuits sur Coursera couvrant les structures de données et les algorithmes par Robert Sedgewick et Kevin Wayne, tous deux professeurs d'informatique.

Robert Sedgewick est également l'auteur du livre _[Algorithms (4th Edition)](https://www.amazon.com/Algorithms-4th-Robert-Sedgewick/dp/032157351X)_, l'un des livres les plus populaires sur les algorithmes pour les développeurs Java.

Dans [cette partie](https://www.coursera.org/learn/algorithms-part2), vous apprendrez les algorithmes de traitement de graphes et de chaînes de caractères.

Vous apprendrez également certaines structures de données et algorithmes avancés utilisés dans le développement d'applications.

En parlant de preuve sociale, [Algorithmes Partie 1](https://www.coursera.org/learn/algorithms-part1) et [Algorithmes Partie 2](https://www.coursera.org/learn/algorithms-part2) sont des cours très recommandés et ont des critiques et des notes impressionnantes.

## Cours #4. Structures de données en Java

Il s'agit d'un [cours en ligne d'introduction aux structures de données](https://www.udemy.com/course/data-structures-part-1-lognacademy/), aux algorithmes et aux analyses de complexité. Il vous enseignera la conception, la mise en œuvre et les analyses des structures de données de base en utilisant le langage Java.

La meilleure partie de [ce cours](https://www.udemy.com/course/data-structures-part-1-lognacademy/) est le nombre d'exemples pratiques et le fait qu'il se concentre sur l'intuition plutôt que sur les formules et les preuves mathématiques.

![Image](https://cdn-media-1.freecodecamp.org/images/BidoQMlvn0BpD8u95vR7IaGcorlUbo87lAMw)
_**[Structure de données en Java](https://www.udemy.com/course/data-structures-part-1-lognacademy/)**_

Le [cours](https://www.udemy.com/course/data-structures-part-1-lognacademy/) fournit une bonne introduction à l'« analyse de complexité », ce qui permet de repérer un programme performant (ou non), même sans avoir besoin de l'exécuter.

C'est tout sur certains des **cours gratuits sur les structures de données et les algorithmes** disponibles en ligne. Vous pouvez utiliser ces cours pour apprendre les structures de données et les algorithmes à votre propre rythme.

Ils ne sont peut-être pas tous complets, mais ils fournissent une bonne introduction. Une fois que vous aurez suivi ces cours, vous pourrez choisir un bon livre comme [Introduction to Algorithms](http://javarevisited.blogspot.com/2015/07/5-data-structure-and-algorithm-books-best-must-read.html#axzz54hiGw48b) pour compléter vos connaissances.
