Par Gitter

Appel à tous les passionnés de science des données, de machine learning et de deep learning ! Nous avons de nombreux projets de pointe à vous proposer, discutés dans les canaux [Gitter](https://gitter.im), allant de la cartographie de l'activité cérébrale à grande échelle, à la vision par ordinateur, aux réseaux de neurones, à la visualisation de données et au traitement du langage naturel. Consultez la liste ci-dessous et n'hésitez pas à rejoindre la conversation.

* [**Pandas**](https://gitter.im/pydata/pandas?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Bibliothèque flexible et puissante d'analyse et de manipulation de données pour Python, offrant des structures de données étiquetées similaires aux objets R data.frame, des fonctions statistiques, et bien plus encore.
* [**OpenAI Gym**](https://gitter.im/openai/gym) — OpenAI Gym est une boîte à outils pour développer et comparer des algorithmes d'apprentissage par renforcement. Elle permet d'enseigner aux agents tout, depuis la [marche](https://gym.openai.com/envs/Humanoid-v1) jusqu'à jouer à des jeux comme [Pong](https://gym.openai.com/envs/Pong-ram-v0) ou [Go](https://gym.openai.com/envs/Go19x19-v0).
* [**OpenWorm**](https://gitter.im/openworm/PyOpenWorm?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Bibliothèque unifiée et simple pour l'accès aux données et aux faits sur l'anatomie de C. elegans.
* [**Deeplearning4j**](https://gitter.im/deeplearning4j/deeplearning4j?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Projet open-source de deep learning distribué en Java et Scala.
* [**Numenta/public**](https://gitter.im/numenta/public?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Plateforme Numenta pour le calcul intelligent.
* [**Thunder**](https://gitter.im/thunder-project/thunder?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Analyse de données neuronales à grande échelle avec Spark.
* [**scikit-learn**](https://gitter.im/scikit-learn/scikit-learn?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Machine learning en Python.
* [**Redash**](https://gitter.im/getredash/redash?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Repenser la manière dont les données sont interrogées, partagées et visualisées. Une application web qui permet d'interroger facilement une base de données existante, de partager le jeu de données et de le visualiser de différentes manières.
* [**H2O**](https://gitter.im/h2oai/h2o-3?utm_source=blog&utm_medium=content&utm_campaign=frontend) — H2O met à l'échelle les statistiques, le machine learning et les mathématiques sur le Big Data.
* [**dp**](https://gitter.im/nicholas-leonard/dp?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Bibliothèque de deep learning conçue pour rationaliser la recherche et le développement en utilisant [Torch7](http://torch.ch/).
* [**Awesome deep vision**](https://gitter.im/kjw0612/awesome-deep-vision?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Une liste sélectionnée de ressources de deep learning pour la vision par ordinateur.
* [**Caffe**](https://gitter.im/BVLC/caffe?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Un framework rapide pour le deep learning.
* [**Distributed Machine Learning in C++**](https://gitter.im/dmlc/dmlc-core?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Une base de code commune pour le Machine Learning Distribué en C++.
* [**arrayfire-ml**](https://gitter.im/arrayfire/arrayfire-ml?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Bibliothèque de Machine Learning d'ArrayFire.
* [**datascience**](https://gitter.im/data-8/datascience?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Une bibliothèque Python pour l'initiation à la science des données.
* [**dev-ua/ai-ml-ds**](https://gitter.im/dev-ua/ai-ml-ds?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Canal d'Intelligence Artificielle et de Machine Learning de Dev Ukraine.
* [**pylearn2**](https://gitter.im/TNick/pylearn2?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Une bibliothèque de Machine Learning basée sur Theano.
* [**Numerical MOOC**](https://gitter.im/numerical-mooc/numerical-mooc?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Une initiative éducative collaborative en science et ingénierie computationnelles.
* [**LxOpenDataMeetup**](https://gitter.im/LxOpenDataMeetup/LxOpenDataMeetup?utm_source=blog&utm_medium=content&utm_campaign=frontend) — Échange de discussions et d'idées sur le machine learning, les données ouvertes et la science des données.

Avons-nous manqué une communauté particulière dans cette catégorie, ou souhaitez-vous que nous mettions en avant votre canal ? Laissez-nous un message dans [Gitter HQ](https://gitter.im/gitterHQ/gitter) et nous l'ajouterons à la liste.

Vous pouvez également [créer facilement votre propre canal ici.](https://gitter.im/home#createroom)
