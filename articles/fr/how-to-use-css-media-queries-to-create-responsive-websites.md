Les Media Queries CSS vous permettent de créer des sites web responsives qui s'adaptent à toutes les tailles d'écran, du bureau au mobile.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp qui vous apprendra à créer des Media Queries CSS. Après avoir appris les bases, vous apprendrez à utiliser les media queries pour créer trois projets pratiques.

Joy Shaheb a créé ce cours. En plus de créer de nombreux tutoriels vidéo utiles, Joy a également écrit de nombreux articles techniques de qualité sur freeCodeCamp News.

![Image](https://www.freecodecamp.org/news/content/images/2021/05/image-56.png)
_La syntaxe d'une media query._

Voici les sections couvertes dans ce cours :

* Installation
* Projet 1
* Règle @media
* Opérateur And
* Type de média
* Max-Width
* Min-Width
* Projet 2 Bureau
* Projet 2 Mobile
* Projet 3 Bureau
* Projet 3 Mobile
* Conclusion
