Java est l'un des langages de programmation les plus populaires depuis longtemps. Il est considéré comme un langage polyvalent, concurrent, fortement typé et orienté objet basé sur les classes.

Nous venons de publier un cours complet sur la chaîne YouTube de freeCodeCamp.org qui vous enseignera le langage de programmation Java. Ce cours est parfait pour les débutants.

Nous avons déjà publié des cours sur Java, mais celui-ci est l'un des plus faciles à suivre pour un débutant. Vous pouvez suivre l'intégralité du cours directement depuis votre navigateur web. De plus, le cours couvre une version plus récente de Java que nos cours précédents.

Farhan Hasin Chowdhury a créé ce cours. Farhan est un ingénieur logiciel expérimenté ici chez freeCodeCamp et il excelle à décomposer les sujets de programmation pour les débutants.

![Image](https://www.freecodecamp.org/news/content/images/2022/09/image-429.png)
_Après ce cours, vous comprendrez peut-être le code Java dans cette bande dessinée XKCD. Ou vous pouvez simplement [lire une explication](https://explainxkcd.com/wiki/index.php/1188:_Bonding)._

Voici les sections de ce cours :

* Hello World
* Variables
* Types de données
* Opérateurs
* Chaînes de caractères
* Entrées utilisateur
* Instructions conditionnelles
* Cas Switch
* Tableaux
* Boucles For
* Boucles While
* Boucles Do...While
* ArrayLists
* HashMaps
* Programmation Orientée Objet
