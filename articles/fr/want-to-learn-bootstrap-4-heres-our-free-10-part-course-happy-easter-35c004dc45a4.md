Par Per Harald Borgen

![Image](https://cdn-media-1.freecodecamp.org/images/1*zCx3Kx42HlhcBm5oYMD2rA.png)
_[Cliquez ici pour accéder au cours.](https://scrimba.com/g/gbootstrap4?utm_source=freecodecamp.org&amp;utm_medium=referral&amp;utm_campaign=gbootstrap4_launch_article)_

Bootstrap 4 représente une réécriture complète de toute la bibliothèque, et il apporte avec lui plusieurs nouvelles fonctionnalités et concepts à assimiler. Il peut donc sembler écrasant de plonger directement dans la documentation pour l'apprendre.

C'est pourquoi nous nous sommes associés à [Neil Rowe](https://twitter.com/neil_rowe_) (précédemment connu sous le nom de [CodersGuide](https://www.youtube.com/user/CodersGuide)) et avons créé un [cours complet gratuit sur Bootstrap 4](https://scrimba.com/g/gbootstrap4?utm_source=freecodecamp.org&utm_medium=referral&utm_campaign=gbootstrap4_launch_article). Neil a précédemment créé l'un des cours les plus populaires sur Bootstrap 3 sur YouTube, nous sommes donc très heureux qu'il ait choisi d'utiliser Scrimba pour son cours de suivi.

Vous pouvez suivre le cours dès aujourd'hui. Rendez-vous simplement [sur la page du cours](https://scrimba.com/g/gbootstrap4?utm_source=freecodecamp.org&utm_medium=referral&utm_campaign=gbootstrap4_launch_article) et vous deviendrez un expert de Bootstrap 4 en une heure.

Examinons maintenant de plus près ce que vous allez apprendre !

#### #1 Introduction au cours

![Image](https://cdn-media-1.freecodecamp.org/images/1*zCx3Kx42HlhcBm5oYMD2rA.png)

Dans le premier screencast, vous obtiendrez un aperçu de Bootstrap lui-même, ainsi que des utilitaires et composants qu'il apporte. Neil passe également en revue les principaux avantages de la version 4.

#### #2 Le système de grille responsive

![Image](https://cdn-media-1.freecodecamp.org/images/1*oOKLdoC4_mNFPV7HUfV-QA.png)

Le cours aborde ensuite l'un des composants les plus importants de Bootstrap, le système de grille responsive. Neil explique comment créer diverses grilles, avec différents points de rupture, tailles de colonnes et décalages.

#### #3 Barres de navigation responsives

![Image](https://cdn-media-1.freecodecamp.org/images/1*h_AkQ1f7wBUorZ-lA2WXPA.png)

Le composant de barre de navigation est une raison majeure pour laquelle beaucoup de gens utilisent Bootstrap. Dans cette leçon, vous apprendrez à construire une barre de navigation responsive qui se réduit en un menu hamburger sur les petits écrans.

#### #4 Modales

![Image](https://cdn-media-1.freecodecamp.org/images/1*be0Vaj3kUcchHvRb3uE1ZA.png)

Bootstrap facilite également la création de modales, qui sont souvent utiles. Dans cette leçon, vous apprendrez à les construire étape par étape.

#### #5 Formulaires

![Image](https://cdn-media-1.freecodecamp.org/images/1*2UPJmuKyHcySzxVcrzk3OA.png)

Les formulaires sont essentiels pour la plupart des sites web, il est donc bon de savoir comment les créer. Dans cette leçon, vous apprendrez à créer un formulaire d'inscription responsive. Ici, vos compétences acquises dans la leçon sur les grilles responsives seront également utiles.

#### #6 Groupes de listes

![Image](https://cdn-media-1.freecodecamp.org/images/1*N-WC9LzY_meovlsapChJUQ.png)

Les groupes de listes peuvent être utilisés pour de nombreuses fins différentes. Ils peuvent, par exemple, être utilisés comme une barre latérale, un menu, une liste d'articles, ou tout ce que vous voulez, essentiellement.

#### #7 Cartes

![Image](https://cdn-media-1.freecodecamp.org/images/1*UYPC0IHtsW6d6WMYYQ9_OQ.png)

Bootstrap 4 introduit un tout nouveau composant appelé cartes, qui remplacent les [panels](https://getbootstrap.com/docs/3.3/components/#panels), [wells](https://getbootstrap.com/docs/3.3/components/#wells) et [thumbnails](https://getbootstrap.com/docs/3.3/components/#thumbnails). Une carte est un conteneur de contenu flexible et extensible. Dans cette leçon, vous apprendrez à en construire une en utilisant divers composants qui constituent généralement une carte.

#### #8 Tableaux

![Image](https://cdn-media-1.freecodecamp.org/images/1*FEODV-0VvpaGC89Q8C-3aw.png)

Bootstrap 4 propose également un ensemble de classes pour modifier les tableaux, et cette leçon en démontrera la plupart. Elles sont optionnelles, ce qui signifie que vous devez activement ajouter une classe de tableau pour transformer un tableau en un tableau Bootstrap. Cela diffère de la manière dont cela fonctionnait auparavant, où les tableaux HTML réguliers étaient automatiquement transformés en tableaux Bootstrap.

#### #9 Alertes

![Image](https://cdn-media-1.freecodecamp.org/images/1*uVdJv8jn0yGSpgpHwLw0JQ.png)

Les alertes ne sont pas particulièrement excitantes, et il n'y a pas trop de changements par rapport aux versions précédentes de la bibliothèque, donc si vous avez utilisé Bootstrap 2 ou 3, vous devriez être assez familier avec tout cela. Cependant, il est important de les connaître, car donner des messages et des notifications à vos utilisateurs via des alertes peut aider à rendre votre site web plus convivial.

#### #10 Options de navigation

![Image](https://cdn-media-1.freecodecamp.org/images/1*9gHQ3xg00UBaO5xbaBHM9g.png)

Enfin, le cours se termine là où il a commencé, en vous donnant plus d'outils pour les barres de navigation, comme les onglets et les menus déroulants. C'est utile à savoir, mais pas aussi fondamental que le contenu de la deuxième leçon, c'est pourquoi cela vient à la fin du cours.

En suivant toutes ces leçons à la suite, vous devriez être en mesure de commencer à construire des sites web avec Bootstrap 4. Et si vous êtes bloqué, vous serez bien équipé pour plonger dans [la documentation](https://getbootstrap.com/) pour résoudre les problèmes que vous pourriez rencontrer.

### Le format Scrimba

Enfin, examinons également rapidement la technologie derrière le cours. Il est construit en utilisant Scrimba, un outil de screencast de codage interactif. Un "scrim" ressemble à une vidéo normale, cependant, il est entièrement interactif. Cela signifie que vous pouvez modifier le code à l'intérieur du screencast.

Voici un gif qui explique le concept :

![Pausez le screencast → Modifiez le code → Exécutez-le ! → Voyez vos modifications](https://cdn-media-1.freecodecamp.org/images/1*4PWxbgV--7ZHlB-YVqavJg.gif)

Pausez le screencast → Modifiez le code → Exécutez-le ! → Voyez vos modifications

C'est idéal lorsque vous sentez que vous devez expérimenter avec le code pour bien le comprendre, ou lorsque vous souhaitez simplement copier un morceau de code.
