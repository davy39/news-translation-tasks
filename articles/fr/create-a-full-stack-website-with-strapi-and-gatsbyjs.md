Strapi est le CMS Headless open-source le plus populaire et est basé sur Node.js. Gatsby permet aux développeurs de construire rapidement des sites web et des applications statiques très rapides. Vous pouvez les combiner pour créer un site web full-stack en utilisant principalement JavaScript.

Nous avons publié un cours complet pour vous aider à apprendre à utiliser Gatsby et Strapi pour coder un projet de portfolio. Le cours a été développé par John Smilga, qui est un enseignant expérimenté et créateur de cours.

Le projet inclut une page de portfolio avec du texte mis à jour dynamiquement, récupéré depuis Strapi via une requête. Vous apprendrez à implémenter des bascules, une section héro, des liens, des articles, plusieurs pages et plus encore. Le projet est entièrement responsive.

Une grande partie du contenu sera créée de manière programmatique à partir de fichiers markdown en utilisant GatsbyJS. Le projet intègre également Formspree pour collecter les données de formulaire.

En plus d'apprendre les bases de la construction d'un site full-stack avec Gatsby et Strapi, vous apprendrez également les sujets suivants :

* Google Fonts
* Déploiement avec Digital Ocean
* Optimisation pour les moteurs de recherche
* Gatsby Browser
* Sitemaps

Regardez le [cours complet sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=Oc_ITwxiG-Y&feature=youtu.be) (5 heures de visionnage).
