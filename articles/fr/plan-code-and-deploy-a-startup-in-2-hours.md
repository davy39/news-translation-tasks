Il existe de nombreux tutoriels sur la création de projets. Mais il peut être difficile de comprendre comment tout s'assemble, de l'idée initiale au déploiement du projet.

Dans ce tutoriel vidéo de [Aaron Jack](https://www.freecodecamp.org/news/plan-code-and-deploy-a-startup-in-2-hours/_aaronjack) de [Code Drip](https://www.youtube.com/channel/UCRLEADhMcb8WUdnQ5_Alk7g), vous apprendrez à planifier, coder et déployer une application full stack JavaScript en seulement deux heures.

Cette vidéo couvre :

* Création d'un front end React avec Hooks
* Codage d'une API Node-Express simple
* Démarrage d'un worker cron pour récupérer des données
* Développement d'un algorithme de filtrage simple
* Utilisation de Redis et node-redis
* Déploiement avec NGINX sur un droplet DigitalOcean

Vous apprendrez à construire une application de tableau d'offres d'emploi qui pourrait être une vraie startup utilisée par des gens. L'application utilisera des API publiques pour récupérer des données d'offres d'emploi à partir de diverses sources. Vous apprendrez tout, de la construction de l'application à l'achat d'un nom de domaine et au déploiement en utilisant Digital Ocean.

Vous pouvez regarder la vidéo complète ci-dessous ou sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=lauywdXKEXI) (2 heures de visionnage).
