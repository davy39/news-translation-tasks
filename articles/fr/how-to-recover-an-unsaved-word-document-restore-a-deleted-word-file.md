Parfois, vous pourriez fermer accidentellement votre application Word sans avoir enregistré votre fichier. Vous pourriez penser que votre travail est perdu, mais il existe en réalité un moyen de récupérer le fichier sans perdre le moindre contenu.

Et c'est exactement ce que cet article va vous montrer comment faire.

Pour suivre ce tutoriel, il sera utile d'avoir un document Word avec lequel travailler. Voici celui que j'utiliserai :

![Image](https://www.freecodecamp.org/news/content/images/2022/01/word.png)

J'ai fermé l'application sans enregistrer le document. Commençons maintenant le processus de restauration facile.

### **Étape 1 – Créer un document vierge dans Word**

* Relancez l'application.
* Sur la page de démarrage, cliquez sur "Document vierge".

![Image](https://www.freecodecamp.org/news/content/images/2022/01/blank.png)

### **Étape 2 – Utiliser l'option Ouvrir**

* Appuyez sur **Ctrl+O**. 
* Alternativement, vous pouvez cliquer sur l'option **Fichier** dans l'en-tête de votre document vierge, puis cliquer sur **Ouvrir** dans la barre latérale.

### **Étape 3 – Cliquer sur "Récupérer les documents non enregistrés"**

* Cliquez sur **Récupérer les documents non enregistrés**, qui se trouve généralement en bas de la page.

![Image](https://www.freecodecamp.org/news/content/images/2022/01/recover.png)

Si vous ne voyez pas le bouton **Récupérer les documents non enregistrés**, suivez ces étapes pour afficher les options :

1. Dans votre document vierge, appuyez sur **Ctrl+O** ou cliquez sur **Fichier**, qui se trouve dans la section d'en-tête.
2. Cliquez sur **Informations**, que vous trouverez dans la barre latérale.
3. Cliquez sur **Gérer le document**.
4. Cliquez sur **Récupérer le document non enregistré**.

![Image](https://www.freecodecamp.org/news/content/images/2022/01/recover-unsaved-document.png)

### **Étape 4 – Choisir les documents non enregistrés**

Après avoir cliqué sur le bouton **Récupérer les documents non enregistrés**, une nouvelle fenêtre devrait s'ouvrir avec une liste de documents non enregistrés. Parcourez la liste, ouvrez votre document et tout votre travail devrait être intact.

![Image](https://www.freecodecamp.org/news/content/images/2022/01/pop-up.png)

## Conclusion

Dans cet article, nous avons passé en revue les étapes nécessaires pour récupérer un document Word non enregistré. Ces étapes peuvent être utilisées à chaque fois que vous fermez Word sans enregistrer un document. 

N'oubliez pas d'enregistrer cette fois-ci !
