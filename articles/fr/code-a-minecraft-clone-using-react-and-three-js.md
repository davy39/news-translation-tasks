Il n'est peut-être pas possible de coder un [clone de Minecraft en 30 secondes](https://www.youtube.com/watch?v=MAlSjtxy5ak), mais vous pouvez certainement en créer un en moins de deux heures.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un clone de Minecraft entièrement fonctionnel en utilisant React, JavaScript et Three.js.

Daniel Bark a créé ce cours. Daniel est un ingénieur logiciel senior originaire de Suède et est un expert en Three.js.

Three.js est une bibliothèque JavaScript 3D qui permet aux développeurs de créer et d'afficher des graphiques 3D animés dans les navigateurs web.

Dans ce cours, vous allez créer un jeu qui prend en charge cinq types de blocs Minecraft. Il inclura le déplacement, le saut, le changement et le placement de blocs, la sauvegarde, et plus encore.

Vous apprendrez de nombreux concepts React tels que useState, useEffect, useRef et les Hooks personnalisés pour la gestion d'état. Et vous serez en mesure d'appliquer ces principes pour créer d'autres jeux 3D en JavaScript.
