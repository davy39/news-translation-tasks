Apprenez et maîtrisez les structures de données les plus courantes dans ce cours vidéo gratuit de 8 heures dispensé par l'ingénieur Google William Fiset.

Ce cours enseigne les structures de données à l'aide d'animations pour les représenter visuellement. Et il est conçu pour les débutants.

Vous apprendrez à coder diverses structures de données et à les combiner grâce aux instructions pas à pas de Finet. 

Chaque structure de données est accompagnée d'un code source fonctionnel pour vous aider à consolider votre compréhension. Les extraits de code sont en Java, mais si vous connaissez JavaScript ou Python, vous pourrez probablement les comprendre sur le plan conceptuel.

Vous apprendrez les structures de données suivantes :

* Tableaux statiques et dynamiques
* Listes simplement et doublement chaînées
* Piles
* Files d'attente
* Tas/Files de priorité
* Arbres binaires/Arbres de recherche binaires
* Union find/Ensemble disjoint
* Tables de hachage
* Arbres de Fenwick
* Arbres AVL

Vous pouvez regarder [la vidéo complète sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=RBSGKlAvoiM) (8 heures de visionnage). Comme toujours, c'est gratuit et il n'y a pas de publicités pour vous interrompre.
