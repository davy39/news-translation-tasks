Pensez à la tâche la plus ennuyeuse que vous ayez jamais faite. Il y a des chances que vous puissiez l'automatiser avec Python !

Nous venons de publier un cours d'automatisation Python sur la chaîne YouTube freeCodeCamp.org.

Frank Andrade a créé ce cours. Frank est un scientifique des données et un enseignant expérimenté.

Dans ce cours, vous apprendrez à automatiser des tâches ennuyeuses et répétitives telles que la création d'un rapport Excel, l'envoi de messages texte, l'extraction de tableaux depuis des sites web, l'interaction avec des sites web, et bien plus encore. Vous apprendrez à utiliser plusieurs bibliothèques Python différentes pour vous aider dans l'automatisation, telles que Path, Selenium, XPath, et bien d'autres.

Voici les projets et sections couverts dans ce cours :

### Extraction de tableaux

* Extraire des tableaux depuis des sites web
* Extraire des fichiers CSV depuis des sites web
* Extraire des tableaux depuis des PDF

### Automatisation web et scraping web

* Bases du HTML - Balises et éléments
* Bases du HTML - Structure arborescente
* XPath - Syntaxe, fonctions et opérateurs
* XPath - Testez votre XPath
* XPath - Caractères spéciaux et syntaxe

### Automatiser les actualités

* Installation de Selenium et ChromeDriver
* Création du pilote
* Trouver des éléments
* Exporter des données vers un fichier CSV
* Mode sans tête (headless)
* Préparer le script pour qu'il s'exécute tous les jours
* Convertir py en exe
* Planifier un script Python avec crontab (macOS)

### Automatiser un rapport Excel

* Créer un tableau croisé dynamique avec Python
* Ajouter un graphique en barres
* Écrire des formules Excel avec Python
* Formater des cellules
* Convertir un tableau croisé dynamique en rapport Excel
* Générer des rapports Excel en un clic (py en exe)

### Automatiser WhatsApp

* Créer un script d'automatisation pour WhatsApp
