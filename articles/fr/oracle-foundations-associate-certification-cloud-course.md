Cet examen de certification Oracle Cloud demandera – en moyenne – environ une semaine d'étude pour s'y préparer. La plupart des personnes qui s'engagent sérieusement dans leurs études sont prêtes à réussir l'examen en environ quatre jours.

Mon avis est que vous devriez tenter l'examen même si vous ne vous sentez pas prêt, afin d'acquérir de l'expérience grâce à l'épreuve de l'examen.

## Qu'est-ce qu'Oracle Cloud Infrastructure ?

Oracle est une entreprise technologique qui existe depuis la fin des années 70 et qui est bien connue pour la création de logiciels d'entreprise sur site (on-premises) et plus particulièrement pour sa base de données Oracle.

Oracle possède son propre fournisseur de cloud, tout comme AWS, et ils appellent leur plateforme Oracle Cloud Infrastructure (OCI) – également connue sous le nom d'Oracle Cloud.

Oracle ne propose pas autant de services cloud et d'intégrations cloud qu'AWS. Mais considérez qu'Oracle était la deuxième plus grande entreprise de logiciels en termes de revenus et de parts de marché en 2019.

Oracle a creusé sa niche sur le marché du cloud en faisant ce qu'ils savent faire de mieux : fournir des fonctionnalités cloud riches autour de leur base de données Oracle.

## La certification Oracle Foundations Associate

L'Oracle Foundations Associate est la certification de niveau débutant pour Oracle Cloud Infrastructure.

![Image](https://www.freecodecamp.org/news/content/images/2020/05/certimage.png)
_Un graphique des 6 certifications cloud d'Oracle. La certification couverte dans ce cours sert de base à toutes les autres certifications._

Cette certification est très similaire à l' [AWS Certified Cloud Practitioner](https://www.freecodecamp.org/news/aws-certified-cloud-practitioner-training-2019-free-video-course/). Elle se concentre sur votre familiarisation avec :

* Les concepts du Cloud
* L'infrastructure mondiale d'Oracle
* Les services de base d'Oracle Cloud (Calcul, Stockage, Bases de données, Réseautage)
* La facturation et la tarification d'Oracle Cloud
* Les services de sécurité d'Oracle Cloud

Si vous débutez dans le Cloud, vous pouvez réussir cet examen avec moins d'une semaine d'étude.

Si vous détenez déjà l'AWS Certified Cloud Practitioner, vous pourriez réussir cet examen avec quelques heures d'étude. Il vous suffirait de vous familiariser avec les terminologies d'OCI et certains concepts concernant la gestion de leur compte et la facturation.

Dans ce cours, je vous montrerai quelques exemples de la manière dont AWS et OCI proposent des offres très similaires en termes de technologies, mais avec des noms différents. Pour n'en citer que quelques-uns :

* AWS Virtual Private Cloud (VPC) est l'équivalent d'Oracle Virtual Cloud Network (VCN)
* AWS Availability Zones (AZs) est l'équivalent d'Oracle Availability Domains (ADs)
* AWS Key Management System est l'équivalent d'Oracle Vault
* AWS S3 est l'équivalent d'Oracle Object Storage
* AWS Elastic Block Store (EBS) est l'équivalent d'Oracle Block Storage
* AWS EC2 est l'équivalent d'Oracle VM

Le score de passage pour cet examen est de 68 %. Il y a 60 questions – à la fois à choix multiples et à réponses multiples. Cela signifie que vous pouvez vous permettre de rater environ 19 questions et tout de même réussir. Ce qui constitue une marge d'erreur considérable.

Les frais d'examen coûtent généralement 95 $ USD, et cette certification Oracle est valide pendant 1,5 an avant que vous ne deviez repasser l'examen.

## Le cours gratuit Oracle Foundations

Mon cours restera gratuit pour toujours. Et il est également sans publicité.

Ce cours gratuit dure près de 3 heures. Il comprend des exercices pratiques. Vous pouvez suivre et configurer votre propre compte Oracle Cloud. Ensemble, nous explorerons les services de Réseautage, de Calcul, de Stockage et de Base de données.

Un compte sandbox signifie que vous ne serez jamais facturé à moins que vous ne choisissiez de quitter le sandbox. Vous pouvez donc explorer en toute sécurité sans être facturé.
