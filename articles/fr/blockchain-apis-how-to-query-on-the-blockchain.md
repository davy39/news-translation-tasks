Si vous souhaitez construire des applications sur la blockchain, vous aurez probablement besoin d'une manière d'interroger des données.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à interroger des données sur la blockchain afin de construire vos propres applications décentralisées.

Ania Kubów a développé ce cours. Elle est une instructrice populaire qui a créé des cours incroyables sur une variété de sujets de programmation.

Ce cours est entièrement dédié à l'interrogation de données dans les applications décentralisées Web3. Vous obtiendrez une introduction à l'indexation et à l'interrogation de données sur la blockchain Ethereum. L'interrogation de données sur des réseaux décentralisés est très différente de ce à quoi vous pourriez être habitué sur le web traditionnel.

À la fin de ce cours, vous aurez les connaissances nécessaires pour interroger des données à partir du mainnet Ethereum ainsi que d'une variété d'autres chaînes afin de construire vos propres applications décentralisées.

Vous apprendrez à utiliser The Graph. C'est un protocole d'indexation pour interroger des réseaux comme Ethereum et IPFS. N'importe qui peut construire et publier des API ouvertes, appelées sous-graphes, rendant les données plus facilement accessibles.

Ce cours a été rendu possible grâce à une subvention de The Graph.

Voici les sections couvertes dans ce cours :

* Ce que nous allons couvrir
* À quoi sert The Graph ?
* Sous-graphes
* Manifestes de sous-graphes
* Pourquoi construire un sous-graphe ?
* Construisons un sous-graphe !
* Interrogation depuis un frontend
* Gestion des clés API
