JavaScript est le langage de script le plus largement utilisé sur Terre. Et il possède le plus grand écosystème de bibliothèques de tous les langages de programmation.

JavaScript est le langage central du web, et le seul langage de programmation capable de s'exécuter dans tous les principaux navigateurs web.

Notamment, JavaScript n'a aucun lien avec Java. [JavaScript: Le langage de programmation le plus incompris au monde](http://www.crockford.com/javascript/javascript.html)

Le nom officiel de JavaScript est ECMAScript défini sous la norme [ECMA-262](https://www.ecma-international.org/publications/standards/Ecma-262.htm)

Si vous souhaitez en savoir plus sur le langage JavaScript et pourquoi il est si largement utilisé, lisez l'article de Quincy Larson - [Quel langage de programmation devrais-je apprendre en premier ?](https://medium.freecodecamp.org/what-programming-language-should-i-learn-first-%CA%87d%C4%B1%C9%B9%C9%94s%C9%90%CA%8C%C9%90%C9%BE-%C9%B9%C7%9D%CA%8Dsu%C9%90-19a33b0a467d) - ou regardez cette [vidéo inspirante de Preethi Kasireddy](https://www.youtube.com/watch?v=VqiEhZYmvKk).

## **Moteurs JavaScript autonomes**

SpiderMonkey de Mozilla, le premier moteur JavaScript jamais écrit, actuellement utilisé dans Mozilla Firefox. V8, le moteur JavaScript de Google, utilisé dans Google Chrome.

Google Apps Script, un interpréteur cloud/serveur qui fournit un contrôle "macro-like" programmatique des services et documents Google Apps.

Node.js, construit sur V8, une plateforme qui permet d'écrire des applications côté serveur en JavaScript.

Windows inclut JScript, une variante de JavaScript dans Windows Script Host.

Chakra, un fork de JScript, est développé par Microsoft et utilisé dans leur navigateur Edge.

Mozilla propose également Rhino, une implémentation de JavaScript construite en Java, généralement intégrée dans les applications Java pour fournir des scripts aux utilisateurs finaux.

WebKit (à l'exception du projet Chromium) implémente le moteur JavaScriptCore.

## **Frameworks JavaScript**

Les frameworks JavaScript les plus fréquemment utilisés sont React JS, Angular JS, Create JS, jQuery, NodeJS, etc. Pour plus de détails, suivez ce [lien](https://javascriptreport.com/the-ultimate-guide-to-javascript-frameworks/).
