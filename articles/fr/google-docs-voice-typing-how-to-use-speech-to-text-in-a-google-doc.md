La reconnaissance vocale est un outil d'accessibilité vraiment utile si la frappe est difficile pour vous. Ou vous pouvez également l'utiliser si vous aimez réfléchir à voix haute et transcrire ce que vous dites.

Dans ce tutoriel, je vais vous montrer comment utiliser la reconnaissance vocale dans Google Docs. Notez cependant que cette fonctionnalité est disponible uniquement si vous utilisez le navigateur Chrome de Google.

## Comment activer l'outil de reconnaissance vocale

Vous pouvez activer la reconnaissance vocale à partir du menu Outils – c'est le quatrième élément du menu en partant du bas. Ou vous pouvez utiliser le raccourci `Ctrl+Shift+S` (sur Windows) ou `Command+Shift+S` (sur Mac) pour l'activer.

![Image](https://www.freecodecamp.org/news/content/images/2021/07/image-101.png)
*Le menu Outils. L'élément de menu Saisie vocale est le dixième élément en partant du haut, ou le quatrième en partant du bas.*

Cela ouvrira un petit onglet avec un microphone et un menu déroulant.

![Image](https://www.freecodecamp.org/news/content/images/2021/07/image-102.png)
*L'onglet de saisie vocale.*

Vous pouvez le déplacer avec les trois points horizontaux en haut. À côté de ces points, vous pouvez cliquer sur la croix pour le fermer. Juste en dessous, vous verrez le menu déroulant où vous pouvez changer de langue (et il indique quelle langue est actuellement définie). Ensuite, en dessous, il y a le grand bouton carré avec un microphone gris au milieu.

Si vous souhaitez changer le paramètre de langue pour l'outil de saisie vocale, vous devez utiliser le menu déroulant. Il vous permettra de choisir entre de nombreuses langues. Il vous suffit de sélectionner la langue que vous préférez, et vous êtes prêt à commencer à l'utiliser.

![Image](https://www.freecodecamp.org/news/content/images/2021/07/image-105.png)
*Le menu déroulant développé. Il propose de nombreuses langues parmi lesquelles vous pouvez choisir.*

## Comment utiliser la saisie vocale dans Google Docs

Vous pouvez commencer la saisie vocale en cliquant sur le bouton du microphone. Lorsqu'il est activé, il devient rouge.

Si vous recevez une alerte de Google Docs vous demandant l'autorisation d'utiliser votre microphone, accordez l'autorisation ou vous ne pourrez pas utiliser la saisie vocale.

![Image](https://www.freecodecamp.org/news/content/images/2021/07/image-104.png)
*L'icône du microphone lorsqu'elle est activée : toutes les autres options disparaissent, et l'icône du microphone devient rouge et est désormais entourée d'un cercle rouge.*

Maintenant, commencez à parler, et Google Docs transcrira ce que vous dites. Vous devez parler clairement et pas trop rapidement pour de meilleurs résultats.

Pendant que Google interprète ce que vous avez dit, un motif de petits carrés sera affiché là où les mots apparaîtront après qu'il ait fini de les élaborer.

![Image](https://www.freecodecamp.org/news/content/images/2021/07/image-106.png)
*Pendant que Google élabore, un motif de carrés similaire à celui-ci sera affiché. Lorsque l'élaboration est terminée, il sera remplacé par les mots.*

## Commandes vocales

Écrire un document inclut souvent l'édition et la mise en forme. Les commandes vocales vous permettent de faire toutes les éditions et mises en forme en utilisant uniquement votre voix.

Les commandes vocales sont disponibles uniquement pour la langue anglaise. Lorsque vous dites certains mots et phrases, comme "aller à la fin de la ligne" ou "sélectionner le paragraphe", ces actions seront exécutées.

Vous pouvez trouver la liste complète des commandes disponibles en disant "liste des commandes vocales", ou dans [l'article du guide Google Docs avec la liste complète des commandes vocales disponibles](https://support.google.com/docs/answer/4492226).
