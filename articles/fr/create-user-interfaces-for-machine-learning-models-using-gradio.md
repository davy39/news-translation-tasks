Ajuster, visualiser et partager des modèles de machine learning peut être beaucoup plus facile avec une interface utilisateur élégante.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Gradio pour créer des interfaces utilisateur intuitives pour des modèles de machine learning.

Abdul de 1littlecoder a développé ce cours. Il a aidé de nombreuses personnes à apprendre le machine learning avec des centaines de vidéos sur sa chaîne YouTube.

Gradio est une bibliothèque Python qui permet de démontrer votre modèle de machine learning avec une interface web conviviale afin que n'importe qui puisse l'utiliser, n'importe où.

Voici les sections abordées dans ce cours :

* Quel est le problème avec le prototypage actuel des modèles ML  
* Qu'est-ce que Gradio et ses avantages    
* Prise en main de Gradio        
* Composants de Gradio        
* Entrées multiples dans Gradio        
* Débogage et signalement        
* Amélioration de l'UI + UX        
* Partage et authentification avec Gradio        
* Déploiement d'une application Gradio DL avec Hugging Face Spaces        
* Résumé de fin de cours
