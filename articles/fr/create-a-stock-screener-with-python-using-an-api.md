Les stock screeners sont utilisés pour scanner et filtrer des actions. Saviez-vous que vous pouvez créer votre propre stock screener en utilisant Python gratuitement ?

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous montrera comment utiliser l'API Ameritrade pour programmer un stock screener avec Pandas et Python.

**Ce cours est destiné à des fins éducatives uniquement. Les informations présentées dans cette vidéo ne constituent pas des conseils en investissement.**

Sam Focht a développé ce cours. Il gère la chaîne Python Basics et a réalisé de nombreux streams en direct sur la chaîne YouTube freeCodeCamp.org.

Vous pouvez regarder [le cours complet sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/HXGtLIoiv3Q) (70 minutes de visionnage).
