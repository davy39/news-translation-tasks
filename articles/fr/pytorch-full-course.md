PyTorch est une bibliothèque open source d'apprentissage automatique pour Python qui facilite la création de projets de deep learning. Nous avons publié un cours de 10 heures qui vous mènera de débutant complet en PyTorch à son utilisation pour coder vos propres GANs (réseaux antagonistes génératifs). Vous n'avez même pas besoin de savoir ce qu'est un GAN pour commencer !

Ce cours axé sur la pratique est accessible aux personnes débutant avec le deep learning et les réseaux de neurones. Le cours a été développé par [Aakash](https://twitter.com/aakashns) de [Jovian.ml](http://jovian.ml/). Il s'agit d'un cours complet qui couvre les sujets suivants :

* Bases de PyTorch et régression linéaire
* Classification d'images avec régression logistique
* Entraînement de réseaux de neurones profonds sur GPU avec PyTorch
* Classification d'images utilisant des réseaux de neurones convolutifs
* Réseaux résiduels, augmentation de données et régularisation
* Entraînement des réseaux antagonistes génératifs (GANs)

Il y a du code et des notes détaillées pour accompagner chaque section de ce cours. Vous pouvez accéder au code dans les Jupyter Notebooks fournis. Cela vous permet d'essayer le code vous-même à chaque étape.

Si vous avez voulu en savoir plus sur le deep learning mais que vous ne saviez pas par où commencer, c'est un excellent point de départ pour votre voyage d'apprentissage du deep learning. Il sera utile d'avoir une compréhension de base de Python avant de commencer.

Vous pouvez regarder le cours ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=GIsg-ZUy0MY).
