SQLite est un moteur de base de données très facile à utiliser inclus avec Python. SQLite est open source et constitue une excellente base de données pour les petits projets, les projets de loisirs, ou les tests et le développement.

Nous avons publié un cours vidéo complet pour vous aider à apprendre les bases de l'utilisation de SQLite3 avec Python. Vous apprendrez à créer des bases de données et des tables, à ajouter des données, à trier des données, à créer des rapports, à extraire des données spécifiques, et bien plus encore. Le cours est enseigné par John Elder de Codemy.com.

Pour ce cours, vous devriez déjà avoir une connaissance de base de la programmation Python, mais vous n'avez pas besoin de connaître quoi que ce soit sur les bases de données ou SQLite.

Dans ce cours, vous apprendrez :

* qu'est-ce qu'une base de données
* comment installer Python
* comment installer le terminal git bash
* comment utiliser sqlite dans un programme Python
* comment se connecter à une base de données en Python
* comment créer une base de données
* comment créer une table
* comment insérer un enregistrement dans une table
* comment insérer plusieurs enregistrements dans une table
* comprendre les types de données
* comment sélectionner des données dans une table
* comment formater vos résultats
* comment utiliser la clause where
* comment utiliser la clause like et les caractères génériques
* comment utiliser AND et OR
* comment mettre à jour des enregistrements
* comment limiter et ordonner les résultats
* comment supprimer des enregistrements
* comment supprimer (drop) une table et faire des sauvegardes
* et bien plus encore !

À la fin du cours, vous apprendrez à construire une application très basique pour renforcer tout ce que vous avez appris dans le cours.

Vous pouvez regarder le cours vidéo ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=byHcYRpMgI4) (1,5 heure de visionnage).
