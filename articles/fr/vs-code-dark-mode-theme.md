Je suis ravi de partager la première extension officielle VS Code de freeCodeCamp – un thème de mode sombre.

Ce thème utilise la palette de couleurs distinctive, la typographie et les sensibilités de conception d'interface utilisateur de freeCodeCamp.org.

Ce thème de mode nuit est minimaliste, agréable pour les yeux et vous aidera à atteindre un état de flux de codage.

Voici un aperçu de ce à quoi ressemble ce thème de mode nuit :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-16-at-12.43.07-PM.png)
_Thème sombre VS Code – Style freeCodeCamp_

Vous pouvez télécharger ce thème depuis le [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=freeCodeCamp.freecodecamp-dark-vscode-theme).

Ou vous pouvez simplement ouvrir l'onglet des extensions de votre éditeur VS Code et rechercher "freeCodeCamp Dark Theme" pour l'installer.

## À propos du thème de mode sombre

Ce thème est basé sur l'esthétique visuelle "Command Line Chic" de freeCodeCamp. Si vous voulez vous sentir comme dans Mr. Robot, ce style est fait pour vous.

Nous avons conçu ce thème de mode nuit pour trouver un équilibre entre contraste et saveur, lisibilité et panache.

Si vous êtes un passionné de design, nous avons codifié Command Line Chic dans le [Guide de style de design](https://design-style-guide.freecodecamp.org/) de notre organisation à but non lucratif.

Je pense que vous allez l'adorer. Bon codage.
