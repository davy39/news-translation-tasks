De nombreux cours de C++ vous apprennent à construire des projets factices sans application réelle. Mais il est souvent plus utile d'écrire des programmes que les gens utiliseraient réellement.

Nous venons de publier un cours de C++ de 5 heures qui vous apprendra à écrire du code C++ moderne en créant un plugin audio à l'aide du framework JUCE.

Ce cours a été développé par Charles Schiermeyer, connu sous le nom de Matkat Music. Non seulement Charles est un programmeur expérimenté, mais il est aussi un excellent musicien et a été nommé 3 fois aux Grammy Awards.

Dans ce cours, vous apprendrez à créer un plugin audio doté d'un égaliseur à 3 bandes avec un analyseur de spectre en haut. Il dispose d'une bande coupe-bas (low cut), d'une bande coupe-haut (high cut) et d'une bande paramétrique. Vous ne savez pas ce que c'est ? Pas de problème. Vous n'apprendrez pas seulement la programmation, mais vous en apprendrez aussi un peu sur la musique.

Vous apprendrez à utiliser le framework JUCE pour créer le plugin. Il s'agit d'un framework d'application C++ multiplateforme, utilisé pour le développement d'applications de bureau et mobiles.

Voici les sections abordées dans ce cours :

* Configuration du projet
* Création des paramètres audio
* Configuration du DSP
* Configuration de l'hôte du plugin audio
* Connexion des paramètres Peak
* Connexion des paramètres LowCut
* Refactorisation du DSP
* Ajout de curseurs à la GUI
* Dessiner la courbe de réponse
* Construire le composant de courbe de réponse
* Personnaliser les visuels des curseurs
* Grille de la courbe de réponse
* Analyseur de spectre
* Boutons de bypass
