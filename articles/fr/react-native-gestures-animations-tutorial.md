React Native permet aux développeurs de créer des applications natives Android et iOS en utilisant React et JavaScript. React Native utilise les mêmes blocs de construction fondamentaux de l'interface utilisateur que les applications iOS et Android régulières. Pour faire ressortir vos applications, vous devriez inclure des gestes et des animations.

Nous avons publié un cours vidéo de William Candillon qui vous apprendra comment ajouter des gestes et des animations déclaratifs à vos projets React Native.

Dans ce cours, vous apprendrez :

1. Pourquoi ? Vous apprendrez pourquoi les gestes et les animations sont importants à comprendre lors de l'utilisation de React Native. Vous apprendrez également des stratégies et des APIs importantes.
2. Transitions. Les transitions sont le moyen le plus simple d'animer des composants dans React Native.
3. Animations. Vous apprendrez à créer une simple animation de timing qui peut boucler, pause, reprendre, pause, reprendre, en utilisant les APIs d'animation Bare Metal.
4. Gestes. Vous apprendrez à ajouter des gestes pour interagir avec des cartes et des éléments à l'écran.
5. Animations SVG. Vous apprendrez comment ces gestes et animations s'intègrent très bien avec SVG et vous apprendrez à créer un curseur circulaire en utilisant des gestes et des animations SVG.
