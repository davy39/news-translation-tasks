Dans ce tutoriel intermédiaire Python Django de JustDjango, vous apprendrez à créer un système multi-utilisateurs en utilisant React et le Django Rest Framework. 

Le tutoriel montre comment créer un système de gestion de l'apprentissage similaire à Moodle ou Blackboard. Les utilisateurs ayant le rôle d'"enseignant" peuvent créer des devoirs. Les utilisateurs ayant le rôle d'"étudiant" peuvent compléter les devoirs et voir leurs résultats.

Le Django REST framework est un outil puissant et flexible pour construire des APIs Web. Utiliser ce framework en backend et React en frontend est une combinaison puissante.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=JIFqqdRxmVo) (3 heures de visionnage).
