Le Low-Code peut vous permettre de créer des applications rapidement et avec des composants UI prêts à l'emploi.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser des solutions Low-Code en construisant 3 applications financières.

Ania Kubów a développé ce cours. Elle est une ancienne courtière financière devenue développeuse logicielle.

Pensez aux plateformes low-code comme des constructeurs d'applications qui vous fournissent essentiellement tous les composants UI dont vous avez besoin et vous donnent également la possibilité de les intégrer en utilisant des entrées pré-configurées pour les API.

Ce type de solution est parfait pour les suiveurs personnels de vos finances, les applications internes qui doivent être créées rapidement, ou peut-être des prototypes de preuve de concept à la recherche d'investissements.

La première application que vous allez construire est un tableau de bord de trading pour que les courtiers puissent enregistrer leurs transactions ainsi que suivre leurs commissions. L'application communique également avec leurs back offices grâce à un envoi de messages intégré via l'API Slack.

La deuxième application est un tableau de bord Crypto utilisant le service de CoinAPI pour obtenir des prix en direct. Il utilise également l'API de Twilio pour envoyer des messages texte à vos amis ou à votre famille concernant les changements que vous voyez instantanément.

La dernière application est un suiveur de factures qui communique avec une base de données externe en utilisant l'API GraphQL. Il vous permet d'intégrer des vidéos instructionnelles pour ceux qui l'utilisent.


Le cours couvre :

* Low-Code VS No-Code
* Cas d'utilisation
* Installation de Retool
* Intégrations compatibles
* Construction d'un tableau de bord de trading
* Construction d'un tableau de bord Crypto
* Construction d'un suiveur de factures
* Et bien sûr, où aller ensuite
