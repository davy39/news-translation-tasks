Si vous devez connaître le nombre de mots ou de caractères dans un document Google, il est facile de le découvrir en utilisant les outils que Google Docs lui-même vous offre.

Que ce soit pour un document Google ou un fichier Word importé, cela fonctionne de la même manière – et je vais vous montrer comment dans ce guide rapide.

## Où trouver l'outil de compte de mots dans Google Docs

Vous pouvez trouver l'outil de compte de mots dans le menu Outils – c'est le deuxième élément du menu en haut. Ou vous pouvez utiliser le raccourci clavier `Ctrl+Shift+C`.

![Image](https://www.freecodecamp.org/news/content/images/2021/04/image-53.png)
_Menu Outils et fonctionnalité de compte de mots._

## Comment fonctionne le compte de mots dans Google Docs

L'outil de compte de mots affiche les statistiques de votre document actuel. Donc, si vous cliquez sur "Compte de mots" dans le menu Outils, ou utilisez ce raccourci clavier, il affichera les statistiques du document sur lequel vous travaillez.

Il affiche d'abord le nombre de pages dans votre document, puis le nombre de mots, puis le nombre de caractères, et enfin le nombre de caractères excluant les espaces. Vous pouvez voir un exemple ci-dessous :

![Image](https://www.freecodecamp.org/news/content/images/2021/04/image-54.png)

Il suffit de cliquer sur "Ok" ou "Annuler" pour fermer cette fenêtre.

## Comment avoir les statistiques du document toujours visibles sur la page

Si vous visez un certain nombre de mots dans un document, vous pouvez vouloir surveiller constamment votre compte de mots – sans avoir à rouvrir l'outil tout le temps.

Vous pouvez le configurer pour que les statistiques soient toujours visibles sur votre page actuelle. Dans l'outil de compte de mots, en dessous de tout le reste, il y a une case à cocher appelée "Afficher le compte de mots pendant la frappe". Si vous cochez cette case, cela fera apparaître le compte de mots dans le coin inférieur droit de la fenêtre du document, comme ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/04/image-55.png)

Le nombre de mots est en fait un menu déroulant. Vous pouvez cliquer dessus pour voir toutes les statistiques et changer quelle statistique est visible lorsque le menu déroulant est fermé. Il vous donne également une option pour masquer à nouveau l'outil de compte de mots.

![Image](https://www.freecodecamp.org/news/content/images/2021/04/image-56.png)

## Conclusion

Google Docs vous montre facilement le nombre de mots, de pages et de caractères dans votre document en utilisant l'outil de compte de mots. Et il est facile de l'ouvrir à partir du menu Outils, ou en utilisant le raccourci clavier `Ctrl+Shift+C`.
