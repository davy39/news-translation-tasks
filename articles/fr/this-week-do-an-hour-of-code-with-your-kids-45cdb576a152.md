L'Heure de Code a commencé en 2013 avec pour objectif de susciter l'enthousiasme des enfants pour la programmation et l'informatique.

Cet événement d'une semaine encourage les enseignants et les parents à passer une heure à coder avec leurs enfants.

C'est devenu un mouvement énorme dans le monde entier, avec des participants célèbres comme Malala et même le président Obama.

Cette année, l'Heure de Code se déroule du 5 au 11 décembre.

Combien un enfant peut-il apprendre sur la programmation en une seule heure ? Pas grand-chose. Mais le but est de les aider à s'amuser et à découvrir en eux le désir d'en apprendre davantage.

Pour vous éviter la peine de concevoir votre propre programme adapté aux enfants, plusieurs organisations ont conçu des défis spéciaux d'une heure. Vous et vos enfants pouvez en relever un ou plusieurs.

### Le défi de l'Heure de Code de Free Code Camp sur le développement front-end

![Image](https://cdn-media-1.freecodecamp.org/images/1*NRvQ5wp36Oz4lb9R-fq0Kw.png)

Notre communauté open source se concentre sur l'aide aux adultes pour apprendre à coder afin qu'ils puissent obtenir des emplois de développeurs. Mais nous voulons aussi aider les enfants à s'intéresser à la programmation dès leur plus jeune âge. Nous avons donc créé un défi de codage d'une heure.

Les enfants n'ont pas besoin de créer un compte ou de faire quoi que ce soit pour se préparer. Ils peuvent simplement aller sur notre page [Heure de Code](http://freecodecamp.com/hour-of-code) et commencer.

Ces défis de codage spéciaux se concentrent sur le développement front-end avec JavaScript. Ils sont [totalement open source](https://github.com/FreeCodeCamp/hour-of-code).

La semaine dernière, deux campeurs qui avaient récemment terminé les premières 1 200 heures du programme de Free Code Camp se sont rencontrés pour concevoir et coder ces défis :

* Cindy Hsu est une étudiante de première année en informatique à l'Université de la Colombie-Britannique. En dehors du codage, elle aime les jeux vidéo et diffuse souvent [en direct sur Twitch](https://www.twitch.tv/princekun).
* Danny Libin est pharmacien et s'intéresse depuis longtemps à la technologie. Il s'est récemment [mis plus sérieusement à la programmation](https://github.com/Daynil).

Voici [un lien vers ce défi](http://www.freecodecamp.com/hour-of-code) pour commencer.

### L'Heure de Dessin avec Code de Khan Academy

![Image](https://cdn-media-1.freecodecamp.org/images/1*k3eqex_Gr9enNK23PmsdAw.png)

Vous pouvez utiliser JavaScript pour faire toutes sortes de graphiques dans votre navigateur.

Ce défi de l'Heure de Code guidera vos enfants à travers l'utilisation de formes pour construire un bonhomme de neige.

Voici [un lien vers ce défi](https://www.khanacademy.org/computing/hour-of-code/hour-of-drawing-code/v/welcome-hour-of-code) pour commencer.

### L'Heure de SQL de Khan Academy

![Image](https://cdn-media-1.freecodecamp.org/images/1*_JWuFaOQtR5f4NHHvYYfHQ.png)

Ce défi plus avancé donne aux enfants l'occasion d'apprendre comment fonctionnent les bases de données relationnelles.

Il propose plusieurs vidéos conceptuelles, ainsi que des instructions pour créer des requêtes SQL simples.

Voici [un lien vers ce défi](https://www.khanacademy.org/computing/hour-of-code/hour-of-sql/v/welcome-to-sql) pour commencer.

### Le défi de l'Heure de Code de Minecraft

![Image](https://cdn-media-1.freecodecamp.org/images/1*VbyFMFKpn4JMdUPnWqXluA.png)

Il s'agit d'un défi de codage amusant qui utilise le langage de programmation Scratch, un langage conçu spécifiquement pour que les enfants apprennent les concepts de base de la programmation.

Voici [un lien vers ce défi](https://studio.code.org/s/minecraft/stage/1/puzzle/1) pour commencer.
