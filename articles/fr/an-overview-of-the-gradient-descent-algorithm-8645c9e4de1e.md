#### L'algorithme subtil mais puissant qui optimise les paramètres

Optimiser les paramètres est l'objectif ultime de chaque algorithme d'apprentissage automatique. Vous voulez obtenir la valeur optimale de la pente et de l'ordonnée à l'origine pour obtenir la ligne de meilleure ajustement dans les problèmes de régression linéaire. Vous voulez également obtenir la valeur optimale pour les paramètres d'une courbe sigmoïdale dans les problèmes de régression logistique. Alors, que diriez-vous si je vous disais que la descente de gradient fait tout cela ?
