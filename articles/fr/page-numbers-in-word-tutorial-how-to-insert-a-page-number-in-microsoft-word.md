Si vous écrivez un livre ou un document pour l'école, vous voudrez probablement inclure des numéros de page. Ils aideront les lecteurs à suivre leur progression et leur permettront de référencer des passages spécifiques dans le texte. 

Et si vous écrivez dans Microsoft Word, il existe un moyen simple d'ajouter des numéros de page à votre travail. Voyons comment faire.

## Comment ajouter des numéros de page dans Word

Lorsque vous avez un document Word ouvert, vous verrez le menu principal de Word en haut de votre écran, comme ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-12.56.18-PM.png)
_Menu principal de Word en haut de votre écran._

### Étape 1 : Cliquez sur l'onglet "Insertion" dans le menu principal de Word

Cliquez simplement sur l'onglet "Insertion", et un menu déroulant avec de nombreuses options apparaîtra. Environ aux deux tiers du menu, vous verrez une option "Numéro de page", comme ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-12.59.02-PM.png)
_Cliquez simplement sur "Insertion" dans le menu principal, puis faites défiler jusqu'à "Numéro de page"._

### Étape 2 : Faites défiler et sélectionnez l'option "Numéro de page"

Lorsque vous cliquez sur "Numéro de page", une petite boîte apparaîtra vous demandant comment vous souhaitez formater vos numéros de page.

### Étape 3 : Formatez vos numéros de page

Vous aurez des options pour choisir où placer les numéros ("Position"), comment les aligner sur la page ("Alignement"), et d'autres options de formatage. Cela ressemblera à ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-1.02.04-PM.png)
_Comment formater vos numéros de page._

### Étape 4 (facultatif) : Personnalisez vos numéros de page

Si vous cliquez simplement sur les menus déroulants à côté de chaque option, vous pourrez choisir exactement où et comment vous souhaitez que vos numéros apparaissent.

Par exemple, si je veux que mes numéros apparaissent en bas à droite, je laisserai ces options par défaut. Mais si je veux qu'ils apparaissent, par exemple, en haut et à l'intérieur de chaque page, je peux apporter les modifications suivantes :

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-1.40.25-PM.png)
_Comment personnaliser la numérotation de vos pages._

Vous pouvez également choisir si vous souhaitez qu'un numéro apparaisse sur la première page (peut-être pas si c'est une page de titre, etc.).

Si vous souhaitez vraiment entrer dans les détails de vos numéros de page, vous pouvez cliquer sur le bouton "Format...". Là, vous pouvez choisir l'apparence de vos numéros (vous pouvez avoir des chiffres romains si vous le souhaitez !), si vous souhaitez inclure des numéros de chapitre, et comment vos numéros doivent commencer.

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-2.19.33-PM.png)
_Plus d'options pour les numéros de page._

Et voilà – vos pages sont maintenant numérotées !

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-5.18.53-PM.png)

## Comment trouver l'onglet Numéros de page dans Word

Voici une fonctionnalité intéressante de Word : si vous allez dans l'onglet "Aide" pour rechercher une fonctionnalité, il vous montrera où la trouver dans le menu principal.

Voici ce que je veux dire :

Supposons que je veux trouver où ajouter des numéros de page (et que je n'ai pas ce tutoriel pratique). Je pourrais simplement cliquer sur l'onglet "Aide" dans le menu principal et taper "Numéros de page".

Mais ce n'est pas la partie intéressante – pendant que vous tapez, vous verrez une correspondance apparaître (surlignée en rose ci-dessous). Lorsque vous survolez cette option (ne cliquez pas encore), Word affiche où trouver cet outil ou cette fonctionnalité (dans la boîte rouge ci-dessous) et pointe vers votre requête avec une flèche bleue pulsante ! Comme ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/03/Screen-Shot-2021-03-04-at-2.46.58-PM.png)
_Cliquez sur "Aide", tapez "numéros de page", survolez "Numéros de page", et le menu apparaîtra à gauche._

Ensuite, si vous cliquez réellement sur votre requête de numéros de page dans l'outil d'aide, il vous y emmènera directement et vous verrez la boîte "Numéros de page" apparaître.

Maintenant, vous savez comment ajouter des numéros de page dans Microsoft Word et vous pouvez personnaliser ces numéros à votre guise. Bonnes écritures !
