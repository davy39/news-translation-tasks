Pour protéger vos données Excel contre les modifications et les manipulations indésirables, vous pouvez souhaiter verrouiller les cellules et protéger la feuille de calcul.

Heureusement, Excel est un outil d'analyse de données hautement personnalisable qui vous permet de verrouiller des cellules et de protéger vos feuilles de calcul avec un mot de passe en même temps.

Dans ce guide, je vais vous montrer comment verrouiller des cellules et protéger une feuille de calcul avec un mot de passe dans Excel.

## Comment verrouiller toutes les cellules dans Excel

Par défaut, toutes les cellules sont verrouillées dans Excel, mais cela n'est vraiment efficace que lorsque la feuille est protégée.

Pour vérifier que toutes les cellules sont verrouillées, appuyez sur `Ctrl + A` pour sélectionner toutes les cellules.

Faites un clic droit n'importe où dans la sélection et cliquez sur « Format de cellule »
![ss5-1](https://www.freecodecamp.org/news/content/images/2022/05/ss5-1.png)

Passez à l'onglet Protection
![ss6b](https://www.freecodecamp.org/news/content/images/2022/05/ss6b.png)
Vous pouvez voir que toutes les cellules sont verrouillées par défaut.

## Comment protéger la feuille de calcul

Pour rendre ce verrouillage de cellules efficace, vous devez protéger la feuille de calcul.

Pour protéger la feuille de calcul, cliquez sur l'onglet Révision et sélectionnez « Protéger la feuille ».
![ss1-1](https://www.freecodecamp.org/news/content/images/2022/05/ss1-1.png)

Entrez un mot de passe qui sera requis pour déprotéger la feuille afin qu'elle puisse être modifiée et cliquez sur « Ok ».
![ss2-1](https://www.freecodecamp.org/news/content/images/2022/05/ss2-1.png)

Maintenant, si quelqu'un essaie de modifier les cellules, il reçoit un message indiquant que la feuille est protégée.
![ss3-1](https://www.freecodecamp.org/news/content/images/2022/05/ss3-1.png)

**N.B. :** Si vous n'entrez pas de mot de passe, n'importe qui peut déverrouiller la feuille et manipuler vos données.

## Comment verrouiller des cellules spécifiques dans Excel

**Tout d'abord, vous devez déverrouiller toutes les cellules.**

Par défaut, toutes les cellules sont verrouillées dans Excel, comme je l'ai discuté ci-dessus. Donc, la première chose que vous devez faire pour verrouiller certaines cellules est de déverrouiller toutes les cellules.

Pour déverrouiller toutes les cellules, appuyez sur `Ctrl + A` pour sélectionner toutes les cellules.
![ss4-1](https://www.freecodecamp.org/news/content/images/2022/05/ss4-1.png)

Faites un clic droit n'importe où dans la sélection et cliquez sur « Format de cellule ».
![ss5-1](https://www.freecodecamp.org/news/content/images/2022/05/ss5-1.png)

Passez à l'onglet Protection et décochez « Verrouillé », puis cliquez sur « Ok ».
![ss6-1](https://www.freecodecamp.org/news/content/images/2022/05/ss6-1.png)

**Pour verrouiller des cellules spécifiques :**
Sélectionnez les cellules que vous souhaitez verrouiller, faites un clic droit dans la sélection et cliquez sur « Format de cellule ».
![ss7-1](https://www.freecodecamp.org/news/content/images/2022/05/ss7-1.png)

Passez à l'onglet Protection, cochez « Verrouillé », et cliquez sur « Ok ».
![ss8-1](https://www.freecodecamp.org/news/content/images/2022/05/ss8-1.png)

Rappelez-vous que le verrouillage des cellules n'est pas efficace tant que vous ne protégez pas la feuille de calcul. Donc, passez à nouveau par [les processus pour protéger la feuille de calcul](#heading-comment-proteger-la-feuille-de-calcul).

Maintenant, si quelqu'un essaie de modifier l'une des cellules, Excel lui indiquera qu'elle est protégée.
![ss9](https://www.freecodecamp.org/news/content/images/2022/05/ss9.png)

## Conclusion

Dans ce guide, vous avez appris non seulement à verrouiller des cellules, mais aussi à les protéger pour que personne ne puisse modifier vos données Excel.

Une autre raison pour laquelle vous devriez envisager de verrouiller des cellules dans Excel est de protéger vos formules afin que vos calculs ne soient pas manipulés.
