Python est l'un des langages de programmation les plus populaires au monde. Il est utilisé partout, du développement web aux scripts simples, en passant par l'apprentissage automatique (machine learning) et l'IA.

Nous venons de publier un tutoriel Python sur la chaîne YouTube de freeCodeCamp.org qui vous enseignera les bases de Python.

Bobby Stearman a développé ce cours. Bobby est ingénieur logiciel senior et a créé de nombreux cours techniques de qualité.

Ce cours est idéal pour les débutants qui souhaitent apprendre à utiliser le langage de programmation Python. Vous apprendrez tout, de la configuration du système à la syntaxe de base, jusqu'à l'utilisation des API.

Voici les sections abordées dans ce cours :

* Installation
* Interpréteur Python
* Éditeur de texte
* Git : documents d'accompagnement
* Nombres
* Chaînes de caractères (Strings)
* Listes
* Tuples
* Ensembles (Sets)
* Dictionnaires
* Instructions If
* Instructions Match
* Boucles
* Clauses de boucle
* Modules
* Clauses de boucle
* Erreurs et exceptions
* Classes
* Environnements virtuels
* Construire un projet

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/8124kv-632k) (3 heures de visionnage).
