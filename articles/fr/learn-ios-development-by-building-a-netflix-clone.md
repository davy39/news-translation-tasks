Swift est l'un des principaux langages utilisés pour le développement iOS. Quelle meilleure façon d'apprendre à créer une application iOS qu'en recréant une application populaire ?

Nous venons de publier un cours de 5 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Swift 5, UIKit et Xcode pour développer des applications iOS en créant un clone de Netflix.

Amr a créé ce cours. Amr est un développeur iOS qui a créé de nombreux tutoriels populaires.

Dans ce cours, vous apprendrez à implémenter le modèle de conception MVVM et à créer l'application à partir de zéro dans Xcode. Le code Netflix lira des vidéos en utilisant l'API YouTube.

Voici les sections de ce cours :

* Introduction et démonstration de l'application
* Création d'un nouveau projet Xcode
* Création de MainTabBarViewController
* Configuration de HomeViewController TableView
* Configuration de home TableViewCell et de sa CollectionView
* Création de l'en-tête de la table (HeaderView)
* Personnalisation de la barre de navigation
* Définition des titres des sections de la TableView
* Envoi de requêtes URL et analyse de la réponse JSON
* Utilisation des extensions
* Consommation de l'API pour récupérer les données de chaque section
* Refactorisation des modèles
* Création d'une CollectionViewCell personnalisée
* Passage de données à la CollectionView
* Affichage des images de posters dans CollectionViewCell
* Création de la TableView "À venir" dans l'onglet Upcoming
* Création d'une TableViewCell personnalisée à partir de la table upcoming
* Création de TitleViewModel
* Création de la TableView "Top Recherche" dans l'onglet TopSearch
* Création de SearchResultsViewController pour afficher les résultats de recherche
* Interrogation de la base de données pour un film individuel
* Utilisation de l'API YouTube
* Analyse de la réponse de l'API YouTube
* Gestion des sélections de cellules (Appuyer sur les cellules)
* Création de TitlePreviewViewController
* Refactorisation du titre Hero de TableViewHeader
* Gestion des appuis dans tous les ViewControllers
* Core Data (Bonnes pratiques)
* Utilisation du Centre de notification pour mettre à jour les ViewControllers
