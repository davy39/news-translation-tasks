Par Per Harald Borgen

Le HTML est l'un des éléments de base du web, car il encapsule le contenu des sites web que vous visitez. Cependant, malgré son énorme influence, très peu de personnes savent réellement comment il fonctionne.

Afin d'améliorer la culture technologique dans le monde, nous nous sommes associés avec le développeur, designer et enseignant [Eric Tirado](https://medium.com/u/3b3bb1053d15) et avons créé un [cours gratuit sur le HTML5](https://scrimba.com/g/ghtml?utm_source=freecodecamp.org&utm_medium=referral&utm_campaign=ghtml_launch_article).

Tout au long du cours, Eric vous guidera de débutant à compétent en HTML tout en vous montrant comment construire un site web. Et le meilleur dans tout ça : il vous faudra moins d'une heure pour compléter le cours !

Examinons maintenant comment il est structuré.

### Contenu du cours

Le cours contient 14 screencasts interactifs. Chaque screencast traite d'un concept clé du HTML, et la plupart durent entre trois et six minutes. Dans certains d'entre eux, vous serez encouragé à plonger dans le code et à expérimenter par vous-même, car cela est possible avec la plateforme Scrimba.

#### Leçon #1 : Introduction au web

![Image](https://cdn-media-1.freecodecamp.org/images/1*uY2K9BXnJ-uP4X9vAPvfDg.png)

Il commence par vous enseigner un peu sur le web en général, en regardant les clients, les serveurs et les trois langages que parlent les navigateurs : HTML, CSS et JavaScript.

Cela vous donne une bonne base pour les prochaines leçons, car vous comprendrez mieux le rôle global du HTML sur le world wide web.

#### Leçon #2 : Création d'un document HTML

![Image](https://cdn-media-1.freecodecamp.org/images/1*O0F7kNLF412jOLz6dP3XkQ.png)

L'étape suivante consiste à créer votre premier document HTML. Cette leçon expliquera comment fonctionnent les fichiers HTML et vous donnera également un rapide aperçu de la barre d'URL dans le navigateur.

#### Leçon #3 : Imbrication des éléments

![Image](https://cdn-media-1.freecodecamp.org/images/1*84ti63CghL17e22bu_7mIg.png)

L'imbrication est un concept critique en HTML, donc il sera couvert dans la troisième leçon. L'imbrication signifie essentiellement que vous pouvez imbriquer des balises les unes dans les autres. Pour imbriquer un élément HTML, il suffit de l'ajouter entre les balises d'ouverture et de fermeture d'un autre élément HTML.

#### Leçon #4 : Éléments de tête et scripts

```html
<head>  
  <title>Apprendre le HTML</title>  
  <meta charset="UTF-8">  
  <meta name="description" content="Mon premier site web HTML">  
  <meta name="viewport" content="width=device-width, initial-scale=1.0">  
  <link rel="stylesheet" href="index.css">  
  <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.6/css/all.css">  
  <style>  
    h1 {  
      text-align: center;  
    }  
  </style>  
  <script>  
    document.getElementById("h1").innerHTML = "Bonjour Univers !";  
  </script>  
</head>

```

L'élément head est la première balise HTML après la balise `<html>` elle-même. C'est un conteneur pour les métadonnées et définit souvent le titre, le jeu de caractères, les styles, les liens, les scripts et d'autres informations méta.

Dans cette leçon, vous apprendrez donc à remplir la balise meta avec une série de différents types de contenu, comme vous pouvez le voir dans l'extrait ci-dessus.

#### Leçon #5 : Éléments de mise en page

![Image](https://cdn-media-1.freecodecamp.org/images/1*F39HrDLec0NvKoaMK3lZDg.png)

Le HTML5 a introduit une série d'éléments de mise en page qui rendent le HTML plus sémantique. Ces nouvelles balises renforcent désormais la signification des éléments de mise en page les plus couramment utilisés, comme `<header>`, `<article>`, `<aside>` et `<footer>`, alors qu'autrefois nous n'avions qu'un élément sans signification comme `<div>` à utiliser. Dans ce screencast, vous apprendrez les plus utiles et les plus importants.

#### Leçon #6 : Éléments de figure et d'image

![Image](https://cdn-media-1.freecodecamp.org/images/1*5TENKIfD5Q3klY2skLS14Q.png)

Les images sont un élément central de presque tous les sites web. Dans cette leçon, vous apprendrez à ajouter des images au site, ainsi qu'à ajouter des légendes.

#### Leçon #7 : Intégration de vidéos

![Image](https://cdn-media-1.freecodecamp.org/images/1*N45m6ef7HyUyDEM4oTJ-qw.png)

Vous avez peut-être remarqué que les vidéos YouTube sont partout sur le web, n'est-ce pas ? Pas seulement sur youtube.com. Cela est possible grâce à l'une des fonctionnalités HTML pour intégrer des médias dans une page depuis ailleurs sur le web. Dans cette leçon, nous couvrons les méthodes pour intégrer des vidéos depuis des sites comme YouTube et également directement depuis un fichier vidéo.

#### Leçon #8 : Barre de navigation et éléments de liste

![Image](https://cdn-media-1.freecodecamp.org/images/1*sGbn8eMzacjpzJQoO8UaFw.png)

Dans cette leçon, vous apprendrez à construire la barre de navigation en utilisant une liste non ordonnée et des éléments de liste. Cela implique à la fois une barre de navigation horizontale et verticale, pour une utilisation dans l'en-tête et dans la barre latérale de notre site.

#### Leçon #9 : En-têtes

![Image](https://cdn-media-1.freecodecamp.org/images/1*Xh0zp6eosxQsCp9C3s8s5Q.png)

Les en-têtes sont créés en utilisant la balise `<h*>`, où le signe * doit être remplacé par des nombres de 1 à 6. L'importance de l'en-tête diminue à mesure que le nombre augmente, ce qui signifie que `<h1>` est celui que vous utiliseriez normalement pour votre titre ou texte le plus important sur votre page.

#### Leçon #10 : Paragraphe et mise en forme du texte (partie 1)

![Image](https://cdn-media-1.freecodecamp.org/images/1*aydm1Ni1yLwAQn3xn8HxpA.png)

La mise en forme du texte est également importante à connaître pour construire des sites web, et dans cette leçon, vous apprendrez les bases. Vous reconnaîtrez probablement beaucoup des concepts ici depuis les traitements de texte comme Word et Pages.

#### Leçon #11 : Mise en forme du texte (partie 2)

![Image](https://cdn-media-1.freecodecamp.org/images/1*SZ3DuSMio-lNb1EQ5vHf-g.png)

Dans cette leçon, vous apprendrez davantage sur la mise en forme, mais avec des balises qui affectent de grands blocs de texte. Des exemples de cela sont l'élément `<code>` et l'élément `<blockquote>`.

#### Leçon #12 : Tableaux (partie 1)

![Image](https://cdn-media-1.freecodecamp.org/images/1*AGQ66VeOeDOJzO1qK-KLig.png)

Les tableaux peuvent être un peu compliqués, mais cette leçon essaie de l'expliquer aussi simplement que possible. Deuxièmement, deux leçons seront dédiées à ce sujet afin que vous le compreniez correctement.

#### Leçon #13 : Tableaux (partie 2)

![Image](https://cdn-media-1.freecodecamp.org/images/1*9KLwnPQq2FDXhEgnB5ulAQ.png)

Dans cette leçon, vous apprendrez des éléments plus avancés dans les tableaux, comme `<thead>`, `<tbody>` et `<tfoot>`. Ceux-ci vous permettent de contrôler les éléments à l'intérieur de l'en-tête, du corps et du pied de page séparément.

Vous serez également exposé à `<colgroup>`, qui vous permet d'ajouter des styles à des colonnes spécifiques.

#### Leçon #14 : Lien de page et dernières touches

![Image](https://cdn-media-1.freecodecamp.org/images/1*Fj7VP0FXfgB7KExjKwhMgw.png)

Dans le dernier screencast, nous conclurons et terminerons le site web. Nous ajouterons un lien depuis la navigation vers diverses sections de la page, et nous introduirons également une autre page afin que vous compreniez comment fonctionnent les liens entre les pages.

À ce stade, vous devriez avoir une solide compréhension du HTML et des balises les plus importantes du langage.
