Apprenez à utiliser Angular Material dans ce cours complet pour débutants de Codevolution. 

Angular Material fournit un ensemble de composants UI réutilisables et accessibles basés sur les spécifications Material Design de Google. Les composants suivent les principes modernes de conception web comme la portabilité des navigateurs, l'indépendance des appareils et la dégradation élégante.

Ce cours couvre :

* Module Material
* Typographie
* Bouton
* Bouton bascule
* Icônes
* Badges
* Indicateur de progression
* Barre de navigation
* Navigation latérale
* Menu
* Liste
* Grille de liste
* Panneau d'expansion
* Cartes
* Onglets
* Sélecteur d'étapes
* Entrée
* Sélection
* Saisie automatique
* Case à cocher et bouton radio
* Sélecteur de date
* Info-bulle
* Barre de notification
* Boîte de dialogue
* Tableau de données
* Exploration du tableau de données
* Filtrage du tableau de données
* Tri du tableau de données
* Pagination du tableau de données
* Défilement virtuel

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=jUfEn032IL8) (3 heures de visionnage).
