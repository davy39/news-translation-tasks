La plupart des sites web ont des pages de destination et quelqu'un doit créer toutes ces pages de destination.

Nous avons publié un cours de 5 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un site web responsive à partir de zéro en utilisant HTML, SCSS et JavaScript.

Jessica Chan a créé ce cours. Elle gère la populaire chaîne YouTube Coder Coder et a créé de nombreux cours de logiciels. Dans ce cours, Jessica explique pourquoi elle fait les choses d'une certaine manière, et pas seulement comment les faire.

Dans la première partie de ce cours, vous apprendrez à configurer le projet et à créer la barre de navigation de bureau. La deuxième partie couvre la création de l'animation du menu hamburger et de l'overlay. Vous pourrez voir tout le processus, y compris l'écriture de code et la résolution de problèmes. Ensuite, dans la troisième partie, vous apprendrez à créer le menu mobile hors-canvas et à ajuster l'overlay.

Dans les prochaines parties, vous apprendrez à créer le héros responsive avec une image de fond et à construire une mise en page responsive à 4 colonnes pour les sections Fonctionnalités et Articles de blog en utilisant flexbox et CSS grid. Enfin, vous apprendrez à utiliser les zones de modèle de grille CSS pour construire le pied de page de la page de destination.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=aoQ6S1a32j8) (5 heures de visionnage).
