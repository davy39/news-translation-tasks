Les éditeurs de code source sont les programmes qui permettent la création et l'édition de code. 

N'importe quel éditeur de texte peut être utilisé pour écrire du code. Mais les éditeurs de code dédiés offrent de nombreuses fonctionnalités utiles, notamment : la complétion de texte, la coloration de la syntaxe, et les onglets/panneaux pour l'édition côte à côte. Ces fonctionnalités peuvent être utiles même pour les nouveaux utilisateurs qui ne connaissent pas toutes les options avancées.

Il existe de nombreux éditeurs disponibles à télécharger ou à utiliser en ligne, gratuits ou payants. Ils offrent de nombreuses fonctionnalités de base similaires, mais chacun a ses propres subtilités. Des recherches supplémentaires peuvent aider à trouver celui qui vous convient.

Les éditeurs suivants sont tous multiplateformes et gratuits à utiliser ou à essayer.

## **[Sublime Text](https://www.sublimetext.com/)**

![Icône de Sublime Text](https://discourse-user-assets.s3.amazonaws.com/original/2X/1/1249e18855b9c9ef177e636e6c4e308b4877bf2d.png)

![Capture d'écran de Sublime Text](https://discourse-user-assets.s3.amazonaws.com/original/2X/6/63f774c31a7b1a439bcef9d11e95463b6998310e.png)

Sublime Text est un éditeur très populaire qui existe [depuis 2008](https://www.sublimetext.com/blog/articles/one-point-oh). Il existe de nombreuses options et [extensions](https://packagecontrol.io/) disponibles, ainsi qu'une grande quantité d'[aide en ligne](http://stackoverflow.com/search?q=sublime+text) pour vous aider à [le configurer](http://www.websightdesigns.com/wiki/Setting_up_Sublime_Text_3).

(La licence pour une utilisation continue coûte 70 $. Cependant, Sublime Text est gratuit à télécharger et à évaluer, avec une fenêtre contextuelle de rappel.)

## **[Brackets](http://brackets.io/)**

![Icône de Brackets](https://discourse-user-assets.s3.amazonaws.com/original/2X/d/df30640741461569071875cf4ff6cad9924f8b32.png)

![Capture d'écran de Brackets](https://discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

Brackets est un éditeur open-source relativement nouveau développé par Adobe. Il est très convivial, surtout pour les personnes qui ne sont pas habituées aux interfaces en ligne de commande ou aux paramètres/préférences de style JSON. 

Les extensions et les thèmes sont rapides et faciles à trouver et à installer via le gestionnaire d'extensions.

## **[Atom](https://atom.io/)**

![Icône d'Atom](https://discourse-user-assets.s3.amazonaws.com/original/2X/7/7f6ba67d0ea310e1c1024f83c9acc1e0ecf255a8.png)

![Capture d'écran d'Atom](https://discourse-user-assets.s3.amazonaws.com/original/2X/8/8016e7c0937e8ce6fcb6104c9aee0026443912f5.png)

Atom est un éditeur [open source](https://github.com/atom/atom), développé par [GitHub](https://github.com/). Comme Sublime Text, Atom est assez populaire.

Il est très configurable ("hackable", comme ils disent) mais les fonctionnalités avancées peuvent être déroutantes pour les nouveaux utilisateurs. Il existe également de nombreuses [documentations utiles](http://readwrite.com/2014/05/20/github-atom-5-tips-getting-started-tutorial-corey-johnson/) [disponibles](http://flight-manual.atom.io/) [en ligne](http://stackoverflow.com/search?q=atom).

## **[Visual Studio Code](https://code.visualstudio.com/)**

![Icône de Visual Studio Code](https://i.imgur.com/b4vFsKa.png)

![Capture d'écran de Visual Studio Code](https://code.visualstudio.com/home/home-screenshot-mac-lg.png)

Visual Studio Code est un éditeur de code source développé par Microsoft en 2015, et il est maintenant l'un des éditeurs de code les plus connus pour le développement web. 

Il prend en charge de nombreux langages de programmation, et comme les autres, vous pouvez télécharger des plugins ou des extensions pour le rendre plus configurable selon votre style de codage. Ses principales fonctionnalités sont :

* IntelliSense - Pour la coloration syntaxique, l'autocomplétion, les définitions de fonctions et la complétion intelligente.
* Débogage - Capacité à déboguer en utilisant des points d'arrêt, des piles d'appels et une console interactive.
* Git intégré - Passer en revue les différences, préparer les fichiers, faire des commits, pousser et tirer, tout cela est disponible dans l'éditeur.
* Extensions - Installer des extensions pour ajouter de nouveaux langages, thèmes, débogueurs et connecter des services supplémentaires.
