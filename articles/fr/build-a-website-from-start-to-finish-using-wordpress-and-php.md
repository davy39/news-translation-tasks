Environ 16 % des sites web les plus populaires utilisent WordPress, et près de 80 % de tous les sites web ayant un code back-end utilisent PHP comme langage de programmation côté serveur.

PHP détient une part de marché énorme, il peut donc être utile de comprendre ces technologies.

Nous avons publié un cours de 5 heures sur la chaîne YouTube freeCodeCamp.org qui vous montrera comment créer un site web entièrement personnalisé en utilisant WordPress, PHP, HTML et CSS.

Eric Oliver, de la chaîne YouTube The Dev Drawer, a créé ce cours complet. Eric a plus de 20 ans d'expérience dans le travail avec des logiciels et l'enseignement des compétences en programmation.

Ce cours enseigne tout ce que vous devez savoir pour créer un site web de A à Z. Vous apprendrez à configurer votre serveur local, à créer le design et à construire un thème WordPress personnalisé qui fonctionne avec Elementor.

Le cours est très pratique. Vous pouvez suivre le cours en utilisant votre propre design de site web ou utiliser le design de site exemple montré dans le cours.

Il y a un time-lapse montrant comment Eric a conçu le site exemple en utilisant Illustrator. Eric explique le processus de réflexion derrière ses décisions de design afin que vous puissiez appliquer ces principes à votre propre design de site web.

Une fois que vous aurez appris à construire votre site web, vous apprendrez à le transférer de votre serveur local vers un serveur de production. Vous apprendrez également à configurer le SEO, la sécurité et la mise en cache.

Il peut être utile d'avoir une certaine expérience avec PHP au préalable, mais ce n'est pas obligatoire. Le cours explique tout ce que vous devez savoir.

Voici les sujets abordés dans ce cours :

* Concevoir votre site web
* Configurer votre serveur local
* Créer votre base de données WordPress
* Télécharger WordPress
* Installer WordPress
* Installer Elementor et les plugins de thème
* Créer les fichiers de votre thème
* Utiliser functions.php pour configurer les paramètres par défaut de votre thème
* Créer les pages par défaut du thème
* Installer votre thème
* Configuration générale
* Créer les blocs d'en-tête et de pied de page
* Créer votre page d'accueil
* Créer un type de publication personnalisé et des méta-boxes
* Créer des shortcodes en utilisant vos CPT
* Modifier les colonnes pour vos CPT
* Créer vos menus
* Créer des pages supplémentaires
* Créer des pages individuelles pour vos CPT
* Créer un formulaire
* Créer des modèles de section Elementor
* Créer un autre CPT
* Valider et modifier la version mobile
* Exporter votre site web pour la production
* Plugins recommandés pour la sécurité, le SEO et la mise en cache
* Configurer la sécurité
* Configurer la mise en cache
* Configurer le SEO
