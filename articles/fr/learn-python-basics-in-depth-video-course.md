Si vous avez toujours voulu apprendre Python, vous avez de la chance. Mike Dane a créé ce cours vidéo approfondi pour Python. Il dure 4 heures et demie, donc il vous faudra probablement au moins un week-end pour le suivre.

Dans cette vidéo, Mike vous guidera à travers des concepts importants de Python et vous aidera à construire quelques projets de base en Python. Voici ce que vous pouvez vous attendre à accomplir :

* Installer Python & PyCharm
* Installation et Hello World
* Dessiner une forme
* Variables et types de données
* Travailler avec des chaînes de caractères
* Travailler avec des nombres
* Obtenir des entrées des utilisateurs
* Construire une calculatrice basique
* Construire un jeu de Mad Libs
* Listes
* Fonctions de liste
* Tuples
* Fonctions
* Instructions de retour
* Instructions If
* Instructions If et comparaisons
* Construire une meilleure calculatrice
* Dictionnaires
* Boucles While
* Construire un jeu de devinettes
* Boucles For
* Fonctions exponentielles
* Listes 2D et boucles imbriquées
* Construire un traducteur
* Commentaires
* Utiliser Try / Except
* Lire des fichiers
* Écrire dans des fichiers
* Modules et Pip
* Classes et objets
* Construire un quiz à choix multiples
* Fonctions d'objet
* Héritage
* ...Et l'interpréteur Python

Vous pouvez regarder la vidéo de Mike sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=rfscVS0vtbw) (4 heures de visionnage).
