Ces termes régissent l'utilisation du site web [https://www.freecodecamp.org](https://www.freecodecamp.org/). Pour utiliser le site web, vous devez accepter ces termes avec Free Code Camp, Inc., l'organisation à but non lucratif qui gère le site web.

L'organisation peut offrir d'autres produits et services, sous des termes différents. Ces termes s'appliquent uniquement à l'utilisation du site web.

## Termes importants

Ces termes incluent un certain nombre de dispositions importantes qui affectent vos droits et responsabilités, telles que les renonciations dans les clauses de non-responsabilité, les limites de la responsabilité de l'organisation à votre égard dans les sections sur les limites de responsabilité, votre accord pour couvrir l'organisation en cas de dommages causés par votre mauvaise utilisation du site web dans la section sur la responsabilité de votre utilisation, et un accord pour arbitrer les différends dans la section sur les litiges.

## Votre permission d'utiliser le site web

Sous réserve de ces termes, l'organisation vous donne la permission d'utiliser le site web. Cette permission n'est pas exclusive à vous, et vous ne pouvez pas la transférer à quelqu'un d'autre. Les autres doivent accepter ces termes pour eux-mêmes afin d'utiliser le site web.

## Conditions d'utilisation du site web

Votre permission d'utiliser le site web est soumise aux conditions suivantes :

Vous devez avoir au moins treize ans.

Vous ne pouvez plus utiliser le site web si l'organisation vous contacte directement pour vous dire que vous ne pouvez pas.

Vous devez utiliser le site web conformément aux normes d'utilisation acceptable et de contenu.

## Utilisation acceptable

Vous ne devez pas enfreindre la loi en utilisant le site web.

Vous ne devez pas utiliser ou tenter d'utiliser le compte d'un autre sur le site web sans sa permission spécifique.

Vous ne devez pas acheter, vendre ou échanger des noms d'utilisateur ou d'autres identifiants uniques sur le site web.

Vous ne devez pas envoyer de publicités, de chaînes de lettres ou d'autres sollicitations via le site web, ou utiliser le site web pour collecter des adresses pour des listes de diffusion commerciales.

Vous ne devez pas automatiser l'accès au site web, ou surveiller le site web, par exemple avec un robot d'indexation, un plug-in ou une extension de navigateur, ou un autre programme informatique qui n'est pas un navigateur web. Vous pouvez indexer le site web pour un moteur de recherche public.

Vous ne devez pas utiliser le site web pour envoyer des e-mails à des listes de diffusion, des groupes de discussion ou des alias de messagerie de groupe.

Vous ne devez pas laisser entendre faussement que vous êtes affilié à l'organisation ou approuvé par celle-ci.

Vous ne devez pas créer de liens hypertextes vers des images ou d'autres contenus non hypertextes sur le site web.

Vous ne devez pas supprimer les marques de propriété intellectuelle des matériaux que vous téléchargez depuis le site web.

Vous ne devez pas afficher une partie du site web sur d'autres sites web en utilisant des iframes.

Vous ne devez pas désactiver, éviter ou contourner les restrictions de sécurité ou d'accès du site web.

Vous ne devez pas surcharger l'infrastructure du site web avec un volume déraisonnable de requêtes, ou des requêtes conçues pour imposer une charge déraisonnable sur les systèmes d'information sous-jacents du site web.

Vous ne devez pas encourager ou aider quiconque à violer ces termes.

Vous ne devez pas usurper l'identité d'autres personnes via le site web.

## Normes de contenu

Vous ne devez pas soumettre de contenu au site web qui est illégal, offensant ou autrement nuisible aux autres. Cela inclut le contenu qui est harcelant, inapproprié ou abusif.

Vous ne devez pas soumettre de contenu au site web qui viole la loi, porte atteinte aux droits de propriété intellectuelle de quiconque, viole la vie privée de quiconque ou enfreint les accords que vous avez avec d'autres.

Vous ne devez pas soumettre de contenu au site web contenant du code informatique malveillant, tel que des virus ou des logiciels espions.

Vous ne devez pas soumettre de contenu au site web en tant que simple espace réservé, pour conserver une adresse particulière, un nom d'utilisateur ou un autre identifiant unique.

Vous ne devez pas utiliser le site web pour divulguer des informations que vous n'avez pas le droit de divulguer, comme les informations confidentielles d'autres personnes.

Vous devez créer et vous connecter à un compte pour utiliser certaines fonctionnalités du site web.

Pour créer un compte, vous devez fournir certaines informations sur vous-même. Si vous créez un compte, vous acceptez de fournir, au minimum, une adresse e-mail valide, et de maintenir cette adresse à jour. Vous pouvez fermer votre compte à tout moment en vous connectant à votre compte et en cliquant sur le bouton de la page des paramètres de votre compte.

Vous acceptez d'être responsable de toutes les actions entreprises en utilisant votre compte, que vous les ayez autorisées ou non, jusqu'à ce que vous fermiez votre compte ou que vous informiez l'organisation que votre compte a été compromis. Vous acceptez d'informer immédiatement l'organisation si vous suspectez que votre compte a été compromis. Vous acceptez de sélectionner un mot de passe sécurisé pour votre compte et de le garder secret.

L'organisation peut restreindre, suspendre ou fermer votre compte sur le site web conformément à sa politique de gestion des demandes de retrait liées au droit d'auteur, ou si l'organisation croit raisonnablement que vous avez violé ces termes.
