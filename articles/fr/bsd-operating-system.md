Berkeley Software Distribution (BSD) est un groupe de systèmes d'exploitation (OS) open source de type Unix apparentés, dont les origines remontent aux premières versions de Research Unix chez Bell Labs. FreeBSD est le membre le plus populaire.

BSD est configuré pour l'hébergement internet, l'hébergement web et l'hébergement de nombreux serveurs sur un seul système. C'est le premier OS à avoir ajouté un protocole internet. Les OS BSD disposent d'un système de temps partagé très strictement appliqué, ce qui les rend particulièrement utiles lorsque les ressources sont partagées entre plusieurs processus.

À titre de comparaison, le système d'exploitation Linux est connu pour être préféré pour les processus monotâches, tels que les supercalculateurs et les ordinateurs de bureau. La partie d'interruption forcée du multitâche efficace de BSD dans le cadre du temps partagé entrave les processus dédiés uniques.

BSD inclut un système de "Jails" qui est quelque peu analogue aux conteneurs Linux, mais avec une sécurité et une flexibilité supplémentaires dans la mise en œuvre.
