TensorFlow peut faire des choses incroyables en matière de vision par ordinateur.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser TensorFlow 2 pour des applications de vision par ordinateur.

Nour Islam Mokhtari a créé ce cours. Nour est ingénieur en apprentissage automatique et enseignant expérimenté.

Le cours vous montre comment créer deux projets de vision par ordinateur. Le premier implique un modèle de classification d'images avec un ensemble de données préparé. Le second est un problème plus réaliste où vous devrez nettoyer et préparer un ensemble de données avant de l'utiliser.
