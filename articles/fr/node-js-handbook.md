Le développeur et campeur freeCodeCamp Flavio Copes a publié son guide complet sur Node.js en ligne gratuitement - à la fois sur la publication Medium de freeCodeCamp et sous forme de fichier .pdf. Vous pouvez [le lire ici](https://medium.freecodecamp.org/6912378afc6e).

Dans son guide, Flavio couvre une variété de concepts Node.js, y compris :

* Comment installer Node.js
* Le moteur JavaScript V8
* Comment lire les variables d'environnement depuis Node.js
* Comment utiliser le REPL de Node.js
* Une introduction à npm
* Le guide package.json
* Le fichier package-lock.json
* Comment mettre à jour toutes les dépendances Node à leur dernière version
* La version sémantique avec npm
* Le lanceur de paquets Node npx
* La boucle d'événements
* Les temporisateurs
* La programmation asynchrone et les rappels
* Les promesses
* Async et Await
* L'émetteur d'événements Node
* Comment fonctionnent les requêtes HTTP
* Utiliser les WebSockets dans Node.js
* Lire et écrire des fichiers avec Node
* Les flux Node.js
* Les bases du travail avec MySQL et Node
* Et bien d'autres sujets

Vous pouvez [lire le guide complet de Node.js de Flavio ici](https://medium.freecodecamp.org/6912378afc6e) (95 minutes de lecture).
