Les jeux sont les projets de programmation les plus amusants. Dès que vous avez terminé de créer les jeux, vous pouvez y jouer !

Nous avons publié un tutoriel sur la chaîne YouTube de freeCodeCamp.org qui vous aidera à améliorer vos compétences en JavaScript en construisant deux jeux populaires de l'ère millénaire.

Dans ce tutoriel, la développeuse et enseignante Ania Kubow vous montrera comment créer Doodle Jump et Flappy Bird en JavaScript. Ce tutoriel est parfait pour les personnes qui connaissent un peu JavaScript et qui veulent commencer à tout assembler pour réaliser des projets concrets.

Vous pratiquerez plus de 32 des méthodes les plus populaires de JavaScript afin de construire les deux jeux. Cela inclut .forEach(), .push(), .shift(), .addEventListener(), .appendChild(), les boucles while, les classes, les constructeurs, et bien plus encore.
