React est une bibliothèque JavaScript populaire largement utilisée pour créer des interfaces utilisateur dynamiques. Elle a gagné une grande popularité au fil des ans grâce à sa flexibilité, sa scalabilité et sa facilité d'utilisation.

Si vous êtes un débutant souhaitant apprendre React 18 avec Redux Toolkit, vous avez de la chance. Nous venons de publier un cours complet sur la chaîne YouTube de freeCodeCamp.org.

John Smilga a développé un cours vidéo complet qui couvre toutes les bases nécessaires pour commencer. John est l'un des contributeurs les plus populaires de la chaîne freeCodeCamp.

Le cours est conçu pour vous fournir un guide étape par étape sur la façon de créer des applications web interactives en utilisant React 18 et Redux Toolkit. Ce cours est idéal pour toute personne débutant avec React ou ayant des connaissances de base et souhaitant en savoir plus sur Redux Toolkit.

Le cours est divisé en plusieurs sections qui couvrent différents aspects du développement React. Vous commencerez par l'installation (setup) et la structure des dossiers d'un projet React, puis vous passerez à la création de votre premier composant. Vous apprendrez à travailler avec les extensions et le fichier settings.json, le JSX et l'imbrication de composants.

Ensuite, vous apprendrez à travailler avec Booklist, le CSS, les images, JSX - CSS et JSX - Javascript. Vous apprendrez également à manipuler les Props, y compris comment y accéder et les utiliser dans des configurations dynamiques. Vous découvrirez aussi la Prop Children, Simple List, Proper List, la Prop Key et les options de Props.

Après cela, vous découvrirez les événements (Events), la soumission de formulaires (Form Submission), Mind Grenade, les fonctionnalités des composants (Component Feature), le flux de données React (React Data Flow), les défis (Challenge) et les modules ES6. Vous apprendrez aussi à travailler avec les images locales (dossier src), d'autres défis, le déploiement, le Hook useState, le Hook useEffect, les retours multiples (Multiple Returns) et les formulaires.

Plus tard, le cours couvre le Hook useRef, les Hooks personnalisés (Custom Hooks), l'API Context, le Hook useReducer, la performance et Redux Toolkit. Vous apprendrez à travailler avec Redux Toolkit et à l'utiliser pour gérer l'état de vos applications React.

Dans l'ensemble, ce cours vous offre une introduction complète à React 18 et Redux Toolkit. C'est un excellent moyen de débuter dans le développement React et d'apprendre les bases de la création d'applications web interactives. Le cours est bien structuré et facile à suivre, et le style d'enseignement de John Smilga est captivant et informatif. Que vous soyez un débutant complet ou que vous ayez des connaissances de base en React, ce cours est une excellente ressource pour vous aider à améliorer vos compétences et à commencer à créer vos propres applications React.

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=2-crBg6wpp0) (13 heures de visionnage).
