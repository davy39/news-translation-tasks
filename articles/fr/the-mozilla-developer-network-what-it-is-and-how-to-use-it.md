MDN Web Docs (anciennement connu sous le nom de Mozilla Developer Network ou MDN) est une ressource gratuite pour une documentation approfondie sur les standards du web tels que HTML5, CSS, JavaScript, et bien plus encore.

> La mission de MDN est simple : fournir aux développeurs les informations dont ils ont besoin pour construire facilement des projets sur le [Web ouvert](https://developer.mozilla.org/en-US/docs/Web). Si c'est une technologie ouverte exposée au Web, nous voulons la documenter.  
>   
> – [Déclaration de mission de MDN Web Docs](https://developer.mozilla.org/en-US/docs/MDN/About#Our_mission)

Une communauté de développeurs et d'autres contributeurs maintient MDN Web Docs à jour. L'ensemble du site est un wiki, ce qui signifie que n'importe qui peut ajouter ou modifier la collection croissante de documentation et de tutoriels.

Mais vous n'avez pas besoin d'être un développeur professionnel pour contribuer à MDN Web Docs ! Comme de nombreux autres projets open source, il existe de nombreuses façons d'aider.

Si vous êtes intéressé à contribuer à MDN Web Docs, consultez leur page [Getting Started](https://developer.mozilla.org/en-US/docs/MDN/Getting_started).

Voici quelques liens rapides vers des ressources utiles :

* [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML), [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS), [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
* [Web APIs](https://developer.mozilla.org/en-US/docs/Web/API)
* [Graphics](https://developer.mozilla.org/en-US/docs/Web/Guide/Graphics)
* [MathML](https://developer.mozilla.org/en-US/docs/Web/MathML)
* [Web Assembly](https://developer.mozilla.org/en-US/docs/WebAssembly)

## Comment utiliser MDN Web Docs

MDN a bâti une réputation pour avoir certaines des meilleures documentations sur les technologies web courantes. Si on vous a déjà demandé : "Avez-vous vérifié la documentation ?", il y a des chances qu'ils fassent référence à MDN Web Docs.

Imaginons que vous travaillez sur un projet et que vous souhaitez utiliser la méthode `map()` de JavaScript, mais que vous avez oublié la syntaxe exacte.

Tout ce que vous avez à faire est d'ouvrir la page pour `[map()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map)` et de faire défiler jusqu'à la section [Syntaxe](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map#Syntax).

Là, vous obtiendrez un aperçu rapide de la syntaxe :

```js
let new_array = arr.map(function callback( currentValue[, index[, array]]) {
    // retourner l'élément pour new_array
}[, thisArg])
```

Et une liste des arguments que la méthode accepte, avec une courte description de chacun.

Si vous souhaitez voir quelques exemples de base et des cas d'utilisation courants, faites défiler un peu plus jusqu'à la section [Exemples](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map#Examples). Il y a de bonnes chances que vous trouviez un extrait que vous pourrez utiliser dans votre propre projet.

De plus, si vous devez un jour supporter des navigateurs plus anciens, la plupart, sinon toutes les documentations JavaScript, incluent une section [Polyfill](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map#Polyfill).
