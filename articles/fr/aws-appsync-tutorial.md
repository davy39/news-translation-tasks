AWS AppSync met automatiquement à jour les données dans les applications web et mobiles en temps réel, et met à jour les données pour les utilisateurs hors ligne dès qu'ils se reconnectent. 

Dans ce tutoriel de Manoj Fernando, vous apprendrez à utiliser AWS AppSync pour créer une API GraphQL en utilisant Amplify et Angular.

Voici quelques sujets abordés dans ce tutoriel :

* Introduction aux API GraphQL avec AppSync, Amplify et Angular
* Modélisation des relations dans GraphQL (1:1 | 1:N | M:N)
* Comment envoyer des mises à jour en temps réel avec les abonnements GraphQL
* Création d'une application web hors ligne

Vous pouvez regarder le tutoriel vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/QEMfnr5MO1w) (2 heures de visionnage).
