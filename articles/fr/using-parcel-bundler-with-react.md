Parcel est un bundler pour applications web qui offre des performances ultra-rapides avec une configuration minimale comparé à webpack.

Dans cette vidéo de Nick Karnik, vous verrez comment créer un projet React de base en utilisant Parcel Bundler. Nick explique également les avantages de l'utilisation de Parcel Bundler.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=hCxvp3_o0gM) (11 minutes de visionnage).
