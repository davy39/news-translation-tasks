Tout le monde parle de contribuer à l'open source, mais comment exactement contribuer à un projet ?

Nous avons publié un cours vidéo sur la chaîne YouTube de freeCodeCamp.org qui explique pourquoi vous devriez contribuer à l'open source et comment le faire.

Ce cours complet explique tout, depuis l'ouverture de votre première pull request jusqu'à l'aide à la maintenance d'un projet.

Ce cours a été développé par Eddie Jaoude. Eddie est un ingénieur logiciel senior avec plus de 15 ans d'expérience. GitHub l'a récemment classé parmi les 500 meilleurs contributeurs open source au monde.

Contribuer à l'open source peut améliorer vos compétences techniques et vos compétences en communication. Cela peut vous aider à établir un réseau avec d'autres développeurs et peut même vous aider à décrocher un emploi.

N'importe qui peut faire une contribution utile à l'open source, d'un débutant complet à un développeur expérimenté.

Voici les sujets abordés dans ce cours :

* Qu'est-ce que l'Open Source
* Pourquoi vous devriez vous intéresser à l'Open Source
* Qu'est-ce que Git
* Qu'est-ce que GitHub
* Exemple de profil GitHub personnalisé
* Fonctionnalités de GitHub
* GitHub Actions pour l'Intégration Continue (CI)
* Onglet Insights pour plus d'informations sur le projet
* GitHub Discussions pour les conversations filaires
* Tableau GitHub Projects similaire à Trello
* Wiki GitHub
* Comment trouver des projets Open Source
* Comment écrire en Markdown
* Rédiger une Pull Request (PR)
* Gagner de l'argent directement avec GitHub Sponsors
* Gagner de l'argent indirectement grâce à l'Open Source
* Ressources Open Source de freeCodeCamp
* Tout le monde est un Mainteneur de Projet
* Comment personnaliser votre profil GitHub

Regardez le cours sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/yzeVMecydCE) (1 heure de visionnage).
