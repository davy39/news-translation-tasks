React reste l'une des bibliothèques JavaScript les plus populaires pour construire des interfaces utilisateur. Si vous êtes un développeur front-end, il est très utile de connaître React.

Nous venons de publier un tout nouveau cours React de 12 heures sur la chaîne YouTube freeCodeCamp.org. Dans ce cours, vous apprendrez React en construisant huit projets concrets et en résolvant 140+ défis de codage.

Bob Ziroll enseigne ce cours. Bob est le Responsable de l'Éducation chez Scrimba et son précédent cours React est l'un des cours React les plus populaires sur YouTube.

Voici les projets que vous allez construire :

Votre toute première application React sera un site statique sur React ! Cela vous donne une introduction aux principes de base de la bibliothèque.
