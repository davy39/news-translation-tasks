Il existe une tonne de tutoriels pour des projets Python. Mais presque tous supposent que vous avez un certain niveau de connaissances en Python avant de commencer le tutoriel. 

Si vous êtes un débutant complet en Python et que vous souhaitez construire votre premier projet, nous avons publié le [tutoriel parfait pour vous](https://www.youtube.com/watch?v=_ZqAVck-WeM) sur la chaîne YouTube de freeCodeCamp.org.

Dans ce tutoriel de Tech With Tim, vous apprendrez à construire un jeu simple à partir de zéro, même si vous n'avez jamais utilisé Python auparavant. En travaillant sur ce projet Python pour débutants, vous apprendrez les bases et les fondamentaux de Python et appliquerez ces compétences.

![Image](https://www.freecodecamp.org/news/content/images/2020/08/image-107.png)

Ce tutoriel est conçu pour vous faire démarrer et fonctionner en Python le plus rapidement possible. Vous apprendrez certaines des commandes et des concepts Python les plus populaires, notamment :

* print
* if/else et autres logiques conditionnelles
* imbrication des instructions if/else
* capture d'entrée
* conversions de type
* et bien plus encore.

Et vous apprendrez à tout faire dans votre navigateur, donc vous n'aurez pas besoin d'installer de logiciel.

Regardez le cours complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=_ZqAVck-WeM) (1 heure de visionnage).
