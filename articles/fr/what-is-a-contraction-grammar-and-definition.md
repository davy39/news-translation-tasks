Jetez un coup d'œil aux phrases suivantes :

J'apprends à coder.

Je vais obtenir un emploi de développeur logiciel.

Apprendre à coder ne doit pas vous coûter de l'argent.

Les mots **J'apprends**, **Je vais** et **ne doit pas** sont appelés contractions.

Dans cet article, vous apprendrez tout sur les contractions – ce qu'elles sont, quand et où vous les rencontrerez probablement, quand les utiliser et quand ne pas les utiliser. Enfin, vous verrez certaines des contractions les plus courantes utilisées en anglais.

Commençons !

## Qu'est-ce que les contractions ?

Les contractions, également connues sous le nom de "formes courtes", sont des mots raccourcis.

Plus précisément, une contraction est lorsque deux mots sont raccourcis et combinés pour former un nouveau mot.

Par exemple, `you` et `are` peuvent être combinés pour créer un mot plus court, `you're`.

Lorsque deux mots sont combinés, certaines lettres disparaissent. Vous pouvez perdre une seule lettre ou plus, selon la contraction.

Lorsque vous combinez ces deux mots et que la ou les lettres disparaissent, une apostrophe (`'`) prend leur place.

Les lettres manquantes d'origine sont remplacées par l'apostrophe pour montrer l'endroit où les lettres manquantes devraient être. Ces lettres n'apparaîtront pas dans la contraction (puisqu'elles ont été remplacées par l'apostrophe).

Par exemple, prenons le mot `isn't`. Cette contraction combine les mots `is` et `not`. Lorsque ces deux mots sont associés, la lettre `o` disparaît. Une apostrophe prend maintenant sa place pour montrer où se trouvait la lettre manquante.

Un autre exemple est lorsque `you` et `will` sont combinés pour former `you'll`. Maintenant, deux lettres disparaissent, `w` et `i`, et l'apostrophe remplit cet espace de deux lettres manquantes.

## Quand utilise-t-on les contractions ?

Nous utilisons les contractions tous les jours, à l'oral comme à l'écrit.

Vous les entendrez de la part de vos amis, de votre famille et à la télévision. Vous les verrez dans les romans, les livres non fictifs, les journaux, les manuels d'instructions, les articles de blog, les supports pédagogiques, et bien plus encore. Il y en a quelques-unes dans ce paragraphe même, et je les utilise tout au long de cet article.

Elles sont informelles et décontractées, car elles donnent à votre écriture un ton plus amical, léger, accessible et abordable.

Les contractions peuvent donner au lecteur l'impression que vous lui parlez directement et que vous avez une conversation. Cela aide à rendre votre écriture plus simple pour que tout le monde puisse la comprendre et lui donner un sens.

Parce que les contractions sont plus courtes, cela signifie également qu'elles prennent moins de place. Pour cette raison, vous les verrez souvent dans les publicités où l'espace est précieux.

En résumé, vous pouvez utiliser confortablement les contractions dans une écriture plus informelle et lors d'occasions décontractées.

### Quand éviter d'utiliser les contractions

Il est préférable d'éviter d'utiliser les contractions lorsque vous souhaitez maintenir un ton plus sérieux et formel dans votre écriture.

Par exemple, évitez de les utiliser dans les articles de recherche académiques, les présentations commerciales importantes, ou dans toute situation où une écriture ou un discours informel n'aurait pas beaucoup de sens.

## Contractions courantes

Voici quelques-unes des contractions les plus largement utilisées et courantes que vous rencontrerez en parlant et en écrivant en anglais.

### Contractions courantes se terminant par `-ll`

Les contractions se terminant par `-ll` incluent le mot `will`. Les lettres `w` et `i` sont supprimées.

| Contracté      | Non contracté |
| ----------- | ----------- |
| Je vais   | Je + vais       |
| Elle va      | Elle + va       |
| Il va   | Il + va       |
| Nous allons      |    Nous + allons    |
| Vous allez   | Vous + allez       |
| Ils vont   | Ils + vont        |
| Qui va   | Qui + va       |
| Cela va   | Cela + va       |
| Il y a   | Il y a + va       |
| Qu'est-ce qui va   | Qu'est-ce qui + va       |
| Quand va   | Quand + va       |
| Où va   | Où + va       |
| Comment va   | Comment + va       |

### Contractions courantes se terminant par `-re`

Les contractions se terminant par `-re` incluent le mot `are`. La lettre `a` est supprimée.

| Contracté      | Non contracté |
| ----------- | ----------- |
| Nous sommes   | Nous + sommes      |
| Vous êtes   |    Vous + êtes   |
| Ils sont      | Vous + êtes       |
| Qui sont      | Qui + sont       |
| Qu'est-ce qui sont      | Qu'est-ce qui + sont       |
| Quand sont      | Quand + sont       |
| Où sont      | Où + sont       |
| Pourquoi sont      | Pourquoi + sont       |
| Comment sont      | Comment + sont       |

### Contractions courantes se terminant par `-s`

Les contractions se terminant par `-s` incluent soit le mot `is` ou `has`.

Par exemple :

- `Il va = Il va`
- `C'est parti = C'est parti`.

La lettre `i` ou les lettres `h` et `a`, respectivement, sont supprimées.

| Contracté      | Non contracté |
| ----------- | ----------- |
| Il est   | Il + est / Il + a     |
| Elle est   |  Elle + est / Elle + a  |
| C'est      | C'est + est / C'est + a       |
| Qui est   |  Qui + est / Qui + a  |
| Il y a   |  Il y a + est / Il y a + a  |
| Cela est   |  Cela + est / Cela + a  |
| Qu'est-ce qui est   |  Qu'est-ce qui + est  |
| Quand est   |  Quand + est  |
| Où est   |  Où + est |
| Pourquoi est   |  Pourquoi + est |
| Comment est   |  Comment + est |

Et `let's` ? C'est un peu différent. `let's` vient de `let + us`.

### Contractions courantes se terminant par `-ve`

Les contractions se terminant par `-ve` incluent le mot `have`. Les lettres `h` et `a` sont supprimées.

| Contracté      | Non contracté |
| ----------- | ----------- |
| J'ai  |  Je + ai  |
| Vous avez    | Vous + avez      |
| Nous avons  |  Nous + avons |
| Ils ont  |  Ils + ont  |
| Pourrait avoir  | Pourrait + avoir    |
| Voudrait avoir   |  Voudrait + avoir |
| Devrait avoir  |  Devrait + avoir  |
| Doit avoir  |  Doit + avoir  |
| Pourrait avoir  |  Pourrait + avoir  |
| Qui a  |  Qui + ont  |
| Qu'est-ce qui a  |  Qu'est-ce qui + ont  |
| Quand a  |  Quand + ont  |
| Où a  |  Où + ont  |
| Pourquoi a  |  Pourquoi + ont  |
| Comment a  |  Comment + ont  |

### Contractions courantes se terminant par `-d`

Les contractions se terminant par `-d` incluent soit le mot `had` ou `would`.

Quelques exemples avec le mot `had` :

- `Je ferais mieux d'y aller maintenant = J'avais mieux d'y aller maintenant`
- `Je souhaite ne jamais être parti = Je souhaite ne jamais être parti`.

Quelques exemples avec le mot `would` :

- `Je préférerais ne pas y aller = Je préférerais ne pas y aller`
- `Je voudrais quelque chose à boire s'il vous plaît = Je voudrais quelque chose à boire s'il vous plaît`.


| Contracté      | Non contracté |
| ----------- | ----------- |
| J'aurais    | Je + avais / Je + voudrais     |
| Elle aurait  | Elle + avait / Elle + voudrait |
| Nous aurions  | Nous + avions / Nous + voudrions  |
| Ils auraient |  Ils + avaient / Ils + voudraient |
| Qui aurait  |  Qui + avait / Qui + voudrait |
| Cela aurait  |  Cela + avait / Cela + voudrait |
| Qu'est-ce qui aurait  |  Qu'est-ce qui + avait / Qu'est-ce qui + voudrait |
|Il y aurait |  Il y a + avait / Il y a + voudrait |
| Quand aurait  |  Quand + avait / Quand + voudrait |
| Où aurait  |  Où + avait / Où + voudrait |
| Pourquoi aurait  |  Pourquoi + avait / Pourquoi + voudrait |
| Comment aurait  |  Comment + avait / Comment + voudrait |

### Contractions courantes se terminant par `-m`

Les contractions se terminant par `-m` incluent le mot `am`.

La lettre `a` est supprimée.

| Contracté      | Non contracté |
| ----------- | ----------- |
| Je suis  | Je + suis  |

### Contractions négatives

Les contractions négatives sont celles qui se terminent par `-nt`.

Vous obtenez cela en ajoutant le mot `not` à un verbe, le rendant négatif.

Dans ce cas, la lettre `o` est supprimée.

| Contracté      | Non contracté |
| ----------- | ----------- |
| Ne peux pas  | Peut + pas  |
| N'avais pas  |Avais + pas |
| N'a pas  | A + pas  |
| N'ont pas  | Avoir + pas |
| N'ai pas  | Faisais + pas |
| Ne fait pas  | Fait + pas |
| Ne fais pas  | Fais + pas |
| N'oses pas  | Oser + pas |
| Ne pourrais pas  | Pourrais + pas |
| Ne voudrais pas  | Voudrais + pas |
| Ne devrais pas  | Devrais + pas |
| Ne dois pas  | Dois + pas |
| Ne pourrais pas  | Pourrais + pas |
| Ne devrais pas | Devrais + pas |
| N'ai pas besoin  |Avoir besoin + pas |
| N'étais pas  |Étais+ pas |
| N'est pas |Est+ pas |
| Ne sont pas  |Sont + pas |
| N'étions pas  |Étions + pas |
| Ne devrais pas  | Devrais + pas |

Un mot qui est un peu différent et une exception à ce qui a été montré jusqu'à présent est `won't` – il vient de `will + not = won't`.

`will` ne se transforme pas en `willn't`. En fait, le mot "will" n'apparaît pas du tout dans la contraction. Considérez-le simplement comme une contraction irrégulière (comme nous avons des verbes irréguliers).

## Erreurs courantes avec les contractions

Quelques contractions causent souvent de la confusion et les gens les utilisent souvent de manière incorrecte – vous verrez même des locuteurs natifs anglais faire ces erreurs.

Ces contractions sonnent exactement de la même manière que d'autres mots, donc ces erreurs se produisent couramment à l'écrit.

### You're et Your

`You're` est une contraction, une combinaison des mots `you` et `are`.

Par exemple, `You're learning how to code` ou `You're doing great!`.

`Your` est un pronom possessif, il est utilisé pour indiquer que quelque chose est possédé par/appartient à quelqu'un.

Par exemple, `Your cat is so cuddly!` ou `Your cooking always tastes so good`.

`Your doing great` n'a pas de sens puisque cela n'indique pas que quelque chose appartient à quelqu'un. `You're doing great` en a, puisque cela indique une action et que les verbes sont utilisés pour cela.

Si vous êtes confus et ne savez pas lequel utiliser, lisez-le à voix haute comme `you are` et voyez si cela sonne correctement.

`Your are cat is so cuddly!` n'a pas de sens ou ne sonne pas correctement, par exemple.

### It's et Its

`It's` est une contraction – une combinaison de `it` et `is` ou `has`.

Par exemple, `It's raining outside` ou `It's been great for me so far, I'm really enjoying it here`.

`Its` est un pronom possessif. `Its` montre la possession.

Par exemple, `Don't judge a book by its cover` ou `The cat is in its sleeping basket`.

De manière similaire à l'exemple de la section précédente, si vous êtes confus, utilisez les verbes `is` ou `has` dans votre phrase et vérifiez si cela a du sens d'ajouter l'apostrophe/faire une contraction : `Don't judge a book by it is cover` n'a pas de sens, donc vous utilisez le possessif "its" sans apostrophe.

`It is raining outside` a du sens, donc vous savez maintenant que vous pouvez utiliser une apostrophe.

### They're, Their, et There

Ces trois mots sonnent de la même manière.

`They're` est une contraction. `They` et `are` ont été combinés.

Par exemple, `They're going away for the holiday season` ou `They're buying a house together`.

La phrase sonne-t-elle correctement lorsque vous utilisez `they are` ? Alors utilisez `they're`.

`Their` montre la possession.

Par exemple, `Their dog bit me last night` ou `I don't like their attitude`.

Enfin, `There` indique un lieu, un endroit.

Par exemple, `I wish I was there instead` ou `I'm never going there again`.
