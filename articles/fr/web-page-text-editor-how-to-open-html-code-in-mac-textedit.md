Dans cet article, je vais montrer comment créer, ouvrir et modifier des fichiers HTML dans TextEdit. 

## Qu'est-ce que TextEdit ?

TextEdit est un traitement de texte gratuit qui est préinstallé sur votre ordinateur Mac. Il existe depuis plus de 25 ans et c'est un moyen facile d'ouvrir et de lire des fichiers word, des fichiers texte et même des fichiers HTML. 

## Comment créer un fichier HTML dans TextEdit 

1. Allez dans Finder > Applications > TextEdit. Double-cliquez sur TextEdit pour l'ouvrir. 

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-7.14.26-AM.png)

2. Cliquez sur Format et ensuite sur Faire du texte brut.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.07.01-AM.png)

3. Ajoutez un [modèle HTML de base](https://www.freecodecamp.org/news/basic-html5-template-boilerplate-code-example/) à votre fichier. 

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Démonstration TextEdit</title>
  </head>
  <body>
    <h1>Ceci est une démonstration</h1>
  </body>
</html>
```

4. Allez dans Fichier > Enregistrer...

5. Nommez votre fichier avec l'extension `.html`. Cela indiquera à l'ordinateur qu'il s'agit d'un fichier HTML. Cliquez sur Enregistrer.

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.12.32-AM.png)

6. Cliquez sur Utiliser .html

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.13.58-AM.png)

Allez dans le fichier nouvellement créé sur votre ordinateur et double-cliquez pour l'ouvrir dans le navigateur. Vous devriez voir ce résultat :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.26.53-AM.png)

## Comment ouvrir des documents HTML dans TextEdit

Il existe deux façons d'ouvrir un fichier HTML dans TextEdit. 

### Option 1

Allez dans le fichier HTML que vous souhaitez ouvrir, faites un clic droit sur le fichier et choisissez TextEdit pour l'ouvrir. 

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-7.41.10-AM.png)

### Option 2

Ouvrez TextEdit et cliquez sur Fichier > Ouvrir... Vous pouvez également survoler l'option Ouvrir récent avec votre souris si vous avez ouvert ce fichier récemment. 

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-7.46.38-AM.png)

## Mode d'édition de code de TextEdit

Il existe une option dans TextEdit pour toujours afficher vos fichiers HTML en tant que code HTML.

1. Cliquez sur TextEdit > Préférences...

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.37.38-AM.png)

2. Cliquez sur Ouvrir et Enregistrer :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.38.21-AM.png)

3. Cochez la case qui dit Afficher les fichiers HTML en tant que code HTML au lieu de texte formaté :

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.38.47-AM.png)

Maintenant, vous pouvez apporter des modifications à votre fichier HTML et voir les résultats dans le navigateur.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Démonstration TextEdit</title>
  </head>
  <body>
    <h1>Ceci est une démonstration</h1>
    <p>Ajout de plus de contenu à mon fichier de démonstration</p>
  </body>
</html>
```

![Image](https://www.freecodecamp.org/news/content/images/2021/09/Screen-Shot-2021-09-15-at-8.42.45-AM.png)

Voici comment créer, enregistrer, ouvrir et modifier des fichiers HTML dans TextEdit.
