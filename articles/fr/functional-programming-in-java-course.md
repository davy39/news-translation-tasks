La programmation fonctionnelle est une manière de programmer où les valeurs sont passées dans des fonctions, et ces fonctions sont également des valeurs. Les principes de la programmation fonctionnelle peuvent aider les développeurs à écrire un code élégant, maintenable, évolutif et prévisible.

Nous avons publié un cours complet sur la programmation fonctionnelle en Java sur la chaîne YouTube freeCodeCamp.org. Ce cours a été développé par Nelson Djalo d'Amigoscode. Bien que ce cours soit destiné aux débutants, vous devriez déjà avoir [certaines connaissances en Java](https://www.youtube.com/watch?v=grEKMHGYyns).

Dans ce cours, vous apprendrez quand, pourquoi et comment utiliser la programmation fonctionnelle. Vous découvrirez la puissance des interfaces fonctionnelles de Java et comment elles peuvent être utilisées pour créer des instructions puissantes et des modèles de conception. Vous apprendrez également à écrire du code déclaratif en combinant des interfaces fonctionnelles avec des flux.

Voici les sujets et compétences abordés dans ce cours :

* Impératif vs Déclaratif
* Interfaces Fonctionnelles
* Fonctions, Prédicats, Consommateurs et Fournisseurs
* Flux et Optionnels
* Modèle de Combinateur
* Lambdas
* Rappels (Callbacks)
* Propriétés de la Programmation Fonctionnelle
