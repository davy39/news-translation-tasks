ASP.NET Core est un framework multiplateforme pour créer des applications connectées à Internet. Avec ASP.NET Core fonctionnant sur .NET, vous pouvez créer des applications et services web, des applications pour l'Internet des objets (IoT) et des backends pour mobiles.

Nous avons publié un [cours](https://youtu.be/Pi46L7UYP8I) de 2 heures et demie pour débutants qui vous aidera à apprendre le développement d'applications dans ASP.NET Core 5 en utilisant MVC. Ce cours a été développé par Bhrugen Patel.

Après avoir appris l'histoire et les bases d'ASP.NET Core, vous apprendrez à créer une petite application de liste de livres avec des opérations CRUD. Vous apprendrez également à utiliser Entity Framework pour l'intégration avec une base de données avec ASP.NET Core Razor Pages.

Après avoir créé une application CRUD avec des validations, vous apprendrez à déployer l'application sur Azure ! Vous découvrirez à quel point il peut être rapide de créer un site web fonctionnel pour une entreprise.

Regardez le [cours complet sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/Pi46L7UYP8I) (2 heures et demie de visionnage).
