Node.js et Express.js sont des technologies populaires utilisées pour le développement backend web en JavaScript.

Nous venons de publier un cours de 10 heures sur la chaîne YouTube freeCodeCamp.org qui vous montrera comment créer quatre projets en utilisant Node et Express.

John Smilga a développé ce cours. John est l'un des contributeurs les plus prolifiques de la chaîne YouTube freeCodeCamp. Il a créé une série de cours très appréciés.

![Image](https://www.freecodecamp.org/news/content/images/2021/10/image-48.png)
_John Smilga_

En plus de Node et Express, le cours met également en avant MongoDB, Mongoose, JWT et de nombreuses autres petites bibliothèques JavaScript.

Dans ce cours, vous allez créer les projets Node suivants :

* Gestionnaire de tâches
* API de magasin
* Bases de JWT
* API d'emplois

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=qwfE7fSVaZM) (10 heures de visionnage).
