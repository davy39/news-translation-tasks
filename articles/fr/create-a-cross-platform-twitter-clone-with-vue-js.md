Une façon amusante d'apprendre de nouvelles compétences en programmation est de créer un clone d'une application populaire. Nous avons publié un cours qui vous apprendra à créer un clone Twitter en utilisant Vue.js, le framework Quasar et Firebase.

Danny Connell, de la chaîne Make Apps With Danny, a créé ce cours. Vous apprendrez à créer une belle application Twitter responsive et multiplateforme à partir de zéro et à la faire fonctionner sur 5 plateformes différentes : iOS, Android, Mac, Windows et navigateur web.

Vue.js est un framework JavaScript populaire pour construire des interfaces utilisateur. Le framework Quasar vous permettra de prendre votre code Vue.js et de créer des applications sur plusieurs plateformes avec une seule base de code.

Voici les sections couvertes dans ce cours :

* Ce que nous allons créer
* Qu'est-ce que le framework Quasar ?
* Créer un projet Quasar
* Structure des dossiers Quasar
* Créer une mise en page avec le générateur de mise en page
* En-tête
* Tiroir de gauche (Navigation)
* Tiroir de droite (Barre de recherche et fil d'actualité)
* En-tête mobile
* Pages et routes
* Page À propos
* Formulaire de nouveau tweet
* Liste des tweets - Design
* Liste des tweets - Données
* Ajouter un tweet
* Supprimer un tweet
* Animation
* Dernières retouches de design
* Firebase - Configuration de Cloud Firestore
* Firebase - Obtenir et afficher les tweets
* Firebase - Ajouter un nouveau tweet
* Firebase - Supprimer un tweet
* Firebase - Mettre à jour un tweet (Like)
* Corriger les derniers problèmes
* Déployer sur Mac (Electron)
* Déployer sur Windows (Electron)
* Déployer sur iOS (Cordova)
* Déployer sur Android (Cordova)
