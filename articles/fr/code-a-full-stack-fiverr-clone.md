Nous venons de publier un cours disponible dès maintenant sur la chaîne YouTube [freeCodeCamp.org](http://freeCodeCamp.org). Ce tutoriel complet vous guidera à travers le processus de création d'un clone de Fiverr en utilisant une pile de technologies de pointe.

Vuk Rosić a créé ce cours. Vuk est ingénieur logiciel, développeur en IA et créateur de cours.

Dans ce cours, vous apprendrez à utiliser plusieurs technologies web :

* **NextJS** : Un framework React puissant qui permet des fonctionnalités telles que le rendu côté serveur et la génération de sites web statiques.

* **React** : Une bibliothèque JavaScript pour construire des interfaces utilisateur, maintenant la nature dynamique et réactive de l'application.

* **Convex** : Un outil de développement back-end qui gère la logique serveur et les interactions avec la base de données de manière transparente.

* **Typescript** : Un sur-ensemble de JavaScript qui ajoute des types statiques au langage, aidant à maintenir des bases de code plus grandes.

* **Tailwind CSS** : Un framework CSS basé sur les utilitaires pour construire rapidement des designs personnalisés.

* **ShadCN** : Une collection de composants React magnifiquement conçus, accessibles et personnalisables que vous pouvez utiliser pour construire des applications web modernes.

Le cours est structuré en plusieurs sections, chacune se concentrant sur différents aspects de la construction d'une place de marché de services complète :

1. **Démonstration et Introduction** : Obtenez un aperçu complet du produit final et une vue d'ensemble de la structure du cours.

2. **Installation** : Apprenez à configurer votre environnement de développement avec Convex, Clerk, NextJS et ShadCN.

3. **Développement Front-end** : De la conception d'une page "Créer un service" à la création d'une barre de navigation avancée, apprenez à gérer l'interface utilisateur efficacement en utilisant React et Tailwind CSS.

4. **Intégration Back-end** : Plongez dans les fonctions back-end avec `gig.ts`, utilisez `useApiMutation.ts` pour les interactions avec l'API et gérez les données utilisateur.

5. **Conception de la Base de Données et du Schéma** : Définissez le schéma de la base de données et intégrez-le avec le front-end pour stocker et gérer les catégories, sous-catégories et informations utilisateur.

6. **Gestion de Contenu Dynamique** : Créez et éditez des pages de services, ajoutez des images et gérez le contenu avec un éditeur de titre personnalisé et un éditeur d'offres.

7. **Paiements et Transactions** : Implémentez Stripe pour gérer les paiements, y compris l'ajout de prix, les liens de paiement et les connexions de comptes vendeurs.

8. **Fonctionnalités Interactives** : Ajoutez une fenêtre modale de confirmation, gérez la navigation des services, la recherche et le filtrage sur la page d'accueil.

9. **Interaction Utilisateur** : Améliorez la plateforme avec une page de profil utilisateur, une section de commentaires, un système de messagerie et une page pour gérer les services.

Ce cours vidéo enseigne non seulement les compétences techniques nécessaires pour construire une application web complexe, mais couvre également l'implémentation pratique de ces technologies dans un projet réel. À la fin de ce cours, vous aurez une solide compréhension du développement full stack et la confiance nécessaire pour appliquer ces compétences dans vos futurs projets.
