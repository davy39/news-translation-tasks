Stable Diffusion permet aux gens de créer facilement de l'art IA avec de simples entrées de texte. Je viens de publier un cours vidéo sur Stable Diffusion sur la chaîne YouTube freeCodeCamp.org. Cet article vous présentera le cours et vous fournira des liens importants pour l'installation et la lecture.

Ce cours se concentre sur l'apprentissage de l'utilisation de Stable Diffusion en tant qu'outil, plutôt que d'entrer dans les détails techniques. Pour comprendre certains termes complexes comme VAEs ou embeddings, vous devrez avoir quelques connaissances en apprentissage automatique et faire quelques recherches par vous-même.

Vous devrez avoir accès à un GPU local pour essayer le matériel du cours. Un accès cloud à un GPU (à l'exception de Google Colab) fonctionnerait également. C'est malheureux, mais Google Colab a interdit l'exécution du code Stable Diffusion dans leurs notebooks.

Dans ce tutoriel, nous aborderons les sujets suivants :

1. Comment utiliser Stable Diffusion

2. Comment entraîner votre propre modèle pour un personnage ou un style artistique spécifique (modèles LoRA)

3. Comment utiliser Control Net, un plugin populaire de Stable Diffusion

4. Comment utiliser le point de terminaison API de Stable Diffusion
