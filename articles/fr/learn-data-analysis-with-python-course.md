Python, SQL et d'autres outils open source ont rendu l'analyse de données plus accessible que jamais.

Vous pouvez apprendre à utiliser Python pour l'analyse de données dans ce nouveau cours de 4 heures sur la chaîne YouTube freeCodeCamp.

Ce cours gratuit a été créé par Santiago Basulto de RMOTR.

Santiago enseigne les capacités de Python et de toute la stack PyData pour effectuer des analyses de données.

Le cours passe en revue quelques concepts de base de Python, puis vous apprend :

* comment lire des données à partir de plusieurs sources (CSVs, SQL, Excel)
* comment nettoyer et transformer des données en utilisant NumPy et Pandas
* comment visualiser des données en utilisant Matplotlib et Seaborn
* comment traiter des données pour créer des rapports.

En plus d'apprendre les bases de l'analyse de données, vous apprendrez également à utiliser Jupyter Notebook.

Nous fournissons des fichiers Notebook et du code pour accompagner chaque section afin que vous puissiez exécuter le code vous-même.

Nous incluons également de nombreux exercices pour que vous puissiez pratiquer ce que vous apprenez.

Regardez le cours complet ci-dessous, ou sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/r-uOLxNrNk8) (4 heures de visionnage).
