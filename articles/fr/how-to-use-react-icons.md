seo_desc: "Par Miracle Ugorji\nLorsque vous construisez une application web React, il est probable que vous allez utiliser des icônes. Les icônes sont des représentations graphiques d'un concept, d'une idée, d'un fichier, d'un programme, d'une application, d'une entreprise, et ainsi de suite. \nVous pouvez utiliser des icônes pour représenter l'identité d'une fonctionnalité. Là où vous ne voulez pas écrire le nom d'un programme, une icône appropriée peut le représenter pour une identification facile.\nIl existe de nombreuses bibliothèques d'icônes différentes - comment les ajouter à votre application React ? Nous allons apprendre cela, ainsi que bien plus, dans cet article.\n## Dans cet article, vous apprendrez :\n\n* Qu'est-ce que la bibliothèque react-icons ?\n* Qu'est-ce que les icônes Font Awesome ?\n* Comment configurer l'environnement de développement\n* Comment installer les icônes React\n* Comment importer les icônes React\n* Comment utiliser les icônes React dans vos applications\n* Comment utiliser plus d'une icône\n* Comment rendre plusieurs icônes à partir de différentes bibliothèques d'icônes\n* Comment afficher les icônes en ligne\n* Comment styliser les icônes Font Awesome dans une application React\n* Bonus : Tous les codes d'importation des bibliothèques d'icônes pour React Icons\n* Conclusion\n## Qu'est-ce que react-icons ?\n\nReact-icons est une petite bibliothèque qui vous aide à ajouter des icônes (provenant de différentes bibliothèques d'icônes) à vos applications React. Elle livre les icônes à votre application sous forme de composants pour qu'elles soient plus faciles à utiliser, et elle vous permet de les styliser pour qu'elles soient cohérentes avec le style global de votre application.\n\nReact-icons utilise les fonctionnalités ES6 pour importer les icônes dans votre application React. Et il fait en sorte que seules les icônes que vous utilisez réellement à partir de chaque bibliothèque soient importées.\n\nAvec React-icons, vous n'avez besoin d'exécuter qu'une seule commande pour utiliser les icônes que vous voulez à partir d'une certaine bibliothèque. Vous devez simplement choisir le bon code d'importation et votre icône est prête.\n\nReact-icons fonctionne avec un grand nombre de bibliothèques d'icônes différentes comme Font Awesome, Material UI, Bootstrap icons, et bien d'autres.\n## Qu'est-ce que les icônes Font Awesome ?\n\nIl existe de nombreuses bibliothèques d'icônes différentes, mais l'une des plus populaires est [Font Awesome](https://www.freecodecamp.org/news/how-to-add-font-awesome-icons-to-your-buttons/). Font Awesome est une bibliothèque ou une boîte à outils de graphiques qui sont créés pour se comporter comme des polices.\n\nLes icônes Font Awesome sont comme des polices régulières mais vous obtenez des graphiques au lieu de lettres. Ce sont des graphiques vectoriels scalables (SVG) ce qui signifie que vous pouvez changer leurs couleurs, les redimensionner, et ainsi de suite sans compromettre la qualité.\n\nMais sans react-icons, pour utiliser les icônes Font Awesome dans une application web React, vous devez passer par un long processus d'installation des différents packages en utilisant plusieurs lignes de commande.\n\nParfois, vous devrez peut-être installer le package pro également. Ensuite, vous devez les importer universellement ou dans des fichiers individuels. Tout ce processus, juste pour obtenir quelques icônes sur votre page web ?\n\nC'est là que react-icons intervient. Dans cet article, je vais montrer comment utiliser react-icons pour ajouter des icônes Font Awesome à votre application web React. Commençons.\n## Comment configurer l'environnement de développement\n\nSuivez les étapes ci-dessous pour configurer votre environnement de développement.\n\n> Note : Si vous avez déjà fait votre configuration, il n'est pas nécessaire de la répéter. Passez simplement à la section suivante.\n\n1. Rendez-vous sur [Node.js](https://nodejs.org/en/) pour le télécharger et l'installer si vous ne l'avez pas déjà.\n2. Ouvrez le terminal de votre appareil.\n3. Tapez cette commande : `npm i -g create-react-app fonts-app`. Cela installera une application React avec tous les packages nécessaires pour votre projet.\n4. Ensuite, tapez `cd fonts-app/`. `fonts-app` est le nom de votre dossier de projet. Vous pouvez choisir n'importe quel nom que vous voulez. `cd fonts-app` vous emmène à votre dossier de projet.\n5. Ensuite, exécutez `npm start` qui démarrera le projet sur votre serveur de développement.\n6. À partir de votre éditeur de code, ouvrez le dossier src du projet et trouvez le fichier `App.js`.\n7. Supprimez l'élément React original rendu là et ajoutez votre propre div ou h1 ou tout autre élément que vous préférez (juste à des fins d'exemple).\n\nMaintenant, vous êtes prêt.\n## Comment installer les icônes React\n\nPour installer la bibliothèque `react-icons`, procédez comme suit :\n\n* Dans votre dossier de projet, ouvrez le terminal de votre éditeur de code.\n* Exécutez la commande `npm install react-icons` pour installer la bibliothèque dans votre dossier de projet.\n\nCela peut prendre un certain temps en fonction de la vitesse de votre système. Une fois terminé, vous êtes prêt à utiliser vos icônes React.\n## Comment importer des icônes avec React-Icons\n\nAllez sur la page [React-icons](https://react-icons.github.io/react-icons/). Vous verrez un certain nombre de bibliothèques d'icônes que vous pouvez utiliser. Chaque bibliothèque d'icônes a un code d'importation qui lui est propre. Pour ce tutoriel, nous allons nous concentrer sur la bibliothèque d'icônes Font Awesome.\n\nLe code d'importation de Font Awesome est `import { IconName } from "react-icons/fa";`.\n\n#### Exemple de code\n\n```jsx\nimport { IconName } from "react-icons/fa";\n\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/FontAwesomePage.png)\n_Icônes Font Awesome sur la page React icons_\n\n## Comment utiliser les icônes dans vos applications\n\nSuivez les étapes ci-dessous pour utiliser les icônes Font Awesome dans votre application.\n\n1. Dans `App.js`, collez le code d'importation en haut du fichier après le code d'importation React\n2. Retournez à la page des icônes React et choisissez une icône parmi les icônes Font Awesome\n3. Cliquez sur l'icône pour la copier\n4. Retournez à votre code d'importation dans le fichier `App.js`\n5. Collez et remplacez le `IconName` dans les accolades par l'icône copiée\n6. Rendez le nom de l'icône en tant qu'élément React.\n\n###### Exemple de code\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart } from "react-icons/fa";\n\nclass Like extends Component {\n  render() {\n    return <FaHeart />  \n  }\n}\n\nexport default Like;\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/HeartIcon.png)\n_Icône de cœur Font Awesome rendue_\n\nL'ajout de l'icône `<FaHeart />` à la méthode de rendu affiche l'icône sur votre page web. Cela est possible car nous avons déjà importé l'icône en haut du fichier.\n\n> Note : Vous devez faire correspondre toute icône copiée à son code d'importation. Vous ne pouvez donc pas coller une icône Bootstrap dans un code d'importation d'icône Font Awesome. Faire cela ne vous donnera pas le bon résultat.\n\n## Comment utiliser plus d'une icône\n\nPuisque vous avez déjà importé la bibliothèque d'icônes que vous voulez, vous n'aurez pas besoin de l'importer une deuxième fois. Tout ce que vous avez à faire est de retourner à la page des icônes React. Copiez une autre icône que vous voulez et collez-la dans les accolades.\n\nC'est ainsi que vous pouvez utiliser autant d'icônes que vous le souhaitez à partir de la bibliothèque Font Awesome.\n\n###### Exemple de code\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart />\n      <FaRegHeart />\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/doubleHeart.png)\n_Deux icônes de cœur Font Awesome rendues_\n\n## Comment rendre plusieurs icônes à partir de différentes bibliothèques d'icônes\n\nIl existe des différences subtiles entre les différentes bibliothèques d'icônes. Et parfois, vous ne pourrez peut-être pas trouver toutes les icônes dont vous avez besoin dans une seule bibliothèque.\n\nDans ce cas, vous pouvez utiliser des icônes provenant de plus d'une bibliothèque d'icônes. Vous devez simplement importer les bibliothèques d'icônes que vous voulez. Par exemple, vous pouvez importer à la fois Bootstrap et Font Awesome. Ensuite, vous devez simplement rendre les icônes comme avant.\n\n###### Exemple de code\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\nimport { BsHeartFill, BsHeart } from "react-icons/bs";\n\nclass Icons extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart />\n      <FaRegHeart />\n      <BsHeartFill />\n      <BsHeart />\n    </div>\n    ) \n  }\n}\n\nexport default Icons\n```\n\nLa possibilité d'importer différentes bibliothèques d'icônes vous donne accès à une large gamme d'icônes. Vous pouvez toujours trouver ce dont vous avez besoin au moment où vous recherchez à travers toutes les bibliothèques d'icônes.\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/FourHearts.png)\n_Cœurs rendus à partir des bibliothèques d'icônes Font Awesome et Bootstrap_\n\n> Note : N'utilisez des icônes provenant de plusieurs bibliothèques que si c'est absolument nécessaire, car cela peut ralentir votre application.\n\n## Comment afficher les icônes en ligne\n\nVous avez peut-être remarqué que les icônes s'affichent en disposition de bloc. Cela est dû au fait que nous avons placé les éléments d'icônes les uns après les autres. Si vous voulez qu'ils s'affichent en ligne, placez-les simplement côte à côte. Consultez l'exemple de code ci-dessous :\n\n```\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\nimport { BsHeartFill, BsHeart } from "react-icons/bs";\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart /> <FaRegHeart /> <BsHeartFill /> <BsHeart />\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/separatedHearts.png)\n_Quatre cœurs en ligne_\n\n## Comment styliser les icônes Font Awesome dans une application React\n\nGénéralement, les icônes viennent dans une couleur terne et une taille de base que vous n'aimerez peut-être pas pour votre page. Vous devrez donc styliser les icônes selon vos préférences.\n\nLes icônes React sont assez faciles à styliser. Vous pouvez utiliser les styles suivants pour styliser une icône React :\n\n* Styles en ligne\n* Redimensionnement avec les balises de titre HTML\n* Styles d'objet\n* Feuille de style CSS\n* Contexte d'icône\n\n### Comment utiliser les styles en ligne pour styliser les icônes\n\nVous pouvez styliser une icône React en utilisant des styles CSS en ligne. Cela signifie que vous styliserez l'icône à partir de la balise auto-fermante de l'élément.\n\nUtilisez cette méthode lorsque vous avez une ou deux icônes à styliser. Vous pouvez également l'utiliser lorsque vous avez besoin d'icônes dans différents styles. Consultez l'exemple de code ci-dessous :\n\n###### Exemple de code\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart style={{color: 'red', fontSize: '50px'}}/>\n      <FaRegHeart style={{color: 'green', fontSize: '50px'}}/>\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/twoColoredHearts.png)\n_cœurs Font Awesome colorés_\n\nDans l'exemple ci-dessus, nous passons la couleur et la taille de la police à la propriété style. La première icône est rouge solide. La deuxième icône est en vert.\n\nRemarquez que la taille de la police est écrite en camel case. Les propriétés CSS avec deux mots sont écrites en camel case au lieu d'utiliser `-` pour les séparer en CSS vanilla. C'est la convention dans React.\n\n### Comment redimensionner les icônes avec les balises de titre HTML\n\nVous pouvez également utiliser les balises de titre HTML pour faire en sorte que vos icônes prennent la même taille que la balise de titre utilisée.\n\nPour ce faire, ajoutez l'élément icône entre les balises d'ouverture et de fermeture du titre.\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <h1><FaHeart style={{color: 'red'}}/></h1>\n \t  <h6><FaRegHeart style={{color: 'green'}}/></h6>\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/BigAndSmallHearts.png)\n\nDans le code ci-dessus, nous avons ajouté les deux balises d'icônes entre les balises de titre HTML respectivement.\n\n### Comment styliser les icônes avec des styles d'objet\n\nVous pouvez stocker et utiliser les styles que vous voulez pour vos icônes dans une variable en tant que propriétés d'objet. Déclarez simplement un objet et attribuez tous les styles que vous voulez pour les icônes.\n\nUtilisez cette méthode lorsque vous avez un certain nombre de styles à passer aux icônes. Vous pouvez également utiliser ce style lorsque vous ciblez une icône spécifique. Consultez l'exemple de code ci-dessous :\n\n###### Exemple de code\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\n\nconst fontStyles = {color: 'blue', fontSize: '80px'};\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart style={fontStyles}/>\n      <FaRegHeart style={fontStyles}/>\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/BlueHearts.png)\n\nDans l'exemple de code ci-dessus, nous avons stocké les styles que nous voulions dans un objet. Ensuite, nous avons passé l'objet comme valeur à la propriété style sur l'icône.\n\nToutes les icônes auxquelles les styles d'objet sont passés prennent le même style. Donc dans l'exemple ci-dessus, les deux icônes ont maintenant la même couleur bleue et une taille de 80px.\n\n### Comment styliser les icônes avec des feuilles de style CSS\n\nVous pouvez également utiliser une feuille de style CSS pour styliser vos icônes. L'utilisation d'une feuille de style est la plus adaptée lorsque vous avez plusieurs styles et icônes à incorporer dans votre application.\n\nPour utiliser une feuille de style, vous devez importer la feuille de style dans le fichier où vous avez les icônes. Passez une classe sur les éléments d'icône que vous voulez. Stylisez les éléments d'icône en utilisant la classe dans la feuille de style. Consultez les exemples de code ci-dessous :\n\n###### Exemple de code\n\n```css\n//style.css file\n.yellow{\nfont-size: 80px;\ncolor: yellow\n}\n\n.purple{\nfont-size: 80px;\ncolor: purple\n}\n```\n\n```jsx\nimport React, { Component } from 'react';\nimport { FaHeart, FaRegHeart } from 'react-icons/fa';\nimport "./style.css";\n\nclass Like extends Component {\n  render() {\n    return (\n    <div>\n      <FaHeart className='yellow'/>\n       <FaRegHeart className='purple'/>\n    </div>\n    ) \n  }\n}\n\nexport default Like\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/YellowAndPurpleHearts-1.png)\n_Icônes jaunes et violettes_\n\nLes exemples de code montrent une feuille de style CSS avec des styles sur deux classes. Nous avons importé cette feuille de style dans notre fichier React. Nous avons passé ces classes aux styles sur les éléments React où nous les voulons.\n\n### Comment styliser les icônes avec le contexte d'icône\n\nVous pouvez également importer le contexte d'icône React plutôt que d'ajouter plusieurs noms de classe sur différentes icônes. Donnez au contexte d'icône les valeurs que vous voulez. Ajoutez vos éléments d'icône à l'intérieur du contexte comme montré ci-dessous :\n\n###### Exemple de code\n\n```\nimport React from 'react';\nimport { IconContext } from "react-icons";\n\nconst Like = () => {\n  return (\n    <IconContext.Provider\n      value={{style: { color: '#f4a200', fontSize: '50px' }}}\n    >\n      <div>\n        <FaHeart />\n        <FaRegHeart />\n      </div>\n    </IconContext.Provider>\n  );\n}\n \nexport default Like;\n```\n\n![Image](https://www.freecodecamp.org/news/content/images/2021/09/Two-YellowHearts.png)\n_Deux cœurs jaunes en ligne_\n\nDans l'exemple de code, nous avons importé le contexte d'icône React. Nous avons ensuite utilisé une fonction sans état pour rendre les icônes dans le contexte d'icône.\n\nRemarquez que le contexte d'icône a donné aux icônes un affichage en ligne par défaut. Si vous voulez que les icônes conservent un affichage en bloc, ajoutez alors `display: 'block'` comme l'une des propriétés. En d'autres termes, vous pouvez passer n'importe quel style que vous voulez pour les polices dans la balise de contexte d'icône.\n\n## Tous les codes d'importation des bibliothèques d'icônes pour React-icons\n\nAu cas où vous voudriez utiliser des icônes provenant d'autres bibliothèques (en plus de celles que nous avons discutées ici), voici leurs codes d'importation pour que vous puissiez commencer rapidement avec eux :\n\n```\nimport { IconName } from "react-icons/ai"; //Ant Design Icons\nimport { IconName } from "react-icons/bs"; //Bootstrap Icons\nimport { IconName } from "react-icons/bi"; //Boxicons\nimport { IconName } from "react-icons/di"; //Devicon Icons\nimport { IconName } from "react-icons/fi"; //Feather\nimport { IconName } from "react-icons/fc"; //Flat Color Icons\nimport { IconName } from "react-icons/fa"; //Font Awesome Icons\nimport { IconName } from "react-icons/gi"; //Game Icons\nimport { IconName } from "react-icons/go"; //Github Octicons Icons\nimport { IconName } from "react-icons/gr"; //Grommet-Icons\nimport { IconName } from "react-icons/hi"; //HeroIcons\nimport { IconName } from "react-icons/im"; //IcoMoon Free\nimport { IconName } from "react-icons/io"; //Ionicon4\nimport { IconName } from "react-icons/io5"; //Ionicon5\nimport { IconName } from "react-icons/md"; //Material Design Icons\nimport { IconName } from "react-icons/ri"; //Remix Icons\nimport { IconName } from "react-icons/si"; //Simple Icons\nimport { IconName } from "react-icons/ti"; //Typicons\nimport { IconName } from "react-icons/vsc"; //VS Code Icons\nimport { IconName } from "react-icons/wi"; //Weather Icons\nimport { IconName } from "react-icons/cg"; //css.gg\n\n```\n\nToutes les bibliothèques d'icônes disponibles sur la page React-icons sont utilisées de la même manière que discuté dans cet article.\n\n## Conclusion\n\nLes icônes React sont hautement optimisées, scalables et faciles à utiliser.
