Qu'est-ce qu'une API exactement ? Comment utilise-t-on une API ? Nous venons de publier un cours complet pour débutants sur les Interfaces de Programmation d'Applications (APIs) sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=GZvSYJDk-us).

Ce cours a été créé par [Craig Dennis](https://twitter.com/craigsdennis), éducateur développeur chez Twilio. Il enseigne ce que font les APIs, pourquoi elles existent et quels sont leurs avantages. Les APIs sont utilisées tout le temps en programmation et en développement web, il est donc important de comprendre comment les utiliser.

Vous obtiendrez une expérience pratique avec quelques APIs web populaires, y compris les APIs de Spotify et Twilio. Tant que vous connaissez les bases absolues de la programmation et du web, vous n'aurez aucun problème à suivre.

Dans la première section du cours (Unité 1), vous apprendrez ce qu'est une API et obtiendrez un aperçu du fonctionnement du web. Dans la deuxième section (Unité 2), vous commencerez à explorer les APIs en ligne. Vous apprendrez à utiliser Postman et des bibliothèques d'aide en JavaScript et Python. Dans la section finale (Unité 3), vous apprendrez à créer un projet utilisant une API avec Node.js et Flask.

À la fin du cours, vous aurez une compréhension complète des APIs et serez capable de rire à la bande dessinée suivante. ?
