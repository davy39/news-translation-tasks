Le passage vers les architectures serverless devient rapidement un aspect pivot du développement d'applications. Même Node.js, traditionnellement utilisé avec des serveurs, peut être utilisé pour construire une application serverless.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à développer des applications serverless en utilisant Node.js. Vous apprendrez les nuances du déploiement d'une application Express.js et Node.js sur AWS Lambda.

Vous apprendrez également à tirer parti des capacités de pointe de [Neon Serverless Postgres](https://neon.tech/education) (qui a créé une subvention pour freeCodeCamp pour aider à rendre ce cours possible) et du Serverless Framework pour améliorer votre stratégie de développement et de déploiement d'applications.

Justin Mitchel a créé ce cours. Il est un instructeur populaire et créateur de la plateforme de cours Coding for Entrepreneurs. Justin apporte son expertise approfondie, garantissant que vous ne comprenez pas seulement les aspects théoriques, mais que vous acquérez également une expérience pratique grâce à une approche d'apprentissage basée sur des projets.

Voici quelques-unes des sections clés de ce cours.

* **Ce que le Serverless signifie pour le développement d'applications** : Un aperçu fondamental, préparant les apprenants au changement de paradigme dans la façon dont les applications sont construites et déployées.
* **Prérequis et aperçu technologique** : Familiarisez-vous avec les outils et technologies qui seront utilisés tout au long du cours.
* **Configuration du projet jusqu'au déploiement** : De la configuration initiale à l'exécution d'Express localement avec le Serverless Framework, ce cours garantit une approche pratique pour maîtriser le déploiement serverless.
* **Sécurisation et gestion de votre déploiement** : Apprenez à sécuriser votre déploiement avec AWS System Manager Parameter Store et à gérer efficacement les ressources Neon avec le CLI Neon.
* **Intégration et gestion de la base de données** : Plongez profondément dans [l'intégration de NodeJS avec Neon Postgres](https://neon.tech/education), la gestion des schémas de base de données et l'automatisation des secrets de base de données Neon branchés.
* **Techniques de déploiement avancées** : Maîtrisez les déploiements automatisés via GitHub Actions et explorez l'intégration avec les réécritures dans Next.js et Vercel, aboutissant au déploiement d'Express.js sur Vercel.

Les architectures serverless offrent une scalabilité et une efficacité de coût inégalées, permettant aux développeurs de se concentrer sur le code plutôt que sur la gestion des serveurs. Cette approche non seulement rationalise les processus de déploiement, mais réduit également considérablement les coûts opérationnels, car vous ne payez que pour le temps de calcul que vous utilisez, ce qui en fait une solution idéale pour des projets de toute taille.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=cxgAN7T3rq8) (4 heures de visionnage).
