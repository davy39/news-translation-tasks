Créer votre propre jeu est plus facile que jamais ! GDevelop est un moteur de jeu convivial pour les débutants qui vous aide à créer des jeux 2D avec peu ou pas de code.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser GDevelop pour créer des jeux 2D.

Wesley et Max ont développé ce cours. Tous deux sont développeurs de jeux et créateurs de cours. Ils font partie de l'équipe GDevelop.

En moins d'une heure, vous apprendrez suffisamment pour commencer à créer vos propres jeux.

[Cliquez ici pour essayer un jeu simple créé avec GDevelop](https://games.gdevelop-app.com/game-798e7afe-03c9-4492-8974-28c6076eb896/index.html).

![Image](https://www.freecodecamp.org/news/content/images/2022/01/Screen-Shot-2022-01-27-at-1.19.31-PM.png)
_Jeu créé avec GDevelop._

Voici les sujets abordés dans ce cours :

* Disposition générale
* Types d'objets
* Système d'événements
* Démarrage rapide
* Variables
* Constructeur d'expressions
* Extensions
* Re-création d'Asteroids
* Exporter un jeu

Regardez le cours complet ci-dessous ou sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=iHF5fwsqu4I) (1 heure de visionnage).
