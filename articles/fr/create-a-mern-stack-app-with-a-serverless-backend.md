La stack MERN est un moyen populaire de créer une application web full stack. MERN signifie MongoDB, Express, React et Node.js.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à utiliser la stack MERN en créant une application web de critiques de restaurants. Tout ce que vous devez savoir au préalable, ce sont les bases de JavaScript.

Mais ce cours est encore meilleur que la plupart des tutoriels sur la stack MERN. Une fois l'application MERN stack terminée, vous apprendrez à refactoriser l'application pour rendre le backend complètement serverless. Le nouveau backend utilisera MongoDB Realm au lieu d'Express et Node.

Non seulement vous pouvez héberger votre backend gratuitement sur MongoDB Realm, mais vous pouvez également y héberger votre frontend React. Vous apprendrez comment faire tout cela dans ce cours.

Voici toutes les sections de ce cours :

* Présentation de MongoDB
* Configurer la base de données cloud MongoDB Atlas
* Charger des données d'exemple dans la base de données
* Créer le backend Node / Express
* Créer le frontend React
* Configurer MongoDB Realm et remplacer le backend
* Héberger le frontend sur MongoDB Realm

Ce cours a été rendu possible grâce à une subvention de MongoDB.

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/mrHNSanmqQ4) (2 heures de visionnage).
