Il existe plusieurs façons d'enregistrer vos appels téléphoniques sur votre iPhone. Mais il est vraiment important de demander d'abord la permission de l'autre interlocuteur avant de faire un enregistrement.

## Existe-t-il une option d'enregistrement intégrée ? 

Il n'existe pas d'option intégrée sur iPhone pour enregistrer vos appels téléphoniques.

Dans la plupart des régions du monde, il est illégal d'enregistrer quelqu'un sans sa permission expresse. 

Il existe d'autres applications et appareils que vous pouvez utiliser pour enregistrer vos appels téléphoniques, mais n'oubliez pas de toujours demander la permission au préalable. 

Il est également préférable de vérifier les lois en vigueur dans votre région concernant l'enregistrement des appels téléphoniques. 

## Comment utiliser un appareil séparé pour enregistrer les appels sur iPhone

L'une des façons les plus simples d'enregistrer un appel téléphonique est d'utiliser un autre appareil comme une tablette, un enregistreur ou un ordinateur. Une fois que vous avez la permission, appuyez sur enregistrer sur l'autre appareil.

Une fois l'enregistrement terminé, vous pouvez sauvegarder cet enregistrement sur l'appareil ou le télécharger sur l'iCloud. 

## Utilisation de Google Voice

Google Voice propose une option pour enregistrer les appels téléphoniques, mais avec quelques restrictions.

La première chose à considérer est que Google Voice n'enregistrera que les appels entrants vers votre numéro Google Voice. Cela ne fonctionnera pas pour les appels sortants.

Il est gratuit d'obtenir un [numéro Google Voice](https://voice.google.com/about) et de télécharger l'application sur votre téléphone.

Vous devrez vous connecter à votre compte et suivre les [instructions](https://support.google.com/voice/answer/115061?hl=en&co=GENIE.Platform%3DDesktop) sur la façon de configurer votre numéro Google Voice. Vous pouvez ensuite aller sur l'app store et rechercher l'application Google Voice.

![Image](https://www.freecodecamp.org/news/content/images/2021/10/IMG_D0B0AB533519-1-1.jpeg)

Une fois que vous êtes configuré avec votre nouveau numéro Google Voice, vous pouvez commencer à recevoir des appels entrants. Lorsque quelqu'un vous appelle, appuyez sur le chiffre 4 de votre clavier pour commencer l'enregistrement de l'appel.

N'oubliez pas de demander la permission à tous les participants de l'appel avant de commencer l'enregistrement. 

Pour arrêter l'enregistrement, appuyez à nouveau sur le chiffre 4 de votre clavier.

## Comment utiliser d'autres applications tierces pour enregistrer les appels sur iPhone

Il existe d'autres applications que vous pouvez utiliser pour enregistrer vos appels téléphoniques. Certaines de ces options incluent Call Recorder ou TapeACall.

Beaucoup de ces applications seront gratuites à télécharger, mais vous factureront à la minute pour les appels téléphoniques. Certaines applications offriront des périodes d'essai gratuites, mais il est vraiment important de lire les petites lignes avant de télécharger les applications. 

## Quelle est la meilleure option ?

Si vous cherchez une option facile et sans tracas, alors utiliser un autre appareil pour enregistrer vos appels téléphoniques est la solution idéale. 

Si vous ne voyez pas d'inconvénient à payer pour une application qui enregistre vos appels téléphoniques, alors faites des recherches à travers les avis dans l'app store pour choisir une application qui répond à vos besoins.
