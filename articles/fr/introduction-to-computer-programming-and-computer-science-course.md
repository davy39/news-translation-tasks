Il existe de nombreux concepts fondamentaux que tous les programmeurs doivent comprendre, peu importe le langage de programmation qu'ils utilisent. Nous avons publié un cours vidéo de 2 heures pour enseigner les bases de l'informatique. Les concepts enseignés s'appliquent à tous les langages de programmation.

Ce cours est destiné aux personnes intéressées par l'informatique et la programmation, mais ayant peu de connaissances en codage. Il a été développé par Steven et Sean de NullPointer Exception.

Ce cours répond à toutes ces questions :

* Qu'est-ce que la programmation ?
* Comment écrivons-nous du code ?
* Comment obtenons-nous des informations des ordinateurs ?
* Que peuvent faire les ordinateurs ?
* Qu'est-ce que les variables ?
* Comment manipulons-nous les variables ?
* Qu'est-ce que les instructions conditionnelles ?
* Qu'est-ce que les tableaux ?
* Qu'est-ce que les boucles ?
* Qu'est-ce que les erreurs ?
* Comment déboguons-nous le code ?
* Qu'est-ce que les fonctions ?
* Comment pouvons-nous importer des fonctions ?
* Comment créons-nous nos propres fonctions ?
* Qu'est-ce que les ArrayLists et les dictionnaires ?
* Comment pouvons-nous utiliser les structures de données ?
* Qu'est-ce que la récursivité ?
* Qu'est-ce que le pseudocode ?
* Comment choisir le bon langage ?

Vous pouvez regarder le cours ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=zOjov-2OZ0E).
