Elementor simplifie la création d'un site WordPress grâce à une interface glisser-déposer. Pas besoin de savoir coder !

Nous venons de publier un cours sur Elementor sur la chaîne YouTube freeCodeCamp.org, qui vous apprendra à créer facilement votre propre site web.

J'ai créé ce cours. J'ai également réalisé d'autres cours populaires sur WordPress pour la chaîne YouTube freeCodeCamp.org.

Tout d'abord, vous apprendrez à obtenir un hébergement web et à enregistrer un nom de domaine. Nous utilisons Hostinger pour cela. Hostinger a fourni une subvention qui a rendu ce cours possible.

Ensuite, vous apprendrez à installer WordPress, Elementor et quelques autres plugins utiles.

Une fois tout installé, vous apprendrez étape par étape comment créer un site web avec Elementor et WordPress. Les compétences que vous acquerrez dans ce cours peuvent vous aider à créer différents types de sites web.

Voici les sections de ce cours :

* Hébergement et nom de domaine
* Configuration de l'email personnalisé et SSH
* Installer WordPress
* Installer Elementor et les modèles
* Présentation de l'édition avec Elementor
* Mise à jour de la page d'accueil avec Elementor
* Ajout d'images personnalisées
* Personnalisation du texte et d'autres fonctionnalités
* Ajout de widgets avec Elementor
* Ajout de vidéo au site web
* Personnalisation des arrière-plans
* Rendre le site web esthétique sur mobile
* Mise à jour de la page "À propos"
* Mise à jour de la page de contact
* Configuration du formulaire de contact
* Mise à jour de l'en-tête de page
* Mise à jour du pied de page
* Comment ajouter et supprimer des pages
* Mise à jour des paramètres finaux
* Conclusion
