Par Siraj Raval

#### **Introduction**

Le Machine Learning a traditionnellement été une technologie accessible uniquement aux docteurs et aux institutions disposant de nombreuses ressources financières. Mais de nos jours, il existe tant d'outils qui permettent à chacun de commencer à apprendre le Machine Learning. Plus d'excuses !

Dans cet article de blog, je mettrai en lumière les quatre pierres angulaires du Machine Learning, et comment chacune d'entre elles a été démocratisée ces dernières années.

Si vous souhaitez rester à jour avec mon dernier contenu sur l'IA, assurez-vous de vous [abonner](https://goo.gl/qzhYkG) à ma chaîne YouTube.

![Image](https://cdn-media-1.freecodecamp.org/images/UXNXIvSCjPM4h4d47P-uFcpmLvegCmoQ4yz-)

Les quatre pierres angulaires du Machine Learning sont les **données**, les **calculs**, les **algorithmes** et l'**éducation**.

#### **Données**

Consultez [cette](https://github.com/awesomedata/awesome-public-datasets) liste incroyable de jeux de données publics sur GitHub. Et si cela ne suffit pas, [Kaggle](https://www.kaggle.com/datasets) propose également des jeux de données remarquablement bien préparés. Que vous utilisiez des jeux de données publics ou que vous récupériez des données depuis le web via des bibliothèques Python comme [scrapy](https://www.analyticsvidhya.com/blog/2017/07/web-scraping-in-python-using-scrapy/), tout le monde a désormais accès à des jeux de données de qualité.

Bien sûr, les grandes entreprises technologiques ont leurs propres jardins de données cloisonnés, mais des startups décentralisées comme [OpenMined](https://openmined.org/) travaillent dur pour créer des services permettant aux scientifiques des données d'entraîner leurs modèles sur ces données également.

#### **Calculs**

Vous avez des GPUs ? Le Machine Learning, et surtout le deep learning, nécessitent de nombreux calculs coûteux. Les réseaux de neurones nécessitent l'utilisation de calculs massivement parallèles, pour lesquels les GPUs sont bien adaptés.

Malheureusement, les GPUs peuvent être très chers. Mais avec des outils comme [CoLab](https://colab.research.google.com/) de Google ou [Kernels](https://www.kaggle.com/c/invasive-species-monitoring/kernels) de Kaggle, chacun peut exécuter du code de Machine Learning dans le navigateur en utilisant des GPUs gratuits (Tesla K80).

#### **Algorithmes**

Les algorithmes sont une denrée courante. Heureusement pour nous, le domaine du Machine Learning a construit une culture de code open source et de partage de nombreux résultats. Que ce soit lors de la conférence annuelle NIPS ou ICLR, les chercheurs sont généralement très heureux de partager leurs résultats.

Si vous souhaitez suivre les dernières recherches, vous pouvez utiliser [Arxiv Sanity Preserver](http://www.arxiv-sanity.com/) pour lire les derniers articles de manière magnifiquement indexée. Et il y a bien sûr le [subreddit](https://www.reddit.com/r/MachineLearning/) sur le Machine Learning. Vous pouvez soit utiliser du code existant, soit utiliser la bibliothèque gratuite [Tensorflow](https://www.tensorflow.org/guide/) pour construire vos propres modèles.

#### **Éducation**

Un grand pouvoir implique de grandes responsabilités. Vous avez le code, vous avez les données, vous avez la puissance de calcul. Maintenant, vous devez simplement être formé sur la manière de les utiliser !

Outre ma chaîne [YouTube](https://youtube.com/c/sirajraval), bien sûr, il existe une multitude de ressources éducatives gratuites pour vous aider à apprendre à utiliser les outils du Machine Learning. J'ai créé un programme de Machine Learning de trois mois qui utilise toutes les ressources mentionnées ci-dessus et plusieurs autres que j'ai trouvées sur le Web pour aider un débutant à commencer. Vous trouverez cela [ici](https://www.youtube.com/watch?v=Cr6VqTRO1v0).

![Image](https://cdn-media-1.freecodecamp.org/images/sGuzsG4UpLaGmEHAJLBt8fPM7ZU0GT3BC-fS)

#### Allez de l'avant et apprenez !

Vous devriez être excité en ce moment. C'est un moment incroyable pour être en vie ! Il se passe tant de changements si rapidement. Au milieu de toute cette complexité, le Machine Learning peut nous aider à comprendre notre monde de manières que nous n'aurions pas pu autrement. Il peut nous aider à créer et à découvrir de nouvelles choses de manière exponentiellement plus efficace qu'auparavant. Vous avez le pouvoir, utilisez-le judicieusement.
