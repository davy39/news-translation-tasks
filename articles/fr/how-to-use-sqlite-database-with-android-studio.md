La plupart des applications Android doivent stocker des données quelque part et le moyen le plus courant de stocker des données sur Android est d'utiliser une base de données SQLite.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org sur l'utilisation de la base de données SQLite avec Android Studio. Vous apprendrez tout ce que vous devez savoir sur SQLite en créant une application Android dans Android Studio.

Ce cours a été développé par Shad Sluiter. Shad est professeur d'informatique à la Grand Canyon University. Il enseigne la technologie depuis plus de 20 ans.

Ce cours montre chaque étape de la programmation d'une application Android de démonstration qui utilise une base de données locale SQLite.

Vous apprendrez comment :

* créer une base de données locale SQLite sur l'appareil Android,
* configurer de nouvelles tables avec la méthode onCreate,
* configurer les colonnes et les types de données,
* étendre la classe SQLiteHelper,
* créer des méthodes CRUD dans une classe DAO (Data Access Object),
* afficher les résultats dans une listview,
* et plus encore.

Regardez le cours complet sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=312RhjfetP8) (2 heures de visionnage).
