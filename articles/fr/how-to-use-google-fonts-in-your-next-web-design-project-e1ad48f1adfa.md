Par Hayden Mills

Vous avez probablement entendu dire que Google propose des centaines de polices web gratuites prêtes à être utilisées pour votre prochain projet. Mais comment les utiliser sur votre site web ? Cet article vous guidera à travers les étapes à suivre pour les intégrer à votre site web. Cela devrait prendre moins de 10 minutes !

#### Qu'est-ce que Google Fonts ?

> « Google Fonts [a été lancé pour la première fois en 2010](https://googlecode.blogspot.com/2010/05/introducing-google-font-api-google-font.html) en tant qu'initiative technique pour faire avancer le web et le rendre plus rapide. » — [Google Design](https://design.google/library/reimagining-google-fonts/)

Google Fonts a été lancé en 2010, devenant rapidement la plus grande sélection gratuite et open-source de polices sur Internet. Toutes les polices Google Fonts sont gratuites pour un usage commercial et personnel. Le site web de Google Fonts permet à chacun de sélectionner et d'utiliser rapidement différentes polices pour ses propres besoins de conception.

#### Qui utilise Google Fonts ?

Tout le monde ! Graphistes, designers UX, chercheurs, développeurs, designers web, blogueurs, gestionnaires de réseaux sociaux, entrepreneurs, artistes, étudiants, enseignants, photographes, et bien d'autres. J'ai vu des polices Google Fonts utilisées sur des panneaux publicitaires, des affiches, des présentations, des faire-part de mariage, des sites web et des livres.

![Image](https://cdn-media-1.freecodecamp.org/images/kXD2sxTadamOSqShbdvGWdoiVGhDl5K0BXMi)
_Tout le monde et leur mère utilisent Google Fonts_

#### Qui crée les polices ?

Google Fonts collabore avec des designers de polices, des fonderies typographiques et la communauté du design du monde entier. Ces personnes et organisations conçoivent les polices que vous voyez sur Google Fonts. Par exemple, [Łukasz Dziedzic](http://www.latofonts.com/) est un designer de polices indépendant qui a conçu la police populaire [Lato](https://fonts.google.com/specimen/Lato). Une fonderie typographique nommée [ParaType](http://www.paratype.com/) a conçu la police [PT Serif](https://fonts.google.com/specimen/PT+Serif).

#### Pourquoi Google se soucie-t-il des polices ?

Chaque lien vers l'API Google Fonts sur un site web donne à Google l'opportunité de faire plus de collecte de données. Cela semble déplaisant ? Oui, mais je ne pense pas que cela devrait vous empêcher d'utiliser Google Fonts.

Une autre raison plus rassurante pour laquelle Google se soucie des polices est qu'ils veulent rendre les polices web open-source accessibles au grand public. Ils veulent créer un web plus rapide, plus beau et plus facile à naviguer et à utiliser. Ils ont également lancé Google Fonts en 2010, à une époque où un service comme celui-ci était très nécessaire, résolvant ainsi un autre problème que nous avions sur Internet. #dontbeevil

#### Pourquoi devrais-je envisager de les utiliser ?

On ne peut pas battre gratuit. N'ai-je pas raison ? En plus d'être gratuites, vous devriez envisager d'utiliser Google Fonts dans votre prochain projet de conception web parce que...

* Les polices sont faciles à implémenter sur votre site web
* Il y a plus de 850 polices parmi lesquelles choisir (et ce n'est pas fini)
* La qualité globale des polices continue de s'améliorer
* Les polices Google Fonts sont également téléchargeables pour une utilisation en impression

D'accord, d'accord, d'accord ! Passons maintenant à la manière de les utiliser.

### Comment utiliser Google Fonts — Étape par étape

#### 1) Allez sur le [site web de Google Fonts](https://fonts.google.com/)

Visitez le site web de Google Fonts où vous pourrez choisir parmi 853 familles de polices et plus ! Le site vous permet de filtrer les résultats par style de police, langues, popularité et poids. Google Fonts présente également de nouvelles polices si vous visitez la page [« en vedette »](https://fonts.google.com/featured).

![Image](https://cdn-media-1.freecodecamp.org/images/nbfKNbQr2w6SzVWBAyKhFMrZqO3QC51AKDvd)
_[fonts.google.com](https://fonts.google.com/" rel="noopener" target="_blank" title=")_

#### 2) Recherchez les polices que vous aimez

Si vous avez un nom de famille de polices en tête, vous pouvez le rechercher.

![Image](https://cdn-media-1.freecodecamp.org/images/L27yKqethigRjO14Puq6HCrNQGaRdj2vcqUM)

#### 3) Cliquez sur « + » lorsque vous êtes prêt à l'utiliser

Chaque famille de polices a un bouton plus dans le coin supérieur droit. Cliquez sur le bouton plus pour ajouter la famille de polices au « tiroir de sélection » qui apparaît en bas de l'écran.

![Image](https://cdn-media-1.freecodecamp.org/images/EMm412pU1mVXOE3r4b1JelMJU6YKne8kK4jS)

#### 4) Répétez les étapes 1–3 si vous cherchez plus d'une police à utiliser

#### 5) Cliquez sur le « tiroir de sélection » pour afficher toutes les polices que vous avez choisies

Le « tiroir de sélection » est l'endroit où vous verrez toutes les polices que vous avez ajoutées. C'est là que vous trouverez le code à ajouter à votre site web. Vous avez également la possibilité de télécharger les polices sur votre bureau.

![Image](https://cdn-media-1.freecodecamp.org/images/SUC-QsEZKrHmUvO-AFtQn6rXT3xvbiGECwNa)

#### 6) Le « tiroir de sélection » est l'endroit où vous récupérerez le code que vous copierez/collerez dans le code HTML et CSS de votre site web

À partir du « tiroir de sélection », vous copierez le code à coller dans votre HTML et CSS. Pour que Google Fonts fonctionne sur votre site web, vous devez avoir à la fois la famille de polices liée à l'API Google Fonts (dans le HTML) et vous devez avoir la famille de polices spécifiée (dans le CSS).

![Image](https://cdn-media-1.freecodecamp.org/images/hjNGEz9NTW6dU8-8X9CwOvFAZcQRMakSNETO)

#### 7) Ajoutez différents poids et styles de police si vous le souhaitez en cliquant sur l'onglet « personnaliser »

C'est votre dernière chance d'ajouter différents poids et styles à vos familles de polices avant de les ajouter à votre site web. N'ajoutez que les poids et styles de police dont vous savez que vous aurez besoin, car plus vous ajoutez de polices, plus elles mettront de temps à se charger sur votre site.

![Image](https://cdn-media-1.freecodecamp.org/images/Igaa9YS1Q-6Bjxf7O5O4duLWibXYB-dCSH6N)

#### 8) Copiez le code HTML et collez-le dans la balise <head> de votre site web

Nous y sommes presque ! Tout ce que nous devons faire maintenant est de copier et coller le code HTML et CSS dans votre site web. Tout d'abord, copiez le lien HTML dans l'en-tête de votre document HTML.

![Image](https://cdn-media-1.freecodecamp.org/images/1byguIwDXOsgpWt-EpWqwy9D3UH8olcMaaIB)

![Image](https://cdn-media-1.freecodecamp.org/images/5uHnAL8reL3ug7q3JN-sxzLqxAW-7c5T2J2W)

#### 9) Copiez la règle CSS et collez-la dans le fichier CSS de votre site web

Enfin, nous devons également copier et coller les règles CSS dans votre fichier CSS. Si vous ajoutez une seule police Google pour votre site web, vous pouvez ajouter les règles de famille de polices CSS dans votre balise body. Sinon, vous devrez vous assurer de l'ajouter aux bons éléments dans vos fichiers CSS.

![Image](https://cdn-media-1.freecodecamp.org/images/3ZoUNz5Dh8l0yUDpTiKBKwonWSVz8BrIYo9T)

![Image](https://cdn-media-1.freecodecamp.org/images/ZyenjIqIONRL6U5ydFHML-Kal82YeBnnF03F)

#### 10) Enregistrez les deux fichiers (HTML et CSS), puis actualisez votre site web et voyez la nouvelle police

N'oubliez pas d'enregistrer et d'actualiser !

![Image](https://cdn-media-1.freecodecamp.org/images/oYCFHWTVYfgM9iltxKzMKqQD4KVg7s84BaiB)
_Actualisez vos fichiers locaux et vous devriez voir la nouvelle famille de polices... dans ce cas, il s'agit d'une police Google appelée [Lato](https://fonts.google.com/specimen/Lato" rel="noopener" target="_blank" title=")._

Voilà ! Vous avez réussi à implémenter votre première police Google sur votre site web. Félicitations, vous, le codeur. Il est juste de dire que Google Fonts sont essentiellement...
