Par javinpaul

Après l'introduction du cycle de publication de six mois sur Java 10, il est très difficile de suivre les dernières modifications de chaque version de Java. Mais si vous cherchez des ressources gratuites sur Java, comme des livres, des tutoriels et d'autres matériaux d'apprentissage, alors vous êtes au bon endroit.

Il existe de nombreuses ressources disponibles pour maîtriser les fonctionnalités introduites dans [Java 8](https://www.oracle.com/technetwork/java/javase/overview/java8-2100321.html) et [Java 9](https://www.oracle.com/java/java9.html).

Par le passé, j'ai partagé certains des [meilleurs tutoriels Java 8](http://www.java67.com/2014/09/top-10-java-8-tutorials-best-of-lot.html) et [livres](https://javarevisited.blogspot.com/2018/07/java-8-tutorials-resources-and-examples-lambda-expression-stream-api-functional-interfaces.html). Dans cet article, je vais partager certains des meilleurs cours gratuits sur Java 8 et Java 9 de [Udemy](https://www.udemy.com/) et [Pluarlsight](https://www.pluralsight.com/).

Ces cours sont assez similaires à tout cours payant que vous pourriez normalement acheter et beaucoup de ceux-ci sont rendus gratuits à des fins promotionnelles ou éducatives. Vous pouvez rejoindre ces cours pour apprendre les fonctionnalités de JDK 8 et JDK 9 en seulement quelques jours.

J'avais l'habitude de commencer à apprendre de nouvelles fonctionnalités de programmation avec des livres, mais de nos jours, les cours en ligne sont ma méthode préférée d'apprentissage. Il existe de nombreux bons cours gratuits disponibles.

Cependant, parfois, j'ai remarqué que les cours gratuits deviennent payants, surtout après que l'instructeur ait atteint ses objectifs promotionnels, donc vous devriez vérifier le prix avant de rejoindre ces cours.

Une bonne idée est de rejoindre les cours maintenant, pendant qu'ils sont gratuits. Une fois inscrit au cours, vous aurez un accès gratuit et illimité, même après qu'il soit devenu un cours payant. Cela signifie que vous pouvez apprendre quand vous le souhaitez.

Même si vous n'apprenez pas maintenant, vous pouvez en apprendre davantage lorsque vous aurez du temps libre ou que vos priorités changeront — il n'y a aucun mal à s'inscrire.

### Pourquoi devriez-vous apprendre Java 8 et Java 9 ?

Si vous vous demandez quel est l'avantage d'apprendre les fonctionnalités de Java 8 et Java 9, alors laissez-moi vous le dire. Si vous voulez être pertinent en tant que développeur Java sur le marché du travail d'aujourd'hui, vous devriez apprendre Java 8 maintenant.

J'en ai déjà parlé dans mon article [10 choses que les développeurs Java devraient apprendre en 2018](https://javarevisited.blogspot.com/2017/12/10-things-java-programmers-should-learn.html#axzz53ENLS1RB), et je le répète : apprenez Java 8 le plus tôt possible.

Cela fait plus de quatre ans que Java 8 a été publié pour la première fois, et il a été très bien accueilli par la communauté Java. Il comportait plusieurs fonctionnalités et améliorations de langage intéressantes, comme les [expressions lambda](https://javarevisited.blogspot.sg/2014/02/10-example-of-lambda-expressions-in-java8.html), les [API Stream](http://www.java67.com/2014/04/java-8-stream-examples-and-tutorial.html) pour les opérations en masse, les [nouvelles API de Date et d'Heure](http://javarevisited.blogspot.sg/2015/03/20-examples-of-date-and-time-api-from-Java8.html) pour une meilleure gestion des dates, les méthodes [Default](http://www.java67.com/2017/08/java-8-default-methods-on-interface-example.html) et Static sur l'interface, les [références de méthode](https://javarevisited.blogspot.com/2017/03/what-is-method-references-in-java-8-example.html), et bien d'autres.

De nos jours, de plus en plus d'entreprises adoptent le style de codage Java 8. Si vous ne savez pas écrire du code dans le style Java 8, comme l'utilisation des concepts de [lambda](https://en.wikipedia.org/wiki/Lambda_expression) et de [programmation fonctionnelle](https://en.wikipedia.org/wiki/Functional_programming), vous risquez d'être laissé pour compte.

Non seulement Java 8 est essentiel pour rester pertinent dans la communauté Java, mais il améliore également votre productivité et rend l'écriture de code Java amusante, une fois que vous maîtrisez les bases. C'est là que ces cours peuvent aider !

En ce qui concerne Java 9, il n'a pas eu le même impact que Java 8. Mais il était tout de même rempli de certaines [fonctionnalités intéressantes](http://www.java67.com/2018/01/top-10-java-9-tutorials-and-courses.html), comme les modules et certaines améliorations d'API. Connaître ces fonctionnalités vous aidera, à terme, à écrire un meilleur code Java et à créer une meilleure application en Java.

### Cours pour apprendre Java 8 et Java 9

Voici ma liste des cours en ligne gratuits pour apprendre Java 8 et Java 9. Cette liste comprend des cours adaptés aux débutants ainsi qu'aux programmeurs Java expérimentés qui ne sont pas familiers avec les [fonctionnalités de JDK 8](http://www.java67.com/2014/09/top-10-java-8-tutorials-best-of-lot.html) et [JDK 9](http://www.java67.com/2018/01/top-10-java-9-tutorials-and-courses.html).

Vous n'avez pas besoin de suivre tous les cours. Vous pouvez regarder un aperçu et décider si vous souhaitez vous inscrire ou non. Au minimum, vous pouvez suivre un cours sur Java 8 et un autre sur Java 9 pour vous mettre à niveau.

Et, comme ces cours sont absolument gratuits, vous n'avez rien à perdre en vous inscrivant.

#### Programmation Java 9 pour les débutants complets en 250 étapes

C'est un excellent [cours Java pour débutants](https://www.udemy.com/java-programming-tutorial-for-beginners/), surtout pour ceux qui commencent pour la première fois. Ce cours couvre la dernière version de Java, ou Java 9.

Si vous voulez apprendre Java à partir de zéro, vous pouvez rejoindre ce cours. Vous apprendrez non seulement les nouvelles fonctionnalités de Java 8 et Java 9, comme la programmation fonctionnelle, les lambdas et les streams, mais aussi d'autres concepts importants de Java.

Vous devez agir rapidement pour obtenir ce cours gratuitement car je suis sûr que l'instructeur Ranga Karanam rendra ce cours payant très bientôt, une fois qu'il aura atteint son objectif promotionnel avec plus de 31K étudiants qui ont déjà rejoint.

Quoi qu'il en soit, j'apprécie personnellement le style d'enseignement de Ranga et la manière dont il présente les concepts, comme JShell et les concepts Spring dans ses [cours sur le Framework Spring](https://javarevisited.blogspot.com/2018/06/top-6-spring-framework-online-courses-Java-programmers.html). Vous apprendrez beaucoup en peu de temps — il n'y a aucun doute là-dessus.

#### Java 8 : Les bases pour les débutants

C'est un [cours très court](https://www.udemy.com/java-8-basics-for-beginners-c/) pour apprendre certaines fonctionnalités clés de Java 8, y compris les expressions lambda, les références de méthode, l'[interface fonctionnelle](https://javarevisited.blogspot.sg/2018/01/what-is-functional-interface-in-java-8.html), les [méthodes par défaut](http://www.java67.com/2017/08/java-8-default-methods-faq-frequently-questions-answers.html), les [API Stream](https://javarevisited.blogspot.com/2018/05/java-8-filter-map-collect-stream-example.html), et la classe [Optional](https://javarevisited.blogspot.com/2017/04/10-examples-of-optional-in-java-8.html).

Vous apprendrez non seulement ces concepts, mais aussi la nouvelle API de Date et d'Heure et d'autres modifications moins populaires mais utiles.

Bien que ce cours ne fournisse pas un aperçu complet, vous pouvez le consulter pour un aperçu rapide, surtout si vous êtes pressé et que vous n'avez pas beaucoup de temps à consacrer à l'auto-apprentissage.

#### Qu'y a-t-il de nouveau dans Java 8 ?

C'est un [cours](https://pluralsight.pxf.io/c/1193463/424552/7490?u=https%3A%2F%2Fwww.pluralsight.com%2Fcourses%2Fjava-8-whats-new) plus complet sur Java 8, et il couvre presque toutes les fonctionnalités que vous devez connaître.

L'auteur, Jose Paumard, a 20 ans d'expérience en programmation informatique et cela se voit dans ce cours. Vous apprendrez comment les [expressions lambda](https://javarevisited.blogspot.com/2015/01/how-to-use-lambda-expression-in-place-anonymous-class-java8.html#axzz5HKrDEIos) et l'[API Stream](https://javarevisited.blogspot.com/2014/03/2-examples-of-streams-with-Java8-collections.html#axzz5SDm16SbN) rendent la programmation en Java amusante à nouveau.

Il ne couvre pas seulement les principales fonctionnalités, comme l'API de Date et d'Heure, mais aussi les petites améliorations d'API, comme `[StringJoiner](https://javarevisited.blogspot.com/2016/06/10-examples-of-joining-string-in-java-8.html)` et la méthode `[join()](https://javarevisited.blogspot.com/2016/06/10-examples-of-joining-string-in-java-8.html)` dans la classe String pour joindre des chaînes en Java 8.

Vous apprendrez également à propos de [JavaFX](https://en.wikipedia.org/wiki/JavaFX) et de la programmation JavaScript en utilisant le moteur [Nashorn](https://en.wikipedia.org/wiki/Nashorn_(JavaScript_engine)). Dans l'ensemble, c'est un excellent cours pour apprendre Java 8, mais le cours, malheureusement, n'est pas exactement gratuit.

#### Qu'y a-t-il de nouveau dans Java 9 ?

C'est un cours formidable pour apprendre les [fonctionnalités de Java 9](https://www.pluralsight.com/courses/java-9-whats-new?). Dans ce cours, Sander Mak, auteur du livre populaire Java 9 Modularity, a expliqué certains changements importants de Java 9.

Le cours est divisé en 6 sections. La première section couvre la modularité de Java, et comme l'instructeur est également l'auteur du livre [Java Modularity](https://javamodularity.com/), vous pouvez vous attendre à ce que ce soit la couverture la plus complète que vous puissiez obtenir.

La deuxième section se concentre sur un changement plus intéressant et interactif de Java 9 — JShell. Vous apprendrez à propos de REPL et comment utiliser JShell.

La troisième section se concentre sur les petites améliorations de langage et de bibliothèque, comme les changements dans l'[API Stream](http://www.java67.com/2016/08/java-8-stream-filter-method-example.html), [Optional](http://www.java67.com/2018/06/java-8-optional-example-ispresent-orElse-get.html), les [méthodes de fabrication sur les Collections](https://javarevisited.blogspot.com/2018/02/java-9-example-factory-methods-for-collections-immutable-list-set-map.html), et d'autres améliorations. C'est la section que vous allez adorer car vous utiliserez ces changements dans votre vie quotidienne.

![Image](https://cdn-media-1.freecodecamp.org/images/2hSREmri2JEt3WGpMrD5-KIPwiCL9C79XsFZ)
_[Qu'y a-t-il de nouveau dans Java 9 ?](https://www.pluralsight.com/courses/java-9-whats-new?" rel="noopener" target="_blank" title=")_

La quatrième section se concentre sur les changements de l'API HTTP/2 et Process, tandis que la cinquième section se concentre sur les améliorations de Java Desktop, comme les mises à jour de JavaFX.

La sixième et dernière section se concentre sur les améliorations de performance et de sécurité. Elle explique les changements de la collecte des déchets G1 et la fonction de chaîne compacte introduite dans Java 9.

Comme pour tout autre cours Pluralsight, vous avez besoin d'un abonnement pour rejoindre ce cours, soit un abonnement mensuel ou annuel. Bien que l'abonnement ne coûte pas cher (35 $ par mois pour accéder à leurs 5000+ cours), vous pouvez toujours obtenir ce cours et le cours Java 8 gratuitement en vous inscrivant pour un [essai gratuit de 10 jours](http://pluralsight.pxf.io/c/1193463/424552/7490?u=https%3A%2F%2Fwww.pluralsight.com%2Flearn) ce qui est plus que suffisant pour compléter ces cours.
