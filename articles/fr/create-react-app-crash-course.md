Create React App est un outil qui vous fait gagner beaucoup de temps lors de la création d'applications React. Il facilite la création d'applications React pour les débutants et les développeurs expérimentés en automatisant la configuration et la mise en place chronophages. Vous exécutez simplement une commande et Create React App configure les outils dont vous avez besoin pour démarrer votre projet React.

Dans ce cours accéléré d'Ipenywis, vous apprendrez tout ce que vous devez savoir pour commencer à utiliser Create React App pour vos propres projets React.

Vous pouvez regarder le tutoriel sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=999E9t4v9So) (22 minutes de visionnage).
