Unreal Engine est l'un des outils les plus populaires pour créer des jeux 3D et 2D. Le système de script visuel Blueprint pour le gameplay rend la création de jeux encore plus facile.

Nous avons publié un cours gratuit sur la chaîne YouTube freeCodeCamp.org qui vous montrera comment créer un jeu de plateforme 2.5D avec Unreal Engine et Blueprints. Le cours a été développé par Awesome Tuts.

Vous apprendrez à faire bouger une balle, collecter des points d'énergie, perdre de la santé, interagir avec l'environnement et les objets, et plus encore. Le jeu servira de modèle que vous pourrez développer et auquel vous pourrez ajouter vos propres fonctionnalités.

Des actifs 3D vous sont fournis pour être utilisés dans le jeu. Ces actifs peuvent être utilisés pour ce jeu et pour d'autres jeux que vous créez.

Ceci est un excellent tutoriel général si vous souhaitez améliorer vos compétences en développement de jeux.

Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=9Fd9hw329fY) (3 heures de visionnage).
