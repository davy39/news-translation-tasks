Vous avez peut-être entendu parler de about:blank mais ne comprenez pas ce que c'est. Ou vous l'avez vu apparaître lorsque vous chargez une nouvelle session dans le navigateur Chrome – juste une page blanche vide avec about:blank affiché dans votre barre d'adresse.

Vous pouvez même le voir lorsque vous cliquez sur un lien pour télécharger quelque chose depuis le web ou lorsque vous cliquez sur un lien pour accéder à une autre page web.

Vous vous êtes demandé pourquoi about:blank apparaît et ce que c'est exactement ? Découvrons-le.

## Devriez-vous vous inquiéter de about:blank ?

Donc, about:blank est-il simplement une page blanche comme son nom l'indique, ou y a-t-il plus que ce que l'on voit ?

![Screenshot-2021-07-21-at-1.43.36-PM](https://www.freecodecamp.org/news/content/images/2021/07/Screenshot-2021-07-21-at-1.43.36-PM.png)

A-t-il des avantages, ou est-ce une fonctionnalité utile ? Devriez-vous le garder et l'utiliser, ou devriez-vous vous en débarrasser ?

Lorsque about:blank apparaît, cela peut être déroutant pour les gens et les inquiéter. Ils voient une page blanche et pensent qu'il s'agit d'un virus – un malware dangereux – ou que cela signifie que quelque chose est arrivé à leur navigateur. Ils ne comprennent pas ce que c'est et ce qui le cause.

Cet article répond aux questions ci-dessus. Il explique pourquoi cette page apparaît dans le champ d'URL de votre navigateur, pourquoi vous pourriez vouloir la garder, et si ce n'est pas le cas, comment la faire disparaître et l'empêcher de s'afficher.

## Pourquoi votre navigateur affiche-t-il about:blank ?

Si vous voyez une page vide avec about:blank comme adresse, vous consultez effectivement une page vide intégrée à Chrome. En fait, cela est courant dans la plupart des navigateurs web modernes comme Mozilla Firefox, Apple Safari, Microsoft Edge, Internet Explorer et autres.

Cette page est chargée par défaut lorsque le navigateur n'a rien d'autre à afficher.

Lorsque l'utilisateur n'a pas spécifié de page web particulière à charger lorsqu'il ouvre le navigateur pour la première fois, le navigateur lui-même ne sait pas quoi afficher. Il choisit donc d'afficher une page blanche spéciale qui est intégrée.

Le navigateur doit toujours afficher quelque chose après tout, même si c'est vide.

### Est-ce que about:blank est un virus ?

En soi, about:blank n'est pas un malware. Il est aussi inoffensif que possible.

En général, vous n'avez pas à vous en soucier.

Dans certains cas très rares, about:blank peut être un *symptôme* d'un virus.

Il peut indiquer que votre ordinateur a détecté quelque chose de dangereux et bloque les virus/malware lorsqu'il rencontre un mauvais lien/URL.

Il peut également apparaître après avoir supprimé un malware. Après un tel processus, le navigateur ne saura pas quelle action effectuer.

Dans tous les cas, si vous pensez que votre ordinateur a un virus, il est bon de faire une analyse avec un programme antimalware de votre choix.

## Qu'est-ce que about:blank ?

Comme mentionné précédemment, il s'agit simplement d'une page intégrée à votre navigateur.

Ce n'est pas une page web, c'est-à-dire une page provenant d'Internet. Ce n'est rien de plus qu'un écran vide apparaissant dans votre navigateur.

Plus précisément, il fait partie du "about" URI Scheme qui est implémenté par une variété de navigateurs différents.

Le "about" URI Scheme fait partie d'une convention de nommage qui indique au navigateur d'afficher ses pages internes intégrées.

Outre about:blank, il existe une longue liste d'URL Chrome, dont certaines sont listées ci-dessous :

- about:downloads pour voir la liste des téléchargements de Chrome
- about:bookmarks pour voir vos signets enregistrés
- about:accessibility montre les informations d'accessibilité pour chaque onglet ouvert dans le navigateur
- about:apps pour voir les applications qui accompagnent le navigateur Chrome et celles installées par l'utilisateur

## Avantages de about:blank

Cette page est un choix de page d'accueil préféré pour de nombreux utilisateurs. Elle garantit que leur navigateur se charge avec une page blanche vide lorsqu'il s'ouvre pour la première fois.

Cela en fait une page d'accueil extrêmement rapide, car elle démarre le navigateur très rapidement. Le navigateur ne perd pas de temps à faire une requête pour une page sur Internet et à attendre la réponse.

Ainsi, cela peut aider à réduire votre consommation de données si votre connexion Internet est lente et que vous avez une bande passante limitée. Vous pouvez économiser des ressources informatiques et réseau.

Une page blanche par défaut peut également vous aider à rester concentré et à ne pas être distrait. Vous pouvez simplement vous concentrer sur la recherche et la visite de la page que vous souhaitez consulter.

## Comment définir about:blank comme page d'accueil

Si vous souhaitez faire de about:blank votre page de démarrage par défaut, vous devrez configurer les paramètres de votre navigateur.

Allez dans le menu des paramètres et sélectionnez "Paramètres" :

![Screenshot-2021-07-21-at-2.00.05-PM](https://www.freecodecamp.org/news/content/images/2021/07/Screenshot-2021-07-21-at-2.00.05-PM.png)

(vous auriez même pu taper about:settings !)

- Dans la barre de recherche, tapez "démarrage".
- Sélectionnez "Ouvrir une page spécifique ou un ensemble de pages".
- Cliquez sur "Ajouter une nouvelle page" et tapez "about:blank"
![Screenshot-2021-07-21-at-2.06.29-PM](https://www.freecodecamp.org/news/content/images/2021/07/Screenshot-2021-07-21-at-2.06.29-PM.png)
- Enfin, appuyez sur "Ajouter"

Lorsque vous ouvrez un nouvel onglet, vous verrez une page vide.

## Comment empêcher about:blank de s'afficher

Cette page n'est pas quelque chose dont vous pouvez simplement "vous débarrasser", car il s'agit d'une fonctionnalité intégrée à votre navigateur et elle sera toujours là, en arrière-plan.

Si votre navigateur s'ouvre toujours avec une page blanche et que cela ne vous plaît pas, vous pouvez empêcher cela. Il est probablement défini comme page d'accueil par défaut et vous devez changer vos paramètres.

En suivant les étapes mentionnées dans la section précédente, supprimez "about:blank" de l'option "Ouvrir une page spécifique ou un ensemble de pages" et sélectionnez votre page web favorite. Ou vous pouvez sélectionner "Ouvrir la page Nouveau onglet" à la place.

## Conclusion

Dans l'ensemble, about:blank est une fonctionnalité sûre et ne devrait pas susciter d'inquiétudes.

Et si vous ne souhaitez pas le voir lorsque vous ouvrez votre navigateur, vous pouvez simplement le changer dans vos paramètres.
