Swift est un langage de programmation populaire souvent utilisé pour créer des applications iOS. 

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra le langage de programmation Swift.

Vandad Nahavandipoor enseigne ce cours. Il travaille actuellement comme développeur iOS principal et a créé de nombreux autres cours populaires.

Dans ce cours, vous apprendrez tous les aspects modernes de Swift en tant que langage de programmation, y compris les variables, les constantes, les fonctions, les structures, les classes, les protocoles, les extensions, la programmation asynchrone, les génériques et bien plus encore. Ce cours posera les bases de l'apprentissage de Swift pour ceux qui ne sont pas déjà familiers avec Swift.

Voici les sections de ce cours :

* Variables
* Opérateurs
* If et else
* Fonctions
* Closures
* Structures
* Énumérations
* Classes
* Protocoles
* Extensions
* Génériques
* Optionnels
* Gestion des erreurs
* Collections
* Égalité et hachage
* Opérateurs personnalisés
* Programmation asynchrone

Regardez le cours complet [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/8Xg7E9shq0U) (7 heures de visionnage).
