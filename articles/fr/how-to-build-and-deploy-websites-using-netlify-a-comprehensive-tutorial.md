Netlify aide les développeurs à déployer rapidement des sites web statiques. Apprenez à utiliser Netlify dans ce cours vidéo complet pour débutants créé par swyx.

Netlify peut être utilisé pour tout ce dont vous avez besoin dans le développement web moderne, de la configuration locale au déploiement mondial. Ce cours est le point de départ idéal pour quiconque souhaite apprendre Netlify.

Le cours couvre les sujets suivants :

* Netlify Edge
* Netlify Build
* Netlify Dev
* Netlify Forms
* Netlify Identity
* NetlifyCMS
* Netlify Addons : Tout le reste !
* Netlify API
* Où apprendre davantage

Vous pouvez regarder la vidéo complète ci-dessous ou sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=mT5siI19gtc) (3,5 heures de visionnage).
