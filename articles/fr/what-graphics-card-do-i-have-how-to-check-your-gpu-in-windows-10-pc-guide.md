Chaque ordinateur possède une carte graphique qui gère le rendu de chaque pixel que vous voyez sur le moniteur. Cette carte graphique joue également un rôle crucial dans l'affichage des vidéos, l'exécution des applications et, surtout, la gestion des graphismes des jeux vidéo. 

Les ordinateurs possèdent également une unité de traitement graphique (GPU), qui rend les images et les vidéos. Votre ordinateur dispose d'un GPU d'Intel, d'AMD ou de NVIDIA. 

Si votre ordinateur n'a pas de carte graphique avec un GPU puissant, vous pourriez être incapable de jouer aux derniers jeux, d'exécuter des infographies et d'utiliser des applications gourmandes en vidéo. Et certaines applications pourraient ne pas s'afficher correctement.

Dans ce guide, je vais vous montrer 3 façons de vérifier votre GPU dans Windows 10, afin que vous sachiez ce que vous avez et si vous devez le mettre à niveau.

## Comment vérifier le GPU avec l'outil de diagnostic DirectX

**Étape 1** : Cliquez sur le menu Démarrer, recherchez « dxdiag » et ouvrez-le.
![ss-1-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-1-1.png)

**Étape 2** : Il se peut que vous soyez invité à sélectionner « Oui ». Assurez-vous de sélectionner « Oui » pour continuer.

**Étape 3** : Cliquez sur l'onglet Affichage et vous verrez le fabricant, le modèle et autres informations de votre GPU :
![ss-2-3](https://www.freecodecamp.org/news/content/images/2021/10/ss-2-3.jpg)

## Comment vérifier le GPU via le Gestionnaire de périphériques

**Étape 1** : Pour vérifier le GPU de votre ordinateur Windows 10 dans le Gestionnaire de périphériques, cliquez sur Démarrer ou sur l'icône de recherche et recherchez « gestionnaire de périphériques ». Ensuite, cliquez sur Gestionnaire de périphériques ou appuyez sur Entrée.
![ss-3-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-3-2.png)

**Étape 2** : Développez la branche Adaptateurs d'affichage pour voir le modèle et le fabricant du GPU de votre ordinateur :
![ss-4-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-4-2.jpg)

## Comment vérifier le GPU via les Paramètres

**Étape 1** : Pour vérifier le GPU de votre ordinateur dans les Paramètres, cliquez sur Démarrer et cliquez sur l'icône d'engrenage pour ouvrir les paramètres.
![ss-5-3](https://www.freecodecamp.org/news/content/images/2021/10/ss-5-3.png)

**Étape 2** : Sélectionnez « Système » dans la liste que vous voyez :
![ss-6-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-6-1.jpg)

**Étape 3** : Assurez-vous que « Affichage » est sélectionné. Bien qu'il soit toujours le premier dans la liste.

**Étape 4** : Sous « Plusieurs affichages », cliquez sur « Paramètres d'affichage avancés » :
![ss-7-1](https://www.freecodecamp.org/news/content/images/2021/10/ss-7-1.jpg)

**Étape 5** : Vous devriez voir les informations du GPU de l'affichage par défaut immédiatement, y compris le modèle et le fabricant (Intel, AMD, NVIDIA).

Si vous avez plusieurs affichages installés, sélectionnez celui pour lequel vous souhaitez voir les informations du GPU. 
![ss-8](https://www.freecodecamp.org/news/content/images/2021/10/ss-8.jpg)
