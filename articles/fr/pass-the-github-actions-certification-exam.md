GitHub a récemment lancé un examen de certification GitHub Actions. Et nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous préparera à passer cet examen. Ce cours est idéal pour les développeurs, les professionnels DevOps et toute personne intéressée par l'automatisation des flux de travail logiciels à l'aide de la puissante plateforme CI/CD de GitHub, GitHub Actions.

Andrew Brown d'ExamPro a créé ce cours. Il est un instructeur populaire qui a créé des cours de préparation pour une série d'examens de certification différents.

GitHub Actions facilite l'automatisation de tous vos flux de travail logiciels avec une CI/CD de classe mondiale. Construisez, testez et déployez votre code directement depuis GitHub. Ce cours vous aidera à comprendre comment vos clients peuvent utiliser GitHub Actions pour automatiser leurs flux de travail de développement logiciel.

### Aperçu du cours

Le cours est structuré en plusieurs sections détaillées, chacune se concentrant sur différents aspects de GitHub Actions, des bases aux techniques plus avancées. Voici ce à quoi vous pouvez vous attendre :

* **Introduction et analyse de l'examen** : Commencez par une introduction approfondie au cours et une analyse détaillée de l'examen de certification. Cette section pose les bases de ce que vous apprendrez et des compétences que vous développerez.

* **Bases de GitHub Actions** : Plongez dans les fondamentaux de GitHub Actions, y compris les workflows, les composants de workflow et la gestion de divers types d'événements tels que les événements planifiés, manuels et webhook. Cette section est cruciale pour construire une base solide.

* **Runners et commandes** : Apprenez les runners hébergés par GitHub et auto-hébergés, explorez comment exécuter des commandes de workflow et engagez-vous dans des labs pratiques pour appliquer vos connaissances.

* **Workflows avancés** : Acquérez des connaissances sur des configurations de workflow plus complexes, la gestion des secrets chiffrés, la définition des variables d'environnement et le scripting dans les workflows.

* **Publication et déploiement** : Découvrez comment utiliser les workflows pour publier des packages et déployer des releases dans différents environnements d'hébergement, y compris Docker Hub et GitHub Container Registry.

* **Optimisation et gestion** : Optimisez vos workflows grâce au caching, gérez les conteneurs de service et apprenez à configurer les matrices de jobs et les protections de workflow.

* **GitHub Actions avancées** : Approfondissez votre expertise avec des sujets avancés tels que les actions personnalisées, la réutilisation de templates et la configuration de runners auto-hébergés de niveau entreprise.

Ce cours gratuit couvre tout le contenu nécessaire défini par la certification officielle GitHub Actions. Le cours vous aidera à :

* **Améliorer vos compétences** : Apprenez à automatiser les tests, les builds et les déploiements pour rationaliser vos processus de développement logiciel.

* **Obtenir une reconnaissance** : Préparez-vous et passez l'examen de certification GitHub Actions pour ajouter une précieuse certification à votre CV.

* **Contribuer plus efficacement** : Implémentez des workflows efficaces dans vos projets ou sur votre lieu de travail, contribuant à une meilleure productivité d'équipe et à la réussite des projets.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/Tz7FsunBbfQ) (3 heures de visionnage).
