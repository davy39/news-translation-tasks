Next.js simplifie le développement d'applications React. Il fournit une structure commune qui vous permet de construire des applications frontend React, et gère de manière transparente le rendu côté serveur pour vous.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous enseignera les bases de Next.js.

Kapehe a développé le cours. Elle est spécialiste devrel chez [Sanity.io](http://sanity.io/) et c'est une excellente enseignante.

Dans ce cours, vous apprendrez à maîtriser Next.js. En construisant une application full-stack de recettes avec Next.js, Sanity.io et Vercel, vous apprendrez à configurer le routage dynamique, le pré-rendu, à récupérer du contenu depuis des APIs externes, à configurer des fonctions serverless, à offrir des aperçus de contenu en temps réel et à déployer votre application sur le web.

Voici les sections couvertes dans ce cours :

* Qu'est-ce que Next.js
* Outils : Node.js, VS Code et la ligne de commande
* Installation de Next.js
* Configuration du CMS Sanity Studio
* Connexion de Next.js avec votre lac de contenu
* Création d'une barre de navigation simple dans _app.js
* Création de votre premier modèle de page dans index.js
* Configuration des routes dynamiques avec [slug].js
* Plongez dans la récupération de données et le pré-rendu avec getStaticPaths et getStaticProps
* Créer un bouton like avec les routes API et les fonctions serverless
* Ajout d'un aperçu en temps réel avec le lac de contenu de Sanity.io
* Configuration du déploiement automatique avec GitHub et Vercel
* Résumé : Ce que vous avez appris et les prochaines étapes

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=1WmNXEVia8I) (3 heures de visionnage).
