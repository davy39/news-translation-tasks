React est l'une des bibliothèques JavaScript les plus populaires et elle est fréquemment utilisée pour créer des interfaces utilisateur. 

Nous venons de publier un cours de 9 heures qui vous apprendra à construire 15 projets différents en utilisant React. Il y a quelques semaines, nous avons publié un [cours de 10 heures sur les fondamentaux de React](https://www.youtube.com/watch?v=4UZrsTqkcW4) et ce nouveau cours basé sur des projets est le moyen idéal pour consolider les compétences que vous avez apprises dans le cours sur les fondamentaux de React.

Ce cours a été développé par John Smilga, qui anime la chaîne YouTube Coding Addict. Il crée des cours depuis de nombreuses années sur diverses plateformes.

Vous apprendrez à construire les projets suivants en React :

* Rappel d'anniversaire
* Page de visites
* Page de critiques
* Accordion
* Menu
* Portfolio à onglets
* Curseur
* Générateur de Lorem Ipsum
* Générateur de couleurs
* Liste de courses
* Barre de navigation
* Barre latérale et modèle
* Menu Stripe
* Panier d'achat
* Page de cocktails

Vous pouvez voir une version fonctionnelle de [tous les projets ici](https://react-projects.netlify.app/). De plus, le code pour tous les projets est fourni. 

En plus d'apprendre à créer les projets, vous apprendrez à les déployer et à les héberger.

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=a_7Z7C_JCyo) (9 heures de visionnage).
