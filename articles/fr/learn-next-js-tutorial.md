Next.js est un framework populaire pour construire des applications JavaScript rendues côté serveur.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous aidera à apprendre à utiliser Next.js.

Next.js est construit sur React, une bibliothèque JavaScript populaire pour construire des interfaces utilisateur. Next.js facilite la création d'applications web efficaces et évolutives, et il inclut des fonctionnalités telles que le fractionnement automatique du code et des performances optimisées dès la sortie de la boîte. Il est souvent utilisé pour construire des applications React rendues côté serveur, et c'est un choix populaire parmi les développeurs qui veulent construire des applications web performantes et évolutives rapidement et facilement.

Dans ce cours Next.js pour débutants par Alicia Rodriguez, vous apprendrez à construire une application full-stack à partir de zéro. Vous apprendrez également à déployer l'application en utilisant Vercel et à synchroniser GitHub avec Vercel pour déployer l'application automatiquement.

Voici les sections de ce cours :

* Qu'est-ce que Next.js ?
* Fonctionnalité principale : rendu côté serveur
* Créer un nouveau projet Next.js
* Analyser l'application finale que vous allez construire
* Structure des fichiers Next.js
* Pages Next.js et construction des pages
* Récupération de données
* Construire les composants - UI (interface utilisateur)
* Ajouter du CSS - Styles
* Routage API dans Next.js
* Déployer l'application sur Vercel
* Suggestions pour améliorer l'application
