De nombreux sites web ont des paniers d'achat et vous pouvez créer le vôtre en utilisant React et TypeScript.

Nous avons publié un tutoriel sur la chaîne YouTube de freeCodeCamp.org qui vous aidera à améliorer vos compétences en React et TypeScript. Vous apprendrez également à utiliser :

* Material UI
* Styled Components
* et React-Query pour les hooks React

L'ingénieur logiciel suédois Thomas Weibenfalk a créé ce tutoriel. Il a réalisé de nombreux excellents tutoriels sur sa propre chaîne YouTube ainsi que sur la chaîne freeCodeCamp.

Ce tutoriel utilise une API gratuite qui vous fournit des données d'exemple pour les articles de votre boutique. Vous utiliserez les hooks React-Query pour récupérer les données de l'API, et Styled Components avec Material UI pour personnaliser les styles.

Regardez le [tutoriel complet sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=sfmL6bGbiN8) (1 heure de visionnage).
