Python est un excellent langage de programmation à apprendre et vous pouvez l'utiliser dans divers domaines du développement logiciel. Vous pouvez utiliser Python pour le développement web, l'analyse de données, l'apprentissage automatique, l'intelligence artificielle, et bien plus encore. 

Avec la vaste quantité de ressources disponibles, il est parfois difficile de savoir quelles sont les meilleures options. 

J'ai compilé une liste de 16 tutoriels Python pour débutants. Cette liste inclura des ressources gratuites et payantes. 

Voici la liste complète. Une fois que vous trouvez une ressource qui vous intéresse, cliquez sur ce lien et il vous emmènera à une description plus détaillée de cette ressource. Sauf indication contraire, les ressources sont gratuites.

## Programmation pour Tous (Démarrer avec Python)

![Image](https://www.freecodecamp.org/news/content/images/2021/08/Screen-Shot-2021-08-06-at-4.13.09-AM.png)

Dans ce [cours de l'Université du Michigan](https://www.freecodecamp.org/news/python-for-everybody/), vous apprendrez les fonctions, les boucles, les conditionnelles, les variables et bien plus encore avec le célèbre "Dr. Chuck". 

Une fois que vous aurez appris les bases, vous pourrez continuer avec le reste de la [spécialisation](https://www.coursera.org/specializations/python) et suivre le [cours sur les Structures de Données en Python](https://www.coursera.org/learn/python-data?specialization=python), le [cours sur l'Utilisation de Python pour Accéder aux Données Web](https://www.coursera.org/learn/python-network-data?specialization=python), et le [cours sur l'Utilisation des Bases de Données avec Python](https://www.coursera.org/learn/python-databases?specialization=python).    

## Studytonight

![Image](https://www.freecodecamp.org/news/content/images/2021/08/Screen-Shot-2021-08-06-at-1.57.39-AM.png)

Dans [Studytonight](https://www.studytonight.com/python/), vous apprendrez les bases de Python, la gestion des erreurs, la POO, la gestion des fichiers, les types de données complexes, le Multithreading et bien plus encore. 

Une fois que vous aurez appris les bases, vous pourrez passer à leurs autres modules qui incluent, la [bibliothèque NumPy](https://www.studytonight.com/numpy), [Matplotlib](https://www.studytonight.com/matplotlib), [Tkinter](https://www.studytonight.com/tkinter), la [Programmation Réseau en Python](https://www.studytonight.com/network-programming-in-python/), et le [Web Scraping avec Beautiful Soup](https://www.studytonight.com/python/web-scraping/).   

## Python Core

Pour commencer, vous devrez créer un compte gratuit. 

## Real Python

![Image](https://www.freecodecamp.org/news/content/images/2021/08/Screen-Shot-2021-08-06-at-1.33.04-AM.png)

[Les Tutoriels Real Python](https://realpython.com/) offrent une variété de ressources pour vous aider à apprendre Python à travers des articles, des vidéos, des projets et des quiz. Vous pouvez commencer avec les [cours de base](https://realpython.com/search?level=basics&q=basics) sur les types de données, les boucles, les fonctions, les chaînes de caractères et les structures de données. 

Vous pouvez ensuite passer à la liste exhaustive de [projets](https://realpython.com/search?level=basics&q=projects) qui inclut la création d'un journal personnel, d'une application de portfolio, et d'un service de notification du prix du bitcoin. 

Pour accéder au contenu complet, vous devrez devenir un abonné [payant](https://realpython.com/account/join/).
