JavaScript a constamment évolué, repoussant toujours les limites de ce que les développeurs pensaient possible. Maintenant, sous les projecteurs, voici Bun, un environnement d'exécution révolutionnaire qui promet d'emporter les vitesses de JavaScript à des niveaux sans précédent.

Nous venons de publier un cours accéléré sur Bun sur la chaîne YouTube freeCodeCamp.org. Haider Malik a développé ce cours. Haider a enseigné de nombreux cours excellents sur Udemy et apporte maintenant son expertise à la chaîne YouTube freeCodeCamp.org.

Bun est une boîte à outils de pointe conçue pour supercharger vos applications JavaScript et TypeScript. Avec ses temps de démarrage ultra-rapides, son support intégré pour TypeScript et JSX, et une gamme de fonctionnalités puissantes conçues pour remplacer Node.js de manière transparente, Bun promet de révolutionner votre expérience de développement.

Que vous cherchiez à optimiser des fonctions serverless ou à améliorer votre flux de travail, ce cours vous donnera les moyens de tirer pleinement parti de Bun et de l'intégrer de manière transparente dans vos projets.

![Image](https://www.freecodecamp.org/news/content/images/2023/10/image-45.png)
_De nombreux développeurs sont enthousiastes à propos de Bun._

Voici les sujets abordés dans ce cours :

* Prise en main de Bun
* Création d'un serveur Web avec Bun
* Création de routes
* Gestion des erreurs
* Diffusion de fichiers texte
* Installation du serveur Web dans Elysia
* Route et gestionnaire
* État et décoration
* Plugin
* Groupe
* Schéma
* Validations des paramètres de chemin
* Construire un frontend avec Vite et Bun
* Construire un serveur Express avec Bun

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/eTB0UCDnMQo) (1 heure de visionnage).
