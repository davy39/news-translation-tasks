React est l'un des frameworks frontend JavaScript les plus populaires. Si vous voulez devenir développeur frontend, vous devrez probablement connaître React.

Nous venons de publier un cours complet sur React pour débutants sur la chaîne YouTube freeCodeCamp.org.

Ce cours est différent de la plupart des autres cours sur React disponibles aujourd'hui. 
Au lieu d'un seul enseignant, nous avons réuni une équipe d'instructeurs all-star pour enseigner ce cours. Ces trois instructeurs sont parmi les plus populaires sur notre chaîne et ont reçu bien plus de dix millions de vues sur d'autres tutoriels. Apprendre à partir de différentes perspectives peut vous aider à consolider votre compréhension des concepts clés de React.

Il s'agit d'un cours basé sur des projets, et il se concentre sur l'enseignement de tout ce que vous devez savoir pour commencer à créer vos propres applications React. Tout d'abord, Gavin Lon partira du tout début et vous enseignera les bases de React en construisant une simple application monopage pour suivre les membres de l'équipe. Gavin est développeur logiciel depuis plus de 20 ans et il est excellent pour transmettre la sagesse qu'il a acquise au fil des années.

Ensuite, vous apprendrez à créer une application de repas avec John Smilga. John est l'un des instructeurs les plus vus sur notre chaîne entière, vous êtes donc définitivement entre de bonnes mains pour ce deuxième projet. Il vous apprendra à récupérer des données depuis une API externe pour les utiliser dans cette application React.

Enfin, vous apprendrez à créer une simple application de trading d'actions avec Sanjeev Thiyagarajan. Vous utiliserez ce que vous avez appris jusqu'à présent et l'appliquerez à une application qui récupère des données depuis une API et stocke des données dans le localStorage. Parmi d'autres choses, Sanjeev a précédemment créé l'un des cours de développement d'API les plus populaires et complets au monde, il est donc la personne parfaite pour vous enseigner davantage sur l'utilisation des API avec React.

Et pour suivre ce cours, tout ce dont vous avez besoin est un navigateur web. Chaque projet de ce cours est développé en utilisant l'IDE en ligne Replit. Replit a fourni une subvention qui a rendu ce cours possible.

Voici les sections couvertes dans ce cours.
