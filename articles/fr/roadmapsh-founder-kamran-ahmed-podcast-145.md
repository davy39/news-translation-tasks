Dans l'épisode de cette semaine du podcast, j'interviewe Kamran Ahmed. Il est ingénieur logiciel et fondateur de [Roadmap.sh](http://Roadmap.sh), qui propose des roadmaps d'arbres de compétences pour de nombreux domaines de développement, tels que DevOps. En tant qu'enseignant, il est également un Google Developer Expert et une GitHub Star.

Nous parlons de :

* Les conseils de Kamran pour trouver les bons projets open source auxquels contribuer

* L'histoire derrière [Roadmap.sh](http://Roadmap.sh), son site web populaire pour les développeurs

* Autres projets open source spécialisés que Kamran a construits au fil des ans

* Comment Kamran est devenu un Google Developer Expert et une GitHub Star
