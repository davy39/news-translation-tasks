GitHub est largement connu comme l'un des dépôts de contrôle de version les plus célèbres. Vous pouvez y trouver d'innombrables projets publics dans de nombreux langages de programmation. Il vous permet de collaborer facilement avec les membres de votre équipe et d'autres développeurs à travers le monde. Ensemble, vous pouvez apprendre de ces projets et y contribuer.

#### Trouver et travailler sur des projets tendance

Entre autres choses, GitHub affiche les **projets tendance** (**Trending Projects**). Cela vous permet de savoir plus facilement quels projets ont reçu le plus d'attention. Vous pouvez trouver des projets qui ont reçu beaucoup d'étoiles de la part d'autres développeurs ces derniers temps.

Le processus de recherche de nouveaux projets tendance est devenu encore plus facile : il existe désormais une extension Chrome, appelée GitHunt, dédiée à augmenter votre exposition aux projets open source.

GitHunt peut être installé en tant qu'extension Chrome dans votre navigateur, vous donnant la possibilité de voir les projets les plus populaires sur GitHub directement dans vos **nouveaux onglets**. Vous pouvez voir les projets tendance sur une base quotidienne, hebdomadaire ou mensuelle.

Vous pouvez choisir d'afficher les projets tendance dans une seule langue ou dans toutes les langues. Mais il n'y a actuellement aucune option pour choisir plus d'une langue sans les inclure toutes.

Outre le titre du projet et le nombre d'étoiles, vous pouvez également consulter une brève description du projet (si elle a déjà été ajoutée) et le nombre de tickets (*issues*) ouverts. Vous pouvez également visiter ce projet en cliquant simplement sur le projet affiché dans le nouvel onglet.

### Pourquoi devriez-vous installer cette extension ?

GitHub a été [appelé le nouveau CV](http://code.dblock.org/2011/07/14/github-is-your-new-resume.html) pour les développeurs de logiciels. C'est un excellent moyen de promouvoir vos connaissances, vos compétences, votre expertise et votre style de codage auprès de recruteurs potentiels.

En conséquence, de nombreux développeurs publient certains de leurs meilleurs travaux sur GitHub. Cela permet à d'autres développeurs d'en apprendre davantage, d'y contribuer et de publier leurs propres innovations.

Malgré cela, il est parfois difficile de tomber sur les meilleurs projets. Vous n'avez peut-être pas l'habitude de consulter directement GitHub juste pour voir les projets tendance. Mais si vous les avez directement dans vos nouveaux onglets dans Chrome, vous augmenterez vos chances de découvrir de nouveaux projets intéressants qui peuvent vous aider.

Les grandes entreprises comme [Facebook, Google,](https://readwrite.com/2013/10/17/is-facebook-the-worlds-largest-open-source-company/) et [Microsoft](https://opensource.org/node/901) tirent tellement profit des projets open source. Laissez cela vous convaincre d'utiliser encore plus cette extension dans votre navigateur.

Vous pourriez trouver dans vos nouveaux onglets un projet simple qui a déjà résolu le problème qui vous tracasse depuis longtemps. Peut-être aviez-vous prévu de chercher une solution plausible sur Google, mais cette solution se trouve peut-être juste là.

Ces projets peuvent également vous inspirer de nouvelles connaissances, idées et modèles pour vous aider à construire vos prochains projets open source tendance.

GitHunt est également un projet open source que vous pouvez [trouver sur GitHub](https://github.com/kamranahmedse/githunt). Si vous avez des idées sur la façon de l'améliorer, l'[auteur](https://github.com/kamranahmedse) a déjà lancé un appel à contributions. J'ai déjà effectué quelques [petites corrections de fautes de frappe](https://github.com/kamranahmedse/githunt/pull/15) et je vous invite à continuer d'améliorer ce superbe projet.

Vous pouvez en savoir plus et voir le code source de cette extension en visitant sa [page GitHub](https://github.com/kamranahmedse/githunt). Vous pouvez la télécharger et l'installer via le [Chrome Webstore](https://chrome.google.com/webstore/detail/githunt/khpcnaokfebphakjgdgpinmglconplhp).
