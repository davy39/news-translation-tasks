## **À quoi sert Python ?**

Python peut facilement être utilisé pour des projets petits, grands, en ligne et hors ligne. Les meilleures options pour utiliser Python sont le développement web, le scripting simple et l'analyse de données. Voici quelques exemples de ce que Python vous permettra de faire :

##### **Développement Web :**

Vous pouvez utiliser Python pour créer des applications web de différents niveaux de complexité. Il existe de nombreux excellents frameworks web Python, dont Pyramid, Django et Flask, pour n'en nommer que quelques-uns.

##### **Analyse de données :**

Python est le langage de choix pour de nombreux scientifiques des données. Python a gagné en popularité dans ce domaine grâce à la disponibilité de nombreuses excellentes bibliothèques axées sur la science des données (dont NumPy et Pandas sont deux des plus connues) et la visualisation des données (comme Matplotlib et Seaborn). Python a vraiment rendu le traitement des données amusant avec toutes ses nombreuses bibliothèques disponibles. Ipython avec JupyterLab est une autre forme de Python qui améliore l'utilisation de Python dans le domaine de la science des données.

##### **Automatisation :**

Python est un langage très flexible qui peut être utilisé pour automatiser des tâches ennuyeuses ou répétitives. Les administrateurs système l'utilisent souvent en écrivant des scripts qui peuvent être facilement exécutés depuis le terminal. Python peut également être utilisé pour créer des bots qui automatisent certaines de nos tâches quotidiennes.

##### **Intelligence Artificielle :**

Python est également largement utilisé dans le domaine en croissance de l'Intelligence Artificielle (IA). Google a sélectionné Python pour être l'un des premiers langages de programmation bien supportés pour l'entraînement et l'interaction avec des modèles utilisant Tensorflow.

#### **Développement d'applications mobiles**

Les applications mobiles et les jeux peuvent être créés avec Python en utilisant Kivy, Pygame et PyQt.

#### **Sécurité et Réseautage :**

Python est utilisé pour construire des outils de réseautage et de sécurité largement utilisés. L'automatisation à distance de Python est la plus sécurisée, rapide et efficace pour les tests cloud des frameworks. C'est pourquoi les développeurs professionnels utilisent Python pour créer les frameworks les plus sécurisés et pour la programmation de sockets.

##### **Apprentissage automatique, Deep Learning**

Python est l'un des meilleurs langages adaptés à l'apprentissage automatique, au deep learning et à l'analyse de données avec une forte emprise dans tous ces domaines.

Il existe des langages spécialisés mieux adaptés à divers rôles tels que R et MATLAB, mais lorsqu'il s'agit de domaines d'applications chevauchants, Python l'emporte haut la main grâce à sa flexibilité, son prototypage rapide et la disponibilité des bibliothèques.

#### **Développement de Bots Telegram**

Vous pouvez utiliser Python et certaines bibliothèques Python pour développer vos propres [Telegram Bots](https://core.telegram.org/bots).

#### **Collecte de données par crawling et scraping**

Python peut également être utilisé pour analyser les codes sources des pages et récupérer leurs données. L'utilisation de certains modules Python, tels que Scrapy et également (pour certaines pages qui utilisent JavaScript) Selenium devrait faire l'affaire !

## **Devez-vous utiliser Python 2 ou Python 3 ?**

Les deux versions sont similaires. Si vous connaissez l'une, passer à l'écriture de code dans l'autre est facile.

* [Python 2.x ne sera plus maintenu après 2020.](https://www.python.org/dev/peps/pep-0373/)
* 3.x est en développement actif. Cela signifie que toutes les améliorations récentes de la bibliothèque standard, par exemple, sont uniquement disponibles par défaut dans Python 3.x.
* L'écosystème Python a accumulé une quantité significative de logiciels de qualité au fil des ans. L'inconvénient de la rupture de la compatibilité ascendante dans 3.x est que certains de ces logiciels (en particulier les logiciels internes des entreprises) ne fonctionnent toujours pas sur 3.x.

## **Installation**

La plupart des systèmes d'exploitation basés sur *nix sont livrés avec Python installé (généralement Python 2, Python 3 dans les plus récents). Remplacer l'installation par défaut de Python de votre système n'est pas recommandé et peut causer des problèmes. Cependant, différentes versions de Python peuvent être installées en toute sécurité aux côtés de la version par défaut de votre système. Voir [Installation et Utilisation de Python](https://docs.python.org/3/using/index.html).

Windows n'est pas livré avec Python, mais l'installeur et les instructions peuvent être trouvés [ici](https://docs.python.org/3/using/windows.html).

## **Interpréteur Python**

L'interpréteur Python est ce qui est utilisé pour exécuter les scripts Python.

S'il est disponible et dans le chemin de recherche du shell Unix, il est possible de le démarrer en tapant la commande `python` suivie du nom du script. Cela invoquera l'interpréteur et exécutera le script.

`hello_campers.py`

```python
print('Hello campers!')
```

Depuis le terminal :

```text
$ python hello_campers.py
Hello campers!
```

Lorsque plusieurs versions de Python sont installées, il est possible de les appeler par version en fonction de la configuration de l'installation. Dans l'environnement personnalisé de l'IDE Cloud9, elles peuvent être invoquées comme suit :

```text
$ python --version
Python 2.7.6
$ python3 --version
Python 3.4.3
$ python3.5 --version
Python 3.5.1
$ python3.6 --version
Python 3.6.2 
$ python3.7 --version
Python 3.7.1
```

## **Mode interactif de l'interpréteur Python**

Le mode interactif peut être démarré en invoquant l'interpréteur Python avec le drapeau `-i` ou sans aucun argument.

Le mode interactif a un prompt où les commandes Python peuvent être entrées et exécutées :

```text
$ python3.5
Python 3.5.1 (default, Dec 18 2015, 00:00:00)
GCC 4.8.4 on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> print("Hello campers!")

>>> 1 + 2
3
>>> exit()
$
```

## **Avantages et inconvénients de Python**

### **Avantages**

1. Langage interactif avec un support de module pour presque toutes les fonctionnalités.
2. Open Source : Vous pouvez donc contribuer à la communauté, les fonctions que vous avez développées pour une utilisation future et pour aider les autres.
3. Beaucoup de bons interpréteurs et notebooks disponibles pour une meilleure expérience comme le notebook Jupyter.

#### **Inconvénients**

1. Étant open source, de nombreuses méthodes différentes se sont développées au fil des ans pour les mêmes fonctions. Cela crée parfois du chaos pour les autres pour lire le code de quelqu'un d'autre.
2. C'est un langage lent. Donc, c'est un très mauvais langage à utiliser pour développer des algorithmes généraux.

## **Bonjour le monde !**

En revenant à la documentation, nous pouvons lire à propos de la fonction [`print`](https://docs.python.org/3/library/functions.html#print), une [_fonction intégrée_](https://docs.python.org/3/library/functions.html) de la [Bibliothèque Standard de Python](https://docs.python.org/3/library/index.html).

```text
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

Les fonctions intégrées sont listées par ordre alphabétique. Le nom est suivi d'une liste parenthésée de paramètres formels avec des valeurs par défaut optionnelles. En dessous se trouve une courte description de la fonction et de ses paramètres, et il y a parfois un exemple.

La fonction [`print`](https://docs.python.org/3/library/functions.html#print) dans Python 3 remplace l'instruction [`print`](https://docs.python.org/2/reference/simple_stmts.html#print) dans Python 2.

```text
>>> print("Hello world!")
Hello world!
```

Une fonction est appelée lorsque le nom de la fonction est suivi de `()`. Pour l'exemple Hello world!, la fonction print est appelée avec une chaîne comme argument pour le premier paramètre. Pour le reste des paramètres, les valeurs par défaut sont utilisées.

L'argument avec lequel nous avons appelé la fonction `print` est un objet `str` ou _chaîne_, l'un des [_types intégrés_](https://docs.python.org/3/library/stdtypes.html#text-sequence-type-str) de Python. De plus, la chose la plus importante à propos de Python est que vous n'avez pas à spécifier le type de données lors de la déclaration d'une variable ; le compilateur de Python le fera lui-même en fonction du type de valeur assignée.

Le paramètre `objects` est précédé d'un `*` qui indique que la fonction prendra un nombre arbitraire d'arguments pour ce paramètre.
