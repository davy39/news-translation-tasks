Pouvez-vous coder un éditeur de code dans un éditeur de code que vous avez codé ?

Nous venons de publier un nouveau cours sur la chaîne YouTube de [freeCodeCamp.org](http://freeCodeCamp.org) où vous apprendrez à construire votre propre éditeur de code basé sur le navigateur.

Dans ce tutoriel, le développeur Mohammed Al Abrah vous guide à travers la création d'un éditeur simple qui s'exécute entièrement sur une seule page HTML. Vous construirez des onglets pour le HTML, le CSS et le JavaScript, et mettrez en place un aperçu en direct instantané afin de voir votre code en action immédiatement.

Le cours couvre également des fonctionnalités pratiques telles que les raccourcis clavier, la validation JavaScript et les fonctionnalités de sauvegarde/chargement avec JSON et le stockage local (local storage). Vous ajouterez même des améliorations d'accessibilité telles que la gestion du focus et la navigation complète au clavier.

À la fin, vous aurez votre propre éditeur léger que vous pourrez utiliser pour vous entraîner, enseigner ou pour de petits projets.

Regardez le cours complet [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=Vs4hQsDpxww) (durée de 4 heures).
