La recherche est l'une des choses les plus importantes dans le développement web moderne.

Elasticsearch est un moteur de recherche open-source, de qualité entreprise. Accessible via une API extensive et élaborée, Elasticsearch peut alimenter des recherches extrêmement rapides qui supportent les applications de découverte de données.

Dans cette vidéo, apprenez-en plus sur Elasticsearch et comment vous pouvez commencer à l'utiliser dans vos applications Node.js.

Vous pouvez regarder la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=DR-zJXxtefE) (19 minutes de visionnage).
