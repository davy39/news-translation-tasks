## **Qu'est-ce que** CSS **Flexbox** ?

CSS Flexbox signifie CSS Flexible Box Layout. C'est un modèle de mise en page qui permet aux éléments réactifs d'être réorganisés dans un conteneur en fonction des propriétés CSS qui définissent comment les enfants doivent "bouger" pour s'adapter à la taille de leur conteneur.

Avec Flexbox, vous pouvez aligner, distribuer et espacer les éléments en utilisant des propriétés simples, afin d'éviter des calculs complexes dans votre CSS.

Pendant le cours, vous apprendrez à utiliser ces propriétés :

* `**flex-direction**`
* `**justify-content**`
* `**flex-wrap**`
* `**align-items**`
* `**align-content**`
* `**flex-grow**`
* `**flex-basis**`
* `**flex-shrink**`

Et plus encore ! Vous apprendrez également à les combiner avec des requêtes média pour créer des mises en page réactives encore plus puissantes.
