Vous pouvez apprendre tout ce dont vous avez besoin pour obtenir la certification Microsoft 365 Fundamentals en suivant ce cours gratuit de 4 heures.

## **Qu'est-ce que** la certification Microsoft 365 Fundamentals ?

La certification Microsoft 365 Fundamentals est une certification Microsoft de niveau débutant couvrant un large éventail de produits Microsoft Software-as-a-Service (SaaS).

Ces produits Microsoft SaaS sont axés sur l'amélioration des opérations commerciales et administratives ainsi que sur la gestion d'équipe et la collaboration.

Le code du cours pour les Microsoft 365 Fundamentals est le MS-900.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/image-4.png)



## À qui s'adresse la certification Microsoft 365 Fundamentals ?

* Vous êtes nouveau dans le cloud et souhaitez apprendre les fondamentaux et les avantages de l'adoption des services cloud en général, et du modèle cloud logiciel en tant que service (SaaS) en particulier.
* Vous êtes un utilisateur professionnel, un administrateur ou un professionnel de l'informatique.
* Vous souhaitez comprendre les capacités de Microsoft 365 et comment construire des solutions de base, et déployer des services cloud dans votre organisation.
* Vous êtes un ingénieur cloud senior ou un administrateur Microsoft 365 qui doit réinitialiser ou rafraîchir ses connaissances après plusieurs années de travail.


## Aperçu des Microsoft 365 Fundamentals

La certification Microsoft 365 Fundamentals couvre les sujets suivants :

* Les fondamentaux du cloud, par exemple, les concepts du cloud, les types de cloud computing, etc.
* Les services et concepts principaux de Microsoft 365, par exemple, SharePoint Online, Microsoft Viva, Teams.
* Les options de point de terminaison et de déploiement dans Microsoft 365, par exemple, Endpoint Manager, AVD, etc.
* La sécurité, la conformité, la confidentialité et la confiance dans Microsoft 365, par exemple, MFA, Defender 365.
* Les tarifs de Microsoft 365 pour les abonnements, les licences et les offres de support pour les services.

## **Comment obtenir la certification ?**

Vous pouvez obtenir la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft.

Microsoft Azure est partenaire avec Pearson Vue et PSI Online, qui disposent d'un réseau de centres de test dans le monde entier. Ils proposent des examens en personne et en ligne. Si vous en avez l'opportunité, je vous recommande de passer l'examen en personne.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous pouvez vous inscrire et réserver votre examen.

Les frais d'examen s'élèvent à **99 USD**.
