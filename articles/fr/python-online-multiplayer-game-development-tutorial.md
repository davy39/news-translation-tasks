Ce tutoriel de jeu en ligne Python de Tech with Tim vous montrera comment coder un jeu multijoueur évolutif avec Python en utilisant des sockets/réseaux et pygame. Vous apprendrez à déployer votre jeu afin que des personnes du monde entier puissent jouer les unes contre les autres.

Après ce tutoriel, vous apprendrez et comprendrez les points suivants :
• Comment fonctionne un système Client/Serveur
• Comment développer des applications pour un serveur externe
• Comment coder un Client
• Comment coder un Serveur
• Envoyer des données chiffrées telles que des objets via un port
• Connecter plusieurs clients à un serveur

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=McoDjOCb2Zo) (2,5 heures de visionnage).
