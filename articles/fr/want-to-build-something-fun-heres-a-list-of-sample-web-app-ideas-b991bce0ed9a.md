> Intéressé par l'apprentissage de JavaScript ? Obtenez mon ebook sur [jshandbook.com](https://jshandbook.com/)

Si vous lisez cet article, vous cherchez probablement une idée. Vous voulez probablement construire une application simple que vous pouvez utiliser dans un tutoriel ou dans votre projet d'exemple pour tester un nouveau framework ou une API. Mais vous ne trouvez rien qui vous parle vraiment.

Elle doit être suffisamment simple pour ne pas prendre tout votre temps, mais en même temps suffisamment complexe pour valoir la peine d'être faite.

"Je ne veux pas construire une autre application de liste de tâches !" Je vous entends penser.

Je comprends. J'ai écrit cet article pour m'aider moi-même, et j'espère qu'il vous aidera également.

### Ce que vous trouverez ci-dessous

Certaines des idées sont autonomes (ne nécessitant pas l'utilisation d'une API externe), tandis que d'autres utilisent des API publiques célèbres où vous pouvez facilement obtenir des données pré-construites.

Certaines nécessitent une partie serveur, et d'autres non, ce qui peut également dépendre de votre implémentation.

Mais j'ai essayé de m'assurer que ces idées sont :

* bonnes pour un tutoriel
* bonnes pour expérimenter avec les technologies web
* pas quelque chose qui prendra une semaine à comprendre
* pas des "idées de startup"
* ciblées pour les applications web
* faciles à expliquer
* faciles à construire (moins de 24 heures, si préparé)
* faciles à étendre avec de nouvelles fonctionnalités

Alors, assez parlé, voici la liste !

### Applications simples sans dépendances externes

#### Une application de suivi de poids

* Elle accepte un ensemble d'entrées manuelles de mesures de poids prises à différentes dates
* Elle peut tracer un graphique
* Elle peut vous permettre de suivre plusieurs entités, par exemple le poids de plus d'une personne
* Elle stocke les poids quelque part

#### Une application de calculatrice

* Une calculatrice standard : nombres, +, -, *, /, et le résultat

#### Une base de données de livres

* Entrez les livres que vous possédez
* Entrez les livres que vous aimeriez acheter
* Stockez les informations et les images des livres

#### Une application de recettes

* Entrez un nom et une description avec les étapes
* Ayez des images
* Ayez un classement pour la difficulté et la qualité
* Ajoutez le temps nécessaire
* Ayez différentes étapes avec une image pour chacune
* Stockez-les quelque part

#### Un suivi des factures

* Enregistrez les factures, les montants et les dates
* Listez les factures
* Ayez quelques graphiques (cette année / l'année dernière)
* Stockez-les quelque part

#### Un suivi des dépenses

* Enregistrez les dépenses et étiquetez-les (ou avez des catégories)
* Listez les dépenses
* Ayez quelques graphiques (le mois dernier / l'année dernière)
* Stockez-les quelque part

#### Une application de chat

* Une sorte de Slack simplifié
* Les gens entrent sans authentification et se voient attribuer un nom, stocké pour quand ils reviennent
* Stockez l'historique
* Ajoutez des notifications

#### Une application de notes

* Ajoutez une nouvelle note
* Listez toutes vos notes dans la barre latérale
* Stockez-les quelque part

#### Une application de journal personnel

* Ajoutez des entrées avec une date et du texte
* Montrez les plus récentes en premier
* Joignez des images
* Stockez-les quelque part

#### Une application Pomodoro

* Entrez un temps
* Démarrez le minuteur
* Alertez lorsque le temps est écoulé

#### Un générateur de memes

* Ayez 10 images de memes populaires
* Laissez l'utilisateur ajouter le texte
* Le résultat est une image + du texte
* Stockez l'historique

#### Jeu du morpion

Nous savons tous ce qu'est un jeu de morpion ?

#### Le jeu de la vie

Un excellent projet impliquant des mathématiques et des graphiques.

#### Un moteur de blog

* Permettez à l'utilisateur de se connecter et d'ajouter des articles
* Les visiteurs peuvent ajouter des commentaires
* Stockez les données quelque part

#### Un moteur de Q&A

* Permettez à l'utilisateur de se connecter
* Ajoutez des questions
* Répondez aux questions
* Permettez à l'utilisateur original de choisir la meilleure question
* Stockez les données quelque part

#### Un moteur de forum

* Permettez à l'utilisateur de se connecter
* Ajoutez des articles
* Commentez les articles
* Stockez les données quelque part

#### Un chat en direct intégrable

Pensez à Intercom ou Olark.

* Ayez un "backend" où vous répondez
* Intégrez sur une page web
* Laissez les gens vous écrire en privé

### Applications alimentées par API

#### Un client Hacker News

* Listez les articles populaires
* Montrez les commentaires d'un article
* Montrez le profil d'un utilisateur
* Recherchez sur HN

Consultez [HNPWA](https://hnpwa.com/) et [Awesome Hacker News](https://github.com/cheeaun/awesome-hacker-news) pour vous inspirer.

#### Un client Reddit

* Listez les articles populaires
* Listez les commentaires d'un article
* Montrez le profil d'un utilisateur

#### Un client Instagram

* Entrez un hashtag et obtenez les derniers articles
* Entrez un nom d'utilisateur et obtenez les derniers articles
* Permettez de stocker un ou plusieurs hashtags/noms d'utilisateur et obtenez tous les derniers articles de ceux-ci

#### Un client de l'API GitHub

* Listez les dépôts populaires d'aujourd'hui / de la semaine / du mois
* Listez les derniers commits dans un dépôt
* Montrez les dépôts publics d'une personne ou d'une organisation classés par étoiles

#### Un client de l'API Unsplash

* Recherchez des images par sujet
* Laissez l'utilisateur entrer un terme, montrez des images pertinentes

Commencez par [Unsplash API](https://unsplash.com/developers) pour vous inspirer.
