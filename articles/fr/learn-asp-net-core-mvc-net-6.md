ASP.NET Core MVC est un framework riche créé par Microsoft pour construire des applications web et des API en utilisant le modèle de conception Modèle-Vue-Contrôleur.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser la dernière version d'ASP.NET Core MVC.

Bhrugen Patel a développé ce cours. Bhrugen est un développeur expérimenté qui enseigne ASP.NET depuis des années.

Dans ce cours, vous apprendrez les bases de .NET Core (.NET 6) puis vous construirez une application CRUD simple avec MVC. Le cours s'adresse à toute personne familiarisée avec les bases d'ASP.NET et qui souhaite savoir comment architecturer et coder des applications réelles dans ASP.NET Core.
