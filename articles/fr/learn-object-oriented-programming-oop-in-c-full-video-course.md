La Programmation Orientée Objet (POO) est couramment utilisée lors de l'écriture de code en C++. C'est un concept utilisé dans de nombreux autres langages de programmation et il est important que tous les programmeurs le comprennent.

La Programmation Orientée Objet est un paradigme de programmation qui utilise les concepts d'objets et de classes. C'est une norme en programmation utilisée pour résoudre un type spécifique de problème. La POO est une manière de structurer un programme en utilisant des morceaux de code réutilisables (appelés classes) qui peuvent être utilisés pour créer des instances individuelles d'objets.

Nous avons publié un cours complet sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra tout sur la Programmation Orientée Objet et comment implémenter les concepts en utilisant le C++.

Ce cours a été développé par Saldina Nurak, connue sur YouTube sous le nom de CodeBeauty. Saldina est une ingénieure logicielle expérimentée en C++ qui a créé de nombreux cours sur le C++.

Tout d'abord, vous apprendrez les bases de la programmation orientée objet. Ensuite, vous apprendrez les concepts avancés de la POO et comment les implémenter en utilisant du code C++.

Voici les sujets abordés dans ce cours :

* Introduction à la POO
* Classes et objets
* Modificateurs d'accès
* Constructeurs
* Encapsulation
* Abstraction
* Héritage
* Polymorphisme
* Conclusion
* Bêtisier
