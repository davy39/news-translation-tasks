Les frameworks MVC tels que Ruby on Rails, Laravel et Angular sont souvent utilisés dans le développement web. MVC signifie "Modèle Vue Contrôleur" et est une manière courante de séparer les principales parties d'une application.

Et nous avons publié un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire votre propre framework MVC à partir de zéro en utilisant PHP.

Ce cours vous donnera une meilleure compréhension du fonctionnement interne de ces frameworks.

Le framework MVC que vous construirez dans ce cours ne sera pas exactement prêt pour la production. Mais il vous apprendra à appliquer ces concepts fondamentaux à vos propres projets de développement web.

Ce cours est enseigné par le développeur PHP expérimenté Zura Sekhniashvili, connu sur YouTube sous le nom de The Codeholic.

Vous pouvez coder chez vous et apprendre à implémenter les fonctionnalités suivantes du framework MVC :

* Routage personnalisé
* Composer
* Contrôleurs
* Vues / Mises en page
* Modèles
* Migrations
* Classes de widgets de formulaire
* Traitement des données de requête
* Validations
* Inscription/Connexion
* Simple Active Record
* Messages flash de session
* Middlewares
* Événements d'application
* Noyau du framework réutilisable / installable

En implémentant ces fonctionnalités, vous développerez une compréhension plus solide de :

* Comment travailler avec le système de fichiers
* Comment travailler avec les bases de données
* Comment utiliser l'abstraction dans des projets réels
* Comment implémenter un routage personnalisé
* Comment installer des packages tiers en utilisant Composer
* Comment créer des packages Composer réutilisables
* et d'autres compétences importantes que vous pourrez appliquer à de futurs projets de développement logiciel.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/6ERdu4k62wI) (6 heures de visionnage).
