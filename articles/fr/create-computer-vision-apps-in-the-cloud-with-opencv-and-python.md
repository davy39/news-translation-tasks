OpenCV est une bibliothèque de fonctions de programmation principalement destinée à la vision par ordinateur en temps réel. 

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à utiliser OpenCV dans le cloud avec Python.

Misbah Mohammed a créé ce cours. Il possède une grande expérience avec le machine learning et rend le suivi simple si vous connaissez déjà un peu Python.

Vous apprendrez à créer des applications de vision par ordinateur dans le cloud sur Google Colab. Vous utiliserez l'AI et le machine learning.

![Image](https://www.freecodecamp.org/news/content/images/2021/05/opencv.gif)

Voici les sections de cette vidéo :

* Leçon 1 : Changer les profils de couleur d'une image
* Propriétés de l'image
* Leçon 2 : Détection de contours (Edge Detection)
* Érosion et dilatation
* Leçon 3 : Manipulation d'image - Suppression du bruit
* Leçon 4 : Dessiner des formes et écrire du texte sur des images
* Exercice intermédiaire 1 : Détection de couleurs
* Exercice intermédiaire 2 : Détection de visages
* Exercice intermédiaire 3 : Détection de formes
* Projet 1 : Suivi de balle
* Projet 2 : Reconnaissance faciale

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/iXNsAYOTzgM) (3 heures de visionnage).
