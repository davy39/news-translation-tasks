### La structure du cours

Le cours est structuré autour de trois sections. Les deux premières sont linéaires et doivent être regardées dans l'ordre. Elles contiennent les concepts de CSS Grid qui me semblent les plus importants à apprendre.

La troisième est une section bonus, dans laquelle vous apprendrez des concepts qui n'ont pas été inclus dans les deux premières parties, et vous verrez quelques exemples de ce que vous pouvez construire avec CSS Grid.

#### Section 1 — Les bases

Nous commencerons par **pourquoi** vous devriez apprendre CSS Grid. Je vous montrerai les avantages de CSS Grid par rapport à un Framework comme Bootstrap, en comparant le code entre les deux approches.

![Image](https://cdn-media-1.freecodecamp.org/images/1*084K3NDHrgIlOZtnJySCvA.png)

Nous plongerons ensuite directement dans CSS Grid, en commençant par une grille très simple qui vous apprendra à définir des **lignes** et des **colonnes**, ainsi que les différentes valeurs que vous pouvez utiliser pour définir la largeur et la hauteur. Cela vous apprendra également les bases du responsive design.

![Image](https://cdn-media-1.freecodecamp.org/images/1*zEiqVORP00FEjMbVYUVrsw.png)

L'étape suivante consiste à créer une maquette de site web. Ici, vous apprendrez à positionner des éléments dans la grille. Nous terminerons la première section par une expérience passionnante utilisant la propriété `grid-template-areas`, qui vous permet de prototyper des mises en page rapidement.

![Image](https://cdn-media-1.freecodecamp.org/images/1*mSUhBPxbppAI_IU3ioGv6w.png)

À la fin de la section une, vous devriez en savoir assez pour commencer à utiliser CSS Grid dans vos projets personnels. Voici les principaux concepts que vous apprendrez :

* `display: grid;`
* `grid-template-columns`, `grid-template-rows`
* `grid-template` (raccourci)
* `grid-column`, `grid-row`
* `grid-template-areas`
* `grid-area`
* `grid-gap`
* `fr`

Pour avoir un aperçu de cette section, vous pouvez également consulter mes deux articles [Apprenez CSS Grid en 5 minutes](https://medium.freecodecamp.org/learn-css-grid-in-5-minutes-f582e87b1228) et [Comment prototyper des sites web rapidement avec CSS Grid](https://medium.freecodecamp.org/how-to-prototype-websites-quickly-with-css-grid-ffc9cba08583). Ils abordent quelques-uns de ces concepts, bien qu'ils ne soient pas aussi approfondis que le cours.

#### Section 2 — Les concepts avancés

Ici, vous en apprendrez davantage sur le responsive design avancé. Cela vous permettra de créer une grille d'images super cool à la fin de la section :

![Image](https://cdn-media-1.freecodecamp.org/images/1*PEWfT_RM3cM3LO3wP8DP7Q.gif)

  
Remarquez comment la grille se réorganise autour des éléments pour fonctionner quelle que soit la largeur.

Comme vous pouvez le voir, elle fait varier le nombre de colonnes en fonction de la largeur de l'écran et réorganise les éléments (qui ont des tailles et des formes différentes) afin qu'il n'y ait pas d'espaces vides.

L'essentiel de la magie opère [en une seule ligne de CSS](https://medium.freecodecamp.org/how-to-make-your-html-responsive-by-adding-a-single-line-of-css-2a62de81e431). Cependant, cette ligne de code est un peu complexe, nous allons donc détailler chaque étape pour que vous la compreniez parfaitement.

Voici ce que vous apprendrez :

* `auto-fit`
* `repeat`
* `minmax`
* `grid-auto-flow`
* lignes implicites
* comment la grille dispose les éléments

À ce stade, vous devriez être à l'aise pour créer pratiquement n'importe quelle mise en page à l'aide de CSS Grid. Et quel que soit le concept que vous ne connaissez pas encore, vous serez tout à fait capable de le découvrir par vous-même, car vous aurez une solide compréhension de base de CSS Grid.

#### Section Bonus

La section finale constitue le matériel bonus. Elle n'a pas besoin d'être regardée dans l'ordre, car les screencasts ne dépendent pas les uns des autres. Cette section se compose principalement de divers concepts CSS Grid que nous n'avons pas pu intégrer dans les deux premières parties. Cependant, vous trouverez également une comparaison entre Flexbox et Grid, où j'examine comment les deux modules diffèrent et comment ils peuvent être utilisés ensemble.

De plus, mon co-fondateur [Magnus](https://medium.com/u/1a7998d688dd) vous donnera un exemple très cool sur la façon de recréer des mises en page d'articles avec CSS Grid. Cette leçon m'a également inspiré à écrire un [article sur le même sujet.](https://medium.freecodecamp.org/how-to-recreate-mediums-article-layout-with-css-grid-b4608792bad1)

![Magnus vous accompagnera dans la création d'une mise en page d'article à l'aide de CSS Grid.](https://cdn-media-1.freecodecamp.org/images/1*DB00c0ObL68bIXhjD5USgQ.png)

Si vous estimez qu'il manque quelque chose au cours, [dites-le nous](https://gitter.im/scrimba_community/Lobby) simplement et nous créerons un nouveau screencast sur le sujet dans la section bonus.

Voici ce que vous apprendrez dans la section Bonus :

* lignes nommées

* création d'une mise en page d'article de style Medium (par [Magnus](https://medium.com/u/1a7998d688dd))

### Le format Scrimba

Le cours est construit avec Scrimba, un outil de screencast de code interactif dont je suis le co-fondateur avec [Sindre](https://twitter.com/sindreaars).

Les screencasts Scrimba ressemblent à des vidéos normales, mais ils sont entièrement interactifs. Vous pouvez modifier le code à l'intérieur des casts.

Voici un gif qui explique le concept :

![Mettez le screencast en pause → Modifiez le code → Lancez-le ! → Visualisez vos modifications](https://cdn-media-1.freecodecamp.org/images/1*4PWxbgV--7ZHlB-YVqavJg.gif)

  
Mettez le screencast en pause → Modifiez le code → Lancez-le ! → Visualisez vos modifications

C'est idéal lorsque vous ressentez le besoin d'expérimenter le code pour bien le comprendre, ou lorsque vous voulez simplement copier un morceau de code. De plus, les screencasts Scrimba pèsent 1 % de la taille des fichiers vidéo, ce qui signifie qu'ils sont beaucoup plus faciles à diffuser, même si votre connexion internet est lente.
