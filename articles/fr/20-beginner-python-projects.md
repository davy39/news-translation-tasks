Vous cherchez à améliorer vos compétences en Python ? Essayez de coder ces 20 projets Python !

Nous venons de publier un cours vidéo sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer 20 projets Python amusants.

Tomi Tokko a créé ce cours. Tomi a réalisé de nombreux tutoriels très populaires, à la fois sur sa propre chaîne et sur la chaîne freeCodeCamp.org.

![Image](https://www.freecodecamp.org/news/content/images/2022/08/giphy-2.gif)

Ces projets pour débutants sont parfaits pour ceux qui veulent pratiquer leurs compétences en Python. Voici les projets que vous allez réaliser dans ce cours :

* Envoi d'e-mails
* Programme de remplacement de mots
* Calculatrice basique
* Découpeur d'e-mails
* Algorithme de recherche binaire
* Programme de quiz
* Générateur de codes QR
* Calculateur de paiements d'intérêts
* Générateur de mots de passe aléatoires
* Simulateur de lancer de dés
* Vérificateur de connectivité de site
* Convertisseur de devises
* Vérificateur d'années bissextiles
* Dictionnaire de mots
* Pierre, Papier, Ciseaux
* Détection de visages avec Python
* Automatisation avec Python
* Scraper Web
* Redimensionneur d'images
* Traceur de graphiques

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/pdy3nh1tn6I) (3 heures de visionnage).
