Il est important pour tous les développeurs web de maîtriser le CSS, ou Cascading Style Sheets. Le CSS est responsable du style et de l'apparence d'un site web. Apprenez à l'utiliser dans ce cours complet de 6 heures pour débutants par Jad Khalili.

Ce cours vidéo gratuit couvre le CSS de A à Z. Vous apprendrez tout, des compétences de base, comme la coloration et le texte, aux compétences hautement avancées, comme les animations personnalisées.

Voici quelques-unes des choses que vous apprendrez dans ce cours :

* comment colorer du texte et des éléments,
* comment formater des sections sur un site web,
* comment changer le style et la taille du texte,
* comment utiliser le CSS pour disposer des éléments sur une page,
* comment utiliser CSS Grid,
* comment utiliser CSS Flexbox,
* comment animer des éléments sur une page,
* comment ajouter des transitions aux animations,
* et bien plus encore !

Regardez le cours complet de 6 heures ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/1Rs2ND1ryYc).
