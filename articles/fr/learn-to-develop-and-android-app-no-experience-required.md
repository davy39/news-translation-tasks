Il y a plus de 2,5 milliards d'appareils Android dans le monde, tous attendant que votre application soit développée.

Nous avons publié un cours complet de 15 heures développé par MeiCode qui vous apprendra à développer une application Android. Le meilleur dans tout ça : Vous n'avez même pas besoin de savoir programmer avant de commencer.

Dans ce cours, vous apprendrez à construire une application Android réelle à partir de zéro en utilisant Java. En cours de route, vous apprendrez les bases de la programmation en Java et de la programmation orientée objet.

Voici quelques sujets abordés dans ce cours :

### • Introduction au langage de programmation Java, incluant :

* Variables
* Opérateurs
* Boucles
* Instructions conditionnelles
* Collections en Java
* Programmation Orientée Objet
* Interfaces et Classes Abstraites
* Singleton Pattern
* Statics
* Concurrence en Java
* Gestion des Exceptions

### • Conception d'interfaces utilisateur, incluant :

* Material Design
* TextView et Button
* ListView et Spinner
* RecyclerView
* Différents Layouts
* Différents Fichiers XML
* Images
* Polices
* Animations
* SnackBar et CardView

### • Persistance des données avec SharedPreferences

### • WebView

### • Et bien plus encore
