Par Constantin

Que vous soyez un développeur expert ou un débutant complet, il est toujours bon d'apprendre un nouveau langage de programmation.

Bien sûr, il n'y a pas vraiment un seul langage que chaque professionnel ou aspirant développeur doit absolument apprendre. Le prochain langage que vous choisirez d'apprendre dépendra de votre niveau de compétence actuel et de vos aspirations de carrière globales.

Ne vous inquiétez pas : je vais décomposer vos options dans cet article pour vous aider à déterminer quel langage de programmation aborder ensuite. Je recommanderai trois langages (un pour les débutants, un pour les développeurs intermédiaires et un pour les experts), ainsi que plusieurs alternatives que vous pourriez préférer, selon vos préférences.

Mais d'abord, examinons les tendances actuelles pour voir quels langages gagnent ou perdent en popularité.

### Tendances des langages de programmation

Le graphique ci-dessous montre les langages de programmation les plus demandés en 2018 par rapport à 2017.

![Image](https://cdn-media-1.freecodecamp.org/images/ud4566wXED3qlHU7m1VISV7cFeboXxVezFdB)
_Source : Données d'Indeed.com analysées par [Coding Dojo](https://www.codingdojo.com/blog/7-most-in-demand-programming-languages-of-2018/" rel="noopener" target="_blank" title=")._

Malgré le fait que Java ait plus de 20 ans, il reste le langage de programmation le plus populaire — bien que sa popularité ait diminué depuis 2017. Java est très demandé depuis longtemps, probablement parce qu'il peut fonctionner sur n'importe quel matériel ou système d'exploitation. Par exemple, toutes les applications Android sont basées sur Java et la plupart des entreprises du Fortune 500 l'utilisent pour le développement back-end.

Python, largement utilisé pour le développement web, rattrape rapidement Java en popularité globale. Le langage est largement utilisé dans le calcul scientifique, l'extraction de données et l'apprentissage automatique — c'est de là que vient la croissance de la demande.

JavaScript reste très populaire, ce qui n'est pas surprenant puisqu'il est utilisé par la plupart des développeurs et presque tous les sites web avec une logique dynamique. De plus, les frameworks front-end comme React et AngularJS sont employés pour les intégrations IoT et les appareils mobiles, donc il est peu probable que JavaScript disparaisse des graphiques de sitôt.

C++ et C#, utilisés dans le développement de jeux vidéo et de logiciels système/application (C# est utilisé pour la plateforme .NET de Microsoft), sont un peu en baisse en 2018 mais restent solides. PHP gagne en popularité comme moyen d'ajouter des fonctions web que HTML ou CSS ne peuvent pas gérer, et aussi pour son utilisation avec les bases de données MySQL. Enfin, Perl reste un langage de premier plan pour sa popularité parmi les administrateurs système et réseau en tant que « langage colle » pour connecter différents composants logiciels.

Une chose à remarquer concernant le graphique ci-dessus est que les langages les plus populaires sont tous fondamentaux. Ces langages (comme discuté plus en détail ci-dessous) sont importants pour les débutants à apprendre. Mais les experts pourraient être plus intéressés par l'apprentissage d'un langage avancé.

![Image](https://cdn-media-1.freecodecamp.org/images/IsYdyz6AYXkTpQMoVzzH4q9tm07ICM7ES6E0)

### Langages de programmation fondamentaux vs avancés

Les langages les plus demandés sont parfois appelés langages de programmation fondamentaux. Ils sont utilisés depuis les premiers jours pour construire les fondations de l'informatique et des réseaux modernes.

Cependant, il existe également de nombreux langages de niche avec des communautés de développeurs en croissance, tels que Haskell (utilisé pour l'IA et l'apprentissage automatique), Scala (un remplacement pour Java), ML (un langage fonctionnel), Scheme (utilisé pour l'IA et la programmation web), et des centaines d'autres.

Ces langages avancés ne sont pas adaptés aux débutants. Cependant, si vous prévoyez de devenir un expert dans un domaine particulier, comme l'IA et l'apprentissage automatique, connaître un langage comme Haskell pourrait être une compétence inestimable.

Une fois que vous avez appris un ou plusieurs langages fondamentaux, vous devriez être capable d'apprendre à peu près n'importe quel nouveau langage de programmation ou langage plus avancé. N'oubliez pas qu'il est presque toujours bon d'apprendre un nouveau langage, car cela vous aidera à apprendre à penser aux problèmes de différentes manières.

Avec cela en tête, voici trois langages qui sont des incontournables pour les développeurs de différents niveaux de compétence (débutant, intermédiaire et avancé), ainsi que quelques alternatives.

### Un langage de codage pour les débutants

**JavaScript** est un bon choix pour votre premier langage de programmation. Comme mentionné ci-dessus, JavaScript est presque omniprésent. Et vous pouvez l'utiliser pour développer toutes sortes d'applications. De nombreux sites que vous utilisez probablement tous les jours [dépendent de JavaScript](https://generalassemb.ly/blog/learning-to-code-why-you-should-learn-javascript-first/), y compris Gmail, Facebook, Twitter, Instagram et Spotify.

JavaScript est utilisé pour le développement front-end des applications web (en utilisant des frameworks comme AngularJS, Ember.js et React) et le développement back-end (en utilisant le framework Node.js). L'apprentissage de ces frameworks prend plus de temps que l'apprentissage de JavaScript seul, mais cela aidera à augmenter votre employabilité (et votre salaire).

Peut-être mieux encore, JavaScript est en fait intégré dans votre navigateur, donc vous n'avez pas besoin d'installer quoi que ce soit pour commencer à programmer avec. Cela le rend incroyablement accessible. Il y a de bonnes chances que vous trouviez les bases faciles à apprendre, mais ne vous inquiétez pas car il y a beaucoup de concepts avancés pour vous occuper pendant très longtemps.

Si vous êtes un débutant et que vous voulez devenir un développeur professionnel, il est bon d'apprendre JavaScript. Cela dit, n'importe lequel des langages fondamentaux serait également un bon choix. Idéalement, vous devriez apprendre plus d'un langage fondamental tout au long de votre carrière.

#### **Alternatives pour les débutants**

**Python** est relativement facile et amusant à apprendre, avec très peu de règles de syntaxe strictes. En fait, il se lit comme l'anglais, donc même un débutant complet peut comprendre le code. Python a été utilisé pour construire [Instagram, YouTube et Spotify](https://makeawebsitehub.com/which-programming-language/), donc cela pourrait être votre billet pour un emploi dans une grande entreprise.

Comme Python, **Ruby** est également facile à apprendre pour les débutants. Vous n'avez pas besoin de mémoriser de nombreuses commandes ou règles de syntaxe pour l'utiliser. Les startups utilisent de plus en plus le framework full-stack de Ruby appelé Ruby on Rails — il a été utilisé pour construire des sites web comme [Airbnb](https://skillcrush.com/2015/02/02/37-rails-sites/), [Groupon](http://learn.onemonth.com/10-reasons-beginners-should-learn-ruby-on-rails/), et bien d'autres.

### La prochaine étape pour les programmeurs intermédiaires

Une fois que vous avez acquis une certaine expérience en programmation, il est bon de prendre le temps d'apprendre **C** (ainsi que C# et peut-être C++).

C est plus difficile à apprendre que JavaScript, mais c'est une compétence précieuse à avoir car la plupart des langages de programmation sont en fait implémentés en C. Cela est dû au fait que C est un langage de « niveau machine ». Donc l'apprendre vous enseignera comment fonctionne un ordinateur et rendra en fait l'apprentissage de nouveaux langages à l'avenir plus facile.

Apprendre à programmer en C, c'est comme comprendre l'anatomie de base avant de devenir médecin. Si vous voulez devenir un développeur expert, vous devez comprendre comment les ordinateurs et les logiciels fonctionnent ensemble. Apprendre C est une expérience éclairante qui aura l'effet secondaire bénéfique de vous apprendre à coder plus efficacement.

C# est largement utilisé pour le développement d'entreprise sur la plateforme .NET, mais c'est aussi le langage principal pour Unity 3D, un moteur de jeu populaire pour les jeux indépendants et VR. C++ est généralement utilisé pour les applications de bureau intensives en performances (y compris les jeux). Il a également été utilisé pour créer de nombreux grands systèmes par des entreprises comme Microsoft, Oracle et Adobe.

#### **Alternative pour les codeurs intermédiaires**

**Java** est un autre bon choix car il peut être écrit sur n'importe quel appareil et utilisé sur différentes plateformes, ce qui en fait un langage incroyablement polyvalent (rappelons qu'il est toujours le langage [le plus demandé](https://www.techrepublic.com/article/the-10-most-in-demand-programming-languages-of-2018/) aujourd'hui).

Il est couramment utilisé pour créer des applications, donc si votre objectif est de développer des applications mobiles, apprenez définitivement Java.

### Un nouveau défi pour les développeurs experts

Si vous avez maîtrisé les langages de programmation fondamentaux comme Javascript, C#, C++, Java et Python, vous êtes prêt pour un nouveau défi. À mon avis, ce défi devrait être **Rust**.

Selon une [enquête de Stack Overflow](https://insights.stackoverflow.com/survey/2018/#most-loved-dreaded-and-wanted), Rust a été classé comme le langage de programmation le plus apprécié par les développeurs en 2018 pour la troisième année consécutive. C'est un langage relativement nouveau, créé en 2010 par Mozilla, et en constante amélioration.

Rust est un langage compilé comparable à C en termes de performance et d'utilisation. Cependant, contrairement à C, Rust est sécurisé en mémoire : les pointeurs pendants, les débordements de tampon ou d'autres types d'erreurs de mémoire ne seront pas trouvés dans le code écrit en Rust. Il est impossible de faire ce type d'erreur en Rust car toute erreur est détectée pendant la compilation — avant que le code ne soit jamais autorisé à s'exécuter.

Cette caractéristique rend Rust difficile à apprendre pour les débutants, mais les programmeurs expérimentés apprécieront les améliorations de performance inhérentes. Il gagne régulièrement en popularité depuis sa création et il y a de bonnes chances que Rust soit en très forte demande dans les prochaines années. Selon Stack Overflow, c'était la [deuxième compétence de codage la mieux payée](https://insights.stackoverflow.com/survey/2017#most-loved-dreaded-and-wanted) en 2017 et elle est dans le [top 6](https://insights.stackoverflow.com/survey/2018/#top-paying-technologies) pour 2018.

#### **Alternative pour les experts**

Si vous êtes déjà un programmeur Java expérimenté, le prochain langage à apprendre pourrait être **Scala** (abréviation de Scalable Language).

Scala est l'une des nombreuses tentatives d'améliorer Java et peut être utilisé comme un remplacement direct pour celui-ci — en fait, il est compilé pour s'exécuter sur la machine virtuelle Java. C'est un langage complexe à apprendre, mais cela en vaut la peine : Scala a été classé parmi les technologies les mieux payées dans l'enquête [2017](https://insights.stackoverflow.com/survey/2017#technology-top-paying-technologies-by-region) et [2018](https://insights.stackoverflow.com/survey/2018/#top-paying-technologies) de Stack Overflow.

Comme Rust, Scala gagne régulièrement en popularité depuis les 5 dernières années. Des entreprises majeures comme [LinkedIn et Twitter utilisent déjà Scala](http://www.scala-lang.org/old/node/1658) dans leurs bases de code.

### Conclusion

En fin de compte, le langage de programmation que vous décidez d'apprendre ensuite devrait dépendre de votre niveau de compétence actuel et de vos intérêts/aspirations de carrière.

Les débutants devraient commencer par des langages faciles à apprendre comme JavaScript, Python ou Ruby pour se familiariser et commencer à construire des applications.

Les programmeurs plus expérimentés seraient avisés de maîtriser un langage fondamental et polyvalent comme C ou Java pour mieux comprendre comment les ordinateurs et les logiciels interagissent. Cela vous apprendra également à coder plus efficacement.

Enfin, les codeurs experts pourraient vouloir apprendre des langages comme Rust ou Scala qui améliorent les langages fondamentaux (comme C et Java). Ces deux langages avancés sont déjà très populaires parmi les développeurs expérimentés et susceptibles de devenir encore plus importants avec le temps, à mesure que les entreprises décident de construire des applications plus récentes et plus efficaces.
