Par Gitter

PHP (acronyme récursif pour _PHP: Hypertext Preprocessor_) est un langage de script open source à usage général largement utilisé, particulièrement adapté au développement web et pouvant être intégré dans HTML. Créé par [Rasmus Lerdorf](https://en.wikipedia.org/wiki/Rasmus_Lerdorf) en 1994, PHP signifiait à l'origine _Personal Home Page_.

Les forces de PHP résident dans ses API pragmatiques, sa bonne documentation et son déploiement facile. Cependant, de nombreux développeurs se demandent [si PHP vaut encore la peine d'être appris aujourd'hui](https://www.quora.com/Should-I-learn-PHP-today-Is-it-still-worth-it) en raison des incohérences dans sa conception.

Selon [W3Tech Web Technology Surveys](https://w3techs.com/technologies/details/pl-php/all/all), PHP est utilisé par 82,2 % de tous les sites web dont nous connaissons le langage de programmation côté serveur, et l'apprendre pourrait certainement vous aider à en savoir plus sur la programmation en général.

Découvrez ces canaux liés à PHP sur [Gitter](https://gitter.im) et rejoignez la conversation :

* [**laravel/laravel**](https://gitter.im/laravel/laravel) **—** Un Framework PHP pour les artisans du web.
* [**composer/composer**](https://gitter.im/composer/composer?source=explore) **—** Gestionnaire de dépendances pour PHP. [https://getcomposer.org/](https://getcomposer.org/)
* [**deployphp/deployer**](https://gitter.im/deployphp/deployer?source=explore) **—** Un outil de déploiement écrit en PHP avec support pour les frameworks populaires dès la sortie de la boîte [https://deployer.org](https://deployer.org)
* [**Sylius/Sylius**](https://gitter.im/Sylius/Sylius?source=explore) **—** Framework eCommerce PHP construit sur Symfony avec une architecture basée sur les composants et un rendu agnostique de format.
* [**flarum/flarum**](https://gitter.im/flarum/flarum?source=explore) — Un logiciel de forum open-source délicieusement simple construit avec PHP et [Mithril.js](http://mithril.js.org/).
* [**phpunit**](https://gitter.im/sebastianbergmann/phpunit?source=explore) — Le Framework de tests unitaires pour PHP.
* [**nette/nette**](https://gitter.im/nette/nette?source=explore)— Nette Framework est un outil populaire pour le développement web en PHP.
* [**monstra-cms/monstra**](https://gitter.im/monstra-cms/monstra?source=explore) **—** Monstra est un système de gestion de contenu moderne et léger.
* [**Block8/PHPCI**](https://gitter.im/Block8/PHPCI?source=explore) — PHPCI est un outil d'intégration continue gratuit et open source spécialement conçu pour PHP.
* [**Codeception/Codeception**](https://gitter.im/Codeception/Codeception?source=explore) — Codeception est un framework de test full-stack moderne pour PHP.
* [**Behat/Behat**](https://gitter.im/Behat/Behat?source=explore) **—** Behat est un framework BDD pour PHP pour vous aider à tester les attentes métier.
* [**PHPOffice/PHPExcel**](https://gitter.im/PHPOffice/PHPExcel?source=explore) — Une bibliothèque pure PHP pour lire et écrire des fichiers de feuilles de calcul
* [**PHP-DI/PHP-DI**](https://gitter.im/PHP-DI/PHP-DI?source=explore) — Le conteneur d'injection de dépendances conçu pour être pratique, puissant et agnostique de framework.
* [**rocketeers/rocketeer**](https://gitter.im/rocketeers/rocketeer?source=explore) — Rocketeer est un exécuteur de tâches et un package de déploiement moderne en PHP. Il est inspiré par la philosophie du [Laravel Framework](http://laravel.com/) et vise ainsi à être rapide, élégant et surtout facile à utiliser.
* [**appserver-io/appserver**](https://gitter.im/appserver-io/appserver?source=explore) — Un serveur d'applications multithread pour PHP, écrit en PHP.
* [**ruflin/Elastica**](https://gitter.im/ruflin/Elastica?source=explore) — Elastica est un client PHP pour elasticsearch.
