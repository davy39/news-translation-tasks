Par Leonardo Faria

Je travaille chez Thinkific avec plus de 20 autres développeurs (nous [recrutons](https://bit.ly/thnk-eng-team-lead) [pour](https://bit.ly/thnk-eng-lead) [plusieurs](https://bit.ly/thnk-senior-front-end-engineer) postes) sur différentes bases de code. GitHub fait partie de notre boîte à outils, et j'ai décidé d'écrire un article partageant quelques extensions Chrome, intégrations et applications tierces qui m'aident à être productif en tant que contributeur individuel et organisé en tant que réviseur de code et responsable d'équipe.

## Extensions Chrome

### [Refined GitHub](https://chrome.google.com/webstore/detail/refined-github/hlepfoohegkhhmjieoechaddaejaokhf?hl=en)

L'extension de [Sindre Sorhus](https://sindresorhus.com/) apporte [une longue liste d'améliorations](https://github.com/sindresorhus/refined-github#repositories) à l'interface utilisateur de GitHub, et beaucoup d'entre elles ont été intégrées au produit après un certain temps. GOAT. Plusieurs fois, un collègue est passé devant mon ordinateur et a dit "Oh, GitHub a changé cela" alors qu'en fait, il s'agit d'une amélioration de Refined GitHub.

### [Better Pull Request for GitHub](https://chrome.google.com/webstore/detail/better-pull-request-for-g/nfhdjopbhlggibjlimhdbogflgmbiahc?hl=en)

Cette extension liste les fichiers du projet sous forme d'arborescence. Cela est très utile lorsque vous avez une grande pull request et que vous devez naviguer pour mieux comprendre ce qui se passe.

![Image](https://leonardofaria.net/wp-content/uploads/2019/09/better-pull-request-for-github.jpg)

### [GitHub Hovercard](https://chrome.google.com/webstore/detail/github-hovercard/mmoahbbnojgkclgceahhakhnccimnplk?hl=en)

Cette extension affiche des informations sur un utilisateur, un projet, une issue ou un commit. C'est un truc d'interface utilisateur sympa comme le fait Facebook. Cette extension fonctionne même pour les dépôts privés, tant que vous avez un jeton valide pour récupérer les informations du service.

![Image](https://leonardofaria.net/wp-content/uploads/2019/09/hover-cards.jpg)

### [GitHub Repository Size](https://chrome.google.com/webstore/detail/github-repository-size/apnjnioapinblneaedefcnopcjepgkci?hl=en)

Cette extension ajoute automatiquement la taille du dépôt et la taille des fichiers individuels.

### [File Icon for GitHub and GitLab](https://chrome.google.com/webstore/detail/file-icon-for-github-and/ficfmibkjjnpogdcfhfokmihanoldbfe)

Pas grand-chose à dire ici :)

## Intégrations

### [Pull Panda](https://pullpanda.com/)

Pull Panda est une intégration que GitHub a acquise [il y a quelques mois](https://pullpanda.com/github) et qu'ils ont ajoutée gratuitement au produit. C'est très pratique car vous pouvez mapper les utilisateurs GitHub aux utilisateurs Slack et obtenir toutes les informations dans Slack sur le travail qu'ils suivent. Pull Panda livre les commentaires dans les PR et vous informe même des conflits git (j'adore et je déteste celui-ci).

Pull Panda peut également repérer les goulots d'étranglement, montrer qui sont ceux qui poussent/révisent le plus de code, et même assigner automatiquement des personnes aux Pull Requests.

## Applications macOS

### [Gitify](https://www.gitify.io/)

Gitify vit dans la barre de menu de votre Mac et est assez simple à comprendre. Si l'icône est verte, vous avez une notification ! Il est également cool de mentionner que l'application est [open-source](https://github.com/manosim/gitify) (basée sur Electron, si vous êtes curieux).

### [Quests](https://steamclock.com/quests/)

Quests est similaire à Gitify mais avec une différence importante : Quests supporte également GitLab.

### [DevHub](https://devhubapp.com/)

DevHub est une application multiplateforme (iOS, Android, macOS, Windows, Linux) similaire à TweetDeck, mais pour GitHub. Je ne suis pas un grand fan des applications à colonnes multiples comme TweetDeck, mais il est important de mentionner que l'application fonctionne également comme une application de barre de menu comme Gitify ou Quests.

## Applications iOS

### [Githawk](http://githawk.com/)

Githawk est mon application iOS préférée pour GitHub. Elle liste les pull requests de manière conviviale et j'aime la façon dont vous pouvez voir les fichiers affectés dans la PR et toutes les conversations en place. Si vous devez jeter un coup d'œil à une pull request lorsque vous n'êtes pas devant l'ordinateur, cette application est faite pour vous !

### [GitHub for Mobile](https://github.com/mobile)

Au moment de la rédaction de cet article (novembre 2019), GitHub propose une version bêta des applications iOS et Android. Je l'utilise depuis quelques jours sur mon iPhone X et mon iPad Mini, et ma première impression de l'application est très positive. Elle semble soignée et offre les principales fonctionnalités que vous attendez d'une application mobile.
