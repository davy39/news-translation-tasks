Un kilobyte est une collection d'environ 1 000 octets.

Dans le système métrique, kilo représente 1 000, et dans certains cas, un kilobyte est défini comme 1 000 octets. Cette définition est couramment utilisée par les entreprises de supports de stockage et pour mesurer les vitesses de transfert de données.

Mais parce que les ordinateurs fonctionnent en binaire, les informaticiens définissent généralement un kilobyte comme 2<sup>10</sup>, soit 1 024 octets. Les informaticiens utilisent le terme "kilo" parce qu'un kilobyte est suffisamment proche de 1 000 octets.

L'abréviation pour kilobyte est ko dans le système métrique (1 000 octets), ou Ko dans le système binaire (1 024 octets).

Certains groupes ont proposé le nom kibibyte pour une unité de 1 024 octets afin d'éviter la confusion. L'abréviation pour kibibyte est Kio.
