Apprenez Spring Boot dans ce cours complet pour débutants d'Amigoscode. Spring Boot est un framework incroyable pour construire des applications Java. Il facilite la création d'applications Spring autonomes, de qualité production, que vous pouvez simplement "exécuter".

Ce cours couvre :

* HTTP : GET, POST, PUT & DELETE
* Construire une base de données en mémoire
* Comment structurer les applications en utilisant l'architecture N Tier
* Auto-wiring des beans avec des injections de dépendances
* Programmer vers des interfaces
* Exécuter le fichier .jar afin de pouvoir le déployer sur n'importe quel serveur
* Utiliser le client REST Postman
* et plus encore !
