Dans ce cours complet de Fawaz Sammani, vous apprendrez les concepts clés derrière le deep learning et comment appliquer ces concepts à un projet réel en utilisant PyTorch.  

Tout d'abord, vous apprendrez les concepts théoriques nécessaires pour construire un chatbot, qui incluent les RNN, les LSTM et les modèles de séquence avec attention.

Ensuite, vous en apprendrez davantage sur PyTorch, une bibliothèque de deep learning très puissante et avancée. Vous apprendrez comment installer PyTorch et comment l'utiliser.

Enfin, la plus grande partie du cours montre comment appliquer ces concepts pour construire un chatbot dans PyTorch.

Vous pouvez regarder le tutoriel sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=CNuI8OWsppg) (6 heures de visionnage).
