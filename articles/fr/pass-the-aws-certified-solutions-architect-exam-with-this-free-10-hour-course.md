AWS Certified Solutions Architect est l'une des certifications en informatique en nuage les plus populaires. Dans ce cours complet enseigné par un formateur expert, vous apprendrez les principales parties d'Amazon Web Services et vous préparerez à l'examen AWS Certified Solutions Architect de niveau associé.

Andrew Brown d'ExamPro vous préparera à passer l'examen AWS Certified Solutions Architect Associate - et à le réussir !

Vous apprendrez à développer des applications hautement disponibles, scalables et durables. Vous apprendrez également à architecturer des solutions basées sur des cas d'utilisation métier. Il y a des labs pratiques qui vous prépareront à appliquer ce que vous apprenez dans le monde réel.

Ce cours massif couvre les sujets suivants :

* S3
* Snowball
* VPC
* VPC Endpoints
* VPC Flow Logs
* NACL
* Security Groups
* NAT
* VPC Follow Along
* IAM
* Cognito
* AWS CLI & SDK
* DNS
* Route 53
* EC2
* EC2 - Pricing
* AMI
* Auto Scaling Groups
* ELB
* EC2 Follow Along
* (Elastic) File System (EFS)
* (Elastic) Block Store (EBS)
* CloudFront
* RDS
* Aurora
* Redshift
* DynamoDB
* CloudFormation
* CloudWatch
* CloudTrail
* Lambda
* SQS
* SNS
* ElastiCache
* High Availability (HA)
* Elastic Beanstalk
* API Gateway
* Kinesis
* Storage Gateway
* Serverless Follow Along
