Dans le développement de sites web, DOM signifie Document Object Model. Il s'agit d'une interface de programmation qui permet de créer, modifier ou supprimer des éléments d'un document web. La manipulation du DOM consiste à utiliser JavaScript pour ajouter, supprimer et modifier des éléments d'un site web. C'est très courant dans le développement web.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra les bases de la manipulation du DOM avec JavaScript. CodeLab a développé ce cours.

Dans la première partie du cours, vous apprendrez les fonctionnalités de base du DOM d'un site web et les commandes JavaScript que vous pouvez utiliser pour manipuler le DOM. Dans la deuxième partie du cours, vous utiliserez ce que vous avez appris pour créer des exemples pratiques allant de débutant à avancé.
