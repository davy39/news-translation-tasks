Il existe une tonne de cours de base sur Python. Mais une fois que vous avez terminé l'un d'eux, il peut être difficile de savoir quoi apprendre ensuite.

C'est pourquoi nous avons publié un [cours intermédiaire de Python de 6 heures](https://youtu.be/HGOBQPFzWKo) sur la chaîne YouTube freeCodeCamp.org.

Patrick Loeber a créé ce cours pour aider les gens à apprendre les compétences et les concepts Python qui ne sont souvent pas couverts dans les cours de base sur Python.

Tout d'abord, vous passerez en revue des concepts de base tels que les listes, les chaînes de caractères et les dictionnaires, mais avec un accent sur certaines capacités moins connues. Ensuite, vous apprendrez des sujets plus avancés tels que le threading, le multiprocessing, les context managers, les générateurs, et plus encore.

Voici une liste complète de tous les sujets abordés dans ce cours :

* Listes
* Tuples
* Dictionnaires
* Ensembles
* Chaînes de caractères
* Collections
* Itertools
* Fonctions Lambda
* Exceptions et Erreurs
* Journalisation
* JSON
* Nombres aléatoires
* Décorateurs
* Générateurs
* Threading vs Multiprocessing
* Multithreading
* Multiprocessing
* Arguments de fonction
* L'opérateur Asterisk (*)
* Copie superficielle vs copie profonde
* Context Managers

Regardez le [cours complet sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/HGOBQPFzWKo) (6 heures de visionnage).

Et si vous êtes nouveau dans Python, nous vous recommandons de commencer par ce [cours gratuit de 14 heures enseigné par le professeur de l'Université du Michigan, Dr. Chuck](https://www.freecodecamp.org/news/python-for-everybody/).
