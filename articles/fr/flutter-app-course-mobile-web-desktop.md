Flutter est un kit de développement logiciel d'interface utilisateur de Google qui permet aux développeurs de créer des applications compilées nativement pour mobile, web et bureau à partir d'une seule base de code.

Nous avons publié un cours Flutter Essentials sur la chaîne YouTube freeCodeCamp.org. Ce cours pour débutants vous apprendra tout ce que vous devez savoir pour commencer à créer des applications avec Flutter. Le cours a été créé par MTechViral.

Flutter utilise le langage de programmation Dart, mais aucune expérience préalable avec Dart n'est requise pour suivre ce cours. Il serait cependant utile d'avoir de l'expérience avec au moins un autre langage de programmation.

Voici quelques-uns des sujets Flutter abordés dans ce cours :

* [Installation](#heading-installation)
* Widgets Container & Text
* Rows & Columns
* Material Drawer
* ListView
* Circle Avatar
* NetworkImage
* Floating Action Button
* Card, Asset Images, TextField & ScrollView
* Organiser votre code Flutter
* Réseautage avec HTTP
* ListView Builder
* GridView Builder
* Forms, Stack & Navigation
* Stockage persistant avec Shared Preferences

À la fin de ce cours, vous serez en mesure de commencer à créer vos propres applications multiplateformes avec Flutter.

Regardez le cours complet de 3 heures ci-dessous ou sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=P2IGQT3BZQo).
