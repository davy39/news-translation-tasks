Par Samuele Dassatti

Je développe actuellement une application Web progressive qui servira également d'application native pour mon prochain service.

Comme beaucoup d'entre vous le savent, **les PWAs sont des applications qui peuvent s'exécuter dans le navigateur ou dans une fenêtre autonome et peuvent tirer parti des fonctionnalités natives du système d'exploitation**, telles que les notifications push et la capacité à fonctionner hors ligne.
