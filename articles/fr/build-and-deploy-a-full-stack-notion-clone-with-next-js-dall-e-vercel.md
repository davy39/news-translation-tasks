Le développement web n'a jamais été aussi passionnant. L'afflux de nouvelles technologies et outils redéfinit les limites de ce que les développeurs peuvent créer. Au cœur de cette transformation se trouve la capacité d'intégrer des systèmes divers de manière transparente, créant ainsi des applications web puissantes.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org, qui vous apprendra à construire et déployer un clone full stack de Notion en utilisant une variété de technologies.

Elliott Chong a créé ce cours. Il se spécialise dans la création de tutoriels complets sur la construction d'applications alimentées par l'IA.

Voici les technologies principales que vous utiliserez dans le cours :

**NextJS 13's App Router** : La dernière itération du routeur de NextJS, garantissant que les routes de votre application sont efficaces et conviviales.

**Génération d'images AI DALL-E** : Élevez le contenu visuel de votre application en tirant parti des capacités de génération d'images AI de pointe.

**Shadcn & Tailwind CSS** : Créez des interfaces qui non seulement ont une belle apparence, mais suivent également les meilleures pratiques de codage.

**API de modèle de langage d'OpenAI** : Intégrez la puissance du modèle de langage d'OpenAI, portant l'interaction utilisateur de votre application à un niveau supérieur.

**ORMs** : Plongez profondément dans l'art d'interagir avec les bases de données, rendant les transactions de données plus fluides et plus efficaces.

Le cours guide les spectateurs de la configuration initiale au déploiement. Commencez par une introduction et une démonstration, naviguez à travers diverses étapes de configuration, plongez dans la construction de l'UI et des APIs, et terminez par les instructions de déploiement.

Voici toutes les sections du cours :

* Intro & Demo
* Outline Knowledge
* Set Up Project
* Set Up shadcn
* Set Up Clerk Auth
* Landing Page
* Dashboard UI
* DrizzleORM
* Create Note UI
* OpenAI Generate Image Prompt
* Create NoteBook API
* Notebook Page UI
* TipTap Editor & MenuBar
* Debounce Save
* AI Autocomplete
* KBD UI
* Firebase
* Delete Note
* Deploy & Outro

Ce tutoriel est bien plus qu'un simple guide pour construire une application web. C'est un voyage à travers le paysage en évolution du développement web. Vous pouvez regarder le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/qDunJ0wVIec) (3 heures de visionnage).
