Vous êtes sur le point de découvrir 20 excellents exemples de code de 20 développeurs incroyablement talentueux.

Js13kGames est un concours de codage JavaScript ouvert à tous ceux qui souhaitent essayer le développement de jeux. Les gagnants de 2021 ont été annoncés, et vous pouvez jouer aux jeux gratuitement.

Comme le suggère le nom, tout le code et les actifs de jeu doivent être inférieurs ou égaux à 13 kilooctets (soit exactement 13 312 octets, car 13 x 1024) lorsqu'ils sont compressés.

Et les jeux ne sont pas autorisés à utiliser des bibliothèques, des images ou des fichiers de données hébergés sur un serveur ou des services qui fournissent un quelconque type de données.

Nous [avons publié une vidéo](https://youtu.be/UiYhTcqx_wk) sur la chaîne YouTube de freeCodeCamp.org qui présente tous les jeux gagnants. Sous la vidéo, vous verrez une liste de tous les gagnants et des liens pour jouer à leurs jeux.
