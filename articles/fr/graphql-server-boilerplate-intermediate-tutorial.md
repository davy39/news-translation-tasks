Découvrez comment coder un modèle de serveur GraphQL. Ce cours intermédiaire complet de Ben Awad utilise de nombreuses technologies, notamment Typescript, Node.js, PostgreSQL, Redis, React, Jest, OAuth, TypeORM et SparkPost.

Voici ce que couvre le cours :

* Configuration d'un Projet TypeORM (#heading-installation)
* Création d'une Entité TypeORM
* Création d'un Utilisateur avec TypeORM
* Configuration de Jest
* Configuration d'une Base de Données de Test avec TypeORM
* Faire fonctionner GraphQL Yoga et Jest ensemble
* Assemblage de Schéma GraphQL
* Gestion des Erreurs dans GraphQL
* Validation des Entrées GraphQL
* Lien de Confirmation par Email avec Redis
* Configuration Globale de Jest
* Test du Lien Email Redis
* Correction de l'Erreur de Connexion
* Envoyer un Email de Confirmation avec SparkPost
* Mutation de Connexion GraphQL
* Session Express pour la Connexion
* Middleware GraphQL
* Exécution Séquentielle des Tests Jest
* Journalisation ou Middleware d'Authentification GraphQL
* Déconnexion Simple avec GraphQL
* Construction d'un Client de Test de Serveur GraphQL
* Déconnexion de Plusieurs Sessions
* Cas de Test de Mot de Passe Oublié
* Verrouillage du Compte Utilisateur GraphQL
* Implémentation du Mot de Passe Oublié GraphQL
* Limitation de Débit
* OAuth Twitter
* Démo OAuth avec React
* Exécution des Tests Jest en Parallèle
* Fusion d'un Schéma GraphQL Modulaire
* Suppression des Conditions de Course des Tests
