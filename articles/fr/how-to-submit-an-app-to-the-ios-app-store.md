Le processus de soumission sur l'App Store d'iOS peut ressembler à un labyrinthe intimidant, mais il ne doit pas forcément l'être.

Nous venons de publier un cours sur la chaîne YouTube de [freeCodeCamp.org](http://freeCodeCamp.org) conçu pour vous guider tout au long de ce processus, du début à la fin. Shad Rayhan Mazumder a développé ce cours.

Shad commence par expliquer la méthode manuelle traditionnelle. Ce cours vous guidera à travers chaque étape essentielle, en expliquant les concepts fondamentaux des identifiants d'application (Application IDs), des certificats et des profils de provisionnement (Provisioning Profiles). Vous apprendrez exactement ce que vous devez faire pour générer les bonnes informations d'identification, enregistrer vos appareils et faire fonctionner votre application sur un iPhone ou iPad physique pour les tests. Le cours couvre également la distribution de versions ad hoc pour les testeurs avant de plonger dans les étapes finales pour une version publique.

Le cours présente ensuite les concepts puissants de l'Intégration Continue et du Déploiement Continu (CI/CD), en se concentrant spécifiquement sur Xcode Cloud d'Apple. Vous apprendrez comment pousser vos dernières modifications vers un dépôt GitHub et faire en sorte que votre application se construise, se teste et se distribue automatiquement à vos testeurs sans que vous n'ayez à lever le petit doigt.

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/9vkkJ4tC4SQ) (2 heures de vidéo).
