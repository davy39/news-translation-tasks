Créer un tableau de bord pour suivre les clients et les utilisateurs est beaucoup plus facile que vous ne le pensez. Vous pouvez construire un tableau de bord marketing alimenté par une API en utilisant une approche low code. 

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un tableau de bord pour aider à gérer votre site ou votre startup.

Dans ce cours, vous apprendrez à utiliser des outils low code et des APIs pour construire un tableau de bord de support client. Vous apprendrez à créer le tableau de bord en utilisant la plateforme Appsmith, MongoDB, l'API SMTP et le webhook Discord.

Ania Kubów a créé ce cours. Elle est une instructrice populaire qui a publié de nombreux cours à la fois sur la chaîne freeCodeCamp et sur sa propre chaîne YouTube.

Vous apprendrez à construire rapidement des outils que de nombreuses startups utilisent en adoptant l'approche low-code.

Appsmith a fourni une subvention qui a rendu ce cours possible.

![image-60](https://www.freecodecamp.org/news/content/images/2022/04/image-60.png)
_Outil low-code_

Lorsque vous regardez ce cours, il est utile d'avoir déjà des connaissances de base en JavaScript et en méthodes de requêtes HTTP. Cependant, vous devriez toujours être en mesure de suivre si vous êtes nouveau dans ces concepts, car Ania vous guidera étape par étape tout au long du processus.

Voici les sections couvertes dans ce cours :

* Prise en main d'Appsmith
* Connexion à MongoDB
* Ajout de données à l'application Appsmith
* Créer une page d'envoi de messages
* Afficher différentes listes marketing
* Implémentation du webhook Discord

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=ccfpj4eD6YM) (2 heures de visionnage).
