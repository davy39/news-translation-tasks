Par Srebalaji Thirumalai

Salut à tous !

Ce guide est destiné aux personnes qui commencent avec React. J'ai soigneusement sélectionné les meilleures vidéos et articles dans chaque section pour faciliter l'apprentissage.

**Note :** Je ne suis associé à aucun des sites mentionnés ci-dessous. Ce sont purement mes opinions.

### **Prérequis**

1. Connaissances de base en HTML, CSS et JavaScript.
2. Compréhension de base des fonctionnalités ES6. Voici mon [article](https://hackernoon.com/es6-for-beginners-f98120b57414) expliquant certaines des fonctionnalités ES6.  
Pour commencer, vous devriez au moins connaître les fonctionnalités suivantes :  
1. Let  
2. Const  
3. Fonctions fléchées  
4. Imports et Exports  
5. Classes
3. Compréhension de base de l'utilisation de [npm](https://www.npmjs.com/).

### **Commencer**

Vous pouvez utiliser des éditeurs de code en ligne pour vous entraîner, ou utiliser [Create React App](https://github.com/facebook/create-react-app).

J'ai configuré un environnement de développement dans [JSFiddle](https://jsfiddle.net/me4z1xyp/2/) et dans [Codepen](https://codepen.io/srebalaji/pen/MVbWdB).

Pour maîtriser tous les fondamentaux de React, vous pouvez commencer avec les tutoriels suivants :

[Documentation officielle de React](https://reactjs.org/docs/hello-world.html) par React

[Guide du débutant pour React](https://egghead.io/courses/the-beginner-s-guide-to-reactjs) par Kent C. Dodds

[Fondamentaux de React](https://www.freecodecamp.org/news/all-the-fundamental-react-js-concepts-jammed-into-this-single-medium-article-c83f9b53eac2/) par [Samer Buna](https://www.freecodecamp.org/news/learning-react-roadmap-from-scratch-to-advanced-bff7735531b6/undefined)

À ce stade, vous devriez avoir une idée de base des fondamentaux de React. C'est suffisant pour commencer à développer des applications web simples en React.

Maintenant, consultez le tutoriel officiel de React :

[Tutoriel officiel de React](https://reactjs.org/tutorial/tutorial.html) par React

C'est un article bien écrit couvrant les fondamentaux de React. Et il explique également les sujets spécifiques très clairement.

Enfin, apprenez à vous connecter à des API avec des applications React :

[Récupération d'API avec React.js](https://blog.hellojs.org/fetching-api-data-with-react-js-460fe8bbf8f2) par [Ethan Jarrell](https://www.freecodecamp.org/news/learning-react-roadmap-from-scratch-to-advanced-bff7735531b6/undefined)

#### **Commencez à construire quelques projets**

1. Une simple application de liste de tâches
2. Une simple application de calculatrice
3. Construisez un panier d'achat
4. Afficher les statistiques des utilisateurs GitHub en utilisant l'API GitHub

React Router vous aide à créer des routes pour vos applications monopages. C'est très puissant et facile à utiliser avec votre application React.

[Routing des applications React](https://scotch.io/tutorials/routing-react-apps-the-complete-guide) par Scotch.io

Ces articles sont plus que suffisants pour vous lancer dans le routage React.

[React Router](https://reacttraining.com/react-router/web/example/basic) guide complet par React Training

### **Webpack**

Webpack est un bundler de modules JavaScript célèbre. Webpack vous aide à maintenir les dépendances sous forme de fichiers statiques pour votre projet afin que les développeurs n'aient pas à le faire.

Webpack est également livré avec des loaders. Les loaders aident à exécuter des tâches spécifiques autour de votre projet.

Pour en savoir plus sur Webpack, suivez les tutoriels suivants.

[Quand et pourquoi utiliser Webpack](https://blog.andrewray.me/webpack-when-to-use-and-why/) par Andrew Ray

[Tutoriel Webpack](https://www.youtube.com/watch?v=9kJVYpOqcVU) par Learn Code Academy

Pour configurer votre environnement React local en utilisant Webpack, vous pouvez vous référer au dépôt GitHub suivant :

[Modèle React SPA](https://github.com/meetajhu/react-spa-template) par Hanif Roshan

Je pense que les tutoriels ci-dessus sont suffisants pour commencer avec Webpack. Cependant, pour obtenir des connaissances approfondies, vous pouvez vous référer aux guides suivants :

[Introduction à Webpack](https://survivejs.com/webpack/introduction/) par SurviveJS

[Documentation officielle de Webpack](https://webpack.js.org/concepts/)

### **Rendu côté serveur**

Le rendu côté serveur est l'une des fonctionnalités les plus cool de React. Il peut être utilisé avec n'importe quelle technologie backend.

Le rendu côté serveur (SSR) dans React vous aide à créer des composants sur le serveur et à les rendre sous forme de HTML dans votre navigateur. Et lorsque tous les modules JavaScript sont téléchargés dans le navigateur, React prend le relais. Simple !

Tout d'abord, consultez l'API React-DOM :

[API React-DOM](https://reactjs.org/docs/react-dom-server.html) par React

Et suivez les tutoriels ci-dessous pour obtenir des connaissances approfondies :

[Rendu côté serveur React](https://tylermcginnis.com/react-router-server-rendering/) par Tyler McGinnis

[Rendu côté serveur avec React Router](https://medium.com/front-end-hacking/server-side-rendering-with-react-and-express-382591bfc77c) par [Roilan Salinas](https://www.freecodecamp.org/news/learning-react-roadmap-from-scratch-to-advanced-bff7735531b6/undefined)

[Guide de rendu côté serveur React](https://medium.com/styled-components/the-simple-guide-to-server-side-rendering-react-with-styled-components-d31c6b2b8fbf) par [Dennis Brotzky](https://www.freecodecamp.org/news/learning-react-roadmap-from-scratch-to-advanced-bff7735531b6/undefined)

### **Redux**

Redux est une bibliothèque JavaScript développée pour maintenir les états des applications. Lorsque vous construisez une application complexe, cela ajoutera une surcharge pour gérer les états entre les composants. Redux vous aide à stocker tous vos états dans une seule source. Et bien sûr, React fonctionne bien avec Redux :)

Pour commencer :

[Tutoriel Redux](https://www.youtube.com/playlist?list=PLoYCgNOIyGADILc3iUJzygCqC8Tt3bRXt) par Learn Code Academy

[Tutoriel Redux pour débutants](https://dev.to/valentinogagliardi/react-redux-tutorial-for-beginners-learning-redux-in-2018-13hj) par Valentino Gagliardi

[React Redux](https://css-tricks.com/learning-react-redux/) par CSS Tricks

Ces tutoriels sont plus que suffisants pour commencer avec Redux. Cependant, je ne peux m'empêcher de mentionner également le tutoriel ci-dessous. Cela en vaut la peine :)

[Commencer avec Redux](https://egghead.io/courses/getting-started-with-redux) par Dan Abramov
