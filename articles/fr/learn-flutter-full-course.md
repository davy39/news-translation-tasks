Flutter est un kit de développement logiciel d'interface utilisateur open-source utilisé pour créer des applications multiplateformes pour iOS, Android, Windows, Mac et plus encore.

Nous venons de publier un cours Flutter de 37 heures pour débutants sur la chaîne YouTube freeCodeCamp.org.

Vandad Nahavandipoor a développé ce cours. Vandad est un Google Developer Expert et Lead iOS Developer.

Flutter est le SDK moderne et gratuit de Google qui vous permet d'écrire des applications desktop, web et mobiles avec la même base de code.

Ce cours est destiné aux débutants absolus. Vous n'avez même pas besoin de savoir comment écrire du code dans un langage de programmation pour commencer. À la fin, vous saurez comment publier une application iOS et Android écrite en Flutter sur l'App Store et le Google Play Store.

Le cours commence par la configuration de votre environnement de développement. Vous apprendrez à utiliser l'éditeur de code gratuit Visual Studio Code pour écrire une application Flutter. Ensuite, vous apprendrez à créer des écrans et la logique de l'application. Vous apprendrez à utiliser Firebase et Figma. Enfin, vous apprendrez à déployer vos applications iOS et Android sur l'App Store et le Google Play Store.
