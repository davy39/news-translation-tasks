## Qu'est-ce que Microsoft Azure Administrator ?

L'Azure Administrator est la certification Azure la plus populaire car elle fournit la plus grande quantité de connaissances pratiques pour gérer l'infrastructure Azure. Elle vous permet également de **pivoter vers n'importe quel parcours de carrière** pour des rôles basés sur Azure.

Si Azure est la charge de travail principale utilisée par votre entreprise, alors cette certification cloud est une certification Azure essentielle à obtenir.

![Image](https://www.freecodecamp.org/news/content/images/2021/04/az-104-roadmap.jpg)

Si vous débutez complètement sur Azure, même si vous possédez une autre certification cloud, je vous recommande vivement de passer d'abord l'AZ-900. En effet, les certifications Azure de niveau associate sont beaucoup plus difficiles que n'importe quelle autre certification d'autres fournisseurs de services cloud. Et bien qu'Azure semble similaire en termes de services en surface, il est très différent en pratique.

%[https://www.freecodecamp.org/news/azure-fundamentals-course-az900/]

## AWS Solutions Architect vs Azure Administrator

Si vous connaissez les certifications AWS, elle est l'équivalent de l'AWS Solutions Architect Associate en termes d'importance.

En termes de contenu réel, l'Azure Administrator ressemble davantage à un mélange entre l'AWS Developer Associate et l'AWS SysOps Administrator.

C'est logique car l'architecture cloud d'Azure sert de pont vers les solutions sur site (on-premise) héritées de Microsoft. Ainsi, l'ingénierie de solutions cloud-natives de manière abstraite est secondaire, tandis que la compréhension de la manière d'amener les entreprises basées sur Microsoft vers Azure passe en premier.

Pour ceux qui n'ont pas encore travaillé dans une carrière informatique impliquant Active Directory, Windows Servers et la gestion générale de Windows, il y aura une courbe d'apprentissage supplémentaire.

Azure est un fournisseur de services cloud axé sur Windows, et si vous ne possédez pas d'ordinateur Windows, vous ne pourrez peut-être qu'observer puis expérimenter directement comment utiliser certains services cloud Azure.

Comme je l'ai mentionné précédemment, les certifications associate d'Azure sont beaucoup plus difficiles que celles des autres fournisseurs de services cloud, attendez-vous donc à y consacrer plus de temps.

![Image](https://www.freecodecamp.org/news/content/images/2021/04/azure-diff.jpg)

## Présentation de l'Azure Administrator

L'Azure Administrator couvre les points suivants :

* Azure Active Directory
* Utilisateurs, groupes et permissions
* Politiques pour assurer la conformité des ressources
* Infrastructure as Code via Azure Resource Manager
* Gestion des ressources, du périmètre et des abonnements
* Conteneurs et Kubernetes
* Azure Virtual Machines et Azure Disks
* Divers modèles de stockage via les Azure Storage Accounts
* Azure Networking (il y a beaucoup à couvrir ici)
* Disponibilité SLA, Availability Sets, Scale Sets, Application Gateway
* Azure App Services pour les applications entièrement gérées
* Azure Backup, Azure Monitor
* et plus...

Ce qu'elle ne couvre pas, ce sont des éléments tels que le Serverless, les Databases et l'Application Integration. Vous pouvez les trouver dans la certification Azure Developer.

## **Comment obtenir la certification Azure Administrator ?**

Vous pouvez obtenir la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft Azure.

Microsoft Azure est partenaire de Pearson Vue et PSI Online, qui disposent d'un réseau de centres de test dans le monde entier. Ils proposent des examens en personne et en ligne. Si vous en avez l'occasion, je vous recommande de passer l'examen en personne.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous pouvez vous inscrire et réserver votre examen.

Les frais d'examen sont de **165 $ USD**.

## Puis-je simplement regarder les vidéos et réussir l'examen ?

Vous devez appliquer les connaissances, car les questions d'examen testeront vos connaissances pratiques. Une question, par exemple, peut demander l'ordre des étapes pour lancer un service, et vous n'obtiendrez cette connaissance qu'en pratiquant.

Azure a une fréquence de mises à jour beaucoup plus élevée que les autres fournisseurs de services cloud. Parfois, il y a de nouvelles mises à jour chaque mois pour une certification.

L'examen comporte de **40 à 60** questions avec une durée de **120 minutes**.

L'examen contient de nombreux types de questions différents.

La note de passage est d'**environ 70 %**.
