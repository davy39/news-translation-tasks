React Native for Web vous permet de créer des applications qui fonctionnent sur iOS, Android et le web en utilisant la même base de code JavaScript. 

Dans ce cours complet de Ben Awad, vous apprendrez à créer une application d'entraînement en utilisant React Native for Web.

Ce cours couvre :

* React Native Web
* Mobx
* Typescript
* React Navigation
* React Hooks
* AsyncStorage / Localstorage
* le déploiement avec Netlify
* et plus encore

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=_CBYbEGvxYY) (4 heures de visionnage).
