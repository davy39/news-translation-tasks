Touch cat tail. Find man head. Who?

Les sept mots précédents sont des commandes Linux courantes. Si vous êtes programmeur, il est probable que vous deviez utiliser Linux et des commandes terminal à un moment donné.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser les 50 commandes Linux et terminal les plus courantes.

Toutes ces commandes fonctionnent sur Linux, macOS, WSL, et partout où vous avez un environnement UNIX.

Colt Steele a développé ce cours. Colt est l'un des enseignants en développement web les plus populaires et il est un excellent professeur.

![Image](https://www.freecodecamp.org/news/content/images/2021/11/image-4.png)
