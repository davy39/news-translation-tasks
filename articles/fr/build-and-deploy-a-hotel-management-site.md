Construire un site web complexe est une excellente façon d'apprendre comment plusieurs technologies fonctionnent ensemble.

Nous venons de publier un cours de 10 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire un site web de gestion d'hôtel en utilisant Next.js, React, Sanity.io, Tailwind CSS et l'API de paiement Stripe. Lari Bright a développé ce cours.

### Technologies Principales Expliquées

Avant de plonger dans le plan du cours, examinons les technologies principales avec lesquelles vous allez travailler :

* **Next.js** : Un framework React qui permet des fonctionnalités telles que le rendu côté serveur et la génération de sites statiques, rendant votre application plus rapide et plus efficace.
* **React** : Une bibliothèque JavaScript pour construire des interfaces utilisateur, connue pour son efficacité et sa flexibilité dans la création d'applications web dynamiques et performantes.
* **Sanity.io** : Une plateforme pour le contenu structuré qui vient avec un environnement d'édition open-source permettant de gérer et de livrer du contenu numérique à l'échelle mondiale.
* **Tailwind CSS** : Un framework CSS basé sur les utilitaires pour construire rapidement des designs personnalisés, fournissant des classes utilitaires de bas niveau pour créer des designs sur mesure sans quitter votre HTML.
* **Stripe** : Une plateforme de traitement des paiements en ligne hautement flexible qui s'intègre facilement avec diverses applications web pour des transactions financières sécurisées et efficaces.

### Points Forts du Cours

Ce cours complet couvre tous les aspects du développement d'un système de gestion d'hôtel fonctionnel :

* **Opérations CRUD avec Sanity.IO** : Gérez les chambres d'hôtel efficacement avec des capacités complètes de création, lecture, mise à jour et suppression.
* **Réservation de Chambres et Avis** : Implémentez des fonctionnalités pour la réservation de chambres d'hôtel et la fourniture d'avis sur les chambres.
* **Traitement des Paiements** : Intégrez Stripe pour des expériences de paiement sécurisées et efficaces.
* **Contexte React pour les Thèmes UI** : Maîtrisez l'utilisation du Contexte React pour le changement dynamique de thème.
* **Recherche Avancée et Authentification** : Implémentez une fonctionnalité de recherche robuste et une authentification sécurisée des utilisateurs.
* **TypeScript et le Hook useSwr** : Améliorez votre projet avec TypeScript pour une meilleure gestion du code et utilisez le hook useSwr pour la récupération de données.
* **Déploiement sur Vercel** : Apprenez à déployer votre application sur Vercel, assurant un lancement fluide.
