Redux est une bibliothèque JavaScript open-source populaire pour la gestion et la centralisation de l'état d'une application. Elle est couramment utilisée avec React.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser Redux.

Nikhil Thadani a développé ce cours. Nikhil a créé de nombreux cours techniques sur sa chaîne YouTube.

Dans ce cours, vous apprendrez à utiliser Redux avec la bibliothèque ReduxToolkit pour créer une application impliquant des requêtes HTTP.

Voici les sujets abordés dans ce cours :

* Terminologies de base
* Application Compteur avec react-redux
* Utilisation de Redux Toolkit
* Projet Panier d'achat - Introduction
* Projet Panier d'achat - État du slice d'authentification
* Projet Panier d'achat - État du slice du panier
* Incrémentation et décrémentation des articles du panier
* Ajout de l'état de déconnexion
* Utilisation de Firebase avec Redux
* Envoi de requêtes HTTP asynchrones avec Redux
* Ajout de notifications avec Material UI
* Ajout de l'état du slice de notification
* Utilisation du motif Redux Thunk
* Création du motif Thunk avec Redux
* Envoi de requêtes HTTP GET avec Redux

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/zrs7u6bdbUw) (2 heures de visionnage).
