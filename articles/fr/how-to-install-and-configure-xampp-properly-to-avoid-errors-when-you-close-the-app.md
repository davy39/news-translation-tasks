XAMPP est un logiciel populaire pour ceux qui utilisent des bases de données MySQL et PHP. C'est un logiciel gratuit, et il prend en charge Windows, Linux et MacOS. Cela le rend assez populaire parmi les développeurs même s'il peut également nous présenter de nombreux défis.

Un problème courant que vous avez peut-être rencontré est les erreurs que vous pouvez obtenir lorsque vous souhaitez quitter l'application. Vous pouvez facilement atténuer ce problème en ouvrant le lanceur de l'application en tant qu'administrateur. Mais ouvrir le lanceur via l'administrateur correctement peut également être problématique. Cela signifie que vous devrez configurer un paramètre pour qu'il demande un accès administrateur chaque fois que vous souhaitez le lancer.

Dans ce tutoriel rapide, je vais vous guider à travers ce processus pour que vous n'ayez pas à lutter avec.

## Processus étape par étape

Si vous gardez le répertoire d'installation par défaut pendant l'installation, il l'installera à l'intérieur du lecteur `C` de votre système d'exploitation Windows.

Entrez simplement dans le lecteur "C" et allez dans le répertoire "xampp".

![Trouver le répertoire "xampp"](https://cdn.hashnode.com/res/hashnode/image/upload/v1731566127539/9964e362-0f79-4a08-9799-7ea17bd3740e.png align="center")

Faites défiler jusqu'à ce que vous trouviez le fichier "xampp-control.exe".

![Trouver le fichier "xampp-control.exe"](https://cdn.hashnode.com/res/hashnode/image/upload/v1731566176798/8a529579-5545-4a97-b32d-6fdbffe15a95.png align="center")

Faites un clic droit après avoir sélectionné le fichier et ouvrez les **Propriétés**.

![Onglet Propriétés](https://cdn.hashnode.com/res/hashnode/image/upload/v1731566236620/dfaa3a6c-795a-4dab-a03a-66b73a7d5de5.png align="center")

Ensuite, allez dans la section **Compatibilité**. Assurez-vous d'ajouter une coche en cliquant sur la case à cocher "Exécuter le programme en tant qu'administrateur".

![Configurer les propriétés](https://cdn.hashnode.com/res/hashnode/image/upload/v1731566287073/53bf3e86-1a4b-47b3-83e9-96caf1a4195d.png align="center")

Cliquez sur "Appliquer", puis sur "OK".

C'est tout !

Maintenant, chaque fois que vous souhaitez ouvrir le lanceur XAMPP, il s'ouvrira toujours avec un accès administrateur.

### Quitter l'application

Lorsque vous souhaitez quitter l'application, vous devez cliquer sur le bouton "Quitter" au lieu d'utiliser le bouton de sortie "X" en haut à droite de la fenêtre.

![Quitter l'application](https://cdn.hashnode.com/res/hashnode/image/upload/v1731566387255/316d1778-dd5b-4a61-9efe-cbbaf8fc7552.png align="center")

Et maintenant, vous ne devriez plus avoir d'erreurs lors de la fermeture du panneau de contrôle !
