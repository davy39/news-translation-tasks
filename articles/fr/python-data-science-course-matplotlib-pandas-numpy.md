Ce cours gratuit de 12 heures sur Python Data Science vous mènera de l'ignorance totale de Python à la capacité d'analyser des données. Vous apprendrez les bases de Python, ainsi que des outils puissants comme Pandas, NumPy et Matplotlib.

Ce cours est pratique et vous mettrez en pratique tout ce que vous apprenez étape par étape.  

Ce cours inclut [une base de code complète pour référence](https://github.com/datapublishings/Course-python-data-science).   

Il commence par une introduction d'une heure aux concepts de base de la programmation, à la résolution de problèmes et au pseudocode.

Ensuite, il vous guide à travers l'installation de Python et de la puissante plateforme Anaconda pour la science des données. 

Puis, le cours vous montrera comment lancer votre propre Jupyter Notebook. C'est une manière populaire de créer des documents avec du code interactif intégré.

Les Jupyter Notebooks s'exécutent directement dans les navigateurs des utilisateurs. Cela les rend faciles à partager et à utiliser – même pour les non-programmeurs. Cela en fait un outil préféré de longue date par la communauté de la science des données.

Voici quelques autres sujets que ce cours couvrira :

* Coder sur l'iPython Shell
* Variables et opérateurs en Python 
* Booléens et comparaisons en Python
* Fonctions utiles de Python
* Contrôle de flux en Python
* Fonctions en Python
* Modules en Python
* Chaînes de caractères en Python
* Autres structures de données importantes de Python : Listes, Tuples, Ensembles et Dictionnaires
* La bibliothèque NumPy pour la science des données en Python 
* La bibliothèque Pandas pour la science des données en Python 
* La bibliothèque Matplotlib pour la science des données en Python

Et enfin, vous verrez tous ces outils fonctionner ensemble dans le cadre d'une application d'analyse de tendances COVID-19 de base.

Vous pouvez regarder le cours ci-dessous, ou le regarder [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=LHBE6Q9XlzI) (12 heures de visionnage).
