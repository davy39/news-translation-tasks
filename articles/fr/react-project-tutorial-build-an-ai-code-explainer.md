Vous voulez apprendre à combiner React 19 et les LLM d'IA pour construire quelque chose d'utile ?

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer une application d'explication de code alimentée par l'IA à partir de zéro. Tapas Adhikary a créé ce cours.

Apprenez à concevoir l'application, à configurer un backend Express.js, à mettre en place un point de terminaison REST qui communique avec un LLM, puis à intégrer le tout dans une application React 19 moderne (avec Vite + TailwindCSS).

À la fin, vous disposerez d'un outil d'explication de code par IA fonctionnel qui prend n'importe quel extrait de code et renvoie des explications claires et adaptées aux débutants. En chemin, vous découvrirez également les nouvelles fonctionnalités de React 19 comme useActionState et les Server Actions, ainsi que les meilleures pratiques pour l'intégration d'API et la gestion des erreurs.

Ce cours couvre :

* Comment concevoir et planifier un projet d'IA
    
* Configuration d'Express.js pour les API REST
    
* Connexion à un LLM pour l'explication de code
    
* Test des API avec Postman
    
* Construction d'un frontend moderne avec React 19, Vite et TailwindCSS
    
* Utilisation de useActionState et des Server Actions de React 19 dans des projets réels
    
* Gestion des réponses API et formatage de la sortie
    

Regardez le cours complet [sur la chaîne YouTube de freeCodeCamp.org](https://youtu.be/W4M-g9wsEC4) (1 heure de visionnage).
