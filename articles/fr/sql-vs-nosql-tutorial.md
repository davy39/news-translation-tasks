Quand devez-vous utiliser une base de données SQL et quand devez-vous utiliser une base de données NoSQL ?

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra les différences entre les bases de données NoSQL et SQL ainsi que quand et pourquoi utiliser chaque type de base de données.

Ania Kubow a développé ce cours. Ania est l'une des créatrices de tutoriels les plus populaires sur la chaîne YouTube freeCodeCamp.org.

Dans ce cours, vous allez revenir aux bases pour apprendre ce qu'est exactement un système de gestion de base de données (SGBD) et comment il est défini. Vous allez ensuite apprendre la conception de base de données et pourquoi elle est importante ainsi que ce qu'est un système de gestion de base de données.

Vous apprendrez ensuite les bases de données relationnelles suivies d'un cours accéléré sur SQL. Vous apprendrez les bases de données non relationnelles puis les avantages et inconvénients de l'utilisation des bases de données relationnelles par rapport aux bases de données non relationnelles. Enfin, vous apprendrez quelques cas d'utilisation suivis d'un cours accéléré sur NoSQL.

Voici les sections de ce cours :

* Qu'est-ce qu'une base de données
* Qu'est-ce qu'un système de gestion de base de données
* Démo : Création d'une base de données
* Modèles de base de données courants
* Bases de données relationnelles
* SQL
* Bases de données non relationnelles
* Avantages et inconvénients : Comparaison des SGBDR et NoSQL
* Base de données à colonnes larges
* Base de données documentaire
* Base de données clé-valeur
* Bases de données multi-modèles
* Cas d'utilisation : Quand utiliser les SGBDR ou NoSQL
