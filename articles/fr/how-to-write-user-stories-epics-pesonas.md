Les User Stories, les Epics et les Personas aident à organiser tous les besoins d'un projet afin qu'il soit clair ce qui doit être fait. Ils sont utilisés dans de nombreux emplois de développeur pour planifier des projets, il est donc utile de savoir ce qu'ils sont et comment les créer.

Dans cette vidéo, Anissa Deanna explique les user stories, les epics et les personas et donne des exemples de comment les créer pour vos propres projets.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=Fw98L-kcRpc) (8 minutes de visionnage).
