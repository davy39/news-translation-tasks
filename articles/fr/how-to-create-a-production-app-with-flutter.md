Flutter est un kit d'interface utilisateur de Google qui facilite la création d'applications compilées nativement pour mobile, web et bureau à partir d'une seule base de code.

Nous avons publié un cours intensif Flutter d'une heure qui vous apprendra à construire une application Flutter bien planifiée et entièrement testée à partir de zéro. Le cours vous guidera à travers la planification, la construction et les tests de l'application.

Ce cours a été développé par Tadas Petra de la chaîne Amateur Coder. Il a spécifiquement créé ce cours pour enseigner aux débutants complets comment construire une application prête pour la production.

De nombreux cours montrent simplement comment coder un programme mais sautent les phases de planification et de test. Ces phases sont tout aussi importantes que le codage du programme et vous apprendrez comment les réaliser dans ce cours.

Ce cours couvre :

* Planification
* Construction de l'application
* Tests unitaires
* Tests de widgets
* Tests d'intégration
* Maintenabilité

Regardez le cours sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/aiTTClKJbnw) (1 heure de visionnage).
