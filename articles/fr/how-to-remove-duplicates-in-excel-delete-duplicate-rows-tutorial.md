Une feuille de calcul Excel peut être difficile à utiliser si elle contient des doublons – surtout si vous n'êtes pas l'auteur de la feuille.

Si vous en trouvez, vous voudrez peut-être mettre en surbrillance ou supprimer ces doublons dans la feuille Excel pour éviter les erreurs.

Dans cet article, je vais vous montrer 2 façons de supprimer les doublons dans vos feuilles de calcul Excel.

## Comment trouver et supprimer les doublons avec la mise en forme conditionnelle

Si vous devez supprimer sélectivement certains doublons (mais en laisser d'autres), c'est la méthode à utiliser. Voici comment cela fonctionne :

**Étape 1** : Appuyez sur `CTRL` + `A` pour surligner l'ensemble du tableau, basculez vers l'onglet `Accueil`, puis cliquez sur « Mise en forme conditionnelle ».
![ss1](https://www.freecodecamp.org/news/content/images/2022/05/ss1.png)

**Étape 2** : Survolez « Règles de mise en surbrillance des cellules » et sélectionnez « Valeurs en double ».
![ss2](https://www.freecodecamp.org/news/content/images/2022/05/ss2.png)

**Étape 3** : Sélectionnez comment vous souhaitez que les doublons soient mis en surbrillance, puis cliquez sur « Ok ».
![ss3](https://www.freecodecamp.org/news/content/images/2022/05/ss3.png)

Les cellules en double seront maintenant mises en surbrillance pour que vous puissiez supprimer celles dont vous n'avez pas besoin :
![ss4](https://www.freecodecamp.org/news/content/images/2022/05/ss4.png)

Encore une fois, dans certains cas, vous aurez besoin de certaines entrées en double. Cette méthode ne supprime pas automatiquement les doublons, vous pouvez donc le faire vous-même.

La méthode suivante supprime automatiquement les doublons.

## Comment supprimer les doublons avec la commande Supprimer les doublons

C'est l'une des méthodes les plus fiables et les plus simples pour supprimer les doublons.

**Pour utiliser cette méthode, suivez les étapes ci-dessous :**
**Étape 1** : Passez à l'onglet `Données` et cliquez sur « Supprimer les doublons ».
![ss5](https://www.freecodecamp.org/news/content/images/2022/05/ss5.png)

**Étape 2** : Sélectionnez toutes les colonnes du tableau pour qu'Excel puisse les parcourir et vérifier les doublons, puis cliquez sur « Ok ».
![ss6](https://www.freecodecamp.org/news/content/images/2022/05/ss6.png)

**Étape 3** : Vous recevrez un message indiquant que les doublons ont été supprimés. Cliquez sur « Ok » pour vous débarrasser du message.
![ss7](https://www.freecodecamp.org/news/content/images/2022/05/ss7.png)
![ss8](https://www.freecodecamp.org/news/content/images/2022/05/ss8.png)
