Il est essentiel pour les développeurs web frontend de comprendre comment utiliser le CSS. C'est ce qui est utilisé pour styliser un site web.

Nous venons de publier un cours complet sur le CSS sur la chaîne YouTube de freeCodeCamp.org.

Dans ce cours approfondi, vous apprendrez toutes les fonctionnalités clés du CSS. C'est le cours le plus complet sur le CSS que nous ayons publié à ce jour. Donc, si vous voulez devenir un expert en Cascading Style Sheets, ce cours est fait pour vous.

Dave Gray a créé ce cours. Dave Gray est un développeur expérimenté et un excellent enseignant. Nous avons déjà publié un cours populaire sur le HTML de Dave et ce cours est une excellente suite après avoir appris le HTML.

Voici tous les sujets abordés dans ce cours :

* Sélecteurs
* Couleurs
* Unités et tailles
* Modèle de boîte
* Typographie
* Stylisation des liens
* Styles de liste
* Mini projet
* Affichage
* Flottants
* Colonnes
* Position
* Flexbox
* Grille de mise en page
* Images
* Requêtes média
* Projet de carte
* Pseudo
* Variables
* Fonctions
* Animations
* Organisation
* Projet final

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/OXGznpKZ_sA) (11 heures de visionnage).
