Depuis les premiers jours du web sauvage, JavaScript a toujours été présent. Il a évolué d'un simple hack réalisé en deux semaines à un écosystème vibrant qui ne cesse d'évoluer vers le web moderne.

Dans cette conférence, Matt Hippely retrace la chronologie de l'évolution de JavaScript, depuis ses débuts jusqu'à ce qu'il est devenu aujourd'hui. Il utilise des applications de démonstration pour montrer comment les choses étaient construites en JavaScript et comment elles le sont aujourd'hui. Il démontre également comment écrire du JavaScript conforme aux pratiques modernes.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=CseCDFed458) (1 heure de visionnage).
