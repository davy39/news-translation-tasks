Si vous avez été à la recherche d'un cours complet qui plonge en profondeur dans le monde des tests et de l'automatisation des API, votre recherche s'arrête ici. Nous venons de publier un nouveau cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à utiliser Postman pour écrire et automatiser des tests d'API.

Le cours est conçu et curaté par Valentin Despa. Il est un développeur expérimenté et a créé de nombreux cours populaires autour de Postman.

### Aperçu du cours

**Unité 1 - Introduction aux tests d'API** : Le cours commence par établir une base solide en matière de tests d'API. Ici, vous serez introduit au concept des API, à leur test, et à la manière de tester manuellement les API en utilisant Postman. Vous apprendrez à gérer les secrets avec les variables Postman, à écrire des scripts et à les déboguer en utilisant la console Postman. L'unité explore également les bases de JavaScript, et comment accéder et passer des données entre les requêtes.

**Unité 2 - Écriture de tests d'API** : Dans cette unité, vous explorerez la structure des tests d'API, ferez des assertions sur le code de statut, et manipulerez le format JSON. De l'assertion du type de données d'une propriété à la validation des schémas JSON et aux tests des en-têtes de réponse, cette unité est une exploration approfondie des tests d'API.

**Unité 3 - Premiers pas en automatisation** : Cette unité vous guide à travers les premières étapes de l'automatisation. Vous apprendrez à utiliser le collection runner, les exécutions planifiées et le CLI de Postman. C'est une partie passionnante du cours qui démontre comment exécuter une collection en utilisant le CLI de Postman.

**Unité 4 - Intégration des tests Postman dans CI/CD** : La dernière unité plonge au cœur de l'intégration continue et du déploiement continu (CI/CD). De l'apprentissage des fournisseurs CI/CD à l'exécution des tests Postman avec GitHub Actions et d'autres outils CI/CD, cette unité est la dernière pierre de votre parcours pour devenir un testeur d'API compétent.

Au fur et à mesure que vous progressez dans le cours, vous serez présenté à des devoirs pour consolider votre compréhension et gagner de l'expérience pratique. À la réussite du cours, vous aurez l'opportunité de réclamer votre badge en témoignage de vos nouvelles compétences.

Ce cours est une excellente opportunité pour ceux qui cherchent à se spécialiser dans les tests d'API, en exploitant la puissance de Postman pour l'automatisation des tests. Que vous soyez un débutant absolu ou un professionnel expérimenté, ce cours est conçu pour offrir de la valeur à tous les niveaux.

Faites votre premier pas vers la maîtrise des tests d'API avec Postman et rejoignez-nous sur la [chaîne YouTube de freeCodeCamp](https://youtu.be/zp5Jh2FIpF0) dès aujourd'hui pour ce cours de 4 heures.
