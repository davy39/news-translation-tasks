#### Comment passer d'un utilisateur hésitant à un utilisateur confiant

![Image](https://cdn-media-1.freecodecamp.org/images/yOdP1Z2NyjFQgxCQU7lCDfvPvKdLOXU9IXyP)

d3.js est une bibliothèque fantastique — mais je peux vous dire honnêtement que pendant longtemps, je l'ai utilisée sans la comprendre pleinement. N'ayant aucun modèle mental solide de ce avec quoi je travaillais, j'avoue que je tendais à copier/coller des morceaux de code à partir de divers extraits sur [bl.ocks.org](http://bl.ocks.org), à croiser les doigts et à espérer le meilleur.
