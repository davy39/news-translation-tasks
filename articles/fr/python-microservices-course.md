Utiliser des microservices Python vous permet de diviser vos applications en parties plus petites qui communiquent entre elles. Cela peut simplifier la mise à l'échelle de l'application en fonction du trafic. De plus, la séparation des préoccupations facilite le travail sur une seule partie de l'application à la fois.

Nous avons publié un cours sur la chaîne YouTube freeCodeCamp.org qui vous aide à apprendre les microservices Python en vous enseignant à construire une application web utilisant Django, Flask et React.

Antonio de Scaleable Scripts a créé ce cours. Antonio a beaucoup d'expérience dans la création de cours de logiciels.

Voici les sujets abordés dans ce cours :

* Qu'est-ce que les Microservices ?
* Introduction au Projet
* Installation de Django
* Ajout des Fichiers Docker
* Connecter Django avec MySQL avec Docker
* Modèles et Serializers
* Rest APIs avec ViewSets
* Point d'Accès Utilisateur avec APIView
* Installation de Flask avec Docker
* Connecter Flask avec MySQL
* Modèles
* Migrations Flask
* RabbitMQ
* Producteur et Consommateur Django
* Producteur et Consommateur Flask
* Service de File d'Attente
* Cohérence des Données
* Requêtes Http Internes
* Finalisation des Applications Python
* Installation de React
* Produits CRUD
* Complétion de l'Application Principale
