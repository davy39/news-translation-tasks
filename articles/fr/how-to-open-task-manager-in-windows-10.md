Le Gestionnaire des tâches de Windows est un outil utilitaire avancé qui vous aide à gérer vos applications en cours d'exécution. 

Le Gestionnaire des tâches vous permet de voir quelles applications sont ouvertes et lesquelles vous utilisez. Vous pouvez également voir quelles applications s'exécutent en arrière-plan que vous n'avez pas ouvertes vous-même. 

Une raison importante pour laquelle vous pourriez vouloir ouvrir le Gestionnaire des tâches est d'arrêter les applications qui rendent Windows non réactif. 

Il existe plusieurs façons d'ouvrir le Gestionnaire des tâches. Dans cet article, je vais vous montrer 7 façons d'ouvrir le Gestionnaire des tâches sur Windows 10.

## 1. Comment ouvrir le Gestionnaire des tâches avec la recherche Windows 

La manière la plus simple de trouver quoi que ce soit sur un ordinateur Windows est de le rechercher. Le Gestionnaire des tâches n'est pas une exception. 

- Cliquez sur l'icône de recherche, puis tapez "Gestionnaire des tâches" dans la barre de recherche. 
- Le Gestionnaire des tâches apparaîtra comme résultat de recherche, et vous pourrez l'ouvrir à partir de là.
 
![search](https://www.freecodecamp.org/news/content/images/2021/09/search.png)

## 2. Comment ouvrir le Gestionnaire des tâches avec des raccourcis clavier

Certaines applications peuvent devenir incontrôlables et empêcher Windows de répondre, vous pourriez donc ne pas avoir accès à la recherche Windows dans de telles situations.

Windows dispose de deux combinaisons de touches différentes avec lesquelles vous pouvez ouvrir le Gestionnaire des tâches si vous souhaitez voir les applications en cours d'exécution ou arrêter une application.

- Vous pouvez ouvrir le Gestionnaire des tâches en appuyant sur `Ctrl + Shift + Esc` simultanément. Cela ouvre directement le Gestionnaire des tâches. 
- Vous pouvez également ouvrir le Gestionnaire des tâches à partir de l'écran GINA en appuyant et en maintenant `Ctrl + Alt + Del`.

Certaines applications peuvent vous empêcher d'accéder au Gestionnaire des tâches avec Ctrl + Shift + Esc, mais appuyer sur `Ctrl + Alt + Del` ouvrira toujours l'écran d'identification et d'authentification graphique (GINA) pour que vous puissiez sélectionner et ouvrir le Gestionnaire des tâches à partir de là.

![gina-screen](https://www.freecodecamp.org/news/content/images/2021/09/gina-screen.jpg)

## 3. Comment ouvrir le Gestionnaire des tâches via le Panneau de configuration

Vous pouvez utiliser l'option de recherche du Panneau de configuration pour ouvrir le Gestionnaire des tâches. 

Tapez "Gestionnaire des tâches" dans la barre de recherche et vous verrez un lien vers le Gestionnaire des tâches sous "Système".

![ControlPanel](https://www.freecodecamp.org/news/content/images/2021/09/ControlPanel.png)

## 4. Comment ouvrir le Gestionnaire des tâches via le menu Démarrer

Beaucoup de gens préfèrent ouvrir les applications à partir du menu Démarrer. Vous pouvez également ouvrir le Gestionnaire des tâches à partir de là.

- Cliquez sur l'icône Windows ou appuyez sur la touche `WIN` du clavier.
- Faites défiler vers le bas et ouvrez le dossier système Windows
- Le Gestionnaire des tâches sera affiché aux côtés de certaines autres applications – il suffit de le sélectionner là pour l'ouvrir.
 
![startMenu](https://www.freecodecamp.org/news/content/images/2021/09/startMenu.png)

## 5. Comment ouvrir le Gestionnaire des tâches avec la commande Exécuter

De nombreuses applications et dossiers peuvent être ouverts avec la boîte de dialogue Exécuter sur Windows 10. Il existe également une commande pour ouvrir le Gestionnaire des tâches à partir de là.

- Appuyez sur les touches `WIN + R` du clavier pour ouvrir la boîte de dialogue Exécuter
- Tapez "taskmgr" et cliquez sur "Ok" pour ouvrir le Gestionnaire des tâches.

![runCommand](https://www.freecodecamp.org/news/content/images/2021/09/runCommand.png)

## 6. Comment ouvrir le Gestionnaire des tâches à partir de la barre des tâches

Vous pouvez également ouvrir le Gestionnaire des tâches à partir de la barre des tâches. 

Faites un clic droit sur un espace vide dans la barre des tâches, puis sélectionnez "Gestionnaire des tâches".

![taskbar](https://www.freecodecamp.org/news/content/images/2021/09/taskbar.png)

## 7. Comment ouvrir le Gestionnaire des tâches à partir du menu Utilisateur avancé de Windows

En plus des 6 méthodes que nous avons déjà vues, vous pouvez ouvrir le Gestionnaire des tâches en faisant un clic droit sur le logo Windows, puis en sélectionnant "Gestionnaire des tâches" dans le menu qui apparaît.

Vous pouvez également y accéder en appuyant et en maintenant les touches `Win + X` du clavier, puis en sélectionnant "Gestionnaire des tâches" dans le menu.

![powerUser](https://www.freecodecamp.org/news/content/images/2021/09/powerUser.png)

## Conclusion

Le Gestionnaire des tâches de Windows est un outil très puissant qui peut grandement augmenter votre productivité. Avec lui, vous pouvez voir quelles applications sont en cours d'exécution et arrêter celles qui causent des problèmes. 

Dans cet article, vous avez appris plusieurs façons d'ouvrir le Gestionnaire des tâches sur Windows 10 afin de pouvoir avoir une meilleure expérience en l'utilisant.
