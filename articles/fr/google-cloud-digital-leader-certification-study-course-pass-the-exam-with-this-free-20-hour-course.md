## Qu'est-ce que le Google Cloud Associate Cloud Engineer ?

L'Associate Cloud Engineer, également communément appelé ACE, est la certification de niveau associé de Google Cloud.

La différence clé entre l'Associate Cloud Engineer et le Google Cloud Digital Cloud Leader est que l'Associate Cloud Engineer est une certification pratique nécessitant une expérience pratique des services principaux de Google Cloud.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/google-cloud-roadmap.png)

Google Cloud n'a qu'une seule certification de niveau associé. Cela diffère d'AWS qui en a trois, et d'Azure qui a plus de six certifications de niveau associé.

N'avoir qu'une seule certification de niveau associé signifie que vous êtes censé connaître plus d'informations par rapport, par exemple, à l'AWS Solutions Architect Associate.

Cependant, l'Associate Cloud Engineer reflète mieux, tant dans le titre basé sur le rôle que dans les connaissances, les emplois de niveau débutant dans l'industrie du cloud.

## Qu'est-ce qu'un Cloud Engineer ?

Un Cloud Engineer est le rôle IT/DevOps le plus populaire dans l'industrie du cloud. Un Cloud Engineer est un rôle de niveau débutant où vous êtes chargé de concevoir, mettre en œuvre et maintenir l'infrastructure cloud.

Ce rôle cloud vous permet de faire un peu de tout et à partir de là, vous pouvez orienter votre carrière cloud en vous spécialisant dans un rôle avancé tel que Solutions Architect, DevOps Engineer, Data Engineer et bien d'autres.

Pour obtenir un rôle cloud, il est recommandé de détenir une certification de niveau associé d'un fournisseur de services cloud (CSP) comme Google Cloud ainsi que de construire vos projets cloud à présenter dans votre portfolio.

## Aperçu du Google Cloud Associate Cloud Engineer

Le guide d'examen divise les questions d'examen en cinq sections suivantes.

* Section 1. Configuration d'un environnement de solution cloud
* Section 2. Planification et configuration d'une solution cloud
* Section 3. Déploiement et mise en œuvre d'une solution cloud
* Section 4. Assurer le bon fonctionnement d'une solution cloud
* Section 5. Configuration de l'accès et de la sécurité

## **Comment obtenir la certification ?**

Google utilise [Kryterion](https://www.freecodecamp.org/news/p/bc0cbbc3-1a26-43ac-a07c-e158c256003e/Kryterion) comme centre de test. Vous pouvez passer l'examen en personne ou en ligne.

Il y a 50 questions à choix multiples et à sélection multiple et vous devez obtenir 70 % pour réussir.

Vous avez 2 heures pour compléter ce cours.

Le Google Cloud Associate Cloud Engineer coûte 125 USD.
