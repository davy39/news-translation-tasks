Vous souhaitez créer une interface utilisateur graphique pour votre programme Python ? Vous devriez apprendre à utiliser Tkinter ! Tkinter est le moyen le plus courant d'ajouter des interfaces graphiques aux programmes Python.

Tkinter fournit une interface orientée objet pour le toolkit GUI Tk intégré à Python. Vous n'avez même pas besoin d'installer quoi que ce soit d'extra pour utiliser Tkinter.

Tkinter simplifie la création d'une interface graphique qui gère les entrées et sorties utilisateur. Une interface graphique utilise une forme de programmation orientée objet appelée "événementielle". Cela signifie que le programme répond à des événements, qui sont des actions effectuées par l'utilisateur.
