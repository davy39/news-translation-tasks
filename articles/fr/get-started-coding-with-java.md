Java est l'un des langages de programmation les plus populaires. Il peut fonctionner sur tous les principaux systèmes d'exploitation et peut être utilisé pour écrire des applications Android natives.

Nous avons publié un cours de Java pour débutants sur la chaîne YouTube de freeCodeCamp.org.

Ce cours a été développé par Matt Speake. Matt programme depuis plus de 20 ans. Il a de nombreux cours populaires sur Udemy et d'autres plateformes.

Dans ce cours, Matt vous enseignera les fondamentaux suivants de Java :

* un **aperçu de la plateforme Java** - pourquoi elle est si populaire et qui l'utilise.
* une **compréhension des composants de la plateforme Java** - vous découvrirez l'indépendance de la plateforme et apprendrez la signification de JVM, JDK et JRE.
* vous **installerez le Java Development Kit** et **vous préparerez à l'utiliser** - c'est sur quoi tous les programmes Java sont construits.
* vous **installerez un IDE (Eclipse)** et apprendrez à l'utiliser pour programmer en Java.
* vous vous **familiariserez avec la syntaxe de programmation Java** en morceaux faciles à assimiler qui sont basiques mais pratiques.
* vous **explorerez l'écriture de code Java très adapté aux débutants dans l'IDE**.

Après le cours, vous aurez la confiance nécessaire pour continuer et approfondir vos connaissances en Java.
