Angular est l'un des frameworks web les plus populaires.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra comment créer un projet Angular 11 à partir de zéro. Il utilise une API publique pour créer un site web de base de données de jeux. Vous acquerrez d'excellentes bases pour construire des applications Angular.

Ce cours a été créé par Slobodan Gajic. Slobodan utilise Angular depuis des années et explique très bien comment tout s'articule.

Vous apprendrez comment configurer un projet à l'aide d'Angular CLI, comment créer des composants, comment effectuer des appels http, comment implémenter des intercepteurs http et comment créer des services.

Vous apprendrez également comment configurer des routes pour votre application et comment transmettre des données avec des routes entre les composants. Vous apprendrez à utiliser les pipes et diverses directives Angular comme ngFor, ngIf, et plus encore.

Voici toutes les sections de ce cours :

* Installation du projet et des modules
* Composant de barre de recherche
* Stylisation de la barre de recherche
* Création du composant d'accueil (home component)
* Création de routes
* Stylisation du composant d'accueil
* Création du service http
* Création de modèles TypeScript
* Création d'intercepteurs http
* Implémentation du composant d'accueil
* Implémentation du menu de tri
* Création du composant de détails
* Stylisation du composant de détails
* Composant d'onglets de détails
* Stylisation des onglets
* Implémentation des données dans les onglets
* Revue finale

Regardez le cours complet ci-dessous ou [sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=LiOzTQAz13Q) (75 minutes de visionnage).
