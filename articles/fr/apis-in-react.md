Apprenez à utiliser des APIs externes avec React et React Router dans un tutoriel complet de Hamza Mirza. Ce tutoriel montre comment créer une application de recherche de recettes dans React.

L'application récupère des recettes depuis une API externe et permet aux utilisateurs de rechercher des recettes spécifiques. Les concepts que vous apprendrez dans ce tutoriel sont utiles pour une grande variété de projets.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/tvfeBLMA_Q4) (1,5 heure de visionnage).
