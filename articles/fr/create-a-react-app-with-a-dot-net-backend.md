ASP.NET Core peut bien fonctionner comme backend pour vos applications React.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer une application React basique qui utilise un composant .NET Web API écrit en C#.

Afin d'intégrer React avec un composant .NET Web API, ce cours utilise le modèle de projet 'ASP.NET Core avec React.js' disponible dans la version gratuite de Visual Studio 2022 (l'édition communautaire). L'application que vous allez construire permet aux utilisateurs de classer des éléments en les faisant glisser et en les déposant (affichés dans une liste) vers une position de cellule sur la grille de classement (la grille est positionnée au-dessus de la liste des éléments pertinents).

Chaque position de cellule sur la grille désigne une valeur de classement. Par exemple, la cellule en haut à gauche est la position numéro 1 désignant l'élément le mieux classé et la cellule positionnée immédiatement à droite de cette cellule désigne l'élément classé 2ème. Cela continue jusqu'à la cellule en bas à droite qui est la pire position de classement possible.

Coder ce projet vous donnera un aperçu de la manière de construire une application simple en utilisant React en front-end qui exploite un composant .NET Web API en backend.

Gavin Lon a créé ce cours. Il est un développeur logiciel et enseignant expérimenté.

Voici toutes les sections couvertes dans ce cours :

* Création du projet
* Créer le code .NET Web API côté serveur en C#
* Créer le composant React 'RankItems'
* Créer la fonctionnalité de navigation
* Écrire le code pour importer les images de films
* Écrire le code pour afficher les images de films
* Écrire le code pour le composant React 'RankingGrid'
* Écrire le code pour la fonctionnalité de glisser-déposer
* Écrire le code pour classer les albums
* Écrire le code pour initialiser les éléments de classement (bouton de rechargement)
