## Comment commencer en tant que développeur web front-end

![Image](https://www.freecodecamp.org/news/content/images/2021/02/frontend.png align="left")

*Source :* [GitHub](https://github.com/kamranahmedse/developer-roadmap)

Ici, vous pouvez voir une feuille de route générale pour les développeurs web front-end. Si vous ne l'avez jamais vue auparavant, lisez-la pour avoir une idée du chemin global.

Il y a beaucoup de choses à assimiler – mais dans cet article, je vais vous aider à créer un chemin plus court pour apprendre le développement web.

Avant de plonger dans les compétences que vous devez apprendre pour devenir développeur web, vous devez savoir que le développement web n'est pas un concept ou un sujet unique. Il comporte de nombreux sous-domaines.

Vous pouvez vous concentrer sur le développement front-end, le développement back-end, la conception graphique, et ainsi de suite. Nous allons commencer par le front-end maintenant.

Pour le développement front-end, vous devez apprendre HTML, CSS, JavaScript et une bibliothèque ou un framework comme React, Angular ou Vue.js.

Alors maintenant, parlons plus en détail de ce qu'est le développement front-end et de ce que vous devez apprendre.

### Qu'est-ce que le développement front-end ?

> Le front-end d'une application fait généralement référence à la couche qui représente l'interface utilisateur (UI). Cela peut inclure tout, d'un site statique avec HTML et CSS à une application React complète qui alimente l'UI. - [Colby Fayock](https://www.freecodecamp.org/news/front-end-developer-vs-back-end-developer-definition-and-meaning-in-practice/)

Il y a quelques [bases que vous devez connaître](https://www.freecodecamp.org/learn/responsive-web-design/) pour le développement web front-end. Elles sont :

1. HTML

2. CSS

3. JavaScript

HTML et CSS sont utilisés pour créer des sites web statiques.

### Qu'est-ce que HTML ?

HTML signifie Hypertext Markup Language. Il est utilisé pour ajouter du contenu à un site web. Les informations que vous voyez sur un site web particulier ne sont visibles que grâce à HTML. Vous pouvez exécuter du code HTML dans un IDE comme Visual Studio, Sublime Text, Atom, et bien d'autres.

HTML se compose de balises comme `<h1>` (pour les titres) et `<p>` (pour les paragraphes).

Par exemple :

```html
<!DOCTYPE html> 
<html>
<body>
  <h1>Bonjour, Lecteurs</h1>
  <p>Vous êtes prêts à partir.</p> 
</body>
</html>
```

Ceci est une syntaxe de base pour HTML. Nous devons écrire du code à l'intérieur des balises body pour l'afficher sur les pages web. Ici, nous affichons une balise h1 avec une balise de paragraphe.

Le résultat de ce code ressemblera à ceci :

![Image](https://www.freecodecamp.org/news/content/images/2021/02/ht.PNG align="left")

### Qu'est-ce que CSS ?

CSS est l'acronyme de Cascading Style Sheets. Vous l'utilisez pour créer des styles pour un site web afin qu'il ait l'air attrayant.

Encore une fois, inspectez le code HTML ci-dessus. Il n'affiche pas de couleur de fond ou d'ombre. Cela est dû au fait que nous n'avons pas encore appliqué de CSS.

Ainsi, CSS améliore HTML et indique aux éléments comment s'afficher sur les pages web.

CSS se compose principalement de 3 types :

1. CSS en ligne.

2. CSS interne.

3. CSS externe.

Pour vous aider à comprendre un peu mieux, nous allons utiliser le CSS en ligne. Le CSS en ligne est un type de CSS où vous pouvez ajouter des propriétés de style à l'intérieur des balises HTML.

Par exemple, ici nous avons ajouté une propriété de style à la balise body. Nous avons simplement changé la couleur de fond du body en rose :

```python
<!DOCTYPE html> 
<html>
<body style="background-color:pink;">
  <h1>Bonjour, Lecteurs</h1>
  <p>Vous êtes prêts à partir.</p>
</body>
</html>
```

Et voici le résultat :

N'est-ce pas cool ?

### Qu'est-ce que JavaScript ?

Alors, qu'en est-il des sites web dynamiques ? Tout cela est possible grâce à JavaScript.

Même Google a mentionné JavaScript comme le langage de programmation le plus populaire de nos jours.

![Image](https://www.freecodecamp.org/news/content/images/2021/02/java.png align="left")

*Source : Google*

JavaScript est un langage de programmation utilisé principalement côté client pour aider les pages web à être plus interactives.

Alors que HTML et CSS sont des langages qui donnent une structure et un style aux pages web, JavaScript fournit aux pages web des éléments interactifs qui engagent un utilisateur.

Expliquons comment cela fonctionne avec un exemple.

Pensez à votre maison. Construire les fondations et le cadre de votre maison avec des briques et du ciment fait partie de HTML. Peindre et décorer votre maison est le travail de CSS. Et intégrer la technologie – comme l'IoT – fait partie de JavaScript.

C'est tout. Le développement web front-end se compose de HTML, CSS et JavaScript.

Une fois que vous connaissez les bases, vous pouvez ajouter des frameworks et des bibliothèques comme React, Angular ou Vue.js à votre ensemble de compétences.

**Ce sont les compétences de base dont vous avez besoin pour commencer en tant que développeur web front-end.**

## Comment apprendre le développement back-end

![Image](https://www.freecodecamp.org/news/content/images/2021/02/backend.png align="left")

*Source :* [GitHub](https://github.com/kamranahmedse/developer-roadmap)

Maintenant, parlons du développement back-end.

Pour le développement back-end, vous devrez apprendre Node.js, Express, MongoDB et Mongoose.

### Mais qu'est-ce que le développement back-end ?

> Le développement back-end est également connu sous le nom de développement côté serveur. Il s'agit de la pratique de communication entre la base de données et le navigateur.

Les langages et outils de développement back-end incluent :

1. Node.js

2. Express

3. MongoDB (y compris Mongoose)

Ces trois compétences sont importantes à apprendre si vous voulez devenir développeur back-end.

### Qu'est-ce que Node.js ?

Selon la [documentation de Node](https://nodejs.dev/learn), Node.js est un environnement d'exécution JavaScript open-source et multiplateforme. C'est un outil populaire que vous pouvez utiliser pour pratiquement tout type de projet.

Node.js exécute le moteur JavaScript V8, le cœur de Google Chrome, en dehors du navigateur. Cela aide Node.js à être très performant.

### Maintenant, qu'est-ce qu'Express ?

> Express est un framework d'application web Node.js minimal et flexible qui fournit un ensemble robuste de fonctionnalités pour les applications web et mobiles. - [Documentation d'Express](https://expressjs.com/)

Vous pouvez penser à Express comme un package NPM qui aide les applications Node.js.

Permettez-moi d'expliquer comment cela fonctionne à travers du code. Le code ci-dessous est la syntaxe de base pour Express.

```python
const express = require('express');
const app = express();
const port = 5000;

app.get('/', (req, res) => {
  res.send("Bonjour, Monde")
})

app.listen(port, () => {
  console.log("L'application est en cours d'exécution sur le port 5000");
})
```

Ici, nous installons puis nécessitons Express. Vous initialisez simplement l'application en tant que `express()` et le port en tant que `5000`.

Ensuite, nous demandons la route `'/'` pour répondre avec Bonjour, Monde.

Lorsque notre application est en cours d'exécution, nous enregistrons qu'elle est en cours d'exécution sur le port 5000.

### Enfin, qu'est-ce que MongoDB ?

Avant de continuer, je dois expliquer un concept crucial : les bases de données.

Pour créer des sites web, vous avez besoin d'une base de données. Nous savons que les sites web nécessitent des bases de données pour stocker des informations. La base de données nous aide à stocker les informations des utilisateurs, à afficher des informations basées sur les demandes des utilisateurs, et bien plus encore.

Les bases de données se présentent sous deux types : SQL et NoSQL.

Quelle est la différence entre SQL et NoSQL ?

1. SQL est une base de données relationnelle, tandis que NoSQL est une base de données non relationnelle.

2. SQL est représenté sous forme de tables, tandis que NoSQL est sous forme de paires clé-valeur ou de format JSON.

3. Les bases de données SQL les plus populaires sont MySQL et PostgreSQL, tandis que la base de données NoSQL la plus populaire est MongoDB.

Maintenant, parlons de MongoDB. Selon [mongodb.com](https://www.mongodb.com/),

> MongoDB est une base de données générale, basée sur des documents, distribuée, construite pour les développeurs d'applications modernes et l'ère du cloud.

Lorsque vous devez travailler sur une base de données, MongoDB est une excellente option NoSQL.

### Vous devez également apprendre Mongoose.

Mongoose vous aide à travailler plus facilement avec MongoDB.

> [Mongoose](https://mongoosejs.com/) fournit une solution simple et basée sur un schéma pour modéliser les données de votre application. Il inclut le typage intégré, la validation, la construction de requêtes, les hooks de logique métier, et plus encore, prêts à l'emploi.

Ce sont les technologies de base que vous devez apprendre pour devenir développeur back-end.

Une fois que vous avez appris toutes ces compétences, vous pouvez vous appeler développeur web Full Stack.

Au fait, vous avez peut-être entendu des gens parler de certaines piles technologiques, comme "MERN" ou "MEAN".

Eh bien, si vous choisissez d'apprendre React.js, vous serez un développeur web Full Stack MERN (MongoDB, Express, React, Node).

Avec Angular.js, vous serez un développeur web Full Stack MEAN (MongoDB, Express, Angular, Node).
