## Apprendre Python - Cours Complet pour Débutants

Dans ce [Cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=rfscVS0vtbw), vous apprendrez les bases de la programmation telles que les listes, les conditionnelles, les chaînes de caractères, les tuples, les fonctions, les classes et bien plus encore.

Vous construirez également plusieurs petits projets comme une calculatrice basique, un jeu de mad libs, une application de traduction, et un jeu de devinettes.

## Programmation pour Tous (Démarrer avec Python)

Dans ce [cours de l'Université du Michigan](https://www.freecodecamp.org/news/python-for-everybody/), vous apprendrez les fonctions, les boucles, les conditionnelles, les variables et bien plus encore avec le célèbre "Dr. Chuck".

Une fois que vous aurez appris les bases, vous pourrez continuer avec le reste de la [spécialisation](https://www.coursera.org/specializations/python) et suivre le [cours sur les Structures de Données en Python](https://www.coursera.org/learn/python-data?specialization=python), le [cours sur l'Utilisation de Python pour Accéder aux Données Web](https://www.coursera.org/learn/python-network-data?specialization=python), et le [cours sur l'Utilisation des Bases de Données avec Python](https://www.coursera.org/learn/python-databases?specialization=python).

## Le Manuel Python

Dans [ce livre de Flavio Copes](https://www.freecodecamp.org/news/the-python-handbook/), vous apprendrez les chaînes de caractères, les listes, les tuples, la récursivité, et bien plus encore.

Vous apprendrez également comment installer des packages tiers et comment travailler avec des environnements virtuels.

## Tutoriels Python pour Débutants Absolus par CS Dojo

Dans cette série de [vidéos YouTube pour Débutants en Python par CS Dojo](https://www.youtube.com/playlist?list=PLBZBJbE_rGRWeh5mIBhD-hhDwSEDxogDg), vous apprendrez les dictionnaires, les boucles, les fonctions, les objets et bien plus encore.

Vous apprendrez également [comment créer un bot Twitter en utilisant Python](https://www.youtube.com/watch?v=W0wWwglE1Vc&list=PLBZBJbE_rGRWeh5mIBhD-hhDwSEDxogDg&index=15).

## Tutoriel Python - Python pour Débutants

Dans ce [cours YouTube de Programming with Mosh](https://www.youtube.com/watch?v=_uQrJ0TkZlc), vous apprendrez les bases de la programmation comme les variables, les boucles, les chaînes de caractères, les tuples, les fonctions, les classes et bien plus encore.

Vous construirez également trois projets : un sur l'automatisation, un sur l'apprentissage automatique, et un sur la création d'un site web avec Django.

Pour une pratique supplémentaire, vous pouvez suivre les [53 Exercices Python pour Débutants de Mosh](https://programmingwithmosh.com/python/python-exercises-and-questions-for-beginners/).

## Cours Python de Studytonight

Sur [Studytonight](https://www.studytonight.com/python/), vous apprendrez les bases de Python, la gestion des erreurs, la POO, la gestion des fichiers, les types de données complexes, le Multithreading et bien plus encore.

Une fois que vous aurez appris les bases, vous pourrez passer à leurs autres modules qui incluent la [bibliothèque NumPy](https://www.studytonight.com/numpy), [Matplotlib](https://www.studytonight.com/matplotlib), [Tkinter](https://www.studytonight.com/tkinter), la [Programmation Réseau en Python](https://www.studytonight.com/network-programming-in-python/), et le [Web Scraping avec Beautiful Soup](https://www.studytonight.com/python/web-scraping/).

## Cours Accéléré de Python pour Débutants

## Python Core

Dans le [Cours Python de SoloLearn](https://www.sololearn.com/learning/1073), vous apprendrez les chaînes de caractères, les variables, la POO, la programmation fonctionnelle et bien plus encore. Il y a de nombreux quiz, défis et projets que vous pouvez réaliser en cours de route.

Pour commencer, vous devrez créer un compte gratuit.

## Bases de Python avec Sam

Dans [cette série YouTube de Sam Focht](https://www.youtube.com/playlist?list=PLWKjhJtqVAbkmRvnFmOd4KhDdlK1oIq23), vous apprendrez les boucles, les fonctions, les chaînes de caractères, la récursivité et bien plus encore.

Vous construirez également plusieurs projets, dont un jeu de devinettes, une liste de courses, un jeu de plateau, et un générateur de mots de passe aléatoires.

## Tutoriels Python pour Débutants

Dans cette [série de tutoriels pour débutants de Pythonspot](https://pythonspot.com/beginner/), vous apprendrez les types de données, les tuples, les objets, les classes, les dictionnaires et bien plus encore. Une fois que vous aurez couvert les bases, vous pourrez passer à l'[apprentissage automatique](https://pythonspot.com/machine-learning/), aux [bases de données](https://pythonspot.com/python-database/), et aux [interfaces graphiques](https://pythonspot.com/gui/).

## Tutoriel Python
