Apprenez toutes les bases pour construire des applications web dynamiques et interactives avec la bibliothèque JavaScript React dans ce cours complet de Bob Ziroll (@bobziroll).

À la fin de ce cours, vous saurez tout ce dont vous avez besoin pour construire des applications web avec React.

Le cours couvre :

* JSX
* Props et état
* Rendu conditionnel
* Stylisation des composants
* Méthodes du cycle de vie
* Récupération de données depuis une API
* Gestion des événements
* Formulaires et composants contrôlés
* Écriture de code React moderne
* Configuration d'un environnement

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=DLX62G4lc44) (5 heures de visionnage).
