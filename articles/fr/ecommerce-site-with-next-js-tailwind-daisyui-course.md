Lorsque vous apprenez à construire une application web, il peut être utile de voir comment tout fonctionne ensemble, plutôt que d'apprendre un grand nombre de technologies individuellement.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire et déployer un site web e-commerce full-stack avec Next.js 13.4 et un ensemble d'autres technologies importantes. Florian Walther de Coding in Flow. Il est un développeur expérimenté et a créé de nombreux cours populaires.

Dans ce cours, vous apprendrez :

* Comment utiliser les actions serveur de Next.js et les appeler à partir de composants serveur et de composants client.
* Comment exécuter des opérations de base de données et des transactions avec Prisma et comment utiliser les extensions de client Prisma pour intégrer vos requêtes.
* Comment construire un site web e-commerce entièrement fonctionnel similaire à Amazon.com.
* Comment ajouter une authentification avec Next-Auth, l'adaptateur Prisma, MongoDB et la connexion Google.
* Comment implémenter des paniers d'achat anonymes pour les utilisateurs non authentifiés et stocker l'identifiant dans un cookie.
* Comment construire une interface utilisateur incroyable en utilisant TailwindCSS et DaisyUI.
* Comment déployer votre projet sur Vercel et configurer les métadonnées pour chaque page (y compris generateMetadata).
* Comment dédupliquer les requêtes Prisma en utilisant la fonction cache de React.
* Comment configurer des outils comme Prettier, Eslint, et les extensions Prisma et TailwindCSS pour VS Code afin de rendre votre projet facile à utiliser.

Vous acquerrez une connaissance approfondie de la manière de combiner un ensemble de technologies différentes pour effectuer chaque étape nécessaire à la construction et au déploiement d'une application web. Vous pouvez utiliser les compétences que vous apprenez dans ce cours dans vos propres projets.

Voici les sections de ce cours :

* Configuration du projet (#heading-installation)
* Configuration de Prisma + MongoDB
* Ajout de la page produit (Action serveur dans le composant serveur)
* Page de liste des produits
* Page des détails du produit (generateMetadata + cache React)
* Bouton d'ajout au panier (Action serveur dans le composant client)
* Barre de navigation + pied de page
* Page du panier
* Connexion de l'utilisateur (Next-Auth)
* Fusion du panier utilisateur avec le panier anonyme
* Pagination
* Fonctionnalité de recherche
* Extension Prisma
* Déploiement + aperçu social
