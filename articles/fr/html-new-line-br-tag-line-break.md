Ajouter des sauts de ligne dans votre HTML en utilisant la balise <br> est un moyen simple de rendre votre texte plus lisible. 

La balise <br> insérera un saut de ligne partout où vous la placerez dans votre code.

## Qu'est-ce que la balise <br> ?

La balise <br> est une balise auto-fermante qui insère un saut de ligne partout où elle est placée dans votre code. Pour utiliser la balise <br>, il suffit d'ajouter la balise <br> partout où vous voulez qu'un saut de ligne apparaisse dans votre code.

Voici un exemple de la façon dont la balise <br> peut être utilisée :

```
<p>Ceci est un paragraphe.<br>
Ceci est une autre ligne dans le même paragraphe.</p>
```

Dans l'exemple ci-dessus, la balise <br> est utilisée pour insérer un saut de ligne entre les deux lignes de texte, tout cela dans le même élément <p>.

Voici à quoi ressemblera le texte :

> Ceci est un paragraphe.  
> Ceci est une autre ligne dans le même paragraphe.

Voici quelques conseils pour utiliser les balises <br> dans le code HTML. Notez que ce sont des approches "rapides et sales" pour faire en sorte que le texte ait l'apparence que vous souhaitez. À mesure que vous vous améliorez avec HTML, vous voudrez utiliser CSS pour styliser votre texte.

Conseil #1 : Utilisez les balises <br> pour créer des sauts de ligne dans le texte, au lieu d'utiliser plusieurs paragraphes ou d'autres éléments. Cela peut être plus simple et plus facile à maintenir.

Conseil #2 : Utilisez les balises <br> pour créer un espace supplémentaire entre les lignes de texte.

Conseil #3 : Utilisez les balises <br> pour créer un espace supplémentaire entre les éléments sur une page. C'est exact – vous pouvez simplement forcer l'insertion d'un espace vertical vide en tapant `<br><br><br>`. Non recommandé, mais si vous êtes pressé, je ne vous jugerai pas.

Conseil #4 : Et la meilleure pratique la plus importante est, encore une fois, d'utiliser les balises <br> avec parcimonie, et seulement lorsque cela est nécessaire.
