### Projet #1 : Backend Node.js alimenté par [GraphQL](https://graphql.org/)

![Image](https://cdn-media-1.freecodecamp.org/images/1*xpqJwgYw9VJiLQqYOyUQEQ.gif)
_GraphQL et interface Graphiql_

[GraphQL](https://graphql.org/) a révolutionné la manière dont nous envoyons et recevons des données. Absolument digne de votre temps pour le découvrir.

#### Spécifications :

* Configurer un serveur Node.js express avec MongoDB et GraphQL.
* Capacité à créer, lire, mettre à jour, supprimer des données **_(CRUD)_**.
* Récupérer des données à partir d'un simple front-end.

Voici un tutoriel détaillé que j'ai écrit sur le sujet, profitez-en !

### Projet #2 : Application terminal

![Image](https://cdn-media-1.freecodecamp.org/images/1*RgFqiOfL3Q7s_yKNfIvxbA.png)
_[application terminal scope-chain-closures](https://github.com/workshopper/scope-chains-closures" rel="noopener" target="_blank" title=")_

Une application terminal est simplement une application qui s'exécute dans le terminal.

Essayez de commencer petit et créez une application simple qui s'exécute dans le terminal. Les mécanismes derrière les applications terminal sont très simples : vous donnez une entrée et obtenez une sortie _(I/O)_

#### Quelques idées très simples pour commencer :

* Demandez le nom de l'utilisateur et affichez les résultats.
* Affichez quelques équations mathématiques simples et laissez l'utilisateur entrer la valeur de l'équation.

[Voici une excellente liste agrégée d'applications terminal pour apprendre JavaScript.](https://nodeschool.io/)

### Projet #3 : Application de bureau [Electron](https://electronjs.org/)

![Image](https://cdn-media-1.freecodecamp.org/images/1*h1CnCpvq91HafVmXbm2GWA.png)

[Electron](https://electronjs.org/) vous permet de créer des applications de bureau natives avec des technologies web telles que HTML, CSS et JavaScript.

Certaines des applications Electron les plus connues sont : [**Visual Studio Code**](https://code.visualstudio.com/), [Atom](https://atom.io/), [**Slack**](https://slack.com/), et [**Webtorrent.io**](https://webtorrent.io/)

Encore une fois, commencez petit et construisez à partir de là. Par exemple, démarrez votre application Electron, créez une entrée et affichez la sortie dans votre terminal. Si vous êtes familier avec le développement web, Electron sera très conventionnel.

### Projet #4 : Clone d'Instagram

![Image](https://cdn-media-1.freecodecamp.org/images/1*ugjjIDXUIXO7qiyuWmV8gQ.png)

Instagram est une application complexe, et la (re)construire vous apprendra beaucoup. Certains des points forts de ce que vous apprendrez en construisant Instagram sont :

* Gestion de grille complexe (essayez d'utiliser la nouvelle grille CSS pour des points bonus)
* Appel de plusieurs points de terminaison d'API/gestion du flux de données (essayez d'utiliser GraphQL)
* Capacité à livrer des applications scalables et réelles.

N'hésitez pas à utiliser n'importe quelle technologie que vous jugez appropriée.

[Instagram est assez gentil pour nous fournir leur API.](https://www.instagram.com/developer/)
