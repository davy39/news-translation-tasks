Lorsque vous utilisez votre machine Windows, votre WiFi peut se déconnecter de manière inattendue. 

Cela peut être causé par divers problèmes tels qu'un pilote de carte réseau obsolète, un logiciel de routeur obsolète et une gestion optimisée de l'alimentation dans Windows 10.

Si votre WiFi se déconnecte sans cesse et que vous cherchez une solution, vous êtes au bon endroit. Dans cet article, je vais vous montrer 3 façons de réparer votre connexion WiFi.

## Comment empêcher le WiFi de se déconnecter en mettant à jour votre pilote de carte réseau 

Il est important d'avoir un pilote à jour pour que votre matériel fonctionne correctement, y compris le matériel externe. Donc, si vous mettez à jour votre adaptateur WiFi, cela pourrait empêcher votre WiFi de se déconnecter.

Suivez les étapes suivantes pour mettre à jour votre pilote de carte réseau :

**Étape 1** : Cliquez sur Démarrer (logo Windows) et recherchez "gestionnaire de périphériques". Cliquez sur le résultat de recherche "Gestionnaire de périphériques".
![ss-1-7](https://www.freecodecamp.org/news/content/images/2021/11/ss-1-7.jpg)

**Étape 2** : Développez l'option "Cartes réseau".

**Étape 3** : Recherchez votre adaptateur actuel, faites un clic droit dessus et sélectionnez "mettre à jour le pilote".
![ss-2-6](https://www.freecodecamp.org/news/content/images/2021/11/ss-2-6.jpg)

**Étape 4** : Sélectionnez "Rechercher automatiquement les pilotes". Windows recherchera maintenant sur Internet un pilote mis à jour et l'installera pour vous.
![ss-3-6](https://www.freecodecamp.org/news/content/images/2021/11/ss-3-6.jpg)

## Comment empêcher votre WiFi de se déconnecter en mettant à jour le logiciel de votre routeur

Chaque routeur dispose d'un logiciel qui le fait fonctionner correctement. 

Différents fabricants ont différentes interfaces utilisateur auxquelles vous pouvez toujours accéder en tapant l'adresse IP de votre routeur dans la barre d'adresse de votre navigateur.

Une fois connecté avec succès, vérifiez s'il y a une mise à jour disponible pour votre routeur et installez-la.
![ss-4](https://www.freecodecamp.org/news/content/images/2021/11/ss-4.png)

## Comment empêcher votre WiFi de se déconnecter en vérifiant les options de gestion de l'alimentation de votre carte réseau

Windows 10 est optimisé pour économiser la batterie. Une connexion Internet augmente la consommation d'énergie, donc parfois votre appareil peut être automatiquement configuré pour se déconnecter de temps en temps afin d'économiser de l'énergie.

Vous pouvez vérifier si votre carte réseau est configurée pour se déconnecter avec les étapes suivantes :

**Étape 1** : Faites un clic droit sur Démarrer et sélectionnez "Gestionnaire de périphériques".
![ss-5-5](https://www.freecodecamp.org/news/content/images/2021/11/ss-5-5.jpg)

**Étape 2** : Développez Cartes réseau.

**Étape 3** : Faites un clic droit sur l'adaptateur avec lequel vous rencontrez le problème et sélectionnez "Propriétés".
![ss-6-6](https://www.freecodecamp.org/news/content/images/2021/11/ss-6-6.jpg)

**Étape 4** : Cliquez sur l'onglet "Gestion de l'alimentation". Décochez "Autoriser l'ordinateur à éteindre ce périphérique pour économiser l'énergie".
![ss-7-2](https://www.freecodecamp.org/news/content/images/2021/11/ss-7-2.jpg)

## Conclusion

Cet article s'est concentré sur 3 façons de réparer votre WiFi s'il se déconnecte sans cesse.

En plus de ces solutions, vous pouvez également essayer d'autres solutions mineures telles que :

- Ne pas rester trop loin de votre routeur
- Limiter les appareils connectés
- Contacter votre fournisseur de services réseau
- Éviter les interférences radio provenants des téléphones, appareils Bluetooth, et autres.
