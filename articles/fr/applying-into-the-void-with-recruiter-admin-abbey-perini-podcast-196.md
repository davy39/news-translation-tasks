Abbey Perini a appris la programmation en autodidacte à l'âge de 27 ans alors qu'elle travaillait comme administratrice dans une agence de recrutement en ingénierie. Elle a travaillé de manière intensive avec de grandes bases de code héritées (legacy codebases) et a enseigné les meilleures pratiques à des développeurs à l'échelle internationale.

Nous discutons de :

* Comment être opérationnel rapidement avec une grande base de code héritée
    
* Comment faire en sorte que les employeurs se souviennent de vous et vous répondent réellement
    
* Comment elle s'est adaptée à son diagnostic de TDAH, reste concentrée et livre du code
    
* Comment le tricot et le cosplay lui donnent une perspective en tant que développeuse
