C# et ASP.NET sont utilisés comme langage de programmation côté serveur sur de nombreux sites web.

Nous avons publié un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer une application de base de données web dans ASP.NET Core en utilisant Visual Studio et le langage C#.

Le cours a été créé par Shad Sluiter. Shad est professeur d'informatique à la Grand Canyon University. Au cours des 20 dernières années, il a enseigné à des étudiants de la 6e année aux personnes âgées dans de nombreux aspects de la technologie.

L'application web que vous allez créer est une application CRUD de base qui interagit avec une base de données.

Ce tutoriel vous montrera comment :

* Utiliser le modèle de conception MVC (Modèle Vue Contrôleur).
* Configurer des tables de base de données en utilisant le framework Entity.
* Configurer des classes en tant que modèles.
* Personnaliser des formulaires Razor.
* Créer des méthodes à l'intérieur d'un contrôleur.
* Styliser une page ASP.NET avec CSS.
* Utiliser des classes Bootstrap.
* Créer une fonction de recherche pour filtrer les résultats de la base de données.

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://youtu.be/BfEjDD8mWYg) (1 heure de visionnage).
