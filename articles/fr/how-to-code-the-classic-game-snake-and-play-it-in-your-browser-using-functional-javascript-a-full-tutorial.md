Vous souvenez-vous du jeu Snake qui était préinstallé sur tous les téléphones Nokia dans les années 1990 ? Vous dirigiez un serpent sur votre écran, en essayant de le nourrir sans le faire s'écraser contre un mur ou contre son propre corps qui grandissait.

Eh bien, ce tutoriel de développement de jeu va vous montrer comment construire ce jeu classique Snake et le rendre jouable dans votre navigateur.

Vous n'utiliserez aucune bibliothèque, non plus. Juste du JavaScript standard. Et vous aurez également l'occasion d'apprendre quelques concepts de programmation fonctionnelle !

Christopher Okhravi est un étudiant suédois en doctorat qui a créé de nombreux excellents tutoriels de programmation sur YouTube. Il est excellent pour expliquer les concepts - un "conférencier de niveau boss" comme l'a appelé un commentateur.

Et Christopher a mis [tout le code source du tutoriel sur GitHub](https://github.com/chrokh/snake) pour votre référence.

Vous pouvez regarder sa présentation sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=bRlvGoWz6Ig) (93 minutes de visionnage).
