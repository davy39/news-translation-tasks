Presque tous les sites web du monde utilisent le HTML et le CSS. Vous devriez donc définitivement les apprendre si vous voulez créer des sites web.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire des sites web avec HTML et CSS.

Per Borgan a créé ce cours. Per est un développeur front-end et co-fondateur de Scrimba.

Ce cours vous apprendra le HTML et le CSS à travers la construction et le déploiement de cinq projets géniaux. Il est également entièrement interactif, avec plus de 75 défis de codage. Il n'y a pas de prérequis, donc il est adapté aux débutants absolus.

Ce cours contient 101 sections réparties en 6 modules. Voici les modules :

* _Module 1 :_ Construire et déployer votre premier site web
* _Module 2 :_ Construire un clone de Google.com
* _Module 3 :_ Construire une carte de visite numérique
* _Module 4 :_ Construire un site d'exploration spatiale
* _Module 5 :_ Construire un site de GIF d'anniversaire
* _Module 6 :_ Projet solo : Page d'accueil de votre ville natale

Ce cours est conçu pour les débutants absolus, donc il n'y a pas de prérequis, et vous n'avez besoin d'aucun outil autre que votre navigateur.

Les leçons sont amusantes et courtes. Et à la fin de ce cours, vous aurez construit et déployé plusieurs projets élégants que vous pourrez montrer à de potentiels employeurs.

Voici quelques-uns des sujets que vous apprendrez dans ce cours :

* Les bases du HTML
* Les éléments de texte
* Les balises d'ancrage
* Les éléments interactifs
* Les divs et les spans
* Les attributs
* L'imbrication
* Les listes
* Les images
* La structure du document
* Les bases du CSS
* Les classes et les identifiants
* Inline vs block
* Centrer les éléments
* Les bordures
* Le modèle de boîte
* Marges et espacements
* L'héritage
* Flexbox
* Les ombres de texte
* Les images de fond
* Utiliser des polices
* Les palettes de couleurs
* Les dégradés
* Les outils de développement Chrome
* Le débogage avec Google
* Les systèmes de contrôle de version
* Le déploiement sur le web
* Utiliser des outils de design
* \u2026 et bien plus encore !
