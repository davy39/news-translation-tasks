La pile PERN se compose de PostgreSQL, Express, React et Node.js. Ces technologies peuvent être combinées pour construire une application web full-stack avec des opérations CRUD.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui enseigne comment utiliser la pile PERN. Le cours a été développé par Henry Ly de The Stoic Programmers.

Henry démontrera comment utiliser la pile PERN pour construire une liste de tâches. À la fin du cours, vous aurez une meilleure compréhension de la manière dont ces technologies fonctionnent ensemble et vous serez prêt à créer des applications plus complexes.

Voici ce que couvre ce cours :

* Démarrage de notre serveur
* Création de la base de données et de la table PostgreSQL
* Connexion de la base de données et du serveur
* Construction des routes avec des requêtes PostgreSQL
* Discussion sur les APIs Restful
* Configuration du côté client (React)
* Construction du composant d'entrée Todo
* Construction du composant de liste Todo
* Construction du bouton de suppression
* Construction du composant d'édition Todo
* Révision de la pile PERN

Regardez le cours complet ci-dessous ou sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=ldYcgPKEZC8) (1,5 heure de visionnage).
