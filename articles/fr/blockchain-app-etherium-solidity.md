Apprenez à construire une application blockchain en utilisant les contrats intelligents Ethereum dans ce cours vidéo de Dapp University. Un contrat intelligent est une collection de code et de données qui réside à une adresse spécifique sur la blockchain Ethereum.

Vous apprendrez à créer une application de liste de tâches avec des contrats intelligents Ethereum en utilisant le langage de programmation Solidity. Vous apprendrez également à écrire des tests, à déployer sur la blockchain et à créer une application côté client.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=coQ5dg8wM2o) (1,5 heure de visionnage).
