Le langage de programmation Go est un langage open-source et compilé conçu par Google. Go est populaire pour les applications basées sur le cloud ou côté serveur.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous aidera à améliorer vos compétences en Go en vous apprenant à créer onze projets.

Akhil Sharma a développé ce cours. C'est un développeur expérimenté qui utilise Go depuis de nombreuses années. 

Voici les projets que vous allez construire en utilisant Go :

* Serveur Web simple
* API CRUD
* Système de gestion de livres YSQL
* SlackBot pour calculer l'âge
* Slackbot pour le téléchargement de fichiers
* Outil de vérification d'e-mails
* AWS Lambda
* CRM Fiber
* HRMS Fiber
* Stack serverless complète
* Bot d'IA avec wolfram et wit.ai
