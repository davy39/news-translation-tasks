Près de la moitié de tous les jeux sont créés en utilisant le moteur de jeu Unity. Unity est idéal pour les développeurs de jeux, qu'ils soient débutants ou expérimentés.

Nous venons de publier un cours de 7 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer des jeux en utilisant Unity. Ce cours est parfait pour un débutant complet.

Fahir d'AwesomeTuts a développé ce cours. Il crée des tutoriels Unity depuis des années et il sait exactement ce que les débutants doivent connaître.

Le cours commence par enseigner comment installer Unity, une vue d'ensemble de l'utilisation de Unity, et les bases de C#. Ensuite, vous apprendrez à créer un jeu complet de A à Z. Et tous les actifs dont vous avez besoin sont inclus gratuitement—il suffit de vérifier les liens dans la description de la vidéo.

Voici toutes les sections de ce cours :

* Téléchargement de Unity et Unity Hub
* À propos des versions de Unity et création d'un nouveau projet
* Introduction à l'interface de Unity
* Commencer avec les bases de Unity
* Corps rigides et colliders
* Source audio et éléments d'interface utilisateur
* Déplacement de notre personnage avec du code
* Introduction aux variables
* Opérations avec les variables
* Fonctions
* Instructions conditionnelles
* Boucles
* Coroutines
* Classes
* Modificateurs d'accessibilité (Encapsulation des données)
* Héritage
* Obtention de composants
* Introduction au jeu Monster Chase
* Importation des actifs
* Création des animations du joueur
* Calques de tri et ordre dans le calque
* Création de l'arrière-plan du jeu
* Mouvement du joueur
* Animation du joueur via le code
* Saut du joueur
* Caméra suivant le joueur
* Animations de l'ennemi
* Script de l'ennemi
* Générateur d'ennemis
* Collision de l'ennemi
* Le script du collecteur
* Système d'interface utilisateur de Unity
* Création du menu principal
* Navigation entre les scènes
* Sélection d'un personnage
* Variables statiques
* Modèle Singleton
* Événements et délégués
* Instanciation du personnage sélectionné
* Finalisation de notre jeu

Vous pouvez regarder le cours complet ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://youtu.be/gB1F9G0JXOo) (7 heures de visionnage).
