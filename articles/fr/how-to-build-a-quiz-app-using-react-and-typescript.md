Apprenez à créer une application de quiz avec React et TypeScript. Nous avons publié un [cours gratuit](https://www.youtube.com/watch?v=F2JCjVSZlG0) de Thomas Weibenfalk sur la chaîne YouTube freeCodeCamp.org.

Construire des projets est un excellent moyen d'apprendre. Dans ce cours, vous apprendrez à créer un projet d'application de quiz interactive. Thomas démontre toutes les étapes clairement du début à la fin afin que vous puissiez facilement suivre sur votre propre ordinateur.

Tout d'abord, vous apprendrez à démarrer l'application avec create-react-app, en utilisant TypeScript. Ensuite, vous apprendrez à construire le projet avec React et TypeScript, y compris Styled-Components. Les questions du quiz proviendront d'une API de trivia. Vous apprendrez à utiliser l'API avec React.

L'utilisation de TypeScript peut aider à s'assurer que vos variables et props sont ce que vous avez l'intention qu'elles soient, car il évalue votre code pendant la compilation (par opposition à l'exécution). Cela devient particulièrement important à mesure que votre programme devient plus grand.
