Il existe de nombreux tutoriels sur la création de clones d'applications web populaires, mais la plupart sont courts et ne couvrent pas beaucoup de fonctionnalités. 

Nous venons de publier un cours massif de 12 heures qui vous montrera comment créer votre propre version d'Instagram. Ce cours est beaucoup plus complet que la plupart des autres cours sur la création de clones d'applications web.

Karl Hadwen a développé ce cours. Karl a créé de nombreux cours populaires qui sont disponibles sur sa propre chaîne et sur la chaîne freeCodeCamp.org.

Le projet que vous allez créer dans ce cours utilise React (y compris les hooks personnalisés, useContext, useState, useEffect, & useRef), Firebase, Tailwind CSS, LoadTest, Lighthouse, Vercel, React Testing Library, et Cypress E2E Testing.

Vous apprendrez à créer plusieurs pages, y compris une page de connexion, une page d'inscription, un tableau de bord (pour voir/aimer/commenter des photos), et des profils d'utilisateurs. La page de connexion se connectera à Firebase lorsqu'un utilisateur essaiera de se connecter. Lorsqu'un utilisateur s'inscrit, l'application utilise l'authentification Firebase pour stocker l'utilisateur dans la base de données d'authentification Firebase.

Vous apprendrez même à déployer complètement l'application en utilisant Vercel.
