Il existe de nombreux tutoriels qui couvrent une technologie spécifique, mais il peut parfois être difficile de comprendre comment toutes ces technologies s'articulent. Nous venons de [publier une vidéo](https://www.youtube.com/watch?v=JgwI22y_eFA) sur la chaîne YouTube de freeCodeCamp.org qui vous guidera de A à Z dans la création d'une boutique en ligne.

Dans ce cours, Manoj Fernando vous montrera comment utiliser AWS, React et Stripe pour construire une librairie en ligne. Le projet utilise Amplify, AppSync, DynamoDB, S3, Lambda services dans AWS et React Hooks, ainsi que l'API Context dans ReactJS. Vous apprendrez comment toutes ces technologies fonctionnent ensemble.

Vous apprendrez à créer une API AppSync en tant qu'API GraphQL et à créer des buckets S3 pour stocker les images des livres. Manoj ne couvre pas chaque ligne du code React, mais se concentre sur la connexion à AWS depuis React. Il aborde également les différentes relations de base de données nécessaires pour créer le projet et comment implémenter différents niveaux d'autorisation. Enfin, il montre comment créer la page de paiement avec le traitement des paiements. Tout le code utilisé est inclus.

Voici les sujets abordés dans ce cours :

* Démonstration du produit
* Discussion sur le diagramme d'architecture
* Configuration du projet
* Initialisation d'un projet Amplify
* Ajout de l'authentification avec Cognito
* Création d'un bucket S3 pour stocker les images
* Création de fonctions Lambda pour traiter les commandes (Pipeline Resolver)
* Création de ressources cloud
* Traitement des commandes de livres
* Exécution de l'application
* Connexion aux ressources cloud depuis l'application React
* Vérification des commandes clients
* Hébergement du site sur S3

Regardez la vidéo complète sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=JgwI22y_eFA) (1 heure de visionnage).
