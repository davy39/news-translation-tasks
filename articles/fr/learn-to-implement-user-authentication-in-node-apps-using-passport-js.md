Si vous créez une application web, il est probable qu'elle ait des utilisateurs qui doivent être authentifiés.

Nous venons de publier un cours de 6 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à implémenter l'authentification des utilisateurs à partir de zéro dans vos applications Node.js et Express. Vous apprendrez à utiliser Passport.js.

Ce cours a été créé par Zach Gollwitzer. Zach a créé de nombreux cours de programmation bien reçus sur sa chaîne YouTube.

Vous apprendrez à implémenter une authentification qui fonctionne avec un formulaire de connexion, comment utiliser les stratégies d'authentification dans une application Angular, et comment créer une option d'authentification JWT personnalisée.

Voici les sections couvertes dans ce cours :

* Sujets et prérequis
* Introduction aux en-têtes HTTP et aux cookies
* Introduction au middleware Express
* Introduction aux sessions Express
* Implémentation de la stratégie locale Passport
* Introduction à la cryptographie à clé publique
* Comment fonctionnent les JWT ?
* Implémentation de la stratégie JWT Passport
* Implémentation d'une solution d'authentification JWT personnalisée
* Stratégie JWT dans une application front-end Angular
