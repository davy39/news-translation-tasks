React est l'une des bibliothèques JavaScript front-end les plus populaires pour construire des interfaces utilisateur. Utiliser TypeScript pour vos projets React peut conduire à une base de code plus robuste avec moins d'erreurs.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser React avec TypeScript.

Roadside Coder a développé ce cours. Il est un instructeur populaire qui a créé des cours de programmation sur une variété de frameworks et de sujets.

Dans ce cours, vous apprendrez d'abord les bases de TypeScript. Ensuite, vous apprendrez à intégrer TypeScript dans une application React en construisant un projet génial.

Vous apprendrez à utiliser TypeScript avec les React Hooks tels que useState, useRef et useReducers. Vous apprendrez à passer des props d'un composant à un autre en définissant les types de props du composant. Et vous apprendrez beaucoup plus !
