Evan You est le créateur de la célèbre bibliothèque JavaScript Vue pour le développement front-end et de l'outil de build JavaScript Vite, que de nombreux développeurs utilisent comme boilerplate pour leurs nouveaux projets. C'est un développeur autodidacte basé à Singapour.

Il partage des conseils pour :

* S'impliquer dans l'open source
    
* Diriger des projets open source et attirer des sponsors
    
* Et comment utiliser l'IA comme un assistant de réflexion plutôt que comme un simple assistant de codage
    

Nous discutons également de son point de vue sur la scène open source chinoise, d'un nouveau documentaire qui vient de sortir sur Vite, et de son nouveau projet : Void 0.

3. En naviguant sur le web, vous pouvez voir le message d'erreur indiquant que quelque chose a été "bloqué par la politique CORS". CORS signifie Cross-Origin Resource Sharing. Lors de la récupération de données, si le domaine du demandeur est différent de celui du destinataire, votre navigateur rejettera cette requête. C'est une mesure de sécurité importante, mais cela peut aussi être un casse-tête pour les développeurs qui tentent de maintenir leurs applications web. Heureusement, freeCodeCamp vient de publier ce tutoriel – rempli de théorie et d'exemples de code – pour vous aider à comprendre les bases. (Lecture de 20 minutes) : [https://www.freecodecamp.org/news/how-to-fix-cross-origin-errors/](https://www.freecodecamp.org/news/how-to-fix-cross-origin-errors/)
