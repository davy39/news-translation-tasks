Une architecture de microservices est un type d'architecture d'application où l'application est développée comme une collection de services. 

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous enseignera l'architecture des microservices et les systèmes distribués en utilisant une approche pratique. 

Georgio de Kantan Coding enseigne ce cours. Il fait un excellent travail en enseignant comment combiner un ensemble de différentes technologies en une seule application.

Dans ce cours, vous concevrez une application en utilisant Python, Kubernetes, RabbitMQ, MongoDB et mySQL.

Voici les sections de ce cours :

* Aperçu
* Installation et Configuration
* Code du Service d'Authentification
* Aperçu du Flux d'Authentification et JWTs
* Déploiement du Service d'Authentification
* Dockerfile du Service d'Authentification
* Kubernetes
* Code du Service de Passerelle
* MongoDB et GridFs
* Aperçu de l'Architecture (RabbitMQ)
* Communication Interservice Synchrone
* Communication Interservice Asynchrone
* Cohérence Forte
* Cohérence Éventuelle
* RabbitMQ
* Déploiement du Service de Passerelle
* Ingress Kubernetes
* StatefulSet Kubernetes
* Déploiement de RabbitMQ
* Code du Service de Conversion
* Déploiement du Service de Conversion
* Point de Contrôle
* Mise à Jour du Service de Passerelle
* Code du Service de Notification
* Déploiement du Service de Notification
* Vérification de Sanité
