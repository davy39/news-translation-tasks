## **Qu'est-ce que le Microsoft Azure Developer Associate ?**

Le Microsoft Azure Developer Associate (communément appelé AZ-204) est une certification Microsoft Azure axée sur la création d'applications web sur les services cloud Azure.

Le Azure Developer complète parfaitement l'Azure Administrator (Az-104) et est souvent la certification que les gens obtiennent ensuite avant de passer aux certifications professionnelles.

Microsoft Azure se distingue des autres fournisseurs de services cloud (CSP) en ce sens qu'il repose davantage sur ses outils de développement pour interagir avec les services et, dans de nombreux cas, il n'y a pas d'interface graphique (GUI), vous devez donc interagir directement avec l'API.

L'obtention de la certification Azure Developer ainsi que celle d'Azure Administrator fournira les connaissances complètes dont les ingénieurs auront besoin pour tirer le meilleur parti d'Azure.

Les certifications Azure Developer et Azure Administrator mènent généralement aux certifications Expert.

![Image](https://www.freecodecamp.org/news/content/images/2022/08/Screen-Shot-2022-08-13-at-4.04.50-PM.png)

## **À qui s'adresse la certification Microsoft Azure Associate ?**

Cette certification Azure convient aux scénarios suivants :

* Vous êtes un **développeur d'applications web** cherchant à acquérir des compétences cloud ou à transitionner vers un rôle de développeur cloud
* Vous êtes un **développeur cloud** ou un **ingénieur cloud** qui doit intégrer des services Azure dans votre application ou déployer votre application sur Azure.
* Vous devez apprendre beaucoup sur les Azure Functions pour construire des charges de travail serverless

## **Aperçu du** Azure Developer Associate

Le Azure Developer couvre les sujets suivants :

* Plusieurs façons de déployer des applications web sur Azure
* Une plongée approfondie dans les Azure Functions
* Un large aperçu des services d'intégration d'applications
* Beaucoup de pratique avec les Azure CLIs et SDKs pour interagir de manière programmatique avec les services Azure

## **Comment obtenir la certification ?**

Vous pouvez obtenir la certification en payant les frais d'examen et en passant l'examen dans un centre de test partenaire de Microsoft Azure.

Microsoft Azure est partenaire avec Pearson Vue et PSI Online qui disposent d'un réseau de centres de test dans le monde. Ils proposent des examens en personne et en ligne. Si vous en avez l'opportunité, je vous recommande de passer l'examen en personne.

Microsoft dispose d'une [page portail sur Pearson Vue](https://home.pearsonvue.com/Clients/Microsoft.aspx) où vous pouvez vous inscrire et réserver votre examen.

Les frais d'examen sont de **165 USD**.

## **Puis-je simplement regarder les vidéos et réussir l'examen ?**

Non, les certifications Azure sont exigeantes car les examens sont conçus pour tester vos connaissances pratiques. Il est donc important de réaliser tous les labs Azure dans votre propre compte Azure.

Il est fortement recommandé d'utiliser des examens pratiques avant de passer l'examen réel.
