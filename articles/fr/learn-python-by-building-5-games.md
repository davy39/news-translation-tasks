Beaucoup de gens apprennent mieux les langages de programmation en réalisant des projets. Si vous voulez apprendre Python en codant cinq jeux, consultez le cours vidéo de 7 heures que nous venons de publier sur la chaîne YouTube de freeCodeCamp.org. Nous avons rassemblé 5 excellents tutoriels de développement de jeux qui vous aideront à apprendre Python.

Vous allez créer Pong, Snake, Connect Four, Tetris et un simple jeu multijoueur. Ce sont tous des jeux populaires qui sont amusants à jouer ainsi qu'à coder.

![Image](https://www.freecodecamp.org/news/content/images/2019/11/image-5.png)
_Codez des jeux en Python !_

Ce cours adopte une approche basée sur des projets. Si vous apprenez mieux en pratiquant, ce cours est fait pour vous. Vous apprendrez avec Christian Thompson, Keith Galli et Tech with Tim. Ces enseignants expérimentés vous aideront à comprendre chaque ligne de code dans les projets.

Vous n'avez pas besoin d'expérience préalable en programmation pour apprendre avec ce cours. Cependant, il est utile d'être conscient de quelques concepts de programmation de base, car ce cours saute les instructions de syntaxe de base et se lance directement dans la création de projets.

### Créez des jeux en utilisant la ligne de commande, turtle et Pygame

L'un des jeux utilise simplement la ligne de commande. Écrire du code pour la ligne de commande est un excellent moyen d'apprendre les compétences de programmation de base sans avoir à se soucier des graphiques.

Un autre jeu utilise le module turtle graphics. Il s'agit d'un module intégré qui est souvent utilisé pour enseigner les compétences de programmation de base et peut être utilisé pour des jeux simples.

Vous apprendrez également à créer des jeux en utilisant Pygame. Pygame est un ensemble de modules Python utilisé pour créer toutes sortes de jeux, même des jeux avancés. Il facilite l'ajout de graphiques et de sons aux programmes Python.

Dans la dernière section, vous apprendrez à coder un jeu multijoueur évolutif avec Python en utilisant des sockets et des réseaux. Vous apprendrez à déployer votre jeu afin que des personnes du monde entier puissent jouer les unes contre les autres.

En créant un jeu multijoueur, vous apprendrez des concepts tels que le développement de clients et de serveurs, l'envoi de données chiffrées et la connexion de plusieurs clients à un serveur.

Vous pouvez regarder le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=XGf2GcyHPhc) (7 heures de visionnage).
