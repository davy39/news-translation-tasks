Yii est un framework PHP rapide, sécurisé et efficace utilisé pour créer toutes sortes d'applications web. Nous avons publié un cours vidéo complet sur l'utilisation du framework Yii2.

Dans ce cours, vous apprendrez à créer un clone de YouTube avec Zura Sekhniashvili. Ce cours est destiné aux débutants complets, mais vous devriez être [un peu familier avec PHP](https://www.youtube.com/watch?v=OK_JCtrrv-c&vl=en).

![Image](https://www.freecodecamp.org/news/content/images/2020/04/image-253.png)
_Démonstration du clone de YouTube que vous créerez dans ce cours._

Zura montrera comment utiliser le framework Yii2 pour développer à la fois le front-end et le back-end du site. Un utilisateur qui est un créateur pourra télécharger des vidéos et fournir des métadonnées sur les vidéos. Les créateurs pourront marquer les vidéos comme publiées ou non listées, et même voir les analyses vidéo. Un utilisateur qui est un spectateur pourra regarder des vidéos, aimer ou ne pas aimer les vidéos, trouver des vidéos similaires, s'abonner et rechercher des vidéos.

À la fin de ce cours, vous comprendrez comment utiliser le framework Yii2 et comment l'appliquer à vos propres projets.

Regardez la vidéo ci-dessous ou visionnez-la sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=whuIf33v2Ug).
