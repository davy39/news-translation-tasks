Une application météo est un projet de programmation très courant pour les débutants. Mais vous pouvez emmener ce projet de base au niveau supérieur en incorporant plusieurs APIs et l'autocomplétion des lieux.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous aidera à améliorer vos compétences en React en vous guidant à travers la création d'une application météo avec autocomplétion.

Slobodan Gajic a créé ce cours. Slobodan est un développeur frontend senior et c'est un excellent enseignant.

Dans ce cours, vous apprendrez à créer une autocomplétion de lieux/villes en utilisant l'API GeoDB et comment utiliser l'API OpenWeatherMap dans React. Vous apprendrez également à utiliser Promise.all et async/await.

Voici les sections de ce cours :

* Comment obtenir des clés API
* Création de l'application et installation des packages
* Construction du composant de recherche de ville
* Construction du composant météo actuel
* Récupération et mappage des données de l'API météo
* Construction du composant de prévision météo

Regardez le cours complet ci-dessous ou sur la [chaîne YouTube freeCodeCamp.org](https://youtu.be/Reny0cTTv24) (2 heures de visionnage).
