La gestion de l'état dans React est un processus permettant de gérer les données dont les composants React ont besoin pour se rendre. Ces données sont généralement stockées dans l'objet d'état du composant. Lorsque l'objet d'état change, le composant se ré-affiche.

La gestion de l'état dans React représente essentiellement la moitié d'une application React. Elle inclut toutes les données. L'autre moitié est la présentation, incluant le HTML, le CSS et la mise en forme. L'état et la gestion de l'état sont utilisés par la partie présentation de l'application. La seule fois où une application React se ré-affichera est lorsque l'état change.
