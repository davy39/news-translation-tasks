Apprenez à programmer pour le web en utilisant Python et Flask.

Le CS50 de Harvard est l'un des cours d'informatique les plus populaires au monde et nous le publions sur la chaîne YouTube sans publicité de freeCodeCamp.

Ce cours couvre tous les concepts essentiels de l'informatique de manière accessible et facile à suivre pour les débutants. Que vous ayez besoin d'un rappel ou que vous débutiez en programmation, ce cours est idéal à regarder.

Lors de la session avec David J. Malan, vous apprendrez à programmer des applications web en utilisant Python, Flask, JavaScript et jQuery.
