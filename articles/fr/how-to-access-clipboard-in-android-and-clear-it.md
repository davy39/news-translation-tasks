La fonctionnalité de copier-coller sur les smartphones Android peut sembler être une chose "ponctuelle". Une fois que vous copiez un extrait de texte, vous devez le coller avant de copier ou de couper un autre. Si vous ne le faites pas, vous n'aurez plus accès à ce texte. 

De plus, si vous copiez du texte et éteignez votre téléphone avant de le coller, le texte disparaîtra dans la nature. 

C'est parce que les smartphones Android enregistrent le texte copié dans la RAM (Random Access Memory), un type de mémoire à court terme. 

Mais il existe un moyen d'accéder au presse-papiers sur votre téléphone Android et de voir ces éléments copiés – et c'est ce dont parle ce tutoriel.

## Comment accéder au presse-papiers sur un téléphone Android

Vous pouvez accéder au presse-papiers de deux manières : la première est via Gboard, une application de clavier de Google. La seconde est en utilisant Clipper, une application tierce qui vous permet de gérer les éléments copiés de manière légèrement plus efficace. 

Google a interdit les gestionnaires de presse-papiers tiers pour Android 10 et versions ultérieures pour des raisons de sécurité, donc les guides de ce tutoriel seront basés sur l'utilisation de Gboard pour accéder et vider le presse-papiers.

## Comment télécharger et installer Gboard

### Étape 1 : Télécharger Gboard

Tout d'abord, vous devez télécharger Gboard depuis le Google PlayStore en tapant "Gboard" dans la barre de recherche et en appuyant sur l'icône de recherche. 

Installez l'application et ouvrez-la. Dans mon cas, je l'ai déjà installée, et elle ressemble à ceci :

![Screenshot_20210803-113545](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-113545.png)

### Étape 2 : Configurer l'application Gboard

Après avoir ouvert l'application, vous devez passer par quelques configurations de base pour la faire fonctionner. 

La première chose à faire est de sélectionner Gboard dans les paramètres de langue et de saisie. L'application vous y invitera. 

Si ce n'est pas le cas (très rarement), allez dans les paramètres, appuyez sur Système, cliquez sur "Langues et saisie", appuyez sur "Clavier virtuel", puis sélectionnez Gboard.

![Screenshot_20210803-113636-1](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-113636-1.png)

### Étape 3 : Définir Gboard comme clavier par défaut

Ensuite, vous serez invité à sélectionner Gboard comme clavier par défaut. Cliquez sur Méthode de saisie et choisissez Gboard dans la fenêtre contextuelle qui apparaît.

![Screenshot_20210803-113816](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-113816.png)

![Screenshot_20210803-113828](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-113828.png)

C'est tout ce que vous devez configurer. Appuyez sur Terminé pour commencer à utiliser Gboard.

![Screenshot_20210803-113843](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-113843.png)

## Comment accéder au presse-papiers sur Android

Pour accéder à la fonctionnalité du presse-papiers de Gboard, ouvrez une application de notes ou toute autre application dans laquelle vous souhaitez taper. 

Parfois, vous verrez l'icône du presse-papiers directement dans le menu à côté des paramètres, des GIF, et autres. Mais si vous ne voyez pas l'icône, appuyez sur les trois points à droite pour révéler les icônes cachées. Là, vous verrez l'icône du presse-papiers. 

Vous pouvez choisir de continuer à l'utiliser ainsi, ou vous pouvez appuyer, maintenir et glisser l'icône du presse-papiers vers le menu principal comme montré dans la capture d'écran ci-dessous.  

![drag-clipboard-icon](https://www.freecodecamp.org/news/content/images/2021/08/drag-clipboard-icon.gif)

### Comment activer le presse-papiers

Par défaut, la fonctionnalité du presse-papiers est désactivée. Vous pouvez l'activer en cliquant sur l'icône du curseur à droite et en appuyant sur "Activer le presse-papiers" :

![Screenshot_20210803-114610](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-114610.png)

Immédiatement après cela, vous verrez certains éléments par défaut sur le presse-papiers. Ce sont des instructions sur la façon d'utiliser la fonctionnalité du presse-papiers fournie par Gboard. 

Pour coller les éléments, il suffit de les taper, comme je l'ai fait ci-dessous :

![Screenshot_20210803-114800](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-114800.png)

### Comment copier avec le presse-papiers

Pour commencer à copier et coller, tapez votre propre texte, maintenez et faites glisser les extrémités vers l'endroit où vous souhaitez commencer et terminer, puis appuyez sur "Copier", ou "Couper", selon le cas.

![Screenshot_20210803-114901](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-114901.png)


### Comment coller avec le presse-papiers

Pour coller le texte copié, vérifiez votre presse-papiers pour révéler l'extrait copié. Les extraits récemment copiés doivent être sous l'onglet "Éléments récents". 

Appuyez sur l'élément pour le coller dans n'importe quel éditeur de texte que vous souhaitez, qu'il s'agisse de Notes, Messages ou de sites web de médias sociaux. 

![Screenshot_20210803-114940](https://www.freecodecamp.org/news/content/images/2021/08/Screenshot_20210803-114940.png)

## Comment vider le presse-papiers sur Android

Vous pouvez vider les éléments de votre presse-papiers avec Gboard de deux manières.

### Option 1 pour vider le presse-papiers
Ouvrez le presse-papiers, appuyez et maintenez l'élément, puis cliquez sur supprimer. Vous devez faire cela pour chaque élément.

![clear-clipboard](https://www.freecodecamp.org/news/content/images/2021/08/clear-clipboard.gif)

### Option 2 pour vider le presse-papiers
Ouvrez le presse-papiers, cliquez sur l'icône du crayon sur le côté droit, sélectionnez tous les éléments, et cliquez sur l'icône de la corbeille.

![mass-clear-clipboard](https://www.freecodecamp.org/news/content/images/2021/08/mass-clear-clipboard.gif)

## Comment épingler un élément au presse-papiers

Veuillez noter que tout élément copié avec GBoard disparaît après une heure. Pour empêcher cela, vous devez épingler l'élément copié.  

Pour épingler un extrait, ouvrez le presse-papiers, appuyez et maintenez l'extrait de texte, puis cliquez sur épingler. 

Pour épingler un groupe d'extraits, appuyez sur l'icône du crayon, sélectionnez les éléments à épingler, et cliquez sur l'icône d'épingle.
