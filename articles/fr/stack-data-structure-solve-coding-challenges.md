La structure de données pile (stack) est fréquemment utilisée en programmation, parfois en coulisses. Et les questions impliquant la structure de données pile sont très courantes dans les entretiens de codage.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à utiliser la structure de données pile pour résoudre le type de défis de codage proposés dans de nombreux entretiens d'embauche techniques.

Une pile est une structure de données qui permet d'ajouter ou de supprimer des éléments d'une seule extrémité, appelée le sommet (top). Cette structure est souvent utilisée pour stocker des éléments dans un ordre dernier entré, premier sorti (LIFO).

Dans ce cours, vous apprendrez à maîtriser cette structure de données. Le cours commence par des problèmes faciles sur LeetCode et InterviewBit et développe progressivement votre intuition afin que vous soyez capable de résoudre des problèmes difficiles en utilisant des piles. Et toutes les questions sont des questions d'entretien populaires de chez Google, Meta, Microsoft et plus encore !

Tanishq Chaudhary a développé ce cours. Tanishq a aidé des milliers de personnes à se préparer aux entretiens de codage.

Le cours est divisé en trois sections principales, chacune remplie de problèmes d'entretien de codage. Voici le plan du cours :

### SECTION 0 – Introduction

* Introduction au cours sur la structure de données Pile
* Intuitions d'implémentation de Pile + Code en Python3 et C++
* Simplify Path | InterviewBit | LeetCode
* Min Stack | InterviewBit | LeetCode

### SECTION 1 – Parenthèses

* Introduction aux questions sur les parenthèses
* Valid Parentheses | InterviewBit | LeetCode
* Redundant Braces | InterviewBit
* Minimum Remove to Make Valid Parentheses | LeetCode
* Longest Valid Parentheses | LeetCode

### SECTION 2 – Calculatrice

* Introduction aux questions sur les calculatrices
* Convert Infix to Postfix
* Evaluate Reverse Polish Notation | InterviewBit | LeetCode
* Basic Calculator I, II, III | LeetCode
