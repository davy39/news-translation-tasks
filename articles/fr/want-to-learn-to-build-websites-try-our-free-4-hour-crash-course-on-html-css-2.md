# 4. Écrivons un peu de HTML !

Dans ce screencast, Kevin introduit `<!DOCTYPE html>` pour informer le navigateur que le contenu est en HTML et nous écrivons notre première page web HTML et apprenons quelques éléments fondamentaux en cours de route !

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mon premier site web</title>
  </head>

  <body>
    <h1>Ma toute première page web</h1>
    <h2>Les sites web sont construits avec HTML</h2>
    <p>HTML est un langage de balisage qui indique au navigateur ce que tout est</p>
    <h2>Ils utilisent également CSS</h2>
    <p>J'ai hâte de commencer à apprendre CSS !</p>

```

# 5. `strong` et `emphasis`

# 7. Ancre et attributs

Nous pouvons utiliser l'élément d'ancre `<a>` pour lier à un emplacement différent sur la même page ou à une autre page. Pour indiquer à `<a>` où lier, nous utilisons des `attributs`, et pour les ancres, c'est `href`.

```html
<a href="https://scrimba.com">Lien vers Scrimba</a>

```

```html
 -->

```

# 15. CSS interne

Dans ce chapitre, Kevin introduit le CSS interne, une alternative à l'écriture de CSS en ligne.

Le CSS interne est écrit dans le même fichier que le HTML, mais dans une balise `<style>` séparée.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Tout sur la Terre et Mars</title>
    <style>
      h1 {
        font-size: 60px;
      }

      p {
        font-size: 24px;
        color: steelblue;
      }
    </style>
  </head>
  <body>
    <h1>La Terre et Mars</h1>
    <p>La Terre et Mars sont deux planètes de notre système solaire.</p>
  </body>
</html>

```

# 16. CSS externe

Le CSS externe est une autre approche pour gérer le CSS. Cette fois, Kevin nous montre comment nous pouvons extraire le CSS dans un fichier séparé et comment lier les feuilles de style CSS à nos fichiers HTML avec une balise `<link>`.

```html
<link href="css/style.css" rel="stylesheet" />

```

# 17. Classes et IDs

Dans ce screencast, Kevin se concentre sur trois types de sélecteurs en CSS et quand vous pourriez vouloir utiliser chacun.

```css
/* Sélecteur d'élément  */
a {
  color: darksalmon;
}

/* Sélecteur de classe  */
.intro {
  font-size: 24px;
}

/* Sélecteur d'ID  */

  color: lightgreen;
}

```

# 18. Commentaires en HTML et CSS

Nous pouvons ajouter des commentaires en HTML :

```html
<html>
  <head>
    <title>Commentaires !</title>
    <link rel="stylesheet" href="css/style.css" />
  </head>
  <body>
    <h1>Commentaires !</h1>
    <!-- Mon commentaire va ici -->
  </body>
</html>

```

Et en CSS :

```css
/* TODO: changer la couleur du texte en blanc */

body {
  background: #333;
  color: white;
}

/* Quelques commentaires supplémentaires */

h1 {
  color: red;
```


# 20. Introduction au modèle de boîte

Il est maintenant temps de découvrir le _modèle de boîte_.
La plupart des éléments sont des éléments de bloc, ce qui signifie qu'ils ont une largeur de 100 % de leur parent et une hauteur de 0.

C'est un cast brillant, où Kevin explique non seulement de manière simple et succincte comment fonctionne le modèle de boîte, mais nous évite également les pièges courants dans lesquels même les développeurs expérimentés tombent de temps en temps.

# 21. Marge et Remplissage

Ensuite, nous avons les marges et les remplissages.
Les marges sont utilisées pour contrôler la position d'un élément **par rapport à ceux qui l'entourent**, tandis que le remplissage est utilisé pour contrôler le positionnement du contenu **à l'intérieur** de notre élément.
Kevin fait un excellent travail en expliquant les nombreuses façons dont le remplissage et les marges peuvent être définis en CSS.

```css
/*  */
padding-top: 20px;
padding-right: 30px;
padding-bottom: 40px;
padding-left: 50px;

/* La version abrégée serait */
padding: 20px 30px 40px 50px;

margin-top: 500px;
margin-left: 100px;
margin-right: 100px;
margin-bottom: 10px;

/* La version abrégée serait */
margin: 500px 100px 10px;

```

# 22. Bordures

La dernière pièce du modèle de boîte - les bordures. Les bordures sont ajoutées autour de vos éléments. Et elles peuvent être définies de manière similaire aux marges et au remplissage.

```css
border-color: yellow;
border-width: 20px;
border-style: solid;

/* La version abrégée serait */
border: solid yellow 20px;

```
