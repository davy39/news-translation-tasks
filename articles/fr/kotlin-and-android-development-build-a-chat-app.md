Êtes-vous prêt à plonger dans le monde du développement Android en utilisant Kotlin ?

Nous venons de publier un cours Android de 10 heures sur la chaîne YouTube freeCodeCamp.org. Alexandru Cristian a créé ce cours. Je vais vous aider à comprendre comment développer avec Kotlin et Android.

Ce cours vous emmènera à travers :

* Les collections telles que les listes, les ensembles et les maps
* Les génériques et les paramètres de type
* Les fonctions lambda et les fonctions de portée
* Android Studio, sa structure de fichiers et la conception XML
* Les widgets Android comme les boutons, TextView, EditText, et plus encore
* Comment construire une application de quiz entièrement fonctionnelle pour les appareils Android.
