Dans ce tutoriel intermédiaire sur React.js de PortEXE, vous apprendrez à créer un clone de l'application populaire de prise de notes Evernote. Cette application permet aux utilisateurs de créer des notes avec un éditeur de texte complet et de les mettre à jour en direct dans une base de données en utilisant Firebase.

Le clone d'Evernote propose des fonctionnalités de création, d'édition et de suppression de notes. Les notes sont sauvegardées dans le cloud en utilisant Cloud Firestore de Firebase.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=I250xdtUvy8) (2 heures de visionnage).
