Arkanoid (similaire à Breakout) est un jeu vidéo classique. Quelle meilleure façon d'améliorer vos compétences en TypeScript qu'en construisant un jeu amusant ?

![Image](https://www.freecodecamp.org/news/content/images/2021/02/arkenoid.gif)
_C'est le jeu que vous allez construire._

Nous avons publié un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à construire un jeu Arkanoid basé sur un navigateur en utilisant TypeScript.

Thomas Weibenfalk a développé ce cours. Thomas est un programmeur et enseignant expérimenté.

Vous apprendrez à construire le jeu sans aucun framework. Le jeu est implémenté en utilisant des types, des classes, des modules et du code moderne en TypeScript.

De plus, vous apprendrez à utiliser le bundler Parcel. Les bundlers de code combineront et simplifieront votre code, le rendant plus rapide à charger. Parcel facilite le développement en JavaScript et TypeScript.

Regardez le cours complet sur la [chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=7bejSTim38A) (95 minutes de visionnage).
