La technologie du machine learning est désormais si courante que vous l'utilisez probablement des dizaines de fois par jour sans même vous en rendre compte. Et comme elle a tant d'applications, les perspectives d'emploi sont excellentes pour toute personne ayant une solide expérience en machine learning.

Nous venons de publier un cours de machine learning sur la chaîne YouTube de freeCodeCamp.org, qui est l'endroit idéal pour commencer votre parcours d'apprentissage.

Kylie Ying a développé ce cours. Kylie a travaillé dans de nombreux endroits intéressants tels que le MIT, le CERN et freeCodeCamp. Elle est physicienne, ingénieure et une excellente enseignante.

Ce cours est une excellente introduction à de nombreux concepts clés du machine learning. Dans cette vidéo, vous découvrirez la logique et les mathématiques derrière les modèles d'apprentissage supervisé et non supervisé. Vous apprendrez également comment programmer différents modèles de machine learning sur Google Colab.

Google a accordé une subvention qui a rendu ce cours possible.

Voici les sujets abordés dans ce cours :

* Intro Données/Colab
* Intro au Machine Learning
* Caractéristiques (Features)
* Classification/Régression
* Modèle d'entraînement
* Préparation des données
* K-plus proches voisins (K-Nearest Neighbors)
* Implémentation KNN
* Naive Bayes
* Implémentation Naive Bayes
* Régression logistique
* Implémentation de la régression logistique
* Machine à vecteurs de support (SVM)
* Implémentation SVM
* Réseaux de neurones
* Tensorflow
* Classification par réseaux de neurones avec Tensorflow
* Régression linéaire
* Implémentation de la régression linéaire
* Régression linéaire à l'aide d'un neurone
* Régression par réseaux de neurones avec Tensorflow
* Clustering K-Means
* Analyse en composantes principales (PCA)
* Implémentations K-Means et PCA

Regardez le cours complet ci-dessous ou sur la chaîne YouTube de freeCodeCamp.org (durée : 2 heures).
