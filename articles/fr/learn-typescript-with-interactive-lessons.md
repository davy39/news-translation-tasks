TypeScript peut faciliter votre vie en tant que programmeur et rendre votre code moins sujet aux erreurs.

Nous venons de publier un cours TypeScript pour débutants sur la chaîne YouTube freeCodeCamp.org. Bob Ziroll de Scrimba enseigne ce cours. Un code interactif est disponible sur la page du cours Scrimba pour chaque leçon.

Le cours vous offrira une introduction pratique à TypeScript. À la fin, vous aurez une solide compréhension des fondamentaux de TypeScript et de la manière de l'utiliser pour créer des applications JavaScript plus sûres et plus fiables.

TypeScript est un "superset" de JavaScript, ce qui signifie que tout code JavaScript valide est également valide en TypeScript. Cependant, TypeScript améliore votre expérience de codage en vous permettant d'écrire du code avec une plus grande confiance dans la stabilité et la longévité de vos projets. Cela est réalisé via une vérification au moment de la compilation, qui identifie les bugs potentiels avant l'exécution du code, vous permettant de corriger les problèmes de manière proactive et d'éviter les plantages dans vos programmes. Cela garantit un processus de développement plus fluide et plus fiable.

En fournissant un système de typage fort et un support d'outils modernes, TypeScript offre également un environnement de développement robuste. Cela facilite le refactoring du code, la gestion de grandes bases de code et la collaboration avec d'autres développeurs. Ce cours d'introduction vous fournira les connaissances et les compétences nécessaires pour intégrer efficacement TypeScript dans vos projets JavaScript.

Voici les sujets abordés dans le cours :

* Pourquoi TypeScript ?
  
* Types de base
  
* Types littéraux
  
* Types personnalisés (objets)
  
* Tableaux typés
  
* Propriétés optionnelles
  
* Unions
  
* Rétrécissement de type
  
* Types utilitaires
  
* Génériques

Regardez le cours complet sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=SpwzRDUQ1GI) (2 heures de visionnage).
