## Tutoriel Java pour débutants

Dans [ce cours YouTube de Programmation avec Mosh](https://www.youtube.com/watch?v=eIrMbAQSU34), vous apprendrez les chaînes de caractères, les tableaux, les conditionnelles, les boucles et plus encore. Vous construirez également une calculatrice de prêt immobilier.

## Cours Java

Dans [ce cours SoloLearn](https://www.sololearn.com/learning/1068), vous apprendrez les tableaux, les objets, les classes, les boucles et plus encore.

## Programmation Java : Résoudre des problèmes avec des logiciels

Dans ce [cours Coursera](https://www.coursera.org/learn/java-programming?specialization=java-programming), vous apprendrez les classes, les conditionnelles, les boucles, les fonctions et plus encore. Vous apprendrez également à travailler avec des fichiers CSV en Java et à construire un mini-projet.

## Programmation Java : Tableaux, listes et données structurées

Dans ce [cours Coursera](https://www.coursera.org/learn/java-programming-arrays-lists-data?specialization=java-programming), vous apprendrez les tableaux, la cryptographie, les HashMaps et plus encore. Vous construirez également un mini-projet sur le chiffre de Vigenère.

## Programmation Java : Principes de conception de logiciels

Dans ce [cours Coursera](https://www.coursera.org/learn/java-programming-design-principles?specialization=java-programming), vous apprendrez différents algorithmes de tri et les bases du texte prédictif.

## Bases de la programmation Java

Dans ce [cours Udacity](https://www.udacity.com/course/java-programming-basics--ud282), vous apprendrez les tableaux, les objets, les classes, les boucles, le débogage et plus encore.

Une fois que vous aurez terminé le cours pour débutants, vous pourrez suivre ce cours intermédiaire sur la [Programmation Orientée Objet](https://www.udacity.com/course/object-oriented-programming-in-java--ud283).

## Programmation Java

Dans ce [cours de Test Automation University](https://testautomationu.applitools.com/java-programming-course/), vous apprendrez les classes, les boucles, les instructions switch, les conditionnelles, les tableaux, les chaînes de caractères, l'abstraction et plus encore.

## Introduction à la programmation Java pour les vrais débutants

Dans ce [cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=GoXwIVyNvX0), vous apprendrez les tableaux, les objets, les classes, les boucles, le débogage et plus encore.

## Apprendre la programmation fonctionnelle en Java

Dans ce [cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=rPSL1alFIjI), vous apprendrez les propriétés de la programmation fonctionnelle, les callbacks, les interfaces fonctionnelles, le motif combinateur, les lambdas et plus encore.

## Architecture de la machine virtuelle Java expliquée pour les débutants

Dans [cet article de freeCodeCamp](https://www.freecodecamp.org/news/jvm-tutorial-java-virtual-machine-architecture-explained-for-beginners/), vous apprendrez la machine virtuelle Java ainsi que l'architecture qui la sous-tend.
