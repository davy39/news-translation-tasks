Si vous utilisez Instagram, vous avez peut-être dû bloquer certains comptes de temps en temps. Mais que faire si les choses changent et que vous souhaitez débloquer quelqu'un ?

Dans cet article, je vais vous expliquer les trois façons de débloquer quelqu'un sur Instagram en utilisant l'application mobile pour iPhone et Android.

## Comment débloquer quelqu'un sur Instagram via les paramètres

La première façon de débloquer quelqu'un est via l'onglet des paramètres. Voici les étapes à suivre :

1. Connectez-vous à l'application Instagram.
2. Cliquez sur l'icône de profil située en bas à droite.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/profile-on-right.jpeg)

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-profile.jpeg)

3. Cliquez sur le menu hamburger en haut à droite.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-hamburger-menu.jpeg)

4. Cliquez sur Paramètres dans la liste déroulante des options.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-settings.jpeg)

5. Cliquez sur Vie privée.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-privacy.jpeg)

6. Faites défiler vers le bas et cliquez sur Comptes bloqués.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/blocked-accounts.jpeg)

7. Cliquez sur Débloquer à côté du profil.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/unblock-next-to-profile.jpeg)

8. Cliquez sur le texte rouge Débloquer pour confirmer.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-unblock.jpeg)

Si vous réussissez, vous devriez voir ce message :

![Image](https://www.freecodecamp.org/news/content/images/2021/08/unblock-success.jpeg)

## Comment débloquer quelqu'un sur Instagram via les messages directs

Si vous avez correspondu avec quelqu'un via les messages directs, vous pouvez le débloquer là-bas.

1. Cliquez sur l'icône de message située en haut à droite de votre écran.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/Screen-Shot-2021-08-31-at-12.13.47-AM.png)

2. Cliquez sur le profil que vous souhaitez débloquer.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/account-to-block.jpeg)

3. Cliquez sur Débloquer situé en bas du fil de discussion.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-unblock-1.jpeg)

4. Cliquez sur le texte rouge Débloquer pour confirmer.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/unblock-message.jpeg)

## Comment débloquer quelqu'un sur Instagram via son profil

1. Allez sur le profil de la personne que vous avez bloquée.
2. Cliquez sur le bouton Débloquer dans leur profil.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/unblock-option-1.jpeg)

4. Cliquez sur le texte rouge Débloquer pour confirmer.

### Option alternative

1. Sur leur profil, vous pouvez cliquer sur les trois points situés en haut à droite. Sur Android, les trois points peuvent être verticaux.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/click-three-dots.jpeg)

![Image](https://www.freecodecamp.org/news/content/images/2021/08/three-dots.jpeg)

2. Cliquez sur Débloquer.

![Image](https://www.freecodecamp.org/news/content/images/2021/08/unblock-option-2.jpeg)

3. Cliquez sur le texte rouge Débloquer pour confirmer.
