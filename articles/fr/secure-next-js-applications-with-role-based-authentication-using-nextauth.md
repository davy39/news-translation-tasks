Si vous créez des applications Next.js, il est probable que vous deviez inclure l'authentification des utilisateurs à un moment donné. 

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui enseigne comment implémenter l'authentification dans Next.js 13.5 avec la puissante bibliothèque, NextAuth – qui sera bientôt connue sous le nom d'AuthJS.

Jacob Lower de Clarity Coders a créé ce cours. Il est un développeur et enseignant expérimenté.

L'authentification des utilisateurs est un aspect essentiel des applications web modernes, et ce tutoriel vidéo complet est conçu pour vous équiper des connaissances nécessaires pour le faire. Vous apprendrez l'authentification basée sur les rôles, une approche qui non seulement améliore la sécurité, mais vous donne également un contrôle fin sur ce que différents utilisateurs peuvent voir et faire sur votre site.

Le cours commence par les bases de NextAuth, vous guidant à travers la configuration de l'environnement et la création du projet. Ensuite, vous apprendrez à utiliser NextAuth pour les pages rendues côté serveur et côté client, vous offrant une compréhension holistique des processus d'authentification.

Le tutoriel adopte une approche pratique en incorporant des fournisseurs OAuth prêts à l'emploi tels que Google et GitHub, vous montrant comment intégrer ces services largement utilisés dans votre application de manière transparente. Mais pour ceux qui aiment personnaliser, le cours vous couvre également. Il explore comment créer votre propre fournisseur d'authentification personnalisé en utilisant MongoDB, garantissant que vous savez comment adapter l'authentification à vos besoins spécifiques.

Cette vidéo ne traite pas seulement du 'comment faire' mais aussi du 'comment cela fonctionne'. De la configuration des routes et options de l'API à la protection des pages côté serveur et côté client, le cours est structuré pour vous offrir une expérience pratique. Vous apprendrez même à gérer les fonctionnalités de connexion et de déconnexion de manière efficace.

Voici les sections de ce cours :

* Introduction
* Configuration de l'environnement / Revue des extensions
* Création du projet
* Installer NextAuth (Auth.js) / Créer la structure du projet
* Créer l'API NextAuth (Auth.js) - Options / Routes
* Configuration du fournisseur OAuth intégré - (Google / GitHub)
* Fonctionnalités de connexion / déconnexion
* Protection des pages - Page côté serveur
* Protection des pages - Page côté client
* Protection des pages - Middleware
* Création d'un fournisseur personnalisé (Création d'une base de données d'utilisateurs)
* Inscription / Configuration de MongoDB Atlas
* Créer des modèles d'utilisateurs (Schéma Mongoose)
* Formulaire d'utilisateur pour créer des utilisateurs
* Créer l'API des utilisateurs
* Configuration du fournisseur personnalisé
