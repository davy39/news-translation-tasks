Apprenez à utiliser Vue.js dans ce cours complet pour débutants par Gwen Faraday. Vue.js est un framework JavaScript open-source pour construire des interfaces utilisateur et des applications monopages.

Tout d'abord, vous apprendrez les bases de Vue.js. Ensuite, vous verrez comment créer deux projets en utilisant Vue.js. À la fin du cours, vous serez capable de commencer à utiliser Vue.js dans vos propres projets.

Ce cours couvre :

* L'instance Vue
* Les directives
* Les méthodes
* La liaison de données
* Les événements
* Les filtres
* Les composants
* Vue-cli
* Vue devtools
* Props & Slots
* Vue-router
* Vuex
* Les tests unitaires
* Et plus encore !

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=4deVCNJq3qc) (3 heures de visionnage).
