Deno est une alternative à Node.js créée par Ryan Dahl, la même personne qui a créé Node.js.

De nombreuses personnes, y compris Ryan Dahl, pensent que Deno est supérieur à Node.js. Nous avons publié un cours vidéo de 6 heures entièrement consacré à Deno sur la chaîne YouTube de freeCodeCamp.org. Décidez par vous-même s'il est meilleur que Node.

Vous apprendrez les bases de Deno dans ce cours. Deno peut être utilisé avec JavaScript ou TypeScript, mais il est plus courant d'utiliser TypeScript. Ce cours inclut un mini cours accéléré sur TypeScript.

Ce cours a été créé par The Codeholic. À la fin du cours, vous aurez utilisé Deno pour construire une application de sondage avec une API REST utilisant MongoDB. Ensuite, vous serez prêt à commencer à implémenter vos propres applications basées sur Deno.

Voici les sujets abordés dans le cours :

* Qu'est-ce que Deno ?
* Comment installer
* "Hello World" dans Deno
* Commandes Deno
* Bases de TypeScript
* Fonctionnalités principales de la démonstration
* Bibliothèque standard de Deno
* Bibliothèques tierces
* Travailler avec le système de fichiers
* Créer un serveur HTTP basique
* Utilisation des packages npm
* Construction d'un projet réel
* Bonnes pratiques de Deno

Regardez le [cours complet sur la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=TQUy8ENesGY) (6 heures de visionnage).
