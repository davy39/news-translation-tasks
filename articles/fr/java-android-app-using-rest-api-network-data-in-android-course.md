Les API REST peuvent être utilisées pour ajouter beaucoup de fonctionnalités à vos applications Android.

Nous avons publié un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer une application Android Java en utilisant une API REST.

Vous apprendrez à construire une application Android avec Java qui gère les données JSON d'un service d'API REST. L'application utilise la bibliothèque Volley pour créer une application météo en utilisant les données de l'API. 

Le cours a été créé par Shad Sluiter. Shad est professeur d'informatique à la Grand Canyon University. Au cours des 20 dernières années, il a enseigné à des étudiants de la 6e année aux personnes âgées dans de nombreux aspects de la technologie.

Ce cours inclut des conseils sur la construction d'une RequestQueue, la conception d'un Singleton et l'utilisation de callbacks pour les communications réseau asynchrones. 

Vous apprendrez également à utiliser les méthodes de la bibliothèque Volley JsonArrayRequest et JsonArrayObject pour analyser un fichier JSON.

Voici les sections de ce cours :

* API REST expliquée : JSON vs XML
* Introduction à l'application météo Android
* Concevoir la mise en page
* Écouteurs d'événements onClick des boutons
* Demander des données JSON avec Volley
* File d'attente de requêtes Singleton dans Volley
* Callbacks pour les requêtes asynchrones
* Récupérer un JSONObject à partir d'un tableau
* JSONArrayRequest avec Volley
* Récupérer un JSONArray avec Volley
* Obtenir les prévisions par nom de ville
