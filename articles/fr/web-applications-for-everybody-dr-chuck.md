La quantité de compétences diverses requises pour créer une application web moderne peut être intimidante pour les débutants. Mais avec le bon enseignant et le bon état d'esprit, tout le monde peut apprendre à créer des applications web complexes.

Nous venons de publier Applications Web pour Tous sur la chaîne YouTube freeCodeCamp.org. Ce cours de 14 heures est une introduction au développement d'applications web.

Charles Severance (alias Dr. Chuck) enseigne ce cours. Il est professeur clinicien et enseigne à l'École d'Information de l'Université du Michigan. Il est également l'enseignant qui a créé Python pour Tous, l'un des cours de programmation en ligne les plus populaires au monde.

Ce cours est une introduction à la création d'applications web pour toute personne ayant déjà une compréhension de base de la conception web responsive avec JavaScript, HTML et CSS. Vous développerez des applications web et de base de données en PHP, en utilisant SQL pour la création de bases de données, ainsi que des fonctionnalités en JavaScript, jQuery et JSON.

Vous apprendrez des concepts de base, comme la conception de bases de données, tout en travaillant sur des projets qui nécessitent le développement d'applications web de plus en plus complexes. De l'installation d'un éditeur de texte à la compréhension de la manière dont un navigateur web interagit avec un serveur web, en passant par la gestion des événements avec jQuery, vous obtiendrez une vue d'ensemble complète et introductive du développement d'applications web.

Voici quelques-uns des sujets abordés dans ce cours :

* Applications Web et le Cycle Requête/Réponse
* Comprendre le Mode Développeur du Navigateur
* HTML - HyperText Markup Language
* CSS - Cascading Style Sheets
* JavaScript
* jQuery
* JSON
* Installation de MAMP et XAMPP
* PHP
* SQL à Table Unique
* Modélisation de Données
* Construction d'une Application CRUD en PHP

En plus d'apprendre ces sujets, vous aurez une pratique concrète en construisant divers projets.

Regardez le cours ci-dessous ou [sur la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=xr6uZDRTna0) (14 heures de visionnage).
