Si vous travaillez avec des fichiers XML dans Notepad++, les formater manuellement peut être une tâche fastidieuse, surtout si les fichiers XML sont volumineux.

Dans cet article, je vais vous montrer comment formater vos fichiers XML en quelques clics. Je vais également vous montrer comment vérifier que la syntaxe des fichiers XML est dans le bon ordre et que toutes les balises sont fermées.


## Comment formater des fichiers XML dans Notepad++
Vous pouvez voir que le fichier XML ci-dessous n'est pas du tout bien formaté. Avec ce fichier, je vais démontrer comment formater n'importe quel fichier XML dans Notepad++ :

![unformattedxmlfile-1](https://www.freecodecamp.org/news/content/images/2023/05/unformattedxmlfile-1.gif)

**Étape 1** : Ouvrez votre fichier XML avec Notepad++ si ce n'est pas déjà fait :
![ss1](https://www.freecodecamp.org/news/content/images/2023/05/ss1.png) 

**Étape 2** : Dans la barre de menu, cliquez sur "Plugins" et sélectionnez "Plugins Admin" :
![ss2](https://www.freecodecamp.org/news/content/images/2023/05/ss2.png) 

**Étape 3** : Recherchez "xml tools", cochez-le dans les résultats de recherche, cliquez sur installer, et sélectionnez "Oui" :
![ss3](https://www.freecodecamp.org/news/content/images/2023/05/ss3.png) 

Le plugin sera téléchargé et Notepad++ redémarrera :
![ss4](https://www.freecodecamp.org/news/content/images/2023/05/ss4.png) 

**Étape 4** : Cliquez à nouveau sur "Plugins" dans la barre de menu, survolez "XML Tools", et cliquez sur "Pretty print" pour formater le fichier XML :
![ss5](https://www.freecodecamp.org/news/content/images/2023/05/ss5.png) 

Maintenant, le fichier XML est bien formaté :
![formattedxmlfile](https://www.freecodecamp.org/news/content/images/2023/05/formattedxmlfile.gif) 

Alternativement, vous pouvez appuyer sur `CTRL + ALT + SHIFT + B` pour faire la même chose.


## Comment vérifier la syntaxe XML avec Notepad++
Si votre fichier XML est vraiment volumineux, la syntaxe pourrait être incorrecte, et certaines balises pourraient manquer des balises de fermeture. 

Vous pouvez vérifier la syntaxe avec le même plugin XML Tools et il suggérera ce que vous devez faire s'il trouve quelque chose d'anormal.

Encore une fois, cliquez sur "Plugins", survolez "XML Tools" et sélectionnez "Check XML syntax now" :
![ss6](https://www.freecodecamp.org/news/content/images/2023/05/ss6.png) 

Maintenant, il m'indique que je n'ai pas fermé une des balises :
![ss7](https://www.freecodecamp.org/news/content/images/2023/05/ss7.png) 


## Conclusion
Cet article vous a montré comment formater vos fichiers XML dans Notepad++ avec le plugin XML Tools. Comme déjà mentionné, cela peut être utile si vous devez travailler avec des fichiers XML volumineux dans Notepad++.

Le plugin XML Tools est un excellent outil pour travailler avec des fichiers XML dans Notepad++ car il vous offre une bonne expérience utilisateur avec les fichiers XML. Pour plus d'informations, vous pouvez le survoler et voir quelques autres choses utiles que vous pouvez faire avec.
