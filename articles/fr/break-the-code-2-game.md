Break The Code 2.0 est un nouveau jeu navigateur qui vous envoie dans le passé, en 1999.

Vous complétez des missions de décryptage en utilisant vos connaissances en programmation et votre intuition pour résoudre des énigmes.

Le jeu est réparti sur 4 disques de missions :

* Le disque /g1 – "BoxOfChocolates"
* Le disque /g2 – "Fight Club"
* Et les disques /g3 et /g4, que vous pouvez débloquer en résolvant les deux premiers disques

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Break_The_Code_2_by__Tech_Domains_---1.png)
_L'interface inspirée de Windows 98 de Break The Code 2.0. (Et oui, au cas où vous vous poseriez la question, le Démineur est entièrement jouable.)_

Break The Code 2.0 est gratuit, et vous pouvez [y jouer directement dans votre navigateur](https://go.tech/fcc) sans rien installer.

Break The Code 2.0 a été codé par une équipe de développeurs chez .Tech Domains. C'est la suite de leur défi de décryptage de l'année dernière.

.Tech Domains a également accordé une subvention à freeCodeCamp.org pour que nous créions un guide de type "Let's Play" du jeu.

Dans ce guide, les ingénieurs logiciels de freeCodeCamp, Tom et Shaun, travaillent ensemble pour accomplir certaines des missions et craquer certains des chiffres. (1 heure de visionnage) :

%[https://www.youtube.com/watch?v=sWMAsRqcoWc]

Si vous résolvez les énigmes avant le 15 avril, vous pourriez être éligible pour gagner des prix d'une valeur allant jusqu'à 30 000 $.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/Prize-Image.jpg)
_Le catalogue des prix. Le trône de jeu en forme de scorpion peut ne pas s'harmoniser avec les autres meubles de votre pièce. Mais il a l'air bien confortable._

C'est une manière amusante de tester vos compétences en décryptage tout en concourant avec d'autres développeurs du monde entier. Plongez dans le passé en 1999 et [essayez Break The Code 2.0](https://go.tech/fcc).
