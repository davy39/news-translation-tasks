L'analyse de régression est une méthode statistique utilisée pour étudier la relation entre une variable dépendante et une ou plusieurs variables indépendantes. 

L'une des méthodes les plus couramment utilisées pour l'analyse de régression linéaire est le R-carré (R-Squared). 

Dans cet article, vous découvrirez ce qu'est le R-carré et la signification de sa ou ses valeurs. Vous verrez également certains des domaines où il est utilisé. 

## Qu'est-ce que le R-carré ?

Le R-carré (R²) est une mesure statistique utilisée pour déterminer la proportion de variance d'une variable dépendante qui peut être prédite ou expliquée par une variable indépendante.

En d'autres termes, le R-carré montre dans quelle mesure un modèle de régression (variable indépendante) prédit le résultat des données observées (variable dépendante).

Le R-carré est également connu sous le nom de coefficient de détermination. C'est un modèle de mesure de la qualité de l'ajustement pour l'analyse de régression linéaire.

## Que signifie une valeur de R-carré ?

Une valeur de R-carré indique dans quelle mesure le modèle prédit le résultat de la variable dépendante. Les valeurs du R-carré s'étendent de 0 à 1.

Une valeur de R-carré de 0 signifie que le modèle n'explique ou ne prédit aucunement (0 %) la relation entre les variables dépendantes et indépendantes.

Une valeur de 1 indique que le modèle prédit 100 % de la relation, une valeur de 0,5 indique que le modèle prédit 50 %, et ainsi de suite. 

La formule ci-dessous est la plus utilisée pour trouver la valeur du R-carré :

R² = 1 - RSS/TSS

où,

* R² = coefficient de détermination
* RSS = somme des carrés des résidus (sum of squares of residuals)
* TSS = somme totale des carrés (total sum of squares)

## Où le R-carré est-il utilisé ?

Le R-carré est utilisé dans différents domaines. Il peut être utilisé pour les éléments suivants :

* Analyse des risques en finance. 
* Campagnes de marketing. 
* Recherche scientifique. 
* Économie. 
* Analyse sportive.
