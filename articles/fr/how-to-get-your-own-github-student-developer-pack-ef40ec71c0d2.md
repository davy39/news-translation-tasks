Par Vinicius Gularte

Bonjour à tous ! Aujourd'hui, je vais vous montrer comment j'ai obtenu mon pack étudiant de GitHub et comment vous pouvez l'obtenir aussi.

Voici quelques avantages du pack étudiant :

* Dépôts privés illimités.
* Accès gratuit à des outils de développement.
* 50 $ de crédit Digital Ocean.
* [Voici](https://education.github.com/pack) une liste complète.

### Conditions requises

Pour obtenir le pack étudiant, vous devez remplir ces conditions :

* Être inscrit dans un cours menant à un diplôme ou à un certificat.
* Être un étudiant âgé de 13 ans ou plus.
* Avoir un compte GitHub.

Si vous remplissez ces conditions, nous pouvons continuer.

### Comment obtenir le pack

Vous obtenez l'accès [ici](https://education.github.com/pack/offers) et cliquez pour rejoindre GitHub Education.

Vous allez voir ceci :

![Image](https://cdn-media-1.freecodecamp.org/images/1bRUBBB13mBXP0udwjUSjrPqAWM5umNjNY1R)

Remplissez ce formulaire et cliquez sur suivant.

Nous avons maintenant deux options pour valider notre statut scolaire.

Avec un email de votre école comme monnom@ecole.com.

Ou

Téléchargez une preuve de votre statut académique.

Dans mon cas, j'ai téléchargé mon statut académique en portugais brésilien, et cela a très bien fonctionné. Une semaine plus tard, j'ai reçu ceci :

![Image](https://cdn-media-1.freecodecamp.org/images/ciQ6TbOpW5SFyRDBZWKZ3LWezDw4TN67b11e)

Le Student Pack durera deux ans. Vous pouvez renouveler le plan si vous êtes toujours étudiant à ce moment-là.

C'est tout. J'espère que cela vous a plu, n'oubliez pas de ?. À bientôt.
