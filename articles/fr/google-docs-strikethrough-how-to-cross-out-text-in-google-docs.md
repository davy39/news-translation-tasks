Lorsque vous écrivez, il arrive parfois que vous souhaitiez barrer certains textes. Cela signifie ajouter une ligne horizontale qui traverse un morceau de texte. 

Voici à quoi cela ressemble : vous êtes une personne géniale.

Voici divers cas d'utilisation des textes barrés :

* Pour indiquer qu'un morceau ou un bloc de texte doit être supprimé.
* Pour indiquer qu'un morceau ou un bloc de texte est obsolète.
* Pour montrer l'achèvement d'une tâche. Cela se voit généralement dans les listes de choses à faire.
* Pour transmettre un message non lié ou humoristique lors de l'écriture, et ainsi de suite.

Dans cet article, nous verrons comment barrer ou rayer du texte lors de l'écriture en utilisant Google Docs.

## Comment barrer du texte dans Google Docs

Il existe deux méthodes que nous pouvons utiliser pour rayer du texte dans Google Docs – utiliser une commande de raccourci ou choisir l'option de barré dans l'onglet Format de la section d'en-tête de Google Docs. 

### Comment barrer du texte dans Google Docs en utilisant une commande de raccourci

Voici les étapes à suivre pour utiliser une commande de raccourci afin de barrer du texte dans Google Docs :

* Ouvrez [Google Docs](https://docs.google.com/) et créez un document vierge.
* Écrivez du texte dans votre document. 
* Surlignez le texte que vous avez écrit.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/gdocs.png)

* Sur Windows, appuyez sur `Alt` + `Shift` + `5`.
* Sur Mac, appuyez sur `⌘` + `Shift` + `X`.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/gdocs_strikethrough.png)

Et voilà, nous avons utilisé une commande de raccourci pour rayer du texte dans Google Docs. 

### Comment barrer du texte dans Google Docs en utilisant l'onglet Format

Dans cette section, nous verrons comment barrer du texte dans Google Docs en utilisant l'option Format. Voici les étapes :

* Ouvrez [Google Docs](https://docs.google.com/) et créez un document vierge.
* Écrivez du texte dans votre document. 
* Surlignez le texte que vous avez écrit.
* Cliquez sur l'onglet `Format` dans l'en-tête.
* Cliquez sur `Texte`.
* Cliquez sur l'option `Barré`.

![Image](https://www.freecodecamp.org/news/content/images/2022/04/gdocs_tab.png)

Après avoir fait cela, votre texte devrait être rayé.
