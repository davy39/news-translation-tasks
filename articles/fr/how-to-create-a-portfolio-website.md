Lorsque vous postulez pour des emplois de développeur, vous voudrez avoir un portfolio aussi solide que possible. Dans de nombreux cas, votre portfolio peut être plus important pour les responsables de l'embauche que votre CV.

Heureusement, le développeur web Stewart Hines a beaucoup réfléchi à ce qui fait un site portfolio efficace. Dans sa présentation, il donne des conseils sur la façon dont vous pouvez construire un portfolio qui vous ressemble. Il donne des conseils sur :

* les considérations de base en matière de design
* les détails que vous devriez révéler sur vous-même
* les types de projets que vous devriez inclure pour impressionner les employeurs (la qualité plutôt que la quantité)
* ...et bien plus encore.
