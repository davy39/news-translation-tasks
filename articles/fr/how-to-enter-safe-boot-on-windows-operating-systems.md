Si vous utilisez le système d'exploitation Windows, vous avez peut-être entendu l'expression "Redémarrer en mode sans échec" ou quelque chose de similaire.

Dans ce tutoriel, je vais expliquer ce que cela signifie et comment vous pouvez le faire sur votre machine Windows.

## Qu'est-ce que le Mode sans échec ou le Mode sans échec pour Windows ?

Le Mode sans échec ou le Mode sans échec démarre le système d'exploitation Windows dans un état élémentaire où il utilise un ensemble minimal de fichiers et de pilotes.

Vous pourriez vouloir redémarrer en mode sans échec pour de nombreuses raisons, que ce soit pour identifier des problèmes liés à des logiciels ou des pilotes, ou pour faire quelque chose en toute sécurité qui pourrait ne pas être possible dans votre mode par défaut de Windows.

## Comment activer le Mode sans échec

Tout d'abord, vous devez ouvrir le programme **exécuter** depuis votre système d'exploitation. Vous pouvez simplement cliquer sur la boîte de recherche de votre barre des tâches, ou vous pouvez également utiliser les touches de raccourci, `Touche Windows` + `R`.

Ensuite, tapez `msconfig` et appuyez sur la touche **Entrée**, ou cliquez simplement sur **OK**.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--1-.png)

Cela ouvrira la fenêtre **Configuration du système**.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--2-.png)

Rendez-vous simplement dans la section **Démarrage**.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--3-.png)

Cherchez l'option "Démarrage en mode sans échec". Vous devez l'activer. Cochez simplement la case comme dans l'image ci-dessous :

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--4-.png)

Ensuite, vous pouvez simplement cliquer sur "Appliquer", puis sur "OK".

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--5-.png)

Assurez-vous d'avoir sauvegardé tout votre travail car vous devez redémarrer votre ordinateur. Cliquez sur "Redémarrer". Cela redémarrera votre système d'exploitation actuel en mode sans échec.

## Comment désactiver le Mode sans échec

Après avoir terminé votre tâche en mode sans échec, vous voudrez désactiver le mode sans échec pour continuer votre démarrage normal, n'est-ce pas ?

Pour cela, nous allons suivre tous les processus mentionnés ci-dessus, mais en quelque sorte à l'envers. Ne vous inquiétez pas, car je vais vous montrer chaque étape à nouveau.

Ouvrez à nouveau la fenêtre **Exécuter**. Je préfère les touches de raccourci, `Touche Windows` + `R`.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--6-.png)

Tapez `msconfig` et cliquez sur **OK**.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--7-.png)

Cela ouvrira à nouveau la fenêtre **Configuration du système**. Allez dans la section **Démarrage**.

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--8-.png)

Assurez-vous de décocher la case pour "Démarrage en mode sans échec". Ensuite, cliquez sur "Appliquer" puis sur "OK".

![Image](https://www.freecodecamp.org/news/content/images/2022/11/Screenshot--10-.png)

Assurez-vous d'avoir sauvegardé toutes vos tâches et cliquez sur "Redémarrer". Votre ordinateur redémarrera en mode par défaut une fois de plus.
