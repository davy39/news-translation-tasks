Apprenez à construire un organisateur de dépenses avec l'API Dropbox et JavaScript ES6 !

Dropbox est une plateforme de contenu et de collaboration. Son API permet d'ajouter des fonctionnalités Dropbox à vos applications, telles que le stockage de fichiers, le partage, les aperçus et la recherche.

Dans ce cours vidéo, Christian Jensen utilise Scrima.com pour démontrer comment construire une application d'organisateur tout en enseignant les bases de l'utilisation de l'API Dropbox. Vous apprendrez également plusieurs concepts ES6, comme les méthodes de tableau et les fonctions async/await.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://youtu.be/AzCQrjWQJs4) (1 heure de visionnage).
