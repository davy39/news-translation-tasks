L'un des moteurs de jeu les plus largement utilisés est Unreal Engine par Epic Games. Sur la chaîne YouTube freeCodeCamp.org, nous avons publié un cours complet sur l'utilisation d'Unreal Engine avec C++ pour développer des jeux.

Dans ce cours pour débutants de Fahir Mehovic d'Awesome Tuts, vous apprendrez à créer trois jeux complets avec Unreal Engine 4 et Blueprints (un système de script visuel).

Unreal Engine est gratuit à utiliser, tout comme ce cours. C'est une excellente façon de commencer à apprendre le développement de jeux sans frais. En créant ces trois jeux, vous apprendrez de nombreux principes de conception de jeux qui sont utiles pour créer tous types de jeux.

Le cours commence par enseigner un jeu simple afin que vous puissiez apprendre les concepts de base. Vous apprendrez à contrôler un personnage dans un environnement 3D. Le personnage peut ramasser des objets et perdre de la santé.

Ensuite, vous apprendrez à créer un jeu de casse-briques et un pac-man. Ces jeux introduisent une complexité supplémentaire telle que le maintien du score et l'interaction avec les ennemis. Tous les actifs nécessaires pour créer ces jeux sont inclus gratuitement.

Vous pouvez regarder le cours ci-dessous ou sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=LsNW4FPHuZE) (5 heures de visionnage).
