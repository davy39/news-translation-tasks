Les ordinateurs peuvent être intimidants, surtout si vous n'avez pas grandi avec. Certaines tâches qui semblent simples pour les utilisateurs expérimentés peuvent sembler impossibles pour les nouveaux utilisateurs.

Nous venons de publier un cours sur les bases de l'informatique et de la technologie sur la chaîne YouTube de freeCodeCamp.org. Ce cours enseigne les bases de l'utilisation d'un ordinateur et est parfait pour les débutants et les personnes qui souhaitent combler certaines lacunes dans leurs connaissances.

La Goodwill Community Foundation a développé ce cours. Depuis près de 20 ans, leur programme GCFLearnFree.org a aidé des millions de personnes dans le monde à acquérir les compétences essentielles dont elles ont besoin pour vivre et travailler au 21e siècle.

Dans ce cours, vous en apprendrez davantage sur les types d'ordinateurs et les systèmes d'exploitation utilisés aujourd'hui. Vous comprendrez mieux les parties de base d'un ordinateur, l'utilisation des applications, la connexion à Internet et bien plus encore.

Ce cours sur les bases de l'informatique est rempli d'animations pour vous aider à comprendre les sujets.

Voici les sujets abordés dans ce cours :

* Qu'est-ce qu'un ordinateur ?
* Boutons et ports d'un ordinateur
* Parties de base d'un ordinateur
* À l'intérieur d'un ordinateur
* Découverte des ordinateurs portables
* Comprendre les systèmes d'exploitation
* Comprendre les applications
* Installation d'un ordinateur de bureau
* Connexion à Internet
* Qu'est-ce que le Cloud ?
* Nettoyage de votre ordinateur
* Protection de votre ordinateur
* Création d'un espace de travail sécurisé
* Sécurité Internet : Fonctionnalités de sécurité de votre navigateur
* Comprendre le spam et le phishing
* Comprendre le suivi numérique
* Bases de Windows : Prise en main du bureau
* Bases de Mac OS X : Prise en main du bureau
* Bases du navigateur
