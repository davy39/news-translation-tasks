Par Marko Denic

Google est l'outil le plus puissant au monde si vous savez comment l'utiliser correctement. Laissez-moi vous montrer comment devenir meilleur en recherche Google.

## 1. Utiliser des guillemets pour une recherche de correspondance exacte

Vous pouvez utiliser des guillemets pour forcer une recherche de correspondance exacte. C'est extrêmement utile si vous connaissez précisément la phrase de recherche que vous recherchez. Vous obtiendrez simplement des résultats plus précis si vous procédez ainsi.

`"what is javascript"`

![Google search exact match example](https://www.freecodecamp.org/news/content/images/2021/07/google-exact.PNG)

## 2. Utiliser l'opérateur AND

L'opérateur AND ne retournera que les résultats liés aux deux termes. Dans l'exemple ci-dessous, vous pouvez voir le cas d'utilisation parfait de cet opérateur.

`html AND css`

![Google search with AND keyword](https://www.freecodecamp.org/news/content/images/2021/07/google-and.PNG)

## 3. Utiliser l'opérateur OR

Vous pouvez utiliser l'opérateur OR pour obtenir des résultats liés à l'un des termes de recherche que vous saisissez.

`(javascript OR python) free course`

![Google search OR keyword example](https://www.freecodecamp.org/news/content/images/2021/07/google-or.PNG)

## 4. Utiliser l'opérateur - (moins)

L'opérateur - exclura les résultats qui contiennent un terme ou une phrase. Ainsi, dans notre cas, nous voulons obtenir des résultats liés à JavaScript, mais en exclure tout résultat concernant CSS.

`javascript -css`

![Google search with - operator example](https://www.freecodecamp.org/news/content/images/2021/07/google-minus.PNG)

## 5. Utiliser des caractères génériques (*) comme espaces réservés

Vous pouvez utiliser les caractères génériques (*) comme espaces réservés, qui seront remplacés par n'importe quel mot ou phrase. 

C'est mon astuce préférée. Je l'utilise quand je connais presque toute la phrase de recherche et que je remplace simplement la partie que j'ignore par un astérisque. Super utile.

`"how to start * in 6 months"`

![Google search with wildcard example](https://www.freecodecamp.org/news/content/images/2021/07/google-wildcard.PNG)

## 6. Comment effectuer une recherche à l'intérieur d'un seul site web

C'est une autre astuce que j'utilise fréquemment. Si je cherche quelque chose de particulier sur un site web spécifique, voici mon approche.

`site:freecodecamp.org`

![Google search inside a single website example](https://www.freecodecamp.org/news/content/images/2021/07/google-site.PNG)

## 7. Comment trouver un type de fichier spécifique

Vous pouvez également utiliser cette fonctionnalité très utile pour vous aider à trouver un type de fichier spécifique. C'est pratique si vous recherchez des documents comme des PDF. Si vous ne l'avez jamais utilisé auparavant, vous commencerez à le faire dès maintenant.

`filetype:pdf learn css`

![Google search for specific filetype](https://www.freecodecamp.org/news/content/images/2021/07/google-filetype.PNG)

## 8. Comment rechercher une plage de chiffres

Cela peut concerner n'importe quoi. Si vous cherchez à acheter quelque chose qui se situe dans une fourchette de prix spécifique, ou si vous recherchez des résultats incluant une plage d'années spécifique, ceci est pour vous.

`ecmascript 2016..2018`

![Google search for specific range of numbers](https://www.freecodecamp.org/news/content/images/2021/07/google-range.PNG)

## 9. Utiliser l'opérateur `before`

Utilisez l'opérateur before pour ne retourner que les résultats antérieurs à une date donnée.  
Vous devez fournir des dates au format année-mois-jour ou seulement une année. Par exemple :

`javascript before:2020`

![Google search with before operator](https://www.freecodecamp.org/news/content/images/2021/07/google-before.png)

## 10. Utiliser l'opérateur `after`

Utilisez l'opérateur after pour ne retourner que les résultats postérieurs à une date donnée. Vous devez fournir des dates au format année-mois-jour ou seulement une année. Super utile si vous voulez exclure les résultats obsolètes.

`web development after:2020`

![Google search with after operator](https://www.freecodecamp.org/news/content/images/2021/07/google-after.png)

Comme vous pouvez le voir, Google peut être un outil puissant si vous savez comment l'utiliser correctement.
