Créer un site web portfolio est un excellent moyen d'améliorer vos compétences en développement frontend. 

Nous venons de publier un tutoriel React qui vous apprendra à construire un portfolio frontend avec des animations soignées.

Slobodan a développé ce cours. C'est un développeur expérimenté qui a créé de nombreux cours populaires. 

Vous apprendrez à construire un site web attrayant en utilisant React, GreenSock, Animate.css et des animations CSS personnalisées. Vous apprendrez à configurer un formulaire de contact et à inclure une carte.

Dans ce cours, vous apprendrez :

* Comment dessiner des animations SVG en utilisant GSAP et React
* Comment animer des lettres au survol en utilisant Animate.css et React
* Comment ajouter une animation de chargement de page en utilisant React-Loaders
* Comment ajouter une animation Pacman dans React
* Comment créer des animations CSS 3D personnalisées
* Comment implémenter des icônes de police dans React
* Comment créer et implémenter un formulaire de contact dans React en utilisant EmailJS
* Comment ajouter des cartes à votre site web dans React en utilisant Leaflet.js

Regardez le cours complet ci-dessous ou sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=bmpI252DmiI) (1,5 heure de visionnage).
