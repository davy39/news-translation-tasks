Visual Studio Code, également appelé VS Code, est l'un des éditeurs de code les plus populaires. Il a été créé par Microsoft et est totalement gratuit.

VS Code fonctionne sur Mac, Windows et Linux. Il est utilisé par les programmeurs pour écrire des logiciels dans de nombreux langages de programmation différents, mais il est particulièrement puissant pour les développeurs web.

Voici quelques fonctionnalités de Visual Studio Code qui peuvent vous aider à améliorer votre productivité en codage :

* la coloration syntaxique,
* l'indentation automatique du code,
* les raccourcis clavier,
* les plugins,
* la navigation facile entre les fichiers,
* et bien plus encore.

Nous avons publié un cours complet sur la prise en main de Visual Studio Code sur la chaîne YouTube de freeCodeCamp.org. Le cours a été créé par James Q Quick.

Le cours couvre les sujets suivants :

* Installation de Visual Studio Code
* Version stable vs version Insider
* Première ouverture
* L'interface
* Personnalisation de l'interface avec des raccourcis (Système d'onglets en grille, Palette de commandes)
* Recherche et remplacement
* Autres raccourcis pour travailler avec les fichiers
* Raccourcis pour naviguer et éditer du texte
* IntelliSense
* Emmet
* Paramètres
* Extensions et thèmes
* Raccourcis clavier
* Extensions de mappage de touches
* Extraits de code
* Synchronisation des paramètres
* Débogage
* Terminal intégré
* Intégration du contrôle de source

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=WPqXP_kLzpo) (2 heures de visionnage).
