Apprenez à créer des applications de bureau multiplateformes pour Windows, Mac et Linux.

PySide6 est une liaison Python pour le framework d'application Qt6. PySide6 vous permet d'utiliser le framework Qt6 pour créer des interfaces graphiques (GUI) et d'autres applications multiplateformes en Python. Il fournit un moyen pratique d'accéder aux différents composants du framework Qt6, y compris les widgets, les signaux et les slots. PySide6 est un projet gratuit et open-source maintenu par la société Qt.

Nous venons de publier un cours complet sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à utiliser PySide6 et Qt pour développer des applications.
