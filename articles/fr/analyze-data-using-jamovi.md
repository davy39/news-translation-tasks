Apprenez jamovi dans ce cours complet de Barton Poulson. jamovi est une application gratuite et open-source qui rend l'analyse de données facile et intuitive. Les menus et commandes de jamovi sont conçus pour simplifier la transition depuis des programmes comme SPSS mais, sous le capot, jamovi est basé sur le puissant langage de programmation statistique R.

jamovi possède un design propre et convivial qui facilite la compréhension de vos données et rend facile le partage de votre travail avec d'autres. Dans ce cours d'introduction, vous apprendrez comment utiliser jamovi pour affiner, analyser et visualiser vos données afin d'obtenir des informations critiques.

Vous pouvez regarder le [cours vidéo complet](https://www.youtube.com/watch?v=mZomeS0tLxY) sur la chaîne YouTube de freeCodeCamp.org (5 heures de visionnage).
