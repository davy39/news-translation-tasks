Dans ce tutoriel étape par étape de 6 heures de Coding Addict, vous apprendrez à construire un site e-commerce à partir de zéro en utilisant React et create-react-app.

Ce tutoriel vous enseigne comment créer le HTML, le CSS et le JavaScript (avec React) pour implémenter toutes les fonctionnalités principales d'un site e-commerce qui vend des téléphones portables. Le tutoriel explique également comment déployer le site en utilisant Netlify afin que le monde entier puisse le voir.

Une fois que vous aurez terminé de regarder, vous devriez être en mesure d'appliquer les concepts pour créer un site e-commerce qui vend n'importe quel nombre de produits.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=wPQ1-33teR4) (6 heures de visionnage).
