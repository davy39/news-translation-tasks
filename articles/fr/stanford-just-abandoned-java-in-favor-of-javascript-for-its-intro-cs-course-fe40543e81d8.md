Le professeur de Stanford qui a écrit un livre populaire sur Java — et a enseigné Java pendant 15 ans — abandonne désormais Java au profit de JavaScript pour ses cours d'introduction à l'informatique.

Le site web de Stanford explique que leur nouveau cours CS 106J "couvre le même matériel que [l'ancien cours] CS 106A mais le fait en utilisant JavaScript, le langage le plus courant pour implémenter des pages web interactives, au lieu de Java."

Ce n'est pas la fin de Java, qui dispose de décennies d'outils et de bases de code, et est toujours utilisé par de nombreuses grandes entreprises. Plutôt, c'est simplement un signe des temps.

JavaScript est véritablement open source et n'appartient à personne, alors que Java est essentiellement détenu par Oracle à travers son acquisition de Sun Microsystems en 2010. Et Oracle a un [historique moins que stellar](https://fcc.im/2op2Svx) en matière de respect des licences open source.
