Êtes-vous prêt à améliorer vos compétences en développement web avec ASP.NET Core MVC ?

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org conçu pour vous faire passer de débutant à développeur ASP.NET Core compétent. Alen Omeri a créé ce cours.

## Ce que vous allez apprendre

Ce cours est structuré pour offrir une expérience d'apprentissage étape par étape, commençant par un projet full-stack basique et progressant vers des applications plus complexes. Voici ce que couvre le cours :

**Démarrage avec ASP.NET Core MVC**

Commencez par comprendre la structure fondamentale d'ASP.NET Core MVC. Ce segment introductif pose les bases de votre parcours.

**Projet 1 : Application Web de Menu Alimentaire** :

* **Installation** : Apprenez à initialiser votre projet et à vous préparer pour le développement.
* **Modélisation et Intégration de Base de Données** : Plongez dans la création de modèles, la configuration du contexte et la connexion de votre projet à une base de données.
* **Logique du Contrôleur** : Développez le contrôleur et ses méthodes pour gérer le flux de données entre la base de données et vos vues.
* **Améliorations** : Ajoutez plusieurs entrées de données à la base de données manuellement, implémentez une barre de recherche et affinez l'interface utilisateur.

**Projet 2 : Clone de Google Docs** :

* **Introduction et Installation** : Familiarisez-vous avec le projet et configurez votre environnement de développement.
* **Développement Principal** : Créez des modèles, configurez la base de données, développez un contrôleur et gérez la création de pages.
* **Fonctionnalités Avancées** : Intégrez la bibliothèque TinyMCE, gérez l'édition de documents et formatez la page de suppression. Apprenez à convertir des documents en PDF pour un ensemble de fonctionnalités complet.

**Projet 3 : Application Web Stripe** :

* **Introduction à Stripe** : Comprenez les essentiels de l'intégration de Stripe dans votre application web.
* **Initialisation du Projet** : Configurez votre projet dans Visual Studio et préparez-vous pour le développement.
* **Construction de l'Application** : Créez des modèles, configurez la base de données, stockez des produits et affichez-les. Plongez dans les détails techniques de la création d'une méthode de paiement et testez rigoureusement votre application.

## Pourquoi ce cours ?

Ce cours vidéo sur la chaîne YouTube de freeCodeCamp.org est plus qu'un simple tutoriel ; c'est une expérience d'apprentissage pratique qui met l'accent sur l'application dans le monde réel. En construisant trois projets distincts, vous acquerrez une compréhension approfondie d'ASP.NET Core MVC, vous permettant de développer vos propres applications web avec confiance.

Regardez le cours complet sur la [chaîne YouTube freeCodeCamp.org](https://youtu.be/6SAFgcMie4U) (2,5 heures de visionnage).
