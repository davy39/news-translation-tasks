Apprenez à coder un jeu de serpent en utilisant Python et Pygame.

Tim Ruscica partage ce tutoriel de jeu intermédiaire et démontre comment créer un jeu de serpent complet et jouable. Vous apprendrez à utiliser le module Pygame.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=CD4qAhfFuLo) (49 minutes de visionnage).
