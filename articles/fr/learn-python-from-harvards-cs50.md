Apprenez les bases de Python avec Harvard.

Le CS50 de Harvard est l'un des cours d'informatique les plus populaires au monde et nous le publions sur la chaîne YouTube sans publicité de freeCodeCamp.

Ce cours couvre tous les concepts essentiels de l'informatique d'une manière accessible et facile à suivre pour les débutants. Que vous ayez besoin d'une remise à niveau ou que vous débutiez en programmation, ce cours est idéal à regarder.

Lors de la session avec David J. Malan, vous apprendrez les bases du langage de programmation Python et découvrirez comment créer quelques projets simples.

Vous pouvez regarder le cours sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=hnDU1G9hWqU).
