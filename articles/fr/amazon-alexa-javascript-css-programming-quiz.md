Maintenant, vous pouvez apprendre des concepts de codage sans les mains en utilisant un Amazon Echo.

Le contributeur de freeCodeCamp.org [David Jolliffe](https://github.com/OcelotDive) a créé un jeu de quiz avec des questions sur JavaScript, CSS, le réseau et l'informatique.

Si vous avez un Amazon Echo à proximité, vous pouvez jouer au jeu dès maintenant en disant : "Alexa, démarre le quiz free code camp."

L'ensemble du projet, ainsi que ses ~100 questions, est [open source et sur GitHub](https://github.com/freeCodeCamp/FreeCodeCampTriviaQuiz). Nous accepterions volontiers des questions supplémentaires. Ouvrez une pull request pour ajouter votre nouvelle question à la base de code, ou ouvrez simplement une issue GitHub avec la question si vous êtes pressé.

Voici une vidéo de moi et de mon fils Quentin, âgé de 11 mois, démontrant la compétence Alexa. Vous pouvez regarder la vidéo sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=IFAmuQgjMK0) (3 minutes de visionnage).
