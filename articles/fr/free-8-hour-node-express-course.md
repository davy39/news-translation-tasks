Il existe de nombreuses façons de créer le back-end d'une application web. Parmi les méthodes les plus populaires, on trouve l'utilisation d'une combinaison de Node.js et Express.js.

Nous venons de publier un cours complet de 8 heures sur la chaîne YouTube freeCodeCamp.org qui vous apprendra tout sur Node et Express.

Ce cours a été développé par John Smilga. John est un développeur expérimenté et créateur de cours. Il a créé certains des cours les plus populaires sur la chaîne YouTube freeCodeCamp.org.

Dans ce cours, vous apprendrez d'abord les bases de Node et Express. Ensuite, vous apprendrez à construire une API Rest complexe. Enfin, vous construirez une application MERN et d'autres projets Node.
