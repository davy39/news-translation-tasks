Il existe de nombreux outils et ressources en ligne qui peuvent vous aider à garantir que votre application web répond à toutes les exigences d'accessibilité.

### Général

* **[Chrome Accessibility Developer Tools](https://chrome.google.com/webstore/detail/accessibility-developer-t/fpkknkljclfencbdbgkenhalefipecmb?hl=en)**  Il s'agit d'une extension Google Chrome qui ajoutera un nouveau panneau latéral d'accessibilité dans l'onglet Éléments des outils de développement Chrome. Ce nouveau panneau affichera toutes les propriétés pertinentes pour l'aspect accessibilité de l'élément que vous inspectez actuellement. Cette extension ajoute également un audit d'accessibilité qui peut être utilisé pour vérifier si la page web actuelle viole des règles d'accessibilité.
* **[WAVE](https://wave.webaim.org/extension/)**  Il s'agit d'une autre extension Chrome (et Firefox) qui vous permet de trouver des défauts d'accessibilité sur votre page web.
* **[aXe](https://chrome.google.com/webstore/detail/axe/lhdoppojpmngadmnindnejefpokejbdd?hl=en-US)**  Cette extension Google Chrome peut trouver des défauts d'accessibilité sur votre page web.
* **[tota11y](https://khan.github.io/tota11y/)**  Une boîte à outils de visualisation d'accessibilité qui vérifie comment votre site web fonctionne avec les technologies d'assistance.
* **[AccessLint](https://www.accesslint.com/)**  Une application GitHub qui vérifie vos demandes de tirage pour les problèmes d'accessibilité.

### Contraste des couleurs

* **[Check My Colours](https://www.checkmycolours.com/)**  Ce site web vérifie si votre site web répond aux exigences d'accessibilité pour la couleur et le contraste.
* **[Color Oracle](https://colororacle.org/)**  Cette application simule le daltonisme. Elle est disponible sur Windows, Mac et Linux.
* **[Accessible Color Palette Builder](https://toolness.github.io/accessible-color-matrix/)**  Ce site web vous permet de modifier une palette de couleurs et créera un tableau montrant toutes les combinaisons de couleurs possibles, vous indiquant lesquelles sont accessibles et quelles combinaisons éviter d'utiliser.
* **[WebAIM Color Contrast Checker](https://webaim.org/resources/contrastchecker/)**  Outil de site web qui vous permet de tester les valeurs de couleur de premier plan et d'arrière-plan et de voir si elles passent les ratios de contraste de couleur AA et AAA pour le texte petit et grand.
* **[Randoma11y Color Generator](https://randoma11y.com/)**  Il s'agit d'un projet d'application en cours qui génère aléatoirement des palettes de couleurs web accessibles AA et permet aux utilisateurs finaux de voter pour ou contre les combinaisons de couleurs. Ce site est utile pour les concepteurs Web/développeurs Web lors du démarrage d'un projet qui n'a pas de couleurs prédéterminées.

### Lecteur d'écran

Un lecteur d'écran est une application logicielle avec technologie d'assistance qui tente de transmettre ce que les personnes avec une vision normale voient sur un écran à leurs utilisateurs via des moyens non visuels, comme la synthèse vocale. Les développeurs peuvent utiliser un lecteur d'écran pour comprendre comment une page web peut être rendue accessible à ces personnes et développer la page en conséquence. Certains des lecteurs d'écran les plus connus et largement utilisés sont :

* **[NVDA](https://www.nvaccess.org/)**  NVDA(Non Visual Desktop Access) est un logiciel de lecteur d'écran gratuit et open-source téléchargeable pour le système d'exploitation Windows.
* **[JAWS](https://www.freedomscientific.com/Products/Blindness/)**  JAWS(Job Access With Speech) est un programme de lecteur d'écran disponible sous forme de logiciel sous licence pour Windows.
* **[a11yproject](https://a11yproject.com/)**  Ce site fournit des guides pratiques et des exemples pour la mise en œuvre de fonctionnalités et de composants accessibles lors du développement web. Le site contient des guides pratiques, des mythes, des conseils et des outils d'accessibilité recommandés. Le projet est piloté par la communauté et peut être contribué sur Github.
* **[ChromeVox](https://www.chromevox.com/)**  ChromeVox est une extension du navigateur Chrome qui peut être utilisée pour rendre les pages web sous forme de synthèse vocale.
