WordPress est un système de gestion de contenu (CMS) open-source permettant de créer n'importe quel type de site web : un site d'eCommerce, un portfolio, un forum, et bien d'autres encore.

WordPress a débuté comme une simple plateforme de création de blogs, mais aujourd'hui, WordPress propulse plus de 43 % de l'ensemble des sites web sur Terre, et ce chiffre continue de croître.

Une raison notable pour laquelle WordPress est si populaire est que des personnes ayant peu ou pas de connaissances en programmation peuvent créer des sites web avec. Vous pouvez ajouter de nombreuses fonctionnalités grâce aux milliers d'extensions (plugins) et de thèmes disponibles. C'est pourquoi WordPress est largement considéré comme un outil no-code.

Bien qu'il s'agisse d'un outil no-code, beaucoup de professionnels et de freelances gagnent leur vie grâce à WordPress parce qu'ils savent coder.

Ainsi, vous connaissez peut-être WordPress comme un outil no-code, mais certaines personnes gagnent beaucoup d'argent avec grâce à leurs compétences en programmation. Cela signifie que WordPress est à la fois un outil de code et de no-code.

Dans cet article, vous apprendrez pourquoi WordPress est à la fois un outil de code et un outil no-code. Je vous montrerai également comment ajouter du code HTML et CSS à vos sites WordPress.

Je vais créer un site WordPress de démonstration avec un logiciel appelé LocalWP. Vous pouvez le télécharger sur le site officiel.

## Pourquoi WordPress est un outil no-code

Si vous ne savez pas coder, vous pouvez tout de même créer des sites web avec WordPress grâce à un constructeur de pages en glisser-déposer comme [Elementor](https://elementor.com/). Vous pouvez ensuite ajouter des fonctionnalités avec des extensions, optimiser la vitesse de votre site et le sauvegarder.

Elementor est un constructeur de pages en glisser-déposer très populaire pour WordPress. Vous pouvez l'utiliser pour personnaliser vos designs et vos mises en page.
![elementor-en-action](https://www.freecodecamp.org/news/content/images/2022/03/elementor-in-action.gif)

Vous pouvez accomplir beaucoup de choses avec la version gratuite d'Elementor, mais si vous souhaitez en tirer le meilleur parti, vous devriez acheter au moins la version la plus abordable et vous entraîner avec. Cela vous aidera grandement à devenir un meilleur designer et développeur WordPress.

Une préoccupation courante concernant les sites WordPress est leur lenteur au chargement. Si vous avez construit votre site WordPress avec Elementor en particulier, Nat Miletic de [Clio websites](https://cliowebsites.com/) a écrit [un article de blog sur la façon d'accélérer vos sites Elementor](https://cliowebsites.com/does-elementor-slow-down-your-site/).

Outre Elementor, d'autres constructeurs de pages notables en glisser-déposer sont [Divi](https://www.elegantthemes.com/gallery/divi/) et [Beaver builder](https://www.wpbeaverbuilder.com/).

Puisque ces constructeurs de pages vous aident à créer des sites WordPress, comment ajouter des fonctionnalités telles que l'eCommerce, un forum, un formulaire de contact, etc. ? La vaste bibliothèque d'extensions disponibles permet d'ajouter n'importe quelle fonctionnalité souhaitée.

Parmi ces extensions, on trouve [WooCommerce](https://woocommerce.com/) pour ajouter des fonctionnalités d'eCommerce, [WP Forms](https://wpforms.com/) pour les formulaires de contact, [WP Job Manager](https://wordpress.org/plugins/wp-job-manager/) pour créer des sites d'offres d'emploi, [Learn Dash](https://www.learndash.com/) pour ajouter des fonctionnalités de plateforme d'apprentissage, et bien d'autres.

Vous trouverez toujours une extension WordPress pour n'importe quelle fonctionnalité dont vous avez besoin !

En plus des extensions, WordPress propose de nombreux thèmes différents pour vous aider à déterminer l'apparence de votre site. Ils vous donnent également la possibilité d'ajouter des designs personnalisés.

Vous pouvez réellement accomplir énormément de choses avec WordPress sans écrire une seule ligne de code. C'est pourquoi WordPress peut être considéré comme un outil no-code.

## Pourquoi WordPress est un outil de code

Prêt à l'emploi, les développeurs WordPress peuvent faire beaucoup pour votre site avec des extensions et des thèmes. Mais à un certain point, vous pourriez vouloir mettre la main à la pâte et coder un peu si vous souhaitez ajouter des fonctionnalités sur mesure et aller au-delà de ce qui est déjà proposé par les extensions et thèmes.

Vous devrez ajouter du code dans WordPress si vous êtes intéressé par le développement d'extensions et de thèmes, l'ajout de CSS personnalisé pour améliorer le design ou l'ajout de widgets personnalisés.

Si vous voulez créer un thème WordPress, par exemple, vous devez être familier avec le HTML, le CSS et le PHP, car vous devez créer au moins deux fichiers : `index.php` et `styles.css`.

Votre fichier `index.php` contient du HTML mélangé à du code PHP, tandis que votre fichier CSS contient les styles nécessaires pour que le thème soit esthétique.

Si vous vérifiez le thème de votre site WordPress en ce moment, vous y verrez certainement des fichiers PHP et CSS.
![ss-1-3](https://www.freecodecamp.org/news/content/images/2022/03/ss-1-3.png)

Cependant, vous ne devriez probablement pas modifier ces fichiers à moins de savoir ce que vous faites.

Mais si vous avez confiance en vos compétences de base en développement web, voici comment vous pouvez personnaliser un peu votre site WordPress.

### Comment ajouter du CSS personnalisé à votre site WordPress

Comme je l'ai souligné plus haut, l'ajout de CSS personnalisé à votre thème est un domaine où WordPress excelle en tant qu'outil de code.

Pour ajouter du CSS personnalisé à votre site WordPress, suivez les étapes ci-dessous :

Survolez Apparence dans le menu d'administration et sélectionnez Personnaliser :
![custCSS1](https://www.freecodecamp.org/news/content/images/2022/03/custCSS1.png)

Sélectionnez CSS additionnel :
![custCSS2](https://www.freecodecamp.org/news/content/images/2022/03/custCSS2.png)

Ajoutez le code CSS souhaité :
![custCSS3](https://www.freecodecamp.org/news/content/images/2022/03/custCSS3.png)

Le CSS que j'ai ajouté au site de démonstration rend la barre de navigation (navbar) collante (sticky) :
```css
#ast-desktop-header > div.ast-main-header-wrap.main-header-bar-wrap > div > div > div {
        position: fixed;
        top: 0;
        width: 100%;
        margin-top: .1rem;
        z-index: 1;
        background-color: #3498db 
}
```
![stickynav](https://www.freecodecamp.org/news/content/images/2022/03/stickynav.gif)

Au fait, c'est ainsi que l'on rend la barre de navigation collante avec du code dans le thème Astra.

## Conclusion

Sans aucun doute, WordPress excelle à la fois comme outil de code et outil no-code.

Si vous souhaitez vous lancer dans la création de sites web exclusivement avec WordPress, vous n'avez pas besoin de savoir écrire du code, car vous pouvez accomplir beaucoup de choses avec la partie no-code de WordPress.

J'ai personnellement commencé par construire des sites web avec la partie no-code de WordPress avant d'apprendre le HTML, le CSS et le JavaScript.

Mais si vous voulez vous lancer dans la création d'extensions et de thèmes WordPress, et résoudre des problèmes complexes, vous devez être capable de coder en HTML, CSS, JavaScript et PHP.

Si vous souhaitez apprendre à coder en HTML, CSS et JavaScript, vous devriez [consulter le programme de freeCodeCamp dès aujourd'hui](https://www.freecodecamp.org/learn/).

Si vous voulez apprendre le PHP, [ce cours vidéo](https://www.youtube.com/watch?v=OK_JCtrrv-c) de la chaîne YouTube de freeCodeCamp vous aidera à développer vos compétences de base en PHP.
