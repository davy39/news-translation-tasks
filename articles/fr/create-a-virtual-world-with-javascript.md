Apprenez à créer un monde virtuel en utilisant HTML, CSS et JavaScript.

Nous venons de publier un cours sur la chaîne YouTube de freeCodeCamp.org qui vous apprendra à créer un monde entièrement à partir de zéro. Dr. Radu Mariescu-Istodor a créé ce cours. Il est un instructeur créatif et populaire.

Vous pouvez remplir le monde virtuel avec tout ce que vous aimez. Dr. Radu démontrera comment remplir le monde avec des voitures autonomes. L'accent principal sera mis sur la génération de ce monde dynamique, avec l'intégration du code de voiture autonome à la fin. Dans ce cours, vous acquerrez des compétences précieuses, y compris les mathématiques, les algorithmes, la programmation et la conception.

Ce cours est techniquement une "Phase 2", venant après le cours extrêmement populaire de Dr. Radu sur les [voitures autonomes](https://www.youtube.com/watch?v=Rs_rAxEsAvI). Cependant, il est complètement autonome et vous n'avez pas besoin de regarder l'autre cours d'abord.

![Image](https://www.freecodecamp.org/news/content/images/2023/12/Dec-13-2023-17-33-48.gif)
_Une voiture autonome dans un monde virtuel, tout créé avec JavaScript de base._

Voici ce que vous allez apprendre :

* Construire un graphe spatial à partir de zéro et créer un éditeur avec une vue dynamique.
* Comprendre les mathématiques et la logique derrière les interfaces conviviales dans les logiciels modernes.
* Générer des bordures de route, des bâtiments et des arbres en utilisant la géométrie et créer une illusion 3D.
* Implémenter des marquages routiers pour préparer des défis avancés de conduite autonome.
* Améliorer l'éditeur de monde pour le confort de l'utilisateur et maximiser le contenu généré automatiquement.
* Charger des données du monde réel à partir d'OpenStreetMap pour des simulations basées sur des villes.
* Créer une mini-carte pour la navigation dans des mondes étendus.

Regardez le cours complet sur [la chaîne YouTube de freeCodeCamp.org](https://youtu.be/5iHejdqYIa8) (6 heures de visionnage).
