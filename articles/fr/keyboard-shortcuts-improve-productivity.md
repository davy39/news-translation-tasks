Si vous utilisez souvent un ordinateur comme moi, vos doigts passent beaucoup de temps sur un clavier. Et dans cet article, je vais vous montrer quelques raccourcis clavier utiles qui vous rendront plus productif.

## Que sont les raccourcis clavier ?

Les raccourcis clavier peuvent vous faire gagner beaucoup de temps, et ils sont souvent utilisés dans les systèmes d'exploitation modernes et les logiciels informatiques. Vous pouvez les utiliser simplement en invoquant des commandes et quelques pressions de touches. Cela vous évite d'avoir à utiliser uniquement la souris pour accéder aux éléments du menu, etc.

Voici une liste des raccourcis clavier les plus couramment utilisés – et j'en ai également inclus quelques-uns moins connus.

Bien que certains de ces raccourcis fonctionnent sur Mac, la disposition du clavier pour Mac est un peu différente. Je vais donc me concentrer sur les bonnes combinaisons de touches pour un ordinateur Windows.

## C'est parti
Commençons d'abord par les touches de fonction pour voir leurs rôles :

### Raccourcis des touches de fonction
| Touche | Fonction |
| :------------- | :----------: |
| F1 | Afficher l'aide |
| F2 | Renommer l'élément sélectionné |
| F3 | Rechercher un fichier ou un dossier |
| F4 | Afficher la liste de la barre d'adresse dans l'explorateur de fichiers |
| F5 | Actualiser la fenêtre active |
| F6 | Parcourir les éléments de l'écran dans une fenêtre |
| F10 | Activer la barre de menu dans l'application actuelle |
| F12 | Enregistrer sous (dans un document Word) |

## Autres raccourcis clavier
### Combinaisons avec la touche ==Alt== :
Vous pouvez utiliser les combinaisons ci-dessous avec la touche Alt pour effectuer les fonctions décrites :
| Touche | Fonction |
| :------------- | :----------: |
| Alt + Tab | Basculer entre les applications ouvertes |
| Alt + Shift + Tab | Basculer en arrière entre les applications ouvertes |
| Alt + F4 | Fermer le programme ouvert actuel |
| Alt + F | Options du menu Fichier dans le programme actuel |
| Alt + E | Options d'édition dans le programme actuel |

| Touche | Fonction |

| Ctrl + V | Coller le texte copié |
| Ctrl + X | Couper le texte sélectionné |
| Ctrl + Inser | Copier l'élément sélectionné |
| Ctrl + A | Sélectionner tout le texte |
| Ctrl + Début | Aller au début du contenu |
| Ctrl + Fin | Aller à la fin du document |
| Ctrl + Flèche gauche | Déplacer d'un mot vers la gauche à la fois |
| Ctrl + Flèche droite | Déplacer d'un mot vers la droite |
| Ctrl + F | Rechercher un fichier sur le bureau, mon ordinateur ou l'explorateur |
| Ctrl + Z | Annuler la dernière action |
| Ctrl + Y | Rétablir une action |

| Ctrl + Shift | Changer la disposition du clavier |

| Ctrl + (+)/Ctrl + (-) | Zoom avant et arrière, respectivement |

### Combinaisons avec la touche ==Shift== :
Vous pouvez utiliser les combinaisons ci-dessous avec la touche Shift pour effectuer les fonctions décrites :
| Touche | Fonction |
| :------------- | :----------: |
| Shift + Suppr | Supprimer l'élément sélectionné sans le déplacer vers la Corbeille |
| Shift + Flèche | Sélectionner des éléments |
| Shift + F10 | Afficher le menu contextuel |
| Shift + Inser | Coller l'élément sélectionné |
| Shift + Début | Surligner de la position actuelle au début de la ligne |
| Shift + Fin | Surligner de la position actuelle à la fin de la ligne |

Voyons maintenant les raccourcis clavier impliquant la touche avec le logo Windows :
| Touche | Fonction |
| :------------- | :----------: |
| Win + Barre d'espace | Changer la langue de saisie et la disposition du clavier |
| Win + V | Ouvrir l'historique du presse-papiers |
| Win + E | Lancer l'explorateur de fichiers |
| Win + touche Plus | Ouvrir l'outil loupe |
| Win + Entrée | Lancer le narrateur |
| Win + I | Ouvrir les paramètres de Windows |

| Win + R | Ouvrir la fenêtre Exécuter |

Les raccourcis listés ci-dessus fonctionneront sur n'importe quel ordinateur Windows, quel que soit le programme ouvert. Il existe d'autres raccourcis qui fonctionneront sur des programmes spécifiques comme MS Word, Excel, etc. – mais ceux-là feront l'objet d'un autre article.
