Apprenez à programmer en Solidity dans ce tutoriel complet de Dapp University. Solidity est un langage de programmation orienté objet pour écrire des contrats intelligents. Il est utilisé pour implémenter des contrats intelligents sur diverses plateformes de blockchain, notamment Ethereum.

Dans ce cours, vous apprendrez à écrire des contrats intelligents Ethereum pour la blockchain en utilisant le langage de programmation Solidity. Vous n'avez pas besoin de connaître quoi que ce soit sur Solidity ou la blockchain pour suivre.

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=ipwxYa-F1uY) (1,5 heure de visionnage).
