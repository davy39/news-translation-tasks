Dans ce cours, Mike Dane vous enseignera les bases de la gestion des bases de données et de SQL.

Le cours commence par Mike vous aidant à installer MySQL sur Windows ou Mac. Ensuite, il explore des sujets comme la conception de schémas, les opérations Create-Read-Update-Delete (CRUD), et d'autres fondamentaux des bases de données.

Si vous n'avez jamais étudié les bases de données ou SQL auparavant, c'est un excellent point de départ. Il couvrira :

* Qu'est-ce qu'une base de données
* Tables et clés
* Bases de SQL
* Installation de MySQL sur Windows/Mac
* Création de tables
* Insertion de données
* Contraintes
* Mise à jour et suppression
* Requêtes de base
* Introduction à la base de données d'entreprise
* Création d'une base de données d'entreprise
* Plus de requêtes de base
* Fonctions
* Caractères génériques
* Unions
* Jointures
* Requêtes imbriquées
* On Delete
* Déclencheurs
* Diagrammes ER
