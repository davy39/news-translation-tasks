Par Gitter

[Haskell](https://www.haskell.org/) est un langage de programmation fonctionnel pur, généraliste et standardisé. Il utilise une sémantique non stricte et un typage statique fort. En Haskell, la manière la plus naturelle d'écrire du code est à un haut niveau d'abstraction.

Haskell a été nommé d'après Haskell Curry, un mathématicien et logicien américain surtout connu pour son travail en logique combinatoire.

C'est un langage de programmation purement fonctionnel, donc si vous voulez apprendre la programmation fonctionnelle, Haskell est un excellent choix !

Découvrez certains de ces canaux Haskell sur [Gitter](http://gitter.im) et rejoignez la conversation :

* [**unisonweb/unison**](https://gitter.im/unisonweb/unison?source=explore) **—** [Unison](http://unisonweb.org/) est une nouvelle plateforme de programmation, actuellement en développement actif. Ce dépôt contient le code pour le backend du nœud Unison (écrit en Haskell, se trouve dans le répertoire `node`, avec le code source dans `src`), et l'éditeur Unison.
* [**atom-haskell/ide-haskell**](https://gitter.im/atom-haskell/ide-haskell?source=explore) **—** Un plugin IDE Haskell pour l'éditeur Atom.
* [**aurapm/aura**](https://gitter.im/aurapm/aura?source=explore) **—** Un gestionnaire de paquets sécurisé et multilingue pour Arch Linux et l'AUR, écrit en Haskell.
* [**gibiansky/IHaskell**](https://gitter.im/gibiansky/IHaskell?source=explore) **—** IHaskell est un noyau pour le [projet Jupyter](http://ipython.org/), qui permet d'utiliser Haskell dans les interfaces Jupyter, y compris la console et les notebooks dans le navigateur.
* [**sdiehl/wiwinwlh**](https://gitter.im/sdiehl/wiwinwlh?source=explore) — Ce que j'aurais aimé savoir en apprenant Haskell [http://dev.stephendiehl.com/hask/](http://dev.stephendiehl.com/hask/)
* [**suhailshergill/extensible-effects**](https://gitter.im/suhailshergill/extensible-effects?source=explore) **—** Extensible Effects : Une alternative aux transformateurs de monades.
* [**mstksg/auto**](https://gitter.im/mstksg/auto?source=explore) — DSL et plateforme Haskell fournissant une API dénotationnelle et compositionnelle pour des programmes interactifs, des jeux et des automatisations à étapes discrètes et à état local.
* [**guillaume-nargeot/hpc-coveralls**](https://gitter.im/guillaume-nargeot/hpc-coveralls?source=explore) — Support de Coveralls.io pour la couverture de code Haskell avec hpc.
* [**ajtulloch/dnngraph**](https://gitter.im/ajtulloch/dnngraph?source=explore) **—** DNNGraph — Un DSL de génération de modèles de réseaux de neurones profonds en Haskell.
* [**tidalcycles/Tidal**](https://gitter.im/tidalcycles/Tidal?source=explore) — TidalCycles (ou Tidal en abrégé) est un langage pour le [live coding](http://toplap.org/) de motifs. Il permet de créer des motifs musicaux avec du texte, en décrivant des séquences et des moyens de transformation. Vous pouvez ensuite les combiner et explorer des interactions complexes entre des parties simples.
* [**spell-music/csound-expression**](https://gitter.im/spell-music/csound-expression?source=explore) **—** Framework Haskell pour la musique électronique.
* [**dev-ua/haskell**](https://gitter.im/dev-ua/haskell) **—** Communauté des développeurs ukrainiens Haskell.
* [**YoEight/eventstore**](https://gitter.im/YoEight/eventstore?source=explore) **—** Client TCP EventStore pour Haskell.
* [**vu3rdd/functorrent**](https://gitter.im/vu3rdd/functorrent?source=explore) — Un client BitTorrent écrit en Haskell.
* [**nponeccop/n2o.hs**](https://gitter.im/nponeccop/n2o.hs?source=explore) — N2O en Haskell (ce canal est en langue russe).
* [**tanakh/VisualStudioHaskell**](https://gitter.im/tanakh/VisualStudioHaskell?source=explore) — Outils Haskell pour Visual Studio.
* [**eagletmt/ghcmod-vim**](https://gitter.im/eagletmt/ghcmod-vim?source=explore) — Programmation Haskell heureuse sur Vim, alimentée par [ghc-mod](https://github.com/kazu-yamamoto/ghc-mod).
* [**HaskellDC/neo4j-cypher**](https://gitter.im/HaskellDC/neo4j-cypher?source=explore) — Bibliothèque Haskell pour Neo4j axée sur Cypher.
* [**aloiscochard/grpc-haskell**](https://gitter.im/aloiscochard/grpc-haskell?source=explore) **—** Implémentation Haskell de gRPC basée sur une bibliothèque C partagée.
* [**vincenthz/cryptonite**](https://gitter.im/vincenthz/cryptonite?source=explore) **—** Cryptonite est un dépôt Haskell de primitives cryptographiques.
* [**gitter.im/fujimura**](https://gitter.im/fujimura/hi?source=explore) — Générer un échafaudage pour un projet Haskell.
* [**haskell-quantum**](https://gitter.im/ibab/haskell-quantum?source=explore) — Une monade pour simuler des processus quantiques.
