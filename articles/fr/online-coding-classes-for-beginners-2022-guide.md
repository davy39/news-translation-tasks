Apprendre à coder peut vous aider à démarrer une excellente carrière en tant que développeur logiciel. Il existe de nombreux cours gratuits que vous pouvez suivre pour cultiver les compétences nécessaires pour devenir développeur.

Dans cet article, je vais vous montrer 152 cours adaptés aux débutants où vous pouvez apprendre l'informatique et la programmation gratuitement.

Les cours de freeCodeCamp sont entièrement gratuits et certains d'entre eux incluent une certification gratuite que vous pouvez ajouter à votre LinkedIn ou à votre CV.

Notez que certains des cours non-freeCodeCamp que j'ai listés ici (comme les cours universitaires) sont gratuits à auditer, mais nécessitent le paiement de frais si vous souhaitez un certificat.

Voici la liste complète des cours gratuits, organisée par sujet. Une fois que vous avez trouvé un cours qui vous intéresse, cliquez sur ce lien et il vous amènera à une description plus détaillée de ce cours.

## **Cours d'informatique**

### **Introduction à la programmation et à l'informatique**

Dans [ce cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=zOjov-2OZ0E), vous apprendrez les tableaux, les boucles, les fonctions, la récursivité, les conditions et les structures de données.

### **Structures de données - Cours d'informatique pour débutants**

Dans [ce cours YouTube de freeCodeCamp](https://www.youtube.com/watch?v=zg9ih6SVACc), vous apprendrez les tableaux, la notation Big O, les listes chaînées, les tables de hachage, les tas et les tries.

### **CS50's Introduction to Computer Science**

[Ce cours de Harvard](https://www.edx.org/course/cs50s-introduction-to-computer-science) vous enseignera les concepts de base de l'informatique comme les algorithmes et les structures de données à travers une série de problèmes et un projet final.

### **Introduction à l'informatique et à la programmation avec Python**

Ce [cours du MIT](https://www.edx.org/course/introduction-to-computer-science-and-programming-7) vous enseignera les concepts de base de l'informatique comme les algorithmes, les structures de données, les tests et le débogage à travers une série de problèmes.

### **Introduction aux algorithmes**

Dans [ce cours du MIT](https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-006-introduction-to-algorithms-fall-2011/), vous apprendrez la programmation dynamique, le hachage, les graphes, les arbres et les algorithmes de tri.
