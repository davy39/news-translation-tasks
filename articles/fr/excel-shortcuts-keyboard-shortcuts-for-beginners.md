Il existe de nombreux outils d'analyse de données, mais Microsoft Excel reste un choix populaire pour stocker et gérer des données.

Puisqu'Excel possède de nombreuses fonctionnalités intégrées, se familiariser avec les différents raccourcis clavier vous aidera à travailler plus efficacement.

Dans cet article, je vais vous montrer certains de ces raccourcis et à quoi ils servent.

## Raccourcis clavier Excel

| Raccourci      | Description |
| ----------- | ----------- |
|**Raccourcis clavier pour faire des sélections**|
| Ctrl + A      | Sélectionne l'ensemble de la feuille de calcul       |
| Ctrl + Maj + Page up| Sélectionne la feuille de calcul actuelle et précédente        |
|| Ctrl + Maj + Page down | Sélectionne la feuille de calcul actuelle et suivante        |
| Maj + Flèche haut | Étend la sélection d'une cellule d'une cellule vers le haut |
| Maj + Flèche bas | Étend la sélection d'une cellule d'une cellule vers le bas |
| Maj + Flèche droite | Étend la sélection d'une cellule d'une cellule vers l'avant |
| Maj + Flèche gauche | Étend la sélection d'une cellule d'une cellule vers l'arrière |
| F8 | Active le mode étendu pour étendre les sélections avec la touche fléchée enfoncée|
| Entrée | Effectue une entrée de cellule et sélectionne la cellule en dessous |
| Maj + Entrée | Effectue une entrée de cellule et sélectionne la cellule au-dessus|
| Maj + Barre d'espace | Sélectionne une ligne entière |
| Ctrl + Barre d'espace | Sélectionne une colonne entière |
| Ctrl + Maj + Début | Étend la sélection de la cellule actuelle au début de la feuille |
| Ctrl + Maj + Fin | Étend la sélection de la cellule actuelle à la fin de la feuille |
| Début | Sélectionne la première cellule d'une ligne |
|**Raccourcis clavier pour effectuer des actions**|
| Ctrl + Z | Annule la dernière action |
| Ctrl + Y | Répète la dernière action |
| Ctrl + N | Crée une nouvelle feuille de calcul |
| Ctrl + B | Met en gras l'élément dans une cellule |
| Ctrl + I | Met en italique l'élément dans une cellule |
| Ctrl + U | Souligne l'élément dans une cellule |
| Ctrl + 5 | Applique un barré au contenu d'une cellule |
| Ctrl + O | Ouvre une feuille de calcul |
| Ctrl + W | Ferme une feuille de calcul |
| Ctrl + S | Enregistre une feuille de calcul |
| Ctrl + C | Copie une sélection |
| Ctrl + V | Colle une sélection |
| Ctrl + X | Coupe une sélection |
| Ctrl + 9 | Masque une ligne |
| Ctrl + 0 | Masque une colonne |
| Suppr | Supprime le contenu d'une cellule |
| Alt + H, A, C | Centre le contenu d'une cellule |
| Alt + P | Va à l'onglet Mise en page |
| Alt + N | Va à l'onglet Insertion |
| Alt + A | Va à l'onglet Données |
| Alt + W | Va à l'onglet Affichage |
| Alt + M | Va à l'onglet Formules |
| Alt + H, B | Ajoute des bordures |
| Alt + H, D, C | Supprime une colonne |
|**Raccourcis clavier pour naviguer dans les cellules**|
| Flèche haut | Déplace d'une cellule vers le haut dans une feuille de calcul |
| Flèche bas | Déplace d'une cellule vers le bas dans une feuille de calcul |
| Flèche droite | Déplace d'une cellule vers la droite dans une feuille de calcul |
| Flèche gauche | Déplace d'une cellule vers la gauche dans une feuille de calcul |
| Ctrl + Flèche | Déplace vers le bord des données dans la direction de la touche fléchée |
| Fin, flèche | Active le mode fin et déplace vers le bord dans la direction de la touche fléchée enfoncée |
| Ctrl + Fin | Déplace vers la dernière cellule d'une feuille de calcul |
| Ctrl + Début | Déplace vers la première cellule d'une feuille de calcul |
| Page up | Déplace d'un écran vers le haut |
| Page down | Déplace d'un écran vers le bas |
| Alt + Page up | Déplace d'un écran vers la gauche |
| Alt + Page down | Déplace d'un écran vers la droite |
|**Raccourcis clavier pour formater les cellules**|
| Ctrl + 1 | Ouvre la boîte de dialogue Format de cellule |
| Ctrl + Maj + F | Ouvre la boîte de dialogue pour formater les polices dans une cellule|
| F2 | Active le curseur dans une cellule |
| Maj + F2 | Insère une note |
| Ctrl + Maj + F2 | Insère un commentaire en fil de discussion |
| Ctrl + Maj + Plus (+) | Ouvre la boîte de dialogue pour insérer des cellules vides |
| Ctrl + Maj + Moins (-) | Ouvre la boîte de dialogue pour supprimer les cellules sélectionnées |
| Ctrl + Point-virgule | Insère la date actuelle |
| Ctrl + Maj + Deux-points | Insère l'heure actuelle |
| Ctrl + C | Copie la cellule sélectionnée |
| Ctrl + X | Coupe la cellule sélectionnée |
| Ctrl + V | Colle le contenu copié ou sélectionné |
| Ctrl + Alt + V | Ouvre la boîte de dialogue Collage spécial |
| Ctrl + Maj + & | Applique une bordure en contour à la cellule sélectionnée |
| Ctrl + 8 | Affiche et masque les symboles en contour |
| Ctrl + D | Remplace le contenu actuel par le contenu au-dessus |
| Ctrl + $ | Applique le format de devise à un nombre |
| Ctrl + ~ | Applique le format de nombre général |
| Ctrl + Maj + % | Applique le format de pourcentage |
| Ctrl + Maj + ^ | Applique le format de nombre scientifique |
| Ctrl + Maj + # | Applique le format de jour à un nombre avec le jour, le mois et l'année |
| Ctrl + Maj + @ | Applique le format d'heure avec l'heure, la minute et PM ou AM|
| Ctrl + Maj + ! | Applique le format de nombre avec séparateurs de milliers |
| Ctrl + K | Ouvre la boîte de dialogue pour insérer un hyperlien |
| F7 | Vérifie l'orthographe dans la feuille de calcul ou la plage sélectionnée |
| Ctrl + Q | Affiche les options d'analyse rapide |
| Ctrl + T | Affiche la boîte de dialogue Créer un tableau|
| Ctrl + Maj + G | Ouvre la boîte de dialogue Statistiques du classeur |
| **Raccourcis clavier pour travailler avec les fonctions et la barre de formule**|
| Ctrl + Maj + U | Étend et réduit la barre de formule |
| Entrée | Exécute une entrée dans la barre de formule|
| Échap | Annule une entrée dans la barre de formule|
| Maj + F9 | Calcule la feuille de calcul active |
| F9 | Calcule toutes les feuilles de calcul dans les classeurs ouverts |
| Alt + Maj + F10 | Affiche le message ou le menu pour un bouton de vérification d'erreur|
| Alt + = | Insère la formule de somme automatique |
| Maj + F3 | Insère une fonction | 
| Alt + F11 | Ouvre l'éditeur Microsoft Visual Basic pour Applications |
|**Autres raccourcis**|
| F1 | Affiche l'aide Excel|
|Ctrl + F6 | Passe d'une fenêtre Excel à l'autre |
|F7 | Ouvre la boîte de dialogue d'orthographe pour la vérification orthographique |
| Alt + F4 | Ferme Excel |
| Ctrl + F5 | Passe à la taille de fenêtre du classeur actif|
| Maj + F11 | Insère une nouvelle feuille de calcul|
|F12 | Ouvre la boîte de dialogue Enregistrer sous|
| Ctrl + F10 | Agrandit ou restaure la fenêtre du classeur sélectionné|
| Ctrl + F9 | Minimise la fenêtre du classeur |
