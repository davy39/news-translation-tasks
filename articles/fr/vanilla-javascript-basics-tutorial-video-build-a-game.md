Dans ce tutoriel rapide sur JavaScript, Marina vous guidera à travers la création de votre propre jeu de mémoire en utilisant du JavaScript vanilla (du JavaScript pur sans bibliothèques).

Elle aborde les méthodes natives de JavaScript pour sélectionner des éléments dans le DOM, comment parcourir des listes avec `forEach`, les timeouts et les écouteurs d'événements.

En cours de route, Marina expliquera également le modèle de boîte CSS et le fonctionnement des marges et des espacements, ainsi que le positionnement absolu vs relatif.

Vous pouvez regarder la vidéo de Marina sur [la chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=ZniVgo8U7ek) (34 minutes de visionnage).
