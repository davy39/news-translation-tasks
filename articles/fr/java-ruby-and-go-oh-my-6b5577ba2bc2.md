Free Code Camp s'est concentré à 100 % sur le développement full stack JavaScript depuis nos débuts il y a 17 mois. Nous avons enseigné JavaScript pour le front end, JavaScript pour le back end (grâce au [puissant framework Node.js](https://www.quora.com/What-are-the-most-famous-web-apps-built-on-top-of-Node-js/answer/Quincy-Larson)) — et même JavaScript comme langage de requêtes de base de données.

Et depuis le début, notre communauté open source a reçu plusieurs demandes chaque jour nous demandant d'enseigner d'autres langages back end populaires également.

Eh bien aujourd'hui, je suis ravi d'annoncer que nous allons répondre à ces milliers de demandes. Free Code Camp enseignera désormais une grande variété de langages de développement web.

À partir d'aujourd'hui, nous construisons une vaste collection de tutoriels sous licence Creative Commons sur des langages populaires comme Python, Java, Ruby et PHP, ainsi que sur des langages émergents comme Elixir et Go.

Et vous pouvez maintenant compléter notre Certification en Développement Back End — et construire ses dix projets d'API et d'applications web dynamiques — en utilisant les langages et frameworks de votre choix.

#### Quel est le premier de ces langages que je devrais apprendre ?

La réponse est la même qu'avant : **JavaScript**.

Comme virtuellement n'importe quel professeur vous le dirait, vous devriez apprendre un langage en profondeur avant d'essayer d'apprendre un second langage. Et JavaScript est de loin le langage le plus populaire et prometteur en ce moment. JavaScript est également un choix populaire pour un premier langage de programmation, et dispose d'une grande variété de [ressources d'apprentissage gratuites](https://medium.freecodecamp.com/25-free-resources-for-new-javascript-developers-11342cf1f4df#.tuc8r5zav) (y compris [Free Code Camp](http://www.freecodecamp.com) lui-même).

Quelle que soit la structure de développement web que vous utilisez, vous devrez devenir compétent en JavaScript. Cela est dû au fait que JavaScript a la distinction non seulement de détenir un quasi-monopole sur le développement web front end, mais aussi d'être tout à fait compétent sur le back end, grâce à des outils comme Node.js et Express.js.

Donc, si vous commencez tout juste en tant que développeur web, notre conseil reste le même : **concentrez 100 % de votre temps sur la maîtrise de JavaScript**.

#### Si les nouveaux développeurs doivent se concentrer sur le full stack JavaScript, pourquoi Free Code Camp enseignera-t-il des langages back end supplémentaires ?

Environ deux tiers de nos campeurs sont nouveaux dans le développement web. Certains d'entre eux n'ont aucune expérience préalable en programmation. D'autres rejoignent Free Code Camp avec une expérience en design web, en administration système et dans d'autres domaines connexes. Ces deux tiers de campeurs sont les personnes pour lesquelles nous avons spécifiquement conçu notre programme open source.

Mais un autre tiers de notre communauté a déjà fait du développement web — souvent avec des langages comme PHP ou Ruby. Et ils rejoignent Free Code Camp pour réviser — ou construire sur — des compétences existantes.

Et — comme vous l'avez probablement entendu — des centaines de nos campeurs ont obtenu des emplois en développement logiciel après avoir rejoint notre communauté open source. Certains de ces emplois n'étaient pas spécifiquement des emplois en full stack JavaScript, mais plutôt des emplois de développeur web full stack utilisant des frameworks de développement web alternatifs, comme Python Django ou Ruby on Rails. Après avoir accepté ces emplois, ces campeurs ont pu utiliser leurs connaissances de Node.js et Express.js pour utiliser ces autres outils. Mais beaucoup de ces campeurs ont rapporté qu'ils devaient payer pour des ressources d'apprentissage coûteuses dans le processus.

Ce tiers de notre communauté — des développeurs web expérimentés et des campeurs qui viennent d'être embauchés — ont exprimé leur désir que nous couvions des langages et frameworks back end supplémentaires. Et leur voix a été entendue.

Notre communauté open source est désormais assez grande — et nous sommes désormais assez diversifiés dans notre expertise en développement web — pour que nous puissions créer des ressources gratuites étendues sur une large gamme de sujets de développement web.

Donc, c'est ce que nous allons faire.

#### Comment ces langages seront-ils intégrés dans Free Code Camp ?

L'une des forces de Free Code Camp a toujours été que nous offrons un chemin clair et sans ambiguïté vers votre premier emploi en ingénierie logicielle.

Plutôt que de compliquer notre carte de défis avec des options, nous avons choisi de garder notre programme principal à 100 % concentré sur le full stack JavaScript.

À la place, les campeurs construisent ce contenu dans l'« Univers Étendu » de Free Code Camp.

* Nous créons une variété de tutoriels et d'articles sur ces langages — tout, depuis la configuration d'un environnement de développement sur différents systèmes d'exploitation, jusqu'à la construction d'applications exemples utilisant des bibliothèques populaires. Et ceux-ci peuvent être écrits en Markdown et interliés les uns avec les autres, directement sur notre wiki.
* Nous accueillons les campeurs pour diffuser en direct le développement web dans n'importe quel langage qu'ils souhaitent sur notre chaîne [Twitch.tv](https://twitch.tv/freecodecamp) (bientôt 24 heures sur 24).
* Nous invitons les campeurs à contribuer des articles à notre Publication Medium sur ces langages. En voici un que nous avons publié hier sur les [similitudes entre Java et Go](https://medium.freecodecamp.com/comparing-early-days-of-go-and-java-db8163bc6798).
* Nous créons des vidéos qui discutent de divers concepts spécifiques à d'autres langages, tels que le Rails Asset Pipeline et la Java Virtual Machine. Nous intégrerons ces vidéos dans des articles de wiki et les publierons sur notre [chaîne YouTube](https://www.youtube.com/freecodecamp).
