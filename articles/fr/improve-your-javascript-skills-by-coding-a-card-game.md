Réaliser des projets est une excellente façon d'améliorer vos compétences en programmation.

Nous venons de publier un cours sur la chaîne YouTube freeCodeCamp.org qui vous apprendra à créer un jeu de cartes numérique avec JavaScript, HTML et CSS. Ce cours a été créé par Gavin Lon, un développeur logiciel et enseignant expérimenté.

Ce tutoriel ne couvre pas seulement la création d'un jeu de cartes de base en utilisant JavaScript, mais aussi :

* comment créer un code interactif dynamique pour l'utilisateur en utilisant JavaScript vanilla
* Créer quelques effets d'animation
* Conception de mise en page responsive en utilisant CSS grid, CSS FlexBox et les media queries
* Comment changer dynamiquement les positions des cellules de grille, afin de randomiser les positions des cartes contenues dans une grille CSS en utilisant JavaScript
* Fonctionnalité de localStorage
* et bien plus encore…

Voici les sections de ce cours :

* Introduction
* Extension Live Server
* Créer des cartes - HTML
* Créer la grille de jeu
* Créer des cartes dynamiquement - Code JS
* Initialiser les positions des cartes
* Charger le jeu et démarrer le jeu
* Empiler les cartes
* Retourner les cartes
* Mélanger les cartes
* Distribuer les cartes
* Choisir une carte
* Style et mise en page
* Animation
* Mise en page responsive
* Stockage local

Regardez le cours sur [la chaîne YouTube freeCodeCamp.org](https://www.youtube.com/watch?v=Bj6lC93JMi0) (2 heures de visionnage).
