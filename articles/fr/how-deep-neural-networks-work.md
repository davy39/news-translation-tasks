Même si vous êtes complètement nouveau dans le domaine des réseaux de neurones, ce cours de Brandon Rohrer vous rendra à l'aise avec les concepts et les mathématiques qui les sous-tendent.

Les réseaux de neurones sont au cœur de ce que nous appelons aujourd'hui l'intelligence artificielle. Ils peuvent sembler impénétrables, voire mystiques, si vous essayez de les comprendre pour la première fois, mais ils n'ont pas à l'être.

Voici les sujets que vous apprendrez dans ce cours :

* Comment fonctionnent les réseaux de neurones
* Ce que les réseaux de neurones peuvent apprendre et comment ils l'apprennent
* Comment fonctionnent les réseaux de neurones convolutifs (CNN)
* Comment fonctionnent les réseaux de neurones récurrents (RNN) et la mémoire à long et court terme (LSTM)
* Le deep learning démystifié
* Se rapprocher de l'intelligence humaine grâce à la robotique
* Comment fonctionnent les CNN, en profondeur

Vous pouvez regarder le cours vidéo complet sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=dPWYUELwIdM) (4 heures de visionnage).
