Git et GitHub sont fréquemment utilisés par les développeurs partout dans le monde. Ces outils sont utilisés dans la plupart des entreprises de logiciels et il est donc important de les comprendre si vous voulez un emploi dans l'industrie du logiciel. 

Nous avons publié une vidéo de cours accéléré de [Gwen Faraday](https://twitter.com/faradayacademy) qui vous enseignera les bases de Git et GitHub.

Git est un logiciel qui simplifie la gestion des différentes versions de votre propre logiciel dans un dépôt de logiciels. GitHub est un endroit pour partager des dépôts de logiciels et collaborer avec d'autres développeurs. Ces outils facilitent le travail de plusieurs personnes sur le même projet logiciel.

Dans ce cours, vous apprendrez :

* les termes courants
* l'installation de Git
* la création d'un dépôt Git
* le clonage d'un dépôt Git
* l'ajout et la suppression d'un dépôt Git
* l'utilisation de GitHub
* la création de demandes de tirage (pull requests)
* le branchement des dépôts
* la fusion des dépôts
* et plus encore

À la fin du cours, vous en saurez _beaucoup_ plus sur Git que les personnes dans cette bande dessinée web. :)
