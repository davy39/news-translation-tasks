Parfois, vous pourriez avoir besoin de connaître l'adresse IP de votre routeur. Cela vous donnera accès à la page de configuration afin que vous puissiez apporter des modifications de configuration comme changer les mots de passe, expulser les utilisateurs indésirables, limiter les utilisateurs, et ainsi de suite.

Dans cet article, je vais vous montrer comment découvrir quelle est l'adresse IP de votre routeur sur un ordinateur Windows et un smartphone Android.

## Qu'est-ce qu'une adresse IP ?

Mais avant de commencer, clarifions – qu'est-ce qu'une adresse IP ?

L'abréviation IP signifie « internet protocol ». Le protocole internet est un ensemble de règles régissant la manière dont les ordinateurs, les téléphones et autres appareils partagent des données sur internet ou les réseaux locaux.

Une adresse IP est un identifiant numérique spécial qui permet l'envoi d'informations entre divers appareils sur un réseau.

## Comment trouver votre adresse Wifi sur un ordinateur Windows

Pour découvrir quelle est l'adresse IP de votre routeur (ou de tout autre appareil auquel vous avez accès via son Wifi), vous pouvez soit utiliser l'invite de commande, soit y accéder via le panneau de configuration.

### Comment trouver votre adresse Wifi sur un ordinateur Windows via l'invite de commande

**Étape 1** : Cliquez sur Démarrer (logo Windows) ou appuyez sur la touche `WIN` de votre clavier.
**Étape 2** : Recherchez « cmd » et appuyez sur `ENTRÉE` pour lancer l'invite de commande.
![ss-1-6](https://www.freecodecamp.org/news/content/images/2021/10/ss-1-6.jpg)

**Étape 3** : Dans l'invite de commande, tapez « ipconfig » et appuyez sur Entrée.
Le numéro assigné à « Default Gateway » est l'adresse IP de votre routeur.
![ss-2-7](https://www.freecodecamp.org/news/content/images/2021/10/ss-2-7.jpg)

### Comment trouver votre adresse Wifi sur un ordinateur Windows via le panneau de configuration

**Étape 1** : Cliquez sur Démarrer et recherchez « Panneau de configuration », puis cliquez sur le premier résultat de recherche ou appuyez sur `ENTRÉE` pour lancer le Panneau de configuration.
![ss-3-5](https://www.freecodecamp.org/news/content/images/2021/10/ss-3-5.jpg)

**Étape 2** : Sous « Réseau et Internet », cliquez sur « Afficher l'état et les tâches du réseau ».
![ss-4-6](https://www.freecodecamp.org/news/content/images/2021/10/ss-4-6.jpg)

**Étape 3** : Vous verrez le nom de votre routeur sur la droite. Cliquez dessus et une petite fenêtre s'ouvrira.
![ss-5-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-5-2.jpg)

**Étape 4** : Dans la boîte de dialogue, cliquez sur « Détails ».
![ss-6-4](https://www.freecodecamp.org/news/content/images/2021/10/ss-6-4.jpg)

Après avoir cliqué sur le bouton « Détails », une autre petite fenêtre s'ouvrira, affichant diverses informations sur votre routeur. La valeur assignée à IPv4 est l'adresse IP de votre routeur.
![ss-7-2](https://www.freecodecamp.org/news/content/images/2021/10/ss-7-2.jpg)

## Comment trouver votre adresse Wifi sur un smartphone Android

Les appareils Android ne disposent pas d'une option intégrée pour vérifier les adresses du routeur dès la sortie de l'usine. Mais vous pouvez utiliser une application tierce comme Wifi Analyzer.

Si vous avez installé Wifi Analyzer sur votre smartphone Android, lancez l'application.
![InkedScreenshot_20211020-105514-2](https://www.freecodecamp.org/news/content/images/2021/10/InkedScreenshot_20211020-105514-2.jpg)

Vous verrez immédiatement l'onglet du graphique des canaux.
![InkedScreenshot_20211020-105529](https://www.freecodecamp.org/news/content/images/2021/10/InkedScreenshot_20211020-105529.jpg)

Faites glisser vers l'onglet des points d'accès et vous verrez l'adresse IP de votre routeur.
![InkedScreenshot_20211020-105543](https://www.freecodecamp.org/news/content/images/2021/10/InkedScreenshot_20211020-105543.jpg)
