Apprenez à créer un jeu de plateforme en utilisant JavaScript vanilla.

Ce tutoriel commence par enseigner comment organiser le code en utilisant la stratégie Modèle, Vue, Contrôleur (MVC) et les principes de la programmation orientée objet (POO). Ensuite, il montre comment programmer le mouvement, dessiner une carte de tuiles et détecter les collisions. Enfin, le tutoriel enseigne comment animer les sprites, charger des niveaux et collecter des objets.

Après avoir regardé, vous serez en mesure d'appliquer les mêmes principes pour créer un jeu qui vous est propre.

Vous pouvez regarder la vidéo sur la [chaîne YouTube de freeCodeCamp.org](https://www.youtube.com/watch?v=w-OKdSHRlfA) (3 heures de visionnage).
