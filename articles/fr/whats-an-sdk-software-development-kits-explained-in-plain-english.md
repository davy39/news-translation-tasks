Un kit de développement logiciel (SDK) est la boîte à outils du monde du développement logiciel. Mais au lieu d'outils comme des tournevis et des pinces, les SDK contiennent des bibliothèques et des API (Interfaces de Programmation d'Applications) qui facilitent la création d'applications pour des logiciels et du matériel spécifiques.

Certains SDK sont nécessaires pour développer une application spécifique à une plateforme. Par exemple, le développement d'applications Android en Java nécessite le Java Development Kit, les applications iOS nécessitent l'iOS SDK, et pour la plateforme Universal Windows Platform, le SDK .NET Framework.

Tout cela est réalisé grâce aux bibliothèques incluses dans chaque SDK. Une bibliothèque est essentiellement un morceau de logiciel qui permet à votre application de communiquer avec la plateforme spécifique sur laquelle elle s'exécute.

Des outils de débogage et de création d'éléments visuels sont souvent inclus dans les SDK, ce qui facilite la correction des applications ou la création de zones de texte et de boutons par les développeurs. Souvent, les SDK incluent également du code exemple afin que les développeurs puissent rapidement se familiariser avec la plateforme pour laquelle ils codent.
