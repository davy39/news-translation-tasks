Learn to use Godot by building a 2D platformer game in this tutorial for absolute beginners from Awesome Tuts. Godot is a 2D and 3D cross-platform game engine released as open-source software.

In the process of creating a simple game called "Monster Chase", you will learn the basics of Godot that you can start applying to your own games. You will learn how to create and control a player, create monsters, add monster AI, and more.

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=fQXKjmCDkIA) (1 hour watch).
