In a new 4.5 hour course from GTCoding, you will learn to create a completely responsive website from scratch using WordPress, HTML, CSS, JavaScpript, and PHP. The course guides you through creating a custom WordPress theme.

In addition to giving you a solid understanding of WordPress theme development, the course also covers planning, UI Design, and the basics of PHP.

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=KibbYf9avko) (4.5 hour watch).
