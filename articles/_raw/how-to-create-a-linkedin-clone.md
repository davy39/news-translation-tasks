Are you interested in learning how to create a social networking site like LinkedIn using React and Firebase?

We just published a course on the freeCodeCamp.org YouTube channel that will help you learn how to build a LinkedIn clone from start to finish, using the popular front-end library React and the powerful backend-as-a-service Firebase.

The course begins with an introduction and then dives into the base setup for the project. From there, you'll learn how to create a log in and sign up page and manage authentication states. You'll also build the top bar of the site and add posts to Firebase. Throughout the course, you'll gain experience with Firebase's real-time database and authentication services.

As you progress, you'll learn how to build a profile page and implement features like a likes system and a comments system. You'll also learn how to upload images to Firebase and design a file upload modal. Other topics covered in the course include small profile icons, updating and deleting posts, building a connections page, and implementing user search functionality.

By the end of the course, you'll have a complete LinkedIn clone that you can modify and use as the basis for your own projects. You'll also have gained valuable experience with React and Firebase, two of the most popular technologies in web development.

The course is taught by an Nishant Singh. He is an experienced instructor who will guide you through each step of the process. The course is well-structured and easy to follow, making it accessible for developers of all levels. With this course, you'll be able to create your own social networking sites and develop your skills in React and Firebase.

Overall, if you're looking to learn how to create a LinkedIn clone using React and Firebase, this course is an excellent place to start. With its comprehensive content and knowledgeable instructor, you'll be able to take your skills in web development to the next level.

Watch the full course on [the freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=HimR8Xtz17U&t=199s) (11-hour watch).
