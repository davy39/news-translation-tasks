## Essential Tools I Discovered on WebCurate

Here‚Äôs a rundown of a few tools that I found interesting on WebCurate‚Äîsome are longtime favourites, while others were discoveries that have quickly become essentials.

#### **Tools I Already Use (and Still Love)**

* [**Supabase**](https://supabase.com/): An open-source Firebase alternative that simplifies backend development with real-time updates and intuitive database management.
    
* [**GSAP**](https://gsap.com/): A powerful animation library for creating smooth, high-performance animations ([I even wrote an article about how I use it](https://www.freecodecamp.org/news/build-an-interactive-accordion-component-with-react-and-gsap/)!).
    
* [**JsonViewer**](https://chromewebstore.google.com/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh): A practical tool for viewing and formatting JSON data, making debugging and visualization easier.
    
* [**CanIUse**](https://caniuse.com/): An essential resource for checking browser compatibility across HTML5, CSS3, SVG, and more‚Äîperfect for ensuring cross-platform support.
    

#### **New Tools I Use Often or Plan on Using Soon**

* [**Pesticide for Chrome**](https://chromewebstore.google.com/detail/pesticide-for-chrome/bakpbgckdnepkmkeaiomhmfcnejndkbi?hl=en): A Chrome extension that outlines page elements, helping spot layout issues and verify alignment quickly.
    
* [**WindUI**](https://wind-ui.com/): A component library styled with Tailwind CSS, that speeds up development without compromising design quality.
    
* [**Mobbin**](https://mobbin.com/): A curated collection of mobile design patterns, offering inspiration to refine and elevate mobile UI elements.
    
* [**Mightly Meld**](https://www.mightymeld.com/): A collaborative design tool for React that makes feedback gathering and design iteration faster and more streamlined.
    
* [**Rowy**](https://www.rowy.io/): A no-code platform connecting directly to databases, allowing for rapid development and deployment of data-driven applications without extensive code.
