User Stories, Epics, and Personas help organize all the needs of a project so it is clear what needs to be done. They are used in many developer jobs for planning projects so it is helpful to know what they are and how to create them.

In this video, Anissa Deanna explains user stories, epics, and personas and gives examples of how to create them for your own projects.
