Learn to code a magic card trick using JavaScript, Node.js, Express, and MongoDB in this tutorial.

First, you'll see the trick performed in real life. Then, you'll learn how to code it from scratch and deploy it using Heroku. This is a fun card trick you can perform for your friends. And the tutorial may even give you some ideas to create your own magic tricks.

This trick is a great use for the freeCodeCamp.org [Programer Playing Cards](https://www.freecodecamp.org/news/programmer-playing-cards/)!

You can watch the full video tutorial on the [freeCodeCamp.org YouTube channel](https://youtu.be/h21pa3yeW08) (30 minute watch).
