By Nick Karnik

Here are GitHub Extensions that I use. They will enable you to improve your productivity on GitHub. Please share your favorite extensions in the comments.

[**Octotree**](https://github.com/buunguyen/octotree) **allows easy exploration of the source code within the browser through a panel on the left:**

![Image](https://cdn-media-1.freecodecamp.org/images/K91lj5pBvV8HsN0TX37LruGJCDJG1blhqOht)

[**github-dashboard**](https://github.com/muan/github-dashboard) **provides the ability to filter events on the Github.com activity dashboard**

![Image](https://cdn-media-1.freecodecamp.org/images/iN1AAIkMUQTO2tm3ks1g8lYaaAa9M4fXpcuX)

[**Sourcegraph**](https://about.sourcegraph.com/) **allows going to the definition and finding references within the code**

![Image](https://cdn-media-1.freecodecamp.org/images/34Q7gJSeHq3DelG3YEvimvn7d5ipOdqBVJjj)

[**Octolinker**](https://github.com/OctoLinker/OctoLinker) **allows navigating through the code more efficiently by turning module imports into clickable links so you can easily view their source**

![Image](https://cdn-media-1.freecodecamp.org/images/bod3D8IorChh9HIRZwTdyjELHSXqARbIEkUm)

[**github-repo-size**](https://github.com/harshjv/github-repo-size) **displays repository size and individual file sizes on GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/Qsrd1b3RRBr9mSvSJiITQSrSfJuflTS0yKwX)

[**refined-github**](https://github.com/sindresorhus/refined-github) **simplifies the GitHub interface and adds several useful features**

**_No more jumps caused by recently pushed branches (They are moved to the side)_**

![Image](https://cdn-media-1.freecodecamp.org/images/rEZM8tEWaLOAcumeAyigcRIRROL2lktOa6wn)

**_Reaction avatars showing who reacted to a comment_**

![Image](https://cdn-media-1.freecodecamp.org/images/ofubT38pFl9a1t1qYwaO-e3sSSVgiZk4A4RX)

**_The option to wait for checks when merging a P_**

![Image](https://cdn-media-1.freecodecamp.org/images/-JrEtND7QVMoeFZhN0bkl2CfDqjN9hwP48Pc)

**_Clickable references to issues and pull requests_**

![Image](https://cdn-media-1.freecodecamp.org/images/Nfyrn7vHoEy0JD83y-77SEUUtd4dvp3izwaD)

**_Links to an issue’s closing commit or pull request_**

![Image](https://cdn-media-1.freecodecamp.org/images/BnipoAfM0PazABnMq3vvP1rJJplOyLi5Y23Q)

[**ZenHub**](https://www.zenhub.com/) **is a better way to manage your GitHub Issues. It supports Multi-repo Boards, Epics, and reports — all inside of GitHub**

![Image](https://cdn-media-1.freecodecamp.org/images/vC0BibG0DNmiPTvvg1jRYtyTNmslgpLNhiJg)

[**github-npm-stats**](https://github.com/katranci/github-npm-stats) **displays npm download stats on a GitHub repo**

![Image](https://cdn-media-1.freecodecamp.org/images/XA3PgTHXVBEgwgGypM2LiKWIqZytzToWFATh)

[**npmhub**](https://github.com/npmhub/npmhub) **displays npm dependencies on GitHub repos**

![Image](https://cdn-media-1.freecodecamp.org/images/gxzbn81yIRTpC2YYC280lb50K6vqnwiJqZud)

[**isometric-contributions**](https://github.com/jasonlong/isometric-contributions) **renders an isometric pixel art version of GitHub contribution graphs**

![Image](https://cdn-media-1.freecodecamp.org/images/kY4gzq0z4qeboA9gRBy2NfA6dzsPL-VPGZSu)
