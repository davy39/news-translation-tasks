In this comprehensive, 24-hour-long course from Tim Corey, you will learn to use C# to build a complete application from start to finish.  

The course will walk you through the entire process of designing and building a real-world application. Using .net and Visual Studio, you will learn to construct a tournament tracker application that is fit to launch. 

Here are some of the topics covered in this course:

* Interfaces
* Email
* SQL
* Text Files
* App.config data storage and retrieval
* OOP

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=wfWxdh-_k_4) (24 hour watch).
