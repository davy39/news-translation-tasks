Learn how to use Angular Material in this full course for beginners from Codevolution. 

Angular Material provides a set of reusable and accessible UI components based on Google’s Material Design specification. The components follow modern web design principles like browser portability, device independence, and graceful degradation.

This course covers:

* Material Module
* Typography
* Button
* Button Toggle
* Icons
* Badges
* Progress Spinner
* Navbar
* Sidenav
* Menu
* List
* Grid List
* Expansion Panel
* Cards
* Tabs
* Stepper
* Input
* Select
* Autocomplete
* Checkbox and Radio Button
* Date Picker
* Tooltip
* Snackbar
* Dialog
* Data table
* Exploring Data table
* Data table Filtering
* Data table Sorting
* Data table Pagination
* Virtual Scrolling

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=jUfEn032IL8) (3 hour watch).
