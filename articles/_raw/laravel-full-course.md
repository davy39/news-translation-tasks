Learn Laravel by creating an Instagram clone in this full course for beginners developed by Coder's Tape. Laravel is a free, open-source PHP web framework used for creating web applications. This course will prepare you to use Laravel for your own web projects.

This course covers topics such as:

* What is Laravel?
* Installing Laravel
* Setting up views and controllers
* Authentication with Laravel
* Front End with Node and NPM
* Using SQLite with Laravel
* RESTful Resource Controllers
* Passing Data to Views
* Route Model Binding
* Laravel Telescope
* Sending Emails to Users
* and more...

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=ImtZ5yENzgE) (4.5 hour watch).
