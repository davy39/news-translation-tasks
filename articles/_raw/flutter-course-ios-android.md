Learn Flutter in this full course from Nick Manning (of fluttercrashcourse.com). Flutter is Google’s multi-platform mobile development framework used to create apps for Android and iOS using the Dart programming language. 

In this course, Nick teaches you how to get up and running with Flutter quickly and effectively. Nick has been a Flutter engineer since 2017 and has boiled down this real-world know-how into a clear-cut, realistic and step-by-step crash course.

This course covers:

* Flutter basics
* Layout
* Images
* Text
* Models
* Navigation and lists

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=pTJJsmejUOQ) (2 hour watch).
