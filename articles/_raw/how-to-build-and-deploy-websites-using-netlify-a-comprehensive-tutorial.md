Netlify helps developers quickly roll out static websites. Learn how to use Netlify in this comprehensive video course for beginners created by swyx. 

Netlify can be used for everything needed in modern web development, from local setup to global deployment. This course is the perfect starting place for anyone learning Netlify.

The course covers the following topics:

* Netlify Edge
* Netlify Build
* Netlify Dev
* Netlify Forms
* Netlify Identity
* NetlifyCMS
* Netlify Addons: Everything Else!
* Netlify API
* Where to learn more

You can watch the full video below or on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=mT5siI19gtc) (3.5 hour watch).
