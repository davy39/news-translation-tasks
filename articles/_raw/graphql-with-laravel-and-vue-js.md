Learn how to use GraphQL with Laravel and Vue.js in this course from Andre Madarang. The course uses the Lighthouse package on the backend with Laravel and Vue Apollo on the frontend. You'll learn how to build a book recommendation app from scratch as a project. 

Here are the topics covered in the course:

* Lighthouse Package Backend
* Vue Apollo Queries
* Vue Apollo Mutations
* Tailwind CSS Styling
* Deployment
* GraphQL Client in Laravel
* GraphQL Laravel server w/ graphql-laravel
* GraphQL Authentication in Laravel w/ Lighthouse & Passport
* GraphQL Authentication in Vue Apollo

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=4z3EMCc4bP4) (5 hour watch).
