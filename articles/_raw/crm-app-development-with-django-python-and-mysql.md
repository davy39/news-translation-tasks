Are you interested in learning how to build a customer relationship management (CRM) app with Django, Python, and MySQL?

We just published a course on the freeCodeCamp.org YouTube channel that will guide you step-by-step on how to build a fully-functional CRM application using Django, Python, and MySQL.

John Elder developed this course. He is an experienced developer and popular course creator. He has worked with a variety of programming languages, including Python, and is passionate about teaching others how to code. With his guidance, you'll be able to confidently complete this course and build your own CRM app.

Throughout the course, you will learn how to build a fully functional CRM app that utilizes MySQL for the database. You will learn how to register, log in, log out, add records, view records, update records, and delete records. By the end of the course, you will have a strong understanding of how to create a basic app using these technologies.

The course starts with an introduction to the technology and software needed for the project. You will learn how to install and set up the app and configure MYSQL. Additionally, the course covers version control with Git and Github.com.

From there, the course dives into building out the basic app. You will learn how to log in, log out, and register users, as well as set up the database model. The course then covers how to view records on the website using a Bootstrap table and how to display individual records using a Bootstrap card.

The course also covers how to add new records, update records, and delete records. Additionally, you will learn how to push code to Github for version control.

By the end of this course, you will have a solid understanding of how to build a basic CRM app with Django, Python, and MySQL. You'll be able to use your newfound skills to build similar apps for yourself or your clients.

In conclusion, if you're looking to expand your coding skills and learn how to build a CRM app with Django, Python, and MySQL, this video course is perfect for you. With step-by-step instructions and guidance from an experienced developer, you'll be well on your way to creating your own fully functional app.
