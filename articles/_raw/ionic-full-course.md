Learn how to use Ionic 4 in this full course for beginners from Awais Mirza. Ionic Framework is the free, open source mobile UI toolkit for developing high-quality cross-platform apps for native iOS, Android, and the web—all from a single JavaScript codebase.

In this tutorial you will learn how to build a mobile app with Ionic and Angular from scratch. You will learn the basic concepts of Ionic and the Ionic CLI.

Here are a few of the topics covered:

* Ionic Installation
* Project Structure
* Emulating iOS and Android
* Routing and Lazy Loading
* Action Sheet
* Components
* ION Alert and Documentation
* Ionic Native
* Debugging
* Native HTTP Plugins
* Native SMS Plugins
* Native Storage

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=AvbuIRg8_Jg) (3 hour watch).
