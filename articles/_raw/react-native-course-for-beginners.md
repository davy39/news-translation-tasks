React Native allows you to use JavaScript to write apps that run natively on Android, iOS, and Blackberry. This full introductory course from _codedamn_ will teach you the basics of React Native and help you create your first React Native app.

Topics covered in this course are:

* Setting up React Native
* File Structure
* Hello World
* Hello World Explained
* Props
* State
* Stylesheets and Flexbox Layouts
* Getting user text
* Uncontrolled vs. controlled components
* Creating a Calculator
