Creating your own game is easier than ever! GDevelop is a beginner-friendly game engine that helps you create 2D games with little to no code.

We just published a course on the freeCodeCamp.org YouTube channel that will teach you how to use GDevelop to create 2D games.

Wesley and Max developed this course. Both of them are game developers and course creators. They are on the GDevelop team.

In less than an hour you will learn enough to start creating your own games.

[Click here to try out a simple game created using GDevelop](https://games.gdevelop-app.com/game-798e7afe-03c9-4492-8974-28c6076eb896/index.html).

![Image](https://www.freecodecamp.org/news/content/images/2022/01/Screen-Shot-2022-01-27-at-1.19.31-PM.png)
_Game created using GDevelop._

Here are the topics covered in this course:

* General Layout
* Object Types
* Event System
* Jumpstarting
* Variables
* Expression Builder
* Extensions
* Re-creating Asteroids
* Exporting A Game

Watch the full course below or on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=iHF5fwsqu4I) (1-hour watch).
