MongoDB is the most popular NoSQL / document database. In this quickstart course you'll be up and running with MongoDB and Python.

Your teacher is Michael Kennedy. He's the host of the Talk Python to Me Podcast and he works directly with the MongoDB team to help make it easier for people to use MongoDB.

![Image](https://cdn-media-1.freecodecamp.org/ghost/2019/02/download.png)

Here are some of the topics covered in the course:

* Why NoSQL and MongoDB
* Modeling data in document databases
* Introducing MongoEngine
* Building the data layer with MongoEngine

By the end of the course, you'll be ready to start using MongoDB in your own Python projects.

You can watch the full video on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=E-1xI85Zog8) (2 hour watch).
