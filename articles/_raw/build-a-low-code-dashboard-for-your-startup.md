Creating a dashboard to keep track of customers and users is much easier than you may think. You can build an API-powered marketing dashboard using a low code approach. 

We just published a course on the freeCodeCamp.org YouTube channel that will teach you how to create a dashboard to help manage your site or startup.

In this course you will learn how to use low code tools and APIs to build a customer support dashboard. You will learn how create the dashboard using the Appsmith platform, MongoDB, the SMTP API, and the Discord web hook.

You will learn how to quickly build tools that many startups use by using the low-code approach.

Appsmith provided a grant that made this course possible.

![image-60](https://www.freecodecamp.org/news/content/images/2022/04/image-60.png)
_Low-code tool_

When watching this course, it is helpful if you already have basic knowledge of JavaScript and HTTP request methods. However, you should still be able to follow along if you are new to the concepts because Ania will guide you step-by-step through the process.

Here are the sections covered in this course:

* Getting started with Appsmith
* Connected to MongoDB
* Adding data to Appsmith app
* Create send messages page
* Show different marketing lists
* Implementing Discord web hook

Watch the full course below or on [the freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=ccfpj4eD6YM) (2-hour watch).
