React Native for Web allows you to build apps that works on iOS, Android, and the web using the same JavaScript codebase. 

In this full course from Ben Awad, you will learn to create a workout app using React Native for Web.

This course covers:

* React Native Web
* Mobx
* Typescript
* React Navigation
* React Hooks
* AsyncStorage / Localstorage
* deploying with Netlify
* and more

You can watch the full video course on the [freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=_CBYbEGvxYY) (4 hour watch).
