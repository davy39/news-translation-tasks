Learn to analyize tweets in this Python Tutorial. Vincent Russo shows how to use the Tweepy module to stream live tweets directly from Twitter in real-time. The tweets are visualized and then the TextBlob module is used to do sentiment analysis on the tweets.

Here the sections of the video:

* Streaming live tweets
* Cursor and pagination
* Analyizing tweet data
* Visualizing tweet data
* Sentiment analysis

You can watch the video on the [the freeCodeCamp.org YouTube channel](https://www.youtube.com/watch?v=1gQ6uG5Ujiw) (90 minute watch).
