My JavaScript Basics course is now live on [freeCodeCamp’s YouTube channel](https://www.youtube.com/channel/UC8butISFwT-Wl7EV0hUK0BQ).

The great thing about this course is that it also aligns with the Basic JavaScript Challenges on the [freeCodeCamp curriculum](https://learn.freecodecamp.org/javascript-algorithms-and-data-structures/basic-javascript). You can practice what you're learning through the videos using the freeCodeCamp challenges. (The course even teaches some concepts that aren’t yet covered in the freeCodeCamp curriculum.)

These videos are designed to quickly give you key information on each topic. Also, I designed them to be as modular as possible. They work fine as stand-alone videos, if you just want to learn about a single topic.

Besides the JavaScript Basics course, I’ve also developed playlists for ES6, Clean Code, Data Structures, Design Patterns, and jQuery. Below are links and descriptions of the videos to help you quickly find exactly what you want to learn.
